Spring Tool Suite™3

为了您的方便，我们将继续提供Spring Tool Suite 3（上一代Spring Tools）的更新和下载。Spring Tools Suite 3继续作为定制的基于Eclipse的多合一发行版发布，这使应用程序开发变得容易。该工具套件提供了语言支持，框架支持和运行时支持的即用型组合，并将它们与Eclipse中的现有Java，Web和Java EE工具相结合。该发行版中集成的仪表板使其易于安装进一步的扩展，从而增加了对Pivotal Cloud Foundry，Gradle或Pivotal tc Server等技术的支持。
始终可以通过GitHub Issues或StackOverflow标签获得对这些发行版的支持。

Spring Tool Suite 3将在2019年中期达到其停产期。

Spring Tool Suite™3

Spring Tool Suite 3是基于Eclipse的开发环境，为开发Spring应用程序而定制。它提供了一个现成的环境来实现，调试，运行和部署您的Spring应用程序，包括与Pivotal tc Server，Pivotal Cloud Foundry，Git，Maven，AspectJ等的集成。
