Spring Cloud Zookeeper

2.2.0立方米
  • 总览
  • 学习
  • 样品

Spring Cloud Zookeeper通过自动配置并绑定到Spring Environment和其他Spring编程模型习惯用法，为Spring Boot应用程序提供了Apache Zookeeper集成。通过一些简单的注释，您可以快速启用和配置应用程序内的通用模式，并使用Zookeeper构建大型分布式系统。提供的模式包括服务发现和分布式配置。

特征

  • 服务发现：实例可以在Zookeeper中注册，客户端可以使用Spring托管的Bean发现实例

  • 通过Spring Cloud Netflix支持Ribbon，客户端负载均衡器

  • 通过Spring Cloud Netflix支持Zuul，动态路由器和过滤器

  • 分布式配置：将Zookeeper用作数据存储

快速开始

只要Spring Cloud Zookeeper， Apache Curator和Zookeeper Java Client在类路径上，任何具有@EnableDiscoveryClient Spring Boot应用程序@EnableDiscoveryClient将尝试与localhost:2181 （ zookeeper.connectString的默认值）上的Zookeeper代理联系。

@Configuration
@EnableAutoConfiguration
@EnableDiscoveryClient
@RestController
public class Application {

  @RequestMapping("/")
  public String home() {
    return "Hello World";
  }

  public static void main(String[] args) {
    SpringApplication.run(Application.class, args);
  }

}

本地Zookeeper服务器必须正在运行。有关如何运行Zookeeper服务器的信息 ，请参见Zookeeper文档 。

快速开始

使用Spring Initializr引导您的应用程序。
