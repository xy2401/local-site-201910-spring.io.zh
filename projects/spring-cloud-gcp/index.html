Spring Cloud GCP

1.2.0立方米
  • 总览
  • 学习
  • 样品

Spring Cloud GCP项目使Spring Framework成为Google Cloud Platform（GCP）的一等公民。

特征

Spring Cloud GCP项目提供以下功能：

  • Spring Cloud GCP发布/订阅，包括Spring集成通道适配器

  • Spring Cloud GCP Pub / Sub流活页夹

  • 适用于Google Cloud Storage的Spring Resource Abstraction，包括Spring Integration Channel Adapters

  • Spring Data Cloud SQL

  • Spring Data Cloud Spanner

  • Spring Data Cloud数据存储

  • Google Cloud Vision API模板

  • Google Cloud Stackdriver记录与追踪

  • Google Cloud Config（测试版）

春季初始化

Spring Initializr for Spring Cloud GCP中有三个条目。

GCP支持

GCP支持条目包含对每个Spring Cloud GCP集成的自动配置支持。仅当将其他依赖项添加到类路径时，才启用大多数自动配置代码。

Spring Cloud GCP入门 所需的依赖项

设定档

org.springframework.cloud:spring-cloud-gcp-starter-config

云扳手

org.springframework.cloud:spring-cloud-gcp-starter-data-spanner

云数据存储

org.springframework.cloud:spring-cloud-gcp-starter-data-datastore

记录中

org.springframework.cloud:spring-cloud-gcp-starter-logging

SQL-MySQL

org.springframework.cloud:spring-cloud-gcp-starter-sql-mysql

SQL-PostgreSQL

org.springframework.cloud:spring-cloud-gcp-starter-sql-postgres

跟踪

org.springframework.cloud:spring-cloud-gcp-starter-trace

视力

org.springframework.cloud:spring-cloud-gcp-starter-vision

安全性-IAP

org.springframework.cloud:spring-cloud-gcp-starter-security-iap

GCP讯息

“ GCP消息传递”条目添加了“ GCP支持”条目和所有必需的依赖项，因此Google Cloud Pub / Sub集成即开即用。

GCP储存

GCP存储条目会添加GCP支持条目和所有必需的依赖项，以便Google Cloud Storage集成即开即用。
