Spring for Apache Hadoop

2.5.0
  • 总览
  • 学习
  • 样品

注意：Spring for Apache Hadoop项目将于2019年4月5日达到生命周期终止状态。到那时为止，我们将根据需要偶尔发布2.5.x维护版本，然后将项目移至阁楼。当前用于Apache Hadoop 2.5.0的Spring版本是使用Apache Hadoop 2.7.3版构建的，并且应该与最受欢迎的Hadoop发行版的最新版本兼容。

介绍

用于Apache Hadoop的Spring通过提供统一的配置模型和易于使用的API来使用HDFS，MapReduce，Pig和Hive简化了Apache Hadoop的开发。它还提供与其他Spring生态系统项目的集成，例如Spring Integration和Spring Batch，使您能够开发用于大数据接收/导出和Hadoop工作流程编排的解决方案。

查阅O'Reilly Media的《 Spring Data：Enterprise Java的现代数据访问 》一书，其中包含有关将Spring用于Apache Hadoop的几章。本书的示例代码也可以在GitHub项目spring-data-book中找到 。

特征

  • 支持创建使用Dependency Injection配置并作为标准Java应用程序（与使用Hadoop命令行实用程序相比）运行的Hadoop应用程序。

  • 与Spring Boot集成以仅创建连接到HDFS的Spring应用程序以读取和写入数据。

  • 创建和配置使用Java MapReduce，流，Hive，Pig或HBase的应用程序

  • Spring Batch的扩展，以支持为任何类型的Hadoop Job或HDFS操作创建基于Hadoop的工作流。

  • 使用任何基于JVM的脚本语言编写HDFS操作脚本。

  • 轻松创建基于Spring Boot的自定义应用程序，可以将其部署为在YARN上执行。

  • HBase的DAO支持（模板和回调）。

  • 支持Hadoop安全性。

版本和发行支持

适用于Apache的Spring Hadoop支持许多Apache版本以及Pivotal，Hortonworks和Cloudera的商业发行版。

支持的发行版因发行版本而异，有关详细信息，请参见Wiki页面 。

另外，请参阅Wiki页面以获取Maven构建详细信息。

在构建页面上可以看到大多数受支持版本的持续集成构建。

Spring Boot配置

<dependencies>
    <dependency>
        <groupId>org.springframework.data</groupId>
        <artifactId>spring-data-hadoop</artifactId>
        <version>2.5.0.RELEASE</version>
    </dependency>
</dependencies>

快速开始

使用Spring Initializr引导您的应用程序。
