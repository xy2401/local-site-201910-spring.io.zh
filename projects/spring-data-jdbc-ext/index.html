Spring Data JDBC扩展

1.2.1
  • 总览
  • 学习
  • 样品

注意：该项目已不再积极开发，并将于今年晚些时候移至阁楼。

Spring Data JDBC扩展是Spring Data项目的一部分。Spring框架中的JDBC支持广泛，涵盖了最常用的功能。该扩展项目为使用高级Oracle数据库功能提供了额外的支持。

大部分支持由SpringSource项目“ Oracle数据库高级包”中移植的代码组成，这些代码可用于支持订阅客户。现在，我们将该代码提供给所有Spring用户，并且任何新开发都将在Spring Data JDBC Extensions项目中进行。

特征

核心

  • QueryDSL支持（从1.2.0版开始不推荐使用）-支持将QueryDSL SQL模块与QueryDslJdbcTemplate一起使用，该模块结合了QueryDSL和熟悉的JdbcTemplate的功能。这提供了用于查询，插入，更新和删除操作的类型安全的API，并结合了Spring的JdbcTemplate支持的便利性，例如RowMapper，资源管理，声明式事务管理等。

甲骨文

  • RAC“快速连接故障转移”-RAC“快速连接故障转移”提供了在数据库节点发生故障时透明地进行Spring应用程序故障转移的功能。

  • 流AQ（高级队列）-AQ支持提供了使用单个本地事务管理器进行数据库和消息访问的选项，而无需诉诸昂贵的分布式两阶段提交事务管理。

  • XML类型-有关如何将Oracle JDBC扩展用于其本机XML类型的自定义类，示例和文档。

  • 高级数据类型-有关如何将Oracle JDBC扩展用于其高级数据类型（如STRUCT和ARRAY）的自定义类，示例和文档。

  • 自定义数据源连接准备器-此功能提供了一个API，用于使用Oracle特定的会话设置等自定义连接环境。

快速开始

使用Spring Initializr引导您的应用程序。
