Spring Data Couchbase

3.2.0
  • 总览
  • 学习

Couchbase的Spring Data是Spring Data项目的一部分，该项目旨在为新数据存储提供熟悉且一致的基于Spring的编程模型，同时保留特定于存储的功能。

Spring Data Couchbase项目提供了与Couchbase Server数据库的集成。Spring Data Couchbase的关键功能区域是一个以POJO为中心的模型，用于与Couchbase桶进行交互并轻松编写存储库样式的数据访问层。

特征

  • Spring配置支持使用基于Java的@Configuration类或Couchbase驱动程序（Java SDK版本2.x）的XML名称空间。

  • CouchbaseTemplate帮助程序类，可提高执行常见Couchbase操作的效率。包括文档和POJO之间的集成对象映射。

  • 将异常转换为Spring的可移植数据访问异常层次结构。

  • 与Spring的转换服务集成的功能丰富的对象映射。

  • 基于注释的映射元数据，但可扩展以支持其他元数据格式。

  • Repository接口的自动实现，包括对自定义finder方法（由Couchbase的查询语言N1QL ）和PagingAndSortingRepository 。

  • 对于版本低于4.0的Couchbase服务器，仍然可以通过Couchbase视图支持存储库。

  • 支持地理空间和多维查询（由Couchbase Spatial Views支持）

  • JMX管理和监视

  • 可以用作@Cacheable支持的后端，以缓存实现高性能访问所需的任何对象（请参阅Couchbase的github上的同级Spring Cache项目， ouchbaselabs / couchbase-spring-cache ）。

快速开始

使用Spring Initializr引导您的应用程序。
