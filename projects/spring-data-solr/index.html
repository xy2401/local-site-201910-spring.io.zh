Apache Solr的Spring数据

4.1.0
  • 总览
  • 学习
  • 样品

Apache Solr的Spring Data是较大的Spring Data系列的一部分，可轻松配置并从Spring应用程序访问Apache Solr Search Server。它提供了与商店交互的低层和高层抽象。

针对Solr特定功能的派生查询和注释可轻松集成到现有应用程序中。

特征

  • 具有多核支持的高级存储库抽象

  • 增强，方面和突出显示的注释

  • 可自定义的类型映射和类型转换

  • Solr模板支持流利的查询API

  • 异常转换为Spring的可移植数据访问异常层次结构

快速开始

使用Spring Initializr引导您的应用程序。
