Spring Cloud连接器

2.0.6
  • 总览
  • 学习
  • 样品

Spring Cloud Connectors简化了在Cloud Foundry和Heroku等云平台（尤其是Spring应用程序）中连接服务并获得操作环境意识的过程。它旨在实现可扩展性：您可以使用提供的一种云连接器，也可以为您的云平台编写一个，并且可以使用对常用服务（关系数据库，MongoDB，Redis，RabbitMQ）的内置支持或扩展Spring云连接器可与您自己的服务一起使用。

注意
 该项目处于维护模式 ，而支持较新的Java CFEnv项目。我们将继续发布与安全性相关的更新，但不会解决增强要求。

社区扩展

  • 用于SAP HANA云平台（HCP）的Spring Cloud连接器

  • 用于IBM Bluemix的Spring Cloud连接器

  • 用于关键Spring Cloud Services的Spring Cloud连接器

  • 用于Pivotal Gemfire的Spring Cloud连接器

  • 用于关键单点登录服务的Spring Cloud连接器

  • 适用于Amazon S3的Spring Cloud连接器

  • 用于Hashicorp Vault的Spring Cloud连接器

快速开始

使用Spring Initializr引导您的应用程序。
