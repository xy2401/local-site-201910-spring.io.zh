Spring Tool Suite™3

Spring Tool Suite 3是基于Eclipse的开发环境，为开发Spring应用程序而定制。它提供了一个现成的环境来实现，调试，运行和部署您的Spring应用程序，包括与Pivotal tc Server，Pivotal Cloud Foundry，Git，Maven，AspectJ的集成，并且位于最新的Eclipse版本之上。

Spring Tool Suite 3附带有Pivotal tc Server的开发人员版本，它是为Spring优化的Apache Tomcat的直接替代产品。通过其Spring Insight控制台，tc Server Developer Edition提供了应用程序性能指标的图形实时视图，使开发人员可以从其台式机识别和诊断问题。

Spring Tool Suite 3支持将应用程序定位到本地，虚拟和基于云的服务器。它免费提供给开发人员和内部业务运营人员使用，没有时间限制，完全开源，并根据Eclipse Public License的条款获得许可。

功能亮点

了解您的Spring App

Spring Tool Suite 3可以理解您的Spring项目。它解析您的配置文件并显示有关正在定义的Bean，它们之间的依赖性，使用的名称空间的详细信息，并提取某些构造型的概述，例如请求控制器，方面，服务等。

您的Spring配置的全面验证

因为Spring Tool Suite 3可以理解您的Spring项目，所以它提供了一套全面的验证，这些验证将自动应用。这些验证表明，在您真正运行应用程序之前很久就直接在IDE中配置错误。查找问题和配置错误变得容易得多。

对您的Spring App的重构支持

重构支持是当今软件工程中最重要的部分之一。因此，Spring Tool Suite 3为重构Spring应用程序提供了高级支持。您的Spring配置文件中不仅会反映众所周知的Java重构，而且IDE还会为Spring元素添加新的重构（例如，重命名Spring Bean）。

代码协助各地

无论您是编写Spring XML配置文件还是实现JavaConfig Spring应用程序，无论您是单独使用核心Spring框架还是与所有其他其他Spring项目一起使用，Spring Tool Suite 3都可以为您提供有意义的内容帮助遍布各地，以及针对常见错误和问题的快速修复。没有这些代码助手，您将永远无法使用Spring进行编程。

图形查看器和编辑器

是否想在您的Spring应用程序中获得有关bean依赖关系的概述？还是想可视化和编辑Spring Integration，Spring Batch或Spring Webflow定义？只需在您的配置文件中单击一下，即可在您的IDE中检出Spring Tool Suite 3随附的图形编辑器。

可用的最佳AOP支持

Spring Tool Suite 3与用于Eclipse的AspectJ语言工具集成在一起，并为当今的AOP提供了最全面的支持。可以识别方面，并将其逐步编织到您的系统中，并直接在IDE中可视化。并在保存文件后立即查看切入点在哪里匹配。

与Cloud Foundry和Pivotal tc服务器集成

将应用程序直接部署到您选择的Cloud Foundry或tc Server实例中（包括对调试，实例创建，Spring Insight，服务等的支持）。
