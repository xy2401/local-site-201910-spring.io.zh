使用IntelliJ IDEA入门指南

本指南将引导您逐步使用IntelliJ IDEA来构建入门指南之一。

你会建立什么

您将选择一个Spring指南并将其导入IntelliJ IDEA。然后，您可以阅读指南，编写代码并运行项目。

你需要什么

  • 约15分钟

  • IntelliJ IDEA

  • JDK 6或更高版本

安装IntelliJ IDEA

如果您尚未安装IntelliJ IDEA（最终版），请访问上面的链接。您可以从那里下载适合您平台的副本。要安装它，只需解压缩下载的档案。

完成后，继续并启动IntelliJ IDEA。

导入入门指南

要导入现有项目，您需要一些代码，因此请克隆或复制其中一个“入门指南”，例如“ REST服务”指南：

$ git clone https://github.com/spring-guides/gs-rest-service.git

随着IntelliJ IDEA的启动和运行，点击欢迎屏幕或文件 导入项目 |在主菜单上打开 ：

在弹出对话框中，确保选择完整文件夹下的Maven的pom.xml或Gradle的build.gradle文件：

IntelliJ IDEA将使用指南中的所有代码来创建一个项目，以供运行。
