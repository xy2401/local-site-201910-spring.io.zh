SpringSource认证计划

工程
达里尔·亨氏（Daryl Heinz）
2008年1月17日

注意：该帖子已被编辑，以反映Spring向Pivotal的转变。有关Spring培训和认证的最新博客在这里 。

自从我六个月前加入SpringSource担任培训总监以来，我一直在听到一个一致的要求。基于对Spring技能不断增长的需求 ，全球的开发人员和顾问正在寻求量化方法来展示其Spring专业知识。同样，这种需求背后的招聘经理也正在要求认证计划，以帮助确定和聘用对Spring有即时有用的基础知识的技术人员。

就像Spring项目团队响应开源社区需求的方式一样，我们做出了回应，并高兴地宣布了SpringSource认证计划 。该程序将于2008年1月通过Spring Framework Professional认证启动。

要成为认证的Spring Framework Professional，您必须成功完成认证考试。如果某人最近参加了“春季四天核心课程” ，或者是Spring社区中的重要贡献者，或者可以为他或她积极参与的Spring开发工作提供参考，则该人有资格参加认证考试。成功完成考试后，将授予Spring Framework Professional认证。

注意：自从Daryl撰写此博客以来，原始的SpringSource认证计划已扩展到Pivotal的许多产品中，而不仅仅是Spring。现在也可以直接购买认证考试-无需参加培训课程

要了解有关Pivotal认证计划的更多信息，请访问以下网站： https://pivotal.io/training#spring-professional-certificate ，查看我们的培训常见问题解答 ，或使用此联系表与Pivotal Education 联系 。
