SpringSource DM Server入门

工程
罗伯·哈罗普
2008年10月22日

更新的2008年10月28日 ：添加了最新的示例链接，并链接到第三个示例

昨晚，我在费城春季用户小组发表了“ SpringSource dm Server简介”。在此演示中，我创建了一个名为GreenPages的小型应用程序，演示了dm Server的所有主要方面。我向与会者保证，我将在此处发布应用程序和幻灯片。

自dm Server GA发行以来的最近几周，许多人一直在询问开始使用dm Server的最佳方法，因此我正在使用此条目来收集所有相关信息，包括SpringSource dm Server简介介绍。

博客条目

最好的起点是我们在里程碑阶段发布的一系列博客条目（当该产品称为SpringSource Application Platform时）：

  1. 介绍SpringSource应用程序平台
  2. 使用SpringSource应用程序平台存储库
  3. SpringSource应用程序平台部署选项
  4. 在SpringSource应用程序平台上使用EclipseLink
  5. 了解OSGi“使用”指令

资料下载

可以从dm Server下载页面下载SpringSource dm Server和Eclipse的dm Server工具。

文档和样本

dm Server文档有两个组成部分：《 用户指南》旨在提供dm Server概念和日常操作的概述；《 程序员指南》着重于应用程序开发。

我们提供了三个示例： Petclinic ， FormTags和Spring Travel 。如果您正在寻找有关如何为dm Server构建应用程序的示例，那么我建议您下载这些示例并熟悉其结构和方法。

论坛

我们有一个非常活跃的社区论坛，提供了许多有用的内容。dm Server论坛可在此处访问。

SpringSource DM Server简介

可以从此处下载“ SpringSource dm Server简介”演示文稿的幻灯片。

演示文稿中最有趣的部分是GreenPages应用程序。在进行演示时，我将以空的Eclipse工作区开始并以完全运行的模块化Web应用程序结束，从而实时构建应用程序。我已经打包了完整的应用程序代码，可在此处下载。

要使GreenPages运行，您需要在dm Server的repository / bundles / usr目录中安装一些额外的库。我将这些文件打包在此处下载。

另外，您需要使用db目录中提供的脚本来启动H2数据库。我还提供了一个脚本，用于使用一些示例数据预填充数据库。

我从Eclipse中运行GreenPages，因此当前没有Ant构建脚本-您需要安装dm Server工具的Eclipse安装。我的建议是下载并使用SpringSource Tool Suite的个人使用版-我在所有dm Server演示中都使用此版本。
