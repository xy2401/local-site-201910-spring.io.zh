发布第一个Spring Framework 3.0里程碑

工程
于尔根·霍勒（Juergen Hoeller）
2008年12月5日

我很高兴地宣布，Spring Framework 3.0 M1终于可以下载了 ！

此版本具有几个主要更改，包括主要3.0主题的开始，例如EL和REST支持 ：

  • 使用基于模块的源代码修改了项目布局和构建系统
  • 更新了Java 5代码样式 （泛型，varargs）的整个代码库
  • 更新到JUnit 4.5和JRuby 1.1
  • 引入了Spring EL解析器 （ org.springframework.expression包）
  • 在bean定义中引入了对＃{...}表达式的支持
  • 为嵌入式表达式引入了启用表达式的@Value注释
  • 为 MVC处理程序中的URI模板引入了@PathVariable批注
  • 在MVC处理程序中为@RequestParam引入了默认值支持
  • 为 MVC处理程序中的HTTP标头访问引入了@RequestHeader批注
  • 引入了AbstractAtomFeedView和AbstractRssFeedView基类
  • 介绍和 JSP标签

以及各种小的改进。

请注意，Spring Framework 3.0需要Java 5或更高版本以及J2EE 1.4或更高版本。我们将Java 6和Java EE 5作为主要平台级别构建-但请放心，我们将与启用Java 5的J2EE 1.4服务器（如WebLogic 9和WebSphere 6.1）保持兼容性。

我们还删除/弃用了一些过时的类。有关弃用和相应迁移路径的更多信息将在3.0 M2 / M3时间范围内提供。

与SourceForge的CVS中托管的先前发行版相比，Spring Framework 3.0正在我们新的Subversion存储库中开发，可以在以下位置找到： https://src.springframework.org/svn/spring-framework/

在我们接近Spring Framework 3.0 final时随时跟踪正在进行的工作，并让我们知道您是否喜欢自己所看到的内容:-)
