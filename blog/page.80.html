春季课程2.0.0 M4

发布
罗伯·温奇
2017年9月15日

我很高兴代表社区宣布Spring Session 2.0.0的发布。M4 。此版本主要侧重于完善WebFlux支持。重点是：

简化的WebFlux配置

为WebFlux配置Spring Session简化为：

@Configuration
@EnableSpringWebSession
public class HelloWebfluxSessionConfig {

  @Bean
  public MapReactorSessionRepository reactorSessionRepository() {
    return new MapReactorSessionRepository(new ConcurrentHashMap<>());
  }
}

您还可以通过简单地添加一个来切换解决会话ID的策略WebSessionIdResolver豆。例如，要从使用cookie解析会话ID切换为使用标头，可以使用Spring Framework的new HeaderWebSessionIdResolver ：

@Bean
public HeaderWebSessionIdResolver webSessionIdResolver() {
  return new HeaderWebSessionIdResolver();
}
