发布了Spring Data R2DBC 1.0 M2和Spring Boot启动器

发布
马克·帕鲁奇
五月15，2019

Spring Data R2DBC

我很高兴代表社区和所有贡献者宣布Spring Data R2DBC 1.0的第二个里程碑的发布。它基于最近发布的Moore M4版本和R2DBC 0.8.0。M8释放 。请注意，Spring Data R2DBC是在Moore发布火车之外发布的，并且它将是下一发布Neumann火车的一部分。

Spring Data R2DBC附带了32张固定的票证 。最值得注意的功能是：

  • 通过使用jasync-sql支持MySQL。
  • 反应式交易管理器。
  • 流利的API insert / update / delete操作。
  • 协程扩展。
  • 支持自定义转换。
  • 通过使用以下命令转换为本地绑定标记的命名参数Dialect实例。
  • 支持简单类型的单列投影。
  • 重构的包结构。
