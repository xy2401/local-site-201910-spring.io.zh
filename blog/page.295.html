什么是消息传递？为了最好地解释这一点，我将解释一下由Gregor Hohpe和Bobby Woolf（Addison Wesley，2004）开创性的《 企业集成模式》一书提供的示例。当您拨打电话时，您尝试将信息中继给另一方。仅当您拨打电话时第二方可用时，此方法才有效。由于并非总是可以接听电话，因此我们使用语音邮箱将消息排队。呼叫者将消息留在语音信箱中，然后被呼叫者可以自由地在以后的某个时间异步地检索该消息（或者实际上很多消息）。

Spring Web Services 2.0发布

工程
Arjen Poutsma
2011年1月11日

经过将近一年的工作，我很高兴地宣布Spring Web Services 2.0已发布！在本文中，我想介绍一些主要的新功能。

需要Java 5+和Spring 3.0

您可能已经知道，我们将对象XML映射（OXM）模块从Spring-WS项目移至了Spring 3.0。因此，由于org.springframework.oxm包中的类冲突，因此在Spring 3.0中使用Spring-WS 1.5（带有自己的OXM模块）会有些问题。

从2.0版开始，我们不再将OXM模块作为Spring-WS的一部分提供，而是依靠Spring的OXM。结果， Spring Web Services 2.0需要Spring 3.0才能工作。通常，对于版本要求，我们倾向于宽容一些，不一定需要最新的Spring版本，但这是使事情正常进行的唯一方法。
