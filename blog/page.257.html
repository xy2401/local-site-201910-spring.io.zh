Spring Data Gemfire 1.2.1（和Gemfire 7.0）发布了

发布
彼得·汉弗莱（Pieter Humphrey）
2012年10月29日

SpringSource很高兴宣布GA发布Spring Data GemFire 1.2.1 。该版本将与GemFire 7.0同时发布，并为GemFire 7.0中引入的新WAN API添加了Spring XML名称空间支持。通过此版本的GemFire，Spring Data GemFire现在在GemFire生态系统中享有“一流公民”的地位，并在GemFire的开发人员页面和文档中得到了突出体现。

该版本遵循最近宣布的1.2.0版本，该版本提供：

  • 完整的Spring XML名称空间支持，可配置所有GemFire资源，从而无需使用GemFire的本机缓存XML（也支持缓存XML）
  • GemFire的Spring数据存储库

具有Oracle数据库支持1.0.0的Spring Data JDBC扩展。RC4发布

发布
托马斯·里斯伯格
2012年10月17日

亲爱的春季社区，

我们很高兴地宣布，第四个也是最后一个候选版本（1.0.0。带有Oracle数据库支持的Spring Data JDBC Extensions 1.0项目的RC4）现已上市！

创建新的Spring Data JDBC Extensions项目是为了为特定于供应商的JDBC扩展提供附加支持，以及使用QueryDSL之类的JDBC的新方法。大部分支持由从SpringSource项目“ Oracle数据库高级包”移植的代码组成，该代码可用于支持订阅客户。现在，我们将该代码提供给所有Spring用户，并且任何新的开发都将在Spring Data JDBC Extensions项目中进行。
