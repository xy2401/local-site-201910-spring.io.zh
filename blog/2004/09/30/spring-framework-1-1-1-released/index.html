Spring Framework 1.1.1发布

发布
托马斯·里斯伯格
2004年9月30日

我们
很高兴地宣布 Spring
框架1.1.1具有
只是
被释放。这是一个错误修正和次要增强版本。

 


此版本中的新功能包括：

        
    
          
  • "import"

XML bean的元素
定义

  • 支持
    非公开课
    以及非公开的初始化和销毁方法

  • bean
    工厂视 bean
    正确销毁顺序的参考（不需要“依赖”）

  • bean
    工厂检测到JDK 1.4
    和Commons Collections用于链接/身份地图创建

  • 分解
    出
    将“ publishEvent”方法插入ApplicationEventPublisher接口

  • 包括在内
    中的新“测试”包
    模拟JAR，为需要Spring上下文的测试提供超类

  • 添加
    “ applyQueryTimeout”
    JdoDialect接口的方法

  • 添加
    “ addAttribute”和
    RemoteInvocation的“ getAttribute”方法

  • 显式的
    支持EHCache
    通过EhCacheManagerFactoryBean和EhCacheFactoryBean

  • SimpleMessageConverter102
    在没有JMS 1.1的“ getBodyLength”方法的情况下处理BytesMessage

  • 添加
    JmsGateway支持
    便利基础班

  • 添加
    ServletForwardingController和ServletWrappingController

  • 绑定状态
    和BindTag可以是
    在请求中也创建了没有错误实例

  • 变了
    MessageTag的
    对象的“参数”属性

  • 添加
    “ javaScriptEscape”
    MessageTag属性

  • 添加
    EscapeBodyTag aka
    “ Spring ：逃脱的身体”
