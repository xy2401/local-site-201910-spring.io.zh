Spring Framework 1.1.3发布

发布
托马斯·里斯伯格
2004年12月12日

我们
很高兴地宣布春天
框架1.1.3具有
只是
被释放。这是一个错误修正和较小的增强版本，
1.1系列；最重要的是，JDK 1.3兼容性已完全恢复。

此版本中的新功能包括：

        
    
          
  • added

“ setResourceLoader”方法
GenericApplicationContext，用于覆盖默认资源加载
战略

  • 添加
    FileSystemResourceLoader，
    将路径解析为文件系统资源，而不是类路径
    资源

  • BeanWrapperImpl
    自动地
    注册默认的ResourceArrayPropertyEditor

  • 添加
    CustomCollectionEditor，作者
    默认注册以在Set，SortedSet和List之间进行转换（当
    必要）

  • 添加
    “ fileEncoding”和
    PropertyResourceConfigurer的“ propertiesPersister”属性

  • 重做
    DefaultListableBeanFactory的“ getBeansOfType”以返回FactoryBean
    实例通过FactoryBean类型传递时

  • DefaultXmlBeanDefinitionParser
    在JDK 1.4上将“设置”标签解析为LinkedHashSet，回退到HashSet
    在JDK 1.3上

  • 分解
    出
    AbstractRefreshableApplicationContext作为以下对象的基类
    AbstractXmlApplicationContext

  • 添加
    “ setIgnoreUnknownFields”
    DataBinder的方法，在以下情况下强制请求参数匹配字段
    变成“假”



    • 添加
      “执行（ConnectionCallback）”
      JdbcTemplate的方法，允许对给定的任何类型的操作
      连接

    • 添加
      DB2SequenceMaxValueIncrementer，实现
      DB2序列的DataFieldMaxValueIncrementer接口

    • 固定
      休眠/ JTA同步
      也可以使用EJB远程事务传播（> 1
      线
      涉及）

    • 添加
      “ transactionConfigClass”和
      “ transactionConfigProperties”定制属性可以
      SqlMapClientFactoryBean



    • 添加
      可选的“ proxyInterface”
      JndiObjectFactoryBean的属性，用于公开代理而不是
      普通物体

    • 添加
      “ lookupOnStartup”和“缓存”
      JndiObjectFactoryBean的属性，允许在第一次时进行懒惰查找
      访问等

    • 添加
      “ codebaseUrl”属性为
      HttpInvokerClientInterceptor，为动态类指定一个URL
      下载

    • 添加
      支持动态类
      下载到AbstractHttpInvokerRequestExecutor及其子类



    • 分派器
      默认情况下清理
      包含请求后最多请求属性（可通过配置
      “ cleanupOnInclude”）

    • UrlBasedViewResolver
      支持一个
      也是“ forward：”前缀，用于通过以下方式转发到其他控制器
      特殊视图名称

    • 添加
      “属性”（属性）和
      UrlBasedViewResolver的“ attributesMap”（地图）属性为静态
      属性

    • InternalResourceView
      执行
      包括是否已提交响应（->转发
      不
      可能了）

    • jstl视图
      和TilesJstlView尊重
      web.xml中的JSTL“ localizationContext”配置，检查这些消息
      第一

    • 添加
      超载
      VelocityView的“ createVelocityContext”和“ exposeHelpers”方法，
      与传入的servlet响应

    • 添加
      “ toolAttributes”属性为
      VelocityView，采用工具的属性名称/工具类名称对
      揭露

    • 添加
      JasperReportsXxxView允许
      使用Jasper方便地呈现CSV / HTML / PDF / XLS Web视图
      报告书

    • 添加
      MappingDispatchActionSupport
      类，作为Struts 1.2的MappingDispatchAction的便利性子类
