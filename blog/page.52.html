2018年6月20日

  • Spring Cloud Finchley。RELEASE可用 。这是大个子！终于到了！反应式拼图的最后一部分。此发行版结束了将Spring领域引入到响应式编程的Spring Framework 5（2017年9月发布）历时近一年的历程。然后我们看到了Spring Data Kay，Spring Cloud Stream，Spring Security 5，Spring Boot 2（就在几个月前），现在，最后，Spring Cloud Finchley都在任何有意义的地方都采用了响应式编程。Spring Framework 5是新的基准，并随之带来了新的Java基准要求。 Spring Cloud Finchley假定使用Java 8或更高版本。但是，此版本不仅仅是刷新！最远的东西！此版本还包括两个（有效）全新项目，称为Spring Cloud Function和Spring Cloud Gateway。Spring Cloud Function 支持无服务器编程 ，Spring Cloud Gateway是API网关 。
