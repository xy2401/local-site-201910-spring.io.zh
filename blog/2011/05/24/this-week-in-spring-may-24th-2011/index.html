春天的一周：2011年5月24日

工程
乔什·朗（Josh Long）
2011年5月24日


好一个星期！当我们在26日在阿姆斯特丹举行的S2G论坛以及5月31日在下周在伦敦举行的 S2G论坛附近时，令人激动不已。如果您在欧洲，请确保不要错过这些激动人心的日子，其中包括Spring，Grails，云，大数据以及工具等所有主题的讲座。


  1. 马克·费舍尔（Mark Fisher）和拉姆尼瓦斯·拉达德（Ramnivas Laddad）上周在Cloud Foundry上展示了他们的热门网络研讨会-“在60分钟内从零到云”。
    谢谢大家参加并取得成功！如果错过了，您仍然可以在这里获取幻灯片并观看重播。 请注意，完成CloudFoundry网络研讨会后，照例将有许多其他资源。查看其他开发人员网络研讨会（向下滚动，单击“开发人员”选项卡） ，然后查看SpringSource Dev YouTube页面。

  2. 春季项目负责人Juergen Hoeller于今年早些时候在伦敦QCon上介绍了下一代春季–春季3.1及以后版本。他的演讲和幻灯片可在InfoQ.com上找到。

  3. 还提供了“ Spring Data Graph入门”网络研讨会的视频。该网络研讨会介绍了Spring Data Graph项目-Spring和Neo4j工程团队之间的共同努力，旨在为您的Spring应用程序提供对Neo4J的一流支持。如果您想以更自然的方式将NOSQL数据技术集成到现有体系结构中，只是想提高速度，或者想了解自己所缺少的内容，那么您绝对应该查看此网络研讨会。

  4. 在吃那些自产自用的一个梦幻般的例子，马克托马斯- Tomcat的提交者和Apache的Bug跟踪基础设施的维护者-介绍如何在Apache JIRA接口正在whelmed -不是不知所措，但还是低效运行-通过搜索引擎，击中特定JIRAs，但没不会维护会话cookie，从而触发创建多个会话。马克描述了自定义的创建Valve Tomcat 7 （和SpringSource的tcServer ）的支持，它将单个Tomcat会话与每个Web 搜寻器相关联，从而大大减少了它们的占用空间。

  5. Spring Web Services 2.0.2已发布。有关更多信息，请参阅更改日志 。Spring Web Services 1.5.1.0也已经发布。有关此版本中的更改，请参阅changelog 。这两个版本本身都包含一些有价值的更新，但重要的是，两个版本还解决了潜在的安全问题。建议用户尽快升级。

  6.  Google I/O, Google's developer conference, is an exciting time for enterprise Java developers, and of course, this also means Spring developers. One notable announcement was the 1.0 release of the Spring Roo plugin for Vaadin, which is a widget-centric approach to web application development.  Vaadin's a very innovative way to build web applications today, and - of course - it works well with Spring. (NB: those instructions are old, but they should still work, and you can just use the Roo plugin, either way!)
