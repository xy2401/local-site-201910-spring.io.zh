Spring.NET Visual Studio 2010扩展1.0.0现在可用

发布
斯博伦
2011年4月7日

亲爱的春季社区，



我们很高兴地宣布，Spring.NET Visual Studio 2010 Extension 1.0.0现在可用。



下载
| 技术支持 文献资料
| 社区



此版本的Spring.NET Visual Studio 2010扩展在以下区域提供了Intellisense支持，用于编辑Spring XML配置文件：


  • 输入完成

  • 物业名称填写

  • 构造函数参数名称完成

  • “类型”，“枚举”和“布尔”类型的属性的属性值完成



此外，此版本还提供了Visual Studio 2010 XML编辑器体验的以下增强功能：


  • 片段集成（内联或通过菜单）

  • 属性和类型的Quickinfo工具提示
