具有Oracle数据库支持1.0.0的Spring Data JDBC扩展。M1发布

发布
托马斯·里斯伯格
2011年4月1日

亲爱的春季社区，

我们很高兴地宣布，第一个里程碑版本（1.0.0。支持Oracle数据库的Spring Data JDBC Extensions 1.0项目的M1）现已可用！

创建新的Spring Data JDBC Extensions项目是为了为供应商特定的JDBC扩展提供附加支持。大部分支持由从SpringSource项目“ Oracle数据库高级包”移植的代码组成，该代码可用于支持订阅客户。现在，我们将该代码提供给所有Spring用户，并且任何新的开发都将在Spring Data JDBC Extensions项目中进行。

资料下载 JavaDocs | 参考文档 | 变更日志

要了解有关该项目的更多信息，请访问Spring Data 主页 。

Spring Data JDBC Extensions 1.0.0中的功能。M1包括：

    一般
  • 从SpringSource Advance Pack for Oracle Database v 1.1.0移植了所有当前功能
    • RAC“快速连接故障转移”

      RAC“快速连接故障转移”提供了在数据库节点发生故障时使Spring应用程序透明地进行故障转移的功能。

    • 流AQ（高级队列）

      AQ支持提供了使用单个本地事务管理器进行数据库和消息访问的选项，而无需诉诸昂贵的分布式两阶段提交事务管理。

    • XML类型

      有关如何将Oracle JDBC扩展用于其本机XML类型的自定义类，示例和文档。

    • 进阶资料类型

      有关如何将Oracle JDBC扩展用于其高级数据类型（如STRUCT和ARRAY）的定制类，示例和文档。

    • 自定义数据源连接准备器

      此功能提供了一个API，用于使用Oracle特定的会话设置等自定义连接环境。

    • 流AQ
  • DATAJDBC-4添加了一个属性以为条目指定“ connection-factory-type”
