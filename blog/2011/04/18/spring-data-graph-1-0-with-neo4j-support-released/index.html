带有Neo4j支持的Spring Data Graph 1.0发布

发布
迈克尔·汉格
2011年4月18日

亲爱的春季社区，

我们很高兴地宣布， 第一个版本 （1.0.0。现已推出支持Neo4j的Spring Data Graph 1.0项目的RELEASE！
这标志着接下来几个月Spring Data子项目系列的第一个版本。

Spring Data项目的主要目标是使使用新数据访问技术（例如非关系数据库，map-reduce框架和基于云的数据服务）的使用Spring支持的应用程序更容易构建。

在SpringSource Blogs上已发布了详细介绍该版本的来宾博客帖子。

Graph Neo4j模块提供与Neo4j图形数据库的集成。早在2010年，Rod Johnson和Emil Eifrem就关于Spring和Neo4j的集成（包括透明的持久性和跨商店支持）进行了集思广益。在最初的原型之后，它已经在VMware和Neo Technology开发团队之间紧密合作下进一步开发。

要了解有关该项目的更多信息，请访问Spring Data Graph项目主页 ，并确保参加由VMware主办，由Neo Technology首席执行官Emil Eifrem主持的网络研讨会“ Spring Data Graph入门” 。

Spring Data Graph 1.0.0中的主要功能。发行包括：


  • 支持属性图（通过关系连接的节点，每个都有任意属性）

  • 带注释的POJO实体的透明映射

  • Neo4jTemplate具有便捷的API，异常转换和可选的事务管理

  • 将类型信息保留在图中的不同类型表示策略

  • 动态类型投影（鸭式打字）

  • Spring Data Commons存储库支持

  • 对部分JPA的跨商店支持-图形实体

  • Neo4j对动态字段和通过存储库方法的遍历支持

  • Neo4j索引支持（包括全文和数字范围查询）

  • 支持JSR-303（Bean验证）

  • 对Neo4j服务器的支持

  • 支持在Neo4j Server中作为扩展运行
