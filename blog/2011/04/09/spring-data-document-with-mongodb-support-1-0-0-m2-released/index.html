带有MongoDB支持1.0.0的Spring数据文档。M2发布

发布
托马斯·里斯伯格
2011年4月9日

亲爱的 Spring 社区，

我很高兴地宣布，具有MongoDB支持的Spring Data Document 1.0项目的第二个里程碑版本现已发布！

Spring Data项目的主要目标是使使用新数据访问技术（例如非关系数据库，map-reduce框架和基于云的数据服务）的使用Spring支持的应用程序更容易构建。

Spring Data MongoDB子项目提供了与MongoDB文档数据库的集成。

资料下载 JavaDocs | 参考文档 | 变更日志

要了解有关该项目的更多信息，请访问Spring Data Spring Data MongoDB页面 。

Spring Data Document 1.0.0中的更改和新功能。M2包括：

一般

  • 使用基于Java的@Configuration类的Spring配置支持

核心数据访问

  • 持久性和映射生命周期事件
  • 地理空间整合
  • [DATADOC-76]-将对findAndRemove的支持添加到MongoTemplate / MongoOperations
  • [DATADOC-5]-提供Mongo错误到Spring DAO异常的详细映射
  • [DATADOC-51]-解决了对同一数据库实例进行多次身份验证时引发异常的问题

查询方式

  • [DATADOC-72]-添加对Mongo的$ elemMatch和链接的条件的支持
  • [DATADOC-77]-将查询中的“和”方法重命名为“ addCritera”
  • [DATADOC-67]-支持地理搜索关键字的条件API

映射

  • 与Spring的转换服务集成的功能丰富的对象映射
  • 基于注释的映射元数据，但可扩展以支持其他元数据格式
  • [DATADOC-60]-添加名称空间支持以设置MappingMongoConverter
  • [DATADOC-33]-引入注释来划分域对象中的id字段

资料库

  • [DATADOC-47，DATACMNS-17]-修改后的新元模型API
  • [DATADOC-46]-添加了对“ In”和“ NotIn”关键字的支持
  • [DATADOC-49]-修复了“ And”和“ Or”关键字
  • [DATADOC-41]-添加了对执行QueryDsl谓词的支持
  • [DATADOC-69]-让存储库名称空间提取默认的映射上下文Bean并允许配置
  • [DATADOC-24]-允许使用@Query注释定义查询
  • [DATADOC-34]-为查询方法中提到的列创建索引

跨店

  • [DATADOC-48]-跨存储持久性-使用MongoDB支持具有透明持久性/检索性的字段的JPA实体

记录中

  • [DATADOC-66]-Log4j日志附加器
