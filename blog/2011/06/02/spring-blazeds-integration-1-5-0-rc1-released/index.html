Spring BlazeDS Integration, the open source solution for building Spring-powered RIAs with Adobe Flex, is now available.

Download | Reference Documentation | JavaDocs | Changelog

This release is largely focused on stabilization of the new features introduced in the previous milestones. Significant attention has been paid to maturing the refactored Spring Security 3 support and the Hibernate AMF serialization support. New annotations such as @AmfIgnore and @AmfCreator allow deeper customization of the AMF conversion process, and this enhanced AMF support has now been made generally available for application to any object type, not just those persisted with Hibernate.

Spring BlazeDS集成1.5.0。RC1发布

发布
杰里米·格莱尔
2011年6月2日

亲爱的春季社区，

我很高兴地宣布1.5.0。Spring BlazeDS Integration的 RC1发行候选版本已经可用，它是使用Adobe Flex构建基于Spring的RIA的开源解决方案。

下载 | 参考文档 | JavaDocs | 变更日志

此版本主要集中于稳定以前 里程碑中引入的新功能。为了使重构的Spring Security 3支持和Hibernate AMF序列化支持更加成熟，已经给予了极大的关注。诸如@AmfIgnore和@AmfCreator之类的新注释允许对AMF转换过程进行更深入的自定义，并且这种增强的AMF支持现已普遍适用于任何对象类型，而不仅仅是Hibernate所保留的对象类型。

与往常一样，我鼓励有兴趣的人通过试用该版本并在社区论坛和Jira中给我们提供反馈，因为我们期望在1.5.0发行之前只有片刻休息时间。GA。我们不断地从使用Spring BlazeDS Integration的项目中取得成功的人们那里获得大量反馈，并且我们希望听到更多有关您的经验的信息。



杰里米·格莱尔
