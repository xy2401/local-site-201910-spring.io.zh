Spring Social project is now available!

The Spring Social project allows you to connect your Java applications to Software-as-a-Service (SaaS) providers such as Facebook and Twitter.

This release includes:

  • Additions to the Facebook API binding such as photo and video upload, Facebook page support, and Facebook search.
  • Expansion of the Twitter API binding to include mobile notifications, blocking, and geo location operations.
  • Implicit sign up support enabling automatic sign up of a user from a connection.
  • A new quickstart showing how to get up and running quickly.
  • Numerous bug fixes and other improvements based on user feedback (see the changelog for details).

春季社交1.0.0。RC1发布

发布
克雷格·沃尔斯
2011年6月23日

亲爱的春季社区，

我们很高兴地宣布， Spring Social项目的第一个候选版本现已发布！

Spring Social项目允许您将Java应用程序连接到软件即服务（SaaS）提供程序，例如Facebook和Twitter。

此版本包括：

  • Facebook API绑定的附加功能，例如照片和视频上传，Facebook页面支持和Facebook搜索。
  • 扩展了Twitter API绑定，以包括移动通知，阻止和地理位置操作。
  • 隐式注册支持可从连接中自动注册用户。
  • 新的快速入门介绍了如何快速启动和运行。
  • 根据用户反馈进行了许多错误修复和其他改进（有关详细信息，请参阅更改日志 ）。

另外，在此发行版中，我们将提供程序模块（Twitter，Facebook等）提取到了自己的项目中，以便他们可以与主要的Spring Social项目分开的时间表进行进度。与Spring Social 1.0.0的发布同时进行。RC1，Spring Social Twitter和Facebook 1.0.0。RC1现在也可用。

要获得该软件，请下载发行版本（ Core | Facebook | Twitter ），或仅将Maven工件添加到您的项目中。要观看直播，请浏览快速入门并启动展示应用程序 （最近针对1.0.0更新）。RC1）。随行补充参考手册中的信息 。

Spring Social需要运行Spring Framework 3.0.5或>。对于新的应用程序，我们建议使用Spring 3.1 ，以利用核心框架的最新进展。有关依赖关系的完整说明，请参见参考手册。

看到社区对扩展Spring Social与众多SaaS提供商合作的兴趣真是太棒了。查看Matt Wright在Instagram和Foursquare上的工作以及Morten Andersen-Gott的Yammer扩展 。在LinkedIn ， Github ， TripIt和Gowalla上的工作也已经开始。我们致力于与社区合作，以构建优质的Spring Social扩展生态系统。如果要将SaaS API集成到应用程序中，请考虑为Spring Social项目做出贡献。看一下我们扩展Spring Social的指南，并在我们的论坛上与开发团队讨论 。

当我们朝着GA发行版推进时，我们想听听您对RC1发行版的看法。参加论坛，或者，如果您有任何建议或发现任何错误，请将其发布在问题跟踪器中 。我们希望您喜欢使用Spring Social！
