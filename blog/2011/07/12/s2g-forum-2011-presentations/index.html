其他演讲

Spring MVC 3.1更新 -Rossen Stoyanchev ：
该会议将深入探讨有关Spring MVC 3.1的更新。主要主题包括用于处理带注释的控制器方法的新基础结构，名称空间配置的改进，对话管理和Servlet 3.0支持。
 
Spring集成更新 -Josh Long
本课程提供有关Spring Integration 2.0的新功能（包括新的Spring Integration ROO附加组件）的导览。在此过程中，您将了解Spring Integration对Spring Framework 3.0功能的支持，例如Spring Expression Language，ConversionService和RestTemplate。您还将了解一些新适配器，包括AMQP，XMPP，TCP / UDP，JDBC，JMX等。
 
针对虚拟环境调整Java - Ben Corrie
本实践课程详细讨论如何在虚拟运行时如何从JVM中获得最佳性能，尤其着重于内存管理。该会议将花一些时间研究现有的最佳实践及其背后的技术原因，同时还将提供最新的数据和建议以及SpringSource为增强Java所做的工作的前瞻性观点。在虚拟上。
 
Spring开发人员的Cloud Foundry - Josh Long
Cloud Foundry是VMware提供的新的开源平台即服务，它提供了一个出色的部署平台，可以在自动缩放，自动管理的云环境中运行Spring应用程序。本演示文稿描述了如何利用MongoDB，Redis和RabbitMQ等云服务，并阐明了在Cloud Foundry上执行Java背后的架构细节。
 
Apache Tomcat 7更新 -Mark Thomas
Apache Tomcat是当今企业市场中部署最广泛的应用服务器。开发人员，QA团队和IT经理都在各种部署中使用Tomcat，并取得了令人难以置信的成功。本次会议在受欢迎的Apache项目中进行了分析，以分析Apache Tomcat 7中可用的新功能，包括异步请求处理，内存泄漏检测/预防和安全性增强。
 
Spring AMQP和Rabbit消息传递 -Dave Syer
Spring AMQP项目的目标是简化基于AMQP协议的消息传递应用程序的开发。Spring AMQP提供了可跨RabbitMQ和Apache Qpid实现的可移植Java和.NET API，以及方便的抽象，这些抽象促进了基于POJO的编程模型。如果您熟悉Spring的JMS支持，您将感到宾至如归。在本节中，您将浏览Spring AMQP功能，例如发布，消息转换器和创建多线程使用者。还将涵盖对RabbitMQ服务器管理的支持及其使用Spring的配置。
 
使用Gemfire进行缓存和可扩展的数据访问-Costin Leau
关于如何通过缓存在Spring应用程序中轻松获得收益的实用见解。本课程将分析常见应用程序堆栈中常见的瓶颈以及解决这些问题的方法。将讨论各种缓存模式，不仅关注性能，还关注可伸缩性，包括如何充分利用vFabric Gemfire。
 
云中的 Grails-彼得·莱德布鲁克
本演讲将调查提供Grails支持的各种云运行时平台，并讨论每个平台固有的设计约束。该会话将提供有关创建Grails应用程序的建议，以充分利用这些新的部署目标。
 
调整Grails应用程序 -Peter Ledbrook
Grails使Web应用程序的启动和运行变得异常容易，但是并不能保证该应用程序的性能和伸缩性。如果每个请求发出数百个数据库查询，则您的应用程序将不会成为Speedy Gonzalez。了解如何跟踪和修复应用程序各个部分的瓶颈，尤其是数据库访问和视图呈现。
