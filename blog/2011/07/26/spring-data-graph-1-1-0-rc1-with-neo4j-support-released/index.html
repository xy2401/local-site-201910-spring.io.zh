Spring Data Graph 1.1.0。带有Neo4j支持的RC1发布

发布
托马斯·里斯伯格
2011年7月26日

亲爱的春季社区，

我们很高兴地宣布一个新的候选版本（1.1.0。支持Neo4j的Spring Data Graph项目的RC1）现在可用！

Spring Data项目的主要目标是使使用新数据访问技术（例如非关系数据库，map-reduce框架和基于云的数据服务）的使用Spring支持的应用程序更容易构建。

Graph Neo4j模块提供与Neo4j图形数据库的集成。早在2010年，Rod Johnson和Emil Eifrem就关于Spring和Neo4j的集成（包括透明的持久性和跨商店支持）进行了集思广益。在最初的原型之后，它已经在VMware和Neo Technology开发团队之间紧密合作下进一步开发。

要了解有关该项目的更多信息，请访问Spring Data Graph Project Homepage 。

Spring Data Graph 1.1.0中的关键更改。RC1发行候选版包括：


  • 添加了Gremlin支持（嵌入式和REST）

  • QueryEngine.query方法现在获取一个参数映射（对于cypher和gremlin）

  • 文档更新
