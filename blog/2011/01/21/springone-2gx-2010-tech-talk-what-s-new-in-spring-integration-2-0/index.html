SpringOne 2GX 2010技术讲座：Spring Integration 2.0的新增功能？

新闻与活动
亚当·菲茨杰拉德
2011年1月21日

InfoQ发布了SpringOne 2GX 2010的又一届会议。这次是Mark Fisher和Oleg Zhurakousky 在“ Spring Integration 2.0的新功能”中进行介绍。

Mark和Oleg使用以代码为中心的演示来描述如何在Spring Integration中集成Spring 3.0功能，实现了哪些企业集成模式以及在何处更新了通道适配器和网关。他们演示了对Spring Integration的新STS工具支持，并讨论了该项目的一些将来的功能和方向。

该演示文稿是介绍Spring Integration 2.0的好方法。在这次演讲之后不到一个月的时间， GA 2.0版发布了，因此涵盖的所有主题都与当前版本保持最新。再次感谢InfoQ 报道了SpringOne 2GX 2010 。
