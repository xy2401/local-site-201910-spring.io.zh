Spring Social 1.0.0发布

发布
克雷格·沃尔斯
2011年9月8日

亲爱的 Spring 社区，

我们很高兴宣布Spring Social 1.0 全面上市 ！


Spring Social是Spring Framework的扩展，使您能够将Java应用程序连接到软件即服务（SaaS）提供程序，例如Facebook和Twitter。

1.0版大版本包含对自1.0.0版以来报告的所有错误的修复。RC3。请参阅更改日志以获取所有详细信息（ Core | Facebook | Twitter ）

要获得该软件，请下载发行版本（ Core | Facebook | Twitter ），或仅将Maven工件添加到您的项目中。要观看直播，请尝试快速入门并启动示例 。随行补充参考手册中的信息 。

Spring Social需要运行Spring Framework 3.0.5或>。对于新的应用程序，我们建议使用Spring 3.1 ，以利用核心框架的最新进展。有关依赖关系的完整说明，请参见参考手册。

发布此版本已经过去了一年。我们感谢社区的所有参与，这使该项目成为有用，优质的项目。特别感谢Matt Wright，Morton Andersen-Gott，Stefan Fussenegger，Bryce Fischer，Gabriel Axel，Marc Schipperheyn，Domingo Suarez，Gordon Dickens，Arjen Poutsma和Juergen Hoeller的支持。我利用这次机会在我们的团队博客中回顾了我们去年的工作。

与我一起参加将于9月29日举行的网络研讨会。

我们希望您喜欢使用Spring Social！
