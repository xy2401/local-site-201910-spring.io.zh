Spring Data MongoDB 1.0.0。RC1发布

发布
马克·波拉克
2011年12月7日

亲爱的春季社区，

我很高兴地宣布，Spring Data MongoDB 1.0 RC1现在可用！

Spring Data项目的主要目标是为新数据存储库提供熟悉且一致的基于Spring的编程模型，同时保留特定于存储库的功能。Spring Data MongoDB项目提供了与MongoDB文档数据库的集成。关键功能区域是以POJO为中心的模型，用于与MongoDB DBCollection进行交互并轻松编写存储库样式的数据访问层。

这主要是一个错误修复版本，随着我们快速迁移到GA，但进行了一些显着改进并添加了新功能：

  • 提供其他选项以基于每个操作设置WriteConcern
  • 支持获取有关集合或映射类的索引信息。
  • 添加对findAndModify和upsert操作的支持
  • 允许通过类路径扫描包括类型转换器
  • 更新为使用Spring Data Commons 1.2.0。RC1

有关已修复问题的详细列表，请查看更改日志。
