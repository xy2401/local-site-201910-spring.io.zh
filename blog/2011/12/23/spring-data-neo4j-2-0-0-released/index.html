Spring Data Neo4j 2.0.0发布

发布
迈克尔·汉格
2011年12月23日

亲爱的Spring开发人员和Graphistas：

我们很高兴向您介绍Spring Data Neo4j 2.0的发布，这是我们身边的小礼物。Spring Data Neo4j基于Neo4j 1.6。M02

此版本的主要功能是添加了一个简单的映射模式 （spring-data-neo4j）。只需注释您的POJO，并为通常的CRUD和高级查询操作使用GraphRepository 。

对于附有图的POJO和高性能用例，您可以采用高级映射模式 （spring-data-neo4j-aspects），该模式利用AspectJ来增强您的域类。

两种映射模式都使用相同的基础代码，该代码现在基于Spring Data Commons映射基础结构。

通过支持新的Cypher功能，将从finder方法衍生的查询添加到存储库并扩展了结果处理转换以包括对映射接口，Pages等的投影，我们改进了Cypher图查询语言支持。

除了还添加了Neo4j-spatial项目提供的初步地理空间支持外，我们还支持新的类型安全的Cypher-DSL ，它也可以与Query-DSL结合使用。

现在， 示例项目已包含在主源代码树中，因此它们始终是最新的。cineasts教程应用程序还包含在3个版本的示例中（简单映射，高级映射，REST）。

由于最近在Heroku上公开了Neo4j附加组件的公开可用性，因此我们包括了有关如何将Spring Data Neo4j应用程序部署到Heroku云中的一章。您还将找到一个随附的示例应用程序，名为“ todos”，已准备好进行部署。

特别感谢Junisphere的James和Werner所做的所有代码贡献和修复。

在过去的几周中，我们在春季论坛 ， JIRA和Neo4j邮件列表中获得了更多反馈。我们利用您的帮助消除了错误，改善了行为并提供了文档。非常感谢所有报告问题并提供见解的人。

要了解有关Spring Data Neo4j的更多信息，请确保观看项目负责人Michael Hunger的介绍性网络研讨会 ，并阅读大量的指南 。有关Spring One会议的详细介绍 ，请访问InfoQ。

然后请动手，将Spring Data Neo4j包含在您的假期项目中，您可以用它以全新的视角看待您的域。

节日快乐！
Spring-Data和Neo4j团队
