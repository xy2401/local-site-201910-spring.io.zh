Spring 的一周：2011年3月22日

工程
乔什·朗（Josh Long）
2011年3月23日

另一个伟大的一周-很多新的（和小说！）内容类型，的确是新内容的新来源！请享用！



  1. SpringSource推出了新的SpringSource YouTube频道 。该频道提供了有关SpringSource技术的令人兴奋的技术内容，并且应该是新内容以及您可能第一次错过的内容的第一站。立即查看！我个人认为，该频道在支持浏览器或YouTube（Google TV / Apple TV / slingbox / etc。）的大屏幕电视或外围屏幕（例如第二台笔记本电脑或第二台显示器）上都是很棒的。您还可以在通勤途中在便携式设备（例如车载iPhone）中收听音乐。

  2. …谈到YouTube频道，上周网络研讨会的内容和幻灯片-
    Spring和STS入门 -适用于错过它的人。Spring Framework 的负责人Juergen Hoeller几周前在一个在线研讨会上介绍了令人兴奋的Spring Framework 的下一代（3.1）， 也可以在SpringSource YouTube频道上观看 ！


  3. Martin Lippert宣布了SpringSource Tool Suite的最新版本2.6.0。新版本包含新功能，并跟踪各种项目的最新版本（Spring Roo 1.1..2，Eclipse Helios SR2，Groovy 1.7.8，Grails 1.3.7）。它包括对Spring Roo和Grails上的Groovy的更新支持和性能，Spring Web Flow的图形编辑器，以及对内容辅助，对Spring注释的快速修复和重构的新支持（ @Autowired ， @Qualifier ， @RequestMapping等）。

  4. Ramnivas Laddad 在春季的“本周”中指出了这个非常创新的Chrome浏览器插件。要使用它，请在多功能框栏中（搜索/地址栏中）键入“ spring”，后跟一个空格，然后输入要搜索的类的nane。它将自动显示链接到文档的候选结果！棒极了！（注意：我们无疑使用刚刚发布的Chrome 10对此进行了测试，尽管它无疑至少与Chrome 9兼容。）

  5. InfoQ 收录了去年在芝加哥举行的SpringOne2GX活动中Mark Pollack和Chris Richardson对Spring Data 的 精彩介绍的视频。Spring Data是一种以熟悉且习惯于Spring用户的方式利用这些强大的新数据存储选项（有时称为“ NoSQL”存储）的好方法。看看这个！

  6. 具有对Neo4j的支持的Spring Data项目的最新版本已经发布。新版本包括许多新功能，并跟踪Neo4j本身的最新版本。Neo4j是当今可供开发人员使用的许多新的专用数据存储库之一。Neo4j将数据建模为关系和节点。它针对快速的节点遍历进行了优化，例如在Facebook朋友图中。

    此外，SpringSource还将在4月20日举办有关Spring Data（特别是包含Neo4j支持的Spring Data Graph子项目）的网络研讨会。该演讲将针对北美和欧洲进行 。现在注册！

  7. Spring Framework 可帮助您构建最佳的应用程序。Spring Social以此承诺为基础，并允许您将应用程序与用户集成。JTeam小组又来了一次，这次是对Spring Social进行了有趣的更新，包括源代码和示例应用程序 。看看这个！


  8. Spring MVC为许多不同类型的视图提供支持，包括RESTful负载，Velocity模板，JSP和基于Tiles的视图。这篇博客文章解释了如何将Spring MVC与Tiles集成

  9. 来自Chariot Solutions的Ken Rimple已录制了有关SpringSource系列最新成员Wavemaker 的截屏视频 。

  10. 现在使用Spring Framework 的人比以往任何时候都多，其结果是对各种工具中Spring的更新，专业支持。一位UML工具制造商Architexa谈论了基于Eclipse的产品对Spring Framework 的增强支持。
    查看此博客以获取更多信息。

  11. Ashish Jain提供了有关Spring 3.1的新缓存支持的有趣观点。

    Ashish选择简化他的软件堆栈（通过删除Hibernate，而是通过Spring Framework 的直接JDBC使用） JdbcTemplate ，是一个不错的选择！），但担心不得不放弃Hibernate的声明式二级缓存。有了Spring 3.1中的缓存支持，无论底层的数据访问技术如何，他都可以吃蛋糕，也可以吃！


  12. 坚持使用JBoss AS？仍然需要提高生产力，并在您的应用程序中需要更好的安全支持吗？该博客介绍了如何在JBoss上设置Spring Security 3。 看看这个。
