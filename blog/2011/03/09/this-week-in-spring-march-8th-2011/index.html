    春天的一周：2011年3月8日

    工程
    乔什·朗（Josh Long）
    2011年3月9日

    本周又出现了一系列激动人心的新版本和更新，丰富的社区内容以及激动人心的公告，欢迎WaveMaker加入SpringSource系列。

    1. 罗德·约翰逊（Rod Johnson） 今天在SpringSource博客中宣布了对WaveMaker的收购。 WaveMaker是一种广泛使用的图形工具，它使非专业开发人员能够快速构建Web应用程序。摘自文章：“虽然WaveMaker已经成为Spring生态系统的一部分，但现在它将成为Spring系列和VMware云战略不可或缺的一部分。WaveMaker的所有员工都将加入VMware。”令人兴奋的消息！请务必查看该帖子以获取所有详细信息。

    2. Craig Walls上周宣布了Spring Social项目的M2，本周对Spring Social中的Social Service Provider框架进行了很好的介绍。令人兴奋的东西，以及任何希望与此框架进行互操作的东西-要么是因为您想使用社交网络的API，要么是想为您的网络API提供Spring Social绑定- 务必阅读此博客

    3. Tomcat专家社区刚刚宣布Tomcat 7.0.10已发布。新的更新包括一些错误修复和安全修复，以及一些很酷的功能。

    4. 在介绍Spring Web Services的博客教程的第二部分中 ，博客“ panbhatt”介绍了JAXB和Spring OXML（对象到XML）编组框架。
    5. Ravindra Rawat 使用Spring Data和Redis引入了一个很酷的电影推荐应用程序 ，并在Github上附带了示例代码！ 这太酷了 ！如果您对某些NoSQL解决方案的功能完全感到好奇，那么像这样的可用，相关的日常应用程序应该可以缓解您的好奇心！
    6. Giant Flying Saucer博客上的Chad Lung介绍了一个5分钟的教程，以使用免费的SpringSource Tool Suite启动Spring MVC项目。 我喜欢这个博客，但我有一个小抱怨：我按照他的步骤进行，只用了3分钟！ :-)

    7. Developer.com对Spring WebServices 2.0中的所有新功能进行了很好的介绍。声明是一回事-在这种情况下，Spring WS是“基于SOAP的Web服务的最佳Java堆栈”-但本文还提供了这一点。

      总体上提供了有关Spring Web Services的很好的介绍。不错的工作！

    8. 这篇文章演示

      如何将Spring Roo Vaadin表单集成到Magnolia CMS中。 Vaadin是一个Web框架，它将GWT编程范例带到了逻辑极限，并且可以成为传统的以组件为中心的服务器端框架（如JSF）的非常引人注目的替代方案。

    9. 当我仔细阅读本周综述的所有Spring Data内容时，我以一种about回的方式发现， EmilEifrém在去年Oredev会议上介绍Spring Data的精彩演讲已经结束了！EmilEifrém是Neo Technology的首席执行官，也是Neo4j图形数据库项目的创始人。

    10. Moez Ben Rhouma之前还演示了如何将Spring Security与ExtJS集成，并演示了如何使用Wicket设置Spring Security 3 。太酷了！
    11. 想要看看Apress的《 Spring Persistence with Hibernate》一书吗？阅读这本书的评论
