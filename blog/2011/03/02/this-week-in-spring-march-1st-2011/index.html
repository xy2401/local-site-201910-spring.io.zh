Spring 的一周：2011年3月1日

工程
乔什·朗（Josh Long）
2011年3月2日

本周又出现了一系列激动人心的新版本，更新和精彩的社区内容。戴上帽子！让我们开始吧。

  1. Spring Roo 1.1.2已发布。 自上个月1.1.1发布以来，此新版本包含200多项增强功能，新功能和修补程序。其中的一些亮点包括增强的工具可用性，持久性增强，GWT和Spring MVC增强以及库升级。
  2. Martin Lippet刚刚宣布了2.5.2。SR1和2.6.0。M2版本的SpringSource Tool Suite 。2.5.2。SR1“刷新”已更新了对Groovy 1.7.8，Grails 1.3.7，Eclipse Helios SR2（3.2.6）和刚刚发行的Spring Roo 1.1.2的支持。2.6.0。M2 Mileton还包括一个Roo插件管理器，改进了对@RequestMapping ，并对Groovy和Grails工具进行了大量改进。

  3. 马丁·利佩特（Martin Lippet）还收集了有关经常提出的新问题的信息：

    我在SpringSource Tool Suite中提供自定义项目模板吗？

    有一个很棒的项目设置方法，想将其编成代码，并轻松地与公司或社区中的其他STS用户共享吗？现在你可以！


  4. 在继续介绍Spring 3.1的系列文章之后，Costin Leau撰写了有关缓存抽象的介绍，使您可以集成诸如GemFire，Ehcache，基于JDK ConcurrentMap的基本缓存以及与其他应用程序一起实现SPI的其他缓存。

  5. Green Beans系列中的一个新条目已经出现，这是Spring Integration入门 。查阅来了解有关如何使用Spring解决Spring Integration集成问题的所有信息

  6. Spring Web Flow 2.3.0已发布 。新版本具有许多新功能：JSR 303 bean验证支持，嵌入式流程（即，您可以想象页面上的模式对话框完全具有自身的流程-不需要刷新封闭的页面，以便嵌入式流程可以继续（通过Ajax）。将流管理的PersistenceContext自动传播到子流，支持Portlet 2.0资源请求，等等。
  7. Spring 社交1.0.0。M2已发布 。新的里程碑包含了几个主题的新功能。新版本将连接和身份验证的概念与各种软件即服务提供商的API分离开来，因此现在您可以一起或独立地使用这两种工具。新版本还扩展了对包括Github在内的Saas产品的初始支持。检查一下并提供反馈！

  8. 谈到Spring Social，Craig Walls（Spring社会项目负责人） 在去年10月的SpringOne2GX 2010上发表了关于开发支持社交的Web应用程序的演讲。 您可以在InfoQ.com上观看

  9. Blogger Matt Raible撰写了一篇文章，探讨了使用Spring Security，HTTPS和jQuery进行Ajax身份验证。 很酷。又详细！看看这个！
  10. Spring Data Graph项目的最新版本1.0.0。支持Neo4j的M3刚刚发布。此版本带来了许多新功能，包括对Neo4j-1.3的更新。M03，AspectJ 1.6.11。M2，对可拆卸实体的显式支持，bean验证，Java配置简化，功能齐全的Neo4jTemplate等！
  11. Eclipse处女座，3.0.0。M2，现在可供下载！ 新版本在最近的M01里程碑所做的大量更改的基础上，修复了错误。
  12. 是否想使用RESTful服务，但又不想冗长的XML有效负载减慢通信速度？REST只是HTTP的一个应用程序，而HTTP为我们提供了令人惊叹的经过实践检验的真实工具和技术，可以扩展基于HTTP的应用程序。一种常见的方法是使用GZip压缩。在这篇文章中，博客作者Stephan Oudmaijer介绍了一种使用Spring的方法RestTemplate使用GZIP压缩的有效负载来支持该用例！

  13. Sivaprasadreddy Katamreddy又来了，这一次很好地介绍了如何利用Spring 3.0 Java配置模型编写没有XML的Spring / Hibernate应用程序 。下次准备像本书这样的食谱时，总是很方便！

  14. 是否曾经需要通过Java与LDAP交互？仅搜索Spring LDAP项目即可，Spring LDAP项目为此提供了一个干净，全面的工具。这篇关于使用Active Directory（AD Windows

    Java和Spring LDAP的Server 2008）提供了很好的介绍。

  15. 在Spring 3.0中，基于Java的配置机制总是有很多新用途。这次，博客作者Jason Stillwell撰写了有关如何使用Spring Java配置来处理public static void main()方法。
