Spring Data Graph-Neo4j支持1.0.0。M4发布

发布
托马斯·里斯伯格
2011年3月16日

亲爱的春季社区，

我们很高兴地宣布一个新的里程碑版本（1.0.0。支持Neo4j的Spring Data Graph 1.0项目的M4版现已发布！

Spring Data项目的主要目标是使使用新数据访问技术（例如非关系数据库，map-reduce框架和基于云的数据服务）的使用Spring支持的应用程序更容易构建。

Graph Neo4j模块提供与Neo4j图形数据库的集成。

资料下载 JavaDocs | 参考文档 | 变更日志

要了解有关该项目的更多信息，请访问Spring Neo4j主页 。

Spring Data Graph 1.0.0中的新功能。M4包括：

  • 更新到Neo4j-1.3。M04
  • 分离/连接状态的简化
  • 所有节点实体在创建时都是分离的，必须调用persist（）
  • 将attach（）重命名为persist（）
  • 每个领域类别的单独索引
  • 支持全文索引
  • 固定直接查找数字索引值
  • 现在，通过relatedTo的关系也被限制为每个类型，方向和目标之一
  • 为基于实体的Neo4jTemplate回调添加了EntityPath / EntityPathMapper
  • 添加了EntityEvaluator用于基于实体的路径评估
  • 非类型节点上调用的节点类型策略的错误处理
  • FieldTraversalDescriptionBuilder构建方法参数化
  • 重新启用，更新和固定的跨商店持久性
  • 图形管理的字段不再需要使用@Transient注释（方面可以解决此问题）
  • 无需为跨商店实体使用@Transient标记图管理字段
  • 将文档格式更改为指南，并在第一部分中添加了教程
