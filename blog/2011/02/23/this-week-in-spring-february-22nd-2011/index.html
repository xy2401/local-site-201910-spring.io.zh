春天的一周：2011年2月22日

工程
乔什·朗（Josh Long）
2011年2月23日

好一个星期！本周，社区以及社区都提供了许多很好的基础内容。让我们开始吧。 

     
  1.  Rossen Stoyanchev continues the whirlwind introduction to the new features in Spring 3.1, this time with a look at the MVC namespace and @Configuration-based improvements. 
    2.
  2. Chris Beams在博客中介绍了Spring 3.1中即将提供的FeatureSpecification支持。Feature Specification 以Java配置为中心的方式提供了与XML名称空间基础结构元素相同的功能，例如。这是一项非常强大的功能， 这篇博客文章是入门的最佳方法。


  3. SpringSource Tool Suite    负责人Christian Dupuis-发推文 ，用户现在无需下载注册表即可下载STS。很酷！

  4.  Can't get enough Spring?

    5. 在您最喜欢的社交网络上查看Spring！



  5. Glyn Normington刚刚宣布了Eclipse Virgo 3.0.0。现在可以下载M01版本     。新版本
    具有与Eclipse堆栈（包括Eclipse Virgo Jetty服务器）的更紧密集成。此外，它还与Equinox紧密集成。
  6. Last week, the Spring Integration team released 2.0.3 of the Spring Integration framework. Lots of good fixes and features in this release. Additionally, it's a minimum version to work with the latest Spring BlazeDS release, so check it out now.
  7. Dr. Mark Lui and others at Shopzilla have put together a post on their use of the Spring Integration framework at Shopzilla and how to use it. Pretty cool, guys!


  8. Nicolas Labrot对前景感到非常兴奋
    宣布宣布在Spring MVC中使用Grails作为视图的过程
    在即将到来的Spring 3.1功能中，他去了并实现了该功能的原型。看看这个！看起来很有希望。

    9.                  
  9.  Ken Rimple, co-author of the upcoming Spring Roo in Action has written up an
                   introduction to the Spring Container's support for BeanPostProcessors, which allow you to tailor the way beans in the context are registered. This is a very good introduction to one of the Spring container's most powerful features. Check it out!  
    10. 
                   
  10. 

              Cody Burleson  pens an introduction to using Spring, the Dojo JavaScript framework, and WebSphere portlet support in RAD together.  

                    
    11. 
                   
  11. The Java Code Geeks are at it again, this time with a look at their view    domain-driven design using Spring and AspectJ.
