春天的这个星期：2011年2月1日

工程
乔什·朗（Josh Long）
2011年2月2日

本周社区的发展令人着迷。取决于您的外观，令人兴奋的是更新的，更前沿的东西（例如Spring Data，Gemfire和Virgo）的背后，或者是对Tomcat和SpringSource Tool Suite等技术的出色改进的背后。无论哪种方式，本周都有很多事情要做，所以让我们开始吧！

  1. SpringSource Tool Suite 2.6.0。M1已发布。这个新版本具有改进的内容辅助和快速修复，基于Spring MVC @RequestMapping的应用程序的新向导和视图。它还具有针对Groovy的改进的内容辅助性能，以及针对Groovy的改进的语法突出显示功能。
  2. Spring Data Graph-Neo4j支持1.0.0。M2发布 。新版本具有许多新功能，包括改进的索引支持-完整，对节点和关系的命名索引支持，节点和关系实体的删除，对框架方法中实体实例的严格类型检查（根据类型策略），对动态投影的支持到任意图形实体，以及对Neo4j 1.2的新更新支持。
  3. Shekhar Gulati 在IBM的DeveloperWorks上发布了Spring Roo的简介（第1部分，从源代码构建） 。
  4. Tomcat项目的高级撰稿人Mark Thomas 写了有关防止Tomcat 7中的跨站点脚本攻击的文章。
  5. VMWare vFabric GemFire团队刚刚整理了一段视频，演示如何设置GemFire的Hibernate Cache Module 。该视频演示了如何在Maven中配置模块，然后演示了其在基于Spring-DAO的实现中的用法。最后，演示了如何使用Spring Insight监视应用程序的性能。酷视频，更酷的技术！

  6. 塞巴斯蒂安·彼得罗夫斯基（Sebastian Pietrowski）
    已经发布了有关Spring Data Redis的很好的介绍 。

  7. Shekhar Gulati在Spring Data和Redis上发表了另一篇很棒的文章。Spring Data是一系列旨在支持更专业的数据持久性需求的技术的总称。其中很大一部分是对当今可用的许多NoSQL数据存储的深入支持，以及对基于JDBC的持久性的更专业支持。
  8. Java Code Geeks在Spring AOP的面向方面编程方面写了一篇很好的文章
  9. Jose Delgado 在Spring Roo中撰写了一篇不错的文章， 介绍如何配置应用程序以处理两个数据库。
  10. Sivaprasadreddy Katamreddy又来了， 这次是将控制反转和依赖注入应用于方法设计
  11. Glyn Normington 表示，Eclipse Virgo内核现在首次成功运行在使用OSGi 4.3框架挂钩实现的区域的有向图上。恭喜你！Eclipse Virgo是以前的SpringSource dm Server的新改进版本的名称。
  12. David Dossot转推了一条有趣的帖子，探讨了

    上个月初使用Spring 的Activiti BPMN 2引擎 。

  13. 有了Tomcat 6和7的所有令人振奋的消息，很容易忘记Tomcat 5仍在更新和受支持。Tomcat 5.5.32也刚刚发布！
