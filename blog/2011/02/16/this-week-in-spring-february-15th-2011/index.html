Spring 的一周：2011年2月15日

工程
乔什·朗（Josh Long）
2011年2月16日

上个星期看到了创纪录的数量……好吧，记录，如果没有别的！首先，Juergen Hoeller（ Spring 项目负责人，周围都是好人）举行了网络研讨会，介绍了Spring 3.1中的新功能以记录出席人数。然后，似乎还不够，第二天发布了Spring 3.1的第一个里程碑，以记录下载次数。这样就足够了。我不认识你，但我可以在那儿停下来。然后 ，在上个星期的过程中，我们看到了我想象中的其他Spring项目的新版本创纪录数量！终于，昨天，毫无疑问，对于所有其他记录活动，我们看到了SpringSource博客上人数最多的记录 ！总而言之，Spring 3.1里程碑式的狂热，众多新项目以及一如既往的健康社区敬业精神（请加入团队！） 

      <li>Hot on the heels of Juergen's Spring 3.1 webinar, Chris Beams got the ball rolling on the  first three   of many  posts on the numerous new features in Spring 3.1. 

    <a href="http://blog.springsource.com/2011/02/11/spring-framework-3-1-m1-released/">

    第一篇文章通常介绍Spring 3.1 M1。
    第二篇文章介绍了Spring 3.1的“配置文件” ，可让您使用Environments沿特定于环境的行对bean进行分区。第三篇文章介绍了
    环境抽象。     

          </li>


<li>
    Matt Raible has put together a good introduction to extension-less URLs in  
    <a href="http://java.dzone.com/articles/implementing-extensionless" > 
       Spring MVC, Tapestry 5, Struts 2 and JSF on JavaLobby. </a>  Check it out!

</li>
 <li>  The <a href="http://www.springsource.org/node/3025"> first Spring Security 3.1  release candidate has been released</a>, featuring numerous fixes and updates since the last milestone. </li> <li>Jeremy Grelle <a href="http://www.springsource.org/node/3024">announced the 1.5.0 RC2 release of the Spring BlazeDS project</a> - lots of exciting new features (including compatibility with the recently released Spring Integration 2.0) </li>
 <li>Christophe Coenraets provides  a quick introduction <a href="http://www.java-tv.com/2011/02/14/integrating-spring-and-flex/">to integrating  Spring and Flex using the Spring BlazeDS project! A great introduction for even greater technologies!
    </a> </li>
       <li>The hits keep on coming out of the Spring Data project - Thomas Risberg just announced the first milestone of the Spring Data Document (1.0.0.M1) sub project's <a href="http://www.springsource.org/node/3032">support for MongoDB.</a>  </li>  
       <li>Oliver Gierke just announced the first milestone of the Spring Data JPA sub project. The Spring Data JPA project - formerly called the Hades project - provides a conventions-centric approach to build JPA-based repositories to access data with a minimum of fuss. </li>
       <li>Costin Leau also announced the <a href="http://www.springsource.org/node/3023">second milestone of the Spring Data Key Value </a> sub project, encompassing support for the Riak and Redis NoSQL stores. 
         </li>
 <li> 
    For the many Python users out there, <a href="http://forum.springsource.org/showthread.php?t=103530">Greg Turnquist just announced</a> 
    the new, 1.2.0 FINAL release of       <a href="http://www.springsource.org/extensions/se-springpython-py">Spring Python,</a> the framework that brings idiomatic Python  ("Pythonic") dependency injection and support - in the spirit of the Spring framework - to the Python language, has just been released! Check it out! 
    </li>
<li> <a href="http://www.springsource.org/node/3029">Chris Beams' SpringOne2GX 2010 introduction to the new configuration options in Spring 3.1</a> is now available on InfoQ.com 
 <li>
    <a href="http://www.springsource.org/spring-android/news/1.0.0.m2-released">The Spring Android project, version 1.0.0.M2, has just been released.
       </a> 

 </li>

    Gordon Dickens is at it again, this time with a review of the <a href="http://gordondickens.com/wordpress/2011/02/11/spring-3-1-whats-up-and-coming/">cool features coming in Spring 3.1. </a>
     </li>
