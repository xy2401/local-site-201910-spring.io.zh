2011年11月29日春季，本周

工程
乔什·朗（Josh Long）
2011年11月30日

欢迎回到春季本周的另一期。本周有很多话题可以讨论，还有很多新版本，所以让我们开始吧！



  1. 克里斯·比姆斯（Chris Beams）宣布了Spring 3.1 RC2的最新，最强大的发行版 。这是预期的最终版本，因此请试一试并尽快尝试。有关Spring 3.1的内容，请查看发行说明和Spring 3.1博客系列。

    2.  
  2.  
    The steady march to Spring Integration 2.1 GA continues. This week, Spring Integration 2.1 RC1  was  released.  
There are a lot of new features in  Spring Integration 2.1, including support for GemFire, RabbitMQ, MongoDB, and much, much, more.  For the full details, see the release notes.  
    3. 
 
  3.    
SpringSource Tool Suite lead Martin Lippert has announced the latest release of SpringSource Tool Suite, version 2.8.1, which provides a very compelling feature: compliance with both versions of the Maven plugins typically supported: M2E, the new, Eclipse-foundation supported integration, and M2Eclipse, the original integration furnished by Sonatype.  This makes it possible for  developers to upgrade and transition easily. All users are encouraged to upgrade immediately. 
 
    4.  



 
  4.   Spring Roo project lead Alan Stewart  has announced that Spring Roo 1.2.0.RC1 has been released. The new release  is packed with new features, including multi-module Maven projects, JSF 2.0 and RichFaces support, and much more.
     
    5.  
 
  5.

    6. Daniel Mikusa撰写了精彩的文章介绍

    运行Apache Tomcat时的JVM性能调整 。文章介绍了各种命令行选项，包括-Xss ， -XX:PermSize和-XX:MaxPermSize ，以及有关选择JVM垃圾收集器的一些详细信息。好贴！

  6.    In this new video interview from InfoQ, Spring expert Costin Leau talks about Spring Data, caching, data grid architectures and work on a new Spring Hadoop project. 

    
    7.

  7. BSB实验室用Spring Batch撰写了一篇有关远程分区的精彩文章 。

        Spring Batch, as readers already know, is a powerful framework for building batch jobs. Spring Batch makes it dead simple to handle long running, multi-step batch processing jobs, and - with remote partitioning, it is even easier to partition those jobs across multiple nodes. This post introduces some of the cool features in the remote partitioning features available in Spring Batch. Check it out!
  8. eWeek has an interesting post about a recent Evans Data survey that found that developers prefer Cloud Foundry. Not a lot of technical content, of course, but this provides a lot of good insight into why others are choosing Cloud Foundry and can be useful when you're trying to make the case for Cloud Foundry in your organization.
  9. ActiveState, creators of the Stackato cloud, which is based on Cloud Foundry (the open source PaaS from VMware), has announced that their Micro Cloud will remain free after the beta is finished. This ensures that developers can develop against the cloud in perpetuity.


  10. Spring使用代理来发挥其魔力。代理有三种类型，每种都有自己的优点和缺点。
    Tomasz Nurkiewic 介绍了Spring如何使用动态代理，CGLIB代理和AspectJ代理以及一些潜在的陷阱。这是一个非常好的帖子！

  11. Stacey Schneider在新版7.0.23 Apache Tomcat上发表了精彩的文章。
    在新版本的新功能中，您将发现能够并行启动和停止Web应用程序，以缩短启动时间，缓存配置文件和改进对失败的部署重新启动的处理。

  12. Eclipse Virgo 3.0.2已发布 。这个新版本修复了一些错误 。
