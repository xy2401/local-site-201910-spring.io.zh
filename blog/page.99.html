Spring Cloud Pipelines 1.0.0。M3发布

发布
马尔辛·格热兹扎克（Marcin Grzejszczak）
2017年2月9日

我很荣幸代表Spring Cloud团队宣布Spring Cloud Pipelines的新里程碑发布- 1.0.0.M3 。

什么是新的？

除了一些错误和文档修复之外，它还为Cloud Foundry（适用于Concourse和Jenkins）上的蓝绿色部署提供了开箱即用的支持！这是我们的做法。

当您单击deploy to prod

  • 我们正在重命名应用程序的当前实例，例如fooService至fooService-venerable

  • 我们正在将应用程序的新实例部署在fooService名称

  • 现在，同一应用程序的两个实例正在生产中运行

当您单击Complete switch over步：

  • 我们正在删除旧实例（ fooService-venerable ）

重要
 请记住，只有在确认两个实例都可以正常工作之后，才执行此步骤！

春天云卡姆登。SR5可用

发布

2017年2月6日

我很荣幸地代表团队宣布， Spring Cloud Camden Release Train的Service Release 5今天开始可用。该版本可以在我们的Spring Release存储库和Maven Central中找到 。该文档可在此处找到。

此版本中包括所有Spring Cloud项目的Spring Boot 1.5兼容性。除了添加Spring Cloud Task之外，此版本主要包括错误修复。

注意 ：此发行版与Spring Boot 1.3不兼容。换句话说，您的Spring Boot 1.3应用程序将不适用于Camden。SR5。
