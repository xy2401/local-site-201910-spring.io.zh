SpringOne2GX 2015重播：Spring Integration Java DSL

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年3月1日

记录在SpringOne2GX 2015中。
演讲人：Artem Bilan
数据与整合
幻灯片： http ： //www.slideshare.net/SpringCentral/spring-integration-java-dsl

Spring Integration Java DSL是Spring Integration编程模型的有机扩展，结合了Spring Framework Java和注释配置支持，Spring Integration EIP实现和Builder模式，提供了XML配置的替代方案。

在本节中，我们将讨论Spring Integration Java DSL fluent API，以及它如何使您仅使用Spring Java配置即可轻松，快速地向应用程序添加集成功能。我们还将展示Java 8 Lambda支持所提供的好处，以及Spring Integration Java DSL实现的核心Spring Integration的附加功能。我们还将讨论关注点和限制，例如方法链复杂性，Java 8之前的支持等。

网络研讨会重播：使用Spring Boot微服务创建PageRank分析平台

彼得·汉弗莱（Pieter Humphrey）
2016年3月1日

演讲者：肯尼·巴斯塔尼（Kenny Bastani）
幻灯片： https ： //speakerdeck.com/kbastani/creating-a-pagerank-analytics-platform-using-spring-boot-microservices
网络研讨会内容的指导教程： http ： //www.kennybastani.com/2016/01/spring-boot-graph-processing-microservices.html
网络研讨会的参考应用程序： https : //github.com/kbastani/spring-boot-graph-processing-example

该网络研讨会向您介绍了一个示例应用程序，该应用程序将多个微服务与一个图形处理平台结合在一起，可以在Twitter上对用户社区进行排名。我们要解决的问题是如何使用一组种子配置文件作为输入来在Twitter上发现影响者社区。在没有机器学习或社交网络分析背景的情况下解决该问题可能会有些困难，但是我们将使用一点计算机科学历史来对此问题进行探讨。在本文的示例应用程序中，我们将使用一系列流行的工具。我们将按重要性顺序使用的工具将是：Spring Boot，Neo4j，Apache Spark，Docker，RabbitMQ。尽管存在一些遗漏的部分，例如保护REST API访问以及将设备卷映射到容器，但这些重要主题将在以后的网络研讨会中介绍。

  • 允许覆盖默认的RedisSerializer

  • 允许使用redisNamespace在Redis中确定会话范围。谢谢fstegmann ！
  • 允许使用redisFlushMode立即（而不是懒惰地）写入Redis
  • 添加ExpiringSession.setLastAccessedTime（long）
