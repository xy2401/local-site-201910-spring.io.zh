发布
于尔根·霍勒（Juergen Hoeller）
2015年5月26日

尊敬的Spring社区，

我很高兴地宣布，我们的里程碑资料库中现已提供Spring Framework 4.2 RC1。这是4.x产品线中的一项功能发布，重点关注核心改进和现代Web功能：

  • Java 8默认方法上的注释检测（例如， @Bean ）

  • 对HTTP流和服务器发送事件的一流支持
  • Listenable / CompletableFuture作为消息处理程序的返回值
  • 通过TCP和WebSocket通道使用的STOMP客户端
