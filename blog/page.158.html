Spring Boot 1.3.0。M5现在可用

发布
菲尔·韦伯
2015年9月4日

Spring Boot v1.3.0。M5现在可以从Spring里程碑存储库中获得 。刚好赶上SpringOne 2GX，此版本关闭了将近50个问题，并合并了15个请求请求 ！再次感谢大家的贡献。

新版本的重点包括：

  • 走向RC1时，大量的错误修复和稳定化
  • 对Javax的支持@WebServlet ， @WebFilter和@WebListener注解
  • Hazelcast的自动配置
  • 改进的“根优先”异常日志记录

有关更改的完整列表和升级说明，请参阅Wiki上的《 Spring Boot 1.3发行说明 》和更新的参考文档 。

Spring Framework 4.2.1现在可用

发布
斯特凡·尼科尔（StéphaneNicoll）
2015年9月1日

我很高兴地宣布，可从repo.spring.io和Maven Central获得Spring Framework 4.2.1。4.2行中的第一个维护版本包含针对4.2 GA报告的大量回归和其他问题的修复程序 。

4.2.1还包含一些小的增强功能：完善了我们对Jackson的支持（请参阅Sébastien更新的博客文章 ），元注释处理的改进（ @AliasFor ），第三方依赖项对齐（包括Hibernate 5.0 GA）。

我们强烈建议所有4.x用户立即升级到4.2.1。Spring框架团队现在正在10月初致力于Spring Framework 4.2.2，这将是即将发布的Spring Boot 1.3 GA版本的基础。我们还将同时发布4.1.8：如果您仍然想在4.1.x分支中进行修复，现在是时候提高它了！
