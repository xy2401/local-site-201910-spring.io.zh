2014年4月8日

演讲者：马丁·利珀特（Martin Lippert）（关键），迈克·米林科维奇（Mike Milinkovich）（Eclipse Foundation）

Spring Boot 1.0.1。现在发布

发布
戴夫·瑟
2014年4月7日

Spring Boot 1.0.1。RELEASE在Maven Central和repo.spring.io存储库中可用。这是一个错误修复版本 ，尽管添加了一些新功能：

  • MessageSource缓存可以在application.properties
  • ActiveMQ连接凭据可以外部化为application.properties
  • 参考指南中有关于安全性自动配置的新部分

Actuator有一个安全错误修复程序，因此使用Spring Security和Actuator端点的任何人都应该升级。不会出现向后兼容性问题或对现有功能进行更改。

发布了Spring Tool Suite和Groovy / Grails Tool Suite 3.5.0

发布
马丁·利珀特
2014年4月2日

亲爱的春季社区，

我们很高兴宣布今天基于Eclipse的工具的下一个主要版本：Spring Tool Suite（STS）3.5.0和Groovy / Grails Tool Suite（GGTS）3.5.0。

此版本的重点包括：

  • 即用型完整Java8支持（STS和GGTS）
  • 更新到Eclipse Kepler SR2（4.3.2）
  • 用于Spring Boot项目的新的高级内容助手
  • 用于Spring Boot的实时Bean Graph的可用性增强
  • Cloud Foundry集成现在支持Spring Boot
  • Gradle Integration for Eclipse的重大改进（包括重大性能改进）
  • 对Spring Boot CLI应用程序的初始支持
  • Grails更新至2.3.7
  • tc服务器更新为2.9.5。SR1
  • 仪表板改进（包括提要，快捷方式和通知）
