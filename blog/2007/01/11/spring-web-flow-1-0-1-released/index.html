Spring Web Flow 1.0.1发布

发布
基思·唐纳德
2007年1月11日

亲爱的春季社区，

我们很高兴地宣布，Spring Web Flow 1.0.1已经发布。

 
下载 | 实用介绍 | 样品 参考手册 变更日志 | 1.0升级指南

Spring Web Flow是下一代Java Web应用程序控制器框架，允许开发人员将用户操作建模为称为流的高级模块。该框架提供了提高的生产力和可测试性，同时为执行导航规则和管理应用程序状态提供了强大的解决方案。

发行说明

Spring Web Flow 1.0在两个月的时间内下载了10,000次以上，并在《财富》 500强公司中得到大量使用。这是第一个1.0维护版本，其中包括自1.0最终版以来已报告的错误的修复程序，并引入了一些较小的增强功能。请查看更改日志以获取所有详细信息。

入门

开始使用Spring Web Flow的最佳方法是下载发行版 ，阅读Erwin的实际介绍 ，并逐步浏览示例应用程序 。我们建议您从电话簿开始审查所有样本，并根据需要补充参考手册材料。该版本附带了十个示例应用程序，每个示例应用程序都展示了一套独特的产品功能。
