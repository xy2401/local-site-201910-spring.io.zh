Spring Security 2有什么新功能？

工程
本·亚历克斯
2007年12月6日

我今天在博客圈中巡游，遇到了我读过的最短的博客之一。引用几乎整个条目：“每次使用Acegi时，一位仙子都会死亡。可悲的是，周围真的没有什么更好的……”。

在我们的社区论坛 ， 开发人员 列表 ， JIRA ， 用户 会议 BOF ， 培训 ， 支持 ， 咨询和团队博客之间 ，我们收到了大量社区反馈。毫无疑问，许多人都在寻求对Spring Security（以前称为Acegi）配置格式的改进，并且我们投入了大量时间使之成为可能。

正如我将展示在下周的Spring体验大会上，春季安全2.0.0 M1功能大大简化了配置。现在，您可以通过以下三个步骤将Spring Security添加到您的应用程序：


  1. 将所需的JAR添加到您的类路径中。当我们使用Maven进行构建时，Maven 2用户只需在其POM中指定一个依赖项即可。

  2. 将单个过滤器和相应的映射添加到您的web.xml。

  3. 在您的Spring容器配置文件中添加一些元素。


将过滤器添加到web.xml需要以下代码：


<filter>
    <filter-name>_filterChainProxy</filter-name>
    <filter-class>org.springframework.web.filter.DelegatingFilterProxy</filter-class>
</filter>

<filter-mapping>
    <filter-name>_filterChainProxy</filter-name>
    <url-pattern>/*</url-pattern>
</filter-mapping>

第三步（也是最后一步）要求您将新的Spring Security 2名称空间添加到现有配置文件中，或创建一个包含安全配置的新XML文件。对于许多用户而言，该文件将非常短且类似于：



<beans:beans namespace declarations>
    <annotation-driven />

    <http auto-config="true">
        <intercept-url pattern="/**" access="IS_AUTHENTICATED_REMEMBER" />
    </http>

    <repository>
        <user-service hash="md5:hex">
            <user name="rod" password="a564de63c2d0da68cf47586ee05984d7" authorities="ROLE_SUPERVISOR,ROLE_USER" />
        </user-service>
    </repository>
</beans:beans>

上面的配置可靠地为您的Web应用程序提供了以下功能：


  • 基于表单的登录

  • 注销处理程序

  • 自动生成登录页面

  • BASIC身份验证支持（理想的远程协议和Web服务）

  • 通过cookie记住我的支持

  • 匿名身份验证（因此所有匿名用户都具有用户名和角色）

  • 与数据库支持的用户存储库集成（您也可以使用 ）

  • 在Spring管理的对象上自动实施@Secured批注

  • Servlet API集成（即HttpServletRequest.isUserInRole（String）和HttpServletRequest.getPrincipal（））

  • 登录以使用应用程序的任何部分的要求


因此，该语句在单个元素中为标准Web应用程序提供了大多数需求。当然，您可以微调各个设置，例如指定其他领域名称（ ），自定义登录页面详细信息（ ），或切换到更高级的“记住我”令牌存储库（ ）。

包括名称空间声明形式在内，上述配置文件大约运行16行。要获得可比的功能，旧的配置样式将需要大约120行XML。

尽管配置方面的改进相当可观，但Spring Security 2.0.0 M1中还具有其他功能！首先，您现在可以选择在数据库中存储“记住我”令牌，从而提高了安全性。我们还具有分级角色支持，这是对类固醇的一种授权权限别名策略。另外，期待已久的用户管理API，NTLM支持，Portlet集成等等。

如果您想了解有关Spring Security 2的更多信息，请在The Spring Experience上实际使用它 。安排了三场会议，涵盖Spring Security 2，包括简介 ， 新功能和改进功能以及如何使用新的Portlet安全集成 。对于那些无法做到的人，您可以在下周晚些时候下载第一个里程碑版本。

希望这些增强功能可以节省一些仙女。
