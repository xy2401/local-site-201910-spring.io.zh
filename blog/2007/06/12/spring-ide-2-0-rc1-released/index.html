Spring IDE 2.0 RC1发布

发布
克里斯蒂安·杜普依斯（Christian Dupuis）
2007年6月12日

我们很自豪地宣布，已经发布了Spring IDE 2.0的第一个候选版本。阅读Spring IDE博客上的声明 。

Spring IDE 2.0包含许多新功能和许多错误修复。最值得注意的是，我们添加了对基于Spring 2.0基于名称空间的配置，Spring AOP（包括@AspectJ样式的方面），Spring Web Flow和Spring JavaConfig的全面支持。

下载 | 说明文件 | 变更日志

可以从我们的开发人员更新站点上立即找到候选版本。 http://springide.org/updatesite_dev 。请花一些时间进行测试，并就可能发现的任何错误，错误或问题提供反馈。非常感谢已经提供的反馈和错误报告。反馈对于我们真的很有价值。

下一个（最好是最后一个）候选发布者计划在下周发布，Spring IDE 2.0的最终版本应该在SpringOne 2007左右发布 。
