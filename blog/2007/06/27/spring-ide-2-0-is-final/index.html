Spring IDE 2.0最终版

发布
克里斯蒂安·杜普依斯（Christian Dupuis）
2007年6月27日

修复了大约250个错误并且在增加对Spring 2.0，Spring Web Flow，Spring AOP和Spring JavaConfig的支持上花费了无数的时间之后，我们很自豪地宣布Spring IDE 2.0的即时可用性。

下载 | 说明文件 | 变更日志

该发行版可从我们的发行更新站点获得 。Spring IDE 2.0是根据Eclipse Public License -v1.0的条款许可的 。

新的功能

Spring IDE 2.0包含许多新功能和许多错误修复。我们的票务系统中提供了所有已关闭票的列表。对于那些不熟悉Spring IDE最新开发的人，这里列出了一些简短的功能：

  • 支持基于Spring 2.0基于名称空间的配置。我们在其中做了很多工作，以使支持尽可能地扩展。您可以在另一篇文章中了解有关此内容的更多信息 。
  • 对Spring Web Flow的支持，包括对WTP XML编辑器的扩展，用于内容辅助和超链接以及验证和图形编辑。可在此处获得更多信息。
  • 基于Spring AOP的开发工具 。这包括基于@AspectJ样式的方面支持验证配置（切入点表达式的解析）和可视化横切参考。
  • 支持Spring JavaConfig M2。这用作测试Spring IDE核心扩展点的沙箱。在这里和这里阅读有关此内容的更多信息。
  • 可用性和UI改进：一个新的Spring Explorer替换了Beans View ，为Eclipse的Project Explorer贡献了内容，一个Spring Working Set类型减少了项目和Spring Explorer的混乱， 重构了参与者以重命名和移动Java包和类的重构以及Bean名称，New Project和Spring Bean配置文件向导。

Spring IDE 2.0与即将推出的Eclipse 3.3（又名Eclipse Europa）兼容。

在Spring IDE Blog上了解更多信息。
