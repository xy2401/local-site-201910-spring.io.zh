我们为什么筹集1000万美元？

工程
罗德·约翰逊
2007年6月6日

您可能已经听说了Spring背后的公司Interface21最近筹集了1000万美元的公告。鉴于我们已经存在了将近3年，并且迄今为止取得了很多成就，所以您可能想知道为什么。

我们为什么要筹集资金，我们将如何处理？

在过去的两年中，我们建立了一支优秀的团队。Juergen Hoeller，Adrian Colyer，Keith Donald，Colin Sampaleanu，Mark Pollack，Ben Alex，Rob Harrop…开始键入该列表很令人恐惧，因为我知道我不能包括该公司所有有才华的技术人员，而我不不想暗示任何功绩……这个团队很棒。

我们已经将Spring从1.0扩展到2.0到Spring 2.1 M2。我们已经使Spring Web Flow成为成熟，功能强大的产品。我们已经创建了Spring Web Services，Pitchfork，Spring OSGi和其他Spring Portfolio项目。我们确保Spring仍然是可用的最好的Java软件之一 。我们正在举办Spring Experience和SpringOne会议。

我们已经在许多国家/地区培训了数千名开发人员。我们已经在北美，英国，欧洲和澳大利亚开设了办事处。我们已经赢得了数百名客户，其中包括几乎全球15家最大的银行。

我们做得很好。但是我们创建的软件做得更好。Spring Framework 已变得无处不在：一位领先的分析家认为，大多数企业Java受访者都在使用它，并被越来越多的企业供应商所接受。Spring已成为企业Java的核心。而且，这一中心重要性值得重要的中间件供应商支持并帮助其发挥最大潜力。

我们需要投资更多并成长为该中间件供应商。

从外部看，我们在产品开发方面的进展令人印象深刻。（好吧，这是令人印象深刻。）但是我们可以做得更好。以Spring Web Services为例。该软件的才华横溢的创造者Arjen Poutsma在去年必须要在整个北欧范围内支付高昂的费用，才能把面包放在桌子上。（感谢Arjen，我们感谢您的辛勤工作，我们的客户爱您。）但这是有代价的。Spring Web Services的进度放慢了。我们有许多客户对它的承诺感到兴奋，但担心它正在失去动力。我敢肯定，在更广泛的社区中，许多人也是如此。Arjen必须牺牲个人时间来保持项目的进展。最近，我们已经看到Spring Web Services向1.0最终版快速发展。我们已经看到了许多用户要求的文档。我们已经看到了1.0RC1中解决的每个错误。之所以发生这种情况，是因为我们知道我们可以依靠资金，并且我们有能力给Arjen提供时间专注于发展。展望未来，我们希望能够向我们的开发人员付费，使其专注于为数十万用户开发代码，而不仅仅是为数百个客户提供服务。

另一个示例： Acegi Security for Spring 。这是目前可用的最佳企业Java安全解决方案，它是一个很棒的产品，但有两个保留：它相对难于使用，并且尚未利用Spring 2.0的配置增强功能；而且大多数人都不会说出它的名字。（它是“ a-See-Gee”，带有短的“ a”，但很快就没关系了。）资金使我们能够开展工作以解决这两个问题。我们已经将Vishal Puri（Interface21 Australia的高级工程师）投入到安全开发中，以协助Acegi Security的创建者Ben Alex（去年加入Interface21）。您很快就会看到一些Spring 2.0命名空间，这些命名空间使Acegi入门更加容易，并且配置更加优雅。最后，我们终于有时间将Acegi Security重命名为Spring Security，以便大家都知道它是Spring产品组合的组成部分，Interface21将支持它，因此您不必学习如何发音。

这两者都是双赢。为Spring社区带来直接收益（他们可以获得更好的软件）；我们的客户（为其业务需求获得更好解决方案的客户）；和我们（我们赚了更多的钱，因为我们的软件越好，使用的人就越多，我们拥有的潜在客户越多）。

那么，既然我们有了资金，将会发生什么变化？我们的价值观将保持不变。这是一家既有文化又有价值观的老牌公司。但是，您将在产品开发，销售和市场营销中看到明显更好的执行力。春季投资组合已经取得了很多成就，您还没有看到螺母。您将看到更多示例应用程序。更多文章解释了如何通过Spring获得成功；并更快地实现所需的功能。本月下旬（6月20日至22日）来到安特卫普的SpringOne ，您将看到我们的技术发展速度如何，并了解我们的发展方向。

哦，我们最终将有足够的销售人员来确保您知道我们的Spring 支持和培训如何减少您的成本并使您更加成功，并确保您的CIO意识到Spring为何可以为您省钱。但是，无论您是否在Interface21上花钱，您都会从Benchmark的钱中受益。
