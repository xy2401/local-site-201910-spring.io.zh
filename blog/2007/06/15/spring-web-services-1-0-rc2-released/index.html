Spring Web Services 1.0 RC2发布

发布
Arjen Poutsma
2007年6月15日

我们很高兴宣布Spring Web Services 1.0 RC2已发布。

下载 | 参考文档 | API文档

 
这是Spring-WS的第二个候选版本，Spring-WS是Spring社区的产品，致力于创建文档驱动的Web服务。此版本包含自RC1版本以来发现的错误的修复程序以及一些小的改进。此外，“航空公司”示例应用程序已得到增强，可以使用Java 5功能，包括新的@Endpoint编程模型，JPA支持，@ Transactional等。

下一个版本计划在几周内发布。如果未发现主要错误，则此版本将升级为1.0！

有关更多信息，请访问Spring Web Services网站。
