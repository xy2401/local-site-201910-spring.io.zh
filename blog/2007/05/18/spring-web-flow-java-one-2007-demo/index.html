Spring Web Flow Java One 2007演示

工程
基思·唐纳德
2007年5月18日

当Sun在会议的最后一天星期五在Spring Web Flow上安排我的JavaOne 2007会议时，我不确定会发生什么。我很荣幸今年再次被接纳，但我想知道在为期4天的会议的最后一天演讲时会看到什么。

我对事情的进展感到非常满意。当我在星期四的扬声器设置中签到时，有800人预先注册了我的星期五课程。在我开始讲话前十五分钟，房间已经达到了这个数目。最终，有1000名JavaOne与会者来到Moscone中心的307-310室，体验了肾上腺素驱动的Spring Web Flow试驾。

Spring Web Flow测试驱动器

在此博客中，我希望重新创建JavaOne Web Flow演示文稿中最激动人心的部分。请查看下面的屏幕，以查看对我个人而言，最令人振奋的部分是什么—从头开始构建搜索流程，以将其部署在Java Server Faces（JSF）环境中。




通过此搜索流程，您可以选择输入一些搜索条件，查看一些结果，执行新搜索以及浏览结果的详细信息（现有的WebFlowians会将其识别为熟悉的Phonebook示例应用程序的JSF版本）。

观看屏幕广播时，请注意Web Flow如何管理应用程序的所有导航和状态，而标准JSF组件则负责视图渲染和模型数据绑定。还要注意，即使我搞砸了一些东西，我也可以快速修复它并恢复工作，因为我不需要重启服务器或切换编辑器。
