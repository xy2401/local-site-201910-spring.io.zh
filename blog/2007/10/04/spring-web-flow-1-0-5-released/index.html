Spring Web Flow 1.0.5发布

发布
基思·唐纳德
2007年10月4日

亲爱的春季社区，

我们很高兴地宣布Spring Web Flow 1.0.5已经发布。这是一个错误修复和较小的改进版本，解决了针对1.0.4报告的所有问题。我们建议用户从以前的1.0.x版本升级到1.0.5。

 
下载 | 实用介绍 | 样品 参考手册 变更日志

Spring Web Flow是下一代Java Web应用程序控制器框架。该框架提供了一个功能强大的系统，用于在各种环境中一致地实现导航逻辑和管理应用程序状态。
