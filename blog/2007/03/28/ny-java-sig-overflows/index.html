NY Java SIG溢出

工程
内兰·乔克西（Neelan Choksi）
2007年3月28日

每隔一段时间我就会对Spring，Interface21和我们的员工的受欢迎程度感到非常惊奇。

昨晚就是这样的时刻之一。罗德在曼哈顿的纽约Java SIG上发表讲话。长期以来，纽约Java SIG协调员Frank Greco在大约两周前的星期日晚上发了一封电子邮件，宣布Java SIG。到周一早上9点左右，该活动的注册人数达到了300个（Google会议室超过了260个时，就从防火法规中惹了麻烦）。实际上，Java SIG在大约一个工作小时内就销售一空。我认为与朋友和客人一起的总注册人数为340。令人惊讶的是，一旦房间满了260人，人们就被拒之门外。对于他们来说，很抱歉他们没有看到Rod讲话，但是也许他们可以来费城参加Enterprise展会中的Emerging Technologies 。

Google非常荣幸地为NY Java SIG，美食和美酒提供了绝佳的场所。弗兰克（Frank）和纽约Java SIG吸引了众多观众，并度过了一个有趣的夜晚。
