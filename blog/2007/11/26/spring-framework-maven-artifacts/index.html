Spring Framework Maven工件

工程
本·黑尔
2007年11月26日

根据大众的需求 ，Spring Framework Maven工件现在正在上载到Spring Snapshot Maven存储库。你可以找到所有的春季组合Maven仓库的细节在我以前的帖子 ，但我会为转载这里的春天快照库的细节。

Spring Snapshot Maven存储库位于http://s3.amazonaws.com/maven.springframework.org/snapshot 。使用此存储库需要您将条目添加到 POM中的元素。它看起来应该像这样：


<repository>
    <id>spring-snapshot</id>
    <name>Spring Portfolio Snapshot Repository</name>
    <url>http://s3.amazonaws.com/maven.springframework.org/snapshot</url>
</repository>

该存储库中的工件不遵循预期的存储库行为，将定期删除。至少将保留给定工件的最后10个快照构建。如果从发行版中删除工件，其快照构建将立即删除。在发布里程碑或最终版本时，将删除工件的所有快照，并为下一个版本创建新的快照。
