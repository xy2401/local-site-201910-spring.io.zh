亲爱的春季社区，

我们很高兴宣布1.0.0。RC3发布了Spring Social ，包括1.0.0。RC3发行了Spring Social Facebook和Spring Social Twitter。Spring Social使您可以将Java应用程序连接到软件即服务（SaaS）提供程序，例如Facebook和Twitter。

    • 现在可以上传照片以及状态更新。
    • TimelineOperations.updateStatus（）现在为新发布的tweet返回一个Tweet对象。
    • 现在可以将状态发布为对现有状态的答复。
  • 示例应用程序集已更新，包括两个新示例：一个示例演示基于弹出窗口的连接流，另一个示例演示在Facebook Canvas应用程序中使用Spring Social。

适用于Spring开发人员的Micro Cloud Foundry

工程
乔什·朗（Josh Long）
2011年8月24日

今天，VMware团队发布了Micro Cloud Foundry ，这是流行的开源平台即服务的完整本地版本，使开发人员可以在Mac或PC上运行功能全面的云。使用Micro Cloud Foundry，开发人员可以在本地构建端到端的云应用程序，而无须配置中间件的麻烦，同时又可以保留部署位置的选择和扩展其应用程序的能力，而无需更改代码行。

Micro Cloud Foundry当然支持Spring和Java，但也为Scala，Node.js和Ruby提供了运行时环境，因此您可以释放内部的多语言程序员 ！Micro Cloud Foundry还提供了许多服务，例如MongoDB，MySQL和Redis，可以立即使用，而无需进行大量的安装和配置。借助内置的动态DNS支持，开发人员可以在碰巧工作的任何地方（无论是在家，办公室还是咖啡店）运行Micro Cloud Foundry，而无需进行任何重新配置。在Micro Cloud Foundry上创建和测试应用程序之后，您可以轻松部署应用程序，而无需更改到www.cloudfoundry.com或其他Cloud Foundry实例-它实现了跨多种云环境的真正应用程序可移植性。

  5. 印度Xebia的敏捷开发人员Tarun Sapra就何时使用Spring的黄金法则写了一篇不错的文章 singleton概念：仅对不必具有特定于客户状态的bean使用单例。尽管他对黄金法则提出的要求略有不同，但我认为区别很重要。Spring bean本身可以维护状态，但是它们必须保护该状态，以防止多个（通常是并发）客户端的突变。

    A lot of times, too, Spring lets you work as though you have client-specific state, but are in fact using a singleton. An example of this is in the way Spring supports injection of the JPA <CODE>EntityManager</CODE>, which is <EM>not</EM> thread-safe. Spring intercepts calls to the EntityManager proxy that's injected and then, in a thread-local, creates an EntityManager so that each request <em>effectively</em> has client-specific state, but they can program in terms of single-threaded access.

    就是说，这篇文章很不错。
