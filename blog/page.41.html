Spring Vault 2.1 GA发布

发布
马克·帕鲁奇
十月2，2018

我谨代表社区宣布Spring Vault 2.1的全面上市。Spring Vault 2.1需要JDK 8或更高版本，并且特别支持JDK 11作为下一个长期支持版本。Spring Vault 2.1最显着的变化是：

  • 扩展基于基础架构的身份验证以支持Google Cloud IAM和Azure托管服务身份。
  • 集成Vault的版本化键值后端。
  • 包装API支持。
  • Java 11兼容性。

此版本与的Vault版本兼容0.5.2取决于0.11.1 。
