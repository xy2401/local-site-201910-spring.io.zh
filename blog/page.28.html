旧版论坛将于2月28日关闭

新闻与活动
布赖恩·杜索
2019年2月6日

2014年，我们宣布淘汰旧版论坛 “ forum.spring.io”，以便在stackoverflow.com上提供更好的社区体验。作为该公告的一部分，我们将论坛设为只读模式，保留了各种Spring问题跟踪器中引用的论坛帖子。

在2019年2月28日，我们计划使论坛完全脱机。为准备此终止活动，我们已经检查了问题跟踪器，并确定了指向forum.spring.io的链接，其中包含与问题相关的补充信息。基于此分析，我们已将此类文本直接复制到问题跟踪器中，或者确保没有丢失任何基本信息。

Apache Geode / Pivotal GemFire 2.0.8的春季会议。发布可用！

工程
约翰·布鲁姆
2019年2月4日

我很高兴宣布Apache Geode和Pivotal GemFire （SSDG）的Spring Session 2.0.8.RELEASE 。

SSDG 基于 Spring Session 2.0.9.RELEASE ， Spring Data Kay-SR13和Spring Framework 5.0.12.RELEASE与Apache Geode一起1.2.1和关键GemFire 9.1.1 。此外，SSDG 2.0.8.RELEASE与Spring Boot兼容2.0.x基于应用程序。

此版本从以下方面引入了许多增强功能： 2.1.x发布分支以及一些小的修复。重点包括：

  1. 防止保存非脏会话（ 第12版 ）。这主要是针对Apache Geode / Pivotal GemFire错误GEODE-6032的保护 。

  2. 标准化SLF4J上的所有日志记录（ 第18版 ）。

  3. 修复了涉及从创建时从Session对象传递过来的Session属性的错误同步锁定的错误。

  4. 修复了涉及数据序列化的错误，该错误可正确识别用户何时使用自定义DataSerializer来对会话进行反序列化。

Spring Tools 4.1.1发布

发布
马丁·利珀特
一月25，2019

亲爱的春季社区，

我很高兴地宣布适用于Eclipse，Visual Studio Code和Atom的Spring Tools 4的4.1.1版本。

此版本的重点包括：

  • （Spring Boot）现在可以通过符号轻松浏览常规的Spring项目（不使用Spring Boot）
  • （Spring Boot）Bean和Bean接线的实时悬停现在可用于常规Spring项目（不使用Spring Boot）
  • （Spring Boot）对不赞成使用的属性的支持已添加到属性编辑器中，包括相应的快速修复
  • （Eclipse）引导仪表板现在显示环境信息，包括运行引导应用程序的所有属性及其值
  • （Eclipse）默认分发恢复了对本地Tomcat服务器的支持，并将战争打包的启动应用程序部署到这些本地服务器
  • 各种错误修复
