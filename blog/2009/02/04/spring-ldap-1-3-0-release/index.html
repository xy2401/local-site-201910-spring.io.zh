Spring LDAP 1.3.0。发布

发布
马蒂亚斯·阿瑟森（Mattias Arthursson）
2009年2月4日

亲爱的 Spring 社区，

我们很高兴宣布Spring LDAP 1.3.0的可用性。发布。这是一个主要版本，与1.2.1版本相比，包含了50多个修补程序。修改的重要性从内部修改和次要修复到重要的错误修复和重大的功能添加而变化。

最重要的是，我们现在提供了一种直接在LdapTemplate中执行简单LDAP身份验证的方法。其他有用的改进包括TLS连接支持，引荐处理改进以及针对基础Java JNDI API中的问题的几个重要修复。与往常一样，请参阅更改日志以获取此版本中包含的更改的完整列表。
