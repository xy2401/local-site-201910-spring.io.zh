下一次阿姆斯特丹Java聚会-3月13日

工程
阿列夫·阿伦森（Alef Arendsen）
2009年2月24日

由于您的旅行行程确实很长，自去年9月以来我们没有再见面。已经快半年了 ，所以又是时候了。

但是，在继续之前，我想指出一个我们已经建立的新Google网上论坛，以便向您发布最新信息。在过去的几年中，我收到了很多关于建立邮件列表的请求，因此从现在开始， Java Meetup Google Group就是关注新的Java Meetup日期的主要场所。在此处注册，以将日期直接发送到您的收件箱。

好的，接下来的聚会的细节：

  • 时间：2009年3月13日
  • 哪里：德贾伦-阿姆斯特丹新大街（地图，下）
  • 时间：下午5.30开始
  • 什么：喝酒和聊天，关于……好，一切

从中央火车站出发，步行到Jaren可能需要20分钟左右。乘坐电车16、24或25，也将带您到那里，至少，如果您在Muntplein电车站下车！
