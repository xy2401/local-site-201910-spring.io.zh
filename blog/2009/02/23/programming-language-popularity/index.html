编程语言的流行

工程
肖恩·康诺利
2009年2月23日

我13岁的儿子一直在Lua，TI Basic和Assembler，ActionScript，JavaScript中进行编程，如今他已经开始使用C ++ / Dark GDK。在过去的几年中，尽管我主要专注于Java，但我已经使用多种静态类型和动态类型的语言进行编程。

由于我的儿子才刚刚开始编程之旅，我自然想知道：

如今最流行的编程语言是什么？

我用tiobe.com和langpop.com回答了这个问题。这两个站点均提供编程语言受欢迎程度统计信息和排名，我很高兴看到Java，C和C ++在这两个站点中均排名前三。

我对这两个网站的满意之处在于，它们收集了范围广泛的搜索引擎和网站中的信息，以生成受欢迎程度得分。这两个站点都没有将重点放在声明“最佳”编程语言或已编写最多代码行的语言上。它们只是提供信息，正如TIOBE所说，“可用于检查您的编程技能是否仍在掌握之中，或就开始构建新软件系统时应采用哪种编程语言做出战略决策。”

2009年2月TIOBE编程索引
TIOBE收集来自Google，MSN，Yahoo！和YouTube的信息以计算评分，并且它们具有一个不错的网页， 用于定义TIOBE索引的组装方式 。以下是2009年2月最流行的20种编程语言的摘要。我特别喜欢它们与去年排名相比的情况，因此我们可以衡量该语言的年增长率。

在下面，您将看到Java在过去两年中一直排名第一。C ++和C＃呈上升趋势。Visual Basic，PHP和Perl却下降了。我还发现有趣的是，尽管Ruby和Python引起了很大的反响，但在过去一年中，两者的排名都没有上升。

2009年2月2日LangPop规范化比较
下面是LangPop的归一化比较表，该表将Yahoo，Craigslist，Amazon，Freshmeat，Google Code和Delicious的数据收集为29种不同的编程语言。单击图表可更清楚地看到它。

Java在C语言之后排在第二位。VisualBasic的排名远低于TIOBE的排名。Python，Perl和Ruby略高于TIOBE的结果。

[caption id =” attachment_1037” align =” aligncenter” width =” 688” caption =“ LangPop规范化比较于2009年2月2日”] [/字幕]

结论？
由于我在SpringSource工作，并且专注于企业Java（例如Spring ）和在Java上运行的动态语言（例如Groovy / Grails ），因此我很高兴看到Java不仅保持了其相关性，而且继续作为Java主导。顶级编程语言。

话虽如此，对诸如PHP，Python，Ruby和Groovy之类的动态语言的兴趣清楚地告诉我，开发人员渴望更高的生产率和更少的复杂性。我很高兴成为SpringSource的另一个原因，因为简化企业Java是我们关注和关注的领域。

鸣谢 ：感谢TIOBE和LangPop维护和共享他们网站上的信息。他们提供了宝贵的服务，我将继续访问他们的站点，以了解未来几个月和几年中编程语言格局的演变情况。
