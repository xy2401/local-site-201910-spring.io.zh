dm Server Roadmap blog entry, last April, we introduced two new artefact types: “plan” and “configuration file”.

Here is a short screencast demonstrating configuration files, in particular we show how to reference them from a plan.

First, a small web application picks up properties from a configuration properties file—deployed separately from the application. 配置属性截屏

工程
史蒂夫·鲍威尔
2009年9月11日

在去年4月Rob的dm Server Roadmap博客条目中 ，我们引入了两种新的伪像类型：“ 计划 ”和“ 配置文件 ”。

这是演示配置文件的简短截屏，特别是我们展示了如何从计划中引用它们。

首先，一个小型Web应用程序从配置属性文件中获取属性-与该应用程序分开部署。其次，将应用程序和属性放置在存储库中，并构建和部署计划 ，该计划将一步安装并启动Web应用程序及其属性。
