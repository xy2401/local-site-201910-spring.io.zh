SpringSource has released an update to SpringSource Tool Suite (STS), the best Eclipse-powered development environment for building Spring, Groovy and Grails powered enterprise applications. The latest version (2.2.1) is now available for download and includes:
  • Hot on the heels of the latest Spring Roo release, STS now supports Spring Roo 1.0.0.RC3 and enhances the Roo shell with hyperlinking to application files letting you open them for editing with a single click.
  • The STS Grails tools have been updated to include a GSP editor for easy page template editing. The Grails tools can be downloaded as an extension to STS.
  • Support for the latest milestones of SpringSource dm Server: 2.0.0.M6.
  • Improvements to the Cloud Foundry integration that Charles Lee discussed in today's blog post.

现在可用：SpringSource Tool Suite 2.2.1

发布
亚当·菲茨杰拉德
2009年11月11日

今天， SpringSource已经发布了SpringSource Tool Suite （STS）的更新， SpringSource Tool Suite是用于构建Spring，Groovy和Grails支持的企业应用程序的最佳Eclipse驱动开发环境。现在可以下载最新版本（2.2.1），其中包括：

  • 在最新的Spring Roo发行之后，STS现在支持Spring Roo 1.0.0。RC3并通过与应用程序文件的超链接增强了Roo shell，使您可以单击打开它们以进行编辑。
  • STS Grails工具已更新，包括用于简化页面模板编辑的GSP编辑器。可以下载Grails工具作为STS的扩展。
  • 支持SpringSource dm Server的最新里程碑：2.0.0。M6。
  • Charles Lee在今天的博客文章中讨论了Cloud Foundry集成的改进。
