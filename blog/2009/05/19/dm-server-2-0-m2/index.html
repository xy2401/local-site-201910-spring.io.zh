dm Server 2.0 M2

工程
安迪·威尔金森
2009年5月19日

dm服务器 2.0.0。M2已发布，现在可以下载 。在从2.0.0开始的两个sprint中。M1，无论是在新功能还是在缺陷修复方面，我们都朝着最终版本取得了重大进展。如果您有兴趣查看我们一直在努力的所有内容，请查看M2发行说明 。请在我们的博客， 论坛和JIRA上保留您的反馈意见。

新的和值得注意的

Spring 3支持

在这一里程碑中，我们已将dm Server升级为可在Spring 3上运行。当我们朝着dm Server 2.0的最终版本迈进时，随着新的里程碑和快照可用，我们将升级Spring 3的版本。

克隆改进

我们在对自动克隆的支持方面进行了一些重大改进。例如，您现在可以在同一dm服务器中并行运行Spring 2.5.6和基于Spring 3的应用程序。

看过的仓库

已添加对受监视存储库的支持。被监视的存储库被配置为监视文件系统上的目录，并且随着向目录中添加工件，它们将可用于dm Server的供应系统。同样，从目录中删除工件后，它们将不再对dm Server的资源调配系统可用。

这是受监视的存储库的一些示例配置：

"watched-repo" : {
    "type" : "watched",
    "watchDirectory" : "repository/watched",
    "watchInterval" : 5
}
