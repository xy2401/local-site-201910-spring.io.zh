dm服务器2.0.0。M6

工程
本·黑尔
2009年10月27日

dm服务器 2.0.0。M6现在可用，可以从此处下载。

在我们接近第一个2.0.0版本候选版本时，该版本在M5版本之后很快。这次，我们非常专注于修复突出的缺陷，并添加了一些新功能。请查看M6发行说明 ，以获取已修复的缺陷和已添加的功能的完整详细信息。请在我们的博客， 论坛和JIRA上保留您的反馈意见。

新的和值得注意的

将dm内核和dm服务器即服务运行

现在可以在Windows和UNIX计算机上将dm内核和dm服务器作为服务启动。要在Windows上安装该服务，需要编辑$ HOME / bin / service / conf / wrapper.conf文件以正确列出内核或服务器的路径。然后运行$ HOME / bin / service / windows.bat install 。还有其他命令要运行，而dm Server文档则针对其他命令。

UNIX用户甚至更容易，因为$ HOME / bin / service目录包含许多用于各种平台的init.d兼容脚本。

远程存储库缓存

不久前添加的远程存储库支持现在更加高效。两种缓存用于实现此性能改进。首先，从远程存储库请求索引时使用Etag比较。这样可以确保仅在索引更改时才下载索引。其次，当从远程存储库请求工件时，使用哈希比较。存储库索引格式已扩展为嵌入每个工件的哈希摘要。将这些哈希与工件的任何现有本地缓存副本进行比较，如果匹配，则避免下载。
