Git迁移

工程
本·黑尔
2009年7月15日

某些人可能已经注意到，在最近的帖子中，dm Server团队一直在引用Git存储库URL。该团队一直在迁移其所有源代码，现在我很高兴地宣布此迁移已完成。下面的列表包含dm Server团队管理的代码的Git URL。如果您在SVN中引用了等效的存储库，请克隆这些新存储库。

本公告仅涵盖由dm Server团队管理的代码，而不涉及整个Spring产品组合。其他项目将按照自己的进度进行转换，Subversion存储库将保持打开状态，直到他们这样做为止。
