dm Server 2.0 M3

工程
安迪·威尔金森
2009年7月15日

dm服务器 2.0.0。M3已发布，现在可以下载 。

在从2.0.0版开始的sprint中。在M3方面，无论是在新功能还是在缺陷修复方面，我们都朝着最终版本取得了重大进展。如果您有兴趣查看我们一直在努力的所有内容，请查看M3发行说明 。请在我们的博客， 论坛和JIRA上保留您的反馈意见。

新的和值得注意的

OSGi Web容器参考实现的集成

dm Server现在包含OSGi Web容器参考实现，并在其所有Web支持上使用并以此为基础。作为这项工作的一部分，我们还转而使用标准XML格式在dm Server中配置Tomcat。

管理控制台中的转储分析

新的转储检查器已添加到dm Server的管理控制台。转储检查器可用于检查dm Server的可维护性组件产生的诊断转储。

使用Git作为我们的版本控制系统

我们最近将dm Server的源代码移至Git，因为我们认为我们可以从Git的分布式特性及其对分支机构的出色支持中受益匪浅。如果您有兴趣访问dm Server的源代码，并有兴趣从源代码构建dm Server，则可以在下面找到将代码托管在Git中的说明。

文档更新

dm Server 用户指南和程序员指南中现在涵盖了许多新增功能。

使用ConfigurationAdmin

dm Server现在大量使用ConfigurationAdmin来管理其配置。这导致对dm Server的配置文件及其格式进行了更改。新的文件和格式在更新的用户指南中描述。

更新的应用程序开发指南

我们已经更新了使用dm Server创建企业Java应用程序的指南 ，以使其与2.0系列保持同步。

从源代码构建dm服务器

设定

必须先在计算机上进行以下设置，然后才能从源代码构建dm Server：


  • 已安装Java 6，并且已将JAVA_HOME环境变量配置为指向它

  • 安装了Ant 1.7.1或更高版本

  • ANT_OPTS环境变量，配置为提供至少512MB的最大堆，例如ANT_OPTS = Xmx512m

  • Git客户端

    • 完成此设置后，您就可以构建dm Server了。

    包装已发布的二进制文件

    生成打包的dm Server zip文件的最简单方法是从已经发布的现有二进制文件中生成一个：

    git clone git://git.springsource.org/dm-server/dm-server.git
cd dm-server
git checkout --track 2.0.0.M3 -b 2.0.0.M3
ant jar package

    该版本将下载构成2.0.0的所有捆绑软件。M3版本的dm Server并将其打包为安装二进制文件。您可以在target / artifacts目录中找到此二进制文件。

    完全从源代码构建

    要完全从源代码构建dm Server，您必须克隆许多存储库，以便在计算机上拥有整个代码库。然后，有必要在每个存储库中执行签出操作，以确保您使用的是2.0.0版。M3标签。每个存储库的相对位置都很重要，因此请从同一目录执行所有克隆：

    git clone git://git.springsource.org/dm-server/util.git
git clone git://git.springsource.org/dm-server/artifact-repository
git clone git://git.springsource.org/dm-server/osgi-extensions.git
git clone git://git.springsource.org/dm-server/kernel.git
git clone git://git.springsource.org/dm-server/web.git
git clone git://git.springsource.org/dm-server/servlet.git
git clone git://git.springsource.org/dm-server/hosted-repository
git clone git://git.springsource.org/dm-server/documentation.git
git clone git://git.springsource.org/dm-server/dm-server.git

    现在，我们必须更新每个克隆的工作副本以针对2.0.0进行工作。M3标签：

    cd artifact-repository
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../dm-server
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../documentation
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../hosted-repository
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../kernel
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../osgi-extensions
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../servlet
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../util
git checkout --track 2.0.0.M3 -b 2.0.0.M3
cd ../web
git checkout --track 2.0.0.M3 -b 2.0.0.M3

    现在已经设置了各种存储库，因此我们准备从源代码构建dm Server：

    cd ../dm-server/build-dm-server
ant src-build

    构建完成后，可以在目标/工件中找到zip文件形式的dm Server安装二进制文件。
