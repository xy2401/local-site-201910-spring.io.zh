在SpringOne Europe上见！

工程
罗德·约翰逊
2009年3月24日

这是会议季节。尽管日程安排很忙，但是一个会议仍然单独代表Spring内容。SpringOne Europe正在快速发展。今年，它将于4月27日至29日在阿姆斯特丹举行。会议本身不仅是阿姆斯特丹的绝佳目的地，而且在荷兰女王节假期之前结束了会议：如果您可以多花一天的时间，这将是一次有趣的文化体验。

SpringOne一直是一个很棒的会议，提供了深入的技术内容并全面涵盖了Spring主题。在过去的几个月中，我们一直特别忙碌，因此今年应该比以往任何时候都更好，这里有很多新技术可以谈论。

SpringOne会议为与Spring团队见面和倾听提供了独特的机会。所有关键开发人员都将在那里，包括Juergen Hoeller，Adrian Colyer和大多数Spring项目的负责人。无论您是对Spring集成，Spring Batch，Spring Web Flow，Spring Security，Spring 与Flex的新集成还是对Spring生态系统中的其他任何东西最感兴趣，您都将在正确的位置。我们还将欢迎包括埃森哲，Adobe和VMware在内的赞助商，我们正在与三家公司积极合作以开发有趣的技术。

至于了解Spring 的地方，SpringOne上是了解即将到来的地方 Spring 3.0版本 。该功能集包括REST支持 ，Spring MVC的进一步重大改进以及新的Spring Expression Language似乎总是引起开发人员的共鸣。到下个月，我们还将完成将核心Spring Java Configuration模型转移到Spring Framework项目中： 我认为这特别令人兴奋 。一些SpringOne会议将介绍Spring 3.0功能，展示重要的新功能和最佳实践。

随着我们将于4月20日发布GA的tc Server产品以及dm Server即将发布的一个主要新版本，您还将直接从项目团队那里听到有关我们新服务器产品的信息。

今年将特别关注开发人员的生产力。Grails的领导者Graeme Rocher将在Spring开发人员的Grails上发表演讲，而Groovy的领导者Guillaume Laforge则进行了两次会议，即Groovy 1.6和在Groovy中创建DSL。Spring IDE和SpringSource Tool Suite的负责人Christian Dupuis将讨论Spring的企业开发工具 。SpringSource Tool Suite是另一个即将发布的重要产品，它在2.0版中具有许多新功能。

我们还将发布重要的新项目公告，我们认为这将激发我们的开发人员社区。

希望您能加入我的阿姆斯特丹！
