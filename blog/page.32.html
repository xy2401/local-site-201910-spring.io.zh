Spring Cloud Open Service Broker 3.0.0。M3发布

发布
罗伊·克拉克森
十二月21，2018

我们很高兴地宣布发布Spring Cloud Open Service Broker 3.0.0。M3。此3.0版本将支持Spring WebFlux和反应性编程模型，以及Open Service Broker API v2.14中的新增和更改。此版本包括：

  • 升级到Spring Boot 2.1.1。发布（以前针对Spring Boot 2.0构建）
  • 向各种请求对象添加一个便捷属性，以访问Plan对于指定的ServiceDefinition

包括以下Spring Boot启动器：

<dependency>
  <groupId>org.springframework.cloud</groupId>
  <artifactId>spring-cloud-starter-open-service-broker</artifactId>
  <version>3.0.0.M3</version>
</dependency>
