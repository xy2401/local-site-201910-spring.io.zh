宣布Spring Cloud功能的总体可用性-2.0.0。发布

工程
奥列格·朱拉库斯基（Oleg Zhurakousky）
一月8，2019

我们很高兴地宣布Spring Cloud Function-2.0.0全面上市。发布。

Spring Cloud Function 2.0.0的各个模块。RELEASE可以在Spring Release存储库以及Maven Central存储库中使用 。

此版本包含以下内容：

  • Kotlin支持
  • Azure Functions的新编程模型
  • 在Web适配器中启动时自动（可选）导出Supplier
  • 功能Bean注册
  • 支持输入类型为Collection函数的透明类型转换
  • 在Spring Boot 2.1上构建
