Spring Data Redis 1.0.0发布

发布
Costin Leau
2011年12月14日

亲爱的春季社区，
我很高兴地宣布，Spring Data Redis 1.0项目的第一个GA版本现已发布！

资料下载 JavaDocs | 参考文档 | 变更日志

Spring Data Redis功能包括：

  • 连接包作为跨多个驱动程序的低层抽象
  • 对Redis Jedis ， JRedis和RJC驱动程序/连接器的配置支持
  • 异常转换到Spring的便携式数据访问异常层次结构Redis的驱动程序例外
  • 泛化的RedisTemplate，用于异常翻译和序列化支持
  • JDK，JSON和Spring Object / XML映射对象以及哈希序列化器
  • Redis的Spring 3.1缓存抽象实现
  • Pub-Sub支持（例如，消息驱动的POJO的MessageListenerContainer）
  • 排序和流水线功能
  • 专门支持SORT，SORT / GET模式和返回的批量值
  • 原子计数器和JDK 采集实现对Redis的顶部

Spring Data GemFire 1.1.0发布

发布
Costin Leau
2011年12月14日

亲爱的春季社区，

我们很高兴地宣布，Spring GemFire 1.1项目的GA版本现已发布！Spring GemFire项目旨在使使用GemFire作为分布式数据管理平台来构建基于Spring的高度可扩展应用程序变得更加容易。

资料下载 JavaDocs | 参考文档 | 变更日志

1.1版更新包括：


  • 使Maven命名与Spring Data约定保持一致（新ID为'org.springframework.data'/'spring-data-gemfire'）

  • 在缓存和客户端缓存名称空间中引入了PDX属性

  • 升级到Spring Framework 3.1 GA

  • 专用于不确定性的支持

  • 改进区域创建

  • 专门支持连续查询（适用于GemFire的消息驱动POJO）

  • 广泛的客户端缓存支持

  • 命名空间支持区域到期

  • 对即将推出的GemFire 6.6的本地支持

  • CacheServer支持

  • 用于Spring 3.1缓存抽象的GemFire实现

  • 支持带有可变参数的查询

Spring Framework 3.1正式发布

工程
于尔根·霍勒（Juergen Hoeller）
2011年12月13日

我很高兴地宣布，Spring Framework 3.1现已全面上市 ！此版本提供了一些关键功能，这些功能使Spring能够应对2012年及以后的挑战：

  • 环境抽象和关联的bean定义配置文件 ，以及用于占位符解析的可集中配置的属性源。

  • 基于Java的应用程序配置，该配置基于配置类上的@ Enable *annotation ，从而可以方便地进行容器配置：例如，使用@EnableTransactionManagement激活声明性事务处理。

  • 缓存抽象与我们的声明性缓存解决方案（@Cacheable等）位于顶部，着重于应用程序代码与缓存提供程序之间的便捷交互。

  • 基于Servlet 3.0的WebApplicationInitializer机制，用于引导没有web.xml的Spring Web应用程序！ 这是Spring Web配置故事中的关键部分，为基于XML的引导提供了丰富的替代方法。

  • 经过修订的MVC处理，具有Flash属性支持 ，新的@RequestPartannotation以及REST支持的进一步改进 。这个新的HandlerMapping / HandlerAdapter变体也可以高度扩展，以满足自定义MVC的需求。
