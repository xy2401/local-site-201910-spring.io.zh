Spring Data Lovelace M3发布

发布
克里斯多夫·斯特罗布尔
2018年5月17日

我很荣幸地代表Spring Data团队宣布Lovelace发布火车的第三个里程碑。

此版本随附所有商店模块的更新，其中包含新功能，改进和错误修复。显着的变化包括：

  • 修订的文档。
  • JDBC模块的基于注释的审核。（ DATAJDBC-204 ）
  • 同步和响应事务支持，以准备MongoDB 4.0版本。（ DATAMONGO-1920和DATAMONGO-1970 ）
  • 适用于MongoDB的反应式Map / Reduce抽象。 （DATAMONGO-1890 ）
  • Spring Data Gemfire和Apache Geode对Lucene索引支持的增强。（ SGF-739和DATAGEODE-101 ）
  • Spring Data Redis的反应式发布/订阅。（ DATAREDIS-612 ）
  • Spring Data Cassandra的Kotlin扩展。（ DATACASS-484 ）
  • Gemfire 9.5，Apache Geode 1.6，Lettuce 5.1（M1），MongoDB 3.8（beta2），MongoDB Reactive Streams 1.9（beta1）和Cassandra 3.5的驱动程序升级。

Spring Cloud Data Flow 1.5.0发布

发布
马克·波拉克
2018年5月16日

Spring Cloud Data Flow团队很高兴宣布1.5.0 GA发布。遵循本地服务器 ， Cloud Foundry和Kubernetes的入门指南 。

以下是重点内容：

  • 用户界面改进

  • Spring Boot，Spring Cloud Stream 2.0和Spring Cloud Task 2.0支持

  • 更新的应用程序入门

  • 指标改进

  • 组合任务的嵌套拆分

  • Kubernetes的改进

  • 更新的文件提取示例

用户界面改进

我们一直在继续改进仪表板的UI / UX。我们希望您会立即注意到整体较轻的设计。任务选项卡已被重写，以匹配其他选项卡的UX样式。新的分页器组件已添加到所有列表页面。每页可能会从20、30、50或100个项目的列表中切换。这进一步简化了批量操作工作流程。

更新的“ 流构建器”选项卡使部署流定义和更新已部署的流变得容易。您可以编辑应用程序和部署属性，以及更改流中的各个应用程序的版本并重新部署。Data Flow与Skipper的集成可处理升级过程，从而可以轻松回滚，以防升级未按计划进行。Stream Builder选项卡还包括许多优化，包括更好的表单验证和急切的错误报告。试试看！

还进行了大量的重构，以优化代码库并为将来的扩展和功能添加做准备。还添加了使用Selenium和SauceLabs进行的端到端测试。
