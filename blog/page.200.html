适用于Android 2.0.0的Spring。M1发布

发布
罗伊·克拉克森
2014年9月16日

我很高兴宣布Spring for Android 2.0.0。M1现在可以在Spring里程碑存储库中使用 。重点包括：

  • 通过使用泛型支持ParameterizedTypeReference
  • 通过以下方式支持OkHttpOkHttpRequestFactory
  • RestTemplate API与Spring Framework的奇偶校验
  • 错误修复和改进

最低版本

此版本将最低版本从Android 2.1（API级别7）提高到Android 2.2（API级别8）。Android 2.1不再支持Google Play商店应用，并且Android开发人员仪表板上也未跟踪该版本。此外，仅Android 2.3及更高版本支持OkHttp。

Spring Data 发布列车Evans上线

发布
奥利弗·德罗博姆
2014年9月9日

我谨代表整个Spring Data团队，宣布称为Evans的Spring Data发布火车迭代的一般可用性。该版本附带11个模块，可帮助您在关系和非关系数据存储之上构建复杂的数据访问层。

此迭代的主要主题是：

  • 将Spring基准升级到4.0
  • 支持top和first作为派生查询中的关键字
  • 支持MongoDB 2.6功能，例如文本搜索
  • SpEL对Spring Data JPA（Spring Security集成）的支持
  • Redis Sentinel支持
  • Spring Data REST的ALPS和摘录预测
  • 支持CDI扩展中的自定义实现
  • 改进的多商店配置
  • Asciidoctor用于参考文档
