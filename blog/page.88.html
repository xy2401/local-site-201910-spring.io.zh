发布

发布

2017年6月7日

我很荣幸代表Spring Cloud团队宣布Spring Cloud Pipelines的新里程碑发布- 1.0.0.M5 。

什么是新的？

用1.0.0.M5我们在Build阶段称为API Compatibility check 。现在，当作为HTTP /消息传递生产者进行向后不兼容的更改时，您的管道可能会中断。我们正在检查您为最近的生产部署定义的最新合同是什么，从这些合同中我们正在为当前应用程序生成测试。

您可以在Spring Cloud Pipelines Docs中阅读有关此功能的更多信息

另外，唯一的Ryan Baxter在Jenkins中增加了对GPG密钥的支持。

Spring 的这个星期-2017年6月6日

工程
乔什·朗（Josh Long）
2017年6月6日

欢迎来到 Spring 本周的另一期！本周，我将在马来西亚和新加坡，与金融服务领域的客户共度时光，并于6月8日在“ 新加坡 Spring 用户组”上发表演讲。如果您在附近，请出来或打个招呼 ！

  • Spring IO Platform负责人和Spring Boot忍者Andy Wilkinson刚刚宣布了Gradle Dependency Management插件1.0.3
  • Spring Web专家刚刚宣布了Spring Web Flow 2.4.5
  • Baeldung博客对Spring Boot和Kotlin有很好的了解
  • Ranga Karanam 用Spring Boot很好地构建了Web应用程序
  • 我喜欢这个博客，其中提供了在生产应用程序中使用编译时注释处理器Lombok的提示和技巧。显然，Lombok不是Spring项目，但是它很容易与Spring一起使用，并且可以在Spring Initializr上使用（作为依赖项）。
  • 我真的很喜欢Quora帖子中的一些回复， 何时选择Couchbase vs MongoDB vs Redis
  • Cloud Foundry正在将Consul从其内部实施中删除。该设计文档/验尸研究了在这种特殊情况下，为什么Cloud Founddry将Consul从堆栈中移出是合理的
  • Baeldung博客还更新了其Java 9 Reactive Streams内容，始终值得一读
  • 我喜欢马丁·福勒（Martin Fowler） 关于消息传递的不同用法的演讲
  • Pivotal自己的Matt Stine读了一篇不错的文章， 您的分解策略是什么？
  • 福布斯的鲍勃·埃文斯的有趣分析： 对不起，亚马逊，但微软是全球排名第一的云供应商
  • 我喜欢这个Dreamworks案例研究
  • Coda Hale（在许多方面创建了DropWizard Metrics库的人）将介绍Little's Law （一种对系统性能进行建模的方法）和他的新库usl4j进行了很好的结合 。
  • Pivotal自己的Joshua McKenty很好地论述了如何使云产品组合多样化 。
