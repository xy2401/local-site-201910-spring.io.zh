Spring for Apache Hadoop项目生命周期终止公告

新闻与活动
托马斯·里斯伯格
2018年4月5日

尊敬的Spring社区，

Spring团队特此宣布，自2019年4月5日起，十二个月的Spring for Apache Hadoop项目将达到生命周期终止状态。到那时为止，我们将根据需要偶尔发布2.5.x维护版本，然后将项目移至阁楼。当前用于Apache Hadoop 2.5.0的Spring版本是使用Apache Hadoop 2.7.3版构建的，并且应该与最受欢迎的Hadoop发行版的最新版本兼容。

Spring Cloud Stream App Starter-HDFS Sink应用程序依赖于Spring for Apache Hadoop项目。展望未来，我们将所需的功能直接移植到HDFS App Starter项目中。Spring Cloud Data Flow将继续将HDFS接收器应用程序编排为一致的流传输管道。
