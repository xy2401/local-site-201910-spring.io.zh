Spring LDAP 2.0.0。M1发布

发布
罗伯·温奇
2013年11月1日

我很高兴地宣布Spring LDAP 2.0.0。现在可以从SpringSource存储库（位于http://repo.springsource.org）获得M1。请参阅此处 ，以获取有关通过Maven解决这些工件的快速教程。

什么是新的？

该版本增加了许多新功能 ！以下是一些要点：

  • Spring LDAP现在包含Spring Data Repository和QueryDSL支持。
  • 流利的LDAP查询支持已添加。
  • 现在提供了一个自定义XML名称空间，以简化Spring LDAP的配置。
  • Spring LDAP核心已使用Java 5功能（例如泛型和varargs）进行了更新。
  • ODM（对象目录映射）功能已移至核心，LdapOperations / LdapTemplate中有一些新方法，它们使用这种自动转换到ODM注释的类/从ODM注释的类进行自动转换。

Spring Framework 4.0 RC1可用

发布
于尔根·霍勒（Juergen Hoeller）
2013年11月1日

尊敬的Spring社区，

我很高兴地宣布我们的第一个Spring Framework 4.0候选发布版的可用性。自M3以来，此版本通过一些新功能完善了我们的4.0功能集：

  • 与最新的OpenJDK 8 Developer Preview版本完全兼容
  • 对基于Groovy的bean定义的一流支持（如Grails所知）
  • 根据通用类型（例如MyRepository ）自动装配
  • 使用Objenesis创建CGLIB代理实例（允许构造函数注入）
  • 引入ScriptEvaluator机制（包括对JSR-223的支持）
  • Spring MVC中的时区分辨率（连接到JSR-310和Joda-Time）
  • Spring MVC的@ControllerAdvice的更细粒度的配置选项
  • MvcUriComponentsBuilder的引入（受Spring HATEOAS的启发）
  • 最后但并非最不重要的一点是，在Spring的新WebSocket支持中进行了许多微调

网络研讨会重播：Hadoop可以做什么：使用Hadoop和Spring构建大数据应用程序

新闻与活动
克洛伊·杰克逊（Chloe Jackson）
2013年10月29日

主讲人：耐克·沙（Nikesh Shah），马克·费舍尔（Mark Fisher），詹姆斯·威廉姆斯（James Williams）

在Pivotal，我们将两个开源项目放在一起-Hadoop和非常快速且广泛使用的Spring Java应用程序开发框架来帮助大数据应用程序。尽管Hadoop已被证明是存储和处理数据的事实基础，但现实世界中的场景还需要更多。数百万开发人员已经在使用Spring来创建高性能，易于测试，可重用的代码，而没有任何锁定。将Spring添加到Hadoop方程中，我们消除了构建大数据应用程序的复杂性。在本月的网络研讨会中，我们将探讨Spring XD（Xtreme Data）如何满足大数据应用程序的需求，例如：从各种输入源向HDFS提取高吞吐量分布式数据。摄取时进行实时分析，例如收集指标和计数值。通过批处理作业进行Hadoop工作流管理。高吞吐量数据导出，例如从HDFS导出到RDBMS或NoSQL数据库。对于任何想了解Hadoop驱动的应用程序的设计，配置和部署的人来说，这都是一个不能错过的演讲。
