用于Apache Hadoop 2.3.0 GA的Spring发布

发布
托马斯·里斯伯格
2015年12月22日

我们很高兴宣布Apache Hadoop 2.3.0 GA的春季发布。

Spring for Apache Hadoop 2.3增加了以下新功能和改进：

  • 添加了对Hive 1.x的支持，并更新了HiveTemplate以与HiveServer2一起使用
  • 适用于Apache Spark的新批处理Tasklet，有关用法示例 ，请参阅文档
  • 向StoreObjectSupport添加了FlushTimeoutTrigger以允许在写入过程中刷新到磁盘
  • 用新的“ spring-statemachine”项目替换内部状态机的实现
  • 在SpringHadoopProperties中添加了jobHistoryAddress以进行启动配置
  • 更新构建以使用Spring Framework 4.2.4，Spring Batch 3.0.6。发布，Spring Boot 1.3.1。发布和Spring Integration 4.2.4。发布。
