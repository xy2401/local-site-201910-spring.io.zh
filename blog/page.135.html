发布
菲尔·韦伯
2016年4月13日

Spring Boot v1.4.0。M2现在可以从Spring里程碑存储库中获得 。这绝对是一个巨大的版本，关闭了180多个问题和请求请求 ！感谢大家的贡献。

新版本的重点包括：

  • 从图像文件（gif，jpg或png）生成的ASCII Art标语。
  • 容易JsonSerializer和JsonDeserializer登记@JsonComponent 。
  • Couchbase支持。
  • Neo4J支持
  • Narayana事务管理器支持。
  • 升级到Spring Framework 4.3，Hibernate 5.1，Jackson 2.7，Solr 5.5，Spring Data Hopper，Spring Session 1.2和Hazelcast 3.6。
  • 全面的测试支持全面改革，包括新的统一@SpringBootTestannotation， @MockBean和@SpyBean支持，JSON AssertJ支持以及测试的自动配置。

适用于Apache Kafka 1.0 Milestone 2的Spring

发布
阿尔特姆·比兰（Artem Bilan）
2016年4月11日

我很高兴地宣布， 1.0版的spring-kafka （Apache Kafka的Spring）第二里程碑现已可用。

工件org.springframework.kafka:spring-kafka:1.0.0.M2和org.springframework.kafka:spring-kafka-test:1.0.0.M2在Milestone存储库中可用。

自第一个里程碑以来的更改包括：

  • KafkaTemplate.send()方法现在返回ListenableFuture>
    异步跟踪发布到Kafka主题。
  • MessagingMessageListenerAdapter （用于@KafkaListener方法）现在可以从方法参数推断payload转换的类型。
  • 一个StringJsonMessageConverter ，它可以使用类型推断使用Jackson ObjectMapper从JSON内容构建目标对象。
  • 已从Listener添加RECEIVED_标头，以避免在发送时（例如，使用Spring Integration Adapters）与具有相同messageKey的同一topic发生冲突（重新路由）。
  • 更多测试工具。
