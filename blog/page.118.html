立即准备您的Java 8 Reactive应用程序，Reactor 3.0 GA即将发布！

发布
斯蒂芬·马尔蒂尼（Stephane Maldini）
九月27，2016

经过一年多的工作 ， 多个里程碑并根据大量反馈进行了微调，我很高兴宣布Reactor 3正式上市。您会发现Reactor Core 3.0.2。在Maven Central上发布。

什么是Reactor 3？

Reactor 3为基于Java 8的应用程序提供了强大而高效的反应式编程模型。该模型以Reactor 2和RxJava 1的经验为基础，并引入了一种流畅的方式来编写异步可用于背压的事件处理。Spring Framework 5使用Reactor 3来构建并最终公开完整的反应式故事 。

适用于Apache Mesos 1.0 GA的Spring Cloud Data Flow发布

发布
托马斯·里斯伯格
九月26，2016

我很高兴代表团队宣布Apache Mesos 1.0的Spring Cloud Data Flow的常规可用性。

适用于Apache Mesos的Spring Cloud Data Flow提供了对在Apache Mesos上协调长时间运行（流）和短期（任务/批处理）数据微服务的支持。我们使用Marathon上的“应用程序组”启动流应用程序，并将任务作为Chronos一次性任务启动。该版本包含用于在Marathon上部署Spring Cloud Data Flow服务器的模板JSON脚本。我们还包括用于在Marathon上部署Redis，MySQL和Rabbit MQ的示例脚本，以用于测试完整的部署。除了在Apache Mesos集群上运行之外，我们还支持在Mesosphere DC / OS集群上运行。
