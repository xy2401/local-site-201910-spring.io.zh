Spring Python 1.2.0。RC1发布了！

工程
格雷格·特恩奎斯特
2010年11月3日

经过数月的工作， Spring Python的第一个1.2版本已可用 。

该项目已将其文档迁移到Sphinx，后者是用于记录Python语言本身的工具。您可以访问项目站点并以HTML进行查看，也可以下载epub版本以在智能手机或平板电脑设备上查看。

此版本的Spring Python过渡到Python 2.6，放弃了对2.4和2.5的支持。这意味着团队正在加紧利用Python的许多新功能，这也为将来某个时候过渡到Python 3.x铺平了道路。

从一开始，Spring Python就已经与Pyro（Python远程对象）集成在一起，这是通过远程将Python应用程序无缝链接在一起的有效方法。随着Pyro最新版本4 API的发布，该版本的Spring Python通过提供新的Pyro4ServiceExporter和Pyro4ProxyFactory类增加了支持。但是不用担心；如果您仍在使用Pyro 3，则Spring Python 1.2仍支持它。

添加了一种新的远程处理方式：安全XML-RPC。 Python有一个内置的XML-RPC库，但是它的使用挑战性很大，而且肯定具有侵入性。Spring Python使无缝利用这一强大的库成为可能。Spring Python还可以使用Python的内置SSL库轻松地保护通道。

我们还将源代码管理迁移到git。您可以访问该网站以查看源代码 。

谢谢，
