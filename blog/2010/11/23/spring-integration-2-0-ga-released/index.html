Spring Integration 2.0 GA发布

工程
马克·费舍尔
2010年11月23日

我非常高兴地宣布Spring Integration 2.0 GA已发布！

下载

包含Spring Integration JAR，源JAR和文档的发行版zip可以在此处下载。

依赖配置

工件也应该在今天晚些时候的Maven中央存储库中可用。同时，您可以将springframework Maven存储库添加到POM。以下示例显示了对“ spring-integration-core”和springframework存储库条目的依赖。如果您打算使用我们的任何适配器，则可以用适当的适配器替换“ spring-integration-core”，例如“ spring-integration-twitter”（并且所有适配器都将“ core”作为传递对象引入应用程序中依赖项）：


<dependencies>
    <dependency>
        <groupId>org.springframework.integration</groupId>
        <artifactId>spring-integration-core</artifactId>
        <version>2.0.0.RELEASE</version>
    </dependency>
</dependencies>

<repositories>
    <repository>
        <id>repository.springframework.maven.release</id>
        <name>Spring Framework Maven Release Repository</name>
        <url>http://maven.springframework.org/release</url>
    </repository>
</repositories>

什么是新的？

要了解此版本中的新功能，请访问“新功能”。 我们的参考手册部分。从那里，您将找到涉及新功能的特定章节的链接。另外，您可以在我最近的有关第1版候选版本的博客中阅读更多有关这些相同功能的信息。

对于现有的1.0用户，这两个资源应作为一个很好的起点。我们还将在接下来的几周内提供“迁移指南”。请继续关注，但与此同时，JTeam的Roberto van der Linden撰写的此博客涵盖了大多数要点。

抢先看

我很难在没有任何代码示例的情况下编写博客条目，因此我想让您快速了解以下新功能之一：Twitter支持。实际上，我今天早上创建了这个“应用程序”，以监视Twittersphere在发布之日对Spring Integration所说的话。这是整个配置：


<twitter:search-inbound-channel-adapter query="spring integration" channel="logger">
    <poller fixed-rate="60000" max-messages-per-poll="50"/>
</twitter:search-inbound-channel-adapter>

<logging-channel-adapter id="logger" expression="payload.fromUser + ': ' + payload.text"/>

每分钟都会执行一次搜索，如果有任何新的Tweet，它们将被发送到记录器。唯一的代码是用于引导应用程序上下文的main（）方法：


public static void main(String[] args) {
    new ClassPathXmlApplicationContext("twitter/context.xml");
}

如果您不相信它真的那么简单；），可以在github上找到该项目。它也恰好与我们存储库中的Spring Integration示例之一几乎相同。要了解有关我们样本的更多信息，请务必阅读Oleg 最近的博客 。

大图

就在上周，我举办了一个网络研讨会，名为“带有Spring的消息驱动架构”。其中包括对Spring对事件的支持，JMS和AMQP的消息传递，生命周期管理，异步执行，任务计划和触发器的相当广泛的概述。

录音可以在这里找到 。如果您想在短短一小时内尽可能多地学习有关Spring Integration及其在整个Spring平台中的适用性，请检查一下。它还具有几个有趣的示例：“贷款经纪人”（受Enterprise Integration Patterns启发）和Oleg和我在SpringOne2Gx上介绍的广受欢迎的REST / SOAP / Twitter / Email示例。
