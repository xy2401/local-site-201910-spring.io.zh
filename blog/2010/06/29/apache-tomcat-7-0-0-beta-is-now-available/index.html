  • protection against session fixation attacks

  • simplified embedding

  • alias support

Apache Tomcat 7.0.0 Beta现在可用

发布
亚当·菲茨杰拉德
2010年6月29日

现在可以从Apache Software Foundation的Tomcat 7 下载页面获得第一个Apache Tomcat 7版本Tomcat 7.0.0 beta。

SpringSource的Tomcat专家Mark Thomas在TomcatExpert.com上描述了该发行版的详细信息 。除了实现Servlet 3.0，JSP 2.2和EL 2.2规范外，Tomcat 7还具有许多新功能：


  • 内存泄漏预防和检测

  • 防止会话固定攻击

  • 一个简单的过滤器，可为应用程序添加跨站点请求伪造保护

  • 简化的嵌入

  • 别名支持

  • Manager和Host Manager应用程序的更好的安全性

有兴趣使用Tomcat的开发人员还应该查看tc Server Developer Edition ，该版本包含Spring Insight性能分析控制台。tc服务器团队正在跟踪Tomcat beta进程，并将在Tomcat 7版本稳定后提供更新。
