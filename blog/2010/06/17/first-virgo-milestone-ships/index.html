首批处女座里程碑船

工程
格林·诺明顿
2010年6月17日


第一个里程碑（2.1.0。Eclipse Virgo的 M01）现在可以根据Eclipse Public License 下载 。它由一个称为Virgo Web Server的应用程序服务器和一个独立的内核组成。

此里程碑的目标是使dm Server 2.0.x用户能够相对轻松地迁移到该服务器并拥有同样稳定的环境。SpringSource正在为处女座提供商业支持，我们鼓励所有dm Server用户迁移到处女座。现在，与用户的主要沟通渠道是处女座论坛 。还有一个处女座开发者邮件列表和每周一次的处女座社区电话 。

这是Virgo项目的主要步骤，它说明dm Server代码已完成。现在，想要打包和分发Virgo的用户可以按照Eclipse Public License的条款进行打包。

可通过 《 用户指南》 ，《 程序员指南》和《入门指南》草案的形式在线获取文档。然后， 处女座Wiki中出现了越来越多的面向开发人员的信息，包括概念概述 ， 源代码目录和构建技巧 。

尽管基于Eclipse的dm Server应用程序开发工具将在适当时候提供给Eclipse，但在此期间，Virgo支持已添加到SpringSource Tool Suite中。可通过下载STS 2.3.2并从更新站点http://www.springsource.com/snapshot/e3.5安装最新的dm Server工具来获得此功能。

背景

一月份，Adrian Colyer 宣布了我们打算将SpringSource dm Server捐赠给Eclipse的打算。2月批准了项目建议书 ，并创建了处女座项目。从那时起，我们建立了源代码存储库，创建了Wiki，并对代码进行了调整，以准备在正式的Eclipse正式批准下发布。5月，dm Server获得了Eclipse Community Best EclipseRT应用程序奖，这特别合适，因为Virgo是EclipseRT的子项目。

今天的里程碑表明了我们对OSGi和dm Server项目的未来处女座的持续承诺。
