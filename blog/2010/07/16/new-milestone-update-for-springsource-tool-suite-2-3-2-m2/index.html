SpringSource工具套件的新里程碑更新：2.3.2。M2

发布
亚当·菲茨杰拉德
2010年7月16日

SpringSource刚刚发布了SpringSource Tool Suite的新里程碑版本：2.3.2。M2 。STS为构建Spring和Grails支持的企业应用程序提供了最佳的Eclipse支持的开发环境。以下是该发行版中的一些重点内容：

  • 与GWT和Google App Engine集成
  • Spring Integration的可视化编辑器
  • 支持Spring Roo 1.1.0。M2
  • 非锁定类加载器基础结构
  • 新的Grails Project Explorer，Perspective和Plugin Manager
  • 安装了新的Subclipse和Subversive扩展

这也是我们第一次提供基于Eclipse 3.5 和 3.6的捆绑软件。如果您还没有准备好尝试最新的里程碑版本 ，则可以随时下载2.3.2。发布版本。

您还可以在Spring Development Tools支持论坛中获得社区支持。如果您想提供直接的技术反馈，请针对SpringSource Tool Suite项目打开JIRA问题。
