Spring MVC 3展示柜

工程
基思·唐纳德
2010年7月22日

自从去年的Spring 3大型发行版以来，我一直在从事许多应用程序开发项目，并提取各种框架功能的“展示”。这些“展示”不是参考应用程序或教程，它们更像是针对特定框架功能的验收测试。看到展示柜后，您应该对技术可以做什么有一个很好的了解。

我汇集的第一个展示是针对我们的Web框架Spring MVC 3。它包括一个示例项目，以及支持的幻灯片演示和截屏。深入研究之后，您应该对Spring MVC可以做什么有一个很好的了解，并了解它的易用性。

mvc展示柜

在此展示中，您将看到以下内容：


  • 最简单的@Controller

  • 映射请求

  • 获取请求数据

  • 产生回应

  • 讯息转换器

  • 渲染视图

  • 类型转换

  • 验证方式

  • 形式

  • 上传文件

  • 异常处理

我使用jQuery通过应用渐进式Ajax技术来驱动功能图。我强烈建议您使用Firebug或类似工具来深入了解客户端/服务器交互。我发现从客户端检查请求和响应详细信息，然后跳转到在服务器端执行的代码非常有教育意义。

通过在Github上克隆spring-mvc-showcase项目来获取代码。只需将项目作为Maven项目导入STS ，然后将该项目拖放到“ SpringSource tc Server Developer Edition”服务器上即可运行。

摘要

我希望您发现spring-mvc-showcase有用，并发现一些您以前可能不了解的“隐藏的宝石”。如果您想贡献自己的补充内容，也请告诉我。 spring-mvc-showcase今天覆盖了很多领域，但是总有机会覆盖更多内容！
