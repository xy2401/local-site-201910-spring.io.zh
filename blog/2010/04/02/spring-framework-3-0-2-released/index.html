Spring Framework 3.0.2发布

工程
于尔根·霍勒（Juergen Hoeller）
2010年4月2日

我很高兴地宣布，Spring Framework 3.0.2现在已经可用，其中包括针对用户报告的问题的100多个修复程序。从我们的下载页面获取它。

Spring 3.0.2赶上了最近的第三方发行版，例如Hibernate 3.5 final，OpenJPA 2.0 beta 3，Hessian 4.0.3和JasperReports 3.7 。另外，此版本引入了对Web支持的改进（例如新的HttpEntity类），并修复了一些回归问题。

我们建议从所有以前的Spring 3.0发行版升级到Spring 3.0.2-供开发和生产使用。 如果您当前正在使用Spring 2.5，那么3.0.2版本是现在建议进入Spring 3世界的入口点。请享用！

PS：

由于存在关于受支持的第三方版本的若干问题，让我快速总结一下我们的一般方法：Spring附带了对特定库版本（例如Hibernate 3.3.1）的构建依赖关系，但同时在运行时支持一系列版本（例如从3.2版到3.5版休眠）。Spring构建所针对的特定库版本通常不是 “推荐”版本，而只是我们构建所依据的最合适的库版本。

因此，就选择在应用程序中与Spring一起使用的库版本而言，简而言之，可以随意选择自己喜欢的版本-最近或使用了几年。Spring作为框架不会不必要地限制您的选择。我们甚至正在积极尝试支持一系列流行版本。只要我们需要针对特定功能的最低版本的第三方库，我们的组件（例如LocalSessionFactoryBean）通常会在其javadoc中告诉您。
