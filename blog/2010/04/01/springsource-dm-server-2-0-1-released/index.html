SpringSource dm Server 2.0.1发布

工程
格林·诺明顿
2010年4月1日

SpringSource dm Server 2.0.1今天发布 。

该版本修复了一些小错误和一个安全问题。管理控制台的dojo版本已从1.3.2升级到1.3.3，以防止进行开放式重定向攻击 。

该版本还包括OSGi Web容器的最终参考实现。OSGi联盟最近发布了OSGi Web容器规范，作为OSGi 4.2企业规范的一部分 。

没有计划进一步发布dm Server，因为该项目正以Virgo的形式捐赠给Eclipse Foundation。我们的目标是在适当的时候发布Virgo的基准版本，其功能将等效于dm Server 2.0。SpringSource将为处女座提供商业支持。
