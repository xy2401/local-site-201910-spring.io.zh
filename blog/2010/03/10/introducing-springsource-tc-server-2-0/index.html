推出SpringSource TC Server 2.0

工程
布鲁斯·斯奈德
2010年3月10日

在不久的将来，SpringSource tc Server 2.0™将会发布，其中包括一个新的Spring Edition，这代表了该产品的一些重大变化以及Web应用程序开发的一大进步。这也延续了SpringSource的承诺，即tc Server是Apache Tomcat的100％兼容的即插即用替代品，它提供了构建和运行Spring应用程序的最佳场所，非常适合现代虚拟环境。

基于众所周知的Tomcat服务器，tc Server添加了高级诊断，操作管理和关键任务支持功能。与传统的Java EE应用程序服务器相比，这些功能大大降低了成本和复杂性，同时提高了生产率。SpringSource tc Server现在提供三种不同的版本：Spring Edition，Standard Edition和Developer Edition。这三个版本的tc Server确保开发团队可以轻松访问，并为基于Spring和非基于Spring的应用程序提供了市场领先的企业级生产能力。

春季版


关于tc Server的大新闻是我们在tc Server Spring Edition中提供了新发行版 。那里的大多数Tomcat应用程序继续使用Spring框架，对于tc Server的用户来说更是如此。对于在生产环境中运行的Spring应用程序，SpringSource tc Server Spring Edition是最佳选择。Spring Edition的优点是包含了可通过Hyperic HQ提供报告和管理的检测化Spring jar。

  • 深入了解Spring应用程序的运行状况和性能

  • Spring应用程序的性能和SLA管理提供了确保应用程序运行所需的洞察力

  • 自动监视应用程序，数据访问和自定义组件

  • 监视Spring内部的性能并响应不断变化的执行条件执行控制操作

  • 监视应用程序服务器状态，运行状况和响应时间


春季版基于tc Server标准版中包含的新功能。

标准版


SpringSource tc Server Standard Edition通过与SpringSource Hyperic HQ 4.2 Enterprise监视和管理系统集成而向前迈出了重要的一步，该系统包括两个重要的增强功能：

  • 丰富的警报定义，工作流程和灵活的控制措施，可帮助开发人员主动管理应用程序问题

  • 组可用性和事件仪表板使查看问题区域和问题的可能根本原因变得容易


tc Server Standard Edition还提供了从用于管理和监视分布式数据中心的同一仪表板管理和监视tc Server应用程序的功能。这为您提供了对数据中心的全面视图，并使您可以关联各种指标数据的视图。下面的屏幕快照描述了一组正在监视的tc Server实例：


此外，tc Server Standard Edition仅提供tc Server Runtime的下载，而无需额外的附件。这使我们能够提供运行时容器的最精简的程序包。

开发人员版


SpringSource tc Server Developer Edition作为tc Server Runtime的免费开发下载提供。这包括：

  • 100％与Apache Tomcat兼容-随处使用Tomcat

  • 以最新的生产就绪型Apache Tomcat版本为核心

  • 包括SpringSource对Tomcat项目的贡献的错误修复

  • 应用程序模板可让您轻松地从现有应用程序创建模板，然后在几分钟之内轻松快速地启动新的类似应用程序。

  • Spring Insight性能仪表板在开发Web应用程序时可广泛了解。下面提供了Spring Insight的屏幕截图，但是没有替代品可以使用您自己的webapp尝试真实的东西。下载开发人员版以试用一下，亲自体验一下。




SpringSource tc Server Developer Edition也捆绑在基于Eclipse的开发环境SpringSource Tool Suite （STS）中。STS提供了可视化工具，可以使用Spring，Spring Roo和Groovy / Grails快速构建现代应用程序。这使开发人员无需离开STS即可使用tc Server轻松部署和测试其应用程序。 我强烈建议您不要查看Jon的tc Server和Spring Insight截屏视频以及Stefan的Roo，STS和tc Server截屏视频，而不是试图将STS的图像压缩到此博客文章中。这两个截屏视频确实为您提供了这些产品实际运作的良好视角，并展示了它们的真正功能，因此请务必查看它们。

免费的开发人员版为开发人员提供了巨大的价值。
