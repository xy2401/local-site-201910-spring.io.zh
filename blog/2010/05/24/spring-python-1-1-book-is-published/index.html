Spring Python 1.1书出版

工程
格雷格·特恩奎斯特
2010年5月24日

Spring Python 1.1书籍出版了！Spring Python采纳了Spring Framework和Spring Security的概念，并将其应用于Python领域 。

在本书中，您将使用依赖注入 ， 面向切面的编程 ， 数据访问 ， 事务管理 ， 远程处理和安全性来编写代码。稍后在第一个案例研究中，您将构建一个银行Web应用程序，并了解如何将所有这些组件一起使用可以快速满足您的需求。在第二个案例研究中，您将在构建航班预订系统时将Python和Java集成在一起。

本书中充满了代码示例，屏幕截图和图表。它提出了一种务实的，毫无意义的方法，可以将Spring的强大概念与灵活的Python语言一起使用。对于Java开发人员来说，它希望加入一些动态的Python增强功能。它也适用于希望将其应用程序提升到新水平的Python开发人员。

Spring Python是第一个上线的Spring扩展。该项目始于2006年，此后一直在增长。
