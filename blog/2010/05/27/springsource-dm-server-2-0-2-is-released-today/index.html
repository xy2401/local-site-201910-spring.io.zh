SpringSource dm Server 2.0.2今天发布。

工程
克里斯托弗·弗罗斯特
2010年5月27日

该发行版修复了一些错误，可以从JIRA获得发行说明。可以从SpringSource.org的项目页面下载此版本。

  • 增加了内核启动硬超时限制，以允许dm Server在较慢的计算机上运行。
  • 记录了OSGi Web容器中的限制， 不支持 Tomcat 元素。
  • 修复ServiceScoper类以关闭所有输入流。
  • 增加了对允许File.list偶尔返回null的支持，这表现为Pickup目录有时似乎无缘无故地擦除自身。
  • 现在可以将@Configurable与ServerOsgiBundleXmlWebApplicationContext一起使用。

该项目正以Virgo的形式捐赠给Eclipse Foundation。我们的目标是在适当的时候发布Virgo的基准版本，其功能将等效于dm Server 2.0。有关更多信息，请参见处女座网站 。SpringSource将与dm Server一起为处女座提供商业支持。
