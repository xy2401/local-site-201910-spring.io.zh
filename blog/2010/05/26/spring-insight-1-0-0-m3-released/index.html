Spring Insight 1.0.0。M3发布

工程
乔恩·特拉维斯
2010年5月26日

Spring Insight团队自豪地宣布发布Spring Insight 1.0.0。M3 。此版本包含大量的新增强功能，可为您提供更高的生产率 ，并使您的应用程序更快地投入生产。Spring Insight是tc Server Developer Edition和SpringSource Tool Suite的一部分，但是您可以在以下位置获取新的开发人员版本 http://springsource.org/insight 。设置时间将花费您通常的10秒。


速度追踪器


最受关注的新功能是我们与Google Speed Tracer的集成，该功能已在上周的Google IO会议上宣布。这种创新的组合使开发人员可以全面了解Web请求中执行的所有工作-从CSS样式和视图渲染到JDBC查询和JSP渲染时间。与Google的工程师一起对这项技术进行一些尖端的使用，真是太激动了。您会喜欢上此功能的！这是一个无缝集成，可以轻松地使用，并且为您提供了前所未有的大量可见性。

简单的用户界面


我们还为应用程序添加了很多信息：吞吐量和错误率指标，运行状况区域，对Spring控制器的更深入了解，更深入的JDBC支持，应用程序运行状况等等。但是，这些新信息不应该让人不知所措，因为我们在简化方面也投入了大量时间，从而使我们的UI显而易见并且易于浏览。

开发套件


最后，我们发布了我们的洞察力开发套件 （IDK）。我们已经从许多用户那里听说，他们希望能够在Insight中自定义其应用程序，或者希望添加其他指标。Insight现在通过其@InsightOperation和@InsightEndPoint注释以及IDK和用于创建更深层插件的教程来提供此功能。IDK包括Insight随附的所有插件（Apache 2.0许可）以及IDK API的源代码–很好的示例和编写自己的应用程序的说明。

Spring Insight团队正在开发下一个版本，我们的培训正在紧锣密鼓地进行，我们迫不及待想听听您的反馈和建议。去抢释放 http://springsource.org/insight，并在社区论坛中给我们留言！
