春季GemFire 1.0.0。M1为Java和.NET发布

工程
Costin Leau
2010年8月3日

我很高兴宣布Spring家族的最新成员Spring GemFire 项目的第一个里程碑版本 。Spring GemFire（简称SGF）将Spring概念和编程模型引入了SpringSource的分布式数据管理平台GemFire。该版本适用于Java和.NET 。

1.0.0中的功能。M1包括：


  • GemFire基础结构的声明性依赖项注入样式配置（例如Cache ， Region ， Interest等）

  • 异常转换为Spring的可移植DataAccess异常层次结构

  • 模板和回调支持可轻松访问本机API

  • 交易管理支持

  • 弹簧支持的GemFire管理对象布线

  • 基于非反射的实例化器的自动生成


请注意，其中某些功能当前仅在Java版本中可用。


通过SGF，Spring用户在与GemFire交互时应该感到宾至如归，而熟悉GemFire的开发人员将看到Spring容器的好处和灵活性，其强大的AOP集成以及通用的服务抽象。但是请不要相信我-下载项目并试用示例应用程序。它是基于控制台的“外壳”，允许与数据网格进行临时交互。一个人可以启动和停止节点，并查看多个客户端之间透明共享的信息。

我们期待您的反馈！
