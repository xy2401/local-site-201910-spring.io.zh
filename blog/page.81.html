使用Helm轻松安装Kubernetes的Data Flow

工程
托马斯·里斯伯格
2017年8月31日

通过适用于Kubernetes的Spring Cloud Data Flow的新Helm图表，现在有了安装软件的简单得多的方法。

Helm是Kubernetes的软件包管理器，类似于apt，yum或homebrew。它非常容易安装 ，并且极大地简化了应用程序及其依赖关系在Kubernetes集群中的安装。在图表中定义了应用程序包的内容和配置。安装时，您可以覆盖任何默认配置值。除图表中定义的服务外，Helm还将安装所有必需的服务。对于Spring Cloud Data Flow ，您具有三个必需的服务：MySQL和Redis被用作Spring Cloud Data Flow状态的存储，RabbitMQ被用作管道的消息传递层。
