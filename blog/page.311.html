现在可用：SpringSource Tool Suite 2.2.1

发布
亚当·菲茨杰拉德
2009年11月11日

今天， SpringSource已经发布了SpringSource Tool Suite （STS）的更新， SpringSource Tool Suite是用于构建Spring，Groovy和Grails支持的企业应用程序的最佳Eclipse驱动开发环境。现在可以下载最新版本（2.2.1），其中包括：

  • 在最新的Spring Roo发行之后，STS现在支持Spring Roo 1.0.0。RC3并通过与应用程序文件的超链接增强了Roo shell，使您可以单击打开它们以进行编辑。
  • STS Grails工具已更新，包括用于简化页面模板编辑的GSP编辑器。可以下载Grails工具作为STS的扩展。
  • 支持SpringSource dm Server的最新里程碑：2.0.0。M6。
  • Charles Lee在今天的博客文章中讨论了Cloud Foundry集成的改进。

将Cloud Foundry集成到开发人员工具中

工程
查尔斯·李
2009年11月10日

感谢其他SpringSource项目团队的努力，我们很高兴将三个开发人员工具与Cloud Foundry集成在一起，即SpringSource Tool Suite，Grails和Roo。这样的结果是，您可以创建和开发Spring或Grails应用程序，并在云中运行它们，而不必离开自己喜欢的IDE或命令行界面。这些工具利用Cloud Foundry客户端Web服务API（即将发布），并利用云中的其他智能和SpringSource工件存储库，以将Java Web应用程序优化，高效地上传到Cloud Foundry。这意味着您仅需几分钟即可从“代码到云”。

推出tc Server开发人员版-带有Spring Insight

工程
乔恩·特拉维斯
2009年10月21日

我们很荣幸为Spring社区提供一个新工具，该工具可以帮助他们更快地将其应用程序投入生产： tc Server Developer Edition是您熟悉和喜爱的标准100％Tomcat，以及一个新的Spring Insight控制台。

Spring Insight可在逐个请求的基础上深入了解应用程序的实际活动。对于任何请求，您都可以查看它进行的所有JDBC查询，渲染所需的时间或任何主要Spring Bean的计时。

当您单独查看特定请求时，这种细粒度的信息非常有价值，但是在成千上万的群中很难找到特定请求。幸运的是，Spring Insight还具有10k英尺的视图，可以快速将您指向问题区域。
