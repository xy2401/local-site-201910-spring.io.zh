Spring Framework 5.2.0。M1现已上市

发布
布赖恩·克洛泽尔
2019年4月10日

我很高兴代表团队和所有贡献者宣布，Spring Framework 5.2的第一个里程碑已经发布，可以从我们的里程碑存储库中获得 。此版本关闭了140多个问题并提出了请求 。

这个第一个里程碑包含许多功能和修补程序，其中包括：

  • 从使用新的解析注释数据开始，对核心容器进行了许多改进MergedAnnotations API @Configuration类优化
  • 支持Kotlin协程
  • 中的新WebMvc.fn编程模型spring-webmvc模块提供了基于Servlet API的带注释的控制器的功能替代。现在spring-webmvc喜欢spring-webflux提供功能和基于注释的编程模型。
  • Spring MVC和Spring WebFlux中的性能改进可减少请求映射，媒体类型解析，CORS检查等方面的开销
  • RSocket支持，包括通过注释的响应处理 @MessageMapping方法和通过执行请求 RSocketRequester 。
  • 许多有趣的集成测试改进，尤其是当您在测试中处理应用程序和上下文事件时
  • 我们的参考文档焕然一新
