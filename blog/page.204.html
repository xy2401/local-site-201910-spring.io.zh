Spring XD 1.0.0。RC1发布

发布
马克·波拉克
2014年7月18日

Spring XD团队高兴地宣布，现在可以下载 Spring XD Release Candidate 1。您还可以使用homebrew在OSX上以及使用yum在RHEL / CentOs上安装Spring XD。

此版本的重点

  • 直接绑定 ：可以将部署配置为避免模块在同一容器中共处时通过消息总线发送数据。使用此选项可提高吞吐量并降低延迟，但不能应用于所有部署拓扑。
  • 流部署状态 ：流的状态是在部署的整个生命周期中计算的。例如，如果组成流的模块子集发生故障，则流的整体状态将从“已部署”变为“未完成”。解决故障后，流的状态将返回到已部署。
  • 改进的REST API ：URI结构已经变得更加统一以进行作业管理。
  • 支持Rabbit HA和SSL配置。现在，您可以使用镜像队列来提高故障发生时消息总线的可用性。Rabbit Source和Sink模块也支持这些选项 。
  • 符合Spring IO Platform 1.0.1的要求 ：用户对服务器贡献的模块或扩展可以依靠Spring IO Platform来选择正确版本的依赖库。
  • HTTP源改进：现在支持HTTPS 。
  • 支持对filejdbc作业进行分区。
  • 添加了MongoDB接收器 。
  • UI魅力。
  • 除其他五个版本外，还支持Hadoop 2.4.1 。
  • 减少分销空间。
