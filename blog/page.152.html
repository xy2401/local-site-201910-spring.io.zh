Spring XD 1.3 RC1发布

发布
马克·波拉克
2015年11月3日

我非常荣幸地代表Spring XD团队宣布，现在可以下载 Spring XD 1.3的第一个候选版本。您也可以使用brew和rpm进行安装。

Spring Security 3.2.9发布

发布
罗伯·温奇
2015年11月2日

我很高兴宣布Spring Security 3.2.9的发布。发布。此版本提供了错误修复和较小的增强。有关发行版的完整详细信息，请参阅“ 更改日志” 。

该版本的重点包括：

  • SEC-2190-在子项ApplicationContext中注册Spring Security时，修复了与JSP标签库的集成
  • SEC-2521-删除StandardPasswordEncoder中的同步，可以大大提高性能
  • SEC-3108-修复DigestAuthenticationFilter中的潜在竞争条件
  • SEC-3109 -DelegatingSecurityContextExecutor与Concurrent / ThreadPoolTaskScheduler一起使用
