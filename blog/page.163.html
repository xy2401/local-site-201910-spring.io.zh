最终的春季数据服务版本Evans SR3和Dijkstra SR6发布

发布
奥利弗·德罗博姆
2015年7月1日

我谨代表Spring Data团队宣布Spring Data发布火车Evans和Dijkstra的最终服务版本的可用性。埃文斯SR3 固定售出114张门票 ，迪克斯特拉SR6减少了56个错误 。

Spring Framework 4.1.7和3.2.14发布

发布
于尔根·霍勒（Juergen Hoeller）
2015年6月30日

尊敬的Spring社区，

我很高兴地宣布，Spring Framework 4.1.7和3.2.14维护版本现已发布。除了解决整个框架中的各种小问题外 ，这些版本还通过在Spring MVC端点中解析不受信任的XML输入时默认情况下禁用DTD处理来解决XML解析漏洞 。

Spring Integration Kafka 1.2可用，具有0.8.2支持和性能增强

发布
马里乌斯·波哥维奇（Marius Bogoevici）
2015年6月19日

我们很高兴地宣布发布Spring Integration Kafka 1.2。

此新版本提供了相对于先前版本的几个主要改进和升级：

  • 支持Apache Kafka 0.8.2并使用新的Producer API；
  • 使用Project Reactor的无阻塞环形缓冲区实现对内部并发调度进行大修。

对于每个更改，目标都是提高Spring Integration Kafka的生产者和消费者组件的性能。尽管我们没有适当的Spring Integration Kafka独立基准测试，但该发行版的增强功能有助于新发行的Spring XD 1.2中报告的性能指标 。
