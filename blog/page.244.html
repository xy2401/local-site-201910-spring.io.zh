Spring Security 3.2.0。M2发布

发布
罗伯·温奇
2013年7月1日

现在可以从SpringSource存储库获得针对Spring Security 3.2的第二个里程碑版本 http://repo.springsource.org 。请参阅此处 ，以获取有关通过Maven解决这些工件的快速教程。

在此，我要特别感谢通过提交错误，提取请求和反馈为该版本做出贡献的所有人。

此版本的重点包括：

  • Spring Security Java配置支持
  • SEC-2111解决了发生超时并且同一线程同时处理原始请求和超时的异步支持问题。
  • 添加了对通过XML名称空间配置“记住我”参数的支持。特别感谢Oliver Becker为该功能提交拉取请求！
  • SEC-2002添加了SessionFixationProtectionEvent。特别感谢Nick Williams提交了对此功能的拉动请求！
  • 嵌入式LDAP容器的改进

在接下来的一周中，请继续关注SpringSource博客 ，以获取有关Spring Security Java Configuration的更多信息。

工程

2013年6月12日

今天我们很高兴地宣布Spring XD的1.0 M1版本（下载） 。Spring XD是一个统一的，分布式的，可扩展的系统，用于数据摄取，实时分析，批处理和数据导出。该项目的目标是简化大数据应用程序的开发。
