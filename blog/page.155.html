SpringOne2GX 2015重播：Netflix上的Spring Cloud

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年10月19日

记录在SpringOne2GX 2015中。

演讲者：乔恩·施耐德（Jon Schneider），泰勒·威克塞尔（Taylor Wicksell）-Netflix

Web / JavaScript跟踪

幻灯片： http ： //www.slideshare.net/SpringCentral/spring-cloud-at-netflix

云原生跟踪
使用Spring Cloud Netflix，您可以快速使用现有的Spring Boot应用程序，并将其转换为完全支持云的服务（即使您在私有数据中心之外运行）。
本演讲将主要通过实时编码进行，将使用一个简单的Spring Boot应用程序，并通过越来越多的可用于生产的功能逐步扩展该应用程序。
1。服务发现–我们如何扩展Spring Cloud以与Eureka更无缝地集成。
2。实时指标– Spectator / Atlas集成，包括我们如何将Spring Boot Actuator指标折叠到Spectator中。我们将向您展示如何使用Atlas堆栈语言生成实时指标图形。
3。服务间通信/负载平衡-与传统的服务器端负载平衡相比，何时应选择客户端负载平衡（Ribbon）？我们将说明Netflix OSS如何既可以在Ribbon中包含客户端负载平衡器，又可以在Zuul中包含软件路由器，以及何时使用它们。
4。管理故障-提供Hystrix / Spring与Turbine集成的演示。
5，分析-Spring Cloud Netflix如何使Spring XD成为具有实时操作洞察力的功能更强大的实时分析平台。

Spring Statemachine 1.0.0发布

发布
珍妮·瓦尔基亚拉赫蒂
十月13，2015

我们很高兴地宣布发布Spring Statemachine 1.0.0 。首先，我要感谢所有为实现这一目标而做出任何贡献的人。可以从Maven Central或从Spring Repository获得工件。

我们在第一个版本中实际得到的是：

  • 易于使用的单层状态机，用于简单的用例。
  • 分层状态机结构可简化复杂的状态配置。
  • 状态机区域提供甚至更复杂的状态配置。
  • 触发器，过渡，防护和操作的使用。
  • 键入安全配置适配器。
  • 用于在Spring Application上下文之外使用的易于实例化的Builder模式
  • 常用案例食谱
  • 基于Zookeeper的分布式状态机
  • 状态机事件监听器。
  • Spring IOC集成，可将bean与状态机关联。
