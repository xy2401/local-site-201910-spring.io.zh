Spring Framework 5.2正式发布

发布
于尔根·霍勒（Juergen Hoeller）
九月30，2019

尊敬的Spring社区，

我很荣幸代表Spring Framework团队和许多近期的贡献者宣布，通常可以从repo.spring.io以及Maven Central获得Spring Framework 5.2！查看我们更新的文档…

  • Spring Framework 5.2 的新增功能
  • 升级到Spring Framework 5.2
  • 我们修订的参考文档以及Java和Kotlin的并行示例。

Spring Framework 5.2需要JDK 8或更高版本，并且特别支持JDK 11作为当前的长期支持分支以及JDK 13作为最新的OpenJDK版本。它具有许多性能改进（影响启动时间以及峰值性能），并朝着GraalVM本机映像支持采取了进一步措施（仍处于试验阶段）。

春季会议玉米M4发布

发布
罗伯·温奇
九月17，2019

该帖子由VedranPavić撰写

我很高兴代表社区宣布Spring Session的发布Corn-M4 。此版本由Spring Boot挑选2.2.0.M6 。

Spring SessionCorn-M4

的Corn-M4发布基于：

  • Spring Session核心模块2.2.0.M4

  • 春季会议数据Geode2.2.0.M4

  • 春季会议数据MongoDB2.2.0.RC2

春季会议的一些亮点2.2.0.M4是：

  • 支持使用新的自定义会话存储库配置SessionRepositoryCustomizer /ReactiveSessionRepositoryCustomizer

  • 支持配置交易行为JdbcOperationsSessionRepository

  • 对Spring Security的支持AuthenticatedPrincipal在SpringSessionBackedSessionRegistry

这些版本的完整详细信息可以在changelog中找到。
