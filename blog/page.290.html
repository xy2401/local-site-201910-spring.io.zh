.NET 1.0.0的Spring CodeConfig现在可用

发布
斯博伦
2011年4月7日

亲爱的春季社区，

我们很高兴地宣布，针对.NET 1.0.0的Spring CodeConfig现在可用。

下载
| NuGet包 | 技术支持 文献资料
| 社区

       
Note also that Spring CodeConfig for .NET is also available as a NuGet Package.

此版本包含以下主要功能：


  • 基于代码的配置类似于Spring Java中的@Configuration支持

  • 声明式程序集扫描以检测配置类

  • 支持混合和匹配基于代码和基于XML的配置元数据

  • 扩展方法以在现有应用程序上下文上支持基于代码的配置

  • 从Code-First或XML-First方法引导Context配置

使用RabbitMQ路由拓扑以提高性能和可扩展性

工程
海伦娜·埃德森（Helena Edelson）
2011年4月1日

为高度可扩展的系统设计良好的路由拓扑就像映射图形。需要考虑很多事情，例如问题，环境的约束，消息传递实现的约束以及性能策略。我们经常遇到的问题是，在满足我们的需求时缺乏灵活性和表达能力。这是RabbitMQ脱颖而出的地方。

基本概念




AMQP 脚踩

  • 交换 ，其接收从生产者应用程序的消息，并可选地路由这些到消息队列的服务器内的服务器内的实体

  • 交换类型特定交换模型的算法和实现。与“交换实例”相反，后者是在服务器内接收和路由消息的实体

  • 消息队列持有消息并将其转发给使用者应用程序的命名实体

  • 绑定在消息队列和交换之间创建关系的实体

  • 路由密钥交换可用于确定如何路由特定消息的虚拟地址

Spring Data JPA 1.0.0。M2发布

发布
奥利弗·德罗博姆（Oliver Drotbohm）
2011年3月24日

亲爱的春季社区，

我们很高兴地宣布，Spring Data JPA项目的第二个里程碑版本现已发布！该版本包括：

  • 支持Querydsl谓词，从而支持类型安全的JPA查询
  • 在引导时验证@Query带注释的查询
  • 支持基于XML的实体映射
  • 支持查询方法中的Distinct，In和NotIn关键字
  • 将Hades扩展移植到Spring Data Commons核心
  • 各种错误修正。
