Spring IDE WebFlow支持预览版本1

发布
科林·桑帕林（Colin Sampaleanu）
2005年5月21日

克里斯蒂安·杜普依斯（Christian Dupuis）宣布了新的预览版
基于Eclipse的Spring-IDE，具有对Spring的图形编辑器支持
Web流。

这是克里斯蒂安在邮件列表中的公告：

-

亲爱的春季社区，



我们刚刚发布了Spring IDE WebFlow支持的第一个预览版。请使用此版本给我们反馈。



特征：




*用于String WebFlow Xml配置文件的全功能图形编辑器。

-支持最新的Spring WebFlow PR 3功能

-拖放编辑

-编辑期间的验证

-打印和导出（jpg和bmp）功能

-Spring WebFlow配置文件的自动布局，包括连接路由

-并行编辑同一配置文件

您最喜欢的Xml编辑器插件和WebFlow编辑器中

*与Spring IDE Beans View和Beans Graph集成

-Bean ConfigSet可以与WebFlow ConfigSet关联以进行验证

Spring IDE的开发人员可以提供预览版1
更新站点位于http://springide.org/updatesite_dev 。确保你有
安装了Eclipse 3.0以及Spring IDE 1.2.0和GEF 3.0.1。



可在以下位置获得文档

 http://springide.org/project/wiki/WebFlowEditor和

 http://springide.org/project/wiki/WebFlowEditorUsage 。



如果发现任何错误或有功能要求，请提交票证
在 http://springide.org/project/newticket指定版本
'webflow。PR1”。



谢谢你的努力



基督徒问候



由Disqus提供动力的评论
