Spring Framework 1.2发布

发布
科林·桑帕林（Colin Sampaleanu）
2005年5月13日

在13日这个幸运的星期五，我们很高兴地宣布Spring 1.2 final刚刚发布。

（感谢Juergen所做的打包工作以及他通常的编码技巧）。

自1.2 RC2以来的主要新功能是：



  • TopLink支持（同时适用于TopLink 9.0.4和10.1.3）

  • JDO 2.0支持（针对JPOX 1.1 beta 3测试）

  • 支持Hibernate 3.0.3（积极发布Connections）


Spring 1.2的总体主要新功能包括：



  • 细粒度的分发jar文件

  • 简化的XML bean定义

  • JMX支持

  • JDK 1.5交易注释

  • 支持WebLogic JTA扩展

  • JDBC RowSet支持

  • JCA CCI支持

  • JDO2支持

  • Hibernate3支持

  • TopLink支持


感谢Oracle提供原始的TopLink支持
原型，尤其是来自Oracle的Jim Clark-
帮助最终获得TopLink支持！



有关新功能，弃用和错误修复的详细信息，请参阅更改日志 。



仅供参考，我们计划于6月发布1.2.1版本，
不能使1.2最终版增强的功能。请参阅我们的JIRA路线图
细节。
