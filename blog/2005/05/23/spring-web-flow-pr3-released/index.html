Spring Web Flow PR3发布

发布
科林·桑帕林（Colin Sampaleanu）
2005年5月23日

Spring Web Flow PR3已发布。这是Keith的邮件列表公告：

-

亲爱的春季社区，

我很高兴地宣布发布Spring Web Flow Preview 3。这是一个重要的新版本，具有许多新功能：在功能和便利性方面。该版本被认为可稳定用于开发。

要下载，请访问： http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=148517

注意：PR2和PR3之间有一些不兼容的更改。有关列表和移植指南，请在此处访问更改日志： http://opensource.atlassian.com/confluence/spring/display/WEBFLOW/Change+Log

此版本中包含许多新的和值得注意的增强功能。这里是其中的一些：

查看状态增强

  • 现在可以为每个视图状态配置一个或多个设置操作。在进入状态之后但在将控制返回给客户端进行响应呈现之前，将执行此操作。这减少了（在许多情况下消除了）设置动作状态的需要，从而使流程定义更加简洁。
  • 现在，每个视图状态都可以使用新的ViewDescriptorCreator策略以完全动态，可插入的方式选择要渲染的视图和所有支持模型数据。
  • 现在支持强大的重定向表达式，并且完全支持从流RequestContext以动态方式解析重定向参数。

带注释的定义对象

  • 除了动作执行参数，系统中的每个核心定义对象（流，状态和转换）现在都可以使用任意属性元数据进行注释。此类属性可在Webflow xml定义中使用“ property”元素指定，并全面支持从字符串类型转换和方便的类型别名。

增强的流执行侦听器回调

  • 现在，响应执行流的生命周期的侦听器具有更多的钩子来插入自定义逻辑-当流开始，开始，暂停，恢复，进入新状态，进入新状态或结束时，通知会发生，并具有否决权建议的流量状态更改（例如，以促进状态前提条件检查）。

-

这些只是此版本的一些新功能。有关更改的完整列表，请参见 http://opensource.atlassian.com/confluence/spring/display/WEBFLOW/Change+Log

与往常一样，我们非常重视您的反馈。还有一两个PR版本可以发布，我们将进入1.0决赛！

干杯，
