Spring Web Flow 2nd预览版已发布

发布
科林·桑帕林（Colin Sampaleanu）
2005年4月11日

我们很高兴地宣布Spring Web Flow的第二个预览版
与Spring 1.2发布路径一起使用。这个
开发稳定版提供以下主要新功能
特征：

  • 可插拔的流执行存储策略，具有开箱即用的连续性支持。这意味着：
    • 现在，您可以选择将执行流的状态持久保存到任何后备数据存储区，包括HttpSession（默认值），数据库，序列化文件等。
    • 您可以选择使用基于连续性的存储策略，将Spring Web Flow变成一个由连续性驱动的系统，以提供适当的浏览器导航按钮行为（后退，下一步，刷新和新窗口）。您还可以在客户端或服务器端连续存储之间进行选择。还支持流执行状态的GZIP压缩。
  • 支持OGNL过渡条件，并带有TransitionCriteriaCreator策略，可根据需要轻松集成其他表达式语言或自定义过渡表达式。
  • Portlet支持，其中的样本（电话簿）说明了Portlet环境中的Spring Web Flow。
  • 支持“ validatorMethod”操作状态属性，指定要调用的验证方法，以与具有已配置的Validator实例的FormAction一起使用（以支持向导中的简单的逐项验证）
  • 支持JDK 1.3。
  • 样本（电话簿）说明了如何编写集成测试以验证流的预期行为。
  • 说明Struts集成的示例（出生日期）。
  • 一个全新的示例应用程序（出售商品），说明了使用延续的验证向导（提供正确的浏览器导航按钮使用！）和条件转换（使用OGNL表达式！）。

也有一些错误修复和其他常规
此版本中包含的改进/抛光。注意，对于
PR1用户：PR2与PR1有一些不兼容的更改。
有关详细信息，请参见WIKI上的更改日志。

          <br />

与往常一样，有了Spring的品质印记，您就可以
期望，我们提供：

  • javadocs和WIKI文档中的质量文档
  • 强大的测试覆盖率
  • 仔细注意命名，包装和易用性
  • 异常错误报告
  • 最终用户需求驱动的功能集

要下载预览版本：

            <br />

  1. 访问springframework项目文件列表 。

            <br />

  2. 向下滚动到“ spring-webflow”发行包，然后选择
    “ spring-webflow-preview2.zip”可供下载。

            <br />

发布档案包含两个jar：

  • spring-webflow.jar-核心Web流系统（org.springframework.web.flow。*）
  • spring-webflow-support.jar-尚未作为核心Spring Framework 的一部分发布的支持软件包。

由于发行档案未随Spring Framework一起提供，请
如果尚未下载Spring 1.2 RC1，请先下载。

            <br />

一旦下载，请访问的Webflow
审查模块的空间
文档和“快速入门”。
  

            <br />

为了使Spring Web Flow运作起来更好，请检查样本
在samples / webflow目录内的发行档案中。
请参阅示例/webflow/readme.txt文件，了解如何构建和部署
示例应用程序。

            <br />

我们非常重视您的反馈-有很多新的内容，
此版本中令人兴奋的内容，请尝试一下！
