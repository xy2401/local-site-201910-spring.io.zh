Spring Web Flow PR4发布

发布
基思·唐纳德
2005年7月17日

亲爱的春季社区，

我很高兴宣布发布
Spring Web Flow预览4（PR4）。此版本与Spring Web Flow保持一致
适用于Spring 1.2.2+。

这将是最新的Web Flow预览版本；下一个版本将是1.0 RC1。要下载，请访问： http://sourceforge.net/project/showfiles.php?group_id=73357&package_id=148517

注意：PR3和PR4之间有一些不兼容的更改。这包括将根包从org.springframework.web.flow重命名为org.springframework.webflow ，
注意Web Flow作为独立模块的地位，值得
顶级程序包。有关更改的完整列表，请访问
在此处更改日志： http://opensource.atlassian.com/confluence/spring/display/WEBFLOW/Change+Log

特别
WebFlow Portlet用户的注释：PR4不附带Portlet MVC
积分。Portlet MVC用户将需要等待Web Flow RC1
在几个星期内集成了Portlet MVC以便与
春季1.3。对于由此给您带来的任何不便，我们深表歉意：期待
更好的Portlet MVC集成+示例应用程序又回到了
未来几周。
