工程
乔什·朗（Josh Long）
2015年2月10日

这篇文章是社区成员Julien Dubois（@juliendubois）的来宾帖子， Julien Dubois是SpringSource的前雇员，现为Ippon Technologies工作，并且是JHipster项目的创建者。谢谢朱利安！我希望看到更多这些来宾帖子，因此-和往常一样-不要犹豫ping我（@starbuxman） ！ -乔什

介绍

JHipster或“ Java Hipster”是一个方便的应用程序生成器，它将为您创建一个Spring Boot（这是Java部分）和AngularJS（这是时髦部分）应用程序。

在很短的时间内， JHipster在Github上变得非常流行 ，并且在在线杂志（如InfoQ ，Infoworld或SD Times）以及世界各地的会议（巴黎，伦敦，蒙特利尔，奥马哈，台北）中都得到了推荐。 ，里士满，法兰克福！

Spring Integration Kafka扩展1.0。GA可用

发布
阿尔特姆·比兰（Artem Bilan）
2015年2月9日

尊敬的Spring社区，

我们很高兴地宣布，Spring Integration Kafka 1.0 GA扩展现已可用，它为Apache Kafka提供了熟悉的Spring Integration端点。
与往常一样，将发布存储库与Maven或Gradle一起使用
compile "org.springframework.integration:spring-integration-kafka:1.0.0.RELEASE"
或下载发行档案 ，以进行试用。

首先，感谢所有为该项目做出贡献的人-特别感谢该项目的创始人Soby Chacko ，他实施了基础架构，以及基于高级消费者的消息源和生产者，还感谢Marius Bogoevici ，他了解了消息驱动型使用者的简单使用者API的复杂性。
