Spring Framework 5.0路线图更新

发布
于尔根·霍勒（Juergen Hoeller）
2015年12月3日

您可能已经听说过这个消息：JDK 9将推迟半年 ，现在的目标是2017年3月发行，而不是最初的2016年9月目标。

由于Spring 5旨在非常紧密地跟踪JDK 9，因此我们决定也重新审视自己的路线图：尽管如此，我们将在2016年第四季度开始我们的候选发布阶段，但是我们也不会在2017年3月之前发布GA。话虽如此，我们的意思是真的：目标是让Spring Framework 5.0在JDK 9的GA日期之后立即进入GA。

无论如何，您开始使用Spring 5功能的机会将早得多，第一个Spring Framework 5.0里程碑计划于2016年6月发布……

2015年12月3日

我很高兴地宣布Gradle依赖性管理插件 0.5.4。现在可以从Maven Central，JCenter和Gradle Plugin Portal中获得RELEASE。

Spring Cloud Connectors 1.2.1发布

发布
斯科特·弗雷德里克
2015年12月2日

我很高兴地宣布Spring Cloud Connector 1.2.1的可用性。这是一个维护版本，具有一些新功能和一些错误修复。

以下是重点内容：

Heroku连接器

关系数据库

  • Heroku云连接器现在可以识别DATABASE_URL检测PostgreSQL数据库时的环境变量。

雷迪斯

  • 现在支持Heroku Redis 。

Spring 接头

关系数据库

  • 创建一个DataSource ，Spring Cloud Connector将寻找一个池DataSource在类路径上实现并根据默认优先级选择一个。检测的优先级已更改为更接近于Spring Boot中使用的顺序。
  • 应用程序可以自定义池的优先级顺序DataSource使用Java配置或XML配置的实现 。
  • DataSourceConfig现在将接受键/值对的映射，以允许设置创建的任意属性DataSource 。除了可以使用默认的JDBC驱动程序类名称和由连接器设置的验证查询之外，它还可以覆盖连接器设置的其他属性。 DataSource 。可以由Java配置或XML配置使用。

将OAuth2应用程序从Spring Boot 1.2迁移到1.3

工程
戴夫·瑟
2015年11月30日

Spring Boot 1.3中有一些新功能可用于OAuth2客户端和服务器以及Spring Security OAuth2 。其中的一些特性，从移植spring-cloud-security ，因此是在天使释放列车Spring Cloud ，但不是在布里克斯顿发布列车。本文可帮助您浏览更改并更新任何现有应用程序以使用新功能。

依赖管理

如果您不使用Spring Cloud，则应该能够更改Spring Boot依赖项的版本号。由于OAuth2的某些功能已从Spring Cloud Security迁移到1.3中的Spring Boot，因此事情可能会稍微复杂一些。另一篇文章介绍了将Spring Cloud应用程序从Spring Boot 1.2升级到1.3。如果您使用的是Spring Cloud Angel发布培训，则应查阅该文章，以获取有关如何管理依赖项（独立于任何特定功能）的详细信息。
