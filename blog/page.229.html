Spring LDAP 2.0.0。发布发布

发布
罗伯·温奇
2014年1月13日

我很高兴地宣布Spring LDAP 2.0.0。现在可以从Maven Central和Bintray购买RELEASE。特别感谢Mattias Arthursson在此版本中所做的所有工作！

请参阅Spring LDAP 2.0的新增功能以查找此发行版的完整详细信息。可以在JIRA的更改日志中找到更改列表。重点包括：

  • Spring LDAP现在包含Spring Data Repository和QueryDSL支持。
  • 流利的LDAP查询支持已添加。
  • 现在提供了一个自定义XML名称空间，以简化Spring LDAP的配置。
  • Spring LDAP核心已使用Java 5功能（例如泛型和varargs）进行了更新。
  • ODM（对象目录映射）功能已移至核心，LdapOperations / LdapTemplate中有一些新方法，它们使用这种自动转换到ODM注释的类/从ODM注释的类进行自动转换。

Spring XD 1.0.0。M5发布

工程
马克·波拉克
2014年1月10日

Spring XD团队高兴地宣布，现在可以下载 Spring XD 1.0.0 Milestone 5 。

通过Spring XD，可以轻松解决常见的大数据问题，例如数据提取和导出，实时分析以及批处理工作流程编排。此版本包括几个值得注意的新功能：

  • JDBC到HDFS的预定义批处理作业
  • 许多其他作业外壳程序命令可用于控制和报告批处理作业。
  • 使用Kite SDK的Hadoop数据集Avro接收器
  • HDFS接收器现在支持编解码器（gzip，snappy，bzip2，lzo）和对文件命名的细粒度控制
  • 除了队列， JMS源模块现在还支持主题。
  • 改进了对Gemfire定位器使用的支持
  • 用于批处理的聚合器模块 ，还支持用于将事件流检查点指向内存，Redis或关系数据库的支持消息存储。
  • 支持单独的控制和数据传输协议
  • XD服务器现在基于Spring Boot构建

SpringOne2GX 2013重播：Spring RESTBucks-一种超媒体驱动的REST Web服务

新闻与活动
乔什·朗（Josh Long）
2014年1月8日

在加州圣塔克拉拉的SpringOne2GX 2013上录制。

演讲者：奥利弗·吉尔克（Oliver Gierke）

Spring MVC为在Java中实现基于REST的Web服务奠定了坚实的基础。但是，在现实世界的项目中，开发人员在面对REST的高级问题时仍然面临挑战。如何真正利用超媒体？如何使用REST为更复杂的业务功能建模。该演讲介绍了超媒体驱动的REST Web服务的Spring RESTBucks示例实现，并解释了它如何使用超媒体元素实现业务流程以及Spring技术（Spring Data REST，Spring Data JPA和Spring HATEOAS）如何帮助开发人员构建系统。在http://projects.spring.io/spring-framework/（Spring MVC的REST控制器）http://projects.spring.io/spring-hateoas/（超媒体支持）上了解有关REST的更多信息。

SpringOne2GX 2013重播：使用OAuth2进行数据建模和身份管理

新闻与活动
乔什·朗（Josh Long）
2014年1月8日

在加州圣塔克拉拉的SpringOne2GX 2013上录制。

演讲者：David Syer博士

OAuth2规范（明智地）为解释和实现细节留出了很多领域，因此有很多机会可以对流和基础数据进行解释。本演示文稿从OAuth2主要功能的基本指南开始，然后通过示例展示如何利用它们来支持业务和应用程序用例。例如，您应该直接在访问令牌中对访问决策数据进行编码，还是使该令牌完全不透明？您应该签署请求吗？您应为OAuth2范围使用哪种命名约定？您如何注册用户和客户？现有的OAuth2实现中有一些显而易见的模式，如果您希望复制其中之一或制定自己的规则，Spring Security OAuth提供了很多挂钩和扩展点。示例将使用Spring和Spring Security展示如何在规范和库中利用固有的灵活性。了解有关Spring Security OAuth的更多信息：http://projects.spring.io/spring-security-oauth和Spring Security：http://projects.spring.io/spring-security
