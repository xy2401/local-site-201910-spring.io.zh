Spring Cloud Hoxton。M1可用。

发布
斯宾塞·吉布
2019年7月3日

我很高兴代表社区宣布， Spring Cloud Hoxton发布列车的里程碑1（M1）现已上市。该版本可以在Spring Milestone存储库中找到。您可以查看Hoxton发行说明以获取更多信息。

Hoxton发布系列的重大变化

此里程碑版本与Spring Boot 2.2.0兼容。M4。

Spring Cloud Gateway

Spring Cloud Gateway增加了对RSocket的支持。在即将发布的博客文章中查找更多内容。

Spring Cloud合约

Gradle支持已更新至5.5，而Groovy已更新至2.5版本。发生了重大的重构，并尽可能简化并重写为Java。
