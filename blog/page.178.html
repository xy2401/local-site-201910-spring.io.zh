SpringOne2GX 2014重放：使用Spring Cloud Connector简化云应用程序

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年3月10日

记录在SpringOne2GX 2014中。

注意：此项目已重命名为Spring Cloud Connectors，并且是较大的Spring Cloud 伞项目的一部分。

演讲者：斯科特·弗雷德里克（Scott Frederick），拉姆尼瓦斯·拉达（Ramnivas Laddad）

为云跟踪而开发

幻灯片： http ： //www.slideshare.net/SpringCentral/simplify-cloud-applicationsusingspringcloud

将应用程序开发到云平台需要使用已部署的应用程序环境并连接到服务。新项目Spring Cloud在包括Cloud Foundry和Heroku在内的各种云平台中简化了这些任务。Spring Cloud可以将相同的工件（战争或罐子）部署到多个云环境。它通过Cloud Connector的概念支持多个云，并为Cloud Foundry和Heroku提供了开箱即用的实现。Spring Cloud专为扩展而设计，可轻松为其他云平台创建云连接器。Spring Cloud还支持通过服务连接器的概念连接到多个服务。开箱即用，它提供了对许多常用服务的支持，而且还可以轻松地将其扩展到其他服务。尽管使用任何JVM语言和框架的应用程序都可以使用Spring Cloud，但它通过基于Java和XML的配置进一步简化了Spring应用程序。在本讲座中，我们将介绍Spring Cloud项目，展示如何简化配置应用程序以进行云部署，讨论其可扩展性机制，并通过展示该领域的实际示例来充分利用它。

SpringOne2GX 2014重播：神圣整合测试的追求

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年3月10日

记录在SpringOne2GX 2014中。

演讲者：肯·克鲁格（Rob Winch）

Web / JavaScript跟踪

幻灯片： http ： //www.slideshare.net/SpringCentral/spring-one2gx-2014holyintegrationtest

Spring MVC Test可以极大地帮助您彻底测试控制器及其配置。但是，对于基于浏览器的客户端，我们无法像用户一样轻松地与应用程序进行交互。例如，用户将请求一个包含表单的页面，填写表单，提交表单，某些Java脚本可能会执行，然后用户会看到某种结果。在本演示中，我们将概述测试Spring Web应用程序。我们将看到，通过结合使用Spring MVC Test和HtmlUnit，我们能够以与用户相同的方式（包括JavaScript执行）轻松地与我们的应用程序进行交互。我们还将看到如何轻松创建表示视图的可重用组件，以便在重构应用程序时可以轻松更新测试。最后，我们将看到如何将这些技术与BDD相结合，以找到集成测试的圣杯。

Activiti和Spring Boot入门

工程
乔什·朗（Josh Long）
2015年3月8日

这篇文章是Activiti联合创始人兼社区成员Joram Barrez（@jbarrez）在Alfresco工作的来宾帖子。谢谢乔拉姆！我希望看到更多这些社区来宾的帖子，因此-和往常一样-不要犹豫，向我（@starbuxman）征求意见和贡献！ -乔什

介绍

Activiti是Apache许可的业务流程管理（BPM）引擎。这样的引擎的核心目标是获取由人工任务和服务调用组成的流程定义，并按一定顺序执行，同时公开各种API以启动，管理和查询有关该定义的流程实例的数据。与许多竞争对手相反，Activiti轻巧且易于与任何Java技术或项目集成。所有这些，它可以在任何规模上运行-从几十到数千甚至数百万个流程执行。

Spring Cloud 1.0.0现在可用

发布
戴夫·瑟
2015年3月4日

Spring Cloud 1.0.0。现在可以在Maven Central（和repo.spring.io）中使用RELEASE。自RC3以来，变化不大，但是由于社区的参与，我们确实发现了一些错误，因此感谢所有迄今为止尝试过的人。这提醒了Spring Cloud的目标：

Spring Cloud为开发人员提供了工具，可快速构建分布式系统中的一些常见模式（例如，配置管理，服务发现，断路器，智能路由，微代理，控制总线，一次性令牌，全局锁，领导选举，分布式会话，群集状态）。分布式系统的协调导致样板式样，并且使用Spring Cloud开发人员可以快速站起来实现这些样板的服务和应用程序。它们将在任何分布式环境中都能很好地工作，包括开发人员自己的笔记本电脑，裸机数据中心以及诸如Cloud Foundry之类的托管平台。
