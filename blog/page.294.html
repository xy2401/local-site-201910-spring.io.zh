Spring BlazeDS集成1.5.0。M2发布

发布
杰里米·格莱尔
2011年2月10日

亲爱的春季社区，

我很高兴地宣布1.5.0。M2的里程碑版本春天BlazeDS集成 ，构建基于Spring的RIA的与Adobe Flex开源解决方案，现已推出。

下载 | 参考文档 | JavaDocs | 变更日志

此版本中重要的新功能和增强功能的重点包括：

  • 支持使用RESTful Spring MVC @Controllers读写AMF内容。这可以作为远程服务的直接RPC方法的不错选择，从而减少了需要支持多种客户端类型的应用程序中的重复工作。（有关使用示例，请参见更新的“测试驱动器”。）

  • 对Spring Security 3支持的重大重构，使得Flex应用程序与各种Spring Security功能（如“记住我”，“会话固定保护”和“并发会话控制”）的集成更加容易。

  • 1.5.0中引入了对Hibernate序列化支持的进一步发展。M1，包括使用直接字段映射而不是基于属性的映射的选项。

  • 框架提供的服务（例如LoginCommand和MessageInterceptor）的自定义和命名空间支持的配置更加容易。

  • 与Spring Integration 2.0.x的升级兼容性

  • 基于Maven的Test Drive示例的完整修订版，包括使用Flex 4的全面更新，以及演示RESTful编程模型的全新示例。

春季Android 1.0.0。M2发布

发布
罗伊·克拉克森
2011年2月9日

亲爱的春季社区，

我们很高兴地宣布， Spring Android项目的第二个里程碑版本现已发布！

Spring Android支持在Android环境中使用Spring Framework。1.0.0。M2版本着重于在本机Android应用程序中扩展RestTemplate的使用：

  • 现在，默认情况下，我们对所有REST调用都使用HttpComponents HttpClient 4，它是Android平台上的本机HTTP客户端。这取代了第一个里程碑中对Commons HttpClient 3的使用。
  • 通过使用简单XML序列化库，我们添加了对象到XML封送处理支持。Simple占用空间小，并且与Android兼容。
  • 现在，Android Rome Feed Reader支持RSS和Atom Feed。Android Rome是流行的Rome库的端口，该库与Android兼容。
  • 与Jackson JSON Processor的集成继续提供一流的Object-to-JSON封送处理支持。

针对Java和.NET的Spring GemFire 1.0.0发布

发布
Costin Leau
2011年2月2日

亲爱的春季社区，

我们很高兴地宣布，Spring GemFire 1.0项目的第一个GA版本现已可用于Java和.NET！Spring GemFire项目旨在使使用GemFire作为分布式数据管理平台来构建基于Spring的高度可扩展应用程序变得更加容易。

立即下载 ： 适用于Java的Spring GemFire | NET的Spring GemFire

此版本具有：


  • GemFire基础结构的声明性依赖项注入样式配置（例如Cache，Region，interest等）

  • 广泛的名称空间支持，用于配置所有主要的GemFire组件：缓存，复制，分区和客户端区域等等

  • 异常转换为Spring的可移植DataAccess异常层次结构

  • 模板和回调支持可轻松访问本机API

  • 交易管理支持

  • 弹簧支持的GemFire管理对象布线

  • 自动生成基于非反射的实例化器

  • 对GemFire 6.5的原生支持（6.0以后）

  • 声明式缓存建议（适用于.NET）

春天的这个星期：2011年2月1日

工程
乔什·朗（Josh Long）
2011年2月2日

本周社区的发展令人着迷。取决于您的外观，令人兴奋的是更新的，更前沿的东西（例如Spring Data，Gemfire和Virgo）的背后，或者是对Tomcat和SpringSource Tool Suite等技术的出色改进的背后。无论哪种方式，本周都有很多事情要做，所以让我们开始吧！

  1. SpringSource Tool Suite 2.6.0。M1已发布。这个新版本具有改进的内容辅助和快速修复，基于Spring MVC @RequestMapping的应用程序的新向导和视图。它还具有针对Groovy的改进的内容辅助性能，以及针对Groovy的改进的语法突出显示功能。
  2. Spring Data Graph-Neo4j支持1.0.0。M2发布 。新版本具有许多新功能，包括改进的索引支持-完整，对节点和关系的命名索引支持，节点和关系实体的删除，对框架方法中实体实例的严格类型检查（根据类型策略），对动态投影的支持到任意图形实体，以及对Neo4j 1.2的新更新支持。
  3. Shekhar Gulati 在IBM的DeveloperWorks上发布了Spring Roo的简介（第1部分，从源代码构建） 。
  4. Tomcat项目的高级撰稿人Mark Thomas 写了有关防止Tomcat 7中的跨站点脚本攻击的文章。
  5. VMWare vFabric GemFire团队刚刚整理了一个视频，演示了如何设置GemFire的Hibernate Cache Module 。该视频演示了如何在Maven中配置模块，然后演示了其在基于Spring-DAO的实现中的用法。最后，演示了如何使用Spring Insight监视应用程序的性能。酷视频，更酷的技术！

  7. Shekhar Gulati在Spring Data和Redis上发表了另一篇很棒的文章。Spring Data是一系列旨在支持更专业的数据持久性需求的技术的总称。其中很大一部分是对当今可用的许多NoSQL数据存储的深入支持，以及对基于JDBC的持久性的更专业支持。
