Spring Security SAML和本周的SAML漏洞

工程
罗伯·温奇
2018年3月1日

本周，软件界发现了影响多个实现的SAML漏洞 。如果使用Spring Security SAML的默认值，则不会受到此漏洞的影响。

Spring Security SAML使用的基础实现是Shibboleth的OpenSAML Java库。包含漏洞的库中未列出OpenSAML Java实现（Shibboleth openSAML C ++容易受到攻击）。但是，如果ParserPool 已被自定义 ，则可能会受到影响。
