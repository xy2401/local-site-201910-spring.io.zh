    Groovy 2.0发布

    工程
    纪尧姆·拉福格（Guillaume Laforge）
    2012年7月2日

    Groovy开发团队和SpringSource很高兴回应宣布Groovy 2.0的发布的消息， Groovy 2.0是Java平台上非常流行的动态语言。这个重要里程碑的主要亮点是：


    • 静态类型检查器，以使编译器告诉您代码的正确性，

    • 静态编译以提高应用程序关键部分的性能，

    • 模块化 ，将Groovy JAR拆分为较小的面向功能的JAR，并让您创建自己的扩展模块 ，

    • JDK 7 Project Coin 语法增强功能 ，因此Groovy对其Java表亲仍然尽可能友好，

    • 和JDK 7 Invoke Dynamic集成可受益于JVM的动态语言支持。

             Another great post from the VoltDB blog talks about using the Spring Converter API with VoltDB's Data Objects.  The idea is a bit unusual, but pretty slick when you think about it: <a href = "http://voltdb.com/company/blog/using-spring-converter-api-voltdb-data-objects">let Spring's generic <EM>converter</EM> registry handle converting Volt's notion of record sets (objects of type <CODE>VoltTable</CODE>) into regular, domain-specific objects</A>.     Spring Data REST 1.0.0。RC1发布

    发布
    乔恩·布里斯宾
    2012年6月26日

    我很高兴地宣布发布Spring Data REST 1.0.0。RC1！除了许多错误修复之外，此版本还增加了对分页和排序的支持，并使将Spring Data REST集成到现有的Spring MVC应用程序中变得更加容易。

    新功能包括：

    • 分页-添加URL参数，例如“ page = 2”和“ limit = 20”，以控制大型结果集的分页。
    • 排序-添加URL参数（例如“ sort = name”）以控制结果集的排序。
    • 与现有Spring MVC应用程序集成-现在，您只需将JavaConfig bean包含到自己的配置中，就可以轻松地将Spring Data REST集成到现有Spring MVC应用程序中。
