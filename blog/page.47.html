Spring Boot 2.1 M2现已上市

发布
斯特凡·尼科尔（StéphaneNicoll）
2018年8月21日

我谨代表团队和所有贡献者，宣布Spring Boot 2.1的第二个里程碑已经发布，可以从我们的里程碑存储库中获得 。此版本关闭了75个问题并提出了请求 ！

第二个里程碑与Spring Framework 5.1保持一致。RC2 ，是支持Java 11的第一个版本。

第二个里程碑的其他亮点包括：

  • 自动配置任务执行（ @EnableAsync ）和任务调度（ @EnableScheduling ）
  • JPA的可配置引导程序模式
  • OAuth2资源服务器支持
  • 改进了Web应用程序的日志输出
  • 改进了数据大小属性的绑定。

提供Spring Data Lovelace RC2

发布
奥利弗·德罗博姆（Oliver Drotbohm）
八月20，2018

我谨代表Spring Data团队宣布Lovelace RC2版本的可用性。这是为准备Spring Boot 2.1 M2以及我们自己的GA版本（9月中旬）而准备的。该发行带有82张固定的门票 。您可以在更新和策划的发行版Wiki中找到更多详细信息。

一个很晚但很重要的新功能使它成为了该版本：Spring Data JPA的延迟存储库初始化。该功能使您的启动可以更有效地使用Spring Framework的JPA后台初始化选项，并且可以大大减少启动时间。有关详细信息，请参见预览示例 。
