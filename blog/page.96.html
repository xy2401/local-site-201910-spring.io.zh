工程
乔什·朗（Josh Long）
2017年3月7日

欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Spring NinjaStéphaneNicoll 刚刚发布了Spring Boot 1.4.5和1.5.2 ，它们都包含了一些小的改进和错误修复。
  • Spring Security忍者Joe Grandja刚刚宣布了Spring Security OAuth 2.1.0和2.0.13 。此版本包含支持使用JSON Web密钥（JWK）进行JSON Web签名（JWS）验证的新功能。此功能提供对授权服务器的支持，并已实现密钥翻转/轮换。

  • 我喜欢Spring Cloud Data Flow的Eric Bottard的InfoQ采访
  • 上周，继续进行Spring Tips系列， 我们看了Spring 5的程序化bean注册
  • Christophe 通过Spring Cloud很好地结合了Pivotal Web Services上的容器网络 -快来查看！
  • Spring NinjaStéphaneNicoll 也刚刚发布了Spring Framework 4.3.7（维护版本）
  • 消息经纪人Solace的人们整理了一篇不错的文章， 介绍了他们的Solace Spring Boot启动程序
  • NewStack 很好地了解了Apache Geode

  • 来自堪萨斯城春季用户组的社区传奇人物Matt Raible和Micah Silverman的另一个精彩演讲，探讨了使用Spring Boot，Zuul和Stormpath探查安全领域
  • Spring Data负责人Oliver Gierke刚刚宣布了Ingalls SR1和Hopper SR8的发布
  • 我喜欢这个非常有趣的Twitter讨论，从Spring团队的Rob Winch开始， 为集成测试寻找好的选择

Spring Security OAuth 2.1.0和2.0.13发布

发布
乔·格兰贾
2017年3月3日

我很高兴代表社区宣布Spring Security OAuth 2.1.0的发布和维护版本2.0.13。

2.1.0版本包含一项新功能，该功能支持使用JSON Web密钥（JWK）进行 JSON Web签名（JWS）验证。此功能为已实现密钥翻转/轮换的授权服务器提供支持。有关更多详细信息，请参见GitHub Issue 。

2.0.13发行版包含少量错误修复和较小的增强功能，以及支持即将推出的Spring Boot 1.5.2发行版的JWK功能。
