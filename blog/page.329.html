Spring Web Services 1.5.0最终发布

发布
Arjen Poutsma
2008年3月28日

尊敬的Spring社区，
我很高兴地宣布Spring Web Services 1.5.0已发布！

资料下载 网站 | 变更日志 | 公告

此最终版本候选版本在1.0.3之上具有以下新功能：

  • 两种新的传输方式：针对客户端和服务器的JMS和电子邮件，
  • 基于WSS4J的WS-Security实现，该实现允许在非SUN JDK（即WebSphere）和JDK 1.4上进行WS-Security，
  • WS-Addressing对客户端和服务器的支持，支持2004年8月及最终版本的规范，
  • 对Java 6的本机支持，包括JAXP 1.4，捆绑的SAAJ 1.3和JAXB 2.0，
  • 两个新的Spring命名空间，它们大大减少了配置编组器和典型的Spring-WS构造所需的XML数量，
  • Spring-WS jar现在是OSGi捆绑包，
  • 一种新的客户端拦截机制，包括WS-Security支持，
  • @Endpoints现在是@Components，因此在使用Spring 2.5组件扫描时会自动拾取它们
  • 一个新的和改进的XSD到WSDL生成器，可以内联包含和导入的XSD
  • 支持Spring Security
  • 对Java 6 HTTP Server的支持
  • 两个新示例，展示了Java 6 HTTP服务器使用Plain Old XML的用法和WS-Addressing
