适用于AWS 2.0.0的Spring Integration。M2和Spring Cloud Stream Kinesis Binder 1.0.0。M2

发布
阿尔特姆·比兰（Artem Bilan）
2018年4月17日

亲爱的春季社区！

我很高兴宣布支持Amazon Web Services的 Spring项目的里程碑发布： 适用于AWS的Spring Integration 2.0.0.M2和适用于AWS Kinesis的Spring Cloud Stream Binder 1.0.0.M2 。

这两个里程碑在Spring Milestone存储库中都可用，并且可以作为Maven依赖项使用：

<dependency>
    <groupId>org.springframework.integration</groupId>
    <artifactId>spring-integration-aws</artifactId>
    <version>2.0.0.M2</version>
</dependency>

<dependency>
    <groupId>org.springframework.cloud</groupId>
    <artifactId>spring-cloud-stream-binder-kinesis</artifactId>
    <version>1.0.0.M2</version>
</dependency>

Spring Cloud GCP 1.0.0 Milestone 3的新增功能

发布
阿尔特姆·比兰（Artem Bilan）
2018年4月16日

亲爱的春季社区！

Spring Cloud GCP是一组Spring Boot启动器，旨在帮助您直接从Spring Boot利用Google Cloud Platform托管的服务，从而使您可以将Spring Messaging用于Cloud Pub / Sub，将Spring Resource用于Cloud Storage，将Spring Cloud Sleuth用于Stackdriver Trace，带有Cloud SQL的Spring数据等。

我们刚刚发布了Spring Cloud GCP 1.0.0.M3 。查看自上一个里程碑2以来的新功能！

Spring Boot 2.0 GA支持

Spring Cloud GCP 1.0.0.M3现在可以与Spring Boot一起使用2.0.1.RELEASE和Spring Cloud Finchley.M9 。

使用Spring Cloud GCP 1.0.0.M3 ，只需在您的Spring Boot中导入BOM表2.0项目：

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-gcp-dependencies</artifactId>
            <version>1.0.0.M3</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>

Spring Data Lovelace M2发布

发布
马克·帕鲁奇
2018年4月13日

我谨代表Spring Data团队宣布Lovelace发布火车第二个里程碑的可用性。
