发布了Spring Data Moore RC2，Lovelace SR10和Ingalls SR23

发布
马克·帕鲁奇
2019年8月5日

我很高兴代表团队宣布Spring Data发布了Moore RC2，Lovelace SR10和Ingalls SR23。所有发行版都采用了最新的Spring Framework发行版，并将分别由Spring Boot 2.2 RC1、2.1.7和1.5.22所采用。

在许多依赖项升级中，我们迁移了几乎所有的Spring Data核心项目以使用AssertJ断言。根据我们之前的声明 ，Ingalls SR23是Spring Data 1.x产品线的最后一个发布版本，该产品即将到期 。对于所有仍在使用Spring Data 1.x的用户，我们强烈建议迁移到Spring Data Lovelace SR10。

  <groupId>org.springframework.geode</groupId>
  <artifactId>spring-geode-starter</artifactId>
  <version>1.1.0.RC1</version>

Spring Cloud Data Flow 2.2.0发布

发布
珍妮·瓦尔基亚拉赫蒂
2019年7月30日

Spring Cloud Data Flow团队很高兴宣布GA版本2.2.0。

在此GA版本中，Spring Cloud Data Flow团队研究了一些关键功能，包括任务应用程序管理，通过增加跨平台（本地，Kubernetes和Cloud Foundry）的验收测试范围，产品的稳定性，错误修复和增强功能。

任务应用管理

现在可以使用SCDF仪表板和命令行管理程序停止和删除任务执行。
我们增加了对使用千分尺集成的任务应用程序监视的支持。Micrometer集成的核心位于Spring Cloud Task的2.2.0版本中，如果您要尝试Task-metrics和SCDF集成，则这是必需的。
