HBase DAO支持

Android 1.0.0的春季发布

发布
罗伊·克拉克森
2012年5月30日

亲爱的春季社区，

我们很高兴宣布Android 1.0的Spring全面上市！

Spring for Android支持在Android环境中使用Spring Framework。

自RC1版本以来，此GA版本包括一些小的修复和改进。有关更多信息，请参见变更日志和参考手册 。

    @Bean 
	public static CustomScopeConfigurer csc(){ 
	  
	  Map scopes = new HashMap();
	  scopes.put("thread", new SimpleThreadScope());
	  	
	  CustomScopeConfigurer csc = new CustomScopeConfigurer();
	  csc.setScopes(scopes);
	  return csc;	
	}
