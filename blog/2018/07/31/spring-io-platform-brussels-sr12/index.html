Spring IO Platform布鲁塞尔SR12

发布
安迪·威尔金森
七月31，2018

我很高兴地宣布， repo.spring.io和Maven Central现在都可以使用Spring IO Platform Brussels-SR12。

该维护版本升级了平台中多个项目的版本：

  • Spring AMQP 1.7.9
  • Spring Boot 1.5.15
  • Spring Data Ingalls SR14

许多第三方依赖项的版本也已更新。
