Spring Cloud Edgware EOL 2019年8月1日

新闻与活动
斯宾塞·吉布
七月30，2018

Spring Cloud Edgware将遵循Spring Boot 1.x EOL声明 。从2019年8月1日起，我们将在12个月后停止维护Edgware版本培训。

截止到那时，我们将继续发布偶尔的Edgware服务发布，然后结束发布过程。

提醒一下，卡姆登（Camden）发行火车已经到达停产期，达斯顿（Dalston）将在2018年12月底到达停产期。

现有的Spring Cloud用户应进行相应的计划，以确保在该日期之前已升级到最新的Finchley +版本。另外，请注意，Spring Cloud Finchley及更高版本需要Spring Boot 2.x，后者需要Java 8+。因此，如果您运行的是Java 6或7，则还应计划升级JDK。
