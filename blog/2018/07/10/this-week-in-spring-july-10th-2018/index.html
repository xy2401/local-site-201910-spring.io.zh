春天的这个星期-2018年7月10日

工程
乔什·朗（Josh Long）
七月10，2018

嗨，春季粉丝！欢迎来到春季本周的另一期！这星期我阳光明媚（温暖！）洛杉矶参加史诗般的SpringOne Tour洛杉矶活动，然后前往波士顿开会。像往常一样，我们有很多内容需要介绍，所以让我们开始吧！

  • Pivotal Cloud Foundry 2.2已发布， 并且具有更智能的指标支持
  • 您是否要从开源开始，或者至少要开始为Spring Boot做贡献？我们有一吨的开放式问题，我们会欢迎外界的贡献 ！
  • Spring Cloud Data Flow 1.6 M1和1.5.2发布
  • Spring Tool Suite 3.9.5发布
  • 关注Graal支持基于Spring Boot的本机映像应用程序的进展
  • 春季社交报废公告
  • 我喜欢这篇文章，探讨e-Bay如何对基于Spring Boot的应用程序进行更快的集成测试
  • 在Zipkin 2.10中检查向v2的迁移 。
  • 这是我今天需要的老歌，但又是个好东西：它着眼于Solace消息技术的Spring Boot启动器
  • Clever Cloud博客上有一篇不错的文章，介绍了如何调整JHipster Spring Boot和基于Angular的应用程序以在其平台上进行部署
  • https://twitter.com/springcentral/status/1016747383483322368
  • 了解如何使用HAProxy安装多云的企业级CloudFoundry。立即观看视频 。
  • 社区传奇人物Martin Deinum 在关于使用PropertySource和.yaml档案
  • 受Spring Cloud启发的Steeltoe.NET库支持基于.NET的应用程序的微服务模式 ，这很热 ！它在Nuget上的下载量超过50万！
  • 我喜欢在AWS EKS上使用Chaos Monkey进行Spring Boot的这篇文章
  • 春天传奇人物Thomas Darimont刚刚升级了他的示例，演示了如何使用Keycloak 4.1.0保护Spring Boot Admin 2.0.1 UI和Spring Boot 2.0.3 Actuator端点到最新版本。
  • 在哪个社区英雄尼古拉·弗兰克尔（Nicolas Frankel）精妙地重新审视了一个我们可能理所当然的基本问题： 为什么我们应该使用像Spring 这样的框架或依赖项注入容器 ？为什么要完全使用框架或依赖项注入容器？
  • Apache Camel 2.2.2-第一个支持Spring Boot 2的服务器！ - 现在可用！
  • 查看我们自己的Mario Gray对反应式Spring Security进行身份验证的内容 。这是一系列有关安全性的文章之一。在此处查看最后一期 。
  • 本杰明·威尔姆斯（Benjamin Wilms）撰写的这个以代码为中心的帖子，是一般混沌工程必读的宣言， 对于任何考虑将Spring Boot用作混沌猴子的人来说都是有用的背景
  • 这是一个老套的好东西： Oracle的无服务器项目Project Fn支持Spring Cloud Function 。
  • Jonas Hecht 关于Spring Boot和Vue.js的惊人文章现在可以在德语杂志JavaMagazin的最新版中找到。
  • 休眠开发者倡导者弗拉德·米哈尔切亚（Vlad Mihalcea）提出的拉取请求看起来非常有希望， 并且 其中的讨论也很有启发性
  • Graal开发人员拥护者OlegŠelajev的精彩文章展示了在Graal上运行基于Spring Boot的应用程序时的一些独特可能性： 使用R数据科学增强Java Spring应用程序 。
  • Nexocode博客上的Wojciech Marusarz在一篇很棒的文章中介绍了反应式编程 （就基础而言，但有使用Reactor和Spring的简短示例）
  • 这是来自Spring Integration主管Gary Russell的有趣知识块 ：您知道您可以使用QPID JMS桥使用Spring Integration / JMS与QPID（AMQP 1.0）进行对话 ，这应该是将Spring Integration与其他符合AMQP 1.0的经纪人？
  • 查看有关如何使用Open Service Broker API为Cloud Foundry和Kubernetes构建服务的文章
  • 这是一篇关于使用以太坊，web3j和Spring Boot的Java区块链的有趣文章。皮奥特·米科夫斯基（PiotrMińkowski）做好了！
  • 现在有对Spring Cloud Pipelines的PHP支持！
  • 请务必注册此7月18日的网络研讨会： 自带代码与自带容器
  • 祝贺Thibaud Lepretre发布了支持Spring Boot 2的第一个版本（测试版） cas-security-spring-boot-starter在最近的巴黎重要活动中！有关详细信息，请参见项目！
