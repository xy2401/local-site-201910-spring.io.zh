春天的这个星期-2018年5月8日

工程
乔什·朗（Josh Long）
2018年5月8日

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我将在英国曼彻斯特参加曼彻斯特水罐比赛，然后前往英国伦敦进行一些客户访问和史诗般的Devoxx UK活动 。下周这一次，我将在美国丹佛参加SpringOne Tour赛事 。如果您像往常一样在这些地方中的任何一个，请毫不犹豫地伸出援手（@starbuxman）

  • 错过了春季的本周分期付款吗？好吧，不用担心，现在有每周汇总的每月汇总！这是四月版
  • Dormain Drewitz通过这一史诗般的宣布猛烈杀戮：Pivotal和Confluent合作，使Pivotal的Kubernetes环境PKS成为运行Confluent Kafka的最佳场所
  • Spring Framework杰出人物StéphaneNicoll刚刚宣布了Spring Framework .3.17和5.0.6
  • Spring Data传奇人物Mark Paluch刚刚宣布发布Spring Data Ingalls SR12和Kay SR7 。这些发行版像往常一样包含新发行版！
  • Spring Batch和Spring Cloud Task负责人Michael Minella刚刚宣布了Spring Cloud Task 2.0.0。发布 。新版本更新了相关性，支持限制并发任务执行，并在批处理作业执行失败使任务失败时改善了行为。
  • Pivotal博客通过Spring Session研究状态管理 。
  • 在上周的Spring Tips中 ，我研究了在Spring Boot 2的执行器中使用新的度量标准集合外观Micrometer 。看看这个！
  • BOSH是一个抽象层，可让您以与IaaS无关的方式描述不可变的基础结构。有适用于AWS，GCP，Azure，OpenStack，VSphere的BOSH提供程序，以及无数其他服务提供程序。针对应用程序管理进行了优化的Cloud Foundry建立在BOSH之上。这使其具有无与伦比的便携性。现在， 有一个Kubernetes BOSH CPI 。这是非常有前途的！
  • 新堆栈着眼于T-Mobile 与Tibco和CloudFoundry的5G融合
  • 在Okta博客上查看高级Spring粉丝Matt Raible的“使用WireMock，Jest，Protractor和Travis CI测试Spring Boot API和Angular组件的Hitchhiker指南” 。
  • Pivotal首席技术专家Jim Weaver 通过这个出色的演示介绍了反应式编程 。
  • 我喜欢这个帖子！它与Spring无关， 但是如果您是有抱负的云原生用户，则可能需要阅读它。这是Pivotal员工关于软触摸重新配置的一则帖子，他们利用公共云IaaS资源进行了重新配置， 每年可节省多达67.5万美元 。

  • 该Baeldung有一个很好的职位，着眼于对Assert Spring框架中的类 。简短而甜蜜！
