Spring 的这个星期-2018年5月1日

工程
乔什·朗（Josh Long）
2018年5月1日

嗨，春季粉丝！欢迎来到春季本周的另一期！在上周，我从德国（代表JAXON）去了奥地利林茨（代表DevOne）和奥地利维也纳（了聚会），现在我在澳大利亚墨尔本（在泰国曼谷停留了10个小时），参加VOXXED墨尔本活动。和往常一样，如果您在身边， 我很乐意收到您的来信！

事不宜迟，我们有很多工作要做，让我们开始吧！

  • Pivotal刚满5岁！ 祝我们生日快乐！
  • Spring Cloud贡献者Ryan Baxter刚刚宣布了Spring Cloud Finchley RC1。新版本包含各种新内容，并有望成为Spring Cloud Finchley正式发布之前的最终版本之一
  • 嗨，春季粉丝！在上周的Spring Tips中，我研究了如何从业务逻辑与Spring Statemachine中解脱出流程状态，这对于协调长期运行的或多角色的流程非常有用。
  • RabbitMQ 3.7.5-beta2现已上市 。
  • 我非常喜欢这篇中文帖子，内容涉及中小型公司是否可以使用 Spring Boot和Spring Cloud
  • RabbitMQ博客上有一篇不错的文章，介绍了如何在RabbitMQ Java客户端中使用Micrometer集成来与Datadog集成
  • Jeroen Reijn很好地介绍了基于测试驱动的基于Spring REST Docs的文档
  • Ordina博客上有一篇不错的文章，介绍了如何使用Spring Cloud Contract支持消费者驱动的合同 。
  • 我喜欢Gaurav Gupta 在与Apache Kafka和Spring同步请求/回复上的帖子
  • Baeldung的另一篇有趣的文章（与Servlet相关）介绍了@ServletComponentScan在Spring Boot中 ：它检测所有Servlet API WebFilter ， WebListener和WebServlet Servlet组件上的注释。
  • 这是一个从Baeldung博客过时的歌曲，但-A-礼包： 快速介绍到SpringBootServletInitializer ，这是让Spring Boot应用程序在Servlet容器设置中运行的机制，而不是胖的- .jar 。
  • 千万不要错过这个新的 Spring Cloud 网关指南
  • Bartosz Jedrzejewski撰写了一篇很好的文章，介绍了使用Reactor（在Spring WebFlux应用程序中） 获得并行性和反压
  • 还有Baeldung的另一篇文章，这次是对Java状态的回顾 。今年的调查中的数字非常有趣！这些数字非常令人鼓舞：到目前为止，Java 8是Java的大多数部署版本。不过，Java 9和Java 10越来越流行！同样有趣的是：到目前为止，Spring Framework 4和Spring Framework 5是大多数已部署的企业Java框架。确切的数字尚不清楚，但看起来几乎是饼图的75％。我猜Java EE和“其他”技术占剩余部署的20％，其次是Spring Framework 3，大约占5％左右（也许更少？）在基于Spring的部署中，大多数基于Spring Boot，只有不到20％的基于不具有Spring Boot的Spring。IntelliJ，Apache Tomcat和Maven也是明显的领导者。同样有趣的是：Groovy，Kotlin和Scala（按顺序）是非Java语言的领导者。
  • Heroku文档提供了一个很好的步骤列表，这些步骤使Spring Boot应用程序可云就绪。该博客发布在Heroku博客上，但其中很多内容适用于为任何平台构建云原生应用程序
  • 这篇文章演示了如何在 没有 Spring 的Spring应用程序中进行测试 。它建立了一个场景， 应该是很罕见的，字段注入，并演示了如何不利这会影响测试的目标。记住我总是说的：“每次进行现场注入时，单元测试都会消失！”然后，文章探讨了如何测试组件，甚至将Spring从测试过程中删除。
  • Nicolas Frankel着眼于Spring Boot 2执行器的变化
  • 我喜欢使用Java 10的备忘单
  • Spring Cloud团队正在招聘！加入我们！
  • Michael Simons和MichaelPlöd在Spring Boot 2上放了一个漂亮的德语滑梯
  • All and Sundry博客上有一篇不错的文章，介绍如何使用Spring Cloud Gateway配置简单路由
  • 我的关于构建基于Spring的响应式应用程序系列的第二部分现已在Java Magazine中提供
  • Java Revisited博客上有一篇不错的文章，介绍了在Spring Cloud中追求认证的动机
  • 2018年11月12日在英国伦敦的CodeNode上举行的首届RabbitMQ峰会的征集文件现已开放！
