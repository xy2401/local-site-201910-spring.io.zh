Spring 的这个星期-2018年5月22日

工程
乔什·朗（Josh Long）
2018年5月23日

嗨，春季粉丝！ew！多么疯狂的一周！上周四，我降落在乌克兰基辅， 在美丽的基辅参加史诗般的 JEEConf 。然后，这个周末去了爱尔兰的都柏林和北爱尔兰的贝尔法斯特，在那里我有幸与一些了不起的Pivotal客户（如Liberty ）进行了交谈。我今天在贝尔法斯特这里结束了，明天将在西班牙贝拉巴塞罗那举行春季I / O会议 。

像往常一样，这是Spring社区中一个疯狂的令人敬畏的星期，因此，事不宜迟，让我们开始吧！

  • 现在，Spring Security Master中提供了OIDC提供程序配置支持 。太棒了！
  • 一定不要错过6月7日Spring Cloud Stream负责人Oleg Zhurakousky的Spring Cloud Stream新功能研讨会
  • 在Pivotal博客上的提问是 ：“那应该是微服务吗？”并给出了在调用和剪切时要记住的六个因素。
  • 您真的需要更多理由来使用SpringOne平台吗？千万不要错过这个引人注目的邀请，从反应堆项目负责人斯蒂芬·马尔蒂尼！
  • 使用适用于PCF的Weave Cloud可视化和监视您的Pivotal Cloud Foundry部署
  • Spring Data JDBC现在支持@Table和@Column感谢@ kazuki43zoo
  • Spring Data负责人Oliver Gierke宣布，我们正在扩展Spring Data的JavaSlang集成，以支持Try作为查询方法的返回值，并鼓励我们给这些位赋予..try
  • Spring Data传奇人物Christoph Strobl刚刚发布了Spring Data Lovelace M3 ，其中包括修订后的文档，针对JDBC模块的基于注释的审核，为MongoDB 4.0版本做准备的同步和反应式事务支持，针对MongoDB的反应式映射/简化抽象，Spring Data Gemfire和Apache Geode中对Lucene索引支持的增强，Spring Data Redis的反应式发布/订阅，Spring Data Cassandra的Kotlin扩展，Gemfire 9.5，Apache Geode 1.6，Lettuce 5.1（M1），MongoDB 3.8的驱动程序升级（beta2），MongoDB Reactive Streams 1.9（beta1）和Cassandra 3.5等等！
  • Spring Cloud Data Flow负责人Mark Pollack博士刚刚宣布了Spring Cloud Data Flow 1.5 ，其中包括UI改进，
    Spring Boot 2.0，Spring Cloud Stream 2.0和Spring Cloud Task 2.0支持，更新的应用程序启动程序，指标改进，组合任务的嵌套拆分，Kubernetes改进以及更新的File Ingest示例等等！
  • 嗨，春季粉丝！在上周的Spring Tips中，我研究了Project Riff和Spring Cloud Function， 这是针对无服务器功能即服务应用程序的一两次打孔 。Project Riff是来自Pivotal的FaaS平台，并且Spring Cloud Function使编写可移植的，具有Spring Boot意识的功能成为可能，从而简化了许多不同平台的工作。
  • Spring Data反应式驱动程序内部会发生什么，如何在反应式设置中访问数据？从Spring Data忍者Mark Paluch的演讲“反应式数据访问的内幕”中了解更多信息， 该演讲现已在InfoQ上发布。
  • RabbitMQ 3.7.5发布了！
  • Daniel Mikusa有一篇不错的文章， 向我们介绍了如何在Cloud Foundry上安装Wordpress 。本质上，这与Spring无关，但确实演示了如何将相当复杂的应用程序添加到平台上
  • “ Devskiller”博客上有一篇很棒的文章，介绍了如何使用Spring Cloud Contract测试REST和基于消息传递的微服务
  • 微软刚刚发布了适用于Spring Boot的Application Insights入门程序
  • Apache Kafka的Spring Integration和Spring 领导者Gary Russell提供了官方的“专业建议”！
  • 2.0.0。Spring Boot Admin的GA版本刚刚发布！新版本有一吨不错的功能，因此做检查了释放，同时你可以！注意：通过以下方式支持发现其他服务DiscoveryClient取决于尚未发布的GA Spring Cloud Finchley版本，该版本已被排除在外，并将在最终版本发布时发布。无论如何， 趁它们变热的时候把它们弄碎！
  • 这看起来很有趣！这是一个Spring RESTDocs片段实现，可生成可导入的Postman集合 。
  • InfoQ拥有Pivotal和Spring传奇人物的Sannidhi Jalukar和Madhura Bhave的演讲 ，内容涉及使用Boot批注和实用程序为Spring Boot应用程序进行TDD以及处理数据库查询，缓存，反应式编程。很棒的话题，值得一看！
  • Flowable是一个开源BPMN引擎，可以与Spring Boot很好地集成。Flowable项目工程师Tijs Rademakers刚刚宣布：“ Flowable 6.3.1已发布，其中包含针对CMMN Engine的增强的历史信息和可插入性选项，CMMN Engine是一种新的应用程序引擎，可支持Flowable中应用程序的生命周期，改进的Spring Boot 2.0.2支持和更多”
  • 这是有关将Apache Camel与Spring Cloud Stream一起使用的有趣文章
  • 我真的很喜欢这篇文章，它开始研究在欧盟新GDPR法规的背景下，在基于Spring Integration 的应用程序中支持“勿忘我”功能的应用程序。
  • Bogdan D 在使用 Spring Boot的功能切换库Toggles上有一篇不错的文章 。
  • 这看起来很有希望！似乎对Micrometer度量外观的支持刚刚落在Project Reactor中 ！
  • Spring Boot NinjaStéphaneNicoll 在支持Bootsching的Spring Boot 2.1中揭示了一个有趣的新Actuator功能
  • 看到Spring Test主管和JUnit主管Sam Brannen的诗歌被Java架构师Mark Reinold于2018年发表的关于Java状态的演讲中包含的诗作，真是太酷了

  • Okta开发倡导者Matt Raible刚刚发布了Spring Boot 2.0的更新，该视频的视频介绍如何使用Spring构建微服务和保护微服务。
  • “先生。 Devoxx，” Devoxx的创始人Stephan Janssen进行了精彩的演讲，介绍了他的团队使用Spring Boot和JHipster改造Devoxx属性的过程 。看看这个！
