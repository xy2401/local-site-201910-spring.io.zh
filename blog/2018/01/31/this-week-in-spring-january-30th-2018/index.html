春天的这个星期-2018年1月30日

工程
乔什·朗（Josh Long）
一月31，2018

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我在洛杉矶和芝加哥，现在在旧金山，与Pivotal生态系统的人们和客户举行会议。我们面前有一个令人难以置信的忙碌的一周，他们准备了很多东西，所以让我们开始吧！

  • 反应堆项目负责人史蒂芬·马尔蒂尼（Stephane Maldini）宣布，反应堆Bismuth-SR5已修复，其中包括Reactive Streams 1.0.2，Netty 4.1.20和一些新的很酷的功能
  • 查阅Spring Security忍者Joe Grandja的这篇文章 ，了解Spring对OAuth 2的支持的未来
  • Spring Security主管Rob Winch刚刚宣布立即更新Spring Security 5.0.1、4.2.4和4.1.5的CVE
  • Reactive Spring Web Ninja Sebastien Deleuze刚刚宣布在Spring Framework 5.0.4中集成并支持Reactive Smile（二进制JSON） 支持
  • 韦德兰·帕维奇（VedranPavić） 宣布春季会议2.0.1
  • Spring Framework负责人Juergen Hoeller刚刚宣布Spring Framework 5.0.3和4.3.14现已上市
  • James Williams在构建包的形式和功能方面做得很好。这是一个很好的，快速的手表 。
  • 温馨提示： 无论何时何地，我们都会尽力提供帮助。我们很高兴看到你们中的一些人对此表示赞赏：-)如果您可以访问Twitter上的各种Spring频道（ Spring Data ， Spring Security ， Spring Boot ， Spring Cloud ， Spring Framework ， SpringCentral或您真正的您），一个答复。您还可以尝试用于Spring项目 ，尤其是Spring Cloud的各种Gitter.im聊天室。另外，[我们还在StackOverflow上保持强大的存在 。

  • 确保签出在波士顿举行的Cloud Foundry Summit
