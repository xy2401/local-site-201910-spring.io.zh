春天的这个星期-2017年1月9日

工程
乔什·朗（Josh Long）
一月09，2018

嗨，春季粉丝！欢迎来到春季本周的另一期！本周我要去德国， 星期三晚上我将在明斯特的Java用户组演讲。然后， 在12号（本星期五）前往索林根参加Cloud Native一天，我将在整个下午进行演讲 -现在注册！而且，如果您比大西洋更靠近太平洋，那么下周一在夏威夷加入我，我们将在非常有前途的LavaOne会议上讨论Spring的所有问题 。

和往常一样，我们有很多内容需要介绍，所以让我们开始吧。

  • 在Spring Cloud中新增了对自动配置基于Reactor的Cloud Foundry Java客户端的支持 。这是令人兴奋的对我来说，因为Java客户端就是这样一个强大的工具，需要很长时间来进行配置。
  • 社区传奇人物ErdemGünay提供了有关使用Spring Cloud Contract的新指南。
  • Baeldung博客很好地介绍了如何使用Spring引导Hibernate 5
  • 宣布了第一批春季活动 ！千万不要错过在一个城市这些事件在您附近！我会和其他Pivotal Peeps一起去。加入我们。
  • 用于Pivotal Cloud Foundry的Apigee Edge Service Broker 3.0刚刚发布。
  • 查看有关使用Spring Cloud Contract的西班牙语帖子 。
  • Resilience4J项目是一个有趣的外观库，其中包含断路器和重试处理程序之类的实现，并且具有外观美观的Spring Boot集成。我希望有一天Spring Cloud能够整合这一点。同时，将其签出。
  • Apache Geode负责人John Blum的 Epic Spring Data 接受了史诗般的《软件工程日报》播客的采访 ：听一下。
  • Dan Vega有一个有趣的视频， 介绍了在Windows环境中安装Spring Boot CLI的过程。
  • 在上个月史诗般的SpringOne Platform 2017活动期间，我接受了Faros Belgium的采访 。
  • Cloud Foundry Foundation安全团队针对最近在CPU中发现的Spectrum和Meltdown安全漏洞，详细介绍了Cloud Foundry用户的响应 。这些漏洞正是 Cloud Foundry这样的集中式平台可以在世界上产生重大影响的地方-集中缓解问题并自动部署。
  • Tristan Perry 编写了一个不错的Spring Cloud微服务教程 。
