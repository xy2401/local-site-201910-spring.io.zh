Spring Session 2.0.0发布

发布
罗伯·温奇
一月16，2018

该帖子由VedranPavić撰写

我很高兴代表社区宣布Spring Session 2.0.0的发布。发布 。此版本通过2.0.0演变。M1 ， 2.0.0。M2 ， 2.0.0。M3 ， 2.0.0。M4 ， 2.0.0。M5 ， 2.0.0。RC1 ， 2.0.0。RC2和2.0.0。RELEASE ，关闭超过130个问题并拉取请求。

Spring Session 2.0的新增功能

您可以在参考资料的“新功能2.0”部分中找到新功能的重点 。有关详细信息，请参阅上面的变更日志链接。

要求

此发行版移至Java 8和Spring Framework 5.0作为基线要求。整个代码库现在基于Java 8源代码级别。

Spring WebFlux支持

在Spring Framework 5.0中引入反应式编程模型一直是2017年最大的故事之一，我们很高兴为使用Redis ReactiveSessionRepository来管理Spring WebFlux的WebSession提供支持。

以一种熟悉的Spring Session方式，启用Redis支持的WebSession支持就像以下代码片段一样简单：

@EnableRedisWebSession
public class WebSessionConfig {

    @Bean
    public LettuceConnectionFactory redisConnectionFactory() {
        return new LettuceConnectionFactory();
    }

}
