  • Spring Cloud Skipper 1.0.GA is here! A cross-cloud (Cloud Foundry, Kubernetes) option for continuous delivery/updating/rollback/query of Spring Boot-based applications. Spring Cloud Skipper underpins the recently added support in Spring Cloud Data Flow for versioning and rolling back stream definitions, but it’s also useful in its own right.
  • Spring Cloud Stream lead Oleg Zhurakousky just announced Spring Cloud Stream Elmhurst.M4 /2.0.0.M4 Release . The new release includes performance improvements around content-type resolution, Introduction of Polling Source as an alternative to the event-driven message consumption, StreamListener infrastructure enhancements to primarily deal with multiple destination, continuing documentation improvements, various other enhancements and bug fixes, and more.
  • Pivotal Web Services (PWS) is Pivotal’s hosted, multi-tenant version of Cloud Foundry. Cloud Foundry is based on BOSH which makes easy, centralized upgrades and deployments of new releases as easy as possible. This post looks at the timeline for deploying a new release of the Ubuntu “stemcell” that underpins all running Linux-based applications on PWS after the Spectrum/Meltdown exploit hit. It took less than an hour of engineer time to update the root image, and about 16 hours to see the entire platform updated with rolling deploys, automatically, after deploying the updated stemcell change. During this time, service was not disrupted for any of the countless applications deployed to the platform.

This release includes an important fix for security vulnerability CVE-2018-1196, which can affect anyone using Spring Boot’s systemd and init.d service support. It also provides the latest version of Spring Security which fixed CVE-2018-1199. In addition the security fix, Spring Boot 1.5.10 includes over 55 fixes, improvements and dependency updates.

春天的这个星期-2018年2月6日

工程
乔什·朗（Josh Long）
2018年2月6日

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我在瑞典斯德哥尔摩参加史诗（又下雪）的JFokus活动，然后又去了德国法兰克福参加OOP表演。如果您在任何一个活动中，都不要犹豫，在Twitter（@starbuxman）上打招呼！

  • Spring Cloud Skipper 1.0。GA在这里！跨云（Cloud Foundry，Kubernetes）选项， 用于基于Spring Boot的应用程序的持续交付/更新/回滚/查询 。Spring Cloud Skipper支持最近在Spring Cloud Data Flow中添加的对版本控制和回滚流定义的支持，但它本身也很有用。
  • Spring Cloud Stream负责人Oleg Zhurakousky刚刚宣布了Spring Cloud Stream Elmhurst。M4 / 2.0.0。M4发布 。新版本包括围绕内容类型解析的性能改进，轮询源简介（替代事件驱动的消息使用）， StreamListener基础架构增强功能，主要用于处理多个目标，持续的文档改进，各种其他增强功能和错误修复等。
  • Pivotal Web服务（PWS）是Pivotal托管的Cloud Foundry的多租户版本。Cloud Foundry基于BOSH，它使轻松，集中的升级和新版本的部署变得尽可能容易。这篇文章着眼于部署新版Ubuntu“ stemcell”的时间表，该新版本在Spectrum / Meltdown攻击受到打击后，将支持PWS上所有正在运行的基于Linux的应用程序。在部署更新的干细胞变更之后，工程师花费了不到一个小时的时间来更新根映像，并且花费了大约16个小时才能看到整个平台通过滚动部署自动更新 。在此期间，对于部署到平台的无数应用程序，服务都没有中断。
  • 查看最新版本的Spring Tool Suite 4中的新功能
  • 萨比·阿南丹（Sabby Anandan）在刚发布的新功能中发表了精彩的文章
    Spring Cloud Data Flow 1.3支持持续交付，功能和改进的可用性等
  • 想知道Spring Boot中的Spring Social自动配置发生了什么？它们已被提取到Spring Social正式版中！有关更多信息，请查看此Github问题
  • Spring IO Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform“ Brussels” SR7 ，它将升级该平台中许多项目的版本，包括Spring AMQP 1.7.6，Spring Boot 1.5.10，Spring Data Ingalls-SR10，Spring Framework 4.3.14，Spring Integration 4.3.14，Spring Retry 1.2.2，Spring Security 4.2.4和Spring Web Flow 2.4.7。
  • Spring Boot主管Phil Webb刚刚宣布了Spring Boot 1.5.10 。该版本包含针对安全漏洞CVE-2018-1196的重要修复程序，该漏洞可能会影响使用Spring Boot的systemd和init.d服务支持的任何人。它还提供了修复CVE-2018-1199的最新版本的Spring Security。除了安全修补程序外，Spring Boot 1.5.10还包含超过55个修补程序，改进和依赖项更新。
  • 说到Spring Cloud Skipper，如果您想从源头上了解更多信息，请查看Pollack博士在SpringOne Platform 2017上的精彩演讲
  • 这是Project Reactor团队成员Simon Basle的又一篇精彩文章，他探讨了如何将日志记录库的MDC功能与Reactor自己的功能相结合Context功能 。
  • 这是一个老套的好东西：请查阅Victor Grazi在InfoQ上发布的有关使用Spring Boot构建基于REST的微服务的文章。
  • RabbitMQ 3.7有什么新功能 ？好多！继续阅读。
  • 大家好消息！阿里云现在支持本地部署Spring Cloud应用程序 。阿里云是阿里巴巴的一家公司，提供自己的基础设施即服务，主要为中国企业提供服务。这篇中文帖子解释了所有内容。
  • 很酷的一点是，我和我的朋友Kenny Bastani共同开发的这本书的免费样本Cloud Native Java是2017年Pivotal最受欢迎/下载量最大的电子书资产 。
  • 这是使用Spring Security进行无密码身份验证的一个很酷的示例。这个想法很简单，但是代码复杂：您输入电子邮件或用户名，系统向您发送电子邮件，然后单击要登录的电子邮件。

  • 社区传奇人物Aboullaite Mohammed 在Project Reactor，Spring Framework 5，Spring Boot 2.0，Spring Data Kay和Spring Web Flux的响应支持下，撰写了一篇有关跟踪实时飞机状态的出色文章。不错的工作！

  • 查看最新的Oracle Java Magazine分期付款。您将找到有关所有最新和最新的反应技术（Vert.x，RxJava 2，Reactor）的内容，包括您真正在使用Spring Boot 2.0，Project Reactor，Spring Data Kay等方面的文章 。
  • Spring Cloud Data Flow 1.3。GA已发布 ！新版本包括流更新和回滚，Java DSL和改进的基于Angular的UI。
