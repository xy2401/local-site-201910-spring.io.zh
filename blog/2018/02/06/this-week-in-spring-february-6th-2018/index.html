Spring 的这个星期-2018年2月6日

工程
乔什·朗（Josh Long）
2018年2月6日

嗨， Spring 粉丝！欢迎来到 Spring 本周的另一期！本周，我在瑞典斯德哥尔摩参加史诗（又下雪）的JFokus活动，然后又去了德国法兰克福参加OOP表演。如果您在任何一个活动中，都不要犹豫，在Twitter（@starbuxman）上打招呼！

  • Spring Cloud Skipper 1.0。GA在这里！跨云（Cloud Foundry，Kubernetes）选项， 用于基于Spring Boot的应用程序的持续交付/更新/回滚/查询 。Spring Cloud Skipper支持最近在Spring Cloud Data Flow中添加的对版本控制和回滚流定义的支持，但它本身也很有用。
  • Spring Cloud Stream负责人Oleg Zhurakousky刚刚宣布了Spring Cloud Stream Elmhurst。M4 / 2.0.0。M4发布 。新版本包括围绕内容类型解析的性能改进，轮询源简介（替代事件驱动的消息使用）， StreamListener基础架构增强功能，主要用于处理多个目标，持续的文档改进，各种其他增强功能和错误修复等。
  • Pivotal Web服务（PWS）是Pivotal托管的Cloud Foundry的多租户版本。Cloud Foundry基于BOSH，它使轻松，集中的升级和新版本的部署变得尽可能容易。这篇文章着眼于部署新版Ubuntu“ stemcell”的时间表，该新版本在Spectrum / Meltdown攻击受到打击后，将支持PWS上所有正在运行的基于Linux的应用程序。在部署更新的干细胞变更之后，工程师花费了不到一个小时的时间来更新根映像，并且花费了大约16个小时才能看到整个平台通过滚动部署自动更新 。在此期间，对于部署到平台的无数应用程序，服务都没有中断。
  • 查看最新版本的Spring Tool Suite 4中的新功能
  • 萨比·阿南丹（Sabby Anandan）在刚发布的新功能中发表了精彩的文章
    Spring Cloud Data Flow 1.3支持持续交付，功能和改进的可用性等
  • 想知道Spring Boot中的Spring Social自动配置发生了什么？它们已被提取到Spring Social正式版中！有关更多信息，请查看此Github问题
  • Spring IO Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform“ Brussels” SR7 ，它将升级该平台中许多项目的版本，包括Spring AMQP 1.7.6，Spring Boot 1.5.10，Spring Data Ingalls-SR10，Spring Framework 4.3.14，Spring Integration 4.3.14，Spring Retry 1.2.2，Spring Security 4.2.4和Spring Web Flow 2.4.7。
  • Spring Boot主管Phil Webb刚刚宣布了Spring Boot 1.5.10 。该版本包含针对安全漏洞CVE-2018-1196的重要修复程序，该漏洞可能会影响使用Spring Boot的systemd和init.d服务支持的任何人。它还提供了修复CVE-2018-1199的最新版本的Spring Security。除了安全修补程序外，Spring Boot 1.5.10还包含超过55个修补程序，改进和依赖项更新。
  • 说到Spring Cloud Skipper，如果您想从源头上了解更多信息，请查看Pollack博士在SpringOne Platform 2017上的精彩演讲
  • 这是Project Reactor团队成员Simon Basle的又一篇精彩文章，他探讨了如何将日志记录库的MDC功能与Reactor自己的功能相结合Context功能 。
