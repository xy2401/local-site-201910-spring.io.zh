Spring Cloud Skipper 1.0 GA发布

发布
马克·波拉克
2018年2月5日

我很高兴代表团队宣布Spring Cloud Skipper 1.0 GA的发布

Skipper是一款轻巧的工具，可让您发现Spring Boot应用程序并在多个Cloud Platform上管理其生命周期。您可以独立使用Skipper或将其与Continuous Integration管道集成在一起，以帮助实施Continuous Deployment的实践。

参考指南中的“ 入门”部分是开始踢轮胎的最佳位置。

发布要点：

  • 引入了用于管理架构的Flyway以及各种架构调整。

  • 选择删除版本及其软件包。

  • 完善了REST API。

  • 将属性更新为YAML转换器。

  • 在清单模板中添加资源元数据。

  • 将平台部署者分成多个Maven模块。

  • 支持传递给shell命令执行。

  • 更新了文档。

  • 各种错误修复。

下一步是什么？

1.0 GA功能集主要是由Spring Cloud Data Flow部署消息驱动的应用程序的需求驱动的。未来版本计划改进在升级/回滚期间管理http路由的方式，并支持Cloud Foundry应用程序清单的模板化。

请在GitHub Issues ， Stack Overflow和Gitter频道上联系问题，反馈或贡献。
