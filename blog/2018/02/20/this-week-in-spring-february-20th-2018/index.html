春天的这个星期-2018年2月20日

工程
乔什·朗（Josh Long）
2018年2月20日

嗨，春季粉丝！欢迎来到春季本周的另一期！ 本周，我将与马里奥·格雷（Mario Gray）在圣地亚哥水罐公园（JUG）进行春季测试。然后，我将在旧金山召开的IBM Index会议上 ，讨论构建反应式微服务，然后在佐治亚州亚特兰大举行的Devnexus上，我将谈论Kotlin和测试。我希望你和我在一起，如果你在附近，打个招呼 。

  • StéphaneNicoll刚刚宣布了Spring Framework 5.0.4 。
  • 查看有关构建灵活的Spring Cloud Data Flow数据管道的白皮书
  • 现在，使用Spring Initializr中的 Spring Cloud Stream甚至更加容易：现在，您只需要选择一个绑定器实现（Kafka或RabbitMQ），然后选择是否希望该实现是反应式的就可以了。 ！
  • Spring Cloud Task主管Michael Minella刚刚宣布了Spring Cloud Task 2.0.0。M3。新版本包括应用程序关闭时更智能的默认行为，限制任务并发运行的功能以及与Spring Batch的更智能集成
  • 我喜欢Oracle开发人员博客中 Abhishek Gupta的这篇不错的文章，内容涉及使用Zipkin进行分布式跟踪
  • 查看Mark Heckler和我在去年的SpringOne Platform 2017上发表的演讲的录音，以了解有关构建响应式Spring应用程序的信息
  • Apache Tomcat 9.0.5刚刚发布
  • 我喜欢SpringUni社区博客上的这篇文章，该博客介绍了Spring Boot 2.0中的一些新功能。
  • 各种Docker构建插件的最大问题是它们采用了稳定的API，这是一个很大的错误！这篇文章中的方法概述使用了docker指挥自己 。
  • 这是一篇不错的文章，介绍了HATEOAS，模式和Spring HATEOAS（技术）
  • 这篇文章探讨了如何预处理使用Spring Cloud Feign创建的请求
  • 这是一篇简短而有趣的文章，介绍了从头开始创建简单的Spring Boot API的过程
  • 社区的朋友Aboullaite Mohammed 组装了一个Spring Boot启动程序，该程序可以自动配置Minio客户端 。Minio是高性能的分布式对象存储服务器，专为大型私有云基础架构而设计。
  • Grails框架十周年快乐！
  • 该博客介绍了如何在Spring Boot应用程序中设置多个MongoDB连接
  • 法国兴业银行（SociétéGénérale）组装了一个Spring Boot启动器，该启动器支持某些更细微的RabbitMQ用法 -也许这里有您想要的东西？
  • Spring and Cloud Foundry成员Brian McClain整理了一个快速视频，展示了如何在GKE上安装Pivotal的功能即服务产品Riff项目 。
  • Baeldung在使用Dubbo上发表了一篇不错的文章， Dubbo是来自阿里巴巴的基于Spring Framework 的RPC框架 ，多年来一直是阿里巴巴（及其他公司）大量服务的基础。
  • 我们正在招聘！我们正在寻找产品营销经理
  • 是否想通过Reactor项目来实现反应式API的前座之旅？查看反应堆项目两位杰出人物Ben Hale和Paul Harris的演讲

  • Spring和Kotlin 忍者SébastienDeleuze已使用Kotlin更新了出色的Spring Petclinic应用程序
  • 大家好消息！MongoDB支持事务。Spring Data负责人Oliver Gierke 推文说 ：“我能闻到MongoDbTransactionManager…#SpringData” :)
  • InfoQ的Richcard Seroter主持了有关事件外包的讨论，并与Axon项目负责人Allard Buijze进行了对话。Axon是一个框架，用于在Spring上构建以事件驱动，以CQRS为中心的，具有事件源意识的应用程序。
