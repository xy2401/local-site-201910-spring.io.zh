Spring Cloud Data Flow 1.4.0 M1发布

发布
冈纳·希尔特（Gunnar Hillert）
2018年2月27日

Spring Cloud Data Flow团队很高兴宣布发布1.4.0 M1 。遵循本地服务器 ， Cloud Foundry和Kubernetes的入门指南 。

第一个里程碑发布的重点是1.4.x行一直是Dashboard UI 。为了始终保持UI依赖关系的最新状态，我们最终将UI升级到Angular 5.2和Angular CLI 1.6 。而且，UI现在提供了与Spring Cloud Skipper更好的集成。

这样，您现在可以管理“应用程序注册”的版本并设置默认版本。

现在，版本信息也显示在“应用程序注册”列表页面上。除了提供有关应用程序类型 （源，接收器，任务，处理器）的更好的视觉效果外，您现在还可以按应用程序类型搜索应用程序注册。

此外，在浏览“应用程序注册”详细信息时，您可以轻松选择想要查看其详细信息的版本。

现在，在部署流时，您可以将它们部署到多个平台。

最后，在此版本中，我们还完善了参考文档，并在文档中记录了UI对命名目的地的支持， 以方便进行扇入/扇出场景。

与往常一样，我们欢迎您提供反馈和意见，因此请通过Stackoverflow或GitHub或通过Gitter与我们联系。

请尝试一下，分享您的反馈，并考虑为该项目做出贡献！
