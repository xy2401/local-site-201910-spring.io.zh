春天的这个星期-2018年2月27日

工程
乔什·朗（Josh Long）
2018年2月28日

嗨，Spring粉丝们，欢迎来到Spring Tips的另一期！这是一个超级刺激的一周！Spring Boot 2.0即将发布！密切关注Spring Initializr，否则您将错过它！ ：D

今天，在我的伙伴Matt Raible的帮助下 ，我参加了Okta Iterate会议，与正在使用Spring和Okta的开发人员进行了交谈 。高点？我遇到了Stack Overflow的共同创造者Jeff Atwood ！

明天，我将在接下来的两周内进行旋风之旅。首先，去苏格兰的格拉斯哥。然后是澳大利亚的悉尼；然后是迪拜；然后是印度的班加罗尔（ 适用于Agile India 2018 ）;然后在3月13日前往马萨诸塞州的波士顿参加SpringOne Tour活动 。如果您在这些地方中的任何一个， 请随时与我们联系！我很高兴看到您并听到您的消息！

  • 我喜欢这个介绍Project Riff的博客-Pivotal的功能即服务。
  • Spring Cloud Stream负责人Oleg Zhurakousky 刚刚宣布了Spring Cloud Stream Elhurst RC1和2.0.0。RC1 。新版本带来了一系列功能（可轮询的使用者， @StreamListener消息转换配置等）以及多项增强功能（可选地支持基于Spring WebFlux或Servlet的自动配置，在Spring Initializr上更方便地安排选项）以及没有默认的日志记录）。
  • Spring Security忍者Joe Grandja刚刚宣布了Spring Security 2.3.0。RC1在JwkTokenStore添加了对椭圆曲线签名验证的新支持。此版本还包括一些小的增强功能和错误修复。
  • 想更多地了解如何将Spring Cloud Contract与Kotlin结合使用 ，请查看引发这一切的问题。目前，Spring Cloud Kotlin定义是在Groovy中编写的。这种新的DSL增加了受众。这是对Spring Cloud Contract和.yml属性的最新支持的不错补充。
  • Spring Integration的忍者Artem Bilan在 Spring Integration 的最新版本5.0.2上有一篇不错的文章，该版本现在支持Micrometer-metrics出版物 ，更新的UnicastReceivingChannelAdapter ，对IntegrationGraphController支持以及MessageChannel的动态过滤器配置。看看这个！
  • Spring Boot传奇人物Andy Wilkinson刚刚宣布了Spring Boot 2.0.0。RC2。这是非常迫在眉睫2.0之前的最后一个版本。GA发布。取得点点滴滴，立即尝试！或者..只需等待几个小时或几天，您就可以使用Spring Boot 2.0。GA
  • Spring Security负责人Rob Winch 刚刚宣布了Spring Security 5.0.2 ），其中包含大量错误修复
  • Spring Data忍者和Spring Vault负责人Mark Paluch 刚刚宣布了Spring Vault 2.0。GA 。该版本升级到Java 8和Spring Framework 5，提供了JSR-305annotation的API，提供了对Java安全的支持，通过Spring Data KeyValue存储库提供了Vault存储库支持，Kubernetes / AWS ECS / IAM身份验证， RoleId / SecretId为AppId身份验证SecretId展开，Spring Security与VaultBytesEncryptor和VaultRandomBytesKeyGenerator等的集成！
  • Matt Raible更新了他的截屏视频，内容涉及如何开始使用Spring Boot，SAML和@okta。正如他所说：“由于@SpringSecurity的SAML DSL项目，您可以在几分钟内完成！”
  • 确保检查出Java 11的早期版本
  • Spring Cloud Finchley。M7已发布。这个新版本对Spring Cloud Gateway进行了增强，Feign获得了一个顶级项目等等。
