适用于Google Cloud Platform 1.0的Spring Cloud里程碑2可用

发布
阿尔特姆·比兰
2018年2月7日

亲爱的春季社区！

今天我很高兴宣布Spring Cloud系列中的一个新项目。叫做Spring Cloud GCP其目标是将众所周知的Spring模式和Spring Boot约定引入您的应用程序，以使用Google Cloud Platform服务。

该项目当前为版本1.0.0.M2可以从Spring Milestone存储库中获得 ：


   
       
           org.springframework.cloud
           spring-cloud-gcp-dependencies
           1.0.0.M2
           pom
           import

这个项目是作为Google和Spring团队之间合作的实验而启动的。到目前为止，我们已经取得了良好的经验，我们的团队间知识共享为我们带来了几个稳定的模块并由Google团队做出了积极贡献的项目。

该项目通过以下模块为Google Cloud Platform服务提供了集成和访问对象：

Spring Cloud GCP核心

一流的公民抽象像GcpProjectIdProvider和CredentialsProvider及其支持课程。提供属性并配置对Project ID和Cloud Authentication的访问 。这是所有其他模块的传递依赖。

spring-cloud-gcp-autoconfigure和spring-cloud-gcp-starters

该项目提供的组件的Spring Boot自动配置和启动程序，或者仅是现有自动配置的包装程序和帮助程序，例如DataSourceAutoConfiguration 。

Spring Cloudgcp-pubsub

着名的Template和Factory Google Cloud Pub / Sub服务的春季模式实施。此外，该模块还包括围绕的Spring Integration Channel Adapters实现。 PubSubTemplate ， PublisherFactory和SubscriberFactory 。

spring-cloud-gcp-storage

的org.springframework.core.io.ProtocolResolver Google Cloud Storage资源解析的实施。加上Spring Integration Channel Adapters来进行读/写操作Blob和Bucket资源。

spring-cloud-gcp-logging

该模块提供了将Web请求跟踪ID与相应的日志条目相关联的支持。这允许按请求对日志消息进行分组。

春天的云gcp跟踪

Google Cloud Stackdriver Trace的Spring Cloud Sleuth组件实现。

spring-cloud-gcp-pubsub-stream-binder

春季云流 Binder Google Cloud Pub / Sub服务的实施。

请参阅《 参考手册》中有关这些和其他项目功能的更多信息。还请注意项目中提供的全面的样本集。此外，我们还为您提供了《 使用Google Cloud Pub / Sub进行消息传递》入门指南。

最后，不要错过有关此项目的Google公告 ！

即将有更多Google Cloud Platform服务支持和Spring组件实现！
