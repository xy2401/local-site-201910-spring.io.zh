Spring 的这个星期-2018年6月26日

工程
乔什·朗（Josh Long）
六月26，2018

嗨，春季粉丝们，欢迎来到春季本周的另一期！上周很有趣！自从上次发言以来，我去过伦敦的阿姆斯特丹（客户），伦敦（伦敦的SpringOne Tour）；巴黎神父（JHipster Conf）；克拉科夫（PL）（Devoxx PL）；现在是巴西圣保罗（Spring Connect SP）。这是一个疯狂的旅程，很高兴与世界上最大的Java社区之一一起结束在巴西的旅程。

而且，作为顶级的樱桃，在本周的综述中有大量值得关注的东西，所以让我们开始吧！

  • Spring Data Geode / Gemfire负责人John Blum刚刚宣布了Apache Geode和Pivotal Gemfire 1.0.0的Spring Boot集成。M1 。它支持使用Spring的缓存抽象进行后备缓存，使用Spring数据存储库和Spring事务管理的记录系统（SOR），使用函数实现和执行的分布式计算，使用连续查询的事件流处理（ESP），使用域对象的版本化/序列化使用SSL的PDX，安全性，身份验证/授权和TLS。
  • Spring Data Geode / Gemfire的负责人John Blum 宣布了Apache Geode / Pivotal Gemfire 2.0.3的春季会议。发布
  • 看看这个最小的Spring Fu Webapp，它现在可以通过Graal 1.0.0 成功编译为本地映像 。RC2和Kotlin 1.2.60（也将与1.2.51一起使用）和Netty！在我公认的相当快的Linux / Dell Precision 5520上，使用Graal的启动时间小于3毫秒（与Java 10的900毫秒相比）。自给自足的可执行文件大小为50 MB，但是我们有充分的理由认为我们可以减小它。
  • 嗨，春季粉丝！我们已经正式结束了Spring Tips的最新季节，第4季 。如果您想了解更多信息，则应查阅此概述文章，其中列出了迄今为止的每期Spring Tips 。一如既往，我渴望最新的最好的建议 。不用担心，我们会回来的！ :)
  • 既然Spring Cloud Finchley GA出来了，我想我应该总结一下响应式Spring Cloud的发展道路。看一下这个
  • 大家好消息！由于Vaadin 自己的Matti Tahvonen，Spring.io / guides 上的Vaadin指南已更新为Vaadin 10。
  • 这太酷了！Project Blockhead是兼容开放服务代理的实现，也支持以太坊 ，这是黑客马拉松的结果！
  • Stephane Nicoll和Brian Clozel的演讲“ Spring Boot 2.0 Web Applications”现在可以在线获得。这是视频 ，这是代码 。

  • 我喜欢Spring Test主管和JUnit主管Sam Brannen对常规Spring组件中自动装配与在JUnit测试类中使用自动装配的区别的解释。
