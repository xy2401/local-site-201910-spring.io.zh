Spring Cloud GCP 1.0.0。RC1现在可用

发布
阿尔特姆·比兰（Artem Bilan）
六月28，2018

亲爱的春季社区！

今天，我很高兴与Google团队一起宣布Spring Cloud for Google Cloud Platform项目1.0版的第一个候选版本。

通过BOM可从Spring Milestone存储库中获得的工件：


    
        
            org.springframework.cloud
            spring-cloud-gcp-dependencies
            1.0.0.RC1
            pom
            import

从先前宣布的Milestone 3开始 ，Spring Cloud GCP基于最新的Spring Cloud Finchley，并提供以下功能：

讯息传递

所有Spring Messaging支持都位于spring-cloud-gcp-pubsub和spring-cloud-gcp-pubsub-stream-binder模块中，并基于Spring Integration和Spring Cloud Stream 。有PubSubTemplate用于与Google Cloud Pub / Sub进行低级普通消息传递。基于PubSubTemplate基础的Spring Integration Channel Adapters是PubSubInboundChannelAdapter和PubSubMessageHandler 。在该发行候选，我们实现了用于经由他们一致的消息转换PubSubMessageConverter抽象和还添加通过报头映射PubSubHeaderMapper 。

通过基于Spring spring-cloud-gcp-autoconfigure模块中提供的Spring Boot自动配置的PubSubMessageChannelBinder通过平滑的绑定属性来表示Spring Cloud Stream集成。

数据库

Google Cloud SQL支持是作为spring-cloud-gcp-autoconfigure模块中提供的普通Spring Boot自动spring-cloud-gcp-autoconfigure实现的，其他所有内容都只是标准的Spring JDBC支持！spring-cloud-gcp-starter-sql-mysql和spring-cloud-gcp-starter-sql-postgresql旨在帮助您透明地自动配置Spring Boot应用程序中的Google Cloud SQL服务。

档案储存

当您可以使用gs://前缀经典Spring Resource抽象引用Google Cloud资源时，可以通过GoogleStorageProtocolResolver实现Google Cloud Storage集成。此外，还提供了GcsInboundFileSynchronizingMessageSource, `GcsStreamingMessageSource和GcsMessageHandler Spring集成通道适配器，用于通过消息处理Google Cloud资源。

记录中

Google Cloud Stackdriver Logging集成由各种自动配置的utils表示，可以使用我们预定义的logback-appender.xml和logback-json-appender.xml模板在您的日志记录配置中启用。spring-cloud-gcp-starter-logging Spring Boot Starter有助于管理应用程序的适当依赖关系。

跟踪

Spring Cloud GCP与Spring Cloud Sleuth一起提供了Google Cloud Stackdriver Trace的集成。spring-cloud-gcp-starter-trace可帮助您自动配置适当的Spring Cloud和Open Zipkin依赖关系以进行集成。

认证方式

为了对您的Google Cloud Accounts进行正确的身份验证，通过Credentials和GcpProjectIdProvider抽象提供了Spring Boot自动配置。

试试看！

在Spring Cloud GCP GitHub存储库中检查代码示例 ，阅读Spring Cloud GCP参考文档了解更多详细信息，并在代码实验室上进行尝试。最重要的是，在我们计划发布GA的过程中，请让我们知道您的反馈并报告任何问题！同时，我们已开始在将来为您带来更多集成。目前，您可以在进行中的版本1.1试用我们对Spring Data Cloud Spanner和Spring Cloud Config的 beta支持。

来自Spring Cloud GCP团队

千万不要错过SpringOne上平台会议在华盛顿特区，今年九月！我将与Google团队一起在Spring Cloud for Google Cloud Platform项目中介绍这些功能以及其他更多功能。结帐会话并注册！
