Spring Cloud GCP 1.0.0。RC1现在可用

发布
阿尔特姆·比兰（Artem Bilan）
六月28，2018

亲爱的 Spring 社区！

今天，我很高兴与Google团队一起宣布Spring Cloud for Google Cloud Platform项目版本的第一个候选版本。 1.0 。

通过BOM可从Spring Milestone存储库中获得的工件：


    
        
            org.springframework.cloud
            spring-cloud-gcp-dependencies
            1.0.0.RC1
            pom
            import

从先前宣布的Milestone 3开始 ，Spring Cloud GCP基于最新的Spring Cloud Finchley，并提供以下功能：

讯息传递

所有Spring Messaging支持都位于spring-cloud-gcp-pubsub和spring-cloud-gcp-pubsub-stream-binder模块和基于Spring Integration和Spring Cloud Stream的模块 。有PubSubTemplate使用Google Cloud Pub / Sub进行低级别的普通消息传递。Spring Integration通道适配器，基于PubSubTemplate基础，是PubSubInboundChannelAdapter和PubSubMessageHandler 。在此版本候选中，我们通过以下方式为他们实现了一致的消息转换PubSubMessageConverter抽象，还通过添加了标头映射PubSubHeaderMapper 。

Spring Cloud Stream集成通过以下方式表示PubSubMessageChannelBinder具有平滑的绑定属性，该属性基于以下内容中提供的Spring Boot自动配置spring-cloud-gcp-autoconfigure模块。

数据库

Google Cloud SQL支持的实现是作为spring-cloud-gcp-autoconfigure模块和其他所有内容都只是标准的Spring JDBC支持！的spring-cloud-gcp-starter-sql-mysql和spring-cloud-gcp-starter-sql-postgresql旨在帮助您在Spring Boot应用程序中透明地自动配置Google Cloud SQL服务。

档案储存

Google云端存储集成是通过以下方式实现的GoogleStorageProtocolResolver ，当您可以通过gs://前缀古典之春Resource抽象。此外， GcsInboundFileSynchronizingMessageSource, `GcsStreamingMessageSource和GcsMessageHandler提供了Spring Integration Channel Adapter，用于通过消息处理Google Cloud资源。

记录中

Google Cloud Stackdriver Logging集成由各种自动配置的utils表示，可以使用我们的预定义功能在您的日志记录配置中启用logback-appender.xml和logback-json-appender.xml模板。的spring-cloud-gcp-starter-logging Spring Boot Starter帮助您管理应用程序的适当依赖关系。

跟踪

与Spring Cloud Sleuth一起，Spring Cloud GCP为Google Cloud Stackdriver Trace提供了集成。的spring-cloud-gcp-starter-trace帮助您自动配置适当的Spring Cloud和Open Zipkin依赖关系以进行集成。

认证方式

为了对您的Google Cloud帐户进行正确的身份验证，可通过以下方式提供Spring Boot自动配置Credentials和GcpProjectIdProvider抽象。

试试看！

在Spring Cloud GCP GitHub存储库中查看代码示例 ，阅读Spring Cloud GCP参考文档了解更多详细信息，并在代码实验室上进行尝试。最重要的是，在我们计划发布GA的过程中，请让我们知道您的反馈并报告任何问题！同时，我们已开始在将来为您带来更多集成。目前，您可以在进行中的版本中试用我们对Spring Data Cloud Spanner和Spring Cloud Config的 beta支持1.1 。

来自Spring Cloud GCP团队

不要错过今年9月在华盛顿特区举行的SpringOne Platform会议！我将与Google团队一起在Spring Cloud for Google Cloud Platform项目中介绍这些功能以及更多其他功能。结帐会话并注册！
