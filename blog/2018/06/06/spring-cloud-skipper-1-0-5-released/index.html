Spring Cloud Skipper 1.0.5发布

发布
马克·波拉克
六月06，2018

我很高兴代表团队宣布发布Spring Cloud Skipper 1.0.5 GA

Skipper是一款轻巧的工具，可让您发现Spring Boot应用程序并在多个Cloud Platform上管理其生命周期。您可以独立使用Skipper或将其与Continuous Integration管道集成在一起，以帮助实施Continuous Deployment的实践。

参考指南中的“ 入门”部分是开始踢轮胎的最佳位置。

这主要是一个错误修复版本。自1.0 GA版本以来的重大更改是：

  • 通过简化状态机序列化的对象，改进了状态管理。

  • 改进了升级和回滚期间的超时处理。

  • 套餐升级现在可以取消。

  • 改进了在生成的YAML中特殊字符的转义。

  • 将部署程序库升级到最新版本。

  • 更新到Spring Shell 2.0.1。

请在GitHub Issues ， Stack Overflow和Gitter频道上联系问题，反馈或贡献。
