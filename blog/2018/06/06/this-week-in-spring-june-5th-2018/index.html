    春天的这个星期-2018年6月5日

    工程
    乔什·朗（Josh Long）
    六月06，2018

    嗨，春季粉丝！欢迎来到春季本周的另一期！这是一个疯狂而美好的一周。我刚刚从史诗般的VOXXED新加坡活动中回来，只是在春天和充满阳光的旧金山度过了美好的一周假期。我们有一吨的东西覆盖，所以让我们开始吧！

    • Pivotal Cloud Foundry 现在支持Minio ，这是面向开发人员的高可用的类似S3的HA企业级对象存储服务器。
    • Spring Cloud Finchley GA计划于6月15日发布
    • 在“应该是微服务吗？”的第4部分中Nate Schutta 谈到了独立可伸缩性及其重要性
    • Spring Batch忍者Mahmoud Ben Hassine刚刚宣布了Spring Batch 4.1.0。M1包括简化的测试和更容易的远程分块 。立即获取位！
    • 在上周的Spring Tips中，我们研究了JDBC，这不仅是古老的JdbcTemplate ！在这篇文章中，我们来看JdbcTemplate ，各种SQL命令对象， SimpleFlatMapper ，还有更多。看看这个！
