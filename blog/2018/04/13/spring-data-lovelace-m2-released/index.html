Spring Data Lovelace M2发布

发布
马克·帕鲁奇
2018年4月13日

我谨代表Spring Data团队宣布Lovelace发布火车第二个里程碑的可用性。

此版本随附所有商店模块的更新，其中包含新功能，改进和错误修复。显着的变化包括：

  • MongoDB 3.6对变更流的支持， $jsonSchema和客户会话（为MongoDB事务做准备）。
  • 针对地图和元组类型的Apache Cassandra映射改进，以及新引入的生命周期回调。
  • 升级到Apache Solr 7。
  • 改进的连接处理能力，可响应式Redis和静态主/从使用。
  • SCAN支持Redis Cluster。
  • List支持Spring Data JDBC。
  • 通过CDI支持可组合存储库。

在Lovelace M2的过程中，当我们从SolrJ 6.6.2升级到7.2.1时，我们决定将Spring Data Solr增加到新的主版本号4.0 M2。

注意未来几天有关MongoDB事务支持的后续博客文章 。
