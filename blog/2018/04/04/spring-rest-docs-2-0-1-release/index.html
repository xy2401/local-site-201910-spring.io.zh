Spring REST Docs 2.0.1。发布

发布
安迪·威尔金森
2018年4月4日

我很高兴宣布Spring REST Docs 2.0.1。可以从Maven Central，JCenter和我们的发行版存储库中获得RELEASE。感谢所有通过报告错误和打开请求请求对此版本做出贡献的人。

该维护版本包括12个错误修复，文档改进和依赖项升级 。建议所有Spring REST Docs 2.x用户进行升级。
