春天的这个星期-2018年4月10日

工程
乔什·朗（Josh Long）
2018年4月11日

嗨，春季粉丝！这周我在亚特兰大，印第安纳波利斯和辛辛那提！我将于10月10日星期二在佐治亚州亚特兰大举行的春季聚会上发表演讲，我将探讨所有新内容，可引导程序和云原生内容，包括响应式编程，Spring Web Flux，Spring Cloud Finchley，Spring Cloud Function和Project Riff。 。然后去印第安纳波利斯，我将在12日星期四做同样的演讲 。

多么疯狂的一周！这个星期有很多事情要做。首先，我想指出的是，这里有一些CVE，如果您还没有的话，应该立即进行相应的更新。继续读以获取更多信息。

其次，我们令人惊奇的文艺复兴时期的书呆子和疯狂的科学家安迪·威尔金森 （ Andy Wilkinson）在本周的大多数公告中均名列前茅。多么了不起的人和多产的工程师！一周又一周又一年又一年，他孜孜不倦地致力于支持您在此博客上看到的内容，但是很少见到他如此频繁地博客。我只是想指出他，因为他在某个时候使您的生活变得更好，而您甚至可能没有意识到:)

无论如何，事不宜迟！让我们开始吧！

  • 有一个新的Spring Integration 5.0.4版本 。
  • Spring IO Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform Brussels SR9 ，其中包含Spring Boot和Spring Framework 4.3.16的更新版本。
  • Spring Cloud Stream负责人Oleg Zhurakousky刚刚宣布了Spring Cloud Stream 2.0。新版本包含很多改进， 因此一定要阅读博客 。我还在新功能之一Spring Cloud Streams Kafka Streams上做了Spring Tips文章
  • Spring Boot负责人Phil Webb 刚刚宣布了Spring Boot 1.5.12 ，这是所有Spring Boot用户的推荐更新。它包括更新的Spring Framework版本，而该版本又解决了最近的CVE。
  • Spring Framework忍者Rossen Stoyanchev刚刚发布了更新的Spring Framework 版本，以解决CVE 2018 1270 。尽快升级！
  • Spring IO Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform Cairo-RELEASE 。该平台的Cairo一代建立在Spring Framework 5.0和Spring Boot 2.0之上，并且需要Java 8。
  • 春季IO平台负责人安迪·威尔金森（Andy Wilkinson ）刚刚宣布，春季IO平台项目将从今天（2019年4月9日）起十二个月结束 ，该项目将移至阁楼。直到那时，布鲁塞尔和开罗线的维护版本都将继续发布。
  • Spring Framework忍者Rossen Stoyanchev已宣布修复与Spring Framework相关的多CVE。现在升级！
  • Spring Cloud Stream传奇人物Soby Chacko提醒我们， 示例已更新，以准备发布Spring Cloud Stream 2.0
  • Spring Boot忍者StéphaneNicoll刚刚宣布了Spring Boot 2的第一个维护版本，其中包括160多个修复，改进和依赖项更新。
  • Spring Security忍者Joe Grandja刚刚宣布了Spring Security 5.0.4和4.2.5 。两种版本均主要提供错误修复和依赖版本更新以及一些小的改进。
  • Spring Boot 忍者StéphaneNicoll刚刚发布了Spring Boot 1.5.11 ，其中包括70多个修复程序，改进功能和依赖项更新。
  • Spring Vault负责人Mark Paluch刚刚宣布Spring Vault 1.1.2和2.0.1已经发布 。Bot版本主要提供错误修复和依赖版本更新以及一些小的改进。
  • Spring Data忍者Mark Paluch 刚刚宣布了Ingalls SR11和Kay SR6服务版本 。Ingalls服务版本位于刚刚发布的Spring Framework 4.3.15之上，并为即将推出的Spring Boot 1.5.11版本做准备。Kay服务版本使用Spring Framework 5.0.5，为方便起见，将由Spring Boot 2.0.1使用。
  • Spring REST Docs负责人Andy Wilkinson 宣布了针对所有Spring REST Docs 2.x用户的建议升级 。
  • 而且，Spring REST Docs的负责人Andy Wilkinson并不是一个固步自封的人， 还宣布了一个维护版本，其中包括针对所有Spring REST Docs 1.x用户的一些错误修复和文档改进。
  • 在本年度的第一篇Spring Tips中 （!!）我们看一下Spring Cloud Stream Kafka Streams。Apache Kafka Streams在基于JVM的代码中支持类似Spark的流处理。您可以管理集群，而Kafka可以管理其余的集群。您无需部署专用的Spark集群即可获得使用Apache Kafka Streams进行流处理的好处。Spring Cloud Stream Kafka Streams以典型的简洁Spring Cloud Stream方式简化了配置Spring Cloud Streams和Apache Kafka Streams的过程。在此视频中，我们介绍了流处理的一些细节，包括KStream和KTable ，连接，序列化和反序列化等等。
  • 我喜欢AdrianMarszałek的这篇长篇文章，内容涉及使用Spring Boot 2和Kotlin进行反应式微服务
  • 这是个老歌，但又是个好东西：安德鲁·菲茨杰拉德（Andrew Fitzgerald）有一篇很好的文章，介绍了如何将Spring Cloud Gateway与Cloud Foundry路由服务一起使用
  • 新产品有很多值得爱的地方cf CLI命令。查看此博客以获取所有详细信息 。你知道吗cf现在，在某些情况下，CLI速度提高了700％？
  • Simon Moser 总结了Cloud Foundry和Kubernetes的美好历史 。这是一个非常有趣的阅读！
  • Neo4j传播者Extraordinairre Jennifer Reif在使用Spring Data Neo4j和Spring Boot方面发表了精彩的文章-不要错过这篇文章！
  • 具有传奇色彩的Aboullaite Mohammed撰写了一篇非常酷的文章，介绍了如何将Spring Boot和Micrometer项目与TICK堆栈配合使用：Telegrap / InfluxDB / Chronograph / Kapacitor 。
  • 我喜欢这个帖子。它解释了一个开发者Andrei Chernyshev如何将基于Kotlin语言的Spring Boot应用程序从JVM 8迁移到JVM 10 。
  • Spring Security负责人Rob Winch 在InfoQ上就Spring Security 5中的响应位进行了精彩的演讲
  • 想在SpringOne Platform 2018上发表演讲吗？提交您的演讲tosuit-de-suite！CFP即将关闭！
  • Pivotal的功能即服务平台Project Riff刚刚宣布了Riff 0.0.6 。尽快获取信息！
  • 千分尺主管和计量学硕士乔恩·施耐德（Jon Schneider）有一个很好的例子，将Netflix Atlas项目嵌入Spring Boot应用程序
  • Flowable 6.3.0版本已更新为使用Spring Boot 2.0
  • Java冠军和Java传奇人物Simon Maple一直在写博客，介绍Cloud Foundry上Java应用程序的安全性。看一下这个！
