Spring IO Platform开罗发布

发布
安迪·威尔金森
2018年4月9日

我很高兴地宣布，Spring IO平台Cairo-RELEASE现在可以从Spring版本库和Maven Central中获得。该平台的Cairo一代建立在Spring Framework 5.0和Spring Boot 2.0之上，并且需要Java 8。

发生了什么变化

开罗升级了许多项目的版本：

  • 铋反应堆
  • Spring AMQP 2.0
  • Spring Batch 4.0
  • Spring Boot 2.0
  • Spring Cloud连接器2.0
  • Spring Data 凯
  • Spring Framework 5.0
  • Spring Integration 5.0
  • 春天卡夫卡2.1
  • Spring REST Docs 2.0
  • 春季安全5.0
  • Spring Security OAuth 2.2
  • 春季会议2.0
  • Spring Web Flow 2.5
  • Spring Web Services 3.0

开罗从平台中删除了以下项目：

  • 春季整合流程
  • Spring Integration Java DSL（现已成为Spring Integration的一部分）
  • Spring Integration卡夫卡
  • Spring Integration Splunk
  • Spring Mobile
  • 春季社交Facebook
  • 春季社交LinkedIn

有关已删除或替换的依赖项管理的更多信息，请参考参考文档的相关部分 。

许多第三方依赖项的版本也已更新。有关托管依赖项版本的更多信息，请参考参考文档的相关部分 。

接下来是什么

开罗将成为该平台的最后一代。有关更多信息，请参见寿命终止公告 。感谢在过去四年中使用并为平台做出贡献的每个人。
