Spring Security SAML路线图

工程
罗伯·温奇
2018年3月5日

自从将近9年前启动以来，Spring Security SAML项目就一直是Spring生态系统的组成部分。这个至关重要的项目是弗拉基米尔·舍弗 （ VladimírSchäfer）的不懈努力和贡献而诞生的。我想花时间亲自感谢弗拉基米尔和我们出色的社区所做的不懈努力。没有他们的所有努力，这个项目将不会是今天的事情。

弗拉基米尔（Vladimír），我们令人惊叹的社区和Spring工程团队正计划合作来增强Spring Security SAML，以实现以下主要目标：

  • 确保所有依赖项都是最新的

  • 确保所有Spring Security API都不公开任何依赖关系API

  • 从Spring Security扩展扩展到Spring Security SAML

从今天开始， Filip Hanik将全职致力于Spring Security SAML，以帮助社区及时推进这些目标。如果您想参与其中，请加入我们 。您也可以在此博客的评论中ping我，或在Twitter上@rob_winch和@fhanik ping我们。
