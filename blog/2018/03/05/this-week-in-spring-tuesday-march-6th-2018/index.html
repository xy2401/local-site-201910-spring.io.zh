春天的这个星期-2018年3月6日，星期二

工程
乔什·朗（Josh Long）
2018年3月5日

嗨，春季粉丝们，欢迎来到春季本周的另一期！在我撰写本文时，是在澳大利亚悉尼星期二的清晨，我一直在这里与Pivotal的一些出色客户进行拜访，我现在正准备在短短的六个小时内飞往迪拜的航班，在那里我将拜访更多Pivotal的惊人客户。本周晚些时候，我将在印度班加罗尔举行令人惊叹的印度敏捷大会 ，然后-下周二（周二）早些时候-我将在马萨诸塞州波士顿举行首场SpringOne Tour活动 。如果您在附近 ，请像往常一样打招呼 ！

本周，我们有很多很棒的内容来庆祝Spring Boot 2.0的精彩发布，因此，不费吹灰之力，让我们开始吧！

  • 头晕！Spring Boot联合创始人Phil Webb 刚刚宣布了Spring Boot 2.0 GA ！新版本包含太多新功能，因此您在做什么？到那边去，阅读帖子， 对@SpringBoot团队在Twitter上，并开始
  • 加入3月13日举行的Spring Boot 2.0启动网络研讨会，从源头上学习它！
  • Spring Boot 2.0终于终于来了！我在此Twitter线程中撰写了有关Spring Boot 2.0的旅程，回顾了2014年4月1日从1.0开始到这一重要版本的Spring Boot的历史。点击并加入我的行列，以#bootiful的方式回访
  • Spring Security主管Rob Winch预览了Spring Security SAML路线图
  • SandraGómez撰写了一篇非常不错的西班牙语文章，内容涉及使用Spring REST Docs记录API
  • Guillermo Tantachuco对Project Riff进行了很好的概述，Riff是Pivotal的新功能即服务产品，在Kubernetes上运行 。
  • Spring Cloud联合创始人Spencer Gibb刚刚宣布了Spring Cloud Finchley M8 ，这是在最近的Spring Boot 2.0版本之后的第一个版本。
  • Spring Security忍者Joe Grandja刚刚宣布了Spring Security OAuth Boot 2.0自动配置已经发布 。它用于帮助用户在旧的Spring Security OAuth 2.x支持与Spring Security 5中的下一代OAuth 2.0支持之间进行转换。它为Spring Security OAuth 2.x的用户提供了基于Spring Boot 1.5.x的基于Spring Boot 2.0的应用程序中相同的自动配置功能。有关更多详细信息，请参阅文档。
  • Spring Security主管Rob Winch提到了Spring Security SAML的状态以及本周的SAML漏洞
  • Spring Session忍者VedranPavić刚刚宣布了Spring Session Apple SR1。新版本包含BOM表和许多其他功能
  • 项目反应堆忍者SimonBaslé 宣布了反应堆 BISMUTH -SR7 。该版本是Spring Boot 2.0所需的最终版本
  • Spring Security主管Rob Winch刚刚宣布了Spring Security 5.0.3 。此版本引入了重大的性能改进。
  • Spring Data Kay SR5发布
  • 新的Gradle 4.6已经发布，并包含许多很酷的功能，包括初始的JUnit 5支持，初始的物料清单（BOM）依赖关系和可选的依赖关系支持， --fail-fast用于测试，以及用于任务的自定义命令行标志。
  • 我最近在第一次IBM Index会议上发言，在该会议上我执行了更新的Cloud Native Java 。这着眼于使用Spring Cloud构建响应式微服务。
  • 想要进入开源？Open Feign正在寻找维护者
