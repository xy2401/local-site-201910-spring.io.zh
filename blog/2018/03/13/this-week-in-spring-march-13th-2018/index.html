春天的这个星期-2018年3月13日

工程
乔什·朗（Josh Long）
2018年3月13日

嗨，春季粉丝！欢迎来到春季本周的另一期！这个星期，我在马萨诸塞州遭受暴风雪袭击的波士顿，参加了史诗般的“春季一巡回演唱会”波士顿活动。不幸的是，由于这场狂暴的暴风雪/暴风雪，该活动已推迟一天，因为我们所有人都在努力应对天气。希望您能够加入Spring Boot 2.0启动网络研讨会！如果没有，重播将在此处提供 ，不要忘记查看启动博客 ！

下雪或不下雪！节目必须继续进行，至少在Spring博客上是这样，所以事不宜迟：

  • Reactor团队正在寻找有动力的新工程师加入该团队 -立即申请！
  • Spring Cloud Data Flow忍者Gunnar Hillert刚刚宣布了Spring Cloud Data Flow 1.4 RC1
  • Spring IO PlatformCairo-RC1负责人Andy Wilkinson刚刚宣布了最新更新，其中包括Spring Boot 2.0。检查一下，并确保它能按预期工作。
  • 我喜欢Spring团队传奇人物StéphaneNicoll的这篇关于将Spring Initializr（http://start.spring.io）升级到Spring Boot 2.0的帖子 。

  • Spring Web Flow主管Rossen Stoyanchev刚刚宣布了Spring Web Flow 2.5 。该发行版为使用Web Flow到Spring Framework 5（最低要求为Java 8，Servlet 3.1，Hibernate 5，Tiles 3和JSF 2.2）的应用程序提供了升级路径。
