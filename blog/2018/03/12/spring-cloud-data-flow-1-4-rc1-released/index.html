Spring Cloud Data Flow 1.4 RC1发布

发布
冈纳·希尔特（Gunnar Hillert）
2018年3月12日

Spring Cloud Data Flow团队很高兴宣布1.4.0 RC1的发布。遵循本地服务器 ， Cloud Foundry和Kubernetes的入门指南 。

以下是发行要点：

流部署UX

在队长模式下，我们添加了全新的流部署构建器表单，以简化部署流时平台和属性的选择。这些选项包括对部署平台和部署者的覆盖，流管道的应用程序和全局属性。

Docker Compose for Development

为了简化Local Server的入门经验，我们添加了Docker Compose支持，当从CLI作为单个命令运行时，它将自动提供最新版本的Local-server和Apache Kafka以便进行消息传递和H2数据库以支持SCDF存储库。

除了开箱即用的体验，您还可以自定义为切换到RabbitMQ进行消息传递，切换到MySQL进行数据库传递，或者添加Redis进行分析。

安全

根据最近的反馈，此版本增加了对使用Spring Cloud Data Flow中的角色映射 LDAP Active Directory组的支持。无需额外的翻译或过滤；通过Active Directory定义和SCDF中的正确映射，可以自动应用所需的授权角色。

此版本还解决了一些技术债务杂务和错误修复。有关更多详细信息，请查看发行说明 。
