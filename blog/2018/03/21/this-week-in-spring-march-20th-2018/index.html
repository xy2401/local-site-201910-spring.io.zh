春天的这个星期-2018年3月20日

工程
乔什·朗（Josh Long）
2018年3月21日

嗨，春季粉丝，欢迎来到春季的第一天！本周，我在旧金山拜访一些客户，然后出去玩，制作新的Spring Tips分期付款，享受令人惊叹的天气。好像今天还不够令人兴奋，Java 10今天也发布了！我知道一切似乎已经足够了，但请继续阅读，因为本周我们有很多好东西！

  • 我们希望您对此Spring Cloud Contract提议的反馈意见能够支持通过不同协议获取存根
  • Spring Cloud Data Flow忍者Gunnar Hillert刚刚宣布了Spring Cloud Data Flow 1.4.0 。新版本包括改进的仪表板，版本化的流，新的流部署构建器，对Docker组成的支持，安全性改进，对外壳的代理服务器支持，LDAP角色映射支持和改进的文档等。这是一个包含大量好东西的大规模发行版，所以请不要错过！
  • Spring Security和OAuth-ninja Joe Grandja 刚刚发布了Spring Security OAuth 2.3.0 ，该版本现在支持在中的椭圆曲线签名验证JwkTokenStore ， 除其他事项外。
  • 千分尺项目负责人乔恩•施耐德（Jon Schneider）在此博客中对千分尺进行了精彩的介绍 。认真地，去读它。千分尺是衡量指标收集的基础/actuator/events Spring Boot 2+中的端点。但是，它与Spring或Spring Boot无关，无论您使用何种技术，这篇文章都值得一读。
  • Spring Cloud的联合创始人Spencer Gibb一直在玩弄整合Retrofit，作为使用Feign之类的替代方法。这是一个进行中的示例，但反馈始终很有价值，很高兴看到可能会发生什么
  • Spring Cloud Open Service Broker负责人Scott Frederick 刚刚宣布了Spring Cloud Open Service Broker 2.0.0。M1 。这个新版本具有Spring Boot 2基准，同时支持Spring MVC和Spring WebFlux，并提供了改进的Spring Boot自动配置。如果您要为Cloud Foundry或Kubernetes构建服务代理，那么您将希望看到这篇文章
  • 这是Spring Web团队忍者Rossen Stoyanchev 必读的文章 ，内容涉及Spring MVC和Spring WebFlux中针对Spring用户的Servlet API和反应性API之间的选择：“您应该使用Spring MVC还是Spring WebFlux？一个完全有效的问题，但设置了不合理的二分法”
  • Java 10在这里 ！祝贺Java团队！趁热拿到！这是第一个支持本地类型推断的版本（ var ）。
  • SivaLabs博客在第6部分中继续他们对微服务的关注， 使用Spring Cloud Sleuth和Zipkin进行分布式跟踪
  • Trampoline项目已更新，以支持Spring Boot 2.0
  • Asim Aslam喜欢Project Riff，这是Pivotal的功能即服务产品。您也可以 。
  • IBM高管埃里克·安德森（Eric Andersen）在IBM Think 2018活动中发推文，期待Cloud Foundry和Istio代理带来的令人兴奋的机遇。
  • Dhiraj Ray在Java Code Geeks博客上有一篇不错的文章，内容涉及如何在Spring应用程序中外部化配置
  • Spring Cloud Contract负责人Marcin Grzejszczak发推文说，感谢Olga Maciaszek，您现在可以在3秒或3分钟内了解Spring Cloud Contract。还有一个十分钟的版本和一个大约五天的版本
  • Dan Newton的一篇不错的文章介绍了如何使用新的反应式Web运行时Spring WebFlux
  • 在第五篇文章中，有关使用Spring Cloud构建微服务的SivaLabs系列着重介绍了如何使用Spring Cloud Zuul
  • 罗伯特·温克勒（Robert Winkler）刚刚发布了Resilience4j 0.12.0 ，该版本从1.4.3的Spring Boot升级了Vavr。释放到1.5.5。发布，以及许多其他依赖项。它还向断路器添加了重置方法，为断路器添加了禁用和force_open状态，添加了对断路器，隔板和速率限制器的Project Reactor支持， 增加了对Micrometer的支持 ，并确保Reactor运算符可在Flux 。
  • 在Touk Team Blog上查看有关Spring Boot 2.0和测微表的最新帖子
  • Matt McCandless继续使用JHipster ，它基于Spring Boot和Angular构建。看看这个！
