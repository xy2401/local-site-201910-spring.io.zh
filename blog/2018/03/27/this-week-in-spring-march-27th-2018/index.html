    春天的这个星期-2018年3月27日

    工程
    乔什·朗（Josh Long）
    2018年3月27日

    嗨，春季粉丝！欢迎来到春季本周的另一期！本周我将在阳光灿烂的旧金山闲逛，与当地客户交谈，并为下季Spring Tips做更多的分期付款。

    和往常一样，我们有很多内容需要介绍，让我们开始吧！

    • 查看有关使用Spring Cloud Stream进行对象检测的 Pivotal展示。
    • Spring Security SAML DSL主管Filip Hanik 刚刚宣布了1.0.5。发布我们的SAML DSL项目 。此版本适用于Spring Security SAML 1.0.4。发布。这个版本向您现有的Spring Boot应用程序添加了一个简单的Java配置选项。
    • Ryan Baxter刚刚宣布了Spring Cloud Finchley。M9 。这个新版本包括Spring Cloud Gateway中的一些不错的功能，包括支持滚动部署和来自服务发现心跳的路由刷新。
    • Spring Tool Suite负责人Martin Lippert 刚刚宣布了Spring Tool Suite 3.9.3
    • 埃德姆居纳伊股份的旅程重新构建平台Turkcell公司的电视服务，云本地使用Spring引导和Spring云在短短几个月内。
    • Ryan Dawson着眼于使用Spring Cloud Config Server重新配置应用程序
    • Java Dev Journal通过简单的演练介绍了Spring Boot
    • Shubham Aggarwal的一篇不错的文章通过Spring Boot演示了Spring Data JPA
    • Baeldung在Spring Boot上有一篇关于属性配置的不错的文章
    • Devoxx创始人Stephan Janssen 详细介绍了如何使用Spring Boot和Spring Cloud （通过JHipster）来改进一些不同的Voxxed Web属性。干得好斯蒂芬！
    • Sourced Blog上有一篇不错的文章，介绍了在实践中使用Spring在典范的REST中实现“ REST Bucks”示例。 请注意， 这不是“REST雄鹿”的例子 。
    • 春季汽车REST文档2.0.0出来 。它基于Spring REST Docs 2.0.0，并支持Kotlin的KDoc。
    • Java Code Geeks 在用Jenkins持续交付Spring Boot应用程序方面有一篇不错的文章。
    • 查看Pivotal的功能即服务产品Project Riff的0.0.5版本 。新版本包括一个Go函数调用程序，该函数调用程序支持作为Go插件构建的函数，支持改进的Python支持，HTTP标头白名单等等。
