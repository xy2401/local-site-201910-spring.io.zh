Spring Cloud Finchley M8可用

发布
斯宾塞·吉布
2018年3月2日

我很高兴代表社区宣布， Spring Cloud Finchley Release Train的Milestone 8（M8）现已上市。该版本可以在Spring Milestone存储库中找到。您可以查看Finchley 发行说明以获取更多信息 。

Finchley发布火车的显着变化

芬奇利。M8与Spring Boot 2.0.0兼容。发布。

Spring Cloud Gateway

一些错误修复和小的配置增强。

Spring Cloud巴士

修复了自定义远程事件。

Spring Cloud安全性

更新为spring-security-oauth2-autoconfigure 2.0.0。发布。

Spring Cloud Config

支持Gitee webhooks。

春季云流

请参阅Elmhurst。RC2 发行说明 。

以下模块已作为Finchley的一部分进行了更新。M8：

模组 版
Spring Cloud Gateway 2.0.0。M8
Spring Cloud巴士 2.0.0。M7
Spring Cloud安全性 2.0.0。M3
Spring Cloud Commons 2.0.0。M8
Spring Cloud Config 2.0.0。M8
春季云流 埃尔姆赫斯特。RC2

    dependencies {
        classpath "io.spring.gradle:dependency-management-plugin:1.0.2.RELEASE"
    }

    maven {

    }
}

    imports {
        mavenBom 'org.springframework.cloud:spring-cloud-dependencies:Finchley.M8'
    }
}
