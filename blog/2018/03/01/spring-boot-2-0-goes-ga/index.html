Spring Boot 2.0正式发布

发布
菲尔·韦伯
2018年3月1日

我非常荣幸地代表团队宣布，Spring Boot 2.0现在可以通过以下方式获得： 2.0.0.RELEASE来自repo.spring.io和Maven Central ！

此版本是215个不同的人17个月的工作和6800多次提交的总结。衷心感谢所有做出贡献的人，以及所有对里程碑都提供重要反馈的早期采用者。

这是自从4年前发布1.0以来，Spring Boot的第一个主要版本。它也是提供对Spring Framework 5.0的支持的Spring Boot的第一个GA版本。

新版本的重点包括：

  • Java 8基准和Java 9支持。
  • Spring WebFlux / WebFlux.fn提供了反应性的Web编程支持。
  • 适用于反应式Spring Data Cassandra，MongoDB，Couchbase和Redis的自动配置和入门POM。
  • 支持嵌入式Netty。
  • Tomcat，Undertow和Jetty的HTTP / 2。
  • Kotlin支持。
  • 全新的执行器架构，支持Spring MVC，WebFlux和Jersey。
  • 基于微米的指标，包括Atlas，Datadog，Ganglia，Graphite，Influx，JMX，New Relic，Prometheus，SignalFx，StatsD和Wavefront的出口商。
  • Quartz Scheduler支持。
  • 大大简化了安全性自动配置。

有太多功能无法在此处详细列出所有功能，因此请转到我们Wiki上的发行说明页面，或Pivotal.io上的启动博客以了解更多信息。如果您要升级现有的Spring Boot应用程序，则还需要查看迁移指南 。如果您只想入门，则可以轻松地在start.spring.io上引导一个新项目。

在接下来的几周内，我们将发布博客文章，其中详细介绍了Spring Boot 2.0的某些功能。您还应该参加3月13日的网络研讨会 ，我们将深入探讨升级的方式。

再次感谢多年来为Spring和Spring Boot做出贡献的每一个人！特别感谢Spring产品组合中其他项目的开发人员，没有他们，他们将没有Spring Boot。
