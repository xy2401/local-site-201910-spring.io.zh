春天的这个星期-2018年9月18日

工程
乔什·朗（Josh Long）
九月18，2018

嗨，春季粉丝！欢迎来到春季本周的另一期！在SpringOne平台发布的前一周，事情变得疯狂了 ！和往常一样，我们有很多东西要覆盖！ - 但首先！ ..一些诗歌（向克莱门特·克拉克·摩尔致歉）

是在整个Spring之前的SpringOne平台的前一周；
不是一个生物在搅动，只有计算机和东西在搅动。
社区的关心提出了期望；
希望SpringRod会在那里；

那就对了！今年定将是惊人的！Spring创始人Rod Johnson，Jenny Zhang，Siew Choo Soh，Neha Narkhede，Matt Raible，Ray Tsang，Lauren Knausenberger，Juergen Hoeller，Bruno Borges，Andrew Clay Shafer和allllll将拥有Spring团队的其余成员！感觉就像一分钟前（十二月！）我们只有最后一个，所以我不敢相信这一个到货有多快，但我等不及了！

现在，我在加州旧金山跳来跳去；乔治亚州亚特兰大；乔治亚州哥伦布；我和CA的洛杉矶很高兴，我很高兴能与我拜访的会议和客户在一起，但是我真的很高兴能参加SpringOne Platform，因此倒计时了！

想一想？下次我写这篇综述时，我将在华盛顿特区参加SpringOne Platform 2018！

  • Spring Cloud Data Flow 1.7 M1发布
  • 介绍Spring Data JDBC

  • 在第8部分（共8部分）的第8部分中，我研究了如何通过Spring Cloud GCP部署基于Cloud Foundry的应用程序，这些应用程序使用Google Cloud服务
  • Andy Wilkinson 也刚刚宣布了Spring Boot 2.0.5
  • 确保您查看Jim Weaver的网络研讨会“ Reactive Card Magic”
  • Spring Boot 1.5.16
  • Oleg Zhurakousky 在Spring Cloud Function中引入了新的Kotlin支持
