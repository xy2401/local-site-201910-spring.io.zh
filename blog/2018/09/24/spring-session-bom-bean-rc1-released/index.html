  • Project Reactor Californium-RELEASE

Spring Session BOM Bean-RC1发布

发布
罗伯·温奇
九月24，2018

该帖子由VedranPavić撰写

我很高兴代表社区宣布Spring Session BOM Bean-RC1 。该版本基于Spring Session 2.1.0.RC1共解决13个问题 。请继续阅读以了解该版本的重点。

支持Java 11

Spring Session现在支持Java 11，而所需的课程版本仍为Java 8。我们的CI管道得到了增强，因此该项目现在已经针对Java 8、10和11进行了持续验证。

依赖升级

春季会议2.1.0.RC1建立在以下关键依赖关系的最新和最佳版本上：

  • Spring Framework 5.1.0.RELEASE

  • 春季数据Lovelace-RELEASE

  • Spring Security 5.1.0.RELEASE

  • 项目反应堆Californium-RELEASE

  • 淡褐色3.10.5

其他

该版本还提供了一些错误修复和性能改进，最值得注意的是：

  • 通过延迟反序列化属性来提高JdbcOperationsSessionRepository性能： ＃1133

  • 通过利用Hazelcast的Offloadable提高了HazelcastSessionRepository#save性能： ＃1204

  • 修复了在同一Redis实例中使用多个数据库时RedisOperationsSessionRepository不正确的HttpSessionEvent处理的问题： ＃1128

反馈请

随着第一个候选版本的发布，Spring Session Bean现在被认为是功能齐全的，我们求助于您，我们的社区，在发布稳定的Bean-RELEASE之前向我们提供反馈。预计在10月中旬，即Spring Boot 2.1.0.RC1
