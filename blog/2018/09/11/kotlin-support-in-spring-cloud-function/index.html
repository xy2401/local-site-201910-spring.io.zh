Spring Cloud功能中的Kotlin支持

工程
奥列格·朱拉库斯基（Oleg Zhurakousky）
九月11，2018

我们很少在博客中介绍单个功能，但是鉴于此 功能是Spring Cloud Function （相对较年轻的项目）中最受关注的功能之一 ，因此我们认为它很合适，因此就在这里。

Spring Cloud Function已添加了对Kotlin lambdas的初始支持。这意味着Spring Cloud Function现在可以识别与Java的Supplier ， Function或Consumer一个有效匹配的Kotlin lambda，并对其进行相应处理。

那是：

@Bean
open fun kotlinFunction(): (String) -> String {
    return  { it.toUpperCase() }
}

@Bean
open fun kotlinConsumer(): (String) -> Unit {
    return  { println(it) }
}

@Bean
open fun kotlinSupplier(): () -> String {
    return  { "Hello Kotlin" }
}

在此处查看示例项目

就是这样。该功能在当前快照中可用，并将成为Spring Cloud Function 2.0.0的一部分。发布。这意味着增强和修改仍在进行中，因此您的反馈非常重要。
