春天的这个星期-2018年9月11日

工程
乔什·朗（Josh Long）
九月11，2018

嗨，春季粉丝！欢迎来到春季本周的另一期！本周我在土耳其伊斯坦布尔，与客户交谈并参加史诗般的SpringOne Platform Istanbul活动，否则通常在短短几周内为旗舰SpringOne Platform 2018活动做准备

  • Spring Security负责人Josh Cummings刚刚宣布了Spring Security 5.1.0。RC2 。新版本支持大量新功能，包括用于OAuth 2的简化DSL， WebClient扩展，Servlet增强，改进的资源服务器支持，改进RestTemplate支持，对X.509的支持以及使用环境变量来配置LDAP。
  • Mark Paluch刚刚发布了打包好的 Spring Data Ingalls SR15和Spring Data Kay SR10
  • Mark Paluch刚刚宣布了Spring Vault 2.0.2，该版本提供了依赖项更新和一些小错误修复
  • Mark Paluch刚刚宣布了Spring Vault 2.1的候选版本 。Spring Vault 2.1需要JDK 8或更高版本，并且特别支持JDK 11作为下一个长期支持版本。此发行版附带15张票证，并针对Java 9至11进行了一些改进。最显着的变化是支持在VM上使用托管服务实例进行Azure身份验证，代表服务帐户的GCP IAM身份验证支持，并且该版本与从0.5.2到0.11.1的保管库版本兼容。
  • 继续我的第八部分系列文章，该系列文章介绍了如何将Spring Cloud与Google Cloud结合使用， 第7部分探讨了如何使用Spring Cloud GCP连接到其他GCP服务
  • StéphaneNicoll刚刚宣布了Spring Framework 5.1 RC3、5.0.9和4.3.19
  • 继续我的第八部分系列文章，该系列文章探讨了如何将Spring Cloud与Google Cloud结合使用， 第六部分探讨了如何使用Spring Cloud GCP Stackdriver Trace支持可观察性。
  • LaísNeves在WoMakersCode会议上回顾了她的经验，在这篇葡萄牙语文章中，她谈到了通过Spring Boot使Java重新变得有趣。
  • Sajal Chakraborty有一篇不错的文章，介绍了如何在现代Spring Boot上下文中使用Spring的RMI和Hessian服务导出程序支持
  • Packt的这本新书《 带有Spring 5.0的软件体系结构》看起来很有趣。我将其添加到“阅读”列表中。
  • 这篇关于使用Spring Boot和Hazelcast的文章很有帮助。考虑到我在土耳其的Hazelcast首次亮相的伊斯坦布尔，这真是个好建议！
  • 我很荣幸被列入此列表中的20位Java影响者榜单
  • 查看Jakub Pilimon的史诗指南，了解使用Spring堆栈进行CQRS的几种可能方法
  • 看看这篇关于Graal VM的不错的帖子： “ The Graal Frenzy”
  • Alican Akkus在“用Spring Boot进行的消费者驱动的合同”上做得很好。
  • JUnit 5.3已发布！它包括并行测试执行，输出捕获，新TestInstanceFactory扩展API等
  • 在短短的几周内 ， 在史诗般的SpringOne Platform 2018活动中，发表了5个Ops团队意外旅程的精彩文章！
