对于我在这里分享的每张照片或瞬间，还有一千我没有分享过。自从节目如火如荼地开始以来，这已经是一个疯狂的24小时，而且我们还有很长的路要走！

因此，事不宜迟，让我们开始看看本周令人惊叹的，发布版本丰富的综述！

  • Spring Cloud Function 2.0和Azure函数
  • 大家好消息！Spring Tool Suite 4在这里 ！我要亲自祝贺Spring Tool Suite领导Martin Lippert经历了这一不可思议的旅程！STS 4是对支持我们的Eclipse发行版的插件背后的引擎的完整改造。该引擎代表了目标的根本优先级调整：这些工具现在基于运行时行为为您提供验证，而不是通过在设计时尝试对Spring进行第二次猜测而产生的误报。结果是在开发过程中使用Java可以快速高效地进行工作-无需费劲地争吵！ -任何验证和导航信息均由应用程序的实际运行时行为告知。只是更准确了。此外，引擎本身已被提取到独立于Eclipse运行的语言服务器中，并且在Microsoft Visual Studio Code和Atom中受支持。这个版本给我留下了深刻的印象！这是一个漫长而值得的旅程。
  • Spring Boot团队传奇人物Madhura Bhave刚刚宣布了Spring Boot 2.1。M4 ！新的Spring Boot 2.1.0。M4版本附带了大量内容，包括对Spring Data JDBC的初步支持，包括@DataJdbcTest注解。看看这个！
