Spring 的这个星期-2018年8月21日

工程
乔什·朗（Josh Long）
2018年8月21日

嗨， Spring 粉丝们，欢迎来到 Spring 本周的另一期。本周，我去了烟雾弥漫的西雅图，而我刚好来到了纽约市，时常拜访客户和 The Spring Team 友。我们有很多东西要讲，所以不费吹灰之力，让我们开始吧！

当我们接近史诗般的SpringOne Platform 2018活动时 ，Spring团队将全程低落 。你要去那里吗？将于2018年9月24日至27日在华盛顿特区举行。千万不要错过将敏捷，Cloud Foundry，Kubernetes，devops和当然Spring的实践者团结在一起的顶级盛会。

  • 通过Spring Boot 2.1的这一更改 ，您甚至不需要使用以下注释来注释基于Spring的JUnit 5测试：@ExtendWith(SpringExtension.class)
  • 约什-卡明斯刚刚宣布Spring Security 5.1RC1 ，其中包括，在许多其他事情，对OAuth的资源服务器支持和CORS和安全的头更好的支持
  • Spring Boot 2.1 M2现已上市
  • 提供Spring Data Lovelace RC2
  • 嗨， Spring 粉丝！我刚刚开始了一个由八部分组成的博客系列，每个星期一和星期四都会分期发布，其中介绍了适用于Google Cloud Platform的Spring Cloud 。在第一部分中，我们着眼于设置一个基本的Spring Cloud GCP项目。请继续关注下一期，我们将使用Spring Cloud GCP将应用程序连接到RDBMS。
  • Spring Session Bean-M1和Apple-SR4发布
  • Spring Framework 5.1 RC2现在可用
  • 我真的很喜欢Richard Seroter关于如何成功使用不同云的新文章
  • 如果您将Spring Boot 2.0.4与Spring Data Neo4j结合使用，请密切注意此问题
  • 请查看此功能预览，该功能预览支持延迟[Spring Data JPA存储库初始化以加快典型Spring Boot应用程序的启动
  • Paul Czarkowski的文章对KNative和Kubernetes进行了很好的介绍
  • Mike Villager创建了两个ConcourseCI资源。第一个实现Dynatrace UFO，但第二个则更有趣，使用它可以轻松地将有关部署事件的元数据推送到Dynatrace
  • 有趣！消息传递技术NATS具有新的类似Kafka的日志API。 查看此介绍新功能的infoQ采访
  • 深入研究JUnit 5扩展模型
  • 检查出这次谈话是铰接的理查德Seroter和微软的阿西尔Vedamuthu Selvasingh中，他们突出Azure中的事件驱动架构和Cloud Foundry上和Spring是如何为用户排忧解难！
