Servlet

OAuth2资源服务器

开放ID提供商配置

现在，可以通过任何支持Open Id Provider Configuration的发行者端点配置Resource Server：

@Bean
JwtDecoder jwtDecoder() {
    return JwtDecoders.createDefaultFromIssuer("https://issuer-endpoint");
}

索赔验证

用户可以添加自己的验证规则以应用于Jwt通过暴露一个JwtDecoder豆：

@Bean
JwtDecoder jwtDecoder() {
    String jwkSetUri = "https://issuer-endpoint/.well-known/jwks.json";
    NimbusJwtDecoderJwkSupport jwtDecoder =
      new NimbusJwkDecoderJwkSupport(jwkSetUri);
    OAuth2TokenValidator<Jwt> validator =
      new DelegatingOAuth2TokenValidator(
        JwtValidators.createDefault(),
        new MyCustomValidator());
    jwtDecoder.setJwtValidator(validator);
    return jwtDecoder;
}

授予权限提取

用户可以自定义GrantedAuthority s来自a Jwt ：

@Bean
JwtDecoder jwtDecoder() {
    String jwkSetUri = "https://issuer-endpoint/.well-known/jwks.json";
    NimbusJwtDecoderJwkSupport jwtDecoder =
      new NimbusJwkDecoderJwkSupport(jwkSetUri);
    JwtAuthenticationConverter jwtAuthenticationConverter =
      new JwtAuthenticationConverter() {
        protected Collection<GrantedAuthority> extractAuthorities(Jwt jwt) {
          return Arrays.asList(new SimpleGrantedAuthority("app:read"));
        }
    };
    jwtDecoder.setJwtAuthenticationConverter(jwtAuthenticationConverter);
    return jwtDecoder;
}

Web助焊剂

OAuth2资源服务器

添加了对基于响应的OAuth2资源服务器的基本支持。请参阅oauth2resourceserver-webflux

OAuth2登录名/客户端

授权码授予

添加了对基于响应式授权代码授予流程的基本支持。参见authcodegrant-webflux

授权请求解析器

添加了对自定义对授权服务器的身份验证请求的支持 。例如，如果授权服务器要求发送自定义参数， 这将很方便。在多租户场景中，如主机名之类的请求元素可能会更改向授权服务器发出请求的方式，这也很有用。

授权的客户资料库

添加了对自定义请求之间的授权客户端的持久性的支持：

http
    .oauth2()
        .client()
            .authorizedClientRepository(new MyCookieBasedClientRepository());
