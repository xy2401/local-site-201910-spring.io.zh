Bootiful GCP：针对Google Cloud Platform的Spring Cloud入门（1/8）

工程
乔什·朗（Josh Long）
八月20，2018

嗨， Spring 粉丝！在这个简短的8部分系列文章中，我们将研究Google Cloud Platform的Spring Cloud集成，称为Spring Cloud GCP。 Spring Cloud GCP代表了Google与Pivotal的共同努力，致力于在使用Google Cloud Platform时为Spring Cloud开发人员提供一流的体验。Pivotal Cloud Foundry用户将可以更轻松地与GCP服务代理集成 。我用Google Cloud Developer Advocate和我的好友Ray Tsang的投入编写了这些文章。您还可以在我们的Google Next 2018会话Bootiful Google Cloud Platform中观看 Spring Cloud GCP的演练。谢谢哥们儿！与往常一样， 如果您有任何反馈意见 ， 我希望能收到您的来信 。

该系列有八个职位。这些都是：

  • Bootiful GCP：针对Google Cloud Platform的Spring Cloud入门（1/8）
  • Bootiful GCP：使用Spring Cloud GCP进行关系数据访问（2/8）
  • Bootiful GCP：使用Spanner（3/8）实现全球一致的数据访问
  • Bootiful GCP：与Google Cloud Pub / Sub（4/8）集成
  • Bootiful GCP：使用Spring Cloud GCP Runtime Config（5/8）进行运行时配置
  • 引导性GCP：通过Spring Cloud GCP Stackdriver Trace（6/8）支持可观察性
  • 引导性GCP：使用Spring Cloud GCP连接到其他GCP服务（7/8）
  • 可怜的GCP：量产！（8/8）

在这一部分中，我们将介绍Spring Cloud GCP，然后使用Spring Cloud GCP设置一个简单的项目。

Google Cloud Platform非常庞大 ！它具有大量功能。但是其他平台也是如此。那么，为什么要使用Google Cloud Platform？我说使用GCP有两个主要原因。首先，谷歌的数据中心无疑是世界上最复杂的数据中心，其运营敏锐度首屈一指。让Google运作并保护您的数据中心。除此之外，Google还必须通过GCP解决大规模的问题，我们才能从GCP的这些发展中受益。 Google独有的数据服务具有传奇色彩。例如， Google的Tensorflow代表了机器学习的最前沿，GCP甚至可以访问专门针对运行TensorFlow工作负载而优化的专用应用集成电路（ASIC） 张量处理单元 （TPU）。Google的Spanner是唯一为云而构建的企业级，全球分布且高度一致的数据库服务，专门用于将关系数据库结构的好处与非关系水平规模相结合。

认证方式

您需要注册一个GCP帐户 。为了在您的本地计算机上使用Spring Cloud GCP项目，您需要在GCP上设置一个项目，并使用gcloud CLI在本地安装 。

在开始使用GCP进行任何工作之前，需要了解和做一些事情。首先，您需要登录。运行以下命令以登录GCP：

gcloud auth application-default login

这样可以轻松提供默认凭据，以支持您与平台的交互工作。在这种情况下，您是说要代表您允许某些操作。

某些操作不受特定用户的影响。他们可能需要代表您特权的子集的细化权限。它们可能独立于给定的用户运行，例如在批处理作业或每晚运行的作业中。在这种情况下，使用服务帐户是有意义的。稍后我们将在跟踪时需要配置服务帐户 。

了解您的项目ID很有用。您会发现很多咒语都需要了解此值。我有一个这样的小脚本在我的~/.bashrc并贡献给每个外壳。

export PROJECT_ID=$(gcloud config list --format 'value(core.project)')

在这些示例中，我倾向于使用CLI，但是大多数您想做的事情都可以从Web控制台完成。

Spring Cloud GCP入门

至少到目前为止，您将需要Spring Cloud 和 Spring Cloud GCP物料清单（BOM）工件。进行新的Spring Cloud GCP项目的最简单方法是从Spring Initializr生成一个新项目，然后选择GCP Support 。这是最小的pom.xml神器看起来像。

基本的pom.xml用于Spring Cloud GCP。

<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
    xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <modelVersion>4.0.0</modelVersion>

    <groupId>com.example</groupId>
    <artifactId>demo</artifactId>
    <version>0.0.1-SNAPSHOT</version>
    <packaging>jar</packaging>

    <name>demo</name>
    <description>Demo project for Spring Boot</description>

    <parent>
        <groupId>org.springframework.boot</groupId>
        <artifactId>spring-boot-starter-parent</artifactId>
        <version>2.0.4.RELEASE</version>
        <relativePath/> <!-- lookup parent from repository -->
    </parent>

    <properties>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
        <project.reporting.outputEncoding>UTF-8</project.reporting.outputEncoding>
        <java.version>1.8</java.version>
        <spring-cloud-gcp.version>1.0.0.RELEASE</spring-cloud-gcp.version>
        <spring-cloud.version>Finchley.SR1</spring-cloud.version>
    </properties>

    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-gcp-starter</artifactId>
        </dependency>

        <dependency>
            <groupId>org.springframework.boot</groupId>
            <artifactId>spring-boot-starter-test</artifactId>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <dependencyManagement>
        <dependencies>
            <dependency>
                <groupId>org.springframework.cloud</groupId>
                <artifactId>spring-cloud-dependencies</artifactId>
                <version>${spring-cloud.version}</version>
                <type>pom</type>
                <scope>import</scope>
            </dependency>
            <dependency>
                <groupId>org.springframework.cloud</groupId>
                <artifactId>spring-cloud-gcp-dependencies</artifactId>
                <version>${spring-cloud-gcp.version}</version>
                <type>pom</type>
                <scope>import</scope>
            </dependency>
        </dependencies>
    </dependencyManagement>

    <build>
        <plugins>
            <plugin>
                <groupId>org.springframework.boot</groupId>
                <artifactId>spring-boot-maven-plugin</artifactId>
            </plugin>
        </plugins>
    </build>

    <repositories>
        <repository>
            <id>spring-milestones</id>
            <name>Spring Milestones</name>
            <url>https://repo.spring.io/milestone</url>
            <snapshots>
                <enabled>false</enabled>
            </snapshots>
        </repository>
    </repositories>


</project>
