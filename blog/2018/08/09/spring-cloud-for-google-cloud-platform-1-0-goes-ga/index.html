Google Cloud Platform 1.0的Spring Cloud正式发布！

发布
阿尔特姆·比兰（Artem Bilan）
2018年8月9日

亲爱的春季社区！

今天，我很高兴与Google Cloud小组一起宣布Spring Cloud for Google Cloud Platform版本的项目1.0最后普遍可用！

该项目开始于与Google Cloud团队的合作（我们非常感谢他们的努力），旨在实施和公开众所周知的Spring Framework模式和抽象，以将Spring和Spring Boot的好处带给使用GCP服务的Java开发人员。我们在一起度过了很多小时（一年多了！）讨论最佳的API和方法，这些API和方法应方便且有用，以针对旨在使用GCP服务的项目。即使我们多次迭代实现以提供开发人员体验，但现在我们仍然相信该项目已经成熟且稳定，并且相信其潜力。现在，我们非常高兴能够继续合作，以实现更多功能以为Spring Cloud应用程序公开GCP服务！

请查阅先前发布的候选版本1 ，以获取有关该项目到目前为止实现的功能的详细说明。从那时起，我们将项目升级到具有所有最新可传递依赖项的最新Finchley SR1 ，并修复了一些问题。

试试看！

可通过BOM从Spring Release存储库和Maven Central获得的工件：

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-gcp-dependencies</artifactId>
            <version>1.0.0.RELEASE</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>

参考文档和示例应用程序可帮助您快速入门。

接下来是什么？

现在我们正忙于更多的集成，并且我们最近发布了一个1.1.0。M1版本具有Spring Data Cloud Spanner和Spring Cloud Config支持的预览。正在进行的艰苦工作是围绕Google Cloud Datastore的Spring Data抽象。我们还想解决更多Google Cloud功能，并为他们提供基于Spring的API和配置便利性。不要犹豫，给我们任何反馈，甚至是贡献 ！

来自Spring Cloud GCP团队

不要错过今年9月在华盛顿特区举行的SpringOne Platform会议！我将与Google团队一起在Spring Cloud for Google Cloud Platform项目中介绍这些功能以及更多其他功能。结帐会话并注册！
