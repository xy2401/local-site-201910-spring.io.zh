Spring 的这个星期-2018年8月14日

工程
乔什·朗（Josh Long）
八月15，2018

嗨， Spring 粉丝！欢迎来到 Spring 本周的另一期！本周，我在旧金山，西雅图和洛杉矶与客户交谈。

我们Spring团队的成员很沮丧，为今年不可错过的SpringOne Platform做准备 。来了解有关本机云的应用程序开发，云计算，敏捷，开发，大数据，持续交付，测试等等。立即获取门票！该活动于2018年9月24日至27日在华盛顿特区举行

..说起！看到这个孩子吗？他刚刚通过了Java营地认证，如果获得1000次点赞/ RT，Pivotal会将其发送到SpringOne Platform。请转推！

这是绝对疯狂的一周，所以让我们开始吧！

  • 在其中，我们有趣地观察了国防信息系统局（DISA）如何改变其管理操作系统的方式。此举有副作用吗？新的认证：Canonical的Ubuntu。这对Cloud Foundry具有深远的意义-请继续阅读以了解如何
  • Spring Tools 4的M14已针对Eclipse Java IDE，Visual Studio Code和Atom Editor发布。在这里获取位
  • 这次有关通过自动化，开源，安全性管理增强安全性的网络研讨会（8月15日） 看起来非常有趣 ！
  • Spring Cloud Open Service Broker 2.0.1。现在可以发布
  • Spring Cloud Skipper 1.1.0。M1
  • Google Cloud Platform 1.0的Spring Cloud正式发布！
  • Matt Raible 更新了他的“使用OAuth 2.0和JHipster开发微服务架构”以使用JHipster v5.1.0和Spring Boot 2.0.3。我们将忽略Spring Boot 2.0.4已经可用！ ：D
  • 我喜欢关于无服务器计算和容器技术的Cloud Foundry调查结果的JAXEnter博客文章
  • 这太酷了！Spring Boot和Spring开发人员Stephane Nicoll与Vaadin的Matti Tahvonen有一段很好的视频，内容涉及如何使用Vaadin 10构建Spring Boot应用程序 。我已经将此标签加为书签，并且会观看。
  • 我喜欢制作， Tyler Van Gorder及其公司显然也是如此，因为他们刚刚将带有657k行Java代码的基于Spring Boot和基于Spring Cloud的系统迁移到Spring Boot 2.0和Spring Cloud Finchley并投入生产！恭喜你！
  • SUSE的朋友正在招聘一名产品管理职位，负责Cloudfoundry Kkubernetesio和OpenStack。有兴趣吗Ping Devin Davis可以做介绍。
  • Spring Framework Guru网站上有一篇非常有趣的文章，内容涉及使用OpenAPI定义Spring Cloud Contracts 。 太棒了 ！
  • 对于JavaFX客户端，Spring是一个很好的服务选择，特别是在Reactor使之成为响应世界的情况下。这个线程很有启发性 。
  • Knative的riff v0.1.1现在可用 。新版本具有系统卸载和命令功能调用程序支持。这太棒了！
  • 与Jenkins for PCF的持续集成和部署
  • [此日语编程杂志的最新一期-WEB + DB- 在Spring Boot上有一篇不错的文章 。看看这个！
  • Flowable BPMN引擎-理想地与Spring Boot配合使用- 现在支持在MongoDB上运行
  • 我与Java生态系统中的其他许多人一起接受了有关Java新发行版节奏的采访。您可能会喜欢一些见解
  • 《 SD时报》很好地撰写了Spring Cloud GCP中最新，最出色的文章
  • 我们的朋友Marten Deinum写了一篇很好的文章， 介绍了如何将JDBI与Spring Boot集成在一起。
  • Rackspace很好地介绍了组织通过在Cloud Foundry上运行来节省数百万美元的方法-简而言之！
  • 现在我们来看一下JUnit5的瑰宝：参数化测试。使用不同的输入执行相同的测试从未如此舒适
  • 想要详尽地了解如何使用Spring Cloud创建基于Spring的微服务吗？来自Pranav Patil的这个社区示例看起来是一个有趣的开始
  • 要参加在Sergipe联邦大学-圣克里斯托瓦校区计算机系举行的WomakersCode峰会Sergipe吗？会议将讨论发展，工具和良好做法，以及一个由女性主持的专注于职业发展和学习计划准备的小组讨论。有太多值得推荐的活动！但是，如果您只能看到其中一个，请一定要参阅Spring社区女主人公LaísNeves的有关Spring Boot的内容，以及它如何使Java开发人员的生活更轻松 。
  • 泰勒·隆德（Tyler Lund）很好地介绍了他看到的混沌工程新团队犯的五个错误
  • Swapnil Bhartiya发表了一篇不错的文章， 介绍了Cloud Foundry为什么选择Istio和Envoy进行路由
  • Riff 0.1.1现在可用 ！它支持命令功能调用程序和系统卸载支持。
  • 这是Spring Boot忍者Andy Wilkinson的精彩文章。Spring Boot即将为Spring Boot贡献500个贡献者。显然，这值得社区的极大感谢。谢谢。想要贡献？好吧，我们也想要您！阅读安迪·威尔金森（Andy Wilkinson）上有关如何执行此操作的主题！
  • Zoltan Altfatter上有一篇不错的文章，介绍如何使用Spring Cloud Services服务注册表
  • 这是一个老套的东西：Mohamed Sanaulla撰写了有关基于Spring Boot配置文件的日志记录的示例Logback配置
  • Blaze Persistence已为Spring Data JPA交付了一些非常引人注目的集成
  • 我们的 Spring LinkedIn组现在有51,000名成员 ！当然，总是欢迎更多！ :)
  • 哇！Spring Data Neo4j的领导者和令人敬畏的数据传奇人物Michael Hunger刚刚宣布发布Neo4j JDBC驱动程序3.4.0，该驱动程序支持Neo4j 3.4.x中的空间和日期/时间数据类型以及完整的群集/路由支持。
