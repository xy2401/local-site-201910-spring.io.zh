Spring Data R2DBC 1.0 M1发布

发布
马克·帕鲁奇
十二月12，2018

我很高兴宣布Spring Data R2DBC的第一个里程碑。 Spring Data R2DBC是Spring的R2DBC客户端项目。借助Spring Data R2DBC，您可以端到端地进行响应式即席查询执行以及关系数据库的Spring Data存储库。

Spring Data R2DBC通过提供以下功能来支持关系数据库的功能-响应式集成DatabaseClient作为其中心抽象。以下示例显示了如何引导DatabaseClient无需旋转Spring上下文：

PostgresqlConnectionFactory connFactory = new PostgresqlConnectionFactory(
  PostgresqlConnectionConfiguration.builder()
    .host(…)
    .database(…)
    .username(…)
    .password(…)
    .build());

DatabaseClient databaseClient = DatabaseClient.create(connFactory);

Spring Data R2DBC构建在R2DBC SPI之上，因此您可以插入各种驱动程序（从R2DBC 1.0 M6开始）：

  • Postgres （ io.r2dbc:r2dbc-postgresql ）
  • H2 （ io.r2dbc:r2dbc-h2 ）
  • Microsoft SQL Server （ io.r2dbc:r2dbc-mssql ）

这个Spring Data R2DBC里程碑具有以下初始功能集：

  • DatabaseClient作为响应式通用SQL执行的网关。
  • 反应性交易支持通过TransactionalDatabaseClient 。
  • 支持功能，例如select()和insert()与映射对象一起使用。
  • 对Postgres，H2和Microsoft SQL Server的方言支持。
  • Java配置支持AbstractR2dbcConfiguration 。
  • 反应性存储库支持通过@EnableR2dbcRepositories并加注@Query方法。
  • 将R2DBC异常转换为Spring的DAO异常层次结构 。

这是一些示例 DatabaseClient用法。另请注意本机参数绑定标记的用法：

Mono count = databaseClient.execute()
        .sql("INSERT INTO person (id, firstname, lastname) VALUES($1, $2, $3)")
        .bind("$1", 42055)
        .bind("$2", "Walter")
        .bindNull("$3", String.class)
        .fetch()
        .rowsUpdated();

Flux> rows = databaseClient.select()
        .from("person")
        .orderBy(Sort.by(desc("id")))
        .fetch()
        .all();

Flux rows = databaseClient.select()
        .from(Person.class)
        .orderBy(Sort.by(desc("id")))
        .fetch()
        .all();

如果您已经熟悉Spring Data的Repository抽象，那么您将认识到Spring Data R2DBC Repository遵循完全相同的扩展思想。 ReactiveCrudRepository ：

interface PersonRepository extends ReactiveCrudRepository {

  @Query("SELECT * FROM person WHERE firstname like $1")
  Flux findByFirstNameContains(String firstname);

  @Query("SELECT * FROM legoset WHERE firstname = $1 AND lastname = $2")
  Mono findByFirstnameAndLastname(String firstname, String lastname);
}

您可以从我们的里程碑存储库中获取Spring Data R2DBC 1.0 M1。为此，请在您的pom.xml文件：


  spring-libs-milestone
  https://repo.spring.io/libs-milestone



  
    org.springframework.data
    spring-data-r2dbc
    1.0.0.M1
