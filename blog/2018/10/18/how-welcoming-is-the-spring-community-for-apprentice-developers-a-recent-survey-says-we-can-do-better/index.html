Spring Community对学徒开发人员的欢迎程度如何？最近的一项调查显示，我们可以做得更好。

DiógenesRettori
十月18，2018

刚刚开始您的开发生涯？查看有关JVM生态系统的最新报告。Simon Maple，Andrew Binstock和许多其他人已经使用此数据为各种技能的开发人员提供了出色的服务。

< @snyksec >墙上的镜子，这是所有镜子中最流行的Java Web框架？

我们生活在一个 Spring 的世界。实际上，普通的应用程序更有可能使用Spring！54％的应用程序使用Spring Boot / Spring MVC / JHipster。 https://t.co/2dkjt6xauK pic.twitter.com/KcMYWew6Jk

-Simon Maple（@sjmaple） 2018年10月19日

我是Spring的长期开发人员，并且在过去的十年中一直为此提倡。如此广泛地采用也就不足为奇了。Spring Boot主导了Cloud Native Java生态系统。下载数量持续以看不见的速度增长。实际上，根据调查，使用Spring技术进行开发的人数更多。现在是成为Java开发人员的好时机！

不过，我可能会更开心，所以让我们开始关注我的原因。看看下面的推文：

我很容易跳到“下一件事情”，而忘记了人们总是进入我们的领域并学习第一件事。这是有关SQL查询的很棒的@digitalocean教程…… https://t.co/t65H4BXMMm

-Richard Seroter（@rseroter） 2018年10月18日

根据该报告，有38％的受访者年龄在35岁以下。30岁以下的年轻人中有近五分之一的人。尽管我喜欢谈论响应式开发模式如何使您的应用程序更具可扩展性（除其他外），但我确实必须告诉自己，有很多人刚刚开始他们的软件工程职业。这些人需要介绍性主题，而不是黑带水平培训。

Java社区如何吸引这些新兴开发人员？我相信整个JVM生态系统及其社区和供应商-包括我自己在内-所做的工作都不尽人意，这可能不是流行的观点。我相信我们还有许多工作要做，以使新的和新的工程师可以使用技术。考虑一下上述报告中的数据：只有1％的受访者年龄在21岁以下，这对我来说是令人不安的。根据美国劳工统计局的数据，大约5％的软件开发人员和编程人员年龄在24岁以下 ，这暗示我1％的人员至少需要加倍。

您才刚刚开始。这太棒了！

除了拥有涵盖所有年龄段的思维方式之外，我不会在本文中提出任何长期解决方案。但是我确实想分享一些基本资产，这些资产可以帮助开发人员集中精力应对Spring和Java。如果您刚刚开始软件开发，那么这些视频和文章是极好的资源。如果您听说过Spring，但又不知道它的含义或用途，那么这些都是给您的，记住， 当学徒是可以的 。

伟大的first-timers-only标签。

对于那些希望参与为Spring Boot做出贡献的人来说，github repo可能有点压倒性的。实际上，您可能会问自己：我怎么可能在这里参与？我们只有一个标签适用于这种情况：“仅限初学者”。我们有时会在仓库中用这个标签标记问题，以表明如果您是该项目的新参与者，我们将为您提供帮助，并帮助您入门该技术。这是一个例子： https : //github.com/spring-projects/spring-boot/issues/14599大多数项目还使用ideal-for-contribution标签来帮助现有的贡献者找到我们真正需要帮助的问题。
