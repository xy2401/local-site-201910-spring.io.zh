Spring Cloud for Alibaba 0.2.0发布

发布
斯宾塞·吉布
十月30，2018

嗨！今天，我们非常高兴地宣布Spring Cloud Alibaba的第一个预览版。

Spring Cloud Alibaba项目由Alibaba的开源组件和几种Alibaba Cloud产品组成，旨在实现并公开众所周知的Spring Framework模式和抽象，以将Spring Boot和Spring Cloud的好处带给使用Alibaba产品的Java开发人员。

注意： Spring Cloud Alibaba项目是由Alibaba维护的社区项目。

注意： 版本0.2.0。RELEASE与Spring Boot 2.0.x系列兼容。版本0.1.0。RELEASE与Spring Boot 1.x系列兼容。

带有“ spring-cloud-alibaba”前缀的开源组件具有以下功能：

服务发现

的spring-cloud-alibaba-nacos-discovery-starter组件实现了Spring Cloud Commons中定义的注册表约定接口，包括NacosAutoServiceRegistration ， NacosServiceRegistry ， NacosDiscoveryClient等等。通过添加依赖性和一些配置，您可以轻松地将服务注册到Nacos Server。还支持与Ribbon集成。

组态

NacosPropertySourceLocator在里面spring-cloud-alibaba-nacos-config-starter组件实现PropertySoureLocator接口。只需添加依赖项和一些配置，您就可以从Nacos Server获得应用程序配置并将其添加到Spring Environment中。此外，您无需其他组件即可分发配置并实时跟踪分发状态。

保障高可用性

Servlet，RestTemplate，Dubbo和RocketMQ的流控制，电路中断和并发功能已与spring-cloud-alibaba-sentinel-starter默认情况下，组件可以通过添加依赖项直接使用。您还可以通过Sentinel仪表板修改流量控制，断路或并发的策略和阈值，以实时运行应用程序。

以“ spring-cloud-alicloud”为前缀的阿里云产品具有以下功能：

应用程序命名服务

应用程序命名服务（ANS）是阿里云的服务发现服务，可提供更具成本效益的SaaS服务发现服务。通过API调用的加密，您的服务具有额外的保护层。

应用程序配置管理

应用程序配置管理（ACM）是阿里云的Nacos配置管理服务，可提供更具成本效益和安全性的SaaS配置管理服务。您还可以查看整个配置分发轨道。

对象存储服务

对象存储服务（OSS）是阿里巴巴提供的经济高效，高度安全和高度可靠的云存储解决方案，可让您随时随地从网络上的任何位置存储和检索任何类型的数据。只需通过自动注入OSS客户端，您就可以直接使用数据存储和下载功能。

如何使用

这些工件可以通过BOM从Maven Central和Spring Release存储库中获得：

<dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework.cloud</groupId>
            <artifactId>spring-cloud-alibaba-dependencies</artifactId>
            <version>0.2.0.RELEASE</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>

下一步是什么？

火箭MQ

Apache RocketMQ™是一个基于Java的开源分布式消息传递和流计算平台，具有高性能和高吞吐量。以来spring-cloud-stream-binder-rocket这些模块基于Spring Integration和Spring Cloud Stream ，开发人员可以在使用Spring Cloud Stream和Spring Cloud Bus时将RocketMQ用作消息传递中间件。

AliCloud SchedulerX

阿里云SchedulerX是阿里巴巴中间件团队精心设计的分布式任务调度工具，支持重复性任务和在指定时间触发的任务。

阿里云SLS

AliCloud SLS是一站式日志管理服务，是一种经过验证的工具，可在阿里巴巴集团内部应对大数据挑战的浪潮中蓬勃发展。无需费心任何代码，您就可以快速收集，使用，交付，查询和分析日志数据，从而提高操作，维护和营销效率，并获得处理大量日志数据的能力。在DT时代。
