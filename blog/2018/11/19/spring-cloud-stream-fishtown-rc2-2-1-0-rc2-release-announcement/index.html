Spring Cloud流鱼镇。RC2 / 2.1.0。RC2发布公告

工程
奥列格·朱拉库斯基（Oleg Zhurakousky）
十一月19，2018

我们很高兴地宣布Spring Cloud Stream Fishtown发布火车的第二个候选发布者-Fishtown。RC2 / 2.1.0。RC2。

Spring Cloud Stream Fishtown 2.1.0。RC2可在Spring Milestone存储库中使用。

作为后续版本2.0.0。RC2，此版本主要包含较小的增强功能和错误修复。
从目前的情况来看，这应该是General Availability版本2.0.0之前的最终版本候选。几周后发布。

注意：
如果应用程序是通过Spring Initializr创建的，则它们需要在spring-cloud BOM声明之前在maven依赖管理中添加此BOM表片段，否则最终将获得最新快照：


           org.springframework.cloud
           spring-cloud-stream-dependencies
           Fishtown.RC2
           pom
           import
