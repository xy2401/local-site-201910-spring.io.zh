Spring Cloud Task 2.1.0。M1现在可用

发布
格伦·伦弗罗（Glenn Renfro）
十一月05，2018

我们很高兴地宣布，Github和Pivotal下载存储库现已提供Spring Cloud Task 2.1.0M1。非常感谢所有对此版本做出贡献的人。

什么是新的？

Spring Cloud Task 2.1.0。M1旨在成为与Spring Boot 2.1.0保持一致的框架版本。从2.0.x开始的更新包括：

  • 更新所有依赖项。
  • 现在可以通过自动配置启用Spring Cloud Task。
  • 的exitCode的TaskExecution是null当任务正在执行时。
  • Spring Cloud Task可以在Java 8、9、10、11上编译并运行。

让我们更详细地介绍这些更新。

更新到所有依赖项

如前所述，此版本使Spring Cloud Task与Spring Boot 2.1保持一致。这包括更新所有依赖项。

现在可以通过自动配置启用Spring Cloud任务

再见@EnableTask向自动配置问好！现在可以通过自动配置启用Spring Cloud Task。要使用它，请添加spring-cloud-starter-task依赖于您的Maven pom或Gradle构建文件。

任务执行的exitCode执行任务时为空

在Spring Cloud Task的先前版本中， TaskExecution从Spring Cloud Task方法返回的结果始终显示正在运行的任务exitCode原为0 。现在，从Spring Cloud Task返回的TaskExecutions具有一个exitCode的null如果任务仍在执行。

我们期待您在Github ， StackOverflow ， Gitter或直接通过Twitter到@michaelminella或@cppwfs上对这些新功能的反馈！
