项目更新：用于Apache Mesos和Apache YARN的Spring Cloud数据流

新闻与活动
马克·波拉克
2018年11月1日

尊敬的Spring社区，

Spring Cloud Data Flow团队对Apache Mesos和Apache YARN的Spring Cloud Deployer和Spring Cloud Data Flow实现非常满意。

现在，我们认为TrustedChoice.com将是Apache Mesos实施Spring Cloud Deployer和Spring Cloud Data Flow的更好的家，并且我们将项目捐赠给他们以推动其发展。

现在，开发工作将由TrustedChoice.com的团队（ Adam J. Weigold ， Phil Egelston ， Justin Mathieu和Cole Anderson ）直接管理，因为Spring Cloud Data Flow团队将不再维护它。

有关更多详细信息，请参考TrustedChoice.com的GitHub存储库中的项目：

  • 适用于Mesos的Spring Cloud Deployer

  • Mesos的Spring Cloud数据流

但是，对于Apache YARN，由于我们将工作重点放在本地，Kubernetes和Cloud Foundry实施上，因此Spring Cloud Data Flow团队已停止对其进行维护。Spring Cloud Deployer和Spring Cloud Data Flow项目的Apache Yarn实施已于今天（2018年11月1日）正式达到报废（EOL）状态。从宣布的日期（即2019年11月2日）起，一年后，这些项目将正式获得一般支持（EOGS）。

如果对社区感兴趣，我们想开放它并捐赠与Apache Mesos实现类似的相关项目。如果您有兴趣分叉该项目以继续开发和维护它，我们将很乐意交叉链接并推广您的分叉的项目。请在Gitter ping我们进行讨论。

祝一切顺利，

Spring Cloud数据流团队
