SpringOne2GX 2014重放：企业中的Cassandra，Couchbase和Spring Data

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年1月20日

记录在SpringOne2GX 2014中。

演讲者：马修·亚当斯（Matthew Adams）（SCI Spike），迈克尔·尼采格（Michael Nitschinger）（Couchbase）

数据/集成跟踪

幻灯片： http ： //www.slideshare.net/SpringCentral/cassandra-couchbase-andspringdataintheentprise

Spring Data Cassandra将Cassandra支持添加到项目的Spring Data框架中，提供了Spring Data熟悉的Repository概念和POJO持久性。本讲座将首先关注基于Cassandra的POJO持久性，包括自动Cassandra模式生成以及同时使用XML和Java的Spring上下文配置。然后，该演讲将更深入地探讨Spring Data Cassandra所基于的一些较低级别的功能（又名“ Spring CQL”），这些功能使普通的，古老的Cassandra开发变得更加容易。Couchbase Server是NoSQL领域的领导者之一，在企业和初创企业中使用率很高，而低延迟甚至是每秒数十万次操作都非常重要，因此Couchbase Server被广泛使用。本演讲不仅将简短介绍采用Couchbase的好处，还将展示如何通过对Spring Data的支持将其集成到Java Enterprise领域。您将学习如何通过应用程序通过实体和存储库从应用程序完全控制数据库架构，如何部署新应用程序版本或扩展群集而不会造成一秒钟的停机，以及如何轻松地与Elasticsearch集成。
