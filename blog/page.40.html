Spring Framework 5.1.1、5.0.10和4.3.20现在可用

发布
斯特凡·尼科尔（StéphaneNicoll）
十月15，2018

我很高兴代表团队宣布Spring Framework 5.1.1、5.0.10和4.3.20 现已可用 。

5.1系列的第一个维护版本包含30多个修复和改进 。Spring Framework 5.0.10 还包含20项广泛的修复和改进，而Spring Framework 4.3.20 是一个相当小的补丁版本 。

基于Spring Framework 5.1.1的Spring Boot 2.1的第一个候选版本指日可待！我们还将在本周晚些时候跟进相应的Spring Boot 2.0.6和1.5.17版本。
