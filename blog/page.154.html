SpringOne2GX 2015重播：莱迪思：您的 Spring 应用程序的云原生平台

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年10月19日

记录在SpringOne2GX 2015
演讲者：Matt Stine
幻灯片： http ： //www.slideshare.net/SpringCentral/lattice-a-cloud-native-platform-for-your-spring-applications
Lattice是一个云原生应用程序平台，使您能够通过Vagrant在本地计算机上的Docker等容器中运行应用程序。莱迪思的功能包括：
集群调度
HTTP负载均衡
日志汇总
健康管理
莱迪思通过打包在Cloud Foundry弹性运行时中找到的组件的子集来做到这一点。结果是适合于快速应用程序开发的开放式单租户环境，类似于使用Lattice开发的Kubernetes和Mesos应用程序应将其原样迁移到完整的Cloud Foundry部署。

SpringOne2GX 2015重播：Spring Cloud入门

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年10月19日

记录在SpringOne2GX 2015中。

演讲者：Dave Syer博士，Josh Long

幻灯片： http ： //www.slideshare.net/SpringCentral/getting-started-with-spring-cloud

Spring Cloud 1.0就在这里！它提供了一种创建和使用微服务的强大方法。当您引入新服务时，您会遇到集成问题：服务可能不稳定，它们可能消失，并且-因为它们通常通过HTTP公开-与过程中方法调用相比，它们需要更多的工作量。在本次网络研讨会中，我们将重点关注Spring Cloud如何将服务注册（例如Eureka），声明式REST客户端（具有Netflix的Feign），Hystrix的反应式编程和断路器模式集成在一起，以支持简单，强大的服务到服务调用，以及使用Spring Cloud Stream传递消息微服务。这是对如何建立连接和使用微服务的深入探讨，并且是在使用Spring Boot构建微服务的介绍之后自然而然的下一步。

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年10月19日

记录在SpringOne2GX 2015中。

演讲者：Alex Snaps，Chris Dennis-Terracotta / Software AG

幻灯片： http ： //www.slideshare.net/SpringCentral/bringing-javaxcacheing-to-your-app

制作了10年，终于完成了JSR-107。现在，我们有了用于Java平台的标准化缓存API。这样就可以在Spring应用程序中使用了。但是，它真正为您带来了什么？这些可选功能是什么？没有交易支持？您如何有效利用这个新的API解决当今应用程序中的现实问题？我们将研究如何最好地开始将缓存引入Spring应用程序以解决实际问题。并且，当我们更加详细地探索javax.cache API时，我们将看到如何将规范推到极限甚至更多。无论是隐式地通过框架（例如Spring和Hibernate），还是显式地（通过缓存，通过……）。以及当您需要超出规范涵盖的范围（例如事务性缓存）时，如何从给定的提供程序中抽象自己。我们还将介绍一些在选择实现时可能需要考虑的实现者细节，尤其是当您计划将规范推到极致时，例如在分发和缓存TB级数据时。

2015年10月19日

记录在SpringOne2GX 2015中。
