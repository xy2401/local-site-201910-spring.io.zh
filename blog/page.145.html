Spring Cloud Stream 1.0.0。M4现在可用

发布
马里乌斯·波哥维奇（Marius Bogoevici）
2016年2月6日

我很高兴代表团队宣布Spring Cloud Stream 1.0.0的发布。M4 。新版本进行了一些重大更改和增强，它定义了核心抽象和原语，我们认为这些抽象和原语对于分布式实时数据处理应用程序的开发至关重要。列举其中一些：

默认的发布-订阅语义

Spring Cloud Stream 1.0.0中最重要的更改。M4是应用程序相互交互的方式。我们选择了默认的发布-订阅模型，在该模型中，每个从给定目标接收消息的应用程序都将收到消息的副本。这更适合流应用程序的处理模型，在该模型中，中间主题充当各种相交数据流的数据集线器，还使轻拍等场景变得更加高效。此功能与下一个消费者群体紧密相关 。

2016年2月2日

记录在SpringOne2GX 2015中。

Spring Statemachine 1.1.0。M1发布

发布
珍妮·瓦尔基亚拉赫蒂
2016年2月2日

我们很高兴地宣布Spring Statemachine 1.1.0的第一个里程碑版本。M1。

我们发布了1.0.0。2015年4月M1和1.0.0。2015年10月发行。我们还为1.0.x系列完成了2个维护版本，现在是时候使用1.1.x系列了。1.0.x进入维护模式，而主要开发将集中在1.1.x。这并不意味着我们将停止使用1.0.x，对其进行维护修复，直到我们成功过渡到1.1.x以上版本为止。在Spring项目中，重要的是在用户准备好进入下一个次要/主要版本之前，不要放弃维护。我们期望1.0.x和1.1.x并存，直到在可预见的将来解雇1.2.x或2.x系列。
