将Spring Cloud Apps从Spring Boot 1.2迁移到1.3

工程
戴夫·瑟
2015年11月25日

Spring Boot 1.3中有一些有趣的新功能，现在可以在Brixton发行版的Spring Cloud中使用。Spring Cloud的Angel发布系列与Spring Boot 1.3部分不兼容，因此在升级时，需要注意一些重要事项。本文可帮助您浏览更改并更新任何现有应用程序以使用新功能。通常，在尝试将Spring项目的新版本引入现有代码库时，它通常也应有所帮助。

提示：您可以使用mvn dependency:tree要么gradle dependencies列出项目中的依赖项并检查版本。

春天的这个星期-2015年11月23日

工程
乔什·朗（Josh Long）
2015年11月24日

欢迎来到春季本周的另一个装置！在美国，这是感恩节的一周，希望-我们花一点时间来感谢许多事情。我可以肯定地说，我代表整个团队在Pivotal发言：亲爱的社区，我们感谢您。谢谢。

因此，和往常一样，我们还有很多事情要做。实际上，在火鸡过后的懒惰时间里，情况会更好！

  • Mark Pollack博士刚刚宣布发布Spring XD 1.3 GA和Spring XD 1.0的Spring Flo 。这是一个巨大的版本！春天XD 1.3.0支持作业组成的DSL，弗洛春春XD，改进管理用户界面，星火1.3.1更新到春天的数据和Spring Hadoop的分布，一个卡桑德拉水槽和更多 ！这也是Spring XD完成其已部分部署在Spring Cloud Stream和Spring Cloud Dataflow中的架构之前的最后一步！这是技术的令人激动的发展！
  • Spring Security和Spring Session的负责人Rob Winch刚刚宣布了Spring Session 1.1.0。M1具有许多新功能：按用户名搜索HTTP会话， HttpSessionListener支持，定制Cookie创建，覆盖默认值RedisSerializer ，以及Hazelcast的配置和支持。
  • 令人惊叹的Andy Wilkinson刚刚宣布了Spring IO Platform 2.0.0。发布 ！Spring IO平台是一种管理依赖关系的好方法，就像Spring Boot依赖关系管理部分所做的那样，但是着眼于支持a）更长的稳定性和b）支持所有 Spring项目，而不仅仅是支持那些由Spring Boot提供。这是管理应用程序的绝佳解决方案。
  • Spring Statemachine负责人Janne Valkealahti 刚刚宣布了1.0.1 ，其中包括一个新的@EnableStateMachine注释，对4.2.2的更新以及一些修复。
  • Pivotal社区的朋友Dan Vega谈论了他刚刚发布的有关使用Spring Boot的Udemy课程
  • Mattias Severson的帖子解释了如何使用Gradle Spring Boot插件来确保一个项目（一个不使用或不依赖Spring Boot的项目）获得与该项目相同的版本
  • Kai Toedter编写的带有Spring Boot，AngularJS和TypeScript的幻灯片酷Web应用程序似乎很有趣！
  • Stormpath Java开发人员传教士Micah Silverman刚刚在博客上发表了有关他们新的Stormpath Spring Boot自动配置的博客 。幻灯片也可以在线获得
  • Activiti忍者Joram Barrez在博客中写道， 最近在Spring Initializr中 添加了Activiti自动配置 ，并附带了一个不错的演示！
  • Dan还做了一个很好的视频，介绍了席卷全国的Spring Boot 1.3中的一项功能：彩色ASCII艺术横幅 ！

发布
珍妮·瓦尔基亚拉赫蒂
2015年11月20日

我们很高兴宣布Spring Statemachine 1.0.1的第一个维护版本。可以从Maven Central或从Spring Repository获得工件。

版本1.0.1中的更改（2015-11-18）

  • 通过@EnableStateMachineFactory自动启动不起作用[＃113]
  • 更新到框架4.2.2 [＃115]
  • 与@EnableStateMachine和@WithStateMachine关联[＃116]
  • 当有2种状态机配置时，EnableStateMachineFactory失败[＃117]
  • PersistStateMachineHandler返回事件结果[＃118]
  • NPE缺少配置器[＃119]
  • 上下文事件不适用于StateMachineBuilder [＃120]

发布了Spring XD 1.3 GA和适用于Spring XD 1.0 GA的Flo

工程
马克·波拉克
2015年11月19日

今天，我们很高兴地宣布Spring XD 1.3和Flo for Spring XD 1.0的全面上市。

  • Spring XD 1.3 GA： zip ， brew和rpm 。
  • 适用于Spring XD 1.0 GA的Flo： zip 。

除了修正错误外，我们还在1.3版本中添加了一些新功能

  • 作业组合DSL允许创建作业执行的复杂图形。
  • Spring XD Designer的Flo支持创建组合作业 。
  • 管理员界面支持组合作业的执行历史记录。
  • Cassandra Sink和Header Enricher处理器
  • Gpfdist接收器现在支持更新操作和所有控制文件选项
  • Spark 1.3.1支持
  • 用于刷新写入HDFS的超时值，以确保数据保留在HDFS DataNode的磁盘上。
  • 常规依赖项升级，Spring Data Gosling，SI 4.2和Boot 1.2
  • Hadoop发行版更新到Apache Hadoop 2.7.1和Hortonworks Data Platform 2.2。Pivotal Hadoop 2.1、3.0和Cloudera Hadoop 5.3
