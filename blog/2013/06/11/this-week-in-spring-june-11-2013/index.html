Spring 的这个星期-2013年6月11日

工程
乔什·朗（Josh Long）
2013年6月11日

大家好，欢迎来到春季本周的另一期！本周，我在纽约州纽约市，与NYC Java Meetup和ScalaDays的开发人员就Spring进行了交谈。这个月我们有很多网络研讨会，所以一定要检查一下
列出下面的详细信息！

  1. 是否想通过SpringOne 2GX 2013 ？如果您是春季冠军， 请在我们的冠军论坛上炫耀您的东西，并在 2013年6月21日之前按照以下说明进行操作 。您
    可能是我们5位幸运获奖者之一！（如果您是Groovy＆ Grails或Cloud Foundry的冠军 ，请别担心，我们将为您推出未来的竞赛！）

  2. Oliver Gierke宣布了Spring Data Babbage，这是下一届Spring Data发布火车的第一个里程碑 。此版本包括一个
    许多新功能，因此请务必查看发行说明！

  3. Spring Batch 2.2.0现已发布 ！这是一个主要版本，除了许多其他功能之外，还支持Spring Data，Java Configuration，AMQP和SQLFire。Spring Batch和我们
    参与专家组已为最近确定的JSR-352规范提供了大量信息 。

  4. 基于Eclipse Kepler 4.3， Spring Tool Suite和Groovy / Grails Toool Suite 3.3.0 M2已发布。此里程碑版本改进了Java配置支持，并且是Spring Framework 4
    准备。

  5. Craig Walls宣布Spring Social Twitter 1.0.5和Spring Social 1.0.3已发布 ！新版本修复了一些错误，并且正在
    预计将弃用Twitter API的1.0版本。

  6. 加里·罗素（Gary Russell）还宣布了Spring Integration 2.2.4和2.1.6维护版本 ，以合并上述的Spring Social Twitter更新。

  7. 我将在星期四进行有关Spring的RESTful服务设计的网络研讨会。与往常一样，将举行两场会议-一场在格林尼治标准时间下午3点，另一场在太平洋标准时间上午10:00-可以容纳尽可能多的时区
    可能。该网络研讨会将介绍用于构建RESTful服务的Spring堆栈。我们将从一个简单的API开始，然后进行API扩展，通过Spring HATEOAS引入Hypermedia控件，通过Spring Data REST引入基于约定的基于存储库的API，通过Spring Security OAuth和Spring Social引入安全性，并解决文件上传等常见情况，异常处理，记录分页和Ajax。我期待在那里见到你！

  8. 乔恩·布里斯宾（Jon Brisbin）将于2013年6月18日介绍Introducing Reactor-一种用于JVM上异步应用程序的框架，不要错过。反应堆提供了
    在执行相当小的无状态异步处理块时需要高吞吐量的应用程序的基本框架。

  9. 加入6月27日来自自由大学的Tony Erksine，他指导我们如何交谈Spring and Impact People ，这是一门关于软技能和技术采用策略的实用课程，需要使公司中的其他开发人员兴奋并使用新技术，
    在这种情况下，使用Spring。

  10. 我在三月份的春季 DevNexus大会上发表了最新的演讲。
    3月的3.1、3.2和4.0     ，该演讲现在可以在InfoQ上在线获得。请检查演讲，但也请确保从我的SlideShare帐户上的JAXConf演讲中查阅该平台的最新版本。

  11. 转至Pivotal博客，获取有关Hadoop编程的简短入门 ，该教程将引导您完成一个简单的字数统计程序。该示例着眼于
    规范的字数问题，然后研究生态系统中的其他解决方案，例如Pig，Hive和Cascading。本系列的下一个博客将为新手介绍Spring for Apache Hadoop，为本文中讨论的四种不同方法提供统一，一致的替代方法。

  12. 在相关新闻中，如果您在纽约地区，请在星期三晚上与我一起在NYCJava聚会的10代办公室进行同样的演讲。再次感谢您进入10gen，
    MongoDB背后的公司，用于主持聚会。
  13. JavaBeat博客上有一篇不错的文章，介绍了如何使用Spring强大的分段文件上传支持 。 Spring 的支持
    提取用于文件上传的常用API-包括commons-fileupload API和Servlet 3 API-并允许您将这些API用于HTTP miltipart编码的文件上传，通常在Web应用程序或REST服务中使用。绝对值得
    阅读，看看！

  14. 伊丹·弗里德曼（Idan Fridman）将Spring Integration中某些常见类型的组件 （包括splitter ，
    变压器 ， 聚合器等等。

  15. 对于那些希望将Spring技能提高到新的专业水平的人，SpringSource刚刚发布了7月份的Professional Spring培训时间表。

  16. Spring教程博客上有一篇很棒的文章， 介绍Spring的@Async和@Scheduled注解 。
