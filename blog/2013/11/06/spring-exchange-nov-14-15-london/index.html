Spring Exchange为期两天的会议

探索新事物…Spring 4.0 JAVA 8

Juergen Hoeller将在开幕主题演讲中说明选定的Java 8
Spring编程模型中的功能，并探讨对
应用程序架构-包括lambda
表达式，JSR-310日期和时间，参数名称发现以及
java.util.concurrent增强。在这里找到更多。

使用Spring Boot的可量产的Spring应用程序

David Syer将为您提供有关Spring Boot的详细介绍
用于构建下一代Spring应用程序。Spring Boot旨在
使创建基于Spring的生产级应用程序变得容易，并且
最小的麻烦的服务。在这里找到更多。

发现使用Spring XD创建大数据应用程序有多么容易

Mark Fisher将展示Spring XD，这是一个旨在解决大型项目的新项目
数据复杂度。XD基于Spring技术构建，可提供可扩展的运行时
可通过简单的DSL轻松配置和组装的环境。 在这里找到更多。

使用Spring Framework 4.0的WebSocket简介

Rossen Stoyanchev将提供有关WebSocket协议的全面更新
Spring开发人员的观点。他将讨论JSR-356的优势和
限制，WebSocket运行时的当前状态，最重要的是
将介绍全面的WebSocket和
Spring Framework 4中的SockJS支持。在这里找到更多。
