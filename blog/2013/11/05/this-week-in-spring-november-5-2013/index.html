春季本周-2013年11月5日

工程
乔什·朗（Josh Long）
2013年11月5日

欢迎来到春季本周的另一期！我在瑞典马尔默举行的梦幻般的Øredev会议上 ，与开发人员谈论Spring ， Cloud Foundry等问题！下周我将在JMaghreb ，然后在Devoxx 2013上 ，我将与开发人员讨论Spring， Cloud Foundry ， RabbitMQ以及我的朋友Dave Syer博士 ， Alvaro Videla和Andy Piper 。期待在这些地方见到您！如果您想谈论Spring，云，大数据以及其他内容，请在Twitter（@starbuxman）上与我联系。

有了这个，我们在本周有很多很棒的东西，所以让我们开始吧！

  1. 首先，大新闻： Spring Framework 4.0 RC1已发布！新版本包括与OpenJDK 8开发人员预览版的完全兼容性，Groovy配置的Bean，Spring MVC的改进，更智能的代理，时区分辨率（连接到JSR 310和Joda-Time），Spring HATEOAS启发的链接构建，改进的Websocket支持，和更多 ！获取相关信息，测试您的应用程序，并确保让我们知道您的喜好！
  2. Spring Security的负责人Rob Winch再来一次，这次是Spring Security 3.2.0 RC2发行版 ，其中包含针对80多个问题的解决方案，以及一系列更新-有些突破，因此请务必进行更新和验证。
  3. 此外，Rob刚刚发布了Spring LDAP 2.0.0。M1 ，添加了许多新功能，包括Spring数据存储库和QueryDSL支持，流利的LDAP查询构建器，用于LDAP的自定义XML名称空间，Java 5功能以及一些重组
  4. PS：如果您碰巧在伦敦地区或交通便利，请查看Spring eXchange ：11月14/15日为期两天的紧凑表演，由主要的Spring工程师介绍最新和最精彩的-当然包括Spring Framework 4.0 ！
  5. 我在JavaOne上做了一个名为Spring Update的演讲，该演讲探讨了Pivotal的Spring部门带来的许多很棒的新事物。该演讲现已在Parleys.com上在线发布，请查看！
  6. Paul Chapman撰写了一篇非常不错的文章，解释了Spring MVC中的错误处理策略
  7. 来自Magnolia的Daniel Lipp和Tobias Mattsson做了一个不错的网络研讨会，介绍了如何将Magnolia CMS与基于Spring和Grails的应用程序集成 。酷的东西！
  8. Greg Turnquist继续讲述Spring Boot的故事，并讨论如何向自定义Spring Boot模块添加属性支持 。
  9. 博客赫伯特奥利维拉已经把一种史诗般后使用Spring MVC， Spring Data ，春季安全，引导和Angular.js在一起 。绝对值得一看！
  10. 我很高兴看到Thymeleaf 2.1支持自动Spring Security CSRF集成 。不错的工作！
  11. Spring的文档现已移至Asciidoc ！这很有趣，主要是因为现在可以更轻松地更新和维护文档，因此您可以期待获得更好的文档！
  12. Dave Syer博士向Spring Boot致谢：任何类型的bean TomcatConnectorCustomizer在Spring Boot应用程序中将为您提供对Connector然后可以用来自定义嵌入式Tomcat实例。一种可能的应用是将SSL应用于嵌入式Tomcat实例 。
  13. 说到Spring Boot，请查看此Spring Boot Java应用程序 ，这是一个运行在@CloudFoundry上的“功能齐全”的参考应用程序，旨在监视构建。干得好， 本·黑尔 ！
