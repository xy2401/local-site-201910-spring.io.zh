班加罗尔春季最佳实践和Cloud Foundry活动

新闻与活动
马克·巴尔
2013年11月21日

我们很高兴地宣布，我们将于11月24日（星期日）在班加罗尔的SpringPeople办公室举办免费的Spring和Cloud Foundry会议，时间为上午10.00到下午12.30。

您可以通过单击此处进行注册，请确保及时注册，因为我们免费提供此活动的席位有限。

活动环节包括：

  • Cloud Foundry：使用Spring将您的应用程序部署到云端作者：Paul Chapman

保罗·查普曼 （ Paul Chapman）是我们澳大利亚服务提供部门的高级顾问，负责为亚太地区的客户提供咨询，培训和指导。他于2007年加入SpringSource，自那时以来，已经为1000多名学生教过Spring。

Paul将研究用于将应用程序开发和部署到云中的Spring参考架构，尤其是Cloud Foundry ，它是Pivotal的开源PaaS。这些参考架构涉及许多Spring项目，例如Spring Data，Spring Social和Spring Mobile，以演示数据密集型，可以在多个设备上使用并可以部署到云中的现代软件架构。

一旦部署了这些应用程序，我们将展示Cloud Foundry如何帮助您轻松管理这些应用程序。

  • 春季最佳实践：从Spring Petclinic到Michael Datavy的Spring Data Hadoop

Michael Isvy于2008年加入SpringSource（Spring背后的公司，现在是Pivotal的一部分）。从那时起，他一直在10个不同国家的1000多名学生中教授Spring。他曾在Spring的许多会议上发表演讲，并且是spring.io博客上的活跃技术博客作者。Michael担任SpringSource / Pivotal亚太地区教育经理。

他最近研究了新的Spring-Petclinic示例应用程序，该应用程序旨在在Web应用程序中使用Spring和Spring MVC时促进最佳实践。他将使用此应用程序展示Spring最佳实践。他还将带来关于Spring最佳实践如何随着时间演变的有趣观点。
