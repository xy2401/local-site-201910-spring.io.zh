适用于Apache Hadoop 1.0.2 GA和2.0 M2的Spring已发布

发布
托马斯·里斯伯格
2013年11月14日

我们很高兴宣布Apache Hadoop 1.0.2 GA和2.0 M2里程碑发行版的春天。

这两个版本都提供了错误修复，Spring项目版本的升级以及对新的Pivotal HD 1.1版本的支持。1.0.2版本增加了对新的Hadoop 2.2稳定版本的支持。2.0.0。M2版本提供了对CDH4 YARN群集的支持，并且为HdfsResourceLoader提供了改进的实现（此实现将反向移植到下一个1.0.3 GA版本）。

我们将继续在Spring IO里程碑存储库中提供特定于版本的工件及其各自的传递依赖项。1.0.2。发布和1.0.2。Maven Central也提供RELEASE-hadoop22版本。

对于1.0.2。发布：
-1.0.2。发行（默认-Apache Hadoop稳定版1.2.1稳定版）
-1.0.2。RELEASE-hadoop22（Apache Hadoop 2.2.0稳定）
-1.0.2。RELEASE-cdh4（Cloudera CDH 4.3.1 MR1）
-1.0.2。RELEASE-hdp13（Hortonworks HDP 1.3）
-1.0.2。RELEASE-phd1（Pivotal HD 1.1）

请参阅Wiki页面以获取构建配置说明： https : //github.com/spring-projects/spring-hadoop/wiki

对于2.0.0。M2：
-2.0.0。M2（默认-Apache Hadoop稳定版1.2.1）
-2.0.0。M2-hadoop20（Apache Hadoop 2.0.6 alpha）
-2.0.0。M2-phd1（Pivotal HD 1.0）
-2.0.0。M2-cdh4yarn（CDH 4.3.1纱）

我们计划在下一个2.0里程碑版本中增加对新Apache Hadoop 2.2.0稳定版本的支持。

Apache Hadoop项目确实对Hadoop 2.1 beta版本中的YARN代码做了一些API更改，因此这将导致与基于Hadoop 2.0 alpha的代码向后兼容的问题。

有关特定项目的更多信息，请参见项目页面http://projects.spring.io/spring-hadoop/ 。
