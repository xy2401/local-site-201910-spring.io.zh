Spring 的这个星期-2013年11月19日

工程
乔什·朗（Josh Long）
2013年11月19日

欢迎来到 Spring 本周的另一版！和往常一样，我们有很多内容要讲，所以让我们开始吧！

我想鼓励大家在21日（这个星期四！）加入我。在我的网络研讨会上， 您最近见过 Spring 吗？ 。自从我们成为Pivotal冒险的一部分以来，如果您还没有非常关注 Spring 大世界中发生的一切，那就值得一看 ！即使有，我敢打赌，无论如何，您都会学到一些新的和令人兴奋的东西。 :) C'ya在那里！

  1. 我们发表了一篇文章- 您最近见过 Spring 吗？ -今天在Pivotal博客上陪同网络研讨会。
  2. 马克·巴尔斯（Mark Baars）刚刚宣布，我们刚刚在培训中添加了具有Hadoop和Pivotal HD的大数据 。太棒了！
  3. Spring Data忍者Thomas Risberg刚刚宣布了1.0.2。适用于Apache Hadoop的Spring GA发行版现已发布！
  4. Spring XD忍者Gunnar Hillert刚刚宣布了Spring XD 1.0.0。M4刚刚发布 。新版本中包含许多新内容，因此请务必查看此版本！
  5. Spring Data主管Oliver Gierke和Spring Data Couchbase主管Michael Nitschinger刚刚宣布了Spring Data Couchbase模块的里程碑2的可用性！Michael还跟进了最初但很有希望的请求，即请求Spring Boot支持Spring Data Couchbase ！ :)
  6. 对最近的Reactor 1.0版本感到兴奋，并想了解更多？快来参加将于2013年12月3日举行的由Reactor负责人Jon Brisbin主持的网络研讨会 ！我会在那里，热切地听着，我也希望在那里见到你。
  7. Spring and Cloud Foundry的忍者Matt Stine将于12月10日举行名为“ 具有不变性的 Spring ”的网络研讨会，该研讨会将介绍Spring如何帮助Josh Bloch的经典有效Java书籍中的第15项 ： 最小化可变性 。
  8. New Relic： Live Java Web应用程序故障排除中的SpringOne2GX2013会话的重放现在可以在线获得 。
  9. Spring Data GemFire 1.3.3已发布！ 。不过，还没有发布说明。
  10. 请查看此Pivotal CF演示-框中的交钥匙Cloud Foundry，以进行内部部署 。
  11. 使用Redis？我们爱Redis！我们发现这是几年前的出色文章，Redis可以轻松解决各种用例 。请享用！
  12. Pinterest 刚刚启动了他们的第一个Web API 。到目前为止，关于此提及还没有与Spring相关的内容，但是我只是把它扔在那里，因为我押注着我们将在多久后看到社区中某人创建Spring Social Pinterest绑定！ :)
  13. 友善的提醒：您可以从Homebrew获得很多很棒的Spring / Pivotal东西， 充实 Java堆栈：

    $ brew tap pivotal/tap; $ brew install springboot springxd rest-shell maven groovy redis rabbitmq tomcat

  14. 与Spring团队工程师Greg Turnquist一起，我真的很喜欢这种眼神 。
  15. 由我们在Skills Matter上的朋友举办的Spring Exchange最近在伦敦举行。有很多精彩的演讲，而且大多数（全部？）其中已被记录并发布到网上 ！请享用！
