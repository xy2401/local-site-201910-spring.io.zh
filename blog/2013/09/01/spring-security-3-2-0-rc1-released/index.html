    Spring Security 3.2.0。RC1发布

    发布
    罗伯·温奇
    2013年8月31日

    Spring Security 3.2.0。RC1现在可以从SpringSource存储库中获得： http://repo.springsource.org 。请参阅此处 ，以获取有关通过Maven解决这些工件的快速教程。


    此版本包含大量更新和修复。重点包括：


  • 完善Spring Security Java配置

    • 使用内容协商来确定在启用多种身份验证机制（即HTTP Basic和Form登录）时如何提示用户进行身份验证

    • AbstractSecurityWebApplicationInitializer允许直接注册Java配置

    • 修复了许多错误



  • CSRF保护以及与Spring Web MVC jsp标签的自动集成

  • 自动缓存控制支持

  • 防御点击劫持攻击

  • HTTP严格传输安全性支持可减少中间人攻击

  • 样本包含pom.xml，因此可以将它们导入为Maven项目

  • MediaTypeRequestMatcher用于通过内容协商对请求进行匹配

  • 样本目录中已集成了十多个Java配置样本

  • 三个新指南可引导用户逐步了解示例，并提供有关如何执行特定任务的详细说明。这些指南中的更多指南将在以后的版本中发布

    • 参考Spring Security 3.2.0。RC1预览以获取有关此版本的更多详细信息。

    SpringOne2GX

    要了解Spring Security 3.2的所有新功能，请参加2013年9月9日至12日在SpringOne2GX上发表的Spring Security 3.2入门 。如果您还没有拿到票 ，现在就来不及了！
