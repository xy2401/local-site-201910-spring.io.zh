• The future is now and it has a name and version - Spring 4.0. Juergen Hoeller outlined the proposed next iteration of Spring, Spring Framework 4.0!
  • If you missed the live webinar of the Spring Framework 3.2 GA release, check out the replay, which also has a brief discussion Spring Framework 4.0.
  • Costin Leau has announced that Spring for Apache Hadoop 1.0 RC2 is now available! The new release includes dedicated namespace support for Cascading, enhanced compatibility with Cloudera CDH3 and 4, enhanced integration with Spring Batch and Hadoop, safer job cancellation and improved documentation.
  • Andrew Clement announced that the 0.3 release of the Scripted editor is now available. There are many new features, including easier installation (now available through Node Package Manager - npm).
  • Roy Clarkson has announced that Spring Mobile 1.1.0.M2 has been released with many new features including a Spring MVC ViewResolver that resolves the view based on Device and SitePreference, and then forwards to the correct, delegate ViewResolver.
  • Join Scott Andrews as he discusses the role of Javascript in an exciting webinar on January 24, 2013: Architecture of a Modern Web App.
  • Join Brian Cavalier and John Hann as they discuss Inversion of Control (IoC) and Javascript in a Webinar on January 31, 2013: IOC + Javascript.
  • Martin Lippert, Andy Clement, and Andrew Eisenberg's presentation from SpringOne2GX 2012, Tooling for the JavaScript era, is now available online from InfoQ.
  • The SpringOne 2GX 2012 replay of Automated Provisioning of Spring Applications to EC2 and VMware Cloud is now available.
  • Nicolas Frankel has a nice post on how to inherit configuration from parent bean definitions to reduce redundant configuration. Nice!
  • Arnaud Cogoluègnes has a nice post on how to use the Spring MVC test framework
  • Taking the example further, Arnaud also shows how to test Spring MVC REST controllers using the Spring MVC test framework.
  • Tomasz Nurkiewicz has put together a Spring Data Repository implementation for JDBC. That's really cool! I can't wait to give this a try!
  • Tobias Fiohre has a nice introduction to the ideas enabled by Spring HATEOAS (ROCA, or resource-oriented client architectures) complete with a sample application and code! Nicely done, Tobias!
