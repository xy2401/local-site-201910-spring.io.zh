春天的这个星期-2013年1月8日

工程
乔什·朗（Josh Long）
2013年1月8日

欢迎回到春季本周的另一期。和往常一样，我们有很多事情要看，因此，不用多说。

  1. GigaOM现在对大数据生态系统中的一些令人兴奋且重要的工具进行了很好的总结。
    有许多不同的工具可用于用例的不同部分，

    and of course Spring Data is mentioned in there, too!
  2. The ODBMS Industry Watch blog and professor Roberto V. Zicari has a nice interview with Spring ninja David Turanski on Spring Data. Definitely a good read!
  3. Robin Varghese has put together a straight-forward blog on sending email with dynamic templates driven by Velocity. This seems to be a common use case, and it's always good to find more up-to-date resources. Many people will no doubt have found Matt Raible's post from 9 years ago on the same subject, which is also very illustrative but, alas, a little outdated. Nice to see this more updated approach to an old classic! 

  4.  Hey, this is interesting! Did you guys know there was a Spring for Delphi (also, I guess, referred to as Spring4D)? 
Me either... @[email protected]  With that in mind, I found this post 
 introducing  some of the collection libraries that seem both very friendly and powerful in Spring for Delphi. 

   Cool!


  5. Eren Av？阿罗？？有一篇不错的文章，展示了如何在Spring中使用动态语言（尤其是Groovy） 。


  6. Dinesh Rajput汇总了一个博客，介绍了Spring Batch 2.x一代所独有的一些功能 -尽管Spring Batch 2.x已经问世多年，但始终值得一看！


  7. Java all and Sundry博客对Spring MVC的一种机制，即RequestMappingHandlerAdapter 。

  8. 亚历克斯·索托（Alex Soto）回来了，这次带着

    post on unit testing Spring Data  MongoDB with NoSQL Unit! Cool.



  9. Java Success博客上有一篇文章，介绍了企业集成的一些情况
    JVM社区中的解决方案，并演示了一些企业集成的概念，特别是Spring Integration 。

  10. Develop and Conquer博客有一个很好的例子，有一些技巧可以注入java.util.Date
    使用Spring Expression语言和@Value注解     。很酷！


  11. Abhijit Ghosh使用Tomcat，Maven和Eclipse（m2e）组合了一个不错的基于SpringMVC的HelloWorld应用程序 。
