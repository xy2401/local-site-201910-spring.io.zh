乔什·朗（Josh Long）
2013年1月29日 

Welcome back to another installation of This Week in Spring ! 
I've been visiting developers and companies in India, China, and Japan. It's been an exciting time
to see what these emerging and powerful countries are doing with open source and with Spring, in particular! 
Of course, stay tuned to the SpringSource blog in the coming weeks 
some very cool examples and details!

在此期间，像往常一样，我们有相当多的消息，本周回补，包括从Spring 4路线图公布更多的消息上周 。如果您想获得绝对最新的信息，请查看YouTube上的Spring 3.2 GA网络研讨会重播 ，其中将在最后讨论Spring Framework 4.0。让我们开始吧！




  1. InfoQ的Charles Humble 对Juergen Hoeller进行了很好的采访，并撰写了Spring 4的公告 。

  2. Spring Integration 2.2.1和2.1.5维护版本现已普遍可用。

    3.        
  3. Spring HATEOAS 0.4 was released, adding Jackson and HAL support.

  7. Java Code Geeks博客的Blogger Ilias Tsagklis在Spring 4.0路线图公告中也有不错的帖子。

  8. 克里斯·比姆斯（Chris Beams）宣布已发布Spring 3.1.4 ！

  9. Marty Pitt为Spring Web应用程序创建了一个非常不错的扩展-他称之为BakeHouse-
    在应用程序启动时预处理要在Web应用程序中使用的Web工件     。可以进行各种预处理：
    • 更少的CSS

    • 连接并最小化您的Javascript
    • 甚至可以使用一些Typescript或CoffeeScript ，需要对其进行编译

    马蒂，这是一个很酷的扩展！就像我一直希望像JAWR这样的东西一样！我最喜欢的东西是什么？流体使用
    Spring Java @Configuration类！真正的光滑和高产！

    宣布的Spring 4.0路线图
