Spring 的这个星期-2013年7月15日

工程
乔什·朗（Josh Long）
2013年7月16日

欢迎回到春季的本周的另一个安装。 与往常一样，我们有很多内容需要介绍，所以让我们开始吧！本周，我在SenchaCon上，与开发人员讨论构建RESTful应用程序和客户端的问题，然后下周前往OSCON，在那里我将主持Spring BOF，并就Spring 4的最新和最精彩的话题进行演讲。 ，并协助管理Pivotal摊位。如果您在SenchaCon或OSCON上， 请随时联系我 ，我们可以讨论Spring ， Cloud Foundry ， big-data等！

  1. SpringOne 2GX 2013早鸟价即将过期，请立即注册以确保折扣价！
  2. Spring Data忍者Thomas Risberg宣布Spring For Apache Hadoop 1.0.1。RC1已发布 。新版本支持Hadoop 2.0和Pivotal HD等。
  3. 工具套件忍者和领导者马丁·利珀特（Martin Lippert）宣布已发布Spring Tool Suite和Groovy / Grails Tool Suite 3.3.0 。非常好！
  4. Gary Russell宣布已发布Spring AMQP 1.2.0 。查看“ 新增功能”以了解详细信息。
  5. 加入Mattias Severson和Johan Haleby，并在2013年7月18日了解无 Lambdas的函数式编程
  6. 与Hemant Joshi一起在2013年7月30日的网络研讨会上介绍如何使用Spring和Cucumber BDD测试框架 。
  7. 我们在Skills Matter的朋友将于11月14日和11月15日在伦敦举行以春季为中心的会议（ 春季交流 ）。有一些杀手speakers的演讲者，如果可以的话，我强烈建议您加入。
  8. 您在野外使用Spring Social吗？我们想听听 ！
  9. 当使用Spring Security Java配置时， Spring Security主管和忍者Rob Winch撰写了一篇很好的关于可读性的文章。
  10. 刚刚发布的Spring ToolSuite（STS）3.3.0和Groovy Grails工具套件（GGTS）3.3.0中包含一个新的“快速搜索”。Spring和Groovy和Grails工具套件的高级开发人员Kris De Volder刚刚就此新功能撰写了一篇不错的文章 。
  11. 我们的朋友Johnathan Mark Smith再来一次！这次，他写了一篇文章， 如何使用Fongo和nosql单元通过MongoDB，JUnit和Log4J测试Spring Data项目 。看看这个！
  12. 哇！阿米尔Kibbar，在惠普软件开发人员的博客 ，已经把如何开发一个服务层，建立一个Web层，然后测试既是真正全面审视。第一篇文章有关设置服务层 ，第二篇文章是完善服务层并对其进行测试的示例，第三篇文章介绍了如何设置REST端点 ，第四篇文章讨论了测试REST服务 。绝对值得一读（和书签！）可以使用直接的Java配置来完成这些文章中演示的所有内容，也可以...
  13. Igor Artamonov撰写了一篇很好的简短文章，介绍如何使用Spring构建RESTful端点 。
  14. 我们在Baeldung博客中的朋友撰写了一篇非常不错的文章，内容涉及如何在Spring Security中使用摘要身份验证 。
