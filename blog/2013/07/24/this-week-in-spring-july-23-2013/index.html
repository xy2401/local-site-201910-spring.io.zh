春天的这个星期-2013年7月23日

工程
乔什·朗（Josh Long）
2013年7月24日

嘿大家！请记住，SpringOne 2GX 2013早鸟将于8月9日到期， 因此请务必确保折价！另外，请确保在添加新会话后检查议程。本周，我在OSCON上与俄勒冈州波特兰市的开发人员进行交谈，或者谈论关于REST 4，以及与Pivotal的同事讨论Cloud Foundry，大数据等等！如果您想聊天，希望您能参加我们正在进行的会谈，并在展厅的Pivotal展位参观我们！对于Spring和Pivotal来说，这都是重要的一周：

  1. Pivotal HD 1.0是世界上最快的Hadoop发行版，以两种版本发布-社区版和Pivotal单节点版（VM），即虚拟机下载。转到gopivotal.com并进行测试-Community Edition最多可部署50个节点的群集！
  2. 我们正在庆祝Project Reactor的第一个里程碑发布 -1.0.0M1- 已经 在Netty上对TCP进行 了基准测试，其速度比仅Netty快300％ ！当与关键的Spring技术集成时，Fast Data的可能性将把人们的头发吹倒。恭喜乔恩·布里斯宾 ！
  3. Spring Data Arora Service Release 2可供下载 。
  4. Martin Lippert在Spring Tool Suite 3.3.0中提供了有关注释和Java Config支持的出色博客。在整个Spring生态系统中，JavaConfig作为XML替代品的支持已接近普及水平。
  5. 与Hemant Joshi一起在2013年7月30日的网络研讨会上介绍如何使用Spring和Cucumber BDD测试框架 。
  6. Hadoop饿了吗？加入我们的网络研讨会系列-“每月可以使用Hadoop做什么”。2013年8月1日的第一次网络研讨会将提供有关Pivotal HD单节点（VM）中包含的功能和教程的深入详细信息。
    • 立即报名参加上午9点的PT会议
    • 立即注册参加晚上7点的会议
  7. 我的好友Andy Piper（@andypiper）将Cloud Foundry的精彩摘要汇总为Cloud Foundry中的 This Week 。我不能推荐它！他才刚开始，他正在做一份heckuva的工作！
  8. Zenika博客上有一篇非常不错的文章，内容涉及如何使用Swagger记录REST API ，您可以透明地将其层叠在Spring MVC API之上 。
  9. Matt Stine在Spring，Continuous Integration和CloudFoundry上也有出色的文章。
  10. JavaCode Geeks博客上有一篇不错的文章，介绍如何向REST API添加验证
  11. 的Pivotal博客上有一篇非常不错的文章，介绍了Tomcat与Pivotal的tcServer的比较，tcServer是Tomcat的二进制兼容发行版，我们为部署提供支持和扩充
  12. 也就 Pivotal博客，有关Spring Data GemFire（和GemFire）如何真正提高应用程序性能的精彩文章 ！
  13. XavierPadró's对Spring消息传递进行了很好的介绍
