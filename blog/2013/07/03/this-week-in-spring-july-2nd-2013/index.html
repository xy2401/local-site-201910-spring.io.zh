    春天的这个星期-2013年7月2日

    工程
    乔什·朗（Josh Long）
    2013年7月3日

    欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！

    1. Spring and Cloud Foundry的忍者Jennifer Hickey宣布了Spring Data Redis 1.1 M1和1.0.5的可用性。看看这个！
    2. Spring Security主管（和忍者） Rob Winch宣布了Spring Security Java配置支持的初始可用性。Rob还刚刚发布了一篇关于新的Spring Security Java配置支持的非常好的文章（四篇中的第一篇）。在第一篇文章中，您可以找到新的Spring Security Java配置支持 。
    3. 7月18日参加我们的网络研讨会“不带Lambas的功能编程” ，其中介绍了现在就可以在Java中使用功能编程的方法（而不是等待Java 8！）。使用Guava，LambaJ和Functional Java。
    4. Corby Page在他的项目Yoga上写了一篇非常不错的文章，介绍了如何扩展REST API的功能 。特别是，它支持称为selector可以用来提取REST响应的子视图以发送回客户端。这也可以用来支持Lez Hazelwood恰当地描述为实体扩展的内容 。
    5. Crunchify博客上有一篇不错的文章，介绍如何使用Spring MVC上传多个文件 。
    6. SpringSource在9月的日程安排中增加了一个新的Live Online Core Spring类 。
    7. Spring LDAP项目已经社交化并移至GitHub ！
    8. 最近，我们的好朋友丁学峰 （您可能还记得帮过他写博客“ China Scala的Spring” ）刚刚发表了一篇关于用Spring构建REST API的很好的演讲。我认为他的甲板很酷，所以请检查一下！
    9. 谢尔盖·谢尔巴科夫（Sergey Shcherbakov）最近发表了精彩的演讲，介绍了很多有趣的东西。我认为他的示例代码特别值得一看 。该代码具有Spring 4 WebSockets，无XML的Spring Batch，Reactor和AngularJS示例。干得好，谢尔盖！
    10. Johnathan Mark Smith撰写了一个不错的博客，介绍如何在MongoDB中使用Spring Data 。不错的工作！
    11. Nicolas Frankel对Spring 3.2中的一些引人注目的功能进行了很好的介绍。干得好，尼古拉斯！
    12. 尼克·威廉姆斯（Nick Williams）提交了一份拉取请求，以支持在Spring WS的Java配置中使用Java配置MessageDispatcherServlet这样就可以在一个ServletContextListener或ServletContainerInitializer 。干得好，尼克！
    13. 这本身并不是特定于Spring或Spring Batch的，但是Technology AMIS博客对如何使用Batch JSR（基于Spring Batch，并与Spring Batch团队合作设计） 进行了有趣的研究。 建立一个下载管理器 。（不用担心，您不必使用GlassFish与Batch JSR一起使用！）太酷了！如果您知道Spring Batch，那么其中的许多内容将看起来很熟悉，并且由于Spring Batch也将实现JSR，因此对于将来想要使用Spring Batch的任何人来说，应该证明是一个非常不错的入门。
