春天的这个星期-2013年8月27日

工程
乔什·朗（Josh Long）
2013年8月27日

欢迎回到春季本周的另一期。与往常一样，我们有很多内容要讲！Spring Security和领队Rob Winch在本周的综述中扮演了重要角色！因此，为Rob Winch欢呼 ！


  1. Spring Security负责人Rob Winch撰写了一篇文章，介绍了Spring Security中针对跨站点请求伪造（ CSRF ）的一些新的，智能且方便的保护。

  2. Rob还撰写了另一篇史诗般的博客文章，展示了Spring Security 对安全标头的新支持 。

  3. Rob 还很友好地将这些新功能集成到Spring REST堆栈代码库中 ，您可以在其中使用它们，这些功能在全栈，集成的Spring REST服务的上下文中起作用。要查看这些更改以及Spring Security和Spring Security OAuth（均使用Java配置集成），请查看oauth模块 。

  4. 反应堆负责人Jonathan Brisbin宣布了Reactor 1.0.0。M2现在可用 。新版本看起来非常令人兴奋！摘自Jon的文章：“第二个里程碑包括许多错误修正和一些非常令人兴奋的新功能。现在，Reactor包括处理器抽象，它是基于LMAX Disruptor RingBuffer的高度优化的任务处理器……MacBook Pro上的轶事基准测试表明，处理器每秒可以通过管道发送约100,000,000个事件。是的，您没看错： 每秒1亿 ！”

  5. Hyperic，Cloud Foundry，Spring和Spring Data忍者Jennifer Hickey宣布了Spring Data Redis项目的最新版本 。新版本包括对密钥过期命令中毫秒级精度的支持，连接失败时消息侦听器的重新订阅，完整实现ConcurrentMap签约RedisMap和RedisProperties

  6. Spring Batch主管Michael Minella宣布Spring Batch 3.0M1已发布 ！此版本标志着实现JSR-352 Java Batch规范的第一步。

  7. 立即注册参加8月29日的网络研讨会： 使用模块化模式驯服耦合和粘性野兽，并使用Param Rengaiah 驯服春季 。

  8. 与我们来自Pivotal Labs的朋友一起 ，在9月5日，David Frank向您展示如何通过Pivotal Tracker获得敏捷性 。

  9. Spring Security主管Rob Winch 发推文说 ：“＃Gradle使使用#SpringFramework 3构建#SpringSecurity并同时使用#Spring 3/4进行测试变得非常简单”，并链接到这个史诗示例 。严格来讲，这并不是与Spring相关的文章，但这是一个很好的Gradle构建示例，对于像SpringSource一样使用它的人也是如此。

  10. 春季安全负责人Rob Winch （男孩，那家伙肯定走了！）还宣布了Spring Security LDAP的最新版本 。

  11. 我们的好朋友XavierPadró回来了，这一次的帖子介绍了如何在Spring的消息接收中使用本地资源的JMS事务。 JmsTemplate 。

  12. Patrick Grimard撰写了一篇很好的文章，介绍如何将Yeoman Backbone与Spring Web应用程序集成，以及一种通过使用Tuckey的方法来回避他在资源解析方面遇到的一些问题的方法。 UrlRewriteFilter 。现在，我喜欢UrlRewriteFilter 。就是说，我认为（但尚未进行测试。）Patrick可以使用Spring MVC资源处理支持完成同样的事情，因为他在重写代码时开始在代码示例中进行了此操作public void addResourceHandlers(ResourceHandlerRegistry registry)配置方法。也许我弄错了，不管怎么说，很酷的帖子！

  13. SpringSource已针对第3季度计划发布了新的培训，请查看以下培训计划： Core Spring ， 与Spring的企业集成和Spring Web 。
