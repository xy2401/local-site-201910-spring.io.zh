春天的这个星期-2013年10月15日

工程
乔什·朗（Josh Long）
2013年10月15日

欢迎来到春季本周的另一期！另外， Ada Lovelace日快乐！

本周，我要去新加坡的狮城 ，在亚太枢纽峰会上与商业领袖谈论春天！如果您在新加坡并且想与企业Java，Spring，Cloud Foundry和大数据进行聊天，请ping me ！

如果您在旧金山/湾区，那么请在本月末（10月29日至30日，星期二）在Pivotal HQ与我一起在Spring上进行两个演讲 ：一个介绍Spring的REST开发，另一个介绍作为Pivotal的一部分，总的来看一下Spring的最新和最出色的产品。

  1. Spring Integration的负责人Gary Russell宣布Spring Integration 2.2.6现已可用 ！最新版本（2.2.x）分支中的最新维护版本，并包含许多对（S）FTP，脚本，TCP，事件和Mongodb支持的重要错误修复。看看这个！
  2. 渴望Hadoop基础吗？前往gopivotal.com观看最近的Hadoop网络研讨会系列的重播 ：使用Hadoop和Spring构建大数据应用程序。
  3. 10月24日星期四，与Juergen Hoeller一起对Spring进行基于注释的视图：Spring的现代组件设计 ，现在就注册欧洲或北美
  4. 11月19日，与我和John Davies一起使用Spring Integration了解每秒100万次动态路由 。现在注册欧洲或北美 。
  5. 11月21日，参加我的网络研讨会， 您最近看过Spring吗？ 在这里，我讨论了今年春季发生的所有事情，特别是今年SpringOne宣布的事情。立即注册欧洲或北美 。
  6. 同时，在Mkyong.com博客上，还有另一篇很棒的文章，该文章是关于使用Spring MVC和Google Maps进行地理位置定位的 。
  7. JavaBeat博客上有一篇不错的文章，介绍如何使用Spring MVC测试框架来测试Spring MVC应用程序。
  8. ISOS Technology博客在单元测试Spring MVC Portlet上有一篇不错的文章
  9. 我确实介绍了Spring的丰富REST堆栈的系列文章的第三部分现在可以在Safari博客上找到。与往常一样，该代码在我的GitHub帐户上可用 。
  10. 您可以通过Homebrew轻松安装Spring Boot： brew tap pivotal/tap && brew install springboot 。当然，这是您可以通过Homebrew获得的其他一些惊人的Pivotal技术的补充，包括Spring REST Shell（ brew install rest-shell ）和RabbitMQ（ brew install rabbitmq ）
  11. 您是否要通过JMS使用RabbitMQ？Spring忍者Greg Turnquist做得很好，展示了如何使用Spring的RabbitMQ JMS连接器 。查看此帖子以获取更多详细信息！
  12. 微软为使Redis在Windows上可用做了很多工作！观看有关在Windows上运行Redis的MSDN Channel 9视频 。
  13. 的libfugu.so博客提供了RabbitMQ的快速而肮脏的外观 。真好！（以及寻找类似于Linux链接库的域的额外要点！）
  14. 观看此视频，演示如何为在Cloud Foundry上运行的Spring应用程序设置New Relic监视支持。
