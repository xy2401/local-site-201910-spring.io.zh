春天的这个星期-2013年10月1日

工程
乔什·朗（Josh Long）
2013年10月2日

欢迎回到春季本周的另一期！当然，有很多事情要讨论，所以让我们开始吧。

如果您在海湾地区，我想邀请您参加本周末的硅谷密码训练营 。我将进行两个演讲-一个关于使用Spring构建RESTful服务，另一个关于使用令人惊叹的Phill Webb改善Spring Java配置肌肉内存。Pivotal还将在该处设有一个摊位，我们很高兴见到您！

  1. Spring Data Arora SR3已发布！ 新版本折叠在Spring Data Commons 1.5.3，Spring Data JPA 1.3.5，Spring Data MongoDB 1.2.4和Spring Data Neo4j 2.2.3中。和往常一样，Oliver做得很好！
  • 观看我们即将在10月3日举行的有关使用Hadoop和Spring构建大数据应用程序的网络研讨会。
  • 请务必在10月8日之后的一周注册一个在线讲座，介绍如何使用Spring构建websocket应用程序 。在其中，Rossen Stoyanchev将介绍Spring 4的websocket支持。
  • 上周在JavaOne 2013上，我们用新网站大放异彩 ！我将尝试每周在这个空间中介绍一些令人惊奇的参考内容。当然，您不需要每周等待即可阅读所有精彩内容！只需从站点指南页面开始。本周，我希望您能看看如何开始使用Spring开发RESTful服务 。它在大约15分钟的时间内介绍了如何开始Spring REST开发。
  • Spring XD忍者Glenn Renfro撰写了一篇不错的文章，介绍如何向Spring XD添加自定义模块 。
  • 下一个版本的RabbitMQ将优先考虑消费者 ！摘自文章“使用消费者优先级，您可以确保高优先级的使用者处于活动状态时接收消息，而只有当高优先级的使用者阻塞时，消息才发送给较低优先级的使用者。”很酷的东西！
  • Petri Kainulainen整理了一个很好的示例，说明了如何在Spring MVC Web应用程序中使用Spring Social的登录功能 。
  • 我们的好朋友XavierPadró's又回来了！这次，他获得了关于使用Spring Web Services 与Spring WS创建合同优先型 Web服务的不错的文章。
  • 我们的朋友Jan Machacek回来了！这次在Spring 4上发布了有关websockets，Angular.js，RabbitMQ和C ++的文章 。我..呃...它也有计算机视觉。我是否提到过……我的意思是……我什么也没得到。这太酷了 。去看看吧！
  • Matt Stine重新设计了Adrian Colyer在SpringOne 2012主题演讲中演示的代码， 以为Cloud Foundry构建自动缩放器 ！
  • Ben Hale是Spring的长期忍者，他是经过改造的Cloud Foundry Java buildpack的开发者，在Platform CF事件中引起了极大的关注，他刚刚编写了一个简单的实用程序，可以自动合并上游存储库 。好吧，我同意您，该实用程序并不适合所有人。但是，请检查代码！它是使用Groovy和Spring Boot编写的。该代码本身是（Apache 2）许可证的40％！
  • 我只想祝贺Pivotal数据科学团队在由Kaiser Permanente主持的24小时数据科学通识教育课程中表现出色！查看该帖子，了解他们的参赛作品，方法和比赛结果！
