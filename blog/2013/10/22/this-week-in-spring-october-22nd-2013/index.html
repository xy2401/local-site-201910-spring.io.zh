春天的这个星期-2013年10月22日

工程
乔什·朗（Josh Long）
2013年10月22日

欢迎来到春季本周的另一期！本周，我将在新加坡与开发人员讨论Pivotal春季小组的最新成果，这是我参加Pivotal APJ峰会的一部分 。

  1. 春季主管Juergen Hoeller将于10月24日 （从现在起2天！） 与春季进行网络研讨会，名为“ 春季 ” 。一定要在那里，它会很棒！
  2. 11月19日，我将与C24首席执行官John Davies进行演讲， 与Spring Integration进行每秒100万次的动态路由 ，我们将讨论如何使用C24的出色集成技术和Spring Integration扩展处理能力。
  3. 11月21日与我一起参加网络研讨会，介绍Pivotal春季系列的最新成果
  4. 我们的朋友Xavier Padro又回来了！这次，他有一个关于使用Spring Integration的RMI通道适配器的博客。
  5. Spring Data的忍者Oliver Gierke在幻灯片中介绍了他作为Pivotal的一部分开发Spring的演讲，着眼于采用的工具和方法 。如果您想在巧克力工厂内看一看，这是一本很棒的书。
  6. Spring Data忍者Oliver Gierke很好地回答了以下问题：“ Spring HATEOAS和SpringData REST有什么区别 ？“好问题，好答案！
  7. 我们的好朋友Petri Kainulainen再次出现，这次他提供了有关Sping Data Solr的不错的教程 。
  8. Groovy＆Grails现在为期4天，请在此处查看完整的培训计划 。
  9. JavaBeat博客很好地介绍了如何在Spring MVC中使用@RequestHeader annotation 。看看这个！
  10. Java Papers博客上有一篇不错的文章， 介绍如何使用Spring MVC Handler Interceptor 。
  11. InterTech bkig的一篇不错的文章介绍了Spring MVC的@SesssionAttribute 。
