Spring 的这个星期-2013年10月29日

工程
乔什·朗（Josh Long）
2013年10月29日

欢迎回到 Spring 本周的另一期！

如果您在湾区，我希望您今天和明天5:30在5楼Pivotal办公室的旧金山霍华德街875号与我一起。我每天晚上都会讲一个话题-一个是关于用Spring构建RESTful服务的，另一个是关于Spring的最新和最重要的简短概述。IO平台。此外， SpringOne2GX 2013从主题演讲1天和2天现在为您的享受。确保在1:13:40左右查看第1天的主题演讲，以听取Pivotal Labs-核心Ruby开发人员，谈论他们在Spring和Spring Boot方面的经验。

和往常一样，我们有很多内容要讲，所以让我们开始吧！

  1. 对于希望快速增加 Spring 知识的每个人， Spring 教育会在11月和12月的所有 Spring 培训课程中提供年终优惠。请在注册过程中使用以下推荐代码： springtime 。
  2. 在Strata Conference + Hadoop World 2013上，Pivotal宣布适用于Apache Hadoop / Spring XD的Spring已通过Pivotal HD ， Cloudera的CDH和Hortonworks HDP产品的认证 。
  3. 说到Spring XD，现在提供了有关如何使用Spring XD监视Twitter的新入门指南-快来查看！
  4. 如果您想了解Spring for Hadoop和Spring XD，请观看网络研讨会的重播，《 如何使用Hadoop：使用Hadoop和Spring构建大数据应用程序》 。
  5. Spring 和Hadoop仍然饿吗？SpringOne 2GX 2013 Day 2主题重播现已推出-听到Mark Pollack，Mark Fisher，Michael Minella和Graeme Rocher的史诗般主题演讲，包括Spring IO，Groovy / Grails，Spring和Hadoop，大数据，Spring Batch和Integration。
  6. 与Juergen Hoeller一起参加Spring网络研讨会重播：最近一次网络研讨会上的Spring现代组件设计 ！
  7. 在网络研讨会重播中与Rossen Stoyanchev一起了解Websocket的最新知识： 使用Spring构建WebSocket浏览器应用程序
  8. Spring MVC和Web CMS用户应在Campbell-Ewald和Magnolia的“ 将Spring应用程序带到美国海军的网络上”一文中查看该会议，该会议由Spring Team和Magnolia带来。
  9. 11月19日，我将与C24首席执行官John Davies进行演讲，与Spring Integration进行每秒100万次的动态路由，我们将讨论如何使用C24的出色集成技术和Spring Integration扩展处理能力。
  10. 11月21日与我一起参加网络研讨会， 介绍 Pivotal Spring 系列的最新成果
  11. 我们的朋友Petri Kainulainen再次回来了，这次的帖子是关于如何添加Spring MVC Web应用程序注册以及如何使用Spring Social和Spring Social Security模块登录的 。
  12. Nicolas Frankel撰写了一篇有趣的文章： 依赖注入和Rich Domain Objects Compatible 。
  13. Dan Woods回来了，他整理了一篇有关使用Spring Integration和RabbitMQ创建后端处理器（服务层）的文章。
  14. Spring Data忍者Oliver Gierke宣布Spring Data Babbage发布火车SR1已发布 。该发行版捆绑了许多重要的增强功能和错误修复，是建议的升级。您可以在JIRA中找到此版本中修复的所有问题。此发行火车以数学家，哲学家，发明家和机械工程师查尔斯·巴贝奇 （ Charles Babbage ）的名字命名。
