Spring Boot 0.5.0。M5发布

发布
戴夫·瑟
2013年10月9日

Spring Boot 0.5.0。M5在Spring仓库中可用。安装和使用说明位于项目网站或github中 。大量新功能包括：

  • 对JMS，AMQP，AOP，Mobile，MongoDB的自动配置支持
  • 简化版@Grab用法（请参见下面的示例）
  • 用于Groovy脚本的测试命令（支持JUnit和Spock，有关更多详细信息，请参见Greg的博客）
  • 一个新的SpringApplicationBuilder除其他外，还支持应用程序上下文层次结构
  • 一个新的PropertiesLauncher可以从运行时发现的属性启动Java应用程序（例如，从lib目录设置类路径）

作为品尝者，以下是使用示例SpringApplicationBuilder在Java中使用父上下文构建应用程序（如果要从同一代码运行多个应用程序，则很有用）：

@Configuration
@EnableAutoConfiguration
public class Server {

	public static void main(String[] args) {
		new SpringApplicationBuilder(Parent.class)
                    .child(Server.class)
                    .profiles("server")
                    .run(args);
	}

        // ... @Bean definitions follow
}

的Parent.class上面是一个共享的父上下文，可以将其重用于同一模块中的其他应用。

这是缩写的一个例子@Grab在Groovy应用程序中（组和版本信息会自动添加）：

@Grab("spring-boot-starter-actuator")
@RestController
class Example {

	@RequestMapping("/")
	String home() {
		[message: 'Hello World']
	}
}

此应用程序可以独立运行，例如，如果您使用spring启动它的shell脚本：

$ spring run app.groovy
... (app starts up)

在浏览器中访问http：// localhost：8080 / ，然后尝试http：// localhost：8080 / metrics 。
