  • Someone asked me this the other day and I felt like it was worthy of a mention: in your Spring MVC @Controller class handler methods, make sure that the BindingResult argument is immediately after the model or command argument, like this:

     @RequestMapping(...) public String handleRequest( @ModelAttribute @Valid YourCustomPojo attempt, BindingResult result). 

     In this example, handleRequest will validate the POJO (YourCustomPojo) - checking the POJO for JSR303-annotations and attempting to apply the constraints because the POJO is annotated with @Valid - and stash any errors  in the BindingResult, which it makes available if we ask for it.

    春天的这个星期-2013年3月5日

    工程
    乔什·朗（Josh Long）
    2013年3月5日

    欢迎回到春季本周的另一期。

    不过，我们还有很多要讲的，让我们开始吧！



    1. 我将于2013年3月14日举行网络研讨会-Spring的多客户开发 ！和我一起学习REST，OAuth，Spring MVC，Spring Android等！

    2. 与Damien Dallimore和David Turanski一起参加网络研讨会，他们介绍网络研讨会：扩展Splunk的Spring集成 -2013年3月28日

    3. 新的SpringOne2GX回放现在可以在YouTube上以高清格式播放： Spring数据存储库：深入研究和级联介绍

    4. @SpringSource正在发起（迅速）赃物 赠与 活动 ！


    5. Spring Security主管和忍者Rob Winch宣布了对Spring Security中基于Java的配置的最初支持。
      这是一个奇妙的里程碑。最近，我们已经看到为Spring Social，Spring Batch和Spring Security提供的XML DSL的Java配置替代方案。查看基于Spring Security Java的配置以了解更多详细信息。


    6. 我有幸拜访了阿里巴巴集团在中国的团队，他们在中国用Spring做一些令人惊奇的事情。在我的博客《 中国规模的春天：阿里巴巴集团》中了解更多信息。

    7. 前几天有人问我，我觉得值得一提：在您的Spring MVC @Controller类处理程序方法中，确保BindingResult参数紧随模型或命令参数之后，如下所示：

       @RequestMapping(...) public String handleRequest( @ModelAttribute @Valid YourCustomPojo attempt, BindingResult result). 

     In this example, handleRequest will validate the POJO (YourCustomPojo) - checking the POJO for JSR303-annotations and attempting to apply the constraints because the POJO is annotated with @Valid - and stash any errors  in the BindingResult, which it makes available if we ask for it.

    8. 说到使用JSR 303进行验证，我发现2010年的这篇精彩文章值得加入。这篇文章介绍了一个自定义注释，称为@SpelAssert ，其工作方式类似于JSR303的@ScriptAssert 。

    9. 您想将Cloud Foundry与CloudBees提供的持续集成功能一起使用吗？我们让你蒙住了！Cloud Foundry和Cloud Bees团队致力于整合流程，并在此处进行了逐步介绍 。

    10. Alvaro Videla已经介绍了他的RabbitMQ模拟器并将其开源。RabbitMQ模拟器是一个了不起的可视化工具，可以演示RabbitMQ拓扑如何工作。


    11. Gary Russell宣布Spring AMQP 1.1.4现已可用 。


    12. Fstyle博客上有一篇有趣的文章，介绍如何使用Spring MVC测试模型对Spring Security进行单元测试 。

    13. 我们的好朋友Boris Lam回来了，这一次是有关如何
      集成Spring Data，MongoDB和JavaServer Faces 。

    14. 程序员指南博客上的Indika Prasad整理了一个教程，展示了如何将Spring Security与Webdav和密码加密结合使用 。

    15. The Java J2EE SOA Key Points blog has a nice post on using the Spring WS JAXB web service client. There's very little narrative, but lots of code.
    16. Java J2EE SOA关键点博客上有一篇关于使用Spring WS JAXB Web服务客户端的不错的文章。叙述很少，但是代码很多。
