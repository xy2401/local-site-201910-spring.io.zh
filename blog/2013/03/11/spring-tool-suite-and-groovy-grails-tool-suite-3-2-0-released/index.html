发布了Spring Tool Suite和Groovy / Grails Tool Suite 3.2.0

发布
马丁·利珀特
2013年3月11日

亲爱的春季社区，

我们很高兴地宣布今天基于Eclipse的工具的下一个主要版本：Spring Tool Suite（STS）3.2.0和Groovy / Grails Tool Suite（GGTS）3.2.0。

此版本的重点包括：

  • Eclipse Juno SR2更新（包括Mylyn，EGit，m2e，m2e-wtp）
  • 在Mac OSX上增加了对高分辨率显示器的支持
  • 将捆绑的tc服务器更新为2.8.2
  • 使用Spring项目的主要性能改进
  • Live Spring Beans Graph的重大改进
  • 增加了对Spring Integration 2.2的支持
  • 更新为包括Groovy 2.0.7和Grails 2.2.1（可从仪表板获得Groovy 2.1）

我们继续在Eclipse 3.8和Eclipse 4.2之上发布发行版。尽管Eclipse Juno SR2发行版中Eclipse的4.2流已进行了很多改进，但我们仍建议使用基于3.8的版本以实现最佳性能和稳定性。

要下载发行版，请访问：

  • Spring工具套件： http ： //www.springsource.org/sts
  • Groovy / Grails工具套件： http : //grails.org/products/ggts

详细的新的和值得注意的笔记可以在这里找到： STS / GGTS 3.2.0新的和值得注意的 。

通过自动“检查更新”可以获取STS / GGTS 3.0.0和3.1.0的更新。STS 2.9.x（或更早版本）的用户应从全新安装STS / GGTS 3.2.0开始。

下一个版本3.3.0计划在Eclipse Kepler（4.3）发布之后不久于2013年7月发布。如果您想了解最新的开发动态，还请注意3.3.0的里程碑版本。
