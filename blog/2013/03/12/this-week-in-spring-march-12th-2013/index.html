春天的这个星期-2013年3月12日

工程
乔什·朗（Josh Long）
2013年3月12日

欢迎来到春季本周的另一期！本周，有很多 Spring Tool Suite新闻，所以请务必查看新版本并进行尝试。最后提醒您：一定要在星期四参加我的网络研讨会，在格林尼治标准时间3:00 PM（对于欧洲人）和太平洋标准时间10:00 AM（对于北美）中介绍Spring的REST和移动支持 。如果您想学习如何为基于Spring的后端服务构建移动应用程序，那么本讲座适合您。通常，我们将看看Spring对REST，Android和移动平台的丰富支持。 

    <LI>Jonathan Brisbin's announced that <a href="http://www.springsource.org/node/4248">Spring Data REST 1.1.0.M1 has been released</a>. 
    The new release is basically a from-the-ground up rewrite. In the new release, there is support for <EM>all</EM> 
    repositories including MongoDB and GemFire-based repositories. </LI>
<LI> Martin Lippert has announced that <a href="http://www.springsource.org/node/4247">Spring Tool Suite and Groovy/Grails Tool Suite 3.2.0 have been released</a>.  The new version is <EM>much</EM> faster than the previous version, and includes updated support for Eclipse Juno SR2, high-res displays on OSX,
    and updated compliance with various Spring projects, including Spring Integration 2.2.


  1. Rob Winch宣布了Spring Security SAML 1.0.0。RC2已发布 。
    Spring Security SAML是第三方提供的，为Spring Security提供SAML支持。

  2. Spring Integration 2.2.2现在可用 ！新版本
    具有各种重要的错误修复。

  4. 与Damien Dallimore和David Turanski一起参加网络研讨会，他们介绍网络研讨会：扩展Splunk的Spring集成 -2013年3月28日
