    春天的这个星期-2013年2月12日

    工程
    乔什·朗（Josh Long）
    2013年2月12日

    欢迎回到春季本周的另一期！
    和往常一样，我们有很多内容需要介绍，所以让我们开始吧！

    1. Dave Syer将于2013年2月14日在在线观众中进行他的SpringOne2GX讲座网络研讨会- 什么时候以及为什么我要使用OAuth2？
    2. 梦之队Sam Brannen（Swiftmind）和Rossen Stoyanchev（SpringSource）于2013年2月21日联手举办网络研讨会：使用Spring 3.2测试Web应用程序
    3. 新的SpringOne2GX 2012讲座以高清格式发布到YouTube！虚拟化Java应用程序的十大理由以及CloudFoundry的新增功能 。
    4. 我们的好朋友Boris Lam回来了，这次有两篇关于如何一起使用Spring Data MongoDB和JSF的文章 。
    5. PluralSight博客的视频介绍了Spring MVC 拦截器 。该视频摘录自完整的视频课程。
    6. 很棒的演示： 用于Spring Integration和RabbitMQ的Spring Insight插件 。
    7. syntx博客上有一篇不错的文章，内容涉及如何使用Spring Security向受Spring MVC保护的资源添加HTTP Basic身份验证 。
    8. Krishna的Blog有一篇不错的文章，介绍了使用InMemoryDaoImpl单元测试Spring Security层。
    9. 您知道，前几天我正在寻找类似的东西！Mark的Blog 在Spring MVC中为服务传入的HTTP请求提供服务时，介绍了各种对象的工作流程 。您是否可以通过在控制器中插入断点并仅查看调用堆栈来轻松找出这些东西？当然。但这一点也很方便引用，很高兴看到所有内容都按顺序进行了排列，并带有一些叙述。
    10. Java上的Dinesh博客上有一篇不错的文章，介绍如何设置Spring Security 。
    11. 幸运瑞安博客有一个真正的propos张贴在迁移基于XML的Spring MVC配置到Java和注释为中心替代 。
    12. Java Code Geeks博客上有一篇关于REST服务处理异常的非常好的文章
    13. 我们的好朋友Roger Hughes回来了，这次写了一篇很好的关于Spring bean各种生命周期技术的文章 。
    14. CommonJ博客上有一篇不错的文章，内容涉及从JavaScript向Spring MVC传递请求参数 。这与Spring MVC 略有关系，但是已经足够接近了！ :)
