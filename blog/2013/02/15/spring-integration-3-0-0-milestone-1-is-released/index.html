Spring Integration 3.0.0里程碑1已发布！

发布
加里·罗素
2013年2月15日

我们很高兴地宣布Spring Integration 3.0.0。M1现在可用。在第一个里程碑中没有主要的新功能，主要是重构，删除过时的内容等。请浏览文档“新增功能”和发行说明以获取更多信息。

我们很高兴看到更多社区贡献者，并正在继续努力在核心项目和扩展存储库中促进这一增长趋势。

  • 发行说明可在此处获得 。
  • 参考文档在这里 。
  • 可以在SpringSource Milestone仓库和社区下载页面中找到工件。

有关更多信息，请参见项目的主页 。
