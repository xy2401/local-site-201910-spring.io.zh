为了庆祝Spring Framework 4.0发布，Manning 12月12日至13日发行的所有Spring Books均享有50％的折扣

新闻与活动
乔什·朗（Josh Long）
2013年12月12日

为了庆祝Spring Framework 4.0的发布，Pivotal很高兴继续赞助曼宁的“ 2014倒计时”。由Pivotal / Spring团队精心挑选的书籍在特定日期提供50％的折扣 。曼宁还向所有报名参加2014年倒计时的人提供额外的报价-他们会被自动录取，以赢得每日赠送的两本电子书之一，并赢得一等奖iPad Air。

12月12日星期四和12月13日星期五是春季 ！（有关图书的详细信息，请参见下文）

只需访问manning.com，然后选择这些图书中的任何（或全部）即可。结帐时在促销代码框中输入srgdotd13 。
