春天的这个星期-2013年12月24日

工程
乔什·朗（Josh Long）
2013年12月25日

节日快乐！希望随着假期的到来，有机会放松一下，更充分地赶上忙碌的一年中错过的有趣的事物。本专栏的读者将知道，开发人员可以通过许多渠道学习Spring，并研究Spring支持的技术领域。不要忘记我们的SpringSource YouTube页面 ， Twitter帐户 ，我们的15-30分钟的“入门”指南 ， 博客 ，当然还有我们的Facebook和Google+页面 。我个人想回去尽可能多地在YouTube频道上观看SpringOne2GX谈话。

  1. 记住JHipster吗？Julien Dubois的 Yeoman驱动的Spring应用程序代码生成器？好吧，0.0.6已发布，并且没有必需的Spring XML（和Java EE的web.xml是最后一个！）并提供服务的代码生成支持。
  • 您可能已经注意到我们有一些新的入门指南，包括有关使用RESTful Web Service的指南。 rest.js 。
  • 想要在Spring中使用Zero Turnaround的实时重载技术JRebel 4.0.0.RELEASE ？使用JRebel 5.4.3-SNAPSHOT (201312032038) 。
  • SD Times 对Spring 4.0版本有一个有趣的了解。通常是分析师的观点，而不是技术。
  • 罗伊·克拉克森（Roy Clarkson）撰写了一份很棒的入门指南，介绍如何使用Spring Boot （以及扩展名为Cloud Foundry ！）。轻松托管静态文件（如JavaScript，CSS和.html模板 。
  • Petri Kainulainen撰写了一篇非常不错的文章，内容涉及如何从@Scheduled春季工作 。
  • Spring Data负责人Oliver Gierke已将Spring Data REST移至JSON表示形式 HAL作为默认格式。自从Spring HATEOAS 成立以来，HAL变得非常受欢迎，现在已成为社区中的首选。
  • 您正在学习机器吗？在GoPivotal ，这里有很多事情-尤其是以我们的开源为中心的大数据工具集-而且我发现Meta Optimize QA网站在学习/获得答案方面非常有帮助。 。这只是一个仅供参考，与Spring无关。
  • 提供了与Spring 4一起使用的Thymeleaf模板引擎的新版本 。干得好，看看！
  • Viral Patel有一个不错的博客，介绍如何使用Spring MVC处理HTTP cookie 。
  • Spring Mobile提供了一种方便的方法来构建特定于移动设备的应用程序版本。此支持的一部分是一种功能，用于在体验之间切换并维护有关用户对他们的体验的偏好的状态（例如：“切换到移动站点”）。查看有关该主题的出色入门指南 。
  • 最近在Spring 4.0的发行中 ， Application Development Trends有一个有趣的，分析性的文章 。值得一看。
  • Spring Integration负责人Gary Russell刚刚宣布了第二个里程碑（也是第一个公开的里程碑！）Spring Integration 4.0版本 。除其他事项外，该版本的功能是首先将API剪切为基于Spring Framework 4的基础，并基于其受Spring Integration启发的消息传递API（包括Message和MessageChannel ）。
  • Apache BigTop创始人Roman Shaposhnik刚刚通过@GoPivotal加入了我们的团队 。等不及要看即将发生的事情了！

如果您在下周庆祝某事，那么祝您假期快乐健康！下周，除夕，C'ya！
