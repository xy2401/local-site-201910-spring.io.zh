春季本周（春季4版！）-2013年12月17日

工程
乔什·朗（Josh Long）
2013年12月17日

欢迎来到春季本周的另一期！ 好吧，这周，我要放假了：）当然，这意味着本周的综述对我来说更加有趣-我可以使用刚刚发行的Spring 4！ 而且，为使假期更加美好，来自其他Spring项目的基于Spring 4.0的新版本源源不断地开始了！

如果您使用的是Spring（Spring 4，Spring Boot以及其他工具），并且有一些不错的新博客，视频或示例项目，您认为人们应该看到这些，请立即在Twitter上与我分享！Matt Raible已经发表了一篇很有帮助的博客文章： 使用Spring 4和Spring Boot进行Webapp改造，在那里他升级了现有的Spring 3.2.5，Spring Security 3.1.4和Jersey 1.18应用程序以运行Spring Framework 4和Spring Boot 。

  1. 首先， 大新闻！春季CTO Adrian Colyer刚刚宣布，春季4已经结束 ！如果您像我一样一直热切地等待着这一整年，那么不要再等待一秒钟了！尽快抓住这些位。当然，Spring 4是自2009年Spring 3.0以来的第一个主要版本增量，对于应用程序开发人员来说是一次重大的飞跃。在2014年1月9日加入Juergen Hoeller（和许多其他工程师）参加启动网络研讨会： Spring Framework 4.0简介 。
  2. 在Spring 4发行版的同时，我们刚刚向极受欢迎的“ 入门指南”集合中添加了一些新指南。在这些新的指南 ，你会发现CORS，jQuery-，Sencha-帮助，Angular.js整合，以及更多 ！
  3. 随后不久，Rob Winch宣布了Spring Security 3.2.0 RELEASE可用 ！现在，我将最终 将代码更新为使用Spring的REST堆栈以及Spring Security和Spring Security OAuth的新版本！在2014年1月16日加入Rob ，进行有关Spring Security 3.2新版本的演讲 。
  4. 一旦发布了Spring 4，Spring Integration的负责人Gary Russell就会立即将期待已久的Spring Integration 3.0发布出去 ！此新版本具有许多新的改进，这些新改进大部分在候选发布公告中涵盖。
  5. 项目负责人Martin Lippert刚刚宣布了Spring Tool Suite和Groovy / Grails Tool Suite 3.5.0。M1现在可用。此更新修订为Groovy 2.2，Grails 2.3.4和tc Server 2.9.4，以及Spring Boot项目的高级内容助手，改进的仪表板供稿以及对新的客户端入门指南的支持。该剪切基于Eclipse Kepler SR1。看看这个！
  6. Spring Data项目负责人Oliver Gierke刚刚宣布了最新的Spring Data发布火车Spring Data Babbage SR2 。该服务版本捆绑了许多重要的增强功能和错误修复，是建议的升级。您可以在我们的JIRA中找到此版本中修复的所有问题
  7. 春季忍者Greg Turnquist在幕后精心整理了上述入门指南向Asciidotor的迁移 。
  8. Spring ninja和Boot共同负责人Phil Webb，我为SpringOne2GX 2013做了一个题为“ 改善您的Java配置肌肉内存”的演讲，现在可以在我们的YouTube频道上收听 。看看这个！
  9. 帕特里克·格里马德（Patrick Grimard）撰写了一篇文章，介绍了如何设置Spring MVC拦截器来处理CORS请求 。有关CORS主题的更多详细信息，请查看“ 了解CORS”页面 ，然后查看新的“ 入门指南”，其中显示了以Servlet Filter中心的另一种基本方法。这是在Spring Boot的基础上构建的，并且使用Filter而不是interceptor ，但是效果是相同的。
  10. 我们的好朋友Bozhidar Bozhanov写了一篇有关Web套接字的精彩文章，这当然适用于Spring 4！ ，并附有幻灯片和代码！请务必检查一下！这篇文章对Websockets使用了一种更底层的方法，Spring也支持这种方法，所有消息都通过一个处理程序进行收集。我个人而言，我喜欢使用更高级别的STOMP支持，以避免必须通过同一处理程序来汇集所有请求，然后再用switch语句将每个请求分开。无论哪种方式，这都是一篇不错的文章，并且-由于它的级别较低-使您可以更好地了解引擎盖下发生的事情。看看这个！
  11. 随着新版本的发布，更新了Maven工件。上周，我提到Spring 4现在具有非常方便的材料清单 Maven pom.xml 。您应该使用它来简化事情。此外，如果您是BinTray用户，请注意该位置也已提供新版本 。
