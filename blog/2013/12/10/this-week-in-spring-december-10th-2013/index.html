欢迎来到春季本周的另一个装置！本周，我要去洛杉矶，和迪士尼的开发商谈谈春天。如果您在该地区，我也很想看到您在那里。我们将于12月11日主持一个用户组会议。以下是有关该活动的详细信息 ，该活动将于明天举行！我期待在那里见到你！

您可以相信我们已经盯着年底了吗？@ [受电子邮件保护]

这意味着是时候发布Spring Framework 4.0了！我们期待您对版本的反馈。

  1. 不要错过Juergen Hoeller于2014年1月9日发布的Spring Framework 4.0发布网络研讨会“ Spring Framework 4.0简介 ”，这是自2009年以来的第一个主要的Spring Framework版本！
  2. 务必查看2014年1月16日由代码业务最忙的人和Spring Security负责人Rob Winch举行的网络研讨会，以了解有关Spring Security 3.2的信息 。
  3. Spring框架的提交者和Spring Boot的共同负责人Phil Webb在Spring 4中介绍了Spring 4对Java泛型的新支持。
  4. Thomas Darimont介绍了Spring Data MongoDB 1.4 M1中的新增功能，包括惊人的新投影框架和SpEL集成。
  5. 如果您是Spring支持的Broadleaf电子商务引擎的用户 ，那么您可能想注册v3.1 Web事件 。
  6. 我们的好朋友Tobias Flohre返回了另一篇很棒的文章， 批处理和Java Enterprise Edition（JSR-352，JEE7，Spring Batch） 。标题不言而喻-很好地概述了技术的位置。
  7. 而且， Petri Kainulainen再次对它进行了精彩的后续介绍，介绍了他关于使用Spring Social向Spring MVC Web应用程序添加社交功能的系列。这篇文章是关于集成的单元测试的 。
  8. @GoPivotal博客对RabbitMQ在挪威广播公司的部署进行了有趣的案例研究。

  9. 说到Phil Webb和Rob Winch，上周发布的Spring Framework RC2包括的令人惊奇的事情之一就是他们对“材料清单” Maven POM的精彩添加： spring-framework-bom 。像这样使用它：

    <dependencyManagement>
    <dependencies>
        <dependency>
            <groupId>org.springframework</groupId>
            <artifactId>spring-framework-bom</artifactId>
            <version>4.0.0.RC2</version>
            <type>pom</type>
            <scope>import</scope>
        </dependency>
    </dependencies>
</dependencyManagement>

    然后，只需导入所需的Spring依赖版本，而不在您的版本中指定版本pom.xml 。

  10. GoPivotal博客上还有一个案例研究，目前Hulu使用Redis扩展了服务40亿个视频的规模 。棒极了！

  11. Sudhir Dharmadhikari撰写了一篇有关使用Spring Roo的方面方法进行AndroMDA项目的有趣文章
  12. 我很喜欢上周在索非亚再次见到的Bozhidar Bozhanov ，已将Spring Social Google+代码移至GitHub 。
