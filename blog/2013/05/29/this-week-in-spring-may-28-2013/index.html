Spring 的这个星期-2013年5月28日

工程
乔什·朗（Josh Long）
2013年5月29日

欢迎来到 Spring 本周的另一期。万一您上周错过了它， SpringOne2GX 2013议程中的绝大多数已经发布，因此现在预订，即可获得会议的早鸟价和机票 ！像往常一样，本周我们有很多工作要做，让我们开始吧！

  1. Spring Batch主管Michael Minella宣布了Spring Batch 2.2.0 RC2 。新版本包含了许多很棒的新功能，包括对Spring Batch Java配置API和Spring Data GemFire的支持。 ItemReader和ItemWriter 。
  2. Gary Russell刚刚宣布了Spring Integration 3.0里程碑2 。请务必检查新功能并踢轮胎！
  3. 与我一起参加关于在2013年6月13日 Spring 构建REST-ful服务的网络研讨会。我将讨论与REST相关的OAUTH，Spring MVC和Spring HATEOAS。
  4. Rossen Stoyanchev的博客介绍了Spring 4对WebSockets的即将支持 ，它看起来非常引人注目！
  5. Gary Russell还刚刚宣布了Spring Integration MQTT扩展适配器（里程碑1），该适配器使使用Spring Integration的MQTT （一种将其自身应用于轻量级消息传递的消息传递技术）易于使用。
  6. 对于如何在Stack Overflow 上以JSON格式返回Spring Data页面作为问题， Oliver Gierke做出了很好的回答。
  7. 本综述的长期读者将了解Thymeleaf，该模板引擎为Web应用程序视图模板注入了新的活力，并且与Spring一起很好地工作。刚发布了第一个稳定的Thymeleaf-testing 2.0.0版本。
  8. Trifork博客上的Joris Kuipers宣布了一组新的宏，这些宏用于使用Freemarker （一种替代的功能非常强大的模板引擎） 与Spring应用程序进行表单输入 。
  9. Oleg Tsal-Tsalko 在Spring 4上就新位进行了讨论 。做得好，奥列格！
  10. Johnathan Mark Smith又回来了，这次是关于如何使用Spring进行Java配置的视频。看看吧 ！
  11. Maciej Walkowiak撰写了一篇很棒的文章，介绍如何使用Spring Data MongoDB审计实体 。
  12. 命名不佳的Java2J2EE博客上有一篇很棒的简短文章，内容涉及如何使用Spring的Java配置样式设置JPA和Spring MVC 。但是，我不鼓励用户在Spring上调用生命周期方法FactoryBean直接，而是选择取消引用已配置的结果：
    @Bean public EntityManagerFactory emf(){
   LocalContainerEntityManagerFactoryBean lcemfb = ..
   return lcemfb;
} 

@Bean public PlatformTransactionManager transactionManager(){
  EntityManagerFactory emf = emf().getObject();
  return new JpaTransactionManager( emf );
}
