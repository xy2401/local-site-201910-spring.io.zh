Spring 的这个星期-2013年5月14日

工程
乔什·朗（Josh Long）
2013年5月15日

欢迎来到 Spring 本周的另一期！本周已经宣布了一些相当激动人心的项目，如果您可以相信，我们几乎超出了SpringOne 2012的重放内容 ！好消息是，SpringOne 2013议程网格将很快上线，因此我们可以向前看。和往常一样，我们有很多内容需要介绍，让我们开始吧！

为准备议程网格，许多新的SpringOne 2013会议已被接受：

    • Spring Security 3.2入门 （Rob Winch）
    • 性能调整Spring Petclinic示例应用程序 （Julien Dubois-Ippon Technologies）
    • 不可变的 Spring （马特·斯蒂恩）
    • 超越依赖注入 （Mark Secrist）
    • Spring下一代工具 （Martin Lippert，Andy Clement）
    • 期货和Rx Observables：异步使用Web服务的强大抽象 （Chris Richardson）
    • 与 Spring 社交建立联系 （Craig Walls）
    • 对实时Java Web应用程序进行故障排除 （Ashley Puls-新遗物）
    • Spring 和Sencha-天堂或至少是云端的匹配 -（Jay Marshall + TBA，Sencha）
    • Hadoop上的内存中数据和计算 （Jags Ramnarayan）
    • Spring Data和Apache Hadoop入门 （Thomas Risberg）
    • Inside Cloud Foundry：架构审查 -（Dekel Tankel）
    • 建立自己的私有云铸造厂 （Matt Stine）
    • Thymeleaf：使用自然模板改善Spring视图图层 （Emanuel Rabina-HP）
  1. 乔恩·布里斯宾（Jon Brisbin）宣布了反应堆项目 。Reactor项目旨在为基于异步IO的应用程序提供坚实的基础，自然而然地为Grails和Spring等技术提供集成。Reactor已经具有很好的多语言故事，并且支持Groovy和Java（尤其是即将发布的Java 8版本！）如果您对与其他异步技术相比有疑问，请务必检查一下，特别是评论部分！
  2. Spring Security负责人Rob Winch一直在忙于增强Spring Security和Spring Security OAuth Java Configuration的故事。他已经获得了Spring Security OAuth Java Configuration API的第一版，我相信他会很感谢对新DSL的任何反馈，因此一定要检查一下！干得好，罗伯！
  3. 5月16日（星期四）与POJO行动作者Chris Richardson举行的网络研讨会 ，主题是分解应用程序的可部署性和可扩展性。现在注册！
    • 这是了解SpringOne 2012主题演讲中讨论的新应用程序体系结构的重要演讲 -如何将大型应用程序分解为小型，可伸缩的离散服务。很难不使用NetFlix的Asgard作为做得很好的一个很好的例子，这个例子很多人可能会认识到。
  4. 5月28日星期二，参加Broadleaf Commerce的Andre Azzolini的网络研讨会，他们讨论从GWT到SpringMVC的经验教训 。
  5. Paul Chapman在SpringSource博客上介绍了Spring MVC中对内容协商的各种支持 。
  6. 克里斯·哈里斯（Chris Harris）的演讲，来自SpringOne2GX 2012的Spring Data MongoDB项目 ，现在可以在YouTube上以高清格式观看！
  7. Lee Faus在SpringOne2GX 2012上的演讲“ Extreme Makeover-Application Edition”现在可以在YouTube上高清观看！
  8. JIWHIZ博客和博客Yuan Yuan 撰写了一篇不错的文章，介绍了Spring的Java配置支持 。
  9. 这篇博文（来自博主Chris Wong）在1月发布的一篇名为“ JmsTemplate不是邪恶的 ”-解释了使用Spring的一些微妙之处CachingConnectionFactory生的ConnectionFactory然后，为获得加分，特别介绍了一种显着加快ActiveMQ速度的方法。
  10. HMKcode博客上有一篇不错的详尽文章，介绍了如何使用jQuery-file-upload Spring MVC插件 。
  11. 您看过HATEOAS吗？HATEOAS是一种设计模式，是一种用于构建更好的RESTful Web服务的方法。Spring HATEOAS使在Spring MVC上的操作变得如此简单，而这个由Geraint Jones撰写的博客很好地介绍了Spring HATEAOS
  12. Blogger Alexey Zvolinskiy回答了一个常见问题：如何将复选框绑定到在Spring MVC中来回发送到服务器的模型对象上 ？
  13. 我们的朋友@baeldung在Twitter上维护着有关Spring on StackOverflow的精彩帖子的每日Twitter提要，我认为他挖掘了一些绝对令人惊奇的内容。一篇帖子回答了一个我经常被问到的问题： 如何枚举所有Spring MVC @Controller -带注释的bean在运行时？
  14. 我在拖曳@SpringAtSO句柄时发现的另一篇很棒的文章是这篇文章，解释了如何将请求范围的属性传播到当前请求的线程之外。这篇文章通常适用于请求范围属性需要传播到其原始线程和request之外的任何情况。
