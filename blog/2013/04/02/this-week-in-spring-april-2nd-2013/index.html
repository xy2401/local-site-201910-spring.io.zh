春天的这个星期-2013年4月2日

工程
乔什·朗（Josh Long）
2013年4月2日

欢迎来到春季本周的另一期！我刚刚从Devoxx UK和Devoxx France返回，我很高兴与来自各行各业的使用Spring的开发人员交谈。我还在伦敦的Skills Matter 上谈到了使用Spring构建Web应用程序的问题 。感谢伦敦春季用户组的Skills Matter，以及令人惊奇的Rob Harrop拥有我，这真是一种荣幸！如果您有兴趣， 可以在线观看该会议的视频。

  1.  立即注册 ，在2013年9月9日至12日在加利福尼亚州圣塔克拉拉举行的SpringOne 2GX 2013 上享受超级早鸟价！
    2. 

  2. Gary Russell宣布发布Spring AMQP 1.2.0。M1现在可用 。
    3. 


  3. Cujojs团队已宣布rest.js 现在已成为Cujo.js 和rest.js 0.9的 一部分， 并且现已可用 。
    4.  
  4. Gary Russell宣布Spring Integration 2.2.3现已可用 。此版本纠正了一些复杂的消息有效负载在映射到方法参数时的转换问题，以及一些其他小的问题。


  5. Arjen
    Poutsma宣布了Spring Scala 1.0.0。M2现在可用     。
    新版本具有所有功能，包括更新的JdbcTemplate包装器，
    BeanFactory ， ListableBeanFactory和ApplicationContext丰富包装，
    具有隐式转换的AOP建议包装，对Spring 3.2.2的支持，以及
    更新了FunctionConfiguration对急切初始化的支持，以及许多其他错误修复和更改。

  6. 上周的网络研讨会介绍了适用于Splunk的Spring Integration适配器，现在可以在线获取。David Turanski和Damien Dallimore（Splunk）做得很好！
    7. 

  7. 新的SpringOne2GX回放现在可以在YouTube上以高清格式播放： Spring Security 3.1入门，Spring，Grails和云工具
    8. 

  8. JRebel家伙宣布了其插件的更新版本，该插件有助于在Java应用程序中实时重载代码。新版本的功能改进了对Spring的支持，支持基于@Configuration的Spring应用程序的实时重载。



  9. James Rossiter在有关如何使用Spring MVC的<form>标记和Spring MVC模型处理表单绑定的文章中非常出色。有两个职位。第一部分介绍如何绑定父对象和子对象 。第二部分介绍了initBinder机制。

  10. JavaBeat博客上有一篇不错的文章，介绍如何使用Spring的范围

  11. Pravin Chavan在如何使用Spring框架处理HTTP请求拦截方面有一篇不错的文章。

  12. 2Bloggers博客上有一篇不错的文章，介绍了BeanFactory和ApplicationContext对象在Spring中执行的不同职责以及它们之间的关系。

  13. TechRecipes博客上有一篇关于使用Spring的XML配置格式的Spring应用程序的基本配置的不错的文章。当然，这是非常入门的。


  14. Java EE Architectures博客上有一篇不错的文章， 介绍如何一起使用Spring MVC和Spring Data MongoDB 。

  15. 卡米洛·洛佩斯（Camilo Lopes）用葡萄牙语撰写了一本书（用葡萄牙语）的书评， 该书是《 春天框架 》（ Vire o jogo com） 。如果您正在寻找一本介绍Spring的书籍（葡萄牙语），那么这本书值得一读。

  16. Programming Fortune博客撰写了一篇不错的文章，该文章也涉及用表单域数据填充对象的主题 。
