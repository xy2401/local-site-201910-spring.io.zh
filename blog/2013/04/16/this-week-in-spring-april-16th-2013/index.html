Spring 的这个星期-2013年4月16日

工程
乔什·朗（Josh Long）
2013年4月16日

欢迎来到 Spring 本周的另一期！
对于Pivotal的Spring来说，这是令人兴奋的一周，您可以在4月24日重新安排的Pivotal 启动活动中了解到更多信息。



  1. 如果您阅读速度太快，Pivotal已将发布活动重新安排到4月24日。在这里注册 ！

  2. Arjen Poutsma有
    发布了Spring Web Services 2.1.3。发布     。新版本主要包含错误修复，有关完整详细信息， 请查看changelog 。

  3. 不要错过即将举行的网络研讨会，唐纳德·迈纳（Donald Miner）和马克·波拉克（Mark Pollack）讨论了Pivotal HD和Spring Hadoop ，这是对Pivotal HD感兴趣的人的一个很好的入门网络研讨会。

  4. 新的SpringOne2GX回放现在可以在YouTube上以高清格式播放： Cloud Foundry体系结构 ， NewSQL中的有效设计模式


  5. 前几天在Reddit上有一篇很棒的帖子，解释了两者之间的区别

    基于Martin Lawrence的REST和SOAP     。这与Spring 无关 ，但是却被拖累了，值得分享。
