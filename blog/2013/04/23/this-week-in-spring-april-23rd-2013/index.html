春天的这个星期-2013年4月23日

工程
乔什·朗（Josh Long）
2013年4月23日

欢迎回到春季本周的另一期！在旧金山，我们正经历着人生的初体验，以及春天典型的美丽天气。随着我们向4月24日发起的Pivotal Initiative进军， Pivotal开源社区（包括Cloud Foundry和SpringSource）中的事情可能变得忙碌起来，这也很合适。回头见！

事不宜迟，让我们进入本周综述：

  1. 你们看到过Spring Batch 2.2.0中惊人的Java配置支持吗？RC1 ？我刚刚链接到的代码演示了一个完整的Spring Batch作业，该作业读取.csv文件，然后将记录写入数据源，所有这些均在该类中完全用Java配置。这演示了2.2.0中的@EnableBatchProcessingannotation。看看吧 ！
  2. 现在，新的SpringOne2GX可以在YouTube上以高清格式重播： 通过Spring和vFabric实施域驱动设计， 在Cloud Foundry上进行批处理和集成以及一个额外的环节， 了解Java垃圾回收以及您可以采取的措施 。
  3. FuzzyDB开源项目发推文说，他们发布了新版本的FuzzyDB，其中Spring Data绑定与Spring Data Arora发行版保持一致。恭喜，伙计们！
  4. 您是否有机会使用Thymeleaf，HTML5和Spring MVC友好的模板引擎 ？如果您想学习更多 ，您可能会喜欢这个名为Thymeleaf的演示，它会融合吗？
  5. David Welch整理了Spring Data Mongo的快速演示，并提供了工作。他发了一条推文，说他在8分钟内从演示工作开始，其中包括4个课程和一个pom.xml 。干得好！
  6. 在SpringSource YouTube频道SpringSourceDev上观看 RamOnes Laddad 在SpringOne2GX上的精彩演讲CloudFoundry Architecture演讲 。
  7. 春季HATEOAS负责人Oliver Gierke在推文中发布了此帖子的链接“ 我如何向我的妻子解释REST ” ，乍一看似乎只是一个人向一个没有相同想法的人解释一个相当深奥的技术概念的尝试。技术背景，但很快就变成了REST的适用性（我认为非常有见地）。如果您想将REST-fu提升到一个新的水平，请查看Spring HATEOAS 。
  8. 我个人非常喜欢新的Java配置API，包括最近发布的Java配置API和预览版本中当前可用的API。我在上面展示了一个非常简单的Spring Batch API示例。上周，我还花了一些时间来撰写有关功能强大的Spring Social Java配置API的信息，该API即将在1.1.0中可用。M2发布。
  9. 您可以通过使用@Requiredannotation在运行时坚持使用非null值来满足某个属性的要求，从而让Spring执行某种前提条件检查，否则Spring将中止该对象的构造。这有助于避免任何无声的NullPointerException 。这个JavaBeat帖子很好地解释了如何使用@Required 。
  10. Java博客中的操作方法很好地介绍了如何在Spring Security 3中创建自定义UserDetailsService 。
  11. JavaCodeGeeks博客上有一篇不错的文章， 介绍了如何使用Spring MVC创建RESTful服务 。
  12. Brian的Java Blog上有一篇不错的文章，介绍了如何将Spring AOP与注释和以XML为中心的配置选项一起使用。
