Spring Tools 4.2.0发布

发布
马丁·利珀特
2019年3月29日

亲爱的春季社区，

我很高兴地宣布适用于Eclipse，Visual Studio Code和Atom的Spring Tools 4的4.2.0版本。

此版本的重点包括：

  • （春季启动）新增功能：允许为LSP进程“ PropertiesLauncher”配置虚拟机参数（ ＃211 ）
  • （Spring Boot）性能：通过跨语言服务器启动缓存创建的符号，大大改善了符号索引基础结构的性能
  • （Spring Boot）性能：用与JDT（语言服务器）的通信代替了内部类型索引，以节省时间和内存开销来保持我们自己的类型索引
  • （Eclipse）新：发行版更新到Eclipse 2019-03
  • （Eclipse）新增功能：现在默认在Spring Boot启动配置中启用Spring JMX支持
  • 各种其他错误修复和改进

Spring Tool Suite 3.9.8发布

发布
马丁·利珀特
三月27，2019

亲爱的春季社区，

嘿!

我很高兴宣布Spring Tool Suite 3（STS3）的3.9.8维护版本。

此版本的重点包括：

  • 更新到Eclipse 2019-03版本
  • 快速文本搜索使您可以在模式和非模式之间进行切换
  • 快速测试搜索使您可以将搜索范围限制为特定文件
  • 用于运行启动应用程序的新选项卡，这些应用程序显示来自应用程序环境的实时数据
  • 各种错误修复

要下载发行版，请访问：

  • Spring Tool Suite 3： https://spring.io/tools3/sts/all
