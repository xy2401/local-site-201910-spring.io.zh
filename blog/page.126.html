Spring Cloud Data Flow 1.0 GA发布

发布
马克·波拉克
2016年7月12日

我很高兴代表团队宣布Spring Cloud Data Flow的1.0 GA版本！

注意
 开始使用此新版本的一个好方法是遵循参考文档的“ 入门”部分 。它使用在计算机上运行的数据流服务器，并为每个应用程序部署一个新进程。

Spring Cloud Data Flow（SCDF）是现代运行时针对数据微服务的编排服务。SCDF允许您描述可以由长期流传输应用程序或短期任务应用程序组成的数据管道，然后将它们部署到您今天可能已经在使用的平台运行时中，例如Cloud Foundry，Apache YARN，Apache Mesos和Kubernetes。我们提供了广泛的流和任务应用程序，因此您可以立即开始开发针对用例的解决方案，例如数据提取，实时分析和数据导入/导出。

Spring Cloud Task 1.0.1。现在可以发布

发布
迈克尔·米内拉
2016年7月11日

我们很高兴地宣布发布Spring Cloud Task 1.0.1。发布。这是建议用于Spring Cloud Task的GA版本，可从Maven Central以及其他通常位置获得。

我们是怎么来到这里的？

Spring Cloud Task是Spring产品组合中数据应用程序发展的一步。当我们发布此版本时，了解一下如何达到这一点可能会有所帮助。

它从Spring Batch开始

建立在Spring产品组合的基础上，这种类型的有限工作负载的演变实际上始于Spring Batch 。作为Spring框架主要针对工作负载的扩展之一，Spring Batch提供了以企业友好的方式执行有限数量的处理的功能。具有多种处理模型（ Tasklet与Chunk）以及可伸缩性选项，这些选项允许在单个JVM内以及使用多个JVM来优化数据处理。多年来，Spring Batch一直是JVM上有限处理的标准载体。

CVE-2016-5007 Spring Security / MVC路径匹配不一致

工程
罗伯·温奇
2016年7月8日

Spring Framework 4.3.1和Spring Security 4.1.1提供了针对CVE-2016-5007 “ Spring Security / MVC路径匹配不一致”的修复程序。

使用Spring Security和Spring MVC的应用程序应升级到Spring Security 4.1.1+和Spring Framework 4.3.1+并使用MvcRequestMatcher 。
