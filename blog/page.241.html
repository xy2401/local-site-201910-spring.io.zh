Spring Data Babbage RC1发布

发布
奥利弗·德罗博姆
2013年8月5日

我很高兴地宣布Spring Data发布火车的Babbage集的第一个也是最后一个发布候选。我由以下模块组成：

  • Spring Data Commons 1.6 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data JPA 1.4 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data MongoDB 1.3 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Neo4j 2.3 RC1- 工件 -JavaDocs- 文档 -Changelog

该发行版是预期在今年SpringOne左右发布的GA发行版的重要里程碑。我们增加了对MongoDB聚合框架的支持，并改进了多态查询的执行。在JPA方面，我们引入了对在手动定义的查询中使用SpEL表达式的支持，并改进了使用@IdClass现在可以定义Date使用以下方式绑定存储库查询参数@TemporalType 。Neo4j模块增加了对countBy()查询并键入存储库的安全查询执行。您可以在Wiki页面上找到有关Babbage内容的更详细的功能列表。
