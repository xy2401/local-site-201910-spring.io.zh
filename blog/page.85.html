Spring Security 5.0.0 M3发布

发布
乔·格兰贾
2017年7月24日

我很高兴代表社区宣布Spring Security 5.0.0 M3的发布。该发行版包含错误修复，新功能，并且基于Spring Framework 5.0.0 RC3。

此里程碑中包括的新功能是：

  • 通过新模块spring-security-jwt-jose添加了对JSON Web令牌（JWT）和JSON Web签名（JWS）的支持

  • 为OpenID Connect身份验证流程集成了ID令牌

请参阅更改日志以获取有关此版本的更多详细信息。

Spring Framework 5.0 RC3-向GA迈进

发布

2017年7月24日

尊敬的Spring社区，

Spring Integration 4.3.11可用

发布
阿尔特姆·比兰
2017年7月20日

我很高兴代表Spring Integration团队宣布4.3.11。Spring Integration的发布现在可用。

此版本包含一些重要的错误修复以及一些改进。特别是它包含针对Jackson序列化小工具漏洞的修复程序。因此在版本4.3.10自定义中引入ObjectMapper实用程序现在不允许反序列化不受信任程序包中的对象。可以配置一组受信任的程序包，也可以使用*相信所有人。看到JacksonJsonUtils.messagingAwareMapper()想要查询更多的信息。
