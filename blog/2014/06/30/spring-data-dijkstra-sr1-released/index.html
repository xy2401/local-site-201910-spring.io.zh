Spring Data Dijkstra SR1发布

发布
奥利弗·德罗博姆
2014年6月30日

我很高兴代表Spring Data团队宣布Dijkstra发行列车的第一个服务版本的可用性。它总共包括以下模块的59个修复程序 ：

  • Spring Data Commons 1.8.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data JPA 1.6.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data MongoDB 1.5.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Neo4j 3.1.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Solr 1.2.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Couchbase 1.1.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Cassandra 1.0.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Elasticsearch 1.0.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Gemfire 1.4.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Redis 1.3.1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data REST 2.1.1- 工件 -JavaDocs- 文档 -Changelog

该版本是所有Dijkstra用户的建议升级，因为它包含一些重要的错误修复。鼓励仍在发布服务的Codd用户也进行升级。下一个版本的发布是即将发布的Evans培训的第一个里程碑。

如果您想了解更多有关Spring Data的信息，请务必参加今年的SpringOne会议。日程安排包含许多与数据相关的讲座，向您介绍我们将随Evans一起提供的最新功能。早鸟票仍然可用，因此请迅速注册 。
