提供Spring Data Release Train Codd的最终服务版本

发布
奥利弗·德罗博姆
2014年6月18日

我想宣布名为Codd的Spring Data发布系列的第三个也是最后一个服务版本的可用性。该发行版包含许多错误修复程序，这些缺陷已使其包含在Dijkstra发行培训的RC和GA版本中。

  • Spring Data Commons 1.7.3- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data JPA 1.5.3- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data MongoDB 1.4.3- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Neo4j 3.0.3- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Solr 1.1.3- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data REST 2.0.3- 工件 -JavaDocs- 文档 -Changelog

完整的更改列表可在此处找到。由于这是一个错误修正版本，我们建议升级。如果可以升级到当前的Spring Data Dijkstra GA版本 ，那就更好了。SR3是Codd发布系列的最新版本。我们将继续进行Dijkstra的第一个服务版本，然后是Evans的第一个里程碑。

如果您想了解更多有关Spring Data以及我们将随Evans发布火车一起提供的所有新内容的信息，请确保查看时间表并立即注册 。
