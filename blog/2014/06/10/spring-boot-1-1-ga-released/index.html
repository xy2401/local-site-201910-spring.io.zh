Spring Boot 1.1 GA发布

发布
菲尔·韦伯
2014年6月10日

Spring Boot 1.1.1已经发布，现在可以从repo.spring.io和maven central中获得 。该发行版将成为Spring IO平台的一部分，并提供超过1.0的许多新功能和改进。有关升级说明和“ 新功能和值得注意的功能”，请参阅发行说明 。

注意：请使用v1.1.1，因为最初的v1.1.0版本存在一个小问题

这儿是一些精彩片段：

模板支持

除了现有的Thymeleaf和JSP支持之外，还为Freemarker ， Velocity和Groovy添加了其他模板选项。Groovy模板特别酷，带有精美简洁的标记：

html(lang:'en') {
    head {
        meta('http-equiv':'"Content-Type" content="text/html; charset=utf-8"')
        title('My page')
    }
    body {
        p('This is an example of HTML contents')
    }
}

欲了解更多信息，请查阅“ 使用创新的Groovy的模板引擎通过”博客文章塞德里克Champeau 。

指标与健康

/metrics执行器端点现在包括更多信息，包括堆详细信息，类负载计数，线程信息和垃圾回收统计信息。所述HealthIndicator接口已得到改进，允许多个指示器豆，以及支持返回特定的HTTP状态代码时，有一个问题。提供了对JDBC的开箱即用的支持，Redis，Mongo和RabbitMQ。

Flyway和Liquibase支持

支持现在提供迁飞或Liquibase数据库迁移。使用任何一个项目都提供了一种简单的方法，可以随着产品的开发动态地发展数据库模式。

其他自动配置

现在包括针对GemFire ， HornetQ ， Elasticsearch ， Solr ， Spring Social ， Spring Integration ， Spring Integration ， Spring Mobile和Spring HATEOAS的其他自动配置和启动程序POM。如果要升级的项目以前为这些产品中的任何一个定义了配置，则应检查是否可以删除一些代码！

自定义横幅支持

这是我最喜欢的小改进之一，任何DropWizard用户都应该熟悉。现在，您可以通过将一个名为banner.txt的文件添加到类路径中来定义自定义横幅。

文档更新

该参考文档已被更新，以反映新的功能和一些额外的操作方法的已被添加。现在，我们还为spring-boot-maven-plugin发布了一个生成的Maven站点 。


感谢为此版本做出贡献的所有人，我们得到了社区的大量参与，并且许多新功能和改进由外部贡献者开发。请继续做好工作，继续提出这些问题并提出要求 ！
