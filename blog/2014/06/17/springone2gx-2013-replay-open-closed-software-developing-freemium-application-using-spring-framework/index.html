SpringOne2GX 2013重放：打开/关闭软件-使用Spring Framework开发免费增值应用程序

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年6月17日

在加利福尼亚州圣克拉拉的SpringOne2GX 2013上录制

演讲者：Frederic Simon开发涉及OSS的免费增值并非易事。一方面，您需要防止高级代码在您的免费模块中运行，并且应从容进行，而不会出现错误和性能下降。从另一方面讲，高级模块必须易于访问OSS核心。代码的部分公开可用性以及产品的两个不同版本的统一连续交付过程也具有挑战性。在本次演讲中，我们将展示Artifactory，该公司通过严重依赖于Spring中提供的灵活的依赖项注入机制成功地将OSS和Pro版本结合在一起。我们将讨论开发，构建，测试和发布混合免费增值应用程序，并将回顾现有方法，并讨论每种方法的优缺点。

在http://projects.spring.io/spring-framework上了解有关Spring框架的更多信息
