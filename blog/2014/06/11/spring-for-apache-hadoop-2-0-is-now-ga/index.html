适用于Apache Hadoop 2.0的Spring现已发布

发布
托马斯·里斯伯格
2014年6月11日

我们很高兴地宣布，适用于Apache Hadoop 2.0版的Spring现已正式发布。

-

提供对Hadoop v2.0和YARN的支持，同时保持与Hadoop v1.0的兼容性

自从Spring 发布 Apache Hadoop 1.0至今已有一年多了。当时，Spring是VMware的一部分。Spring for Apache Hadoop 1.0支持Hadoop 1.0.4和1.1.1，但不支持最新的2.0 alpha版本。从那以后很多事情改变了。Spring现在是Pivotal的一部分，它将我们与众多开源技术组合在一起。Apache Hadoop项目已经发布了几个新版本的Hadoop，其中包括引入YARN框架的2.2.0。因此，自然而然地继续推进Spring for Apache Hadoop项目。对于Apache Hadoop 2.0的Spring发行版，主要主题是：

  • 提供对新版本和发行版的支持，例如
  • Apache Hadoop 2.2.0、2.4.0
  • 关键高清2.0
  • Cloudera CDH 5
  • Hortonworks HDP 2.1
  • 保持与Hadoop v1.0的兼容性
  • 为开发YARN应用程序提供了很好的编程模型
  • 添加新功能，例如
  • 用于处理HDFS数据的新作家和读者
  • 对使用Kite SDK读写POJO数据集的新支持

我们提供了一些示例应用程序，以展示如何使用更常用的功能。我们也有一些Spring IO指南，用于开发Spring YARN应用程序。

Spring for Apache Hadoop 2.0 GA发行版的目标发行版是当前的Apache Hadoop v2稳定发行版2.2.0。这是用于2.0.0的版本。发布的传递依赖项。

对于其他版本或发行版，我们继续在Spring IO发行版本存储库中提供特定工件及其各自的传递依赖项：
* 2.0.0。发布（默认-Apache Hadoop稳定版2.2.0）
* 2.0.0。RELEASE-hadoop24（Apache Hadoop最新2.4.0）
* 2.0.0。RELEASE-hadoop12（Apache Hadoop稳定版1.2.1）
* 2.0.0。RELEASE-phd1（Pivotal HD 1.1）
* 2.0.0。RELEASE-phd20（Pivotal HD 2.0）
* 2.0.0。发行CDH4（Cloudera CDH4 MR1）
* 2.0.0。RELEASE-cdh5（Cloudera CDH5纱）
* 2.0.0。RELEASE-hdp13（Hortonworks HDP 1.3）
* 2.0.0。RELEASE-hdp20（Hortonworks HDP 2.0）
* 2.0.0。RELEASE-hdp21（Hortonworks HDP 2.1）

有关如何使用这些版本进行构建的说明，请参阅项目Wiki 。

Maven Central存储库中还提供了Apache Hadoop 2.2.0、2.4.0和1.2.1的版本。

有关特定项目的更多信息，请参见项目页面 。
