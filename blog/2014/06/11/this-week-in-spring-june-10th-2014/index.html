春天的这个星期-2014年6月10日

工程
乔什·朗（Josh Long）
2014年6月11日

欢迎来到春季本周的另一期。这一周真是令人惊讶的一周！我是在波兰的克拉科夫开始的，在美妙的第33度会议上与开发人员进行了交谈，然后前往波兰的华沙，在当地的Spring Labs用户组中进行了介绍。明天，它将在伦敦Devoxx UK举行，在那里我将帮助推广Spring Boot很棒。然后，星期五，要去多伦多，下周的春季我将与4个客户和团队见面。这对你意味着什么？好吧，一个，如果你在伦敦和多伦多，打个招呼 ！这也意味着会议季节正全面展开。我们尝试在需要的任何地方。什么会议（当然还有SpringOne2GX 2014！）你今年要去吗？我很想知道您今年要参加哪些会议。请帮助我们确定优先顺序。谢谢！（Twitter标签#twiSpring ）

  • Spring Boot 1.1发布了 ！它包括对GSP，Spring Social，Velocity，Freemarker，Spring Mobile，Spring HATEOAS等的新支持。许多新的自动配置支持也支持：GemFire，HornetQ，Elasticsearch，Solr，Spring Social，Spring Integration，Spring Mobile和Spring HATEOAS。老实说，阅读博客然后再回来可能会更好。那里真棒。
  • 我们的CFSummit会议现在在旧金山举行！Cloud Foundry周围有很多神奇的东西。我发现了许多新闻发布，但这是来自eWeek的关于PCF 1.3中一些新内容的新闻 。
  • 如果您不是本周在旧金山举行的CloudFoundry峰会，您可能已经错过了Matt Stine的精彩演讲： Cloud Foundry和微服务：互惠共生的关系 。您首先在这里听说过：CloudFoundry和Spring Boot将成为微服务的杀手combination。
  • 为庆祝Spring Boot发布，令人惊叹的Greg Turnquist将于7月8日展示使用Spring Boot构建您的应用程序管理工具 。
  • 加入Glenn Renfro，他将带您全面了解Spring Integration 4.0的Boot / JavaConfig / Annotation驱动的经验， 并于6月17日完成了Spring Integration Done Booti 。
  • Casey Doolittle和Phil Dutson展示了终极的电子商务功能防火练习：7月15日使用春季在24小时内构建购物车
  • 回顾一下去年的SpringOne2GX 2013和SpringOne2GX 2013的演讲重播：太空探索：用于具有Sencha Space的安全互连HTML5应用程序的API
  • 令人惊叹的Craig Walls 去年在SpringOne2GX 2013上就Spring Social进行了精彩演讲 ，请查看！毕竟，Spring Social 1.1刚刚下降。现在是赶上并与您的客户联系起来的好时机。
  • Greg Turnquist在撰写Spring IO指南的过程中发布了一个非常不错的博客。格雷格还是指南的主要贡献者之一，因此，如果他批准，那么您就会知道这很好。继续前进，开源！
  • 说到该新指南，您应该检查一下：它是关于如何开始使用Spring WS构建SOAP服务的 。 （嘿，对于很多人来说，这仍然是一件事情，Spring WS的新的基于Java配置的端点模型非常适合该任务！）
  • Ramnivas Laddad，AspectJ背后的大脑之一，Cloud Foundry的架构师之一，以及我们的本地云忍者，最近在Spring Cloud上进行了一次网络研讨会，这使得在干净的， 依赖注入为中心的 PaaS绑定服务的使用变得容易方式 ：检查一下！
  • 曼努埃尔·乔丹（Manuel Jordan）是秘鲁社区的杰出成员（我的几本书的编辑，他在论坛上做出的杰出贡献使他获得了Springy奖，在论坛上他回答了比任何人都多的问题，而且周围都是好人），秘鲁一系列史诗般的博客，介绍了Spring的各个部分。检查出来 。干得好，曼努埃尔！
  • softwarecave博客上有一篇关于在Spring上使用Hibernate的文章。我进行了深入研究，但会提醒人们几乎没有理由不再使用XML，而且-即使有理由，也没有理由再没有理由编写这种配置了。只需使用Spring Boot。令人惊讶的是， @EnableAutoConfiguration产生了什么变化！
  • GeekAByte博客上有一篇不错的文章， 介绍了Spring可能进行的各种以注释为中心的自动装配 。看看这个！
