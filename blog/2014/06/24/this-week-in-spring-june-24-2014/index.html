春天的这个星期-2014年6月24日

工程
乔什·朗（Josh Long）
2014年6月24日

欢迎来到春季“本周”的另一期-本周，我将与惊人的Vaadin团队合作开发一些非常有趣的应用程序。准备在不久的将来获得更多！另外， SpringOne2GX 2014早鸟已延长至6月30日 ，请立即注册！事不宜迟，向前：

  • Spring Boot 1.1.2现已发布 ！新版本的功能主要是错误修复等。看看这个！
  • Adib Saikali进行了精彩的演讲，介绍了Spring IO其余平台从Java 8中受益的地方 ，以及我们之前在Spring框架中所描述的用途。
  • 观看Spring Batch主管Michael Minella在网络研讨会上介绍Spring Batch 3.0的回放
  • 查看Glenn Renfro 在物联网世界中使用Spring Integration的介绍
  • Spring Data负责人Oliver Gierke宣布了Spring Data Codd的最终服务版本 。此发行版包含Dijkstra发行培训的RC和GA版本中已经存在的错误修复。
  • 令人惊讶的Artem Bilan刚刚宣布了Spring Integration Java DSL的M2版本，该版本在最近的Spring Integration 4版本的末尾发布 。新版本具有许多改进和错误修复。检查一下，踢一下轮胎，并确保让我们知道 ！
  • 我很喜欢Spring Data的这个不错的西班牙语高级入门 。
  • Spring一直提供Spring内部的某种事件ApplicationContext 。这些事件为系统的一部分与其他部分通信提供了一种方法。Spring Boot更进一步，并提供诸如Spring Security身份验证和身份验证失败之类的事件。HoserDude博客上的Tim 很好地介绍了在Spring中使用事件的一些方法

  • 是否曾经想过将Spring MVC路由定义集中在一个外部文件中？我要么……但是，许多其他人似乎想要！我碰巧是由我们自己的Brian Clozel负责这个项目的 ，他提供了确切的信息。看看这个！

  • 观看C24夸张的CEO的重播，从SpringOne2GX 2013开始引入分布式规则引擎和复杂事件处理 。
  • Spring Data Gemfire（当然还有Gemfire本身）代表着一拖二的扩展：水平扩展与基于内存的读取和写入相结合。这篇来自SpringOne2GX 2013的Lyndon Adams的演讲很好地介绍了如何使用Gemfire和Spring应对潮汐数据泛滥（物联网）
  • 是否想从Spring Boot使用GORM？看看Grails-lead Graeme Rocher的这个例子 ：太简洁了！
  • 在Spring Security支持的集成中使用后退按钮会让您失望吗？查看Pham Thai Thinh的这篇文章 。本博客介绍了一些使资源快速过期的方法，这些资源不应从浏览器的历史记录中访问。好贴！我可能会修改该示例以使用Java配置，但是无论哪种方式，这都是一个不错的示例。值得注意的是，在Spring Security 3.2+中，这可以通过内置的Spring Security标头支持来完成。
  • Pham Thai Thinh在使用现有的基于JDBC的表作为Spring Security的后端身份提供程序方面 也做得很好。
  • Gretty似乎像Maven生态系统插件Cargo一样，用于在构建中运行Web应用程序， 但Gradle除外， Gretty 现在也支持Spring Boot应用程序
  • 丹尼尔·布莱恩特（Daniel Bryant）撰写了一篇不错的文章，内容涉及在Spring Boot支持的跨（微）服务中实现相关ID 。在处理和跟踪多个分布式组件之间的状态时，这是一件非常方便的事情。
