@SpELAssert(value = "password.equals(passwordVerify)",

春天的这个星期-2014年6月3日

工程
乔什·朗（Josh Long）
2014年6月3日

欢迎来到春季本周的另一期。这一周真是令人兴奋的一周！好吧，说实话。他们都是激动人心的星期。但特别是，本周将使很多人感到高兴。观看博客 ，下周我会再见到您。 :)

  1. 哦，我的天啊！Spring Boot 1.1.0。RC1现在可用 ！新版本保持了史诗与春数据Elasticsearch，HornetQ的，和Spring社会支持，以及更多 ！获取最新版本，不懈努力，并在Twitter或GitHub上提供反馈。
  2. Mark Pollack博士刚刚宣布了Spring XD的最新版本1.0.0。M7现在可用。新版本提供了许多很棒的新功能。我最喜欢的功能是将数据固定到特定流的能力-将其视为使用消息内容的关联 -从而可以保留有状态操作。可以认为这是一种基于有用的业务密钥来路由和划分消息的好方法。发行说明中有一个很好的例子。
  3. Azul摇滚明星Gil Tene发表了一个关于减少SpringOne2gX 2013延迟的精彩演讲， 该新闻现已在线发布 。吉尔是我们这个行业的疯狂科学家之一。我还没有看过这个演讲，但是我会的，我强烈建议您也这样做。Azul将高性能，低延迟的JVM用作设备和可交付软件。因此，他的演讲源于许多令人费解的研究和开发，我希望尽快通过观看和学习他的演讲来发掘自己。去吧，吉尔！
  4. 6月的网络研讨会在这里！Michael Minella于6月10日发布于Spring Batch 3.0.0中 ，而Glenn Renfro于6月17日发布于Spring Integration中 。
  5. 春季忍者格雷格·特恩奎斯特（Greg Turnquist）撰写了有关使用惊人工具的预告片when.js在他的SpringOne2GX 2014演讲之前，在前端REST客户端中实现Promises / A + 。在会议上查看帖子和 他的演讲 ！
  6. Ramnivas Laddad，Spring忍者，AspectJ的原始领导人和Cloud Foundry的架构师，刚刚发布了Spring Cloud的非常酷的外观 ，它使来自不同中间/基础架构服务（数据库，消息队列）的客户端服务可以在各种平台上使用服务即服务（PaaSes）是一个简单的平台解耦配置问题。
  7. 现在，ADP的Jeffery Sologov 关于构建大型应用程序的陷阱的演讲重播了！看看这个！
  8. TTP：//twitter.com/JakubJirutka插话，告诉我们这部史诗 春表达式语言（规划环境地政司）支持的执行Bean验证API的（JSR349分之303） 。GitHub提供了一个有趣的观点，“它对于使用纯Bean验证非常复杂的跨域验证特别有用。”我也喜欢这些例子：
    @SpELAssert(value = "password.equals(passwordVerify)",
        applyIf = "password || passwordVerify",
        message = "{validator.passwords_not_same}")
public class User {
  private String password;
  private String passwordVerify;
}

    不错的工作！

  9. 令人赞叹的Brian Dussault找到了一个有趣的提示 ： Zuul是一个漂亮的应用程序配置管理解决方案 ，提供了干净的Spring客户端API。

  10. 你知道我昨天过得怎样吗？一种HystrixInvocationHandler 。一个InvocationHandler由JDK（和Spring的丰富代理子系统）用来创建包装bean的代理。这个InvocationHandler将方法调用包装在Netflix OSS Hystrix项目的给定bean中Command对象。Hystrix Command包装功能并提供/支持弹性模式 。我等不及要看Spencer Gibb的Halfpipe项目的更多内容了！
  11. 我们的朋友David Welch再次参加了这个有趣的项目，名为Spring Tiered，该项目旨在进一步简化（并规范化） 基于HATEOAS的服务的开发 。有趣...
  12. 此外，谈到构建（并使用）弹性服务，请查看Chris Richardson在SpringOne2GX 2013上的精彩演讲，内容涉及用于异步使用服务的强大抽象 。
  13. 此外，我整理了一篇有关Maven（和替代品）和Spring入门的文章。
