Spring IO Platform 1.0.0发布

发布
乔什·朗（Josh Long）
2014年6月26日

我们很高兴地宣布Spring IO Platform 1.0.0 RELEASE的发布，并代表了一些令人惊叹的工程工作。此版本的重点是：

  • 材料清单发布（Maven BOM依赖项）级别为您设置版本。
  • Spring IO代表精心协调的依赖关系，它们可以一起工作-测试涵盖了所有Spring项目以及许多流行的第三方开放源代码
  • 一个平台，许多工作负载-构建Web（基于HTML，Websocket和REST的Web应用程序），移动，集成，批处理，响应式或大数据应用程序，或者仅是出色的API
  • 使用Spring Boot彻底简化了开发经验
  • 开箱即用提供生产就绪功能（监视，指标等）
  • 模块化平台，允许开发人员仅部署他们需要的部件
  • 跨独立JVM，嵌入式运行时，成熟的Java EE应用程序服务器以及PaaS部署（如Cloud Foundry）的真正可移植性
  • 仅取决于Java SE，并支持Groovy，Grails和某些Java EE
  • 与您现有的依赖管理工具（例如Maven和Gradle）一起使用
  • 已认证Spring IO Platform可在JDK 7和8上运行（尽管各个项目可能会逐个项目地支持较旧的JDK）
  • 构建下一代微服务风格架构的理想平台

Spring IO 1.0现在可用，并且包括所有Spring项目，但Spring XD除外，该产品有望在今年晚些时候全面上市。Pivotal可以从http://spring.io/services获得Spring IO的服务，支持和培训。
