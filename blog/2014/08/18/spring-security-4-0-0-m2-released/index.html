Spring Security 4.0.0。M2发布

发布
罗伯·温奇
2014年8月18日

我很高兴宣布Spring Security 4.0.0的发布。M2可在Spring Milestone存储库中获得 。

SpringOne 2GX 2014指日可待
即将于9月8日至11日在德克萨斯州达拉斯的SpringOne预订您的位置。这是发现第一手一切并提供直接反馈的最佳机会。从0到Spring Security 4.0会话将包含有关如何开始使用Spring Security的详细信息，并提供对Spring Security 4中新功能的深入了解。当然，还有很多其他与Spring相关的令人兴奋的话题 ！

变更日志

您可以在发行说明中找到有关此发行版的详细信息。该版本的重点包括：

  • 使用Spring的消息传递抽象支持WebSocket安全
  • Spring数据集成
  • Spring Session是基于此发行版所需的工作而产生的。通过独立于应用程序服务器的方式，您可以轻松地使用自定义后端（即Redis）作为HttpSession实施。有关此令人兴奋的新项目的其他功能，请参见项目页面。
  • 增强和修复测试支持
  • 易于集成MockMvc通过SecurityMockMvcConfigurers
  • 您不再需要指定WithSecurityContextTestExecutionListener在您的测试中
  • @WithSecurityContext即使自定义也可以使用SecurityContextRepository实作
  • 支持基于摘要和证书的身份验证测试
  • 在LDAP身份验证中支持嵌套的静态组
  • 许多集成测试已添加到示例应用程序中
  • 更新了Spring 4.1 RC2的最低版本。这对于足够的功能来说是必要的，因此有必要全面进行

敬请关注spring.io博客 ，其中有一系列博客介绍了这些令人兴奋的新功能。
