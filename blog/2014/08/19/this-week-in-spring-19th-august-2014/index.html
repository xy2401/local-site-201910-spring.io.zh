2014年8月19日至19日，本周

工程
乔什·朗（Josh Long）
2014年8月19日

欢迎来到春季本周的另一期！已经距离SpringOne2GX 2014几周了！当您玩得开心时光飞逝！2015年初将是春季本周的第4年，每个星期都会更加精彩！我希望在SpringOne2GX 2014上看到更多的人。这肯定是一个了不起的事件。

无论如何，让我们继续本周的综述！

  1. Spring Security主管Rob Winch刚刚宣布了Spring Security 4.0.0。M2
    它提供了许多出色的功能，包括改进的安全意识测试支持，Spring Data集成，对Websocket安全性的支持，以及与该版本的工作衍生的新生的Spring Session项目的联系。看看这个！Rob还宣布了Spring Security 3.2.5和3.1.7 ，此修复程序弥补了一些潜在的安全漏洞。到处都是好东西！
  • Spring框架4.1。RC2刚刚掉线 。春季4.1是非常接近完成，所以一定要在此版本中检查出的最新和最伟大的位！
  • Christoph Strobl刚刚宣布了Spring Data Evans发布火车的第一个RC 。新版本包含许多功能，包括用于Spring Data MongoDB的文本搜索集成，改进的多存储支持，用于Redis标记的配置选项等等。
  • 我们在Codecentric的朋友们刚刚宣布了针对Spring Batch的自定义Spring Boot自动配置的新版本。此实现不同于Spring Boot随附的自动配置，它包括对JSR 352的支持。
  • Netflix开发人员Dan Woods最近发表了精彩的演讲，探讨了他们希望从基于Spring Boot的Grails 3中脱颖而出的想法，幻灯片平台绝对值得一看。
  • Being Java Guys博客上有一篇不错的文章，介绍了如何使用Spring 4和Hibernate 4构建REST服务。该示例使用web.xml和Spring XML配置。这是一个工作的食谱，但我希望读者会记得，你可以得到更短的时间和春天开机做了同样和代码。
  • Oracle确实对WebLogic 12.1.3中的基于Spring的应用程序提供了特别的支持 ，我承认，我并不知道。如果您使用的是WebLogic，这是一个非常有趣的阅读。
  • Tirthal的Java咖啡馆博客介绍了最近宣布的Spring IO 。
  • 关于构建基于Spring的简单Web应用程序的越南语帖子非常简单，值得一读。
  • 这篇文章介绍了如何集成Spring Security 3.0.4和GWT
