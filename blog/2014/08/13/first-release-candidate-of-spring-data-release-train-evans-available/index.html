春季数据发布培训Evans的第一个发布候选版本

发布
克里斯多夫·斯特罗布尔
2014年8月13日

我们很高兴地宣布Spring Data发布火车Evans的第一个候选发布。我们提供以下模块：

  • Spring Data Commons 1.9 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data JPA 1.7 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data MongoDB 1.6 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Neo4j 3.2 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Solr 1.3 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Couchbase 1.2 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Cassandra 1.1 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Elasticsearch 1.1 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Gemfire 1.5 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data Redis 1.4 RC1- 工件 -JavaDocs- 文档 -Changelog
  • Spring Data REST 2.2 RC1- 工件 -JavaDocs- 文档 -Changelog

自第一个里程碑以来发生的一些亮点是：

  • 如果您想并排使用其他商店，则增强了对多商店的支持 。
  • 使用CDI时支持自定义存储库实现 。
  • MongoDB存储库的文本搜索集成。
  • Redis前哨的配置选项 。
  • 弹性查询的地理查询 。
  • 我们将参考文档从Docbook移到了Asciidoctor。

总共处理了95张票，因此请确保您已经预订了达拉斯SpringOne的地方，以第一手了解Spring Data的新变化。同时，请注意有关此版本选定功能的即将发布的博客文章。发行火车Wiki页面也可能是找到相关新内容的良好起点。
