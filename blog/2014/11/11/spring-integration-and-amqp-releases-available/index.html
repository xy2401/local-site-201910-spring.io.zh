Spring集成和AMQP版本可用

发布
加里·罗素
2014年11月11日

亲爱的春季社区，

根据最近发布的Spring Integration和Spring AMQP候选版本公告，我们非常高兴地宣布Spring Integration 4.1.0。发布和Spring AMQP 1.4.0。现在可以下载RELEASE。

2014年11月12日，DZone研究机构在其最新调查结果的发布中，将 Spring Integration评为ESB / Integration框架领域的领导者，占有42％的市场份额。

非常感谢所有贡献者，其中包括一些社区新手。

这些版本现已在Spring存储库中提供，并将在几个小时内在Maven Central中。

有关更多信息以及Spring Integration和Spring AMQP的“新功能”，请参阅先前的公告[5] [6]。

自发布候选版本以来仅有几处更改，主要是文档和一些错误修复。有关详细信息，请参见发行说明[1] [2]。

有关更多详细信息，请参见项目页面[3] [4]。

此外，还提供以下维护版本，其中包含重要的错误修复。单击这些链接以查看相应的发行说明，其中显示了已解决的问题Spring Integration 4.0.5。发布 ， Spring Integration 3.0.6。RELEASE和Spring AMQP 1.3.7。发布 。
