春天的这个星期-2014年11月4日

工程
乔什·朗（Josh Long）
2014年11月4日

欢迎来到春季本周的另一期！这周我在路上交谈卡萨布兰卡，摩洛哥热心Spring开发人员在上JMaghreb会议，那么它的关闭伦敦（再次！）为惊人的Spring eXchange 。如果您在任何一个，请务必打个招呼！

  • Spring Integration忍者，我的个人英雄Artem Bilan 刚刚宣布了Spring Integration RC1版本。我不知道您是否看过Java配置DSL，但这使编写复杂的集成流程变得轻而易举！它利用了Java 8的lamba支持。看看这个！请记住，微服务并非仅以REST为中心。如果您试图构建微服务并将其通过任何其他机制公开，那么我将从Spring Integration开始。
  • 春天的忍者Thomas Risberg刚刚宣布了Hadoop 2.0.3的春天 ！此版本增加了对Apache Hadoop 2.5.1（hadoop25）和Pivotal HD 2.1（phd21）的支持。
  • Spring Data忍者Christoph Strobl刚刚宣布了Spring Data Evans服务版本1（SR1） 。新版本包含许多重要更新，是建议的升级。
  • 我将与ZeroTurnaround的Adam Koblentz进行网络研讨会，介绍如何使用Spring（尤其是Spring Boot）和JRebel 6 。Spring Boot使Java尽其所能地提高了生产力，但是有时您需要更多一点来解决Java传统上对开发过程中实时重新加载代码（如加速迭代）的糟糕支持。在这个网络研讨会中，我们将研究如何使用JRebel！
  • 查看有关使用Spring Boot和Vaadin构建“启动” UI的网络研讨会的回放
  • 观看Russ Danner的网络研讨会的重播，介绍如何构建支持内容的Web和移动应用程序
  • 我喜欢Matt Raible的史诗般的帖子，他介绍了他如何使用Spring MVC，SPring Data REST（HAL支持），Swagger，Liquibase和JAXB构建REST API ，所有这些当然都是通过Spring Boot进行的 。这个帖子赢得了书签！
  • 上周在波士顿举行的聚会上，您是否看到Thomas Risberg在Hadoop Spring Boot上的史诗般的演讲？如果没有， 请在GitHub上检查代码和幻灯片 ！他们在照亮！
  • 丹尼斯·舒尔特，比上CodeCentric把官方博客一起非常好贴使用Logstash，ElasticSearch，和Spring Boot 。看看这个！
  • 我花了一些时间来帮助一些误入歧途的人，帮助他们解决Spring的细微差别@Qualifier注释并在Spring博客上发布了详细信息。希望这对外面的人有帮助！
  • 本·黑尔（Ben Hale）领导Cloud Foundry上的Java经验，并且本身就是Spring的摇滚明星，他为New Relic做了一则客座帖子，介绍了一键式部署体验，以便在基于Cloud Foundry的应用程序上设置New Relic 。绝对值得一读！如果您想了解New Relic对Spring开发人员的意义，我可以谦虚地推荐这个网络研讨会，我和我今年早些时候做了New Relic的Ashly Puls吗？
  • 最近，社区成员Marc Schipperheyn开设了JIRA，以寻求有关Spring在现代单页应用程序样式Web应用程序中的作用的指南 。接下来的讨论很有启发性，原因有很多，除了会激发我们文档中的更改之外！检查一下，如果您有贡献，您的$ 02.00！
  • 在基于Spring的单元测试之前，我们的朋友Petri Kainulainen撰写了一篇不错的文章，介绍如何重置自动增量列
  • 我们的好朋友Rafal Borowiec再次回来了，这次的帖子介绍了如何创建Spring Boot执行器端点
  • 您在使用Apache Camel吗？您是否知道它现在提供了基于Spring Boot的自动配置 ？
