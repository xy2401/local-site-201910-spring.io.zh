Pivotal在DZone的2014年企业集成指南中得到认可

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年11月12日

我们很高兴能在DZone的2014年企业集成指南中被公认为ESB和轻量级集成框架市场的领导者，这是一种专注于企业集成和API管理趋势，策略和工具的高级资源。该指南包括主题介绍，专家意见，最佳实践和解决方案比较。该指南的读者将获得企业集成的概述，并了解开发人员创建无缝集成所面临的障碍。该指南涵盖的主题包括：

  • 消息队列，中间件和ESB在企业中的角色。
  • 用于分解整体架构的分解模式。
  • 用于了解REST API成熟度的模型。
  • 预测将来如何构建具有多个集成的大型项目。

DZone的企业集成指南还通过对500多个开发人员和专家的调查，提供了有关集成和API管理实践的关键见解，使读者可以从技术专业社区的从业人员那里学习趋势。此外，该指南的解决方案目录比较了不同的API管理平台，集成套件，ESB，消息队列和集成框架，以帮助读者明智地选择所需的解决方案。

读者可以在此处下载该指南的免费副本。

关于DZone

DZone为世界各地的开发人员，技术专业人员和聪明人提供专家研究和学习社区。15年来，DZone一直是值得信赖的全球内容来源。

关于Spring Integration 4.1 GA版本

在进行DZone研究的同一天，Spring Integration 4.1 GA提供了第一类Spring Framework 4.1，Java 8，WebSocket支持。长期用户将享受新的内置EIP模式，Redis Queue Gateway支持。最重要的是，请查看即将发布的Java DSL-非常令人兴奋。Rabbit用户还将发现Spring AMQP 1.4 GA充分利用了Spring Framework 4.1，并提供了Rabbit消息传递增强功能。
