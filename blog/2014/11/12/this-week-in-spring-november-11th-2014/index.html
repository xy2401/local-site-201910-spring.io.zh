春天的这个星期-2014年11月11日

工程
乔什·朗（Josh Long）
2014年11月12日

欢迎来到春季本周的另一期！本周，我回到了硅谷，与一些大型技术驱动公司的开发人员进行了交谈，探讨了他们采用云原生架构的第一步。一如既往的有趣反馈！本周，我将在海湾地区举行聚会， 讨论使用Spring Cloud构建Bootiful微服务 。下周，我期待与大家在保加利亚索非亚的Java2Days见面 ！C'ya在那里！

  1. Spring Integration负责人Gary Russell刚刚宣布了Spring Integration 4.1.0和Spring AMQP 1.4.0 final现已发布！
  2. Spring Security OAuth 2.0.4。发布现在可用 。新版本包含一些不错的附加功能和一些错误修复，使其成为推荐的轻松升级。
  3. Spring负责人Juergen Hoeller刚刚宣布了Spring Framework 4.1.2的可用性，并附带了4.0.8和3.2.12维护版本 。
  4. 我将与ZeroTurnaround的Adam Koblentz进行网络研讨会，介绍如何使用Spring（尤其是Spring Boot）和JRebel 6 。Spring Boot使Java尽其所能地提高了生产力，但是有时您需要更多一点来解决Java传统上对开发过程中实时重新加载代码（如加速迭代）的糟糕支持。在这个网络研讨会中，我们将研究如何使用JRebel！
  5. 继续11月18日的微服务网络研讨会系列，标记Redmonk的Matt Stine和Donnie Berkholz的日历-彼此制造：微服务+ PaaS
  6. 除了小组会议和一些根本没有幻灯片的会议：)，SpringOne2GX会议幻灯片都发布在www.springone2gx.com上的日程安排页面的会议详细信息中。
  7. SpringOne2GX 2014的开幕主题演讲- 第1 天和第2天 -现在可以在线获取！我本人从第二天开始，但是那只是我...-哨子-
  8. 说到SpringOne2GX 2014，Splunk传播者Damien Dallimore的演讲“ 获得应用程序生命周期智能”现在可以在线重播！
  9. 2014年SpringOne2GX上新的遗物摇滚明星Ashley Puls的演讲， “通过字节码操作在矩阵中生活”现在也可以重播！如果我这么大胆，Ashley也很乐意与我在今年早些时候进行网络研讨会，内容涉及将New Relic用于Spring应用程序。当我对奥术或复杂事物有疑问时，阿什利（Ashley）只是一小段人。她很聪明。观看这些演讲，看看我的意思！
  10. 谈到入围名单上的辉煌，优秀的Dave Syer博士最近举行了一个网络研讨会，介绍了针对微服务的Spring Security OAuth 。我不能太强调这一点：Syer博士是或曾经是Spring Batch，Spring Security OAuth，Spring Boot，Spring Cloud的联合创始人或项目负责人，他是Spring Framework 的贡献者，当然- Cloud Foundry的一名高级工程师。他讲话时，我会倾听，希望您也能。观看此视频。
  11. 说到Spring Cloud，Syer博士最近还宣布了Spring Cloud的M2版本 。我用来展示Spring Cloud 的Spring Cloud示例应用程序和（希望不久将更容易访问，但琐碎） 应用程序都已更新为使用Spring Cloud M2。
  12. Pivotal自己的Tim Spann pu在一起非常不错的文章，介绍了如何使用Spring XD将文件加载到Pivotal HD的HAWK中
  13. Sergei Egorov撰写了一篇很好的文章，介绍了使用Thrift构建微服务的过程，该序列化机制在大数据空间中非常流行
  14. 是否想要构建REST服务并公开有关它的文档？昂首阔步的路要走。我不确定这是最好的（它是30 MB！），但这是...某物。这是将Swagger与Spring Boot结合使用的不错的帖子
  15. Biju Kunjummen在使用Spring缓存抽象对Google Guava Cache的支持方面有一篇不错的文章。
  16. Concretepage引入了一种简单的以Java配置为中心的方法来构建Spring Batch解决方案
  17. Biju Kunjummen 还 使用Spring Session从websockets捕获（并使用）HTTP会话进行了很好的编写-请检查一下！
  18. 试图访问AOP代理后面的目标对象 ？这篇文章提供了一种实现方法！
