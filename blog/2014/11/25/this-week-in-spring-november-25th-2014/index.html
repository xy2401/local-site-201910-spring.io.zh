Spring 的这个星期-2014年11月25日

工程
乔什·朗（Josh Long）
2014年11月25日

哎呀！您可以相信已经快到年底了吗？时间一直在尖叫！对我们美国的人们来说，这是感恩节的一周。这是一年中度过的美好时光，找到一个不错的Spring IO指南或博客来阅读，然后建立一些很棒的东西..哦，这是与亲朋好友一起度过并反思这些时光的理想时机我们对此深表感谢。当我说我们感谢您，我们富有启发性的社区，用户和客户时，我确定我会代表Spring团队和整个Pivotal。

但是，如果您也想阅读一些内容，本周我们将进行综述！因此，事不宜迟..！

  1. Spring Integration忍者Artem Bilan 刚刚发布了Spring Integration Java Configuration DSL，1.0。发布 ！欢呼！这个版本已经发布了很长一段时间，希望您能检查一下！Artem Bilan Gary Russell在使该API取得成果方面做得非常出色。在博客中查看可以参考的资源列表。
  2. Spring XD联合负责人Mark Pollack博士刚刚宣布了Spring XD 1.1。M1和1.0.2 。1.1正在向GA迈进，并提供对Apache Spark，Python，Apache Kafka，改进的HDFS接收器功能，LDAP认证，Java配置，Redis和JDBC源以及Zookeeper的支持。那只是初学者！好的医生简要地暗示了下一个1.1之前的版本将要发布的内容！阅读博客，了解有关1.0.2中可用内容的更多信息和详细信息！
  3. Spring Boot的共同负责人Phil Webb刚刚宣布了Spring Boot 1.2.0。RC2 ，包含许多新功能！尝试一下，立即踢轮胎 ！
  4. 我写了Spring Boot 1.2中的一些新功能，这些功能旨在支持 JAX-RS和JTA等通用Java EE API ，以及对RedHat的WildFly应用服务器提供支持的Undertow HTTP引擎的新支持。
  5. DISID公司在其领导下发布了Spring Roo的第一个版本1.3，该版本为Spring Roo和Spring Roo生成的应用程序提供了完整的JDK 8支持 。
  6. ZeroTurnaround的Adam Koblentz和我进行了一个网络研讨会，介绍如何使用Spring Boot和JRebel来减少可重载类的迭代时间 。
  7. 观看Ben Hale的SpringOne2gx 2014演讲的重播，该演讲使用Spring Boot创建基于REST的，基于超媒体的微服务 。
  8. Matt Stine在2014年关于使用Spring和Cloud Foundry开发支持PaaS的微服务的演讲也已在线发布。
  9. Cornelia Davis在SpringOne2GX 2014上有关在云中运行Spring应用程序（例如Cloud Foundry）的演讲也已在线发布。
  10. 观看Spring Ninja Greg Turnquist 在5分钟内构建并运行基于Spring Boot的REST服务 。为什么？因为他可以 ！
  11. 在Geekout UK讲完 Spring，Cloud Foundry，微服务等所有内容之后，我进行了一次简短的采访。看看这个！
  12. 请查看Cloud Foundry生态系统合作伙伴ActiveState 撰写的有关为PaaS构建微服务的有趣系列文章 。第一篇文章在那里，向下滚动以获得该系列的下一部分。
  13. 想了解ZookKeeper和Netflix的Consul之间的区别吗？看看 Ivan Glushkov撰写的这张整洁的幻灯片 。 13
  14. 我们在ZeroTurnaround上的好朋友Simon Maple共同探讨了如何快速构建与Facebook进行对话的Spring Boot应用程序
  15. 对象合作伙伴博客上的Jacob Severson在关于使用Reactor项目（和Spring Boot）构建UDP服务的精彩文章中
  16. 在All and Sundry博客上查看此帖子，了解如何使用Spring Session将HTTP会话状态保持在Redis这样的后备存储中。
  17. 查看此博客，了解如何使用带有Spring Boot的嵌入式Apache Tomcat设置HTTPS 。
  18. 查看有关使用Spring Security的中文介绍 。
  19. AirPair有一篇关于使用Java 8的有趣文章Stream ，以将数据流式传输回客户端以用于占用大量内存的用例 。

当我们进入感恩节假期时，我们将在土耳其上班，然后狂欢地观看Netflix，请记住，新的估算表明， Netflix占据了北美互联网上游流量的9.5％ 。甚至不要问下游 ！要了解Neflix用于扩展的微服务堆栈（包括Spring和越来越多的Spring Boot），并了解如何将其与Spring Cloud集成， 请查看我几个月前提供的网络研讨会 ！一旦火鸡吃完了，那就来销售吧！中国刚刚庆祝了一次类似的大型购物活动，一个在线电子零售商阿里巴巴在 Spring 的一天内创造了93 亿美元的销售额。去年，我在博客中写了关于他们疯狂的规模，《中国规模的 Spring 》 。

谢谢 ，感恩节快乐！
