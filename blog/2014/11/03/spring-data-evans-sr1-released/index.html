Spring Data Evans SR1发布

发布
克里斯多夫·斯特罗布尔
2014年11月3日

我很高兴代表Spring Data团队宣布Evans发布系列的第一个服务版本的可用性。参与模块的团队完成了41个问题，包括服务器错误修复。我们要感谢大家花时间报告，自愿修复和测试这些问题-这是我们无法缺少的宝贵反馈！

该发行版包括以下11个模块：
* Spring Data Commons 1.9.1- 工件 -JavaDocs- 文档 -Changelog

* Spring Data MongoDB 1.6.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Neo4j 3.2.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Solr 1.3.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Couchbase 1.2.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Cassandra 1.1.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Elasticsearch 1.1.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Gemfire 1.5.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data Redis 1.4.1- 工件 -JavaDocs- 文档 -Changelog
* Spring Data REST 2.2.1- 工件 -JavaDocs- 文档 -Changelog

建议升级Evans SR 1，因为它包含重要的错误修复。鼓励仍在发布服务的Dijkstra用户也进行升级。下一个发行版将是发行火车Fowler的第一个里程碑。
