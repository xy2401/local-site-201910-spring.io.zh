Spring Integration Java DSL 1.0 GA发布

发布
阿尔特姆·比兰（Artem Bilan）
2014年11月24日

尊敬的Spring社区，

正如我们在Release Candidate 博客文章中所承诺的那样，我们很高兴地宣布Spring Integration Java DSL 1.0 GA现在可用。像往常一样，使用带有Maven或Gradle的Release Repository ，或下载发行档案 ，以进行试用。

有关更多信息，请参见项目主页 。

首先，我们很高兴与您分享，2014年11月12日， DZone研究机构在其最新调查结果的出版物中将Spring Integration评为ESB / Integration框架领域的领导者，占有42％的市场份额。该报告是11月份最受欢迎的DZone指南，下载量已超过12000！不要错过：非常令人兴奋。我们希望Spring Integration Java DSL的发布能带来更多的刺激！

非常感谢所有贡献者，其中包括一些社区新手。

该版本自候选版本以来仅包含一些错误修复，以及许多JavaDocs！

与发布无关，我想在此提供一些有关此问题的资源。

我们在Stack Overflow上观察到许多有价值的DSL问题。

Josh Long的技术技巧展示了如何将Spring Boot，REST，Spring Integration 4.1 WebSocket支持和Spring Integration Java DSL以及Java 8功能一起使用。

的Jdbc Splitter在项目测试中实施。

我的要点展示了如何使用Reactor Stream与Spring Integration Java DSL一起使用。

Dave Syer已开始在Spring Cloud Bus项目中使用Spring Integration Java DSL。

如2014年SpringOne / 2GX大会所示，不要错过si4demo来了解Spring Integration（包括Java DSL）的发展。（视频应会很快出现）。

特殊感谢六必居Kunjummen谁做对DZone一些不错的文章，介绍Spring集成的Java DSL： http://java.dzone.com/articles/spring-integration-java-dsl ， http://java.dzone.com/ articles / spring-integration-java-dsl-0 。

当然，使用最新的Spring XD ，我们可以基于@Configuration包括Spring Integration Java DSL IntegrationFlow定义。

另请参见全面的Cafe Demo：逐行教程 。

一如既往，我们期待您的评论和反馈（ StackOverflow （ spring-integration标签）， Spring JIRA和GitHub ），我们非常欢迎您的贡献 ！
