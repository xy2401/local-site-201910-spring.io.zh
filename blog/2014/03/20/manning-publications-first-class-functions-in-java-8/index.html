曼宁出版物：Java 8中的一流函数

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年3月20日

Java开发人员已经听到了很多有关函数式编程的信息，特别是Java 8增加了lambda表达式。

“ Java 8中的一流功能”是Java 8 in Action的12分钟视频教程，由Raoul-Gabriel Urma，Mario Fusco和Alan Mycroft编写，内容为Java 8 in Action：Lambda，流和函数式编程 。

该视频介绍了方法引用和lambda表达式，Java 8提供了将行为作为参数传递给方法的机制。

要在此书或任何其他Manning书籍上节省38％，只需在manning.com上签出时在“促销代码”框中输入vturma01。

虽然“ lambda表达式”一词听起来很抽象和学术性，但Java 8 Lambda会对您每天的编程方式产生很大的影响。用最简单的术语来说，lambda表达式是一个函数（一些代码），您可以将其作为参数传递给另一个方法。因此，您可以通过使用由lambda表示的行为作为参数来应对不断变化的需求。Java 8的功能性编程功能（例如lambda和新的Stream API）提供了一种更清晰的方式遍历集合的方式，可以帮助您编写简洁，可维护的代码，这些代码可以轻松扩展并在多核体系结构上表现良好。Java 8 in Action是Java 8 lambda和Java函数编程的清晰编写指南。它从实际介绍Java语言中lambda表达式的结构和好处开始。然后，本书介绍了Stream API，并展示了它如何使与集合相关的代码从根本上更易于理解和维护。在此过程中，您将发现Java 8的面向FP的新设计模式，这些代码可用于代码重用，代码可读性，异常处理，数据操作和并发。对于也在JVM上探索其他功能语言的开发人员，本书最后简要介绍了Scala和Clojure中的有用功能。
