    春天的这个星期-2014年3月4日

    工程
    乔什·朗（Josh Long）
    2014年3月5日

    欢迎来到春季本周的另一期。

    和往常一样，我们有很多内容需要介绍，让我们开始吧！

    1. Spring Batch和Boot的联合创始人Dave Syer博士宣布Spring Boot RC4现已上市 。查看所有糖果的最新切割！
    2. 有一个新指南详细介绍了如何使用Spring Data REST
    3. Spring Data忍者Christoph Strobl宣布Spring Data Redis 1.2现在可用。
    4. 在3月18日举行的在线研讨会上，不要错过Mattias Arthursson，该研讨会将展示Spring LDAP 2.0.0 。
    5. 3月25 日，与 Juergen Hoeller和Spring团队一起参加Java 8和Spring Framework 4.0的网络研讨会！
    6. David Turanski撰写了一篇不错的文章， 介绍如何使用Groovy进行bean配置
    7. SpringOne2GX 2013的Alvaro Videla和Jan Machacek的演讲， RabbitMQ是新的King ，现已在线提供
    8. 加里·罗素（Gary Russell）在2013年的SpringOne2GX演讲中介绍了Spring Integration的内部结构
    9. Pance Cavkovski的精彩文章展示了JavaWebSocket项目Java EE 7和Spring 4对Web套接字的支持 。
    10. Norris Shelton Jr.有一篇不错的文章，介绍如何从当前安装的版本访问Spring Security主体Authentication使用自定义注释的对象
    11. Quinten Krijger撰写了一篇不错的文章，介绍如何使用Spring Security和Spring MVC管理会话并发
    12. Tomasz Nurkiewicz撰写了有关如何构建自定义Spring XML名称空间的文章 。不过，他在一开始就提出了一个很重要的观点：Spring 不需要 XML！
    13. “ 不仅仅是另一个博客”博客着眼于如何覆盖Spring Security过滤器链
    14. Thys Michels撰写了一篇简短的文章，内容涉及如何使用Java配置来配置Spring应用程序以使用Heroku上的ElasticSearch服务 。
