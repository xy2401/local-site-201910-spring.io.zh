    Spring Data Release Train Babbage的最终服务发布

    发布
    奥利弗·德罗博姆（Oliver Drotbohm）
    2014年3月10日

    我很高兴地宣布，Spring Data Babbage发布系列的最后一个服务版本已经发布。该发行版包括以下模块：

    • Spring Data Commons 1.6.5- Changelog - JavaDoc - Artifacts
    • Spring Data JPA 1.4.5- 更改日志 -JavaDoc- 工件
    • Spring Data MongoDB 1.3.5- Changelog - JavaDoc- 工件
    • Spring Data Neo4j 2.3.5- Changelog - JavaDoc- 神器

    该发行版捆绑了许多重要的增强功能和错误修复，是建议的升级。该版本是Babbage发布系列的最后一个服务版本。建议用户查看Spring Data Codd的最新版本，然后升级到它们。

    通过在项目中包含以下依赖项，可以轻松升级到服务版本：

    
  
    
      org.springframework.data
      spring-data-releasetrain
      Babbage-SR4
      pom
      import

    下一步将是计划于3月中旬发布的Codd发行列车的第一个服务版本，以及即将发行的Dijkstra发行列车的第一个里程碑。有关此新发行版本火车的更多详细信息，请参见此处 。
