Spring 的这个星期-2014年3月11日

工程
乔什·朗（Josh Long）
2014年3月12日

欢迎来到 Spring 本周的另一期！和往常一样，我们有很多内容要讲，所以让我们开始吧！

  1. Spring AMQP负责人Gary Russell刚刚宣布Spring AMQP 1.3.0刚刚发布 ！
  2. Spring Social负责人Craig Walls刚刚宣布了第一个期待已久的Spring Social 1.1 RC 。欢呼！我很高兴看到这个新版本。
  3. Spring Security负责人Rob Winch刚刚宣布了Spring Security 3.2.2和3.1.6的安全修复程序 。
  4. Spring Boot的共同负责人Dave Syer博士在许多其他令人惊奇的事情中， 为Ratpack（一种用于JVM Web应用程序的工具包）提供了一个看起来很性感的Spring Boot包装器 。这是如此真棒。
  5. Spring Data负责人Oliver Gierke刚刚宣布， Spring Data Babbage的第一个服务版本现已发布
  6. 我撰写了一篇有关Spring Boot应用程序的各种部署策略的文章
  7. 在3月18日举行的在线研讨会上，不要错过Mattias Arthursson，该研讨会将展示Spring LDAP 2.0.0 。
  8. 3月25 日，与 Juergen Hoeller和Spring团队一起参加Java 8和Spring Framework 4.0的网络研讨会！
  9. 谈到Spring Data，请查看本次演讲的重播，其中介绍了SpringOne2GX 2013中的Spring Data存储库最佳实践
  10. 不要错过John Hann 在SpringOne2GX 2013中对实用JavaScript的介绍
  11. Emanuel Rabina 在SpringOne2GX 2013上发表了有关使用自然模板和Thymeleaf改善Spring视图层的演讲 ：请查看！
  12. More Vaadin博客在Spring Boot / Vaadin集成中发表了一篇不错的文章，如果简短的话，我和Vaadin的PetterHolmstrÃ¶m一直在努力 。真好！与往常一样，这是开源的，所以不要害羞，并确保发送您的任何反馈 。
  13. 在相关新闻中， 我和Petter正在研究的Vaadin4Spring 集成的快照现已发布在Sonatype的存储库中 -尽情享受！
  14. 我们的好朋友Eberhard Wolff回来了，这次他在Spring 4.0和Spring Boot上写了一篇不错的（德语）文章，其中包括（很多）其他内容
  15. 戴维·威廉姆斯（David Williams）博客， 《 Cloud Foundry的实现价值的故事》令人深思 ：一种技术能够多快地交付价值 -可衡量的价值？
  16. Martin Fowler已经开始关注微服务领域 。没有什么要报告的，特别是。这只是我建议观看的页面。
  17. CA技术公司的第7层刚刚发布了针对API设计和部署的调查结果。该调查有很多收获，因此请务必阅读，但我认为特别有希望的一件事是：API开发人员中超媒体感知API的预期增长 。如果您对采用超媒体和HATEOAS设计模式感兴趣，我鼓励您查看Spring HATEOAS，并且-如果您想了解有关使用Spring HATEOAS和Spring Boot的更多信息， 请查阅本教程和相关代码 。
