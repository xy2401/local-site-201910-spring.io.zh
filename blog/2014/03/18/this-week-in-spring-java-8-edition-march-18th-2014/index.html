本周春季，Java 8版-2014年3月18日

工程
乔什·朗（Josh Long）
2014年3月18日

欢迎来到春季本周的另一期！

今天，Oracle在EclipseCon上发布了Java 8及其所有出色的功能 。Java 8终于来了！一定要立即获取您的JDK。

有很多优秀的功能（例如lambas）可以增强Spring应用程序的开发。要了解更多信息，请务必查看3月25日Spring项目主持人Juergen Hoeller的网络研讨会，主题是Spring 4和Java 8 。我会在那里！

与往常一样，我们有很多内容需要介绍，所以让我们开始吧。

  1. InfoQ发表了一篇非常不错的文章，由Dan Woods在今天介绍Spring Boot 。丹做得很好，我推荐。
  • Spring Security主管Rob Winch宣布了Spring Test MVC HtmlUnit框架1.0.0。M1已发布。该项目旨在整合HtmlUnit和Spring MVC Test。
  • Spring Data负责人Oliver Gierke刚刚宣布了Spring Data Codd发布火车的第一个服务版本 。
  • Oliver代表Michael Nitschinger 宣布了Spring Data Couchbase 1.0 GA版本 。恭喜发布，团队！
  • 我们的好朋友Petri Kainulainen回来了一篇不错的文章，内容涉及如何编写和集成测试Spring Social和Spring MVC集成 。像往常一样，这是一篇非常深刻的文章，是一系列连续文章的一部分。像往常一样都值得一读。

  • 说到EclipseCon上的，它也非常酷，我们自己的春天工具套件导致马丁·利珀特 keynoted沿与Eclipse基金会总监迈克·米林科维奇在包装的EclipseCon 2014今天炫耀项目通量。

  • 我们的好朋友Tomas Lin又来了一次，这一次的帖子是关于如何使用websockets，Spring Boot，StompJS和AngularJS构建应用程序的 。这篇文章并没有涵盖实际使用这些技术的内容，而是介绍了使用这些技术时遇到的难题。他获得了与决议，JIRA，样本等相关的链接，这对他的旅程有所帮助，并且由于他的贡献，使所有人受益。Spring Boot仍然不是GA，因此获得实际反馈是一个巨大的胜利。托马斯，真是棒极了！

  • 我写了一篇文章来纪念网站的25岁生日之际，2014年3月，12日 。在文章中，我将以Web作为平台的状态进行研究，并考察支持Web开发解决方案的Spring项目的传播。
  • 如果您在海湾地区的时候，我会给出3月25日的谈话在枢纽办公室介绍Spring数据堆栈
  • 免费编程博客上有一篇关于使用Spring MVC 4.0构建RESTful JSON Web服务的详细文章。
  • 最后但并非最不重要的一点是，我们的好朋友Sergi Almar回来了，这次非常有力地看着用Spring 4和RabbitMQ跟踪您的Websocket消息 。很好看！
