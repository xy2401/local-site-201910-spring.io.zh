Michael Nitschinger, I’m happy to announce the 1.0 GA release of the Couchbase Spring Data module.

Between the last release candidate and the final release, several bugs have been fixed and new features have been added. Notable additions are the support for custom converters, JSR-303 Validation support and built-in support for temporal objects like Dates, Calendars and similar JodaTime variants.

The release artifacts are available via Maven Central. You can find the reference documentation, JavaDoc and a change log at the usual expected places.

Spring Data Couchbase 1.0 GA发布

发布
奥利弗·德罗博姆
2014年3月13日

我很高兴代表Michael Nitschinger宣布Couchbase Spring Data模块的1.0 GA版本。

在最后一个候选版本和最终版本之间，修复了多个错误，并添加了新功能。值得注意的新增功能是对自定义转换器的支持，JSR-303验证支持以及对日期，日历和类似的JodaTime变体等时态对象的内置支持。

可通过Maven Central获得发行工件。您可以在通常的预期位置找到参考文档 ， JavaDoc和更改日志 。

从现在开始，Couchbase模块将从Dijkstra开始成为Spring Data正式发布火车的一部分。现在，第一个GA版本已经面世，我们将更加关注与诸如Spring Boot或Spring XD之类的姊妹项目的集成。
