春天的这个星期-2014年4月22日

工程
乔什·朗（Josh Long）
2014年4月22日

欢迎来到春季本周的另一期！本周，我在印度班加罗尔举行的“伟大的印度开发者峰会”上，与开发者讨论（等待）。弹簧！我还在班加罗尔电子城的一个大型系统集成商处见了面，并向使用Spring进行全球项目的大型建筑师和工程师团队致辞。印度是一个有趣的地方，因为它们的大部分业务来自国外的公司，这些公司试图在原本不堪重负的项目中获得更多帮助。自然，任何有助于更快完成任务的事情都在这里引起关注，Spring是最受欢迎的。不用说，Spring Boot引起了很多共鸣！但是，当当地美食和这里的美食一样时，这还不是全部！ :)

本周有许多在线讲座和重播-我们有很多内容要报道，所以让我们开始吧！

  • Hadoop饿了吗？为一场盛宴做准备-Mark Pollack博士和XD团队宣布， Spring XD 1.0.0 M6现已上市 。这个新版本具有新功能，包括纱线运行支持一个TON，对于DIRT分布式运行时动物园管理员支持，机器学习算法，改进的壳制表完成时，反应器供电的TCP / UDP源，可以做1M + TX /秒在便宜的硬件上，默认是FTP到HDFS作业，还有更多！（不，实际上， 我们的意思是 ）。此发行版感觉就像他们以某种方式跳过了所有1.0.x里程碑，并发布了“ 2.0”版本！
  • Reactor项目负责人Jon Brisbin与Typesafe，Netflix，Twitter等公司的其他工程师一起宣布了用于反应流处理的新通用API 。这是巨大的 ！
  • Spring Data负责人Oliver Gierke刚刚宣布服务2的Spring Data Codd现已上市 。这个新版本非常庞大，因此请务必将它们全部检查一遍！
  • Dave Syer博士刚刚宣布发布Spring Security OAuth 2.0.0。RC1 。这个项目的发展非常迅速，我喜欢将代码更新到我的演讲“ Spring REST Stack”中 ，以使用越来越简洁和强大的迭代。我已经更新了该分支（仍在更新单元测试）以使用Spring Security 2.0.0。RC1。这是一项受OAuth保护的有效REST服务，支持HTTP和HTTPS，可以独立运行，也可以传统运行.war ，并使用自定义UserDetailsService bean和Java 8 lambda。不错！
  • Spring Integration正在寻找一个令人难以置信的新版本-完全支持注释和Java配置+一些Spring Boot支持！使用4.0，您将能够制作XML的免费集成应用程序。项目负责人Gary Russell 将于5月13日在网络研讨会Spring Integration 4.0（新领域）中带您领略所有新热点 。
  • 4月30日，我们和New Relic的Ashley Puls一起加入 ，我们将通过Javascript（前端）和Java / Spring（后端）应用程序进行跟踪。
  • 克里斯·比姆斯（Chris Beams）在Sagan项目上撰写了另一篇很棒的文章， 该文章是关于移植到Java 8的 。看看这个！
  • 我最近的演讲“使用Spring Boot构建“ Bootiful”应用程序”的重放 现在可以在我们的YouTube频道上找到 。
  • Spring Security负责人Rob Winch 再次发表了与常见第三方项目进行高级Spring MVC测试集成的文章。这篇文章是关于使用Geb测试Web应用程序的 。
  • Spring Data Neo4j的负责人Michael Hunger和我将在5月20日举行有关Spring Data Neo4j 3.0和Neo4j 2.0的新功能的网络研讨会。
  • 在大脑上搜索？您现在可以在Elasticsearch的前同事和Spring提交人Costin Leau的前排座位上坐90分钟，这些座位现在是Elasticsearch： 您的数据，您的搜索，Elasticsearch 。
  • 不要错过Pivotal的Mike Wiesner，他将在此SpringOne2GX 2013 Replay：应用程序安全陷阱中解决实用的应用程序安全性（不仅仅是春季安全性）。
  • Groovy语言的领导者Guillaume LaForge的史诗般的Groovy Weekly专栏进展顺利！总有好东西，这周也不例外。检查一下是否挖掘了Groovy（和Grails）的所有内容。
  • 该项目由Alexandre Rafalovitch进行，旨在改进SOLR JavaDocs。这里有很多值得佩服的地方，但是我喜欢这个Spring Boot示例，该示例演示了连接到SOLR实例的REST端点 。真好！
  • Fabio Maffioletti撰写了一篇非常不错的文章，其中讨论了如何通过Atomikos将JTA与Spring（尤其是Spring Boot和Spring Data JPA）结合使用，以协调两个不同数据库之间的事务 。这篇文章以我在2011年针对同一主题写的一篇较旧的文章开始，介绍了Spring Boot和Spring Data，并通过JPA将示例从一个数据库和一个JMS代理更改为一个数据库和另一个数据库。这真是太酷了！
  • Yuan Ji一直在与Spring HATEOAS一起构建关于HAL兼容的REST服务的很好的文章，对 它们进行单元测试，以及最近使用Angular-HAL来使用这些服务。
  • 我们的好朋友Brian Hannaway回来了，这次他发表了有关构建Spring Batch应用程序的文章 。内容真的很好，并且对于那些喜欢学习Spring Batch的人来说都是值得一读的。在帖子中，Brian演示了如何使用XML名称空间构建批处理应用程序。Spring Batch还支持非常好的Java配置API，并且使用Spring Boot，可以非常简洁地编写Spring Batch应用程序。我以Brian的非常酷的示例为例，并使用Spring Boot和Java配置API编写了不同版本以供参考 。不过，绝对要从Brian的博客开始，否则我的存储库中的代码将毫无意义！
