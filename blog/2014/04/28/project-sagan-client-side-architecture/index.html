Sagan项目：客户端架构

工程
布赖恩·克洛泽尔
2014年4月28日

现在我们对JavaScript模块有了更多的了解 ，我们已经准备好深入研究Sagan应用程序的客户端体系结构。

注意 ：如果您以前没有阅读过有关Sagan项目的博客文章，则应该知道这是Spring参考应用程序，可为spring.io和其他博客提供支持。以前的文章展示了如何运行此应用程序 ， 如何进行零停机部署以及如何将Sagan升级为使用最新的JDK8功能 。

在这篇文章中，我想遍历Sagan应用程序中客户端体系结构的基础知识：

  1. 为什么将客户端应用程序放在单独的位置sagan-client项目模块？
  2. 它与sagan-site模块？
  3. 什么是npm ， bower和gulp ？
  4. 这些工具如何一起工作以制作客户端应用程序？

为了说明所有这些，下面的截屏视频不仅会向您展示如何添加新的Bower依赖关系并编写一个小的新功能（网站上的键盘快捷键！），而且还将介绍如何增强gulp构建（优化图像！）。

注意 ：请参阅显示说明 ，以获取截屏视频中提到的站点的链接。您也可以在vimeo上以高清观看此截屏视频 。
