Spring 的这个星期-2014年4月8日

工程
乔什·朗（Josh Long）
2014年4月8日

欢迎来到 Spring 本周的另一期！

明天，我将在关于Spring Boot 的虚拟 JUG vJUG上进行演示。不要告诉任何人，但是演示文稿本身将基于Spring Boot联合创始人Phil Webb和我一直在计划的一次演讲。该代码已经在线 ，所以请检查一下，我期待明天见到您！如果你错过了谈话，而你恰好是在海湾地区，然后我们一起在周四晚上的关键办事处在这里旧金山既菲尔和自己同样的谈话更含咖啡因的版本！

然后，这个星期天，我要去传播 Spring 的爱！我将首先在巴黎为Devoxx FR着陆。从那里出发，前往印度班加罗尔参加“ 伟大的印度开发者峰会” 。自2012年以来我一直没有去过那里，我非常期待它并在那里看到令人惊叹的Spring社区。然后，在回家的路上，我将前往比利时， 在比利时JUG演讲 。我将要进行的大多数讨论都是以Boot为中心的，所以不要错过一个机会，看看Spring Boot为什么会引起关注。

  1. Spring联合负责人Syer博士宣布Spring Boot 1.0.1现已上市 。新版本包含安全修复程序和新功能。检查一下，立即升级！
  • Spring ninjaStéphaneNicoll提醒我们，Spring Framework 4.1.0将提供对JSR-107批注的支持。建立快照。真好！
  • http://spring.io架构师Chris Beams在他的系列文章中发布了另一期文章 ，该文章着眼于Sagan项目（ spring.io网站 ）。这篇文章详细介绍了网站如何实现零停机时间部署。看看这个！
  • Spring Data GemFire 1.3.4已发布。我希望我有发行说明或有关内容的详细信息，但我没有。
  • 查看有关EMC如何使用Spring和tcServer的案例研究
  • Mark Pollack博士宣布了Spring Shell 1.1 RC1已发布。新版本具有错误修复，JLine 2的更新以及命令解析器的重写。
  • Thomas Risberg宣布已发布适用于Apache Hadoop 2.0 RC2的Spring
  • Spring web-dude Rossen Stoyanchev宣布已发布Spring WebFlow 2.4 RC1。您知道该版本中我最喜欢的新功能是什么吗？Java配置支持！上周，Spring Integration Java配置API的第一个版本宣布了，我很高兴地报告，Spring WS Java配置API的工作正在进行中。借助Spring Web Flow，Spring Integration和Spring WS，将为所有主要的Spring项目提供Java配置API。 （呜！）
  • 我们的朋友托比亚斯·弗洛赫（Tobias Flohre）又来了，这次发表了一篇帖子， 详细介绍了Spring Batch BatchStatus作品 。真好！
  • Hortonworks撰写了一篇不错的文章，详细介绍了如何使用Spring XD和Hortonworks Hadoop将推文流式传输到Hadoop进行情感分析 。当然，您也可以使用其他Hadoop。
