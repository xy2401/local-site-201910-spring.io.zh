春天的这个星期-2014年4月29日

工程
乔什·朗（Josh Long）
2014年4月29日

欢迎来到春季本周的另一期！上周，我在印度班加罗尔度过了一段绝对美好的时光，昨天，在疯癫的杨树Devoxx会议创始人斯蒂芬·詹森 （ Stephan Janssen ）邀请下，我跳入比利时的鲁汶，快速访问了比利时Java用户组，在那里与我进行了交流一些绝对热情的本地人，当然还有我的pal和Spring框架提交人StéphaneNicoll 。当然，整个视频也应该很快在Parleys上发布！

让我们开始吧！

  1. 春天社会的共同创始人和领导克雷格·沃尔斯刚刚宣布了期待已久的 春天社会1.1.0版本 ！这个新版本太棒了！如果您在过去的一年半中一直在关注预发布版本的削减，那么您将知道对如何轻松展示Spring Social的Java配置有一些重新思考。最终版本既简洁又通用。看到这一点我感到非常兴奋，并将开始相应地更新我的各种以Spring Social为动力的OAuth客户端！很好，克雷格！
  2. 您是否看到新的“ 次要 ” Spring Boot版本1.0.2 ？我使用引号是因为，对于Spring Boot，即使是次要发行版也很麻烦！Syer博士宣布了该版本，并指出（其中包括）新奇的@IntegrationTest注解。太棒了！（现在请原谅，当我去更新Boot项目时...）
  3. 明天4月30日与我和来自New Relic的Ashley Puls一起加入，我们将通过Javascript（前端）和Java / Spring（后端）应用程序进行跟踪。
  4. Spring Integration正在寻找一个令人难以置信的新版本-完全支持注释和Java配置+一些Spring Boot支持！使用4.0，您将能够制作XML的免费集成应用程序。项目负责人Gary Russell 将于5月13日在网络研讨会Spring Integration 4.0（新领域）中带您领略所有新热点 。
  5. Spring Data Neo4j的负责人Michael Hunger和我将在5月20日举行有关Spring Data Neo4j 3.0和Neo4j 2.0的新功能的网络研讨会。
  6. Spring框架忍者Brian Clozel继承了有关Project Sagan的系列，刚刚撰写了关于Project Sagan如何执行客户端JavaScript的很好的介绍。很好地了解了现代客户端应用程序的外观npm ， grunt和gulp 。最好的部分？内容以视频形式提供！完善！
  7. Spring Roo是DISID的主要贡献者 ！查看我的好朋友Pieter Humphrey关于Spring Roo的未来的精彩文章。
  8. 与Chris Richardson在SpringOne2GX 2013上分享90分钟的时间，他讨论Java，Scala和Javascript的未来。不，我们不是指产品路线图。查看他的主题为： Futures和Rx Observables的会议： 异步使用Web服务的强大抽象 。
  9. 您仍然认为Spring仅用于依赖项注入吗？加入Mark Secrist，对基础概念（例如底层设计模式和框架的构建模块）进行令人惊叹和启发性的展示-高度可重用的见解。观看他的SpringOne2GX 2013会议的重播：超越依赖注入 。
  10. XavierPadró撰写了一篇不错的文章，介绍了如何在充当Web服务客户端时使用Spring Integration配置超时
  11. IntelliJ忍者Andrey Cheptsov在有关如何使用Jetbrains起源的语言Kotlin，NoSQL数据库MongoDB，Spring Boot和PaaS Heroku的文章中发表了一篇非常可爱的文章。真好！
  12. Petri Kainulainen再次回来了，这一次，他撰写了一篇很棒的文章，介绍了如何使用jOOQ-这使得使用SQL更加容易-与Spring一起处理分页和排序 。
  13. 罗杰·休斯（Roger Hughes）在Spring的Quartz调度支持中跟踪异常有一个不错的文章。
  14. 关于时间快要用完的一些不错的帖子，请不要失去它。博客。注意的第一个是这个帖子很奇怪ClassNotFoundException在JBoss EAP 6.2的较早版本的Spring Integration上展示
  15. 第二篇文章，稍早一些，是关于使用Spring Integration MQTT适配器与基于MQTT的服务进行通信的 。MQTT是一种轻量级消息传递协议，它是当今许多基于物联网的解决方案的核心。谁知道？您的冰箱可能正在使用它！ :)
  16. 是否想在本地计算机上运行更像生产的Hadoop实例？不想运行完整的虚拟机？查看有关使用Docker部署Pivotal的HD Hadoop发行版的文章 ！（然后，查看Spring for Hadoop和Spring XD ！）
