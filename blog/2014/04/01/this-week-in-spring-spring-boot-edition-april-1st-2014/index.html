春季本周（春季启动版！）-2014年4月1日

工程
乔什·朗（Josh Long）
2014年4月1日

欢迎大家参加重要的本周春季引导版 ！

今天当然是愚人节。4月1日的笑话不断涌现！我们今年没有准备任何实际的笑话，但我总是想向人们指出Spring联合创始人Rod Johnson宣布的将SpringSource（Spring的创立公司命名）出售给Microsoft的这段史诗般的视频！ （在2008年 ， 愚人节 ）

说出来的方式，还有更有趣的东西今天要谈（无任何轴承愚人节）： 春天开机其中-就在今天早上- 终于走到GA ！祝贺Phillip Webb和Dave Syer博士 （以及为他们提供帮助的人们所获得的分数）发布了这一惊人的版本。

我不知道它是什么样子从外面看，在看，但我们对Spring团队已经从概念看菲利普和戴夫驱动器引导，以抛光样机在那里取得了巨大的水花在SpringOne2GX 2013 ，到GA在去年还有一些变化。Boot已经通知了许多其他Spring项目，包括Spring Framework 4.0。一项真正的改变游戏规则的技术，并且不仅限于JVM。

如果您还没有尝试过，那么现在是时候了。有一个关于从STS 3.5使用Spring Boot的视频（也将很快到期） 。完全不需要 STS，但是它确实有一些方便的工具！看看这个。

  1. 大新闻： Spring Boot 1.0.0已经达到了它的第一个主要版本！
  2. Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration 3.0.2 和 Spring Integration 4.0 M4 。Spring Integration 3.0.2中的新功能很棒，但是请向下滚动以查看Spring Integration，Boot和Java配置演示 ！我很高兴看到这个版本。 #boot
  3. 春季领导者Juergen Hoeller宣布了Spring 4.0.3版本，该版本具有Java 8支持并大大改进了websocket支持。
  4. Spring Data传奇人物Thomas Darimont宣布Spring Data Redis 1.2.1现已可用 。新版本包含针对的错误修正RedisCacheManager和RedisTemplate 。
  5. Spring Data负责人Oliver Gierke刚刚发布了Spring Data Dijkstra的第一个里程碑 。看看这个！
  6. 4月9日，我将在全球范围内进行vJUG演示，介绍Spring Boot 。我的希望是-在所分配的短时间内-演示使用Spring Boot构建应用程序的样子。我希望在那里见到您，也请随时在IRC频道上提问。 #boot
  7. 我们的朋友Chris Richardson在博客中对此进行了介绍，该博客描述了一种使用Spring Boot构建微服务的方法 。该示例在Scala中。看看这个！#boot
  8. Spring Boot被提名为最具创新性的Java技术 。我不确定投票程序的工作方式，但是一旦发现，我会在这里发布。无论哪种方式，我希望我们都能依靠您的帮助来推动投票！#boot
  9. Marco Vermeulen进行了非常不错的演讲，介绍了如何使用Spring Boot来构建微服务 。这个例子在Groovy中。 #boot
  10. Jim Drannbauer整理了一个GitHub存储库，演示了如何将最近发布的 Spring MVC Test HtmlUnit和Cucumber一起使用 。有了这个， 您的单元测试几乎就像人类的句子一样阅读 ！#boot
  11. 我们的朋友Matt Raible再一次回来了，这次不是一个，而是关于使用Spring Boot的两篇文章！第一篇文章演示了如何将Swagger集成添加到Spring Boot应用程序中 ，第二篇文章演示了使用Ionic框架为Boot后端应用程序构建iOS客户端的感觉 。 #boot
  12. 杂乱无章的博客又回来了，这次有一篇关于同时使用Spring Boot和Scala的文章 。真好！无论您使用哪种语言，引导都是一个胜利！#boot
  13. 塞尔吉·阿尔玛（Sergi Almar）本周回来了，主题是如何使用JMX监视Websocket线程池 。这是他在Spring 4中有关检测Websocket连接和断开连接的最后一篇文章的很好的后续文章。 #boot
  14. 我们网站背后的团队已经安装了Disqus，以便在我们的博客上发表评论 。我希望可以向您显示这些电子邮件，但是从“想法”到“评论”的转换是实时的！“在这种情况下是出奇的快。敏捷，甚至。
  15. Adam Shook在SpringOne2GX 2013上发表了SpringOne2GX 2013重播：Hadoop-大数据新手的基础知识，现在可以在线查看其重播。不要错过！
  16. 还可在线获得的是SpringOne2GX 2013的David Turanski和Luke Taylor的演讲，即Spring的实时分析 。看看这个！
  17. Spring LDAP主管Mattias Arthursson在去年SpringOne2GX 2013上介绍Spring LDAP 2.0的演讲重播现已上线
  18. 您是否错过了Spring主管Juergen Hoeller的网络研讨会，该研讨会在Java 8上介绍了Spring 4 ？它现已上市，绝对值得一看！
