Spring 的这个星期-2014年4月15日

工程
乔什·朗（Josh Long）
2014年4月15日

欢迎来到 Spring 本周的另一期！本周，我在法国巴黎与Devoxx FR的开发人员讨论了Spring Framework 4.0，Spring Boot等内容。下周将在印度班加罗尔举行的印度大开发者峰会。与往常一样，如果您在任何一个地点附近并想与Spring交谈，请在网上ping我 。

  1. 4月30日（星期二）网络研讨会-不要错过Ashley Puls（新遗物）和我本人，他们将通过现代Web应用程序进行跟踪和追踪： 使用Java和Javascript进行Web应用程序诊断
  2. 4月22日，星期二，网络研讨会-看看EclipseCon 2014主题演讲是怎么引起的：Martin Lippert和Mike Milinkovich演示了Project Flux：将Eclipse连接到Developer Tooling的基于云的时代的alpha版本。立即注册 ！
  3. Thomas Risberg和Janne Valkealahti在此SpringOne2GX 2013重放中展示了如何为Java MapReduce，Streaming，Hive，Pig，Cascading或HBase使用单一编程模型/配置模型，重播： Spring Data和Apache Hadoop入门
  4. 在SpringOne2GX 2013上与Jags Ramnarayan和Anthony Baker进行90分钟的交流： 基于Hadoop的内存数据和计算
  5. 查看Kevin Nilson（Google）和我本人在Spring Profiles上的SpringOne2GX 2013联合演示，以及如何通过常规dev / stage /生产生命周期管理代码的神奇工具： Multi Environment Spring Applications 。
  6. 我在SpringOne2GX 2013上与Roy Clarkson交流时，谈到了Spring Mobile / Android，REST / OAUTH等内容： 使用Spring构建智能客户端
  7. ErdemGünay撰写了一篇非常不错的文章， 介绍了他如何使用Spring Expression Language（SpEL）和Spring Boot动态地向许多Android客户端动态发送远程安装指令。
  8. 我们的好朋友Sam Brannen将在亚特兰大 Spring 用户组上发表演讲 ！聪明的人之一
  9. 莫里茨·舒尔茨（Moritz Schulze）撰写了非常非常详细的文章，介绍了他的公司如何开发时间跟踪，休假时间管理应用程序，称为Trackr with Java 8和Spring Boot。
  10. Mario Arias使用Kotlin语言汇总了Spring 关系数据入门指南的一个版本。Kotlin是JetBrains的用于JVM字节码和JavaScript的静态类型编程语言。太酷了！我以前从未见过kotlinprimavera模块，但是我将其挖掘出来！
  11. 微软企业传播总监Patrick Chanezon参加了我最近在Spring Boot上进行的vJUG见面会，并谈到了黑客！结果是有关如何将Spring Boot应用程序部署到Microsoft的Azure PaaS的一些指导。真好！谢谢，帕特里克！
  12. Hai Nguyen撰写了一篇非常不错的文章， 介绍如何配置/使用嵌入式Jetty的JNDI绑定DataSource
  13. 您是否错过了Spring Framework 负责人Juergen Hoeller在GOTO Amsterdam上关于Spring 4和Java 8的史诗般的演讲？不用担心， 您可以在线观看 ！
  14. 这是 Spring 的特别假期！Kim Saabye Pedersen 在RedHat的OpenShift PaaS上使用Spring Boot撰写了一篇不错的文章
  15. 我们的朋友Petri Kainulainen回来了，这一次是一篇关于在Spring上使用JOOQ的很好的文章 。Petri在社区中是一个了不起的人。我希望你不需要我告诉你。为他的站点添加书签。几乎总是值得的。
  16. 我们在Netflix的朋友TomásLin回来了！这次他有一篇关于用Spring Boot和Gradle捆绑Web内容的文章
  17. 迈克尔·西蒙斯（Michael Simons）发表了一篇非常详细的文章，介绍了他如何开发Boot Web应用程序 。谢谢您的详细撰写，迈克尔。非常有价值的反馈。
