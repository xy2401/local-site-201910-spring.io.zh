春天的这个星期-2014年7月22日

工程
乔什·朗（Josh Long）
2014年7月23日

欢迎来到春季本周的另一期！这周我们将在俄勒冈州波特兰市的OSCON上！这是一个疯狂忙碌的星期，只有数字更忙！与同事和社区成员一起访问真是太好了。博览会地板上的摊位已经挤满了用户。我喜欢这个节目。 :)

  1. 首先，大新闻： Spring Framework 4.1候选发布版现已发布 ！新版本具有更新的JSR-107支持，JMS 2.0和以注释为中心的JMS配置，SpEL编译器等等 。
  • Spring XD的共同负责人Mark Pollack博士刚刚宣布Spring XD RC1现已上市 。安装非常简单 。有许多新功能，这列火车的运行速度很快，因此请检查一下，下载钻头，踢轮胎等。
  • Spring Data忍者Christoph Strobl撰写了一篇不错的博客文章， 介绍Spring Data MongoDB的全文本搜索
  • A My My Code博客探讨了如何将Spring的集成测试与模拟服务一起使用。
  • Tousif Khan 在Spring的视图控制器上发表了一篇不错的文章
  • Tech Zoo博客写了如何使用Spring的Apache Tiles支持
  • 该软件 洞博客介绍了如何使用jqPlot，Spring的REST支持，AJAX和JQuery处理图表
  • Theodora Fragkouli写了一篇关于使用Spring MVC的不错的文章Errors标签
  • 我刚刚偶然发现了有关使用Spring Data Redis的 JavaCodeGeeks 学院路线 -看起来很有趣。
  • 是否想同时使用Spring，Hibernate 4和Ehache？查看本教程 。
