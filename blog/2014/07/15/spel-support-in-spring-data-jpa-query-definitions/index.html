Spring Data JPA @Query定义中的SpEL支持

工程
托马斯·达里蒙特
2014年7月15日

Spring Data JPA允许使用@Queryannotation手动定义要由存储库方法执行的查询。不幸的是，JPQL中的参数绑定非常有限，仅允许您设置值并提供某种类型转换。Evans发行列车的最新Spring Data JPA M1发行版通过添加支持减轻了这种痛苦
使用SpEL表达式在@Queryannotation中的语句内使用动态绑定的参数，这在手动定义查询时提供了更大的灵活性。在此博客文章中，我将向您介绍此功能的功能。

方法参数表达式

SpEL支持提供对查询方法参数的访问。这使您可以直接按原样绑定参数，也可以在绑定之前执行其他操作。

@Query("select u from User u where u.age = ?#{[0]}")
List findUsersByAge(int age);

@Query("select u from User u where u.firstname = :#{#customer.firstname}")
List findUsersByCustomersFirstname(@Param("customer") Customer customer);
