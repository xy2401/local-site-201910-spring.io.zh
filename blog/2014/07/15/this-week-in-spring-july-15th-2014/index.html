春天的这个星期-2014年7月15日

工程
乔什·朗（Josh Long）
2014年7月15日

欢迎来到春季本周的另一期！这是疯狂的一周，下周的数字甚至更疯狂！Spring团队中的一些人-包括Spring Boot的联席负责人Phill Webb ，Spring Batch的负责人Michael Minella和我本人，下周将在OSCON参加，因此，如果您在该地区，并且在展会上来打个招呼！我们也将在各个小时到达展位。

  1. Spring Batch主管Michael Minella 刚刚宣布了Spring Batch 3.0.1 ，以及一个新的Spring Batch扩展存储库 ，该存储库本身包含与Spring Batch的新集成。第一个贡献是ItemReader和ItemWriter用于Elasticsearch 。真好！
  2. Spring Data忍者Thomas Darimont刚刚宣布在Spring Data JPA中支持SpEL @Query注解
  3. Spring和Groovy Tool Suite的负责人Martin Lippert刚刚宣布了基于Eclipse Luna 4.4的Spring Tool Suite 3.6.0和Groovy Tool Suite 3.6.0 ，并支持Groovy 2.3，Grails 2.4，tc Server 2.9.6和其他一些小的改进。和错误修正。
  4. Spring Data忍者Christoph Strobl刚刚发布了Spring Data Evans m1，其中包括许多新修订 。
  5. 说到Spring Data Evans的第一个里程碑，Spring Data REST忍者Greg Turnquist刚刚宣布Spring Data Evans M1里程碑附带了ALPS元数据支持 。FTA： ALPS提供了一种查询RESTful服务的方法，以了解其服务的数据。让我们看一个简单的例子。如果克隆TODO存储库并运行mvn spring-boot：run，则可以非常轻松地浏览其值 。所以..检查出来！我知道许多人一直在询问类似的支持。
  6. 查看即将举行的Michael Plod网络研讨会， 他为什么推荐Spring-了解技术（以及一些业务）案例！迈克尔是一位出色的演讲者，经常参加会议。立即注册参加7月22日的活动。
  7. 7月29日，来自Crafter软件的Russ Danner将向我们介绍如何使用Spring和Groovy征服支持内容的Web和移动应用程序 。
  8. 安德烈亚斯艾泽勒已经把一个非常好的介绍了机器的后面使用@Transactional在Spring托管的单元测试中 ，并分析常见方案，代码等。Vert很酷的讨论，还有一个周到的帖子。
  9. Cegega博客非常酷地介绍了如何使用Spring Batch的异常处理RetryTemplate 。
  10. 这个主题在我本周几乎和我所听到的一样亲切，所以我只想分享这个古老而又美味的东西 ： 如何在Spring的JTA支持下使用Atomikos
  11. 您当然是在查看Groovy语言的领导者Guillaume LaForge的This Week in Groovy ，不是吗？ :-)
