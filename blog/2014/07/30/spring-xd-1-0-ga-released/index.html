Spring XD 1.0 GA发布

发布
马克·波拉克
2014年7月30日

我非常荣幸地代表Spring XD团队宣布Spring XD 1.0的全面上市！您可以下载zip发行版 。您也可以使用自制软件在OSX上安装，以及使用yum安装在RHEL / CentOs上。

Spring XD的目标是成为您开发和部署大数据应用程序的一站式服务。这种应用程序需要广泛的技术来解决不同的用例，同时又需要相互协作。此过程中的步骤包括：

  • 数据采集
  • 实时流和分析
  • 数据清理
  • 批处理（Hadoop上和下）
  • 机器学习和探索性数据分析
  • 可视化和报告
  • 实时和批处理之间的闭环分析

Spring XD将许多步骤整合到一个统一的运行时平台中，因此您可以解决尽可能多的用例。您不必将大量的单个项目组合在一起，每个项目都有自己的配置和编程模型。相反，使用Spring XD，您可以快速开始使用简单但功能强大的DSL开发端到端解决方案并进行扩展。

Spring XD提供：

  • 分布式，容错和可扩展的运行时
  • 流和批处理支持的统一平台
    • 用于流处理的流行资源 ， 处理器和接收器
    • 使用NoSQL Analytics进行流处理
    • Hadoop Batch工作流程编排
    • 离Hadoop批处理作业
    • 通过JPMML从批处理到实时评分的闭环机器学习
  • 运行时可移植性：易于在VM /金属上群集 ， YARN或EC2上进行设置，操作和部署
  • Hadoop发行版的可移植性：针对六个Hadoop发行版进行了认证
  • 它基于成熟的Spring基础项目（例如Spring Integration，Spring Batch和Spring Data）构建的企业就绪状态
  • 便于使用
    • DSL使用许多预构建的组件创建流和作业
    • 带有选项卡补全的Interactive Shell
    • 管理员界面和REST API
    • 易于开发和测试自定义模块以及延长运行时间 。
    • 自定义模块可以轻松合并其他Spring项目，例如Spring Data中的项目，例如用于创建仪表板的Spring Data REST

有几种资源可帮助您开始使用Spring XD。

  • InfoQ文章
  • Spring XD聚会记录
  • 入门指南
  • 项目页面
  • 参考文件
