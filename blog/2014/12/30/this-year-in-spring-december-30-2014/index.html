今年 Spring -2014年12月30日

工程
乔什·朗（Josh Long）
2014年12月30日

欢迎来到 Spring 本周的另一期！本周，按照我们的传统，我们将回顾一些让2014年成为Spring开发人员的美好一年的重要事情。而且，那是一年！

  • 去年年底看到了Spring 4 GA，而今年看到的是Spring 4.1发布，距离4.0之后仅短短9个月！Spring 4.x拥有一系列令人眼花new乱的新功能！Java 8支持，Java EE 7支持，新的Spring MVC视图，JSR 107支持，强大的资源管道和@Conditional注释，这当然使之成为可能...
  • Spring Boot的第一个1.0 GA发布于今年4月，此后经历了两次重大更新（ 1.1和1.2 ）。Spring Boot席卷了整个社区，引发了我们团队以及更重要的是其他所有人的各种活动！比赛试图淡化或复制它。现有的Spring用户蜂拥而至（包括Netflix和Ticketmaster之类）。但是，如果您遵循此每周综述，那么您就不需要进一步的证据来证明社区的普及！观看人们关于使用Spring Boot来做事的博客真是太有趣了……似乎一切！
  • Spring XD 1.0今年早些时候和后续发布， Spring XD 1.1正在迅速接近尾声。Spring XD（也是基于Spring Boot，BTW构建的！）通过在Spring Integration ， Spring Data和Spring Batch的基础上提供了一个数据处理工作台。它光滑，快速，优雅，当然人们很快就将其包含在军械库中 。
  • Spring Cloud于今年在德克萨斯州达拉斯举行的SpringOne2GX 2014上宣布。 Spring Cloud支持构建基于云的微服务风格的应用程序，并基于Spring Boot公开的编程模型构建。它集成了流行的库和基础架构，例如许多Netflix OSS堆栈和Consul 。
  • Cloud Foundry也度过了一年的欢乐时光！去年年底，我们开始看到Cloud Foundry v2的第一个细流，而今年我们看到了它的成熟，并且还看到了Cloud Foundry v2.1！下一代Cloud Foundry的令人兴奋的工作正在进行中。这是对快速运行的Go中所有组件的自下而上的重写，提供了可独立于较高的PaaS-y位使用的运行时（“格”）。当然，Cloud Foundry是Spring Boot和基于Spring Cloud的应用程序的理想目标。Pivotal Cloud Foundry是Cloud Foundry的紧缩包装，其发展迅猛。即将推出：磁贴以及对Spring XD和Spring Cloud的支持！而且，当然，我们在结束本节之前必须提到所有最令人振奋的事情： Cloud Foundry Foundation现在已全面运行 ！

太好了！我和您一样渴望看到我们在2015年有可能超越所有这些！现在，不用多说，让我们快速浏览一下本周的一些链接。

  • 就在新的一年的时候，令人惊叹且独特的Spring项目负责人Juergen Hoeller 宣布了Spring 4.1.4、4.0.9 和3.2.13。
  • 我们的朋友尼古拉斯·弗兰克尔（Nicolas Frankel） 在Spring对Optional依赖关系 -InfoQ拥有Rossen Stoyanchev关于Spring 4 Web应用程序的SpringOne2GX 2014演讲，现已发布
  • 总部位于马德里的Spring用户大会Spring I / O宣布了明年的展会。我个人错过了去年的演出，很高兴能回来！
  • Kielczewski.eu博客上有一篇非常详尽的文章，介绍如何使用Spring Boot构建安全的Web应用程序 。不错的工作！
  • David Kiss在使用Spring Boot（和Spring Social）处理Twitter feed上有一篇不错的文章。

那很有趣！希望您喜欢沿着记忆小路走下去。每年的这个时候总是很好。我也为 Spring 本周的下一期而感到兴奋！这将标志着（大约）四年不间断的分期付款！太着急了！

我敢肯定，我代表发言的所有Spring团队的时候我祝你一个非常， 非常 快乐，新年快乐 ！2015年的C'ya！
