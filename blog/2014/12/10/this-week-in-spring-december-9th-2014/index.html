春天的这个星期-2014年12月9日

工程
乔什·朗（Josh Long）
2014年12月10日

欢迎来到春季本周的另一期！在短短的几周内，我们将在2015年！那会令人兴奋吗？当然。但是，本周也同样令人兴奋！本周好东西太多了。

  • Spring Boot 1.2.0在这里 ！Java EE的许多新功能（Servlet 3.1基线，JTA，JMS和JNDI支持）以及改进的Jackson输出，Actuator端点，CLI更新和性能。Spring Cloud已经在跟踪1.2，并将很快以此为基础。得到的位 。
  • Spring Boot 1.1.10现在提供了许多修复程序 。
  • 疯狂的驻地科学家Andy Wilkinson刚刚宣布了Spring IO Platform库的最新版本 。有很多注意事项，但是特别是它增加了对Spring Integration Flow 1.0，Spring Integration Java DSL 1.0和Spring Integration Splunk 1.1的支持，并更新了一些项目的修订版。
  • Spring Security负责人Rob Winch发布了Spring Security 4.0.0。RC1 。阅读他的发行说明，并尽快尝试！
  • Spring Data忍者Thomas Risbery刚刚宣布了适用于Apache Hadoop 2.0.4的Spring 。
  • Spring Integration的负责人Gary Russell刚刚宣布了Spring Integration和Spring AMQP维护版本 。
  • 1月27日加入Gary Russell，他展示了Spring Integration引领ESB / Integration Framework市场的新原因（2014年DZone研究）： Java DSL for Spring Integration简介 。
  • 内存是您的事吗？加入Luke Shannon 1月13日，在他的演讲“ 使用Spring和GemFire构建可扩展的数据应用程序”中，了解Spring MVC / Spring Boot应用程序如何利用GemFire保持持久性。
  • 观看Stepane Maldini的网络研讨会重播，该网络研讨会介绍了用于异步，非阻塞微服务的Reactor 。
  • Matt Raible宣布了对其颇受欢迎的AppFuse入门项目的更新。主要变化之一： 它现在使用Spring IO Platform依赖项来简化版本和依赖项管理 。
  • 给Spring Data的帽子提示Oliver Gierke给我发了这篇有用的文章， MongoDB JVM库的简单用例比较。

  • Dennis撰写了有关Spring Boot和Logstash，Kibana和ElasticSearch的文章 。

  • CodeCentric 已发布其Spring Batch和基于Spring Boot的启动程序的1.3.0版本 。现在，它包括交易安全指标。
  • 说到Spring Boot启动器，您是否看到Tobias在上个月撰写的有关编写自己的Spring Boot启动器的帖子 ？
