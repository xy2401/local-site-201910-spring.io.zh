Spring 的这个星期-2014年1月7日

工程
乔什·朗（Josh Long）
2014年1月7日

欢迎回到春季本周的另一期！Spring团队的工作开始大量增加。本周9月9日，我们非常期待的Spring 4发布网络研讨会注册非常疯狂！我会在那里，看着并帮助MC。我希望你也会在那里！

另外，如果您在湾区，

我将在1月22日在奥克兰水罐公园（Oakland JUG）上发言，一个晚上的时间是几个小时，我们称您最近是否见过 Spring ？ 。我们将看看去年的史诗般的超赞价值，其中包括Spring 4，Spring Boot和Spring XD的发布。希望您也能加入我们！提出问题！

  1. 2014年1月16日，我们的Spring Security负责人Rob Winch将介绍Spring Security 3.2版本，并讨论其对Java配置，CSRF保护，与安全性相关的HTTP响应标头，可选的Spring MVC集成以及Spring Framework 4.0的支持。
  2. Patrick Grimard 在将Spring Security 3.2.0的CSRF保护与Backbone （或者实际上是任何面向客户端的应用程序）结合使用方面发表了一篇不错的文章 。正如Spring Security Rob Winch指出的那样， 还有一种更简单的方法可以实现这一目标 。
  3. 2月是安全月！我们刚刚与David Syer博士一起发布了SpringOne2GX 2013重播：使用OAuth2进行数据建模和身份管理 。
  4. 在REST方面，还刚刚发布了另一个SpringOne2GX 2013重播：Spring RESTBucks-由Oliver Gierke提供的超媒体驱动的REST Web服务 。
  5. 罗伊·克拉克森（Roy Clarkson） 撰写了一篇有关如何使用WebJars的精彩文章，该文章使您可以使用传统的基于JVM的构建管理工具（例如Gradle和Maven）以及Spring Boot来管理客户端依赖性（如JavaScript）。
  6. 这篇关于日志记录的应用程序检测工具过时了一些，但我认为值得一提，因为它通常很有见地，并且不仅在规范的Spring Pet Clinic应用程序方面，而且在Node.js Node Cellar方面都展示了其概念。以及.NET音乐商店。不错！
  7. 我们的朋友Petri Kainulainen回来了！他写了一篇不错的文章，介绍如何在Spring中使用JOOQ类型安全查询API
  8. 这个帖子有这么很少与Spring的事，但它确实有做的GemFire XD（我们在内存中，分布式数据存储在内存中或与HDFS可以工作）和位聪明的Python的修修补补访问的GemFire来自Python的XD 。
  9. 您是否看到过在Raspberry Pi上运行由Spring Boot驱动的Web服务的史诗性帖子？
  10. Ned Lowe撰写了一篇不错的文章，内容涉及从Spring MVC 2.0样式的MVC应用程序迁移到Spring MVC 2.5之后可用的以注释为中心的方法 。
  11. Thys Michels 在JUnit测试Spring MVC服务方面撰写了一篇不错的文章 。
  12. 还要感谢Learning Spring博客的友好提醒， 如果您只需修改正确的配置文件 ，Spring应用程序就会暴露出许多有价值的日志供您利用 。
  13. Tomas Zezula 在Spring上发表了一篇不错的文章@Primary注解 ，它使您可以从许多可能的依赖项中消除对要使用的依赖项的选择。
