Spring 的这个星期-2014年1月28日

工程
乔什·朗（Josh Long）
2014年1月28日

欢迎来到春季本周的另一期！

快速提醒：我将与一位来自Vaadin的人共同举办一个网络研讨会， 介绍如何使用Spring和Vaadin （丰富的Internet应用程序框架）。我们将介绍一些常见的集成选项，还有一些不太常见的集成选项！请继续关注，我希望在那里见到你！

另外，我将在接下来的几个月中充实我的演讲议程。到目前为止，我知道我将在4月22日至25日在印度班加罗尔举行的“ 伟大的印度开发者峰会”上 ，并从5月14日至5月16日在波兰克拉科夫的Geecon举行 。如果您在那些地区中的任何一个，我很乐意见到您。如果您主持JUG或UG，或者拥有庞大的开发团队和受众，那么我也非常希望有机会与您和您的团队进行交谈。ping 我 。

和往常一样，我们还有很多事情要做，让我们开始吧！

  1. Spring领导者Juergen Hoeller今天宣布了一个双重功能发布： Spring 4.0.1和Spring 3.2.7均已发布 ！看看这个。
  2. Phil Webb刚刚宣布了Spring Boot 1.0.0的第一个候选版本 ！
  3. 加入Apache Committer Stuart Williams和Daniel Mikusa， 了解Apache Tomcat 8简介 -2014年2月8日。立即预订座位，因为这很可能是另一个热门话题。
  4. 2月18日的网络研讨会！当他们告诉我们有关Spring数据存储库的最佳实践时，请不要错过Oliver Gierke和Thomas Darimont。
  5. 您是否错过了Spring Leader Juergen Hoeller的史诗般的网络研讨会，介绍了Spring 4 （他做了两次，一次是针对EMEA，另一次是NorAm时区）？您是否还错过了一周后他举行的两次安可演出，因为第一次网络研讨会使网络会议不堪重负，导致成千上万的参与者无法访问它？那就别担心， 您现在就可以在线观看 ！
  6. 沿着这些思路 ， Greg Turnquist刚刚宣布， spring.io上的所有指南已升级到最近发布的Spring Boot 1.0.0。RC1 ！
  7. 真好！Spring MVC忍者Rossen Stoyanchev刚刚宣布了Spring Web Flow 2.3.3的更新版本 。它支持Spring 4和JSF 2.2。
  8. 在All and Sundry博客上，Biju Kunjummen撰写了一篇不错的文章，介绍了如何使用Spring的``-一种超类型令牌的实现 -正确地使用通用类型来通过网络来封送REST API响应由于擦除，在运行时会丢失其通用组件。他用从Spring REST协议栈，这是我开发的示例代码的谈话有很多的同系 Spring 队员罗布绞盘，罗伊·克拉克森，克雷格·沃尔斯，罗森Stoyanchev，罗保茨马等帮助
  9. 去年11月，我做了一个演讲， “您最近看过 Spring 吗？” 。我在将演讲从Keynote '09转换为Keynote 7时遇到了一些麻烦，最终设法将所有内容都消除了。我刚刚上传了完整的讨论区（其中包含一些额外的内容，这些内容在一些演讲中由于时间不够而没有包括在内）。我在星期日上传了卡片组。您也可以在此处找到该视频的链接。请享用！
  10. 本专栏的读者知道我爱我一些良好的软件包管理，并且OS X的Homebrew 还不错。您是否知道可以使用Homebrew安装Pivotal GemFire和tcServer ？安装完pivotal点按（ brew tap pivotal/tap ），只需发出brew install tcserver gemfire 。那样简单！
  11. Kailash Kutti撰写了有关Spring for Apache Hadoop的演讲的平台。
  12. Blogger Sudhir Dharmadhikari似乎真的很喜欢Spring Data，甚至为该项目提出了一个非常讨人喜欢的重命名。要了解原因， 请查看他的博客 。
  13. Spring Data忍者Thomas Darimont刚刚宣布Spring Data Redis 1.1.1现已可用 。
  14. 格雷格·特恩奎斯特 （ Greg Turnquist ）撰写了一篇非常不错的博客文章，解释了您如何也可以为《 入门指南》 做出贡献 。
  15. 同时，在Pivotal博客上，有一篇不错的文章介绍了如何在（基于SQL的）数据分析中使用窗口函数 。好吧，好吧，我知道它与Spring无关，但是它是如此有用。此外，构建更智能，以数据为中心的应用程序将帮助您构建更智能的Spring应用程序！
  16. 加拿大Pivotal Labs工程部负责人Farhan Thawar 在关于移动应用程序开发的前5个神话中发表了一篇不错的文章 ，这与我们在Pivotal的Spring团队讨论移动技术的观点相吻合。进行检查，然后查看我们的入门指南，以了解Spring如何帮助您改善移动产品！
  17. JHipster 0.8.0现已发布 ！新版本大量建立在Spring Boot的基础上，并且包含了很多功能！看看这个！
  18. 嘿，顺便说一下，您是否尝试过“ 入门指南 ”？他们简直太简单了！我的朋友和同事彼得·汉弗莱（Pieter Humphrey）坐下并叙述了他对一些指南的了解。看看这个！这两个视频演示了有用的工作代码，每个过程大约需要6分钟 ！
  19. 顺便说一句，我想确保人们从去年开始在GoPivotal博客上看到了一些以数据和Spring为中心的出色帖子。这里是_ 通过Spring和NoSQL的扩展为RDBMS增添年头，另一篇关于如何使用GemFire一次运行多个大数据应用程序，另一篇有关如何将Spring应用程序迁移到实时数据网格 。
  20. 说到Gemfire，最近有一篇非常不错的文章，介绍了中国铁路系统如何在1月31日农历新年 （或“春节”） 之前利用GemFire增强容量 。如果您是从中国读到的，祝您新年快乐-马年快乐！您可能还可以查看我去年春季在China Scale上发表的这篇文章。
  21. 查看有关Thymeleaf如何支持各种模板样式的权威指南。模板和设计可能是网站实用程序的一个非常重要的组成部分，因为本文- 为什么简单的WebSite具有科学上的优势 -解释了！
  22. Spring Data负责人Oliver Gierke在Stack Overflow上发表了很多很好的文章。第一部分说明了如何正确使用PagedResourceAssembler使用Spring Data ，第二个回答了这个问题， 您是否应该在JPA中每个表有一个存储库？
  23. 从哲学上和技术上都有很好的深入研究如何使用RabbitMQ防止无限队列 。
