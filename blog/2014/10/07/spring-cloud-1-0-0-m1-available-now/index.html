Spring Cloud 1.0.0。M1现在可用

发布
戴夫·瑟
2014年10月7日

Spring Cloud （9月宣布的新伞项目）已经达到了一个里程碑，这是第一个里程碑， repo.spring.io存储库中提供了新鲜的jar。Spring Cloud将遵循“发布训练”模型进行发布，有点类似于Spring Data，但是我们还没有一个很酷的名字，所以它只有1.0.0。M1。此发行版中的模块是

  • Spring Cloud Config：由git存储库支持的集中式外部配置管理。配置资源直接映射到Spring Environment但如果需要，可以由非Spring应用程序使用。

  • Spring Cloud Netflix：与各种Netflix OSS组件（Eureka，Hystrix，Zuul，Archaius等）集成。

  • Spring Cloud Bus：一种事件总线，用于将服务和服务实例与分布式消息传递链接在一起。对于在群集中传播状态更改（例如配置更改事件）很有用。

  • Spring Cloud Security：一组用于以最少的麻烦构建安全的应用程序和服务的原语。

  • Spring Cloud CLI：Spring Boot CLI插件，用于在Groovy中快速创建Spring Cloud组件应用程序。

  • Spring Cloud Starters：Spring Boot风格的启动程序项目，可以简化对Spring Cloud使用者的依赖关系管理。

以上所有均具有1.0.0。M1版本标记，因此尝试使用它的一个很好的起点是安装CLI并使用以下代码运行Config Server：

@EnableConfigServer
class ConfigServer {
}

然后打http://localhost:8080/foo/default看一些财产来源。

该代码托管在github上，非常欢迎社区贡献，因此请继续阅读并查看。在单独的组织中有一些简洁的示例： spring-cloud-samples ，包括一个具有git子模块和脚本的“脚本”项目，以使完整的演示系统尽快启动并运行。
