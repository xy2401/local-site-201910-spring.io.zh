春天的这个星期-2014年10月7日

工程
乔什·朗（Josh Long）
2014年10月8日

欢迎来到春季本周的另一期！本周，我参加了Couchbase Connect会议，与开发人员讨论Spring Data Couchbase（以及Spring Boot，Spring XD，Spring Session等）。有趣的表演，有趣的人群！下周，我将在JAX London和Geekout UK ，因此，如果您在附近，请务必打个招呼。

现在，让我们开始综述。

  1. 优秀的Pollack博士刚刚宣布发布Spring XD 1.0.1 ，这是构建数据集成和提取解决方案的最简单方法。它可能只是一个1.0.1版本，但请不要误会：此版本中有一些非常酷的功能，包括改进的管理和管理UI功能。
  • 联合安全性是任何分布式，多客户端，（微）服务导向系统的重要组成部分。Spring Security和Spring Security OAuth特别提供了一种引人注目的解决方案。参加即将举行的有关使用OAuth2进行微服务安全性的在线讲座的优秀Dave Syer博士
  • Matt Stine将举行网络研讨会，介绍Pivotal Cloud Foundry和Spring Cloud如何为构建微服务提供自然解决方案 。不要错过这个！
  • Spring和JavaScript忍者Brian Cavalier写了一篇非常有趣的文章，介绍了我们正在做的一些工作，以使JavaScript客户端和服务之间的差异同步更加容易 。这篇文章不仅介绍了所有这些令人兴奋的新内容，而且基于RaveJS（Spring Boot启发的JavaScript方法）构建。
  • 希望您能和Vaadin团队一起参加，我将于10月23日来看看如何构建由Spring Boot驱动的Vaadin应用程序 。
  • 贾斯汀·李·格兰特（Justin Lee Grant）对Spring Boot进行了很好的介绍 。
  • Spring负责人Juergen Hoeller刚刚宣布了Spring框架的4.1.1版本 。此版本包括改进Optional处理，杰克逊ObjectMapper生成器，一些要求JDBC改进的东西等等。
  • 好人Syer博士刚刚宣布了Spring Cloud项目的第一个里程碑版本 。Spring Cloud提供了用于构建微服务和以PaaS为中心的应用程序的基元。它建立在Netflix OSS堆栈和Spring Boot之上。
  • 卡尔西克·艾布拉姆（Karthik Abram） 在春季的一个隐藏的宝石上发表了一篇不错的文章： ClassPathScanningCandidateComponentProvider类，这使得发现某种类型的bean很容易。
  • Nicolas Lejeune撰写了一篇有趣的文章，讲述了他如何设法转换旧版应用程序（它使用EJB3）并将其移至Cloud Foundry环境中
  • 我们的朋友Toshiaki Maki汇集了他理想的Spring Boot Maven原型 ，并与大家分享 ！
  • 您是Groovy粉丝吗？（谁不是？）查看Groovy项目负责人Guillaume LaForge的每周综述，即Groovy的本周 ！
  • 您是否要调试Cloud Foundry上部署的应用程序 ？Daniel Mikusa撰写了一篇非常不错的文章，解释了一种方法。
