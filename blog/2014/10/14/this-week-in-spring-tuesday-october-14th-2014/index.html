春天的这个星期-2014年10月14日，星期二

工程
乔什·朗（Josh Long）
2014年10月14日

欢迎来到春季本周的另一期！本周，我去了硅谷代码营，JAX伦敦和Geekout英国，与开发人员讨论如何在Cloud Foundry上构建可扩展的，以微服务为中心的Spring Boot应用程序。与热情的开发人员（在大西洋两岸！）交谈非常有趣。

事不宜迟，让我们继续综述！

  • Spring Boot的共同负责人Phil Webb刚刚宣布了Spring Boot 1.2.0的可用性。M2 ，除其他外，包括对Jersey的自动配置，log4j， commons-dbcp支持，更智能的磁盘使用状况指示器和改进的RabbitMQ支持
  • Spring和Groovy Tool Suite的负责人Martin Lippert 刚刚宣布了最新和最出色的版本3.6.2 ，这是一个维护版本，其中包括对最新Eclipse Luna SR1安装的更新。
  • 此外，Spring Boot的共同负责人Phil Webb刚刚宣布了Spring Boot 1.1x系列的1.1.8版本，其中包括针对白色标签错误页面跨站点脚本攻击的修复程序。
  • 联合安全性是任何分布式，多客户端，（微）服务导向系统的重要组成部分。Spring Security和Spring Security OAuth特别提供了一种引人注目的解决方案。参加即将举行的有关使用OAuth2进行微服务安全性的在线讲座的优秀Dave Syer博士
  • Matt Stine将举行网络研讨会，介绍Pivotal Cloud Foundry和Spring Cloud如何为构建微服务提供自然解决方案 。不要错过这个！
  • 美国乔治亚州亚特兰大，10月21日，星期二，在Cox Communications的为期一天的路演活动中了解Spring，微服务和Cloud Foundry。
  • 10月23日，星期四，德国慕尼黑，GA，在EMC位于德国伊斯曼宁的办公室举行的为期一天的路演活动中了解Spring，微服务和Cloud Foundry。
  • 希望您能和Vaadin团队一起参加，我将于10月23日来看看如何构建由Spring Boot驱动的Vaadin应用程序 。
  • Java et Moi博客上有一篇关于使用Spring MVC的@SessionAttributeannotation的不错的文章（法语）
  • 我真的很喜欢对Spring Boot应用程序中的配置属性值 （如--D参数，环境变量，属性文件中的值以及JNDI内的值）进行深入研究。
  • 苏德赫Dharmadhikari做得非常 漂亮的工作介绍yeoman基于JHipster代码生成器 ，它建立在顶部春天开机 。
  • 我们的好朋友Miroslav Kopecky回来了，这次有一篇不错的文章，介绍了如何使用Java配置 （通过Scala！） 使用Spring MVC（在Spring 4中） 。
  • Redis Cluster不再是软件 ！查看详细信息。
  • Biju Kunjummen撰写了一篇非常不错的文章，介绍了两种Java配置样式 ：一种是将依赖项声明为@Bean bean定义方法的参数， @Bean一种是通过调用方法简单地引用另一个bean。
  • 查看这篇文章，以避免在Spring应用程序中避免Log4j2和SLF4J之间的依赖关系冲突 。
  • 查阅Daniel Murygin的这篇文章，其中详细介绍了如何构建相当平凡的REST服务（甚至可以处理文件上传！）。和客户端应用程序使用Spring Boot ？
