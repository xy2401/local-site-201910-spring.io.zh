Spring Boot 1.1.0。M1现已上市

发布
菲尔·韦伯
2014年5月14日

Spring 里程碑存储库中现已提供Spring Boot 1.1.0的第一个里程碑版本。

重点包括：
*额外的模板支持。
*改进的指标和健康指标。
*其他数据集成（包括改进的MongoDB支持和对GemFire的支持）。
*许多小的改进和补充。
*更新了许多第三方库的依赖关系。

有关新的和值得注意的功能的完整列表，请参见发行说明 。

我们非常感谢现有Spring Boot 1.0用户的任何反馈。如果要升级，请确保遵循发行说明中的说明 。

Spring Boot 1.1将于5月底发布。
