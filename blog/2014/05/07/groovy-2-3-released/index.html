Groovy 2.3发布

发布
纪尧姆·拉福格（Guillaume Laforge）
2014年5月7日

Groovy开发团队很高兴宣布Groovy 2.3.0的发布 ！

Groovy 2.3是用于JVM的Groovy编程语言的新主要版本，具有以下特点：

  • 在JDK 8上运行Groovy的官方支持
  • 一个新的trait关键字，以定义用于构成行为的新代码单元
  • 新的和改进的编译时代码转换，例如：
    • @TailRecursive ：用于尾部递归的转换方法，以避免炸毁堆栈，
    • @Builder ：轻松实现由Groovy编译器本身生成的流利的生成器
    • @Sortable ：使用类的各种属性来转换类以实现Comparable
  • 具有Path支持的新NIO2模块
  • 如最近有关InfoQ的文章所述， 减轻了快速的JSON解析和构建的麻烦
  • 闭包参数类型推断 ，使用静态类型检查器和静态编译器来推断闭包中的参数类型
  • 标记模板引擎
  • Groovysh和GroovyConsole易用性改进
  • 一个新的GroovyAssert测试实用程序
  • 更多@BaseScript类功能，以及更多。

如果您想了解更多有关Groovy 2.3的新功能的信息，请阅读详细的发行说明 ，如果您想定期获得有关Groovy生态系统的最新消息，可以订阅Groovy Weekly时事通讯 ，每个星期二发布最新消息。 。

有关已解决问题的更多信息，请查看我们的变更日志 。

除了Bintray JCenter和Maven Central中的Groovy库之外 ，您还可以下载Groovy的二进制发行版 。
