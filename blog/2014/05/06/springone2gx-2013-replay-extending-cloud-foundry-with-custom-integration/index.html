SpringOne2GX 2013重放：通过自定义集成扩展Cloud Foundry

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年5月6日

在加州圣塔克拉拉的SpringOne2GX 2013上录制。

主讲人：Cornelia Davis，Scott Frederick

在开源代码库中可以找到它，Cloud Foundry包括一组预打包的服务（Postgres，MySQL，Redis，MongoDB和RabbitMQ）和许多应用程序运行时（Java，Ruby和Node.js）。此外，CloudFoundry.com通过服务网关与许多外部服务提供商集成。部署自己的Cloud Foundry时，可以通过添加其他服务和运行时支持来扩展现有的开源功能。实际上，您可以通过buildpacks将自己的运行时引入任何Cloud Foundry（包括CloudFoundry.com）。在本节中，我们将向您展示如何构建和部署或代理定制服务。我们还将向您介绍buildpack，向您展示如何创建自己的软件包以及如何使您的应用程序使用它们。

在以下网址了解有关使用CloudFoundry的更多信息：http://www.cloudfoundry.com/use
