Grails project lead Graeme Rocher just announced Grails 2.4! The new release is amazing for a slew of reasons, not the least of which is that some of its many amazing features are easy to use with Spring Boot, too. Double win! Congratulations, Graeme and team. (And also thank you, for another amazing release!)
  • Spring Boot 1.1.0.M2 is now available! The new release features improved support for Spring Data Solr, Spring Data Gemfire, and the entire Spring Data Dijkstra release train. The new release also offers GSP (Groovy Server Pages) and Velocity as templating options, along with upgrades to various libraries like Spring Security 3.2.4, and Spring Batch 3.0. This release train is moving quickly, so jump onboard while you can! There's a lot more great stuff, so check out the release notes.
  • June webinars are here! Ramnivas Laddad on launches Spring Cloud on June 3rd in Abstracting PaaS services to be portable with Spring Cloud, Michael Minella in Spring Batch 3.0.0 on June 10th, and Glenn Renfro in Spring Integration Done Boot-ifully on June 17th.
  • This blog introduces some of the limitations of Hibernate's inbuilt JDBC logging and then introduces log4jdbc as used in a simple Spring / Hibernate application.
  • Our pal Eugen Paraschiv has put together a very nice introduction to Spring Data JPA.
  • Matti Tahvonen over on the Vaadin team put together this fantastic introduction to using Spring Data Neo4j, Spring Boot, and of course the Vaadin4Spring library that Petter Holmström and I started. To be fair, it's only usable because of Petter :) So usable, in fact, that Matti was able to put together something beautiful - Bootiful - very quickly. Check it out! The application models (and visualizes!) data stored in Neo4j, so it's not just any old CRUD application, this is very cool!
  • CloudFoundry ninja James Bayer announced the new Pivotal CF 1.2, which now supports VMWare's Hybrid Cloud Service, initial auditing and autoscaling, new data services (Redis, MongoDB, Neo4j, RiakCS, and ElasticSearch are all available!), and a lot more!
  • Curiousity piqued? Want to learn more about CloudFoundry, the open-source Paas from Pivotal? Check out this epic video on how to setup your own Paas using BOSH from SpringOne2GX 2013. And hear about how other teams acutally did it in the SprignOne2GX replay - Free Yourself with CloudFoundry: A Private Cloud Experience.
  • Data ninja Thomas Risberg just announced the new Spring for Apache Hadoop RC4 release, which is awesome! The new release improves upon the Spring YARN integration. YARN, of course, is the distributed, generic runtime on top of which Hadoop 2.0's very specific map/reduce support now sits. You can use YARN for job distribution of your own, however. You might, for example, use YARN to split up Spring Batch workloads. In the new example, Janne Valkealahti demonstrates a simple Spring Boot-powered Spring YARN component. It's amazing how concise this stuff is now!
  • Want to learn more about Janne Valkealahti, the mad (data) scientist behind our Spring YARN support? Check out this Pivotal People profile!
  • Spring Data lead Oliver Gierke does a nice job introducing the new hotness in Spring Data Dijkstra, the new Spring Data umbrella release. This release includes Java 8 Optional support, asynchronous repository method invocations, and more.
  • Spring Security lead Rob Winch has just two small, bugfixe releases: Spring LDAP 2.0.2, and Spring Security 3.2.4
  • Spring Web Services lead Arjen Poutsma has just announced Spring Web Services 2.2.0, which now features a Java configuration API and much more.
  • Let me take a moment to remind everyone: Java configuration is everywhere! Spring framework, Boot, Data, Security, MVC, Integration, Batch, Social, and much more, all offer as-rich-as-the-XML Java configuration integrations. In the case of Boot, Java configuration is the only out-of-the-box option, though of course you can use XML if you'd like. It's just.. not expected.

    春天的这个星期-2014年5月27日

    工程
    乔什·朗（Josh Long）
    2014年5月28日

    欢迎来到春季本周的另一期！在美国，我们有一个为期3天的周末，这很好。观看一些精彩的技术讲座的好机会！如果您像我一样，那么您的技术讨论已经用光了，很高兴看到我们今天还有很多事情要做！

    1. Grails项目负责人Graeme Rocher刚刚宣布了Grails 2.4 ！新版本之所以令人惊叹，有很多原因，其中最重要的一点是它的许多令人惊叹的功能中的一些功能也易于在Spring Boot中使用。双赢！恭喜，格雷姆和团队。（还要感谢您，发布了另一个出色的版本！）
    2. Spring Boot 1.1.0。M2现在可用 ！新版本增强了对Spring Data Solr，Spring Data Gemfire和整个Spring Data Dijkstra发行版的支持。新版本还提供GSP（Groovy服务器页面）和Velocity作为模板选项，以及对各种库的升级，例如Spring Security 3.2.4和Spring Batch 3.0。该发行火车正在快速移动，因此请尽速上船！还有很多很棒的东西，所以请查看发行说明。
    3. 6月的网络研讨会在这里！Ramnivas Laddad将于6月3日启动Spring Cloud，以抽象化PaaS服务使其可移植到Spring Cloud中 ； Michael Minella在6月10日发布在Spring Batch 3.0.0中 ； Glenn Renfro在6月17日启动启动的Spring Integration中 。
    4. 该博客介绍了Hibernate内置JDBC日志记录的一些局限性，然后介绍了在简单的Spring / Hibernate应用程序中使用的log4jdbc 。
    5. 我们的朋友Eugen Paraschiv 对Spring Data JPA进行了很好的介绍 。
    6. Vaadin团队的Matti Tahvonen整理了有关使用Spring Data Neo4j，Spring Boot以及我和PetterHolmström所创建的Vaadin4Spring库的精彩介绍。公平地说，由于Petter的缘故，它仅可用：)实际上，如此有用， Matti能够很快将漂亮的东西-Bootiful-组合在一起 。看看这个！应用程序建模（并可视化！）数据存储在Neo4j中，所以它不仅是任何旧的CRUD应用程序，这非常酷！
    7. CloudFoundry忍者詹姆斯拜耳宣布的新枢纽CF 1.2 ，现在支持VMware的混合云服务，最初的审计和自动缩放，新的数据服务（Redis的，MongoDB的，Neo4j的，RiakCS和ElasticSearch都可用！），以及更多 ！
    8. 好奇吗？想更多地了解Pivotal的开源Paas CloudFoundry？观看这段史诗视频，了解如何使用SpringOne2GX 2013中的BOSH设置自己的 Paas 。并在SprignOne2GX重放中了解其他团队如何做到这一点- 通过CloudFoundry进行自己的自由：私有云体验 。
    9. 数据忍者Thomas Risberg刚刚宣布了适用于Apache Hadoop RC4的新版Spring，太棒了！新版本在Spring YARN集成的基础上进行了改进。当然，YARN是分布式的通用运行时，Hadoop 2.0现在非常特定的映射/归约支持位于该运行时之上。但是，您可以使用YARN进行自己的工作分配。例如，您可能使用YARN来拆分Spring Batch工作负载。在新示例中，Janne Valkealahti演示了一个简单的Spring Boot驱动的Spring YARN组件。现在这些东西多么简洁， 真是太神奇了！
    10. 想更多地了解Spring YARN支持背后的疯狂（数据）科学家Janne Valkealahti？查看此Pivotal People个人资料 ！
    11. Spring Data负责人Oliver Gierke很好地介绍了Spring Data Dijkstra （新的Spring Data伞版）中的新功能。此版本包括Java 8 Optional支持，异步存储库方法调用等。
    12. Spring Security主管Rob Winch仅发布了两个小的错误修正版本： Spring LDAP 2.0.2和Spring Security 3.2.4
    13. Spring Web Services负责人Arjen Poutsma 刚刚宣布了Spring Web Services 2.2.0，该版本现在具有Java配置API以及更多功能。
    14. 让我花点时间提醒大家：Java配置无处不在！Spring框架，启动，数据，安全性，MVC，集成，批处理，社交等等，都提供了XML丰富的Java配置集成。对于Boot来说，Java配置是唯一的即用型选项，尽管您当然可以根据需要使用XML。只是..没想到。
    15. Spring Security主管还写了关于使用Spring Security Test来处理Web安全的博客
    16. 想编写自己的Spring Boot入门程序吗？在Spring ninja Stephane Nicoll中查看此示例， 了解如何为Spring Boot提供HornetQ 启动器
    17. 最后， Spring Social Google 1.0.0。GA已发布 ！恭喜加布里埃尔，到达这里所需的所有工作。我也喜欢易于使用的示例。
    18. Arnaud Giuliani对在Spring Boot中使用GWT进行了非常酷的介绍 。做得好，阿诺！
    19. Netflix 工程师Tomas Lin在推特上发布了指向REST API的便捷Spring MVC异常处理程序的链接。这是解决问题的一种（精细）方法。另外，我建议您看看[使用Spring HATEOAS的VndError （ s ）支持] a）。
