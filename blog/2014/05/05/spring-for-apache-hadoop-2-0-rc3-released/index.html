用于Apache Hadoop 2.0 RC3的Spring发布

发布
托马斯·里斯伯格
2014年5月5日

我们很高兴地宣布，Apache Hadoop 2.0 RC3的Spring版本已经发布。

-

现在包括对最新Hadoop发行版的支持-Pivotal HD 2.0，Cloudera CDH5和Hortonworks HDP 2.1

除了版本升级以外，此版本中的更改还围绕对写入数据集SHDP-327的分区支持。

Spring for Apache Hadoop 2.0的主要重点是添加YARN应用程序开发支持，此外继续提供
HDFS和MapReduce支持方面的改进。您可以看到许多新的YARN功能正在使用中
纱线样品和
新的Spring IO指南 。

Spring for Apache Hadoop 2.0发行版的默认发行版是当前的Apache Hadoop 2稳定发行版2.2.0。

我们将继续在Spring IO里程碑存储库中提供特定于版本的工件及其各自的传递依赖项：
* 2.0.0。RC3（默认-Apache Hadoop stable 2.2.0）
* 2.0.0。RC3-hadoop12（Apache Hadoop稳定版1.2.1）
* 2.0.0。RC3-phd1（Pivotal HD 1.1）
* 2.0.0。RC3-phd20（Pivotal HD 2.0）
* 2.0.0。RC3-cdh4（Cloudera CDH4 MR1）
* 2.0.0。RC3-cdh5（Cloudera CDH5纱）
* 2.0.0。RC3-hdp13（Hortonworks HDP 1.3）
* 2.0.0。RC3-hdp20（Hortonworks HDP 2.0）
* 2.0.0。RC3-hdp21（Hortonworks HDP 2.1）

有关如何使用这些版本进行构建的说明，请参阅项目Wiki 。

有关特定项目的更多信息，请参见项目页面 。
