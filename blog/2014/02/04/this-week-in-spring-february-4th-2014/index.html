春天的这个星期-2014年2月4日

工程
乔什·朗（Josh Long）
2014年2月4日

欢迎回到春季本周的另一期！有很多要讨论的内容，所以让我们开始吧。

  1. Spring Integration 3.0.1维护版本现已发布，其中包含许多良好的错误修复，我们鼓励3.0.0用户立即升级。如果您还没有看到3.0.0的新功能，请查看Gary Russell的博客 ，其中包含许多新功能。
  2. Spring Data的“ Codd”发布系列的第一个候选发布版现已上市！
  3. 加入Spring Data项目的负责人Oliver Geirke和Thomas Darimont将于2月18日带领我们学习Spring Data Repositories-Best Practices 。
  4. 2月11日，不要错过Apache提交者Stuart Williams和Daniel Mikusa的Apache Tomcat 8网络研讨会简介
  5. 在其他Apache Tomcat新闻中， Apache Tomcat 6.0.39的维护版本现已发布。转至TomcatExpert博客以查看详细信息。
  6. Zoltan的博客上有一篇非常不错的文章，内容涉及如何将Spring Boot应用程序构建和部署到Heroku ！看看这个！
  7. Spring Batch主管Michael Minella 就Spring Batch和 SpringOne2GX 的Batch JSR，JSR 352作了精彩的演讲，现在可以重播！
  8. SpringOne2GX在本周的另一个重播中，来自E * Trade架构师Durai Arasan的精彩演讲，涉及在E * Trade上将Spring Integration与RabbitMQ结合使用的现实生活 。SpringOne2GX 2013最好的演讲之一！
  9. spring.io负责人和春季忍者Chris Beams在Pivotal上发表了一篇很棒的文章 ，说明开源对我们所有人的意义。检查出来，并传播这个词！
  10. 我们在Neo4J的朋友获得了有关TechCrunch的广泛报道，内容涉及Neo Technologies如何改变Graph数据库的世界！
  11. Blogger Brian 用WRO4J和Spring巧妙地讨论了一种处理常见Web开发任务的方法，例如资源优化（JavaScript转译，JavaScript和CSS 压缩等）。看看这个！
  12. Spring负责人Juergen Hoeller就如何从Spring 3.2迁移到Spring 4.0发表了一篇不错的文章。1） 。
  13. Cygnet Infotech博客整理了一个有关Grails的简洁图表，标题为Grails为什么是一个很棒的Java Web框架的6个原因 。我不知道该图表对“ Java” Web框架意味着什么，但是它肯定是一个史诗般的JVM Web框架，其余部分都是真实的！
  14. Programming Free博客上有一篇关于使用Spring MVC 4.0构建RESTful服务的文章。这篇文章在技术上是正确的，但可能更简单！（见下文）

  15. 弗雷德·乔治 （ Fred George ）是我去年很高兴与我们的共同朋友克里斯·理查森 （ Chris Richardson ）见面的人（2012年，我在奥雷迪夫（Oredev）讲话），他在2012年就建立微服务进行了精彩的演讲，您可以在这里观看。该视频与Spring没有任何关系。相反，它专注于构建松散耦合的，离散的， 微服务 ，这些服务加在一起描述了一个系统 。我为什么要提这个？因为Spring Boot使构建这样的服务变得非常简单！检查一下Spring Initializr ，选择所需的功能（也许是Web和JPA ），打开Application.java ，然后添加一个REST端点！ 这是一个由Spring Boot提供支持的可正常运行的REST服务的示例 。我唯一忽略的是Maven（或Gradle）构建文件。
