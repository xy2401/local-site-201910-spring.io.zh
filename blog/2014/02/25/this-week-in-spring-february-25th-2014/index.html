Spring 的这个星期-2014年2月25日

工程
乔什·朗（Josh Long）
2014年2月25日

欢迎回到 Spring 本周的另一期。

昨天，我在佐治亚州亚特兰大举行了令人惊叹的DevNexus会议 ，该会议与开发人员讨论了Spring广阔世界中的最新和最伟大的内容，以及关于使用Spring开发REST API的信息。今天，我在加拿大蒙特利尔参加ConFoo会议 。我本周将与开发人员谈谈Spring丰富的NoSQL和大数据支持以及如何使用Spring构建REST API。

和往常一样，我们有很多内容需要介绍，所以让我们开始吧！

  1. Spring Batch主管Michael Minella宣布Spring Batch 3.0 M3现已上市 。我最喜欢的新功能？它实现了JSR 352（批处理JSR）的所有必需组件！祝贺迈克尔和团队，这是一个巨大的里程碑！
  2. Spring Mobile的负责人Roy Clarkson 宣布Spring Mobile 1.1.1现已上市 。
  3. Spring Security负责人Rob Winch宣布Spring Security 3.2.1和3.1.5现在可用
  4. Rob 还宣布Spring LDAP 2.0.1现已可用 。
  5. Christoph Strobl 宣布Spring Data Redis 1.2.0现在也可用！
  6. 加入Juergen Hoeller和Spring团队， 在3月25日了解Java 8和Spring Framework 4.0 。
  7. 不要错过3月18日的 Mattias Arthursson ，他将展示Spring LDAP 2.0.0 。
  8. Tomcat爱好者应查看Apache Committer Stuart Williams和Daniel Mikusa的网络研讨会重播： Apache Tomcat 8简介 。
  9. 现在正在重播SpringOne2GX 2013的Spring Data Community Lightning Talks
  10. SpringOne2GX 2013演讲的重播， _SpringOne2GX 2013重播：使用Spring，Neo4J，Mongo和Redis研究云中的癌症_
  11. Spring Data负责人Oliver Gierke宣布Spring Data Codd现已上市，GA ！查看最新和最伟大的东西！
  12. Axon CQRS框架的最新版本2.1版现已上市 。
  13. Pivotal首席执行官Paul Maritz 在Cloud Foundry周围建立基金会方面写了一篇不错的文章
  14. El Reg 在Cloud Foundry基金会的最近成立上也有一篇不错的文章，标题为Cloud Foundry，引起了EMC，IBM，HP，Rackspace，SAP，VMware love-in的关注 。
  15. Zoltan博客上有一篇不错的文章，介绍了如何在Spring Security中使用两因素身份验证
  16. Java Code Geek博客上的Biju Kunjummen在使用Spring Boot和Spring Integration的第一步中撰写了一篇不错的文章。
  17. 泽维尔·帕德罗（XavierPadró'）已在上周介绍Spring和Thymeleaf的文章中发布了他的后续文章。新文章探讨了如何构建表单以及如何将Ajax与Thymeleaf一起使用 。
  18. 说到Spring Boot， OpenCredo博客在Spring Boot的第一步中也有一篇不错的文章！看看吧 。
  19. JavaLobby在使用自定义方面有一篇不错的文章ViewResolver与Spring MVC一起使用 。
