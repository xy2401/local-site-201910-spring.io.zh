Michael Nitschinger I’m pleased to announce the availability of the first release candidate of the Spring Data module for Couchbase.

The highlights of this release are (usual bugfixes and stability improvements aside) the support for custom repository queries backed by views, the possibility to customize view queries through the @View annotation and @Version support on entities for optimistic locking. Also - when XML configs are used - the beans can now be configured through SpEL expressions. This RC1 release is available from the Spring IO milestone repository. The complete changelog can be found here.

Spring Data Couchbase 1.0 RC1发布

发布
奥利弗·德罗博姆（Oliver Drotbohm）
2014年2月6日

我很荣幸地代表Michael Nitschinger宣布Couchbase的Spring Data模块的第一个候选发布。

此版本的亮点是（除了通常的错误修复和稳定性改进之外）对由视图支持的自定义存储库查询的支持，可以通过@Viewannotation自定义视图查询以及对实体进行@Version支持以实现乐观锁定的可能性。另外-使用XML配置时-现在可以通过SpEL表达式配置Bean。可从Spring IO里程碑存储库中获得此RC1版本。完整的变更日志可以在这里找到。

由于我们现在具有1.0 GA版本的完整功能，因此我们现在专注于性能，稳定性和文档编制。让我们知道发现的任何错误，以便我们可以在最终发行版之前对其进行修复。

如果您想全面了解Couchbase，尤其是Spring Data Couchbase模块，请参加2月27日Michael Nitschinger的网络研讨会 。

有关更多特定于项目的信息，请参见https://github.com/spring-projects/spring-data-couchbase上的项目页面。
