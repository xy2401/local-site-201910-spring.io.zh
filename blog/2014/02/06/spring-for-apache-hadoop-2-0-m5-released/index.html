用于Apache Hadoop 2.0 M5的Spring发布

发布
托马斯·里斯伯格
2014年2月6日

我们很高兴地宣布Apache Hadoop 2.0 M5里程碑发布的春季。我们正在接近发行候选版本，因此是时候重点介绍此2.0版本中的新功能以及与1.0版本相比的好时机。

Spring for Apache Hadoop 1.0的主要目标是将HDFS和MapReduce与MapReduce v1或MapReduce v2（YARN）结合使用。默认发行版是Apache Hadoop 1.2.1，并为其他发行版提供了其他“风味”：Hadoop 2.2.0，Pivotal HD 1.1，Cloudera CDH4 MR1或MR2 YARN和Hortonworks HDP 1.3。

Spring for Apache Hadoop 2.0的主要重点是添加YARN应用程序开发支持，同时继续提供对HDFS和MapReduce支持的改进。未来2.0版本的默认发行版是Apache Hadoop 2.2.0。

我们将继续在Spring IO里程碑存储库中提供特定于版本的工件及其各自的传递依赖项：
* 2.0.0。M5（默认-Apache Hadoop稳定版2.2.0）
* 2.0.0。M5-hadoop12（Apache Hadoop稳定版1.2.1）
* 2.0.0。M5-phd1（Pivotal HD 1.1）
* 2.0.0。M5-cdh4（Cloudera CDH4 MR1）
* 2.0.0。M5-cdh5（Cloudera CDH5 YARN beta）
* 2.0.0。M5-hdp20（Hortonworks HDP 2.0）

Spring在Apache Hadoop 2.0版本中最重要的增强功能：

  • 在新的spring-yarn子项目中提供了对YARN功能的支持。spring-yarn框架使您更轻松地开发基于Spring的应用程序，这些应用程序可以部署为使用YARN在Hadoop v2上运行。
  • 我们还将YARN支持与Spring Boot集成在一起，以便您可以在YARN上部署Spring Boot应用程序。我们将在博客文章中跟进，以更详细地说明此功能。
  • Spring Boot促进了基于注释的编程模型，并且我们为Hadoop YARN功能添加了基于注解的配置。

您可以看到YARN示例中使用了许多这些新的YARN功能

我们还添加了一个spring-data-hadoop-store子项目，以提供更好的支持，该方法使用DataWriter和DataReader实现将数据写入HDFS并支持文本文件和SequenceFiles等格式（带有或不带有压缩）。这个新的子项目还与Kite SDK的数据集支持集成在一起。

有关特定项目的更多信息，请参见项目页面 。
