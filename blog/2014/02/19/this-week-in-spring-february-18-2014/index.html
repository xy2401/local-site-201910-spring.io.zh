Spring Data Babbage SR3 has been released. The new release rolls up Spring Data Commons 1.6.4, Spring Data JPA 1.4.4, Spring Data MongoDB 1.3.4, and Spring Data Neo4J 2.3.4. Like Spring 4 before it, this release train ships with a Maven dependency BOM (bill of materials) that can be used to level set the dependency versions.
  • March 18th Webinar - Join Mattias Arthursson as he introduces Spring LDAP 2.0.0.
  • March 25 Webinar - Be sure to check out Spring lead Juergen Hoeller's upcoming webinar: Spring Framework 4 on top of Java 8.
  • SpringOne2GX 2013 replay: Spring and Sencha - A Match Made In Heaven (or at least the cloud) is now available
  • Check out a founding father of Spring - Rob Harrop talk about Running Spring in Amazon Web Services, a great talk from SpringOne2GX 2013.
  • HCL Technologies AVP Srinivas Ajjarapu penned a nice post on some of the ways that Pivotal's stack is helping IT lower spending and get more.
  • Nicolas Frankel is back, and has put together a nice post on how to chain together Spring MVC ViewResolvers.
  • Daniel Mikusa and Stuart Williams's webinar replay introducing Apache Tomcat 8 is now available online.
  • Steve Perkins put together a nice post on his first steps with Spring Boot. Check it out!
  • John Thoms has put together a nice example of configuring Spring Boot's embedded Tomcat application server with SSL support. Check it out!
  • The stellar JHipster project, which sits on top of Spring Boot, is now available complete with new monitoring endpoints and a lot more!
  • Speaking of Spring Boot, Spring Boot RC 3 is now available. Don't hesitate to check it out.
  • Our pal Xavier Padró is back with a nice look at how to use the view templating library Thymeleaf with Spring
  • My pal Andy Piper and Eclipse Paho (an MQTT client) lead Andy Piper has put together a nice post on how to use Spring Integration's MQTT support.

    春天的这个星期-2014年2月18日

    工程
    乔什·朗（Josh Long）
    2014年2月19日

    欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！

    1. Spring Data负责人Oliver Gierke宣布Spring Data Babbage SR3已发布 。新版本汇总了Spring Data Commons 1.6.4，Spring Data JPA 1.4.4，Spring Data MongoDB 1.3.4和Spring Data Neo4J 2.3.4。像之前的Spring 4一样，此发行版本火车附带一个Maven依赖项BOM （ 物料清单 ） ，可用于对依赖项版本进行级别设置。
    2. 3月18日，网络研讨会-加入Mattias Arthursson ，他介绍了Spring LDAP 2.0.0 。
    3. 3月25日的网络研讨会-请务必查看Spring领导者Juergen Hoeller即将举行的网络研讨会：基于Java 8的Spring Framework 4 。
    4. SpringOne2GX 2013重播： Spring和Sencha-天生一对（或至少是云计算）现已可用
    5. 查看Spring的创始之父-Rob Harrop谈“在Amazon Web Services中运行Spring” ，这是SpringOne2GX 2013的精彩演讲。
    6. HCL Technologies的高级副总裁Srinivas Ajjarapu 在Pivotal堆栈帮助IT降低支出并获得更多收益的某些方式上发表了一篇不错的文章 。
    7. Nicolas Frankel回来了，并且发表了一篇不错的文章， 介绍如何将Spring MVC链接在一起ViewResolver s 。
    8. Daniel Mikusa和Stuart Williams的介绍Apache Tomcat 8的网络研讨会重播现在可以在线获得 。
    9. 史蒂夫·珀金斯（Steve Perkins）在使用Spring Boot的第一步中撰写了一篇不错的文章。看看这个！
    10. John Thoms提出了一个很好的示例，该示例使用SSL支持配置Spring Boot的嵌入式Tomcat应用程序服务器 。看看这个！
    11. 位于Spring Boot之上的恒星JHipster项目现在可以使用新的监视端点完成了 ！
    12. 说到Spring Boot， Spring Boot RC 3现在可用 。不要犹豫，检查一下。
    13. 我们的好朋友XavierPadró回来了，对如何在Spring中使用视图模板库Thymeleaf进行了很好的介绍
    14. 我的好朋友Andy Piper和Eclipse Paho（一个MQTT客户）负责人Andy Piper撰写了一篇不错的文章，介绍了如何使用Spring Integration的MQTT支持 。
