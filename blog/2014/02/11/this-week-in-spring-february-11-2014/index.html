春天的这个星期-2014年2月11日

工程
乔什·朗（Josh Long）
2014年2月11日

欢迎回到春季本周的另一个装置！我急忙准备几个星期。我将于18日在我们的办公室（加利福尼亚州，旧金山，霍华德街875号）进行技术演讲（非常欢迎您！只要在美国太平洋标准时间18日星期二12:30左右出发，然后加入我们，我们一起谈论Boot，baby！），然后在19日接受一些培训，然后我将在亚特兰大参加史诗般的DevNexus表演， 然后第二天，前往加拿大蒙特利尔参加ConFoo会议。我希望在GoPivotal的技术演讲办公室 ， 亚特兰大或蒙特利尔见到您！

本周，我们有很多很棒的东西，特别是关于使用Spring来构建数据驱动的应用程序的方面。看看这个！

  • 想更多地了解Spring 4对Java 8的丰富支持？查看Spring领导者Juergen Hoeller即将举行的有关Spring 4对Java 8的支持的网络研讨会 ！
  • 我们很高兴地宣布一个关于Spring LDAP项目的网络研讨会
  • 在Spring Data Neo4j上查看项目负责人Michael Hunger的史诗般的SpringOne2GX 2013演讲
  • Jeremy Grelle，Mattias Niemela和Tim Branyen在SpringOne2GX 2013上关于AngularJS的演讲，Backbone.js和SpringOne2GX 2013的以客户为中心的设计现已在线发布
  • Spring Data负责人Oliver Gierke刚刚宣布Spring Data Elasticsearch 1.0M1已发布
  • 说到这一点，请查看Oliver的帖子，其中介绍了Spring Data Codd RC1中的一些新功能。
  • JANNE VALKEALAHTI在使用Spring Data YARN项目上有一篇不错的文章
  • Thomas Risberg宣布Spring Hadoop M5现已可用
  • 查看Hadoop专家Kailash Kutti对Apache Hadoop和Spring Data Hadoop的介绍（来自新加坡Spring用户组的视频）
  • 同样来自新加坡Spring用户组的演讲，介绍了Spring Data JPA和Spring Data MongoDB
  • 奥利弗（Oliver）和公司随Spring Data Couchbase 1.0 RC1一起发布
  • Martin Lippert宣布了Spring Tool Suite和Groovy / Grails Tool Suite 3.5.0。M2现在可用
  • 小诺里斯·谢尔顿（Norris Shelton，Jr）写了一篇不错的文章，介绍了如何获取当前有效的Spring配置文件
  • 扬·菲利普（Jan Philipp）撰写了一篇不错的文章，内容涉及如何使用Spring Integration的出色网关支持来公开基于接口的消息传递系统的外观，在这种情况下，该外观最终会与RabbitMQ对话
  • Theodora Fragkouli 在Spring上为Apache Hadoop 2.0 M5撰写了一篇不错的文章 。看看这个！
