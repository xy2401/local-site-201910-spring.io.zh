春天的这个星期-2014年9月16日

工程
乔什·朗（Josh Long）
2014年9月16日

欢迎来到春季本周的另一期！ 我们已经从上周的史诗般的 SpringOne2GX 2014回来了，但还不为自己感到骄傲！从SpringOne2GX返回后，我便着手准备今天上午的网络研讨会， 即带有Spring Boot的Bootiful微服务 。这似乎确实引起了很好的共鸣，这主要是因为Spring Boot和Spring Cloud 很棒 ！我要等到SpringOne2GX上有关该主题的所有精彩视频以及该网络研讨会本身才能与您分享。对我来说，这就像是一种

  1. Android主管和移动忍者Roy Clarkson的Spring宣布了Android 2.0.0的Spring。M1已发布 。新版本支持OkHttp（通过OkHttpRequestFactory ），更新其中包含的内容RestTemplate实施与RestTemplate包含在Spring框架中（包括支持使用ParameterizedTypeReference ）。它有效地提高了Android 2.2的基准，并包括许多错误修复和其他改进。这是很长时间以来的第一个新版本-我建议尽快获取并尝试使用这些功能！
  2. 我并不是要取笑，但您至少应该阅读Syer和Spencer Gibb博士在SpringOne2GX上发表的关于Spring Cloud演讲的幻灯片。好东西。
  • Spring Security主管Rob Winch 在Spring Session项目和websockets上撰写了一篇关于使用Spring Security的不错的文章。基本上，如果您使用的是JSR 356（Websocket JSR），则可能只需要访问HTTP会话即可使其过期，因为Web-socket API无法使HTTP保持活动状态或使访问变得容易。现在可以修复所有问题！
  • 说到Rob，他录制的演讲录音-Spring Boot…自2013年以来推特完整的Spring应用程序 -现在可以在InfoQ上获得！看看这个！
  • 您是否想念上周的史诗般的 SpringOne2GX 2014 ？没有恐惧！像往常一样，影片将可在Spring.io在滴流在未来数个月， 而在此期间，你可以阅读天1和第2天回顾一下帖子。
  • ADTMag 在最近发布的Spring framework 4.1上有一个快速（尽管不是技术性的）新闻。
  • 我刚巧接受了来自QCon 2014的Spring框架负责人Juergen Hoeller的精彩采访，涉及所有主题。面试真好！
  • Nguyen Anh Tuan有一篇有趣的帖子， 介绍了他的团队如何构建无状态且无会话的多租户Spring Security实现
  • 一个有名的博客TheSpringWay上有一篇不错的文章，介绍了便利的Spring MVC 4.1对响应代码处理的改进 。您知道历史上有点泥泞的人吗ResponseEntity现在有一个不错的生成器API？
  • JavaCodeGeeks博客上有一篇关于将Spring Batch与Spring Boot结合使用的不错的文章。这是一篇有趣的文章，其原因有很多，其中最重要的一点是它还捕获了用户在Java配置中的第一步（在Spring Boot中猖ramp运行）。
  • 今天上午的网络研讨会给我带来了很多乐趣。下周加入我，与我的朋友Joram Barrez进行网络研讨会， 以构建以流程为中心的Spring Boot和Activiti BPMN2工作流引擎 ！
