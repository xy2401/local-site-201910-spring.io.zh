春天的这个星期-2014年9月2日

工程
乔什·朗（Josh Long）
2014年9月3日

欢迎来到春季本周的另一期！

这是TWiS的最后一期， 下周 SpringOne2GX 2014 揭幕 ！我个人非常兴奋！我等不及要见你们了。这将非常有趣。所以，让我们继续吧！

  • Spring Data负责人Oliver Gierke刚刚宣布发布Spring Data Dijkstra SR4 ，其中包含错误修复和各种各样值得升级的内容。看看这个！
  • Spring和Groovy＆Grails工具套件负责人Martin Lippert刚刚宣布了3.6.1 。新版本将tcServer更新为3.0.0，其中包括构建性能的改进，AspectJ和AJDT的注释处理器等等。立即下载，并拥有最新，最出色的IDE，可以遵循所有指南并遵循SpringOne2GX 2014上的示例 ！
  • 希望您能在9月16日加入我的网络研讨会，探讨如何使用Spring构建微服务
  • 9月23日，与Joram Barrez和我一起参加网络研讨会， 探讨如何使用开源BPMN2引擎Activiti构建以流程为中心的Spring应用程序。
  • 您看到美丽，更直观的新Spring Initializr吗？Preetttyyyyyy…
  • 不眠不休的Syer博士刚刚发布了Spring Security OAuth 2.0.3 ，其中主要包括针对JWT令牌的错误修复。此外，资源服务器（如果配置了@Configuration ）将在允许访问受保护的资源之前检查客户端和范围的有效性，这意味着更多的即时客户端特权撤销。真好！
  • 春季忍者Greg Turnquist写了一篇不错的文章， 介绍了不断完善的Spring IO指南 。从那里开始，然后继续阅读一些指南！
  • 我做了一个演讲， “您最近看过春天吗？” ，今年初在DevNexus上。它在线上 ，如果您还没有的话，值得关注！（如果我可以这么说..）
  • 利芬Doclo已经把一个非常酷的职位对他的轴突框架Spring启动自动配置 。Axon是CQRS（命令和查询职责隔离）模式的实现。（Axon建立在Spring，Spring Integration等基础之上，本专栏的读者将从以前的综述中了解到！）
  • MacAdie Web Blog 对在Spring Boot上的Austin Groovy和Grails用户组会议上做的笔记进行了粗略但非常有用的回顾。
  • 我喜欢Ralph Wehner的帖子，其中介绍了如何一起使用Spring Boot和Vaadin ：它解释了所有内容，甚至链接了示例示例 -太棒了！
  • Jayway博客上的我们的朋友又回来了！本周， 他们介绍了如何构建自定义的Spring Boot HealthIndicator s
  • 的IT Hood提示有帮助：如果您遇到HTTP状态代码之类的神秘错误406 Not Acceptable调用基于Spring MVC的REST端点时的结果，您可能会检查对CLASSPATH的依赖关系是否正确 。简单，但容易忘记！（当然，除非您使用的是Spring IO Platform BOM或Spring Boot的启动程序依赖项 ！）。
