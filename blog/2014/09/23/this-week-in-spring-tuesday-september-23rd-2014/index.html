春天的这个星期-2014年9月23日，星期二

工程
乔什·朗（Josh Long）
2014年9月23日

欢迎来到春季本周的另一期！今天早上，我很高兴与我的朋友Joram Barrez一起介绍将Activiti BPMN工作流引擎与Spring（和Spring Boot）一起使用。该演讲应该在接下来的几周内上线。我还将在本周的Alfresco峰会上与Joram共同演讲（主题相同）。如果您错过了SpringOne2GX 2014，请查看第1 天和第2天总结，在这里可以了解NetFlix，Spring Boot和微服务体系结构等信息。

  1. 我上周进行的网络研讨会演讲的重播，即带有Spring Boot（和Spring Cloud）的Bootiful Microservices ，现在可以在线获得。由于Boot和主题的原因，此演讲引起了很多关注，因此实际上在我提出演讲后的几天 （不是通常的一两周）内就可以在线观看！请享用！
  • 我真的在挖掘最近来自Codeleak.pl博客的一些帖子！首先，这篇关于单元测试电子邮件集成代码的文章写在Spring Boot的顶部 -快来查看！
  • 在进行微服务讨论后不久，我收到一条推文，向我介绍了最近添加的关于波兰公司如何使用Spring构建微服务的讨论 。演讲大多是高水平的，到处都是春天。这是一个很酷的话题，并且在30分钟之内可以轻松快速地消耗掉）。
  • RafałBorowiec在Codeleak.pl网站上的工作非常出色！他在之前的帖子（关于单元测试电子邮件）上进行了后续研究，着眼于Spring Boot的@ConfigurationProperties支持
  • Rafał 还整理了这篇关于使用验证组来验证Spring MVC模型的文章，以及另一篇关于测试给定Spring MVC模型已注册全局错误的文章。
  • Biju Kunjummen 在定制方面发表了不错的文章HttpMessageConverter在Spring Boot应用程序中
  • Adrian Matei在有关使用Java配置和Spring Boot构建Spring Batch应用程序的文章中撰写了一篇非常不错的文章
  • Netflix（和我们的朋友）Tomas Lin刚刚发表了一篇不错的文章，内容涉及如何以可配置的方式公开来自不同环境的静态资源（例如JavaScript资产）
  • 我的好朋友和春天的忍者格雷格•特恩奎斯特（Greg Turnquist）在他即将出版的关于“春天靴子”（Spring Boot）的书上发表了一篇不错的文章。我等不及了！格雷格是个聪明而热情的家伙，如果您读过他的Spring Python书籍，您也会知道他的读物也很有趣！
  • Develop and Conquer博客上有一篇很棒的文章，内容涉及在Spring Boot应用程序中使用Spring Security的Active Directory LDAP-快来查看！他在其他主题上也有很多不错的文章（例如在Spring 4的websocket支持之后使用ActiveMQ和STOMP支持），所以请检查一下！
