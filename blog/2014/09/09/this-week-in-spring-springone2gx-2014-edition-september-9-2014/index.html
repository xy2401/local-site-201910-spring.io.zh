春季本周（SpringOne2GX 2014版！）-2014年9月9日

工程
乔什·朗（Josh Long）
2014年9月9日

欢迎使用SpringOne2GX 2014直播的本周春季本期的另一期！今年的表演值得一看！昨天，我们在“孤星状态”中以惊人的主题演讲拉开了节目的序幕，该活动由工程枢轴高级副总裁休·威廉姆斯 （ Hugh Williams） 主持 ，春季团队带领Juergen Hoeller ， Dave Syer ， Graeme Rocher和Netflix的Andy Glover 。

当然，我们将做一些总结性博客，请继续关注。在我们开始本周综述之前，请查看此视频，该视频将被投射到Dallas Omni酒店上，以展示SpringOne2GX 2014徽标！

而且，虽然这与录音（无论如何都会很快提供）或什至是精心编写的摘要（敬请期待！）完全不同。也要来..），这是昨晚主题演讲中的一些选择推文，以激发您的胃口：

确实，甚至在SpringOne2GX之前的一周都是史诗级的，因为团队忙于发布令人兴奋的新内容供我们使用！本周综述挤满了好东西。

  1. 刚好赶上 SpringOne2GX， Spring的负责人Juergen Hoeller 刚刚宣布Spring 4.1现已可用 ！新版本是挤满了伟大的东西，包括JMS注释，全面的JCache（JSR 107）的支持，新的Spring MVC的意见，WebSocket的改进，改进的性能，以及更多 ！
  2. Spring Boot的共同负责人Phil Webb刚刚宣布了Spring Boot 1.2.0。M1 ，正在通过Spring Boot 1.1.6修复 。1.1.6是一个强烈建议的错误修复。1.2.0。另一方面，M1是非常令人兴奋的1.2 Spring Boot版本的第一个里程碑。它具有以下新功能：JTA支持，JNDI绑定资源的更容易使用（本身就是一个不错的功能，但是对于轻量级JTA来说特别方便！），Spring Cloud PaaS连接器的自动配置，通过Tomcat和码头和改进的指标DataSource和健康指标。
  3. Spring Data负责人Oliver Gierke刚刚宣布Spring Data Evans已离开GA ！新版本塞满了所有功能！Redis Sentinel，ALPS和摘录投影，多商店配置，Spring 4.0的基线，对top和first作为衍生查询的关键字，MongoDB的2.6全文， 等等 。
  4. 延续数据主题， Apache Hadoop忍者Thomas Risberg的Spring刚刚发布 。新版本增加了对追加模式商店编写器，安全集群的kerberos配置，Spring YARN中的容器分组和集群的支持，并且仍与Hadoop 2.0 +，Pivotal HD 2.0，Cloudera CDH5和Hortonworks 2.1兼容。
  5. 我将很快举行两个网络研讨会：一个是9月16日在Spring上构建微服务，另一个是9月23日在Spring上使用Activiti工作流引擎 。立即注册，并提出您的问题，评论等等！我们将有很多话题，这应该很有趣！
  6. 我的英雄之一， Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration Java配置DSL里程碑3 。该博客包含（许多）新功能和支持代码的列表。我无法在这里做到公正，只需检查一下即可。
  7. Spring忍者和全能的家伙Andy Wilkinson刚刚宣布了Spring IO platform 1.0.2版本的发布 。新版本包含许多新项目，因此请务必将其签出。Spring IO平台试图在多个Spring项目之间协调依赖关系，以便从依赖关系使用者的角度来看，与通用库之间永远不会发生任何冲突。
  8. Spring Integration的负责人Gary Russell刚刚宣布了Spring AMQP（用于Java）1.4里程碑1 。新版本包括注释支持（ @EnableRabbit ）和注释驱动的消息侦听器（ @RabbitListener ）。头晕！这反映了Spring 4.1对JMS的类似支持。
  9. 由于悲伤，复杂的原因，Artem本周不在SpringOne2GX上 （我们想念您，Artem！），但这并没有阻止他在上周的Spring Integration Java配置DSL版本（见上文）中引起轰动。 现在，Spring Integration 4.1 Milestone 1发行了 。Spring Integration版本包含许多很酷的功能，包括将常见类型迁移到Spring 4本身，性能改进，Spring 4.1 SpEL编译器的引入，Web套接字适配器（是的，Web套接字适配器！）， http：// github.com/projectreactor ，一个由Boon驱动的 JSON映射器，是一种EIP拆分器模式的实现，可以返回一个Iterator以获取结果（因此可以流式传输 ）。
  10. 我们的朋友Matt Raible看了使用Spring Boot支持的JHipster项目 。他遇到了一些困难，但博客提供了有关如何解决问题的各种有用信息，以便您可以避免它们并真正飞起来！
  11. Blog4Java博客（由我们的朋友Javier Barquero经营）有一个后续帖子，介绍了他如何使用Spring Batch
  12. 而且，当然，如果没有年度新闻稿 ，SpringOne2GX将是完整的吗？
  13. Chariot Solutions博客上有一篇有趣的文章，内容涉及将Spring Boot应用程序从Java转换为Scala-绝对值得一看。有趣的是，在这个特定的示例中，似乎并没有像使用Java 8那样获得太多收益。
  14. 您在达拉斯沃思堡地区吗？由于我们扔SpringOne2GX 2014在达拉斯， 我们认为我们会花点时间来围捕一些.. ok了很多 .. Spring团队和头部到当地的Java用户组 （称为“Java的用户美达组”（或Java MUG的简称）打招呼。希望您会考虑加入我们，度过一个非常有趣的夜晚！
