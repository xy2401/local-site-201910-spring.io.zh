Pivotal在JavaOne 2014上

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年9月26日

您要去JavaOne 2014吗？如果您愿意，我们邀请所有Java社区成员在Pivotal Booth（展位号5201）停留，并与Pivotal工程师讨论Java SE的最新创新，包括Spring，Groovy，Grails，Redis，RabbitMQ，Tomcat，Cloud代工和Hadoop。展览空间的开放时间为星期一，星期二上午9:30 – 5:30 pm，以及星期三上午9:30至4:30 pm，因此应该有足够的时间来回答您的问题。请参阅以下有关我们赠品的地图和展位详细信息-快来寻找我们！展位上有很多很棒的东西。

关键技术人员也将参加很多会议，因此请确保将以下演讲添加到您的会议构建者中并预留座位：
           
Josh Long-Pivotal的春季倡导者

  • 春季BOF [BOF3868]
  • 春季4TW！[CON3693]

Cornelia Davis — Pivotal的平台工程师

  • 在云中运行Spring应用[CON4327]

John Field — Pivotal解决方案架构师

  • 开源身份和访问管理专家小组，第3部分[BOF3478]
  • 使用Java的安全Web应用程序剖析[CON3479]

Guillaume Laforge — Pivotal的Groovy项目经理

  • 2014年脚本碗：[CON2939]战斗激怒
  • 2014年及以后的Groovy [CON5996]
  • 基于Java 8的Groovy [CON5839]

Martin Lippert-Pivotal的首席软件工程师

  • 将Eclipse IDE连接到开发人员工具的基于云的时代[CON1739]

Oliver Gierke-Pivotal的 Spring Data 项目负责人

  • REST保证：具有Spring MVC的超媒体API [CON6071]

CédricChampeau — Pivotal Groovy团队的高级软件工程师

  • 用Groovy特性重新思考API设计[CON2425]

有趣的非关键会议

Groovy和Grails难题者：像往常一样—陷阱，陷阱和最终案例[CON1764]
JVM上的微服务：实用概述[CON4952]

使用Java，Spring Framework 和Oracle Tuxedo扩展现有的C / C ++ / COBOL应用程序[HOL9447]

在展位上访问我们，就Cloud Foundry，Spring，Groovy，Grails，Redis，RabbitMQ，Tomcat，Hadoop等进行技术专家对话。我们将在展位上演示有关以下内容的精彩演示：



  • 通过JBoss和WebLogic Buildpack演示将Java EE和PaaS与Cloud Foundry结合使用

  • 物联网，带有Spring XD演示的Hadoop

  • 带有Spring Boot的微服务 架构和PaaS ， Spring Cloud演示


关键Web服务连帽衫推！使用Spring Boot构建REST服务 ，将其实时发布到run.pivotal.io并获得连帽衫！您可以提前进行操作，并向我们展示其运行情况，或者在展位上使用我们的笔记本电脑。
