Spring for Apache Hadoop 2.1.0。M1发布

工程
托马斯·里斯伯格
2014年9月8日

一年前，我们及时发布了适用于Apache Hadoop 2.0版项目的Spring的第一个里程碑版本，正好赶上SpringOne 2GX 2013。为了保持这种传统，我们很高兴地宣布Apache Hadoop 2.1的Spring的第一个里程碑版本发布，正好赶在德克萨斯州达拉斯的SpringOne 2GX 2014发布之时。该版本现已在Spring IO存储库中提供

2.1.0的主要更改。M1版本：
*增加了对商店作者中附加模式的支持
*增强了用于安全集群访问的kerberos配置
*在Spring YARN中支持容器分组和集群，它带来了用于运行多种容器类型的功能，新的REST API和具有新客户端命令和命令行外壳的改进的应用程序模型。
*从此版本开始，我们支持Hadoop 2.0 API，但不再提供与较早Hadoop v1发行版的向后兼容性。

Spring Boot 1.2.0。M1现在可用

发布
菲尔·韦伯
2014年9月8日

我很高兴地宣布Spring Boot 1.2.0。M1现在可以在Spring里程碑存储库中使用 。该发行版基于Spring Framework 4.1，并提供了许多改进和新功能。重点包括：

  • 使用Atomkos或Bitronix嵌入式事务管理器或完整的Java EE应用服务器，支持跨多个XA资源的分布式JTA事务。
  • 轻松配置JNDI绑定DataSource和ConnectionFactorybeans。
  • 改进的JMS支持，包括对Spring Framework 4.1的自动配置@EnableJms注解。
  • 自动配置支持spring-cloud 。
  • Tomcat和Jetty的SSL配置更加轻松。
  • 已改善DataSource指标和健康指标。
