嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我去过德克萨斯州的达拉斯和奥斯汀，首先去参加客户会议，然后去参加史诗般的Reactive Summit并发表演讲。如果您在附近，请在Twitter上打个招呼！ 我们有很多内容需要介绍，让我们开始吧！

Spring Web Services 2.4.1。发布/ 3.0.0。RC1发布

工程
格雷格·特恩奎斯特
十月17，2017

春季问候社区，

Spring Web Services现在正式有两个分支： master （3.0+）和2.x （旧2.x系列的维护）。只要支持Spring Framework 4.x，就会维护2.x系列，但是任何新工作都将在master分支上进行。

2.4.1。RELEASE是3.0.1发行版的次要补丁程序。RC1是一项重大升级。以下链接包括相关票证。

2.4.1发行说明 | 2.4.1文档 。

3.0.0。RC1发行说明 | 3.0.0。RC1文档 。

  • 3.0.0。RC1：

    • 重新配置Spring Web Services使其在Spring Framework 5.0和Java 8上运行

    • 删除对org.springframework.ws.soap.security.wss4j软件包所涵盖的Apache WSS4J 1.x较早版本的支持。相反，我们仅支持WSS4J 2.x（在org.springframework.ws.soap.security.wss4j2软件包中）。

    • 将我们的XMPP库Smack支持升级到4.2。

    • 该版本计划由Spring Boot 2.0和Spring IO Platform Cairo所采用。

  • 2.4.1。发布：

    • 维持我们现有的依赖基准，并在最新支持的Spring Framework和Spring Security版本上转移

    • 只要支持Spring Framework 4.x，我们就将维护2.x版本。

    • 根据Spring Framework 4.x和5.x之间的一些差距，我们将不再对2.x分支与Spring Framework 5.x进行兼容性检查。如果您需要Spring Framework 5，则确实需要升级到我们的3.x分支。

Spring Boot 2.0.0 M5现已上市

发布
菲尔·韦伯
十月12，2017

Spring Boot 2.0.0 M5已发布，现在可以从我们的里程碑资料库中获得 。此版本关闭了168个问题并提出了要求，并继续朝着2.0 GA迈进。感谢大家的贡献！

这个里程碑改进了M4中的许多项目，并提供了许多值得注意的新功能：

  • 最初对Java 9的支持 -我们将继续完善对Java 9的支持，但此发行版应适用于基本应用程序。查看此Wiki页面以获取更多详细信息 。
  • Spring Security的OAuth 2.0客户端支持-Spring Security开始提供更多OAuth 2.0功能。该版本的Spring Boot为Spring Security的OAauth 2.0客户端支持提供了自动配置。
  • JSON-B支持 -我们现在与现有的Jackson和Gson集成一起提供对JSON-B的支持。我们继续推荐杰克逊作为我们的默认选择。
  • 会话执行器端点 -Spring Session的用户现在可以直接从执行器查看和终止会话。
