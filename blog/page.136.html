Spring Cloud Stream 1.0.0。RC2现在可用

发布
马里乌斯·波哥维奇（Marius Bogoevici）
2016年4月7日

我很高兴代表团队宣布Spring Cloud Stream 1.0.0的发布。RC2，可在此处获得 。作为候选发布周期的一部分，它包含许多错误修复和少量调整。完整的更改列表在此处 。

一些重点：
-Spring Cloud Stream应用程序现在使用ObjectMapper由Spring Boot提供（如果可用）；
-标头嵌入现在是通用的生产者/消费者属性。可以通过需要嵌入标头传播的中间件来启用或禁用它。这使Spring Cloud Stream外部的互操作性更加容易。 （到现在为止，这是Kafka的专有属性）；
-进一步的文档和样本改进。

春季数据发布火车Hopper进入GA（和SR1）！

发布
奥利弗·德罗博姆
2016年4月7日

我很高兴代表整个Spring Data团队宣布Spring Data发布火车Hopper的全面上市。这列火车包括395张固定的车票，最重要的补充是：

  • 将Spring Framework基准升级到4.2（尤其是4.2.5）。
  • 在Spring Data JPA和MongoDB中支持示例查询。
  • 改进了对组合注释的支持。
  • 在Spring Data REST中支持查找类型和嵌套关联
  • Solr（Solr 5.5上的2.0），Neo4j（4.1），Couchbase（Couchbase 2.0上的2.1），Elasticsearch（ES 2.2上的2.0）的主要升级
  • 升级到Querydsl 4。
  • 支持Spring Data Redis的对象映射和存储库。
  • 支持Redis Cluster 。
  • 支持JPA和MongoDB中存储库查询方法的预测。
  • 将Spring Data Envers添加到发布系列中（以前是单独维护的）。

2016年4月6日

尊敬的Spring社区，

依赖注入改进：

  • @Autowired注入Map / Collection bean和自我引用

  • InjectionPoint / DependencyDescriptor作为@Bean方法的可注入参数

网络研讨会重播：在NetBeans IDE中使用Spring Boot启动并运行

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年3月30日

演讲嘉宾：Geertjan Wielenga
幻灯片：无幻灯片，所有演示
NetBeans IDE是Java平台的官方IDE。NetBeans专注于快速开发各种Java应用程序，它是全面且易于使用的。特别是，其用于端到端开发的免费和开源工具倍受赞誉。
近年来，由于NetBeans与Maven和Gradle的直观而强大的集成，许多开发人员已将NetBeans添加到其工具箱中。在本课程中，您将了解NetBeans如何非常适合使用Spring Boot来设置和开发应用程序。
您还将了解NetBeans IDE 8.0和8.1中已添加的许多新功能，这些功能可帮助您在Spring Boot的上下文中快速有效地使用Java和Maven。在本课程结束时，您将更好地掌握Spring Boot，并全面了解NetBeans提供的与之配合使用的功能！
