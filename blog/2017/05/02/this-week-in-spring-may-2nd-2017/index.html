春天的这个星期-2017年5月2日

工程
乔什·朗（Josh Long）
2017年5月2日

欢迎来到春季本周的另一期！本周，我在澳大利亚珀斯，为惊叹的YOW！West会议 ，与开发人员讨论如何构建云本机Java应用程序。如果你周围，说喜 。和往常一样，我们有很多内容需要介绍，让我们开始吧！

我对5月30日至31日在芝加哥举行的第一场春季活动感到非常兴奋。我会和其他春季大师一起去那里，例如Eddu Melendez，Anupama Pradhan和Michael Minella。希望您能加入我们！

  • 春季IO主管Any Wilkinson宣布了春季IO平台Athens-SR5 。他还宣布了Spring IO Platform Brussels-SR2
  • Spring Session的负责人Rob Winch刚刚宣布了Spring Session 1.3.1
  • Spring Integration的领导者和伟大的人物Gary Russell刚刚宣布了Apache Kafka 2.0M1的Spring
  • Spring Cloud忍者Glenn Renfro 刚刚宣布了Spring Cloud Task 1.2.0
  • 我很喜欢 （以防您不清楚我对本帖的看法！）通过TensorFlow和Spring Cloud Data Flow进行实时情感分析
  • 本月的ZeroTurnaround Web框架报告的结果并不令人惊讶： Spring MVC遥遥领先，其次是JSF和GWT，然后是Spring Boot 。
  • Okta博客上的这篇文章“ 使用Spring Boot和Angular进行Bootiful开发 ”是我们的伙伴Matt Raible撰写的另一篇文章。
  • 查看Spring WebFlux工程师Sebastien Deleuze制作的ZT（ZeroTurnaround的缩写）实时重播，对Reactive Spring进行史诗般的观察。
  • Spring Cloud工程师Jon Schneider正在制作一些非常酷的东西，这可能表示未来的方向。谁知道！？但是您绝对应该检查他的项目Spring Metrics
  • 我的好友Matt Raible和我在最近的2017 Devoxx France的两方演讲中讨论了使用Spring Boot和Angular构建渐进式Web应用程序的问题： 第1 部分和第2部分 。真是太有趣了！Merci mille fois，法国Devoxx！感谢法国Devoxx让我们参加这些会议！
  • 我在Spring Boot上接受了令人惊叹的Software Engineering Daily Podcast的采访。在那里很有趣。我会定期收听播客，非常感谢这次机会！
  • 我喜欢Alex Soto 撰写的有关使用Arquillian测试Spring Data应用程序的文章 。
  • 这是有关在Docker容器中发现微服务公开端口的有趣文章。似乎..痛苦。我很高兴在这里得到如此详尽的记录和阐述！好东西！
