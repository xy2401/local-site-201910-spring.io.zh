Spring Cloud Stream 切尔西。SR2发布

发布
索比·查克（Soby Chacko）
2017年5月17日

我很高兴代表社区宣布发布Spring Cloud Stream切尔西。SR2。这是切尔西发布系列中的第二个常规发布版本，它修复了切尔西的许多问题。SR1。

新版本可以在Maven Central中获得，有关其功能的详细说明可以在参考文档中找到。有关工件和最新更改的信息，请参阅发行说明 。

接下来是什么：

发布之后，我们将着手进行下一轮名为“ Spring Cloud Stream发布”的培训。 Ditmars ，并扩展Spring Cloud Stream生态系统。与开发Ditmars发布火车，我们将开始Elmhurst发布培训将基于Spring Framework 5和Spring Boot 2。

以下是一些即将到来的主题：

  • 改进活页夹中的错误处理支持，并提供用户友好的API进行恢复和错误处理

  • 里程碑（1.0.0。M1）发布用于JMS活页夹

  • 在kafka活页夹中介绍Kafka Streams支持 。

  • 反应源

  • 绩效基准
