    春天的这个星期-2017年5月23日

    工程
    乔什·朗（Josh Long）
    2017年5月23日

    欢迎来到春季本周的另一期！本周，我在荷兰的阿姆斯特丹和德国的埃森，与客户讨论云原生转换。我们有很多内容要介绍，让我们开始吧。

    • Spring Tool Suite的负责人Martin Lippert刚刚宣布了Concourse Pipeline和Cloud Foundry清单编辑支持版本作为语言服务器的预览版。这些语言服务器可由支持它们的任何IDE或文本编辑器使用，在这种情况下，包括Microsoft的Visual Studio Code和更高版本的STS本身。
    • Spring Cloud Stream忍者Soby Chako 刚刚宣布了Spring Cloud Stream。SR2
    • 西蒙·巴斯利（SimonBaslé）刚刚宣布了反应堆铋释放列车M1
    • StéphaneNicoll刚刚宣布了Spring Boot 2.0M1 ！终于到了！获得第一个主要发行版的版本，以响应式Spring Boot 2.0。有很多好东西，包括更新的Gradle插件，轻松的绑定改进，主要依赖项升级等等！
    • 我认为PawełChudzik撰写的有关穷人批处理解决方案的文章-遍历大型数据集-非常有趣。结论很明显：如果要滚动浏览大量记录，请使用java.util.Stream Spring数据存储库中的finder查询。这是一个显而易见的结论。我是否也可以谦虚地建议您查看Spring Batch，并以穷人的价格购买有钱人的批次？ :)
    • 是否想在Spring MVC中更轻松地处理表单编码的JSON？为这个问题投票！
    • 我喜欢如何通过Amir Boroumand的Amir博客上的Code在Spring Boot上实现忘记密码的流程 。
    • Amir Boroumand 在使用Spring Boot创建REST API方面也有很好的演练
    • 我们在JDriven的朋友正在对MQTT，Spring Cloud Stream和Spring Cloud Data Flow进行很棒的操作（尽管我认为是荷兰语）
    • 我喜欢这种相当平易近人的方法，以了解如何由我们自己的Toshiaki Maki 将Hashicorp Vault部署到Cloud Foundry 。
    • 想了解无服务器的未来对Spring的看法吗？查看Spring IO上有关 Dave Syer博士关于Spring Cloud Function的精彩演讲的幻灯片。
    • WeaveWorks的范围现已可用并支持Cloud Foundry
    • 我喜欢史蒂夫叶格（Steve Yegge）的咆哮， 为何Kotlin是一门好话。有趣的阅读！
