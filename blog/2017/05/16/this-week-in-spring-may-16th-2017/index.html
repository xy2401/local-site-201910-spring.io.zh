春天的这个星期-2017年5月16日

工程
乔什·朗（Josh Long）
2017年5月16日

欢迎来到春季本周的另一期！这周我在慕尼黑参加春季聚会 ;柏林的OSDC ;克拉科夫的吉康 ;巴塞罗那用于春季I / O ，埃因霍温用于NextBuild 。如果您在附近，请打个招呼 。

和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Mark Pollack博士刚刚宣布了Spring Cloud Data Flow 1.2GA！ ，包括本地，Kubernetes，YARN和Cloud Foundry的服务器版本。这个新版本包含很多好东西。我特别喜欢对组合任务的支持，这使我能够将任务流程编排为一个有凝聚力的工作单元。这是复杂ETL管道的理想选择。它甚至包括对顺序，并行和条件顺序转换的支持。而且，当然，所有这些都可以混合搭配。以前，要获得这种功能，您必须在Spring Batch作业中手动连接远程分区。或者设置一些非常复杂的Spring Integration流程。而且做不到这一切。它还包括对运行中事件的更好可见性的支持，OAuth支持，基于角色的访问等等。这是一个非常酷的发行版！取得点点滴滴，立即尝试！
  • Spring Boot 2.0M1刚刚发布！
  • Spring Batch 4一直在努力 ！新的里程碑包括对Java配置API的更新（新的构建器！）。和依赖关系更新。
  • Spring REST Docs负责人Andy Wilkinson刚刚发布了维护版本
  • Spring Security主管Rob Winch刚刚宣布了新5.0系列的第一个里程碑 。新版本包括OAuth 2.0和OpenID connect 1.0支持以及反应式集成！
  • Rob再也没有休息过，他还宣布了与Spring Framework 5集成的Spring Session 2.0 M1 。
  • 新的Pivotal Spring教育发生了令人激动的变化。保罗·查普曼（Paul Chapman）在这里有详细信息
  • Spring Data忍者Mark Paluch 刚刚宣布了Spring Data Kay M3 。此版本包括对各种数据存储的大量更新，以及对MongoDB中反应式可尾游标的增强支持等。
  • 查看有关如何将Zipkin引入传统XML Spring应用程序的示例
  • Cereebro 是可与Spring Cloud一起使用的优质微服务仪表板，刚刚达到1.0
  • 让您部署和管理服务的BOSH变得非常简单 ！
  • 看看这个方便的反应性胸腺示例
  • 我喜欢这个介绍了消费者驱动的合同测试的演示文稿
  • 亚历克斯·索托（Alex Soto）返回第二期，介绍如何使用Arquillian测试Spring Boot应用程序
  • 这个针对Spring Cloud Data Flow的Tensorflow集成看起来很棒
  • 并非严格针对Spring，但我喜欢此资源， 了解如何在服务器端以及在哪里应用Kotlin
