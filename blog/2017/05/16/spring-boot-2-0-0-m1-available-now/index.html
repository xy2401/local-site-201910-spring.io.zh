Spring Boot 2.0.0 M1现在可用

发布
斯特凡·尼科尔（StéphaneNicoll）
2017年5月16日

我很高兴代表团队和所有贡献者宣布，Spring Boot 2的第一个里程碑已经发布，可以从我们的里程碑存储库中获得 。此版本关闭了300多个问题并提出了请求 ！

这个第一个里程碑是建立在Spring Framework 5.0之上的，并且需要它。RC1 。Spring Framework 5有许多不错的改进，包括对构建响应式应用程序的广泛支持。

第一个里程碑的重点包括：

  • Gradle插件的完整重写 （包括专用指南 ）
  • 使用WebFlux或WebFlux.fn支持响应式Web应用程序开发：专用spring-boot-starter-webflux初学者默认使用Netty，支持@WebFluxTest用于测试对MongoDB，Redis和Cassandra的反应性控制器和反应性数据支持
  • 整个产品组合中的主要依赖项升级： Spring Data Kay M3 ， Spring Security 5.0 M1 ，Spring Integration 5.0 M4， Spring AMQP 2.0 M4 ， Spring Session 2.0 M1和Spring Batch 4.0 M2
  • 轻松的绑定得到了改善

有关更改的完整列表和升级说明，请参阅WIKI上的《 Spring Boot 2.0发行说明 》和更新的参考文档 。

如果您想开始并发现这些新功能，可以在https://start.spring.io上轻松引导新项目。
