介绍

自从我教我的第一门Core Spring课程以来已经有10年了。那时，几乎所有内容都是XML，例如，配置JPA或Spring Security可能会非常艰巨。同时，借助组件扫描，Java配置和Spring Boot，Spring已经变得非常成熟，这使其使用起来更加有趣。春季项目的数量已大大增加。

Pivotal Training继续增强我们的Spring课程，并推出新课程-包括Spring Boot Developer和Spring Cloud Services。我提供了这些课程的概述以及下面的一些最新功能。最重要的是，您获得认证的方式已经改变。

我们最近宣布了对我们的春季认证计划的一项激动人心的更改。春季认证考试现在可以个人购买，而无需参加课程 。是的，你没看错！。经验丰富的Java开发人员-具备丰富的Spring知识-可以直接从Pivotal购买Spring认证考试。

也就是说，准备春季认证考试的最佳方法是参加相应的春季课程。但是，对我们的认证计划的更改使您更容易证明您的Spring专业知识。

除了您的实际证书之外，我们还与Badge Cert合作，以便您可以将经过验证的证书徽章添加到LinkedIn或其他个人资料中，以证明您拥有真实的证书。这就是徽章的样子。

这是我的Core Spring Instructor 认证的链接。

春季课程

核心弹簧

为开发人员和架构师设计的Core Spring是为期4天的课程，探讨使用Spring的所有基础知识。内容分为三部分：

  • 容器：使用Java Config进行配置，@ Autowired之类的Spring注释，XML bean文件，FactoryBeans，面向方面的编程以及基于Spring的集成测试。
  • 数据管理： JdbcTemplate，使用@Cachable，内存嵌入式数据库进行测试，Spring管理的事务，JPA和Spring Data存储库进行缓存
  • Web应用程序：使用Spring Boot编写Spring MVC控制器，使用Spring MVC，Spring Security和Cloud Native应用程序（Spring Cloud）实现REST

该课程已更新为使用Spring Framework 4.3，并涵盖了自Spring 4.0以来引入的若干功能，例如@Optional参数， @Sql用于数据库测试和@RestController 。

请注意，由于这是基础课程，因此4.x中的许多新功能超出了其范围。本课程还扩展了Web应用程序的范围，包括Spring Boot属性和配置以及使用Spring Cloud构建微服务。

参加我们的春季核心课程是准备行业认可的春季专业认证考试的最有效方法。要了解更多信息，请查看课程数据表并进行注册，请访问Core Spring信息页面。

春季网

是否在深入研究Spring MVC和相关技术？我们为期4天的Spring Web课程教您如何在四个关键领域创建企业Web应用程序：

  • 基础知识： Spring MVC控制器，使用Spring Boot构建Web应用程序，布局管理，定义视图，使用多种视图技术和表单处理
  • 内部构件：使用消息源，了解处理程序映射器，处理程序适配器，视图解析器，拦截器，ControllerAdvice，国际化，过滤器，资源处理，CORS支持，异常处理，移动设备支持
  • 企业：实施REST，AJAX和浏览器支持，Spring Security，使用MockMvc框架，使用Web套接字和服务器发送的事件
  • 可选：深入研究Spring MVC内部，在没有 Spring Boot的情况下配置Spring MVC应用程序，将现有应用程序迁移到Spring Boot

本课程也已更新为使用Spring Framework 4.3和Spring Boot 1.4。Spring 4功能，例如@RestControllers ， ResponseBodyAdvice ，资源处理管道，Web套接字，服务器发送的事件和@CrossOrigin （CORS）。所有实际练习都使用Spring Boot，因为它使编写Spring MVC应用程序变得快速而轻松。但是，尽管这是有益的，但我们希望学生学习如何充分使用MVC，并能够使用自己的配置覆盖Spring Boot的默认值。

Pivotal Training建议不熟悉Spring的学生在参加Spring Web之前先参加Core Spring课程。但是，如果您对Spring有一定的了解，并且主要兴趣是构建Web应用程序，那么我们的Spring Web课程非常适合您。

准备开始了吗？访问Spring Web页面以获取更多信息，查看课程数据表，并在即将开始的课程中进行注册。上完课程后，通过成功完成Spring Web Application Developer认证考试来证明您的技能。

企业之春

使用Spring构建由通信组件和流程组成的松散耦合的事件驱动架构。微服务架构可能是这种方法的一个示例。

Enterprise Spring课程部分基于流行的《 Enterprise Integration Patterns》一书，涵盖使用Web服务（REST）和消息传递（JMS，AMQP）的进程间通信，然后在此基础上构建以涵盖Spring Integration，Spring Batch和Spring Cloud Data流。

作为Spring Framework 4.3更新的一部分，本课程将包括REST的扩展范围，涵盖诸如UriComponentsBuilder ， @JmsListener ，用于配置Spring Integration的Java DSL，以及有关使用Spring Cloud Data Flow构建数据转换和集成管道的全新章节。

假定对Spring框架和Spring Boot相当熟悉。我们建议刚加入Spring的学生在参加Enterprise Spring之前先参加Core Spring课程。

访问Enterprise Spring页面以了解更多信息，查看课程数据表，并立即注册一个课程。完成后，通过注册企业集成专家认证考试来展示您的技能。

Spring Boot开发人员

Spring Boot使构建和配置Spring应用程序变得非常容易。为期2天的Spring Boot开发人员课程探讨了Spring Boot的主要功能，包括自动配置，数据访问，执行器等。

主要功能分为两个主要领域：

  • 基础知识： Spring Boot原理，自动配置，配置文件，使用属性和/或您自己的配置覆盖默认设置
  • 主题： Spring Boot应用于Web应用程序（Spring MVC，GSP，REST），数据访问（JDBC，JPA，Spring Data，NoSQL），安全性和OAuth，消息传递（AMQP，RabbitMQ），监视（执行器）

熟悉Spring框架对于本课程很有用。我们建议刚接触Spring的学生首先参加Core Spring课程。

访问Spring Boot Developer页面以了解更多信息，查看课程数据表并立即注册。

春季云服务

微服务和云本机应用程序是一种流行的模式，可以更快地交付应用程序。Spring Cloud Services是为期2天的课程，探讨如何使用Spring Boot和Spring Cloud支持这种类型的应用程序。课程主题分为四个功能领域：

  • 基础知识：为什么选择Cloud Native？云原生如何改变发展
  • 基础架构：构建Spring Boot应用程序，部署到PaaS（PCF）
  • 微服务：概述，云原生设计，12个因素。
  • Spring Cloud：共享配置，服务发现，客户端负载平衡，使用断路器处理和监视失败的服务

微服务应用程序涉及许多协作过程，因此如果没有智能的基础架构，它们将更难以部署。该课程将应用程序部署到Pivotal的Cloud Foundry PaaS中，以展示一步部署，自动恢复和可伸缩性的好处。

熟悉Spring对于本课程很有用。我们建议刚接触Spring的学生首先参加Core Spring或Spring Boot课程。

要了解更多信息，请查看课程数据表，并在即将举行的课程中注册，请访问我们的Spring Cloud Services页面。
