春天的这个星期-2017年9月12日

工程
乔什·朗（Josh Long）
2017年9月11日

嗨，春季粉丝！这周我在新加坡度过了令人惊叹的 YOW！新加坡活动，并与客户交谈。

另外，我将进行测试聚会 -不要错过！

  • 春天5快到了！Spring Framework负责人Juergen Hoeller刚刚宣布了Spring Framework 5.0。RC4 ，与Spring Boot 2.0一致。M4，JUnit 5和Reactor 3.1 RC1。这可能是您在大发布之前踢轮胎的最后机会！有这么多新的东西在这个版本！
  • 这很有趣！Spring Cloud孵化器中有一个Spring Cloud Config Server MongoDB项目 。试试看！
  • Spring Framework 忍者StéphaneNicoll刚刚宣布了Spring Framework 4.3.11 。
  • Spring Cloud和Spring Data忍者Mark Paluch刚刚宣布了Spring Cloud Vault 1.1.0。RC1 。此版本包括对AppRole身份验证和文档的新HTML样式。
  • Spring Data忍者Mark Paluch刚刚宣布了Spring Data Ingalls Sr7和Kay RC3的上市 。新版本都是推荐的更新。
  • 想更多地了解Spring Framework 5.0中的反应式支持。GA？请勿错过即将于10月4日举行的网络研讨会 ！
  • 我们的好友Matt Raible 在Okta的Java开发人员NoSQL选项博客上做了很好的综述。这篇文章主要是关于技术采用的非常有趣的讨论，但也确实强调了Spring支持哪些模块。
  • 在JUnit 5.0上很热门。GA，JUnit和Spring Framework的贡献者Sam Brannen刚刚更新了一个示例应用程序，以使用JUnit 5.0。看看这个！
  • 最后， JUnit 5现在到了 ！此发行版中有很多好东西，因此请务必查看发行说明 ！
  • Java平台的首席架构师Mark Reinhold 建议将Java迁移到可预测的且更频繁的发布周期 。在此处阅读有关此建议的更多信息。
  • 我喜欢这期Microsoft Azure Fridays，其中Sean McKenna向Microsoft传奇人物Scott Hanselman介绍了Cloud Foundry 。
