Spring Framework 5.0 RC4现在可用

发布
于尔根·霍勒（Juergen Hoeller）
2017年9月11日

尊敬的Spring社区，

我很高兴地宣布，我们的第四个也是最后一个Spring Framework 5.0候选 版本现已发布 ，它与Reactor 3.1 RC1以及昨天的JUnit 5.0 GA保持一致 ，并为即将到来的Spring Boot 2.0 M4奠定了基础！

Spring Framework 5.0 RC4还为Servlet 4.0 API的最终版本提供了一流的支持，与Hibernate Validator 6.0进行了微调的集成，并且与Jackson 2.9.1和Kotlin 1.1.4保持最新。因此，我们的依赖关系基线几乎是完整的，只有Reactor 3.1 GA和一些与JDK 9相关的更新（例如ASM 6.0）仍然可以进行。

接下来：计划于9月28日发布的Spring Framework 5.0 GA！
此后不久，我们将在10月4日发布网络研讨会来庆祝它。随时加入！

干杯，
于尔根
