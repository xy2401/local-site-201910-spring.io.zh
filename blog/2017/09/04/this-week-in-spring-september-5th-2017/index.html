春天的这个星期-2017年9月5日

工程
乔什·朗（Josh Long）
2017年9月4日

嗨，春季粉丝们，欢迎来到春季本周的另一期！本周，我来香港是为了惊艳的YOW！在香港举行的活动，然后将前往新加坡参加YOW！新加坡活动。如果您在任何一个地区，这些会议都是来自澳大利亚的世界级盛会，因此它们在该地区是新的，所以我希望您能尝试一下。

  • 查看有关Spring Cloud Data Flow各个方面的众多介绍中的第一篇。Spring Cloud Data Flow的忍者Eric Bottard的第一篇介绍着眼于Spring Cloud Data Flow shell本身。
  • Spring Cloud的贡献者Ryan Baxter刚刚宣布了Spring Cloud Edgeware M1 。新版本包括对各个项目以及新的Spring Cloud Gateway项目的更新。该版本基于Spring Boot1.5.x。
  • Spring Cloud联合创始人Spencer Gibb 刚刚宣布了Spring Cloud Finchely M2 。这个新版本基于Spring Boot 2.0.x，因此具有Java 8基线。它还在Spring Cloud Gateway中支持许多更酷的功能。在Spring Cloud Gateway和其他版本中，此版本的共同主题是响应式支持。
  • Spring Cloud Data Flow忍者Thomas Risberg在本文中介绍了Kubernetes的Spring Cloud Data Flow的Helm安装程序 。Helm是Kubernetes的软件包经理。
  • 这份有关应用程序现代化的 Pivotal和Apigee白皮书非常不错
  • 这是Matt McCandless今年早些时候的老歌，但又是个好东西，叫做Spring Boot：适合您的靴子
  • 这篇有关Spring Boot的Quartz Scheduler的文章是从过去到过去的文章。我喜欢。
  • Java 9 发行后，新Java版本的发布计划似乎有6个月的节奏 。我个人欢迎这个想法。
  • 我真的很喜欢这本俄语语言的帖子，其中介绍了Spring Boot的应用程序开发 。
  • 上周，Pivotal与Google和VMWare一起宣布了对Pivotal Cloud Foundry生态系统非常重要的下一步-Pivotal Container Service ，这是由BOSH管理的Kubernetes服务。 从这篇TechCrunch文章中了解我们的瘦身方式
  • 我喜欢关于将Spring Boot与SQL数据库（PostgreSQL），JPA和Liquibase集成的Auth0文章 。
  • JHipster的有关Doing API-First开发的帖子很有趣。
  • 带有响应性Spring Framework 5和Java 8基准的Spring Boot 2.0一定会给人留下深刻的印象。在新版本中，我最喜欢的功能之一是对Spring Boot Actuator端点的 REST堆栈不可知（反应式Spring WebFlux，Spring MVC或JAX-RS） 支持 ，此博客文章对此进行了详细介绍。
  • 观看这个出色的网络研讨会，了解IntelliJ IDEA为什么是Grails应用程序的首选 。我之所以喜欢它，是因为Grails 3是基于Spring Boot的，并且因为Jeff Scott Brown是一位出色的演讲者，并且因为IntelliJ制作了一个非常精明的优秀IDE。
  • 这并不是与Spring严格相关的，但值得一读。丹尼尔·布莱恩特（Daniel Bryant）在本文中介绍了一种全面的方法来测试微服务 -以及您可能在其中使用的所有组件。
  • 我喜欢Spring Data负责人Oliver Gierke的StackOverflow响应： 如何在Spring Boot中有选择地升级依赖项？
  • 如果您想快速介绍Cloud Foundry服务代理是什么，请阅读此文章 。
  • 从严格意义上讲，这也与Spring无关，但是对于许多构建云原生应用程序的Spring用户来说，这可能非常有用： Java Cloud Hosting Services的成本降低策略
