适用于Apache Kafka 2.0和1.3发行候选版本的Spring

发布
加里·罗素
九月12，2017

我们很高兴地宣布2.0.0的可用性。Apache Kafka的RC1春季发布候选2.0版。

如1.3.0中所述。M2公告 ，我们将同时发布1.3和2.0，其中1.3包含2.0功能的子集，支持Kafka 0.11.xx客户端，同时仍支持Spring Framework 4.3。因此，1.3.0。RC1发行候选版也可用。

可从Milestone存储库下载它们：

repositories {
    maven { url 'http://repo.spring.io/libs-milestone' }
}
compile "org.springframework.kafka:spring-kafka:2.0.0.RC1"

自上次发布以来，以下是添加的其他功能的摘要：

  • 支持Spring Framework的事务同步。

  • KafkaAdmin现在可用于自动添加主题（针对NewTopic应用上下文中存在的bean）。

  • AckMode.RECORD现在可以在侦听器返回后立即正确提交偏移量。侦听器容器中的新线程模型有助于实现这一点。

  • 若干错误修复

这些功能在1.3版本中也可用。
