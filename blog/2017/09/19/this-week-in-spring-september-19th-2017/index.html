﻿ 春天的这个星期-2017年9月19日 Spring Boot 1.5.7 which includes scores of fixes, dependency updates, and improvements.
  • Spring Integration lead and messaging ninja Gary Russell just announced Spring AMQP 2.0 RC1. The RC1 adds some minor improvements since the last milestone. Gary also announced that maintenance releases 1.7.4 and 1.6.11 are also available now.
  • Not one to sit on his laurels, Gary also announced Spring for Apache Kafka 2.0 and 1.3 release candidates. Both releases support the Kafka 0.11.x.x client, while still supporting Spring Framework 4.3. The new release includes support for Spring Framework’s transaction synchronization, a KafkaAdmin object, and correct handling of AckMode.RECORD. Also, these releases includes several bug fixes.
  • Spring Integration contributor Artem Bilan just announced Spring Integration 5. M7 and 4.3.12. The new release extracts out the reactive Spring Webflux adapter into a separate module, spring-integration-webflux. It also supports encoding message headers in the payload of a message for protocols - like AWS Kinesis, early versions of Apache Kafka, and TCP/IP - that don’t support the notion of headers natively.
  • Spring Integration lead Gary Russell just announcced Spring Cloud Stream Ditmars/1.3 RC1. The new supports working with Kafka Streams KStream references as inputs and outputs to @StreamListener methods. Put another way: you can use Spring Cloud Stream to connect your application, declare all the bindings and so on, just as normal, and then write messaging code that taps Kafka Streams for interactive queries.
  • Spring Boot committer Stéphane Nicoll just announced Spring Boot 2.0.0.M4. This release is the first to include the new web-runtime agnostic Actuator endpoints as well a raft of changes to simplify security configuration.
  • Spring Session lead Rob Winch just announced Spring Session 2.0.0.M4. This release eases configuration for Spring WebFlux-based applications. It also supports a strategy for session ID resolution.
  • Spring ninja Greg Turnquist just announced Spring Session MongoDB 2.0.0.M3. The new release builds on Spring Session 2.0.0.M4, Spring Data Kay RC3, Reactor Bismuth-M4, and Spring Framework 5.0.0.RC4. If you’re using Spring Boot 2.0’s spring-boot-starter-data-mongodb-reactive, then all you’ll need is @EnableMongoWebSession!
  • Spring Security lead Rob Winch just announced Spring Security 5.0.0.M4. This release is full of great new features including OAUth 2 / OpenID Connect support and a reactive integration for Spring WebFlux.
  • Spring web guru Rossen Stoyanchev just announced Spring Web Flow 2.4.6. The release includes a security fix that affects default bindings.
  • Spring Cloud Data Flow contributor Eric Bottard just announced Spring Shell 2.0.M1! The Spring Shell project surfaces a command component model to support developing custom shells. Spring Shell is the project that underpins, for example, the Spring Cloud Data Flow shell. This new release has a lot to offer including positional parameters, use of the Spring conversion API, JLine 3 support, integration with the bean validation API, multi-line commands, and much more.
  • Simon Baslé just announced Reactor 3.1 RC1. This is a huge step on the road to Spring Framework 5.0. Users are strongly encouraged to grab the bits and try things out ASAP!
  • Spring Cloud Data Flow contributor Glenn Renfro announced the 1.3.0.M2 release, packed with great new features. It includes an updated Dashboard/Flo integration supporting batch, task and streaming operations; fan-in and fan-out support; application registry and Maven update policies; security updates; direct named channels; shell improvements and much more.
  • Spring Boot ninja Madhura Bhave looks at the ways that Spring Boot 2.0.0.M4 improves its integration with Spring Security; this is definitely worth a read!
  • Tom Hombergs does a good job looking at messaging for microservices with Spring Boot and RabbitMQ.
  • Another friendly reminder that I’ll be speaking at the Reactive Summit event on Reactive Spring and I hope to see you there!
  • A friendly reminder that I’ll be speaking at Kotlin Conf on Spring Boot and Kotlin in December and I’d love to see you there.
  • Damith Ganegoda does a nice job introducing Spring Boot and Angular 4 in this blog post.
  • The Ordina JWorks blog has a great look at the new Micrometer monitoring abstraction (which is also integrated in Spring Boot 2.0.0.M4).
  • Atomist’s Clay McCoy looks at how Atomist can help you catch breaking REST API changes in your Spring Boot application.
  • I liked this post on ZDNet on Greenplum, big data’s best kept secret
  • Speaking of Spring Boot committer and all around amazing human Madhura BHave, she also put together a nice look at what’s coming in Spring Boot 2.0 for her JavaZone 2017 presentation.
  • I like Dan’s post that introduces the concepts of hypermedia and HATEOAS, in particular, and does so with some sample code based on Spring Boot and Spring HATEOAS.
  • Facebook and Github have just announced Atom IDE - a new IDE that, apparently, supports Java. I haven’t had a chance to try it yet but it looks interesting. I wonder if it’ll have Maven or Gradle support..

    春天的这个星期-2017年9月19日

    工程
    乔什·朗（Josh Long）
    2017年9月19日

    嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我去了Microsoft的Spring和Cloud Foundry团队（这次是在华盛顿州雷德蒙德），然后又去了加利福尼亚的旧金山和德克萨斯州的圣安东尼奥市拜访了一些客户。因此，让我们开始吧！

    • 我喜欢这个Pivotal Engineering帖子，内容涉及我们如何将Pivotal Web Services（我们的托管，多租户Cloud Foundry安装）过渡到CredHub 。CredHub旨在为BOSH管理的环境存储密码，密钥，证书和其他敏感信息。
    • Spring Boot忍者Brian Clozel刚刚发布了Spring Boot 1.5.7 ，其中包括许多修复程序，依赖项更新和改进。
    • Spring Integration主管和消息传递忍者Gary Russell刚刚宣布了Spring AMQP 2.0 RC1 。自上一个里程碑以来，RC1进行了一些小的改进。加里还宣布维护版本1.7.4和1.6.11现在也可用。
    • 盖瑞（Gary） 还没有宣布获奖， 还宣布了适用于Apache Kafka 2.0和1.3发行候选版本的Spring 。两种版本都支持Kafka 0.11.xx客户端，同时仍支持Spring Framework 4.3。新版本包括对Spring Framework事务同步的支持， KafkaAdmin对象，并正确处理AckMode.RECORD 。此外，这些版本还包括一些错误修复。
    • Spring Integration的贡献者Artem Bilan刚刚宣布了Spring Integration 5。M7和4.3.12 。新版本将反应式Spring Webflux适配器提取到一个单独的模块中， spring-integration-webflux 。它还支持在消息的有效负载中对协议的消息头进行编码，例如AWS Kinesis，Apache Kafka的早期版本以及TCP / IP，这些协议本身并不支持头的概念。
    • Spring Integration负责人Gary Russell刚刚宣布了Spring Cloud Stream Ditmars / 1.3 RC1 。新的支持与Kafka Streams一起使用KStream参考作为输入和输出@StreamListener方法。换句话说，您可以像往常一样使用Spring Cloud Stream连接应用程序，声明所有绑定等，然后编写利用Kafka Streams进行交互查询的消息传递代码。
    • Spring Boot的提交者StéphaneNicoll刚刚宣布了Spring Boot 2.0.0。M4 。此版本是第一个包含新的Web运行时不可知的Actuator端点以及大量更改以简化安全配置的版本。
    • Spring Session的负责人Rob Winch刚刚宣布了Spring Session 2.0.0。M4 。此版本简化了基于Spring WebFlux的应用程序的配置。它还支持会话ID解析策略。
    • 春季忍者Greg Turnquist刚刚宣布了Spring Session MongoDB 2.0.0。M3 。新版本建立在Spring Session 2.0.0之上。M4，Spring Data Kay RC3，Reactor Bismuth-M4和Spring Framework 5.0.0。RC4。如果您使用的是Spring Boot 2.0 spring-boot-starter-data-mongodb-reactive ，那么您所需要做的就是@EnableMongoWebSession ！
    • Spring Security主管Rob Winch刚刚宣布了Spring Security 5.0.0。M4 。此版本具有许多新功能，包括OAUth 2 / OpenID Connect支持和Spring WebFlux的反应性集成。
    • Spring网络专家Rossen Stoyanchev刚刚宣布了Spring Web Flow 2.4.6 。该版本包括一个安全修复程序，它会影响默认绑定。
    • Spring Cloud Data Flow的贡献者Eric Bottard 刚刚宣布了Spring Shell 2.0。M1 ！Spring Shell项目显示命令组件模型以支持开发自定义Shell。Spring Shell是支撑Spring Cloud Data Flow Shell的项目。这个新版本提供了很多功能 ，包括位置参数，Spring转换API的使用，JLine 3支持，与bean验证API的集成，多行命令等等。
    • SimonBaslé刚刚发布了Reactor 3.1 RC1 。这是迈向Spring Framework 5.0的重要一步。强烈建议用户抢先体验，并尽快尝试！
    • Spring Cloud Data Flow贡献者Glenn Renfro 宣布了1.3.0。M2版本 ，具有许多新功能。它包括更新的仪表板/ Flo集成，支持批处理，任务和流操作；扇入和扇出支持；应用程序注册表和Maven更新策略；安全更新；直接命名频道；外壳改进等等。
    • Spring Boot忍者Madhura Bhave着眼于Spring Boot 2.0.0的方式。M4改进了与Spring Security的集成 ；这绝对值得一读！
    • Tom Hombergs 用Spring Boot和RabbitMQ很好地研究了微服务的消息传递 。
    • 另一个友善的提醒是，我将在Reactive Spring 的Reactive Summit活动上发言，我希望在那里见到你！
    • 谨在此提醒您，我将于12月在Spring Boot和Kotlin的Kotlin Conf上演讲，我很想在这里见到您。
    • Damith Ganegoda在这篇博客文章中很好地介绍了Spring Boot和Angular 4 。
    • Ordina JWorks博客对新的Micrometer监视抽象 （也集成在Spring Boot 2.0.0中）进行了很好的介绍 。M4）。
    • Atomist的Clay McCoy着眼于Atomist如何帮助您在Spring Boot应用程序中捕获突破性的REST API更改 。
    • 我喜欢Greenplum上ZDNet 上的这篇帖子，大数据的秘密保密
    • 在谈到Spring Boot提交者以及令人惊叹的人类Madhura BHave时，她还 对JavaZone 2017演示文稿中的Spring Boot 2.0进行了很好的介绍 。
    • 我喜欢Dan的帖子，其中特别介绍了超媒体和HATEOAS的概念，并使用了一些基于Spring Boot和Spring HATEOAS的示例代码 。
    • Facebook和Github刚刚宣布了Atom IDE-一种显然支持Java的新IDE。我还没有机会尝试，但是看起来很有趣。我想知道它是否将支持Maven或Gradle。
