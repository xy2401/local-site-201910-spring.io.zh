Spring Framework 5.0正式发布

发布
于尔根·霍勒（Juergen Hoeller）
九月28，2017

尊敬的Spring社区，

我很高兴地宣布，在经历了超过一年的里程碑和RC以及总体上将近两年的开发之后，Spring Framework 5.0最终终于以5.0.0的形式提供。从repo.spring.io和Maven Central发布！

新一代的框架已准备好在2018年及以后推出：支持JDK 9和Java EE 8 API级别（例如Servlet 4.0），以及与Reactor 3.1，JUnit 5和Kotlin语言的全面集成。最重要的是，Spring Framework 5带有许多功能性的API变体，并在基于Servlet的Web框架Spring MVC的修订版之后引入了称为Spring WebFlux的专用响应式Web框架。

干杯，
