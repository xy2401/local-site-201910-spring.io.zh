春季中的本周-2017年9月26日（Java 9版）

工程
乔什·朗（Josh Long）
九月26，2017

嗨，春季粉丝！这是多么疯狂的美好一周！我回到了旧金山，与客户和本地合作伙伴讨论了Pivotal的所有方面，并且还享受了使用Java 9的同时在San Francisco Sun享受一些乐趣。在这个新发行版中有太多令人喜欢的东西，也有很多值得期待的东西，当然，Spring Framework 5是开箱即用的Java 9。

  • Spring IO Platform负责人Andy Wilkinson刚刚宣布了Spring IO Platform Brussels SR5 。新版本包括Spring AMQP 1.7.4，Spring Boot 1.5.7，Spring Data Ingalls SR7，Spring Framework 4.3.11，Spring Integration 4.3.12，Spring Loaded 1.2.8和Spring Web Flow 2.4.6。在这个新版本中有很多值得一看的地方，所以现在就来获取！
  • Spring Tool Suite的负责人Martin Lippert刚刚发布了有关如何在macOS High Sierra 10.13上运行STS时如何克服一些问题的信息。
  • Spring Cloud Task主管Michael Minella刚刚宣布了Spring Cloud Task 1.2.2。发布 。新版本增加了对Spring框架的支持，最近增加了对使用MySQL时序列表的MYISAM以外的数据库引擎的支持。
  • 多年来，经典的Spring PetClinic发生了很大变化！此版本演示了带有Spring和Kotlin的PetClinic 。如果您曾经看过任何原始的PetClinics（其中许多都早于Spring本身），这是一个非常简洁的小应用程序。
  • 是否需要Spring Boot功能的概述 ？该分析报告可能正是医生所订购的！
  • Spring Cloud Pipelines现在也支持Kubernetes
  • Oracle发布了Java 9！这是一个大问题！在Java 9中尝试有很多好处。如果您不使用Java 9，请尝试一下 。Spring Framework 5可以在Java 9（模块路径和类路径）上正常运行。
  • Alex Falappa在Twitter上：“下一个NBSpringBoot插件：用于检测和修复丢失的pom依赖项的Java编辑器提示。https://t.co/n7w22q4sMK”
  • 最近，有人问我一个示例，该示例在Cloud Foundry的Spring Boot应用程序中演示mTLS身份验证。我不知道，就像我经常做的那样，我问过Spring Security负责人Rob Winch，他分享了Java Cloud Foundry经验负责人Ben Hale的这个惊人的例子 。很有意思！
  • 我喜欢与Pivotal现场CTO Josh McKenty一起在云和Cloud Foundry上做的播客
  • 新的RebelLabs开发人员生产力报告已经发布 ，它显示了Spring的持续强劲增长。Spring有多个用户，占46％，而Java EE占33％。它还显示了Java 8的惊人增长，其中有72％的响应者使用了它。祝好消息都好起来！
  • 新的Microsoft和新的Microsoft SQL Server 2017版本已在这里，可以在Linux上运行！我从没想过要过一天。这和Spring有什么关系？绝对没有。只是..真的很酷！
  • 您是否在Spring Framework 5中的功能性Web框架上看到了响应式Spring Ninja Arjen Poutsma的今年7月网络研讨会？
  • 说到反应式 ，您可能还会喜欢Evgeny Poberezkin 关于反应式编程的发展的演讲。
  • Oracle有很多内容可供选择，它们引入了Java 9的新功能 。看看这个！
  • Spring社区的朋友Michael Simons的新书《 Spring Boot》现已推出 。看看这个！
  • Camunda BPM系统的最新版本具有Spring Boot集成 。
  • .NET忍者Richard Seroter说Java和Spring Cloud用户不应该拥有所有乐趣：学习如何使用Pivotal的Steeltoe框架将断路器集成到.NET应用程序中 。
  • Loom Systems的外观结合了Pivotal Cloud Foundry和Pivotal容器服务（基于Kubernetes ）。
  • Jenn Strater-Spring REST Docs的专家，除其他外，他组织了一个不错的演示，介绍了测试驱动的文档 （就Spring REST Docs而言，是natch）。
  • IBM最近开放了其J9 JVM的源代码。 这篇文章对J9背后的特殊历史提供了一些见识。
  • 这篇关于API文档的十个好主意的帖子，与Spring没有特别的关系，但是似乎确实在强化以下内容：Spring REST Docs是一个好主意（TM）！
