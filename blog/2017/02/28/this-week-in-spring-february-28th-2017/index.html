Spring 的这个星期-2017年2月28日

工程
乔什·朗（Josh Long）
2017年2月28日

欢迎来到 Spring 本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！我的朋友肯尼·巴斯塔尼（Kenny Bastani）和我将在3月1日和2日为O'Reilly进行培训，介绍Spring，Spring Boot，Spring Cloud和Cloud Foundry的所有内容 -加入我们！

期待已久的Spring Cloud Services公开测试版现已在我们的公共Cloud Foundry PIvotal Web Services上发布 ！

  • Spring Web忍者Rossen Stoyanchev刚刚宣布了Spring 5的M5里程碑版本，并完整地支持WebClient测试机制
  • Eric Bottard刚刚宣布了适用于Cloud Foundry 1.1.1的Spring Cloud Data Flow，其中包括稳定性改进，更好的错误处理，网络调用优化和更好的负载性能特征
  • 反应堆项目负责人Stephane Maldini刚刚宣布了反应堆铝SR1
  • Pivotal自己的Chris Sterling解释了Cloud Foundry的新容器网络堆栈及其如何应用于基于Spring Cloud的应用程序 。
  • 上周，我继续了Spring Tips系列，我研究了如何将Spring Boot与Apache MyBatis结合使用，Apache MyBatis是在中国的超级商人阿里巴巴这样的高性能组织中常用的技术。
  • Spring Boot团队的Madhura Bhave和Phillip Webb 在DevNexus上发表了精彩的演讲，揭露了Spring Boot的内部运作方式 。
  • 我喜欢Adib Saikali 在令人惊叹的Toronto Spring用户组中使用Spring Cloud Contract来查看消费者驱动的合同
  • 这并非特定于Java或任何特定技术， 而是有关如何对数据库索引进行单元测试的有趣文章 。请务必也阅读评论。
  • 有人指出我在Java 8上的这篇文章CompletableFuture另一天。这是老歌，但又是好东西 。
  • 您是否正在使用Neo4j，并希望向Neo4j浏览器添加一些细节？看看Neo4j自己的迈克尔·汉格（Michael Hunger）创作的Spoon这个项目
  • 我喜欢这个视频，它介绍了 Apache NetBeans中Spring Initializr的一些亮点
  • 克里斯·理查森（Chris Richardson）和肯尼·巴斯塔尼（Kenny Bastani）在几周前就分布式系统的最终一致性发表了精彩演讲，现在可以在线获取。
  • Spring Data忍者Mark Paluch进行了精彩的演讲，介绍了Hashicorp的Vault的秘密管理，该演讲现已在线发布
