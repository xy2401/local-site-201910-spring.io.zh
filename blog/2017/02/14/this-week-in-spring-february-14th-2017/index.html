春天的这个星期-2017年2月14日

工程
乔什·朗（Josh Long）
2017年2月14日

欢迎来到春季本周的另一期！ 今天是情人节，祝您情人节快乐！这周我将在田纳西州的孟菲斯进行商务活动，然后去圣路易斯，在这里我将与我的好友Mark Heckler在Reactive Spring上演出（加入我们！） ，然后下周去佐治亚州的亚特兰大。 DevNexus大展 。看看Pivotal在活动中做了什么！

和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • 上周，在另一期Spring Tips中，我研究了Spring Cloud Sleuth和Zipkin的分布式跟踪
  • Spring Cloud Task主管Michael Minella刚刚宣布了Spring Cloud Task 1.2.0。M1
  • Marcin Grzejszczak刚刚宣布了Spring Cloud Pipelines 1.0.0。M3完全支持Cloud Foundry上的蓝绿色部署
  • 在Spring Data Hazelcast存储库上查看此SpringOne平台2016演讲
  • 在Pivotal Web Services上有一篇关于Spring Boot Actuator支持的好文章
  • 关于Hadoop生态系统中的实时数据流的S1P演讲看起来很有趣
  • Oleg Zhurakousky通过此S1P演讲很好地了解了如何将Apache NiFi与Spring结合使用
  • 我们的朋友Andreas Evers 在这次S1P演讲中很好地介绍了如何使用Spring REST Docs编写综合文档
  • RedHat的Christian Posta关于使用Spring Cloud和Kubernetes进行现实微服务的S1P演讲看起来很有趣
  • 想更多地了解Spring Cloud Data Flow？查看此S1P演讲， Spring Cloud Data Flow：统筹一切
  • Target刚刚将Winnaker的审核工具开源到Spinnaker。Spinnaker是Netflix的Spring Boot支持的连续交付平台。Pivotal，Netflix，Google和其他许多公司都是Spinnaker的贡献者。您也可以在Cloud Foundry上运行Spinnaker
  • 我认为这次对迈克尔·米内拉 （ Michael Minella）的采访是关于短暂的微服务任务的，这是一个很好的聆听
  • hashicorp刚刚推出了他们的Vault Cloud Foundry服务代理
  • 查看451Research对Spring Cloud Data Flow的了解
  • 本讲座将探讨Instagram如何使用RabbitMQ大规模处理异步任务
  • 克里斯·理查森（Chris Richardson）更新了他的微服务模式门户，使其包括新的模式 - 服务部署平台 ， 按子域 分解，按业务能力分解和外部化配置 。
  • 兰迪·肖普（Randy Shoup） 在这次采访中对自动驾驶团队进行了有趣的介绍
  • 是否想使用Java Server Faces和Spring Boot ？
  • Baeldung博客很好地介绍了如何创建可执行.jar应用程序 。它研究了各种选项，包括Spring Boot fat- .jar方法。
  • 我们的朋友尼古拉斯·弗兰克尔又回到了一遍，这次一起来看一下如何实现一个自包含.jar即签署 。
