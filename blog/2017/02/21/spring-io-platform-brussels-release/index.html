Spring IO Platform布鲁塞尔发布

发布
安迪·威尔金森
2017年2月21日

我很高兴地宣布，Spring IO平台Brussels-RELEASE现在可以从Spring版本库和Maven Central中获得。

布鲁塞尔版本向平台引入了以下项目：

  • 春天卡夫卡1.1

布鲁塞尔版本还升级了许多项目的版本：

  • 春季AMQP 1.7
  • 春季靴1.5
  • Spring Data 摘要
  • Spring Integration Java DSL 1.2
  • Spring Integration Kafka 2.1
  • Spring LDAP 2.3
  • 春季安全4.2
  • 春季会议1.3
  • Spring Web Services 2.4

许多第三方依赖项的版本也已更新。
