反应堆Aluminium-SR1已着陆

发布
斯蒂芬·马尔蒂尼（Stephane Maldini）
2017年2月22日

批量发布的更新已经到来，并且随新版本一起提供给您Aluminium-SR1 材料清单 。赶紧发布Spring Framework M5 ！

堆芯3.0.5

发行说明

具有重要功能的重要质量更新，包括基于谓词的窗口和checkpoint操作员，修复程序，测试范围以及3.0.x版本中的第一次参考指南草案 。

我们现在正在努力3.0.6.RELEASE这将是我们之前的最后一次主要3.0内容更新3.1.0.RELEASE 。我们鼓励我们的Spring社区只要遵循任何已弃用的说明，请遵循这些说明@Deprecated API，因此可以在时间到时轻松快速地进行升级。请查看我们的问题积压 ，以获取更详细的范围。

反应堆插件3.0.5

发行说明

已经进行了改进reactor-test通过后验证断言支持丢弃的数据/异常测试。附加组件与反应堆堆芯紧密耦合，我们始终与它的发布保持同步，并在两者之间隔离范围。

我们现在正在努力3.0.6.RELEASE 。我们将至少包括一个新的附加组件（* reactor-math *），并继续进行抛光测试支持。还需要进行其他正在进行的研究，以让公共运营商TCK能够为撰稿人或其他图书馆开发人员提供一种理智的方式，以面对许多排序/优化方案强调其实施。

反应堆净值0.6.1

发行说明

一个推荐的更新，解决了自从引入客户端连接池模式以来面临的大多数问题。 0.6.0.RELEASE默认情况下启用。新功能包括NettyContext连接池中的管道运算符和代理支持。

我们现在正在努力0.6.2.RELEASE 。SPI中计划了许多改进，并且对质量的关注当然也会带来很多修复。我们的目标是根据我们从Cloud Foundry Java Client和Spring Framework团队获得的持续反馈，涵盖和审查用例。
