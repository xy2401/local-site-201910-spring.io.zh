SpringOne Platform 2016重播：PayPal的Spring Boot

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
一月23，2017

记录在SpringOne Platform 2016中。
演讲嘉宾：Fabio Carvalho，Eduardo Solis
幻灯片： http ： //www.slideshare.net/SpringCentral/spring-boot-paypal

尽管PayPal在生产中使用Spring已有多年，但作为迈向微服务的目标之一，PayPal最近将Spring Boot引入了Java框架（称为“ Raptor”），该框架包括RESTful服务，Batch和Message Daemon。应用程序。除了Spring Boot，还使用其他Spring项目，例如Spring框架，Spring Batch和Spring Integration。

PayPal Java RESTful框架提供了微服务架构和部署模型，它依赖于云，CI，Docker，嵌入式容器和Spring Boot上的PaaS基础架构。此外，开发了许多Spring Boot启动器和基于Spring的功能来满足PayPal的需求。

一些示例是：

REST客户端抽象和DI机制
中央应用程序日志记录（在将流量记录到RESTful端点时，抽象URI和支持子资源）
在Spring Integration通道中将应用程序记录到异步活动（用于消息传递应用程序）
一些Spring Boot入门程序，用于自动初始化和配置“有条件的” PayPal组件
RESTEasy Spring Boot启动器
关于REST服务，RESTEasy是PayPal的JAX-RS选择。尽管对于Jersey来说有一个Spring Boot启动器，对于Spring REST有一个启动器，但是RESTEasy没有。因此，贝宝还创建了RESTEasy Spring Boot启动程序并将其开源。
