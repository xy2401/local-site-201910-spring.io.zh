SpringOne Platform 2016重播：设计，实现和使用响应式API

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
一月16，2017

记录在SpringOne Platform 2016中。
演讲者：-本·黑尔，保罗·哈里斯，斯蒂芬·马尔蒂尼
幻灯片： http ： //www.slideshare.net/SpringCentral/designing-implementing-and-using-reactive-apis

Java社区正处于编程模型重大变革的风口浪尖。随着业界朝着高性能微服务架构的方向发展，对反应式编程模型的需求变得显而易见。在本次会议中，Cloud Foundry Java Client的主要开发人员将讨论导致他们选择反应式API的原因。他们将以该项目为视角，探索如何使用Project Reactor设计和实现此API，以及用户在使用反应式API时会有什么期望。如果您是寻求提供反应式API的开发人员，那么这是您获得团队构建大型的，可用于生产的反应式库的经验。
