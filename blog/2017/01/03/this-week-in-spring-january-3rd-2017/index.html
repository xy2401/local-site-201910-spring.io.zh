春天的这个星期-2017年1月3日

工程
乔什·朗（Josh Long）
一月03，2017

欢迎来到春季本周的另一期！我们现在正处于2017年，而且一如既往地为0-有很多需要覆盖的地方！

这也是一年中的第一周。我为此感到自豪，因为它也标志着2011年1月春季本周的第一期发行六周年，此后的每个星期二我都会无休止地生病，度假，旅行或其他方式。这是一次heckuva比赛，我很高兴看到接下来的6年多的春天在本周将有什么收获！我永远感谢多年来为春季本周做出贡献的所有人员（从社区到内部，从VMWare到现在的Pivotal）！在过去3年多的时间里，我的朋友和Pivotal自己的Pieter Humphrey为实现这一目标提供了惊人的帮助，因此，感谢Pieter！

像往常一样，如果您希望在春季的“本周”中看到更多内容，请随时在Twitter上 ping 我或通过电子邮件 ping 我 。

现在，让我们开始吧！

  • Spring Framework负责人和联合创始人Juergen Hoeller 刚刚宣布了Spring Framework 5.0 M4
  • 上周，由于我正在休假，所以我没有发布新的Spring Tips视频（明天也不会），（ 春季除外）！但是我确实查看了每个用户应该知道的资源，以便在Spring上找到有关所有内容的教育内容
  • Spring Integration和消息传递忍者Artem Bilan刚刚宣布了Spring AMQP 1.7RC1 ，其中包括对Amqp Client 4.0的新支持，对Log4j 2.7的更新，Spring Retry 1.2， spring-rabbit-junit为了方便的规则，例如BrokerRunning并且支持配置连接如何从代理本身出现。
  • 如果您想知道如何扩展Spring Boot，Ordina / JWorks的社区贡献者Dieter Hubau就在SpringOne Platform 2016上编写自己的Spring Boot Starter进行了精彩演讲。
  • 格雷格·特恩奎斯特（Greg Turnquist）在本SpringOne Platform 2016重播中谈到了Spinnaker的连续交付 。看完之后，请务必在使用时查看Spring Cloud Spinnaker的最新M3版本 ！
  • 有没有想过为什么Spring on Cloud Foundry会带来如此流畅的体验？还是Groovy？还是Grails？观看SpringOne摇滚明星发言人和buildpack团队负责人Ben Hale在其SpringOne Platform重播中谈论Cloud Foundry Java Buildpack的更新 。Ben在诸如内存计算器之类的惊人功能的基础上，为您提供了有关为什么buildpack提供当今云上最佳Java SE支持的最新信息。
  • Christoph Strobl在SpringOne Platform 2016演讲中向我们介绍了Spring Data Redis的最新知识。
  • Eitan Suez在此次SpringOne Platform 2016重播中解释了Apache Geode如何适应CQRS架构 。
