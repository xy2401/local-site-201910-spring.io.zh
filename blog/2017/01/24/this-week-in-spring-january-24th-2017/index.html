春天的这个星期-2017年1月24日

工程
乔什·朗（Josh Long）
一月24，2017

欢迎来到春季本周的另一期！像往常一样，我们需要处理的事情很多 ，所以让我们开始吧：我们已经从团队中获取了一批新的Spring Cloud软件和本周Pieter Humphrey播放的以微服务为主题的S1P 2016回放，因此请系紧安全带！

  • Spring Cloud Data Flow产品经理Sabby Anandan 在Spring Cloud Data Flow 1.1版本中发表了精彩的文章
  • Cloud Foundry负责人Eric Bottard的 Spring Cloud Data Flow 刚刚发布了最新版本 ， 该版本支持内存优化，Maven缓存和磁盘优化，改进的错误处理以及任务支持。
  • Spring Cloud Stream负责人Marius Bogoevici刚刚宣布了布鲁克林Spring Cloud Stream。SR2和切尔西。M1
  • Spring Integration Java DSL负责人以及伟人Artem Bilan刚刚宣布
    用于Hazelcast 1.0.0.m2的Spring集成扩展 。此版本支持领导者选举，出站适配器，集群监视等。
  • Spring Security负责人Rob Winch刚刚宣布了Spring LDAP 2.3.1 ，由于早期版本中发现了一个问题，它是Spring LDAP 2.3.0的快速跟进。
  • Spring Integration的负责人Gary Russell刚刚宣布了Spring AMQP 1.7.0。发布和1.6.7。发布 。此版本包括对AMQP 4.0客户端的升级，更新的Log4J支持，新的spring-rabbit-junit模块，在运行时动态注册队列的功能以及众多其他功能，这些功能将集成到即将推出的Spring Boot 1.5中！
  • Spring Integration 大师Artem Bilan-从不睡觉！ -也刚刚宣布了针对SMB 0.5.0的Spring Integration Extension。
  • 我真的很喜欢实现Config Server，并从Domo的Clint Checketts 扩展了它 ，并预测它将成为非常流行的S1P重播。
  • Netflix的Tomas Lin将CD和表达语言整合在一起： 在Spornaker上放置SpEL：发展表达语言以实现连续交付
  • 玛丽·安·韦尔（Mary Ann Wayer）和特拉维斯·樱桃（Travis Cherry）在2016年SpringOne平台重播中向Premier，Inc展示了一个过渡故事： 从整体架构过渡到Spring Cloud和微服务
  • Adrian Cole在此SpringOne Platform 2016重播中介绍了分布式跟踪和Zipkin简介 。
  • Marcin Grzejszczak和Reshmi Krishna在此S1P 2016重播中演示了分布式跟踪： 使用Spring Cloud和Zipkin实现微服务跟踪
  • 您是否针对常见的AWS服务（例如SQS，ElastiCache，SNS，CloudFormation）进行编码，
    RDS，S3？Mimacom的Agim Emruil向我们展示了如何在此SpringOne Platform 2016重放中从惯用的Spring构造中访问它们： AWS上的Spring Cloud
  • 克里斯·理查森（Chris Richardson）在本SpringOne Platform 2016重播中专注于微服务设计基础和事件源： 使用聚合开发微服务
  • 来自PayPal的Fabio Carvalho和Eduardo Solis在此S1P 2016重播中介绍了基于Spring Boot的Raptor Web框架： PayPal的Spring Boot
  • 现在注册， 从波兰ZielonaGóra的一位大师Marcin Grzejszczak了解Spring Cloud对分布式跟踪的支持 。
  • 温馨提示， 用于Cloud Foundry的Spring Cloud Services 1.2具有多站点服务发现功能
  • 甚至RedHat都将Spring Cloud作为构建微服务的首选方法 。顺便说一下，本演讲中描述的项目已移至该孵化器存储库 。
  • 查阅这个不断增长的存储库，以了解有关Spring Cloud Function的工作原理草图。
  • 查看此社区推动的工作， 以支持Spring Cloud DiscoryClient Marathon对Apache Mesos进行抽象
  • 在Paradigma Digital博客上，EduardoGonzález发表了一篇精彩的西班牙语文章，内容涉及使用Spring Cloud和Zipkin进行分布式跟踪
  • 在Spring Cloud Config上查看墨西哥墨西哥城的JUG演讲
  • 本文通过设置一个简单的基于Spring Security和基于JPA的Spring Data的应用程序进行了逐步介绍
  • 看看Hazelcast惊人的Viktor Gamov，因为他介绍了如何在Bootiful Hazelcast中将 Hazelcast与Spring Boot 结合使用
  • R. Pidugu对构建Spring Boot和基于JPA的Spring Data的应用程序进行了很好的介绍
  • 我们在JDriven博客上的朋友将他们的Spring Sweets系列放到了另一期中， 这次探讨如何将元数据添加到Spring Boot Actuator中/info终点
