SpringOne Platform 2016重放：Spring数据和内存中数据管理的实际应用

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
一月10，2017

记录在SpringOne Platform 2016中。
主讲人：约翰·布鲁姆（John Blum），卢克·香农（Luke Shannon）
幻灯片： http : //www.slideshare.net/SpringCentral/spring-data-and-inmemory-data-management-in-action-64852094

在本次会议中，我们将介绍并编写一个基于Spring Boot的实时应用程序，该应用程序由Cloud Foundry上运行的Apache Geode（又名Pivotal GemFire）提供支持。与会者将学习内存中的计算和数据管理概念，包括使用Spring数据存储库和GemFire OQL进行数据访问和查询，使用GemFire CQ进行复杂/实时事件处理，使用通过Spring Data GemFire函数注释方便地实现和执行的GemFire函数进行数据亲和力支持和最终有效的测试策略和技术，用于使用Spring的测试框架以及JUnit，Mockito和MultithreadedTC测试高度并发的分布式应用程序。
