春天的这个星期-2017年10月3日-Spring Framework 5.0版！

工程
乔什·朗（Josh Long）
十月03，2017

欢迎来到春季本周的另一期！上周真疯了 ！提供了很多好东西。Spring Framework 5.0现在可用！Spring Framework 5.0附带了大量的发行版，您将在本文中看到所有这些发行版。另外，本周，我-和Pivotal和Spring团队的其他成员-都在JavaOne上，希望您能找到我们，并在我们的展位或各种对话中打招呼 。

  • 上周，我研究了使用Spring Framework 5的支持来构建基于Websocket的响应式应用程序 。
  • 春天AMQP与Spring集成导致加利罗素刚刚宣布春季AMQP 2.0。RC2 。
  • 最后，Spring Framework 5.0。GA在这里！新版本与Project Reactor集成在一起，包括反应式Web运行时，Kotlin扩展以及Java EE 7和Java 8的基准。新版本有太多新事物，希望在这里捕获，因此请查看发布博客 ，然后查看“新功能”文章
  • Spring和Reactor忍者SimonBaslé刚刚宣布发布Reactor Bismuth 。这个新版本为Spring Framework 5.0奠定了基础，并且本身也包含许多功能！
  • Spring Framework 5.0版本为许多其他项目打开了大门，包括Spring Data Kay 。这是自2009年Spring Data推出以来对Spring Data的最大修订！这个新版本的基准是针对Spring Framework 5.0，Java 8和Java EE 7。它包括修订的存储库API（对Optional完整支持Optional ），对反应式数据访问的支持（Cassandra，Couchbase，MongoDB和Redis），在发布系列中添加了Spring Data Geode，使用了可空性注释和高级运行时检查，支持Kotlin对空安全性和不可变数据类的支持通过Kotlin构造函数，Java 9兼容性等等！
  • Spring消息传递和集成忍者Artem Bilan 刚刚宣布了适用于Apache Kafka 2.0的Spring。GA 。新版本包括@KafkaListener和Consumer解析中的Apache Kafka支持，事务支持，标头映射，Apache Kafka Streams支持，新的KafkaAdmin ，错误处理和组支持。它还支持@EmbeddedKafka进行测试。
  • Spring Cloud忍者Ryan Baxter刚刚宣布了Spring Cloud Dalston SR4 。新版本更新了Spring Cloud Contract，Spring Cloud Config，Spring Cloud Commons，Spring Cloud Netflix和Spring Cloud Sleuth。
  • Spring REST Docs领导Andy Wilkinson 刚刚宣布了Spring REST Docs 1.2.2。发布 。此维护版本包括一些错误修复和文档改进，因此建议进行升级
  • 查看Spring Framework 5的新的按构造器自动布线的实现
  • 我喜欢Andreas Falk的德语采访，内容是Spring Framework 5.0和Spring Security 5.0的新功能 。
  • Zoltan Altfatter的这篇文章探讨了如何通过Spring Integration流中的传入JMS消息来启动Spring Batch Job 。
  • Rohit Kelapure研究了为什么在这个史诗般的Twitter线程中Pivotal的Cloud Foundry 是运行Spring Boot应用程序的最佳场所 。
  • 在本文中，Cristina Negrean着眼于使用Spring Cloud Data Flow进行实时分析 。
  • Gabriela Motroc 在新的Spring Framework 5版本中为JAXEnter撰写了一篇不错的文章。有太多喜欢的东西。
  • Aboullaite Mohammed撰写了一篇有关使用Elasticsearch和Kibana和Spring Boot监控指标的非常好的文章
  • Ordina JWorks博客上有一篇很好的文章，介绍了如何使用Spring Cloud保护微服务
