Spring Data Ingalls SR8发布

发布
奥利弗·德罗博姆
十月11，2017

我谨代表Spring Data团队宣布发布火车Ingalls的第八次服务发布。该版本在刚刚发布的Spring Framework 4.3.12之上发布，并为即将发布的Spring Boot 1.5.8做准备。
