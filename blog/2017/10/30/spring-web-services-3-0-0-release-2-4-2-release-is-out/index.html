Spring Web Services 3.0.0。发布/ 2.4.2。发布了！

工程
格雷格·特恩奎斯特
十月30，2017

春季问候社区，

Spring Web Services已发布3.0.0，以实现一般可用性，并发布了次要补丁2.4.2。发布。

如前一篇博客文章所述，只要支持Spring Framework 4.x，就将维护2.x系列，但是任何新工作都将在master分支上进行。

以下链接包括相关票证。

2.4.2版本说明 | 2.4.2文档 。

3.0.0。发行说明 | 3.0.0。发布文档 。

  • 3.0.0。发布：

    • 更好地处理大型附件以避免OutOfMemory异常。

    • Wss4jSecurityInterceptor增强功能包括SAML回调

    • 改进SpringSecurityPasswordValidationCallbackHandler以避免NPE缺少登录用户

    • 修复了SOAP 1.1 / SOAP 1.2消息头处理

    • 修复Axiom SOAP 1.2的内容类型

    • 升级到AspectJ的最新版本

    • 支持新兴的Axiom 1.3（希望确保Spring WS和Axiom 1.3协同工作的社区贡献者）。

  • 2.4.2。发布：

    • 反向移植相同的SpringSecurityPasswordValidationCallbackHandler修复

    • 不小心将wss4j 2.x版本向前移了太远，以确保稳定的兼容性。如果要使用最新的wss4j，则必须移至3.0版本。

我想强调指出，此版本在很大程度上是由社区参与推动的。实际上， 最新的3.0分支版本中有75％来自社区贡献者。

项目页 | GitHub | 问题

这些工件在Maven Central和http://repo.spring.io/上进行 。

重要
 我和大多数Spring团队的同事将参加Spring One Platform 。现在就注册！
