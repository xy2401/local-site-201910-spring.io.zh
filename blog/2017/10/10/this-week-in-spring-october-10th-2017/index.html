春天的这个星期-2017年10月10日

工程
乔什·朗（Josh Long）
2017年10月10日

嗨，春季粉丝！欢迎来到春季本周的另一期！碰巧的是，今天也是 Ada Lovelace的日子。根据Finding Ada网站的说法：Ada Lovelace日（ALD）是一个国际庆祝日，庆祝女性在科学，技术，工程和数学（STEM）中取得的成就。它旨在提高女性在STEM中的形象，并以此创建新的榜样，以鼓励更多的女孩进入STEM职业并支持已经在STEM中工作的女性。 Ada Lovelace日快乐！

我刚刚从波兰的弗罗茨瓦夫举行的史诗般的ByteMyCode会议中回来，现在我在华盛顿的西雅图与客户交谈，并[ 今天在Pivotal Seattle的Cloud Native Java上举行了公开活动 。今天晚些时候，我将去加拿大多伦多拜访客户，并参加本周晚些时候发生的SpringOne Platform Preview展览 。如果您在附近，请不要犹豫，打个招呼！

  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 5.0.0。M5 。新版本包含错误修复和新功能。该版本更新到Spring Framework 5.0.0。释放，反应堆铋释放和Spring Data Kay释放。
  • Spring Security主管Rob Winch刚刚宣布了Spring Session 2.0.0。M5 。新版本带来了Redis ReactiveSessionRepository实施。
  • 在上周的Spring Tips中 ，我研究了响应式Spring Security 5 。
  • Spring Integration忍者Artem Bilan刚刚宣布了Spring AMQP 2.0.0。发布 。这个新版本包括对Java 8和Spring Framework 5.0的全面支持。一个新的spring-rabbit-junit工件和一些有用的测试工具，包括BrokerRunning JUnit的@Rule ;的amqp-client版本5.0的库；一种DirectMessageListenerContainer允许直接在客户端线程上使用消息，而不是在客户端线程上使用基于队列的逻辑SimpleMessageListenerContainer ;一个新的RabbitOperations.invoke()用于在专用的线程绑定通道上进行批量模板操作；编码器对Logback的支持AmqpAppender ;大量的错误修复等等。查看发布！
  • Spring Data负责人John Blum 刚刚宣布了Spring Session Data Geode / GemFire 2.0.0。M2 。这个新版本将升级到Spring Framework 5.0.0，Spring Data Kay RELEASE，Spring Session 2.0.0。M4，Spring Boot 2.0.0。M4，它增加了Pivotal GemFire和Apache Geode PDX序列化支持。
  • Spring Cloud Vault负责人Mark Paluch 刚刚宣布了Spring Vault 1.1.0。GA和2.0.0。M3 。新的1.1.0版本包括对AuthRole身份验证的拉模式支持，使用AWS IAM的保管库登录，对批处理传输操作的支持，基于通用密钥的租用期限的轮换以及对密钥的介绍。 VaultEndpointProvider动态配置端点。2.0。M3包括基于以下内容的保管库@EnableVaultRepositories Spring Data KeyValue顶部的注释，它支持创建和修改以JSON表示的保管库策略。
  • Spring Cloud忍者Vinicius Carvalho 刚刚宣布了Spring Cloud Stream 1.3.0 。新版本包括对Kafka Streams的支持，对RabbitMQ的重试支持，惰性队列（包括DLQ支持），滞后指标（用于了解事物为什么变慢的原因）等等。看看这篇文章和嵌入式YouTube视频以了解更多！
  • Spring Ninja Stephane Nicoll刚刚宣布了Spring Framework 4.3.12 。这是一个维护版本，其中包含一些修复程序和选定的改进。
  • 我喜欢Dave Syer博士在OpenSlava 2017展会上关于无服务器之路的演讲 。很好地了解了Spring Cloud Function中的概念（和代码），真的很有趣。
  • Baeldung已使用Netflix Hystrix断路器更新了Spring Cloud集成的范围。
  • 看到RedHat拥抱Spring Boot真的很酷，但是有关使用Kubernetes ConfigMaps管理机密 （例如密码或credentals）的文章忽略了更强大（更不用说平台无关）的Spring Cloud Config Server 。
  • 我喜欢这篇MemoryNotFound博客文章，内容涉及将Spring Boot与ActiveMQ （或一般而言只是JMS）结合使用。
  • 或者非常有名的Sebastien Deleuze 刚刚使用KotlinX Coroutines项目更新了Project Reactor集成 。
  • 我喜欢这个著名的PetClinic示例的Spring Boot和Kotlin语言版本
  • KotlinX对Reactor的协程支持 -这是Spring Framework 5的Spring WebFlux本身的基础-反应库-已更新为Reactor Bismuth发布系列。
  • 我真的很喜欢Stephen Janssen对 Java 9或更高版本中的功能的了解。
  • TheNewStack 在这篇文章中介绍了Spring Framework 5中的反应式支持
  • Alex Nesterov研究如何将Kotlinx协程扩展用于基于Spring Framework 5的反应式代码 。这是一件有趣的工作。
  • Activiti 7的新版本是一个开源BPMN工作流引擎，它具有物料清单Maven工件以及Spring Boot / Spring Cloud就绪的启动器和集成，看上去很棒 。
  • 2017年Java-JetBrains开发人员生态系统状况
  • Ham Vocke撰写了一篇非常棒的关于测试微服务的文章 。我喜欢这篇文章，尽管我会使用Spring Cloud Contract而不是Pact。Spring Cloud Contract也可以支持Pact文件。
  • 在Ordina JWorks博客上的这篇精彩文章中，Dieter Hubau帮助将Rick（来自Rick＆Morty ）与令人惊叹的McDonald's Mulan Szechuan酱与Spring Cloud Stream连接起来 。
  • Thymeleaf的领导者Daniel Fernandez刚刚宣布Thymeleaf 3.0.8已发布 ！此版本中有很多新功能，但我最喜欢的是Spring 5集成模块的GA版本， thymeleaf-spring5 ，这使得以反应方式轻松使用Thymeleaf。
