    春季数据发布火车Kay前往通用航空

    工程
    奥利弗·德罗博姆
    十月02，2017

    我很高兴代表Spring Data工程团队宣布Spring Data Kay的全面上市。这是自Spring Data于2009年成立以来的第一个主要修订，因此包含了大量功能。最重要的是：

    • 升级到Spring Framework 5.0，Java 8和JavaEE 7作为基准
    • 修订的存储库API（更好的方法名称， Optional等等。）
    • 修订后的存储库组成模型
    • 支持对Cassandra，Couchbase，MongoDB和Redis的反应性数据访问
    • 将Spring Data Geode添加到发布系列中
    • 在其中使用可空性注释和高级运行时检查
    • Kotlin对Kotlin构造函数支持的null安全和不可变数据类的支持
    • 通用Java 9兼容性

    在策划的变更日志， 以前的博客文章 （以及在响应式位上甚至更早的文章 ）以及单个模块的参考文档的新增内容中查找更多信息。

    该版本将包含在即将到来的Spring Boot里程碑中，使我们有时间进行进一步改进，并为Boot 2.0的GA提供了第一个服务版本。除此之外，我们现在开始着手即将发行的发行火车Lovelace 。

    另外，请确保您不会错过今年的SpringOne Platform会议，该会议上充斥着与数据相关的讲座，有机会了解最新和最出色的功能，当然还有一些有关我们计划下一步工作的预览。折扣价即将用完！
