Spring Cloud Edgware。RC1发布

发布
斯宾塞·吉布
十月25，2017

我很高兴代表社区宣布， Spring Cloud Edgware发行培训的发行候选1（RC1）现已开始提供。该版本可以在Spring Milestone存储库中找到。您可以查看Edgware 发行说明以获取更多信息 。

Edgware发布系列中的显着变化

重命名启动器

许多入门者未遵循正常的Spring Cloud命名约定。在Edgware中，使用不建议使用的启动程序将记录一个警告，并使用新启动程序的名称代替它。下表列出了不推荐使用的启动器及其替代品

不推荐使用 Edgware入门
春天云启动器archaius Spring-cloud-starter-netflix-archaius
Spring-Cloud-Starter-Atlas Spring-Cloud-Starter-NetFlix-Atlas
春天云启动器尤里卡 spring-cloud-starter-netflix-eureka客户端
春天云启动器尤里卡服务器 Spring Cloud Starter NetFlix Eureka服务器
春天云启动器假装 春天云启动器openfeign
春天云启动器hystrix 弹簧云启动器netflix-hystrix
Spring Cloud Starter Hystrix仪表板 Spring-Cloud-Starter-NetFlix-Hystrix-仪表板
春天云启动器功能区 弹簧云启动器netflix功能区
春天云启动器观众 spring-cloud-starter-netflix观众
弹簧云启动涡轮 弹簧云启动器netflix涡轮
弹簧云启动涡轮amqp 已删除
弹簧云启动涡轮流 Spring-Cloud-Starter-NetFlix-涡轮流
Spring-cloud-starter-zuul Spring-Cloud-Starter-NetFlix-Zuul

Spring Cloud巴士

更新以允许Bus客户端在独立程序包中实现远程事件。

春季云任务

请参阅发行公告 。

Spring Cloud Netflix

@EnableDiscoveryClient现在是可选的。包括spring-cloud-starter-netflix在内的应用程序将自动假定该应用程序应该注册并成为发现客户端。要禁用自动注册，请设置spring.cloud.service-registry.auto-registration.enabled=false 。Hystrix和Eureka Server现在通过Spring Cloud Contract进行了测试。

Spring Cloud领事

Consul DiscoveryClient现在支持数据中心参数。现在支持带有自签名证书的HTTPS检查。

Spring Cloud合约

Wiremock已更新，现在支持自定义扩展。现在可以在正文响应验证中引用URL段。

Spring Cloud侦探

Zipkin 2支持已添加，并进行了许多次要增强。

春季云流

见发布公告 ;

Spring Cloud Config

创建了一个新的JDBC EnvironmentRepository 。

春季云动物园管理员

设置实例ID的功能已添加。

Spring Cloud Commons

如上所述， @EnableDiscoveryClient现在是可选的。
