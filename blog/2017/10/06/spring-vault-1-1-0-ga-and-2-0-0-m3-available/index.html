提供了Spring Vault 1.1.0 GA和2.0.0 M3

发布
马克·帕鲁奇
十月06，2017

我很荣幸地代表社区宣布Spring Vault 1.1.0的全面可用性和Spring Vault 2.0的第三个里程碑，分别来自Maven Cental和里程碑存储库。

Spring Vault 1.1 GA版本的主要特点是：

  • 对AppRole身份验证的拉模式支持。

  • 通过AWS IAM使用保管库登录。

  • 支持批量运输操作。

  • 根据通用密钥的租用期限进行轮换。

  • 介绍VaultEndpointProvider动态配置端点。

查看Spring Vault 2.0 M3，您会发现以下增强功能：

  • 保管箱存储库通过@EnableVaultRepositories建立在Spring Data KeyValue之上。

  • 支持创建和修改以JSON表示的保管库策略。

下一步是什么？

我们正在将开发方向转向2.x分支，从而使Spring Vault 1.1 GA成为1.x分支的最后一个次要版本。Spring Vault 1.1将看到维护版本，直到其使用寿命终止。我们将在下一个Spring Vault 2.0里程碑中完善响应式支持，这实际上是一个候选版本。

有关更改的完整列表，请参见1.1.0 RELEASE和2.0.0 M3的更改日志。
