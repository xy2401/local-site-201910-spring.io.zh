春天的这个星期-2017年10月17日

工程
乔什·朗（Josh Long）
十月17，2017

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我去过德克萨斯州的达拉斯和奥斯汀，首先去参加客户会议，然后去参加史诗般的Reactive Summit并发表演讲。如果您在附近，请在Twitter上打个招呼！ 我们有很多内容需要介绍，让我们开始吧！

另外，我将与我的朋友和Grails的联合创始人Jeff Scott Brown于2017年11月9日在OCI上的Grails for the Spring Developer上与我们的朋友进行网络研讨会。希望您能加入我们！

  • 上周，SpringSource的联合创始人兼Spring Framework的负责人Juergen Hoeller自从Spring项目开始以来所做的所有出色工作均获得了JAX特别评审团奖 。感谢他，特别评审团指出：“ Spring Framework 自存在以来的所有年份中，都不需要完全重写。相反，它不断适应不断变化的趋势。”感谢Juergen，感谢您所做的一切。
  • 上周，我研究了如何将Scala与基于响应式Spring Boot和基于Spring Webflix的应用程序一起使用，以及如何与其他兼容Reactive Streams的项目（如Akka Streams）进行互操作。
  • Spring Data负责人Oliver Gierke刚刚宣布了Spring Data Ingalls SR8 。该发行版包括许多改进和错误修复，是所有使用Ingalls发行版的人员的建议升级。
  • Spring Boot 负责人Phil Webb刚刚宣布了Spring Boot 2.0.0。M5 。新版本现已在Spring Initializr上提供 。它包括对Java 9的初步支持，Spring Security 5.0的OAuth 2客户端支持，JSON-B支持以及新的Spring Session Actuator端点。
  • Spring Tool Suite的负责人Martin Lippert刚刚宣布了Spring Tool Suite 3.9.1 。Eclipse Oxygen 1.a的新发行版更新包括JDK 9和JUnit 5，更新了Spring Boot Dashboard，以支持对Spring Boot发行版中正在运行的 bean的了解。最后一件事太酷了！还有一个重构要转换application.properties至application.yml ，当然还有许多错误修复和改进。
  • Spring NinjaStéphaneNicoll刚刚宣布Spring Boot 1.5.8现已可用。新版本包含40多个修复程序，改进和依赖更新。这是推荐的更新。
  • Spring Web Services负责人Greg Turnquist 刚刚宣布了Spring WS 2.4.1和3.0.0。RC1 。2.4.1有一些小补丁，其中3.0.0。RC1有许多重大升级。Java 8和Spring Framework 5.0的新发行基准。如果您要构建基于SOAP的服务，那么这是给您的！
  • 您是否看到了针对Google Cloud Platform的新Spring Cloud集成 ？这里有很多值得一提的地方 ，包括与核心GCP功能（如项目ID和运行时配置API）的集成，Google Cloud Pub / Sub，Google Cloud SQL，Google Cloud Storage以及Spring Cloud Sleuth与Google Stackdriver的集成！
  • Spring Data的负责人Oliver Gierke在此播客中接受了一个非常不错的采访，内容涉及从整体到系统的转变。绝对值得一听！
  • Judy Wang很好地了解了Pivotal Cloud Foundry的“ PCF指标”如何以更好的洞察力缩短平均恢复时间 。
  • Spring Framework的创始人兼Atomist的首席执行官Rod Johnson在这篇有趣的博客文章中介绍了Atomist的自动化开发的广阔视角 。Atomist提供了许多对Spring开发人员非常有用的工具。
  • 我们自己的Kenny Bastani 在这个示例应用程序中着眼于将事件驱动的体系结构与各种基于Spring的技术结合使用
  • 我喜欢韦德兰·帕维奇（VedranPavić）的幻灯片， 从零到开源英雄：对春季项目的贡献
  • Spring Framework 的创始人兼Atomist首席执行官Rod Johnson在此博客文章中围绕Atomist关于自动化开发的观点进行了扩展。这对开发人员来说是火箭酱 。
  • 我在Reactive Spring的JetBrains上为人们举办了一个网络研讨会。该网络研讨会现在可以在此处进行 。该网络研讨会介绍了Reactor，Spring Framework 5.0（和Spring WebFlux），Spring Data Kay，Spring Boot 2.0（和Actuator），Spring Security 5.0和WebClient ，全部使用Kotlin语言！我希望你喜欢。
  • 我以为这很酷-Pivotal 有电视广告 ！
  • 春季测试忍者Sam Brannen 提出了一个很好的示例 ，说明如何编写替代通用的JUnit 5 Jupiter扩展@Rule来自JUnit 4编程模型。
  • 雷德蒙克（Redmonk）的詹姆斯·州长（James Governor）对哈德逊之死（上周甲骨文在甲骨文全球大会上公开弃用），詹金斯（Jenkins） 等技术的盛行以及像Pivotal's Concourse这样的后起之秀颇有见识。
  • 此外，Redmonk的James Governor 还回顾了Cloud Foundry峰会的一些新闻，包括Kubo的新兴细节，Kubo是由Pivotal Cloud Foundry管理的Kubernetes实施。
  • 这是Hortonworks的Tim Spann的怀旧之词，但是很好，他创建了一个Spring Boot微服务来读取Apache Phoenix Data 。
  • TechCrunch还对最近的Cloud Foundry Summit新闻进行了综述 ，重点是将容器的Kubernetes集成到Cloud Foundry中 。这对于像Elasticsearch这样的有状态服务是理想的。
  • JAXEnter的Jane Elizabeth对最近发布的Spring Data Kay （反应性Spring Data实现）有很好的了解。
  • JAX创新奖刚刚举行，特别让我们感到高兴的是， Spring Framework 负责人Juergen Hoeller赢得了JAX特别评审团奖。TL; DR：Spring是规模最大且不断维护的代码库之一，而且Java社区中没有其他人这么长时间为该代码库做出贡献和完善。感谢Juergen，代码库不需要完全重写。声明中说：“Jürgen，谢谢您为Spring Framework所做的一切！”
