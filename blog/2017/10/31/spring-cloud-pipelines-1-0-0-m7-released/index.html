Spring Cloud Pipelines 1.0.0。M7发布

发布
马尔辛·格热兹扎克（Marcin Grzejszczak）
2017年10月31日

我很荣幸代表Spring Cloud团队宣布Spring Cloud Pipelines的新里程碑发布- 1.0.0.M7 。

什么是新的？

此版本增加了许多新功能，质量和测试改进。至于主要功能，我们增加了对Jenkins和Concourse的Kubernetes支持。至于质量和测试，我们已经编写了150多个Bash测试，添加了Shellcheck，EditorConfig并编写了更多的Jenkins测试。我们还统一了项目文档，现在可以在Spring Cloud Static gh页下的每个里程碑使用该文档。的文档1.0.0.M7在这里可用-http: //cloud.spring.io/spring-cloud-static/spring-cloud-pipelines/1.0.0.M7/

重大变化

我们将管道描述符文件名从pipeline.yml至sc-pipelines.yml 。

发行说明

您可以单击此处以查看发行说明 ，以获取有关此里程碑所做的更多信息。

谢谢

特别感谢@wybczu ， @malston ， @bijukunjummen ， @anwarchk和参与该项目的每个人！

如果您已经在使用Spring Cloud Pipelines，请在此问题下发表评论，并提供您需要采取的步骤来定制项目以满足公司的需要。没有您的反馈，我将无法自动执行自定义过程！
