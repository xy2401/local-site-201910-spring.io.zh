Spring Cloud Finchley。M3发布

发布
斯宾塞·吉布
2017年10月31日

我很高兴代表社区宣布， Spring Cloud Finchley Release Train的Milestone 3（M3）现已上市。该版本可以在Spring Milestone存储库中找到。您可以查看Finchley 发行说明以获取更多信息 。

Finchley发布火车的显着变化

Finchley发行培训中包含的许多项目中的一个共同主题是在Spring Reactive主题中增加对WebFlux，WebClient，Micrometer或其他集成的支持。

Spring Cloud侦探

再次支持WebFlux和Reactor。不包括OAuth支持。spring-cloud-sleuth-zipkin-stream模块已被移除。

Spring Cloud Gateway

新GatewayFilter接口，以减轻与现有的任何混乱WebFilter从Spring框架。任何自定义实现现在需要实现GatewayFilterFactory而非WebFilterFactory 。另外要引用WebFilterFactories将需要被替换GatewayFilterFactories为Java DSL。

Java DSL也略有变化。uri()方法现在是终止运算符，而不是and() 。

以前是这样写的：

@Bean
public RouteLocator wsRouteLocator() {
    return Routes.locator()
            .route("testws")
                .uri("ws://localhost:"+this.wsPort)
                .predicate(alwaysTrue())
                .and()
            .build();
}

现在将如下所示：

@Bean
public RouteLocator wsRouteLocator() {
    return Routes.locator()
            .route("testws")
                .predicate(alwaysTrue())
                .uri("ws://localhost:"+this.wsPort)
            .build();
}

可在http://start.spring.io上获得对网关的支持。
