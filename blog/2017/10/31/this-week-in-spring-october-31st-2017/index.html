春天的这个星期-2017年10月31日

工程
乔什·朗（Josh Long）
2017年10月31日

嗨，春季粉丝们，欢迎来到春季本周的另一期！本周我将在旧金山参加2017年第一次令人惊讶的Kotlin Conf 。我将谈论Spring Boot和Kotlin，希望您在身边时也能加入我的行列。

这周我们有很多工作要做，让我们开始吧！

  • Spring Batch主管Michael Minella 刚刚宣布了Spring Batch 4.0.0。M5 ，其中包括ItemProcessor支持Java 8的实现Function实施，更新的文档等等。
  • Spring Framework的负责人Juergen Hoeller 宣布了Spring Framework 5.0.1 ，其中包括一些错误修复，这些错误修复进而在Reactor Netty中进行了修复，并欢迎AspectJ 1.8.12中的性能改进。
  • 上周，在Spring Tips的另一部分中， 我研究了基于HTTP的Spring Cloud Contract合同 。
  • Spring Cloud联合创始人Spencer Gibb宣布了Spring Cloud Edgware的首个发布候选版本。各种启动器的工件名称中没有进行一些更改。Spring Cloud Edgware 具有许多新功能，因此请务必研究发行说明。
  • 您是否想知道Apache Geode和Pivotal Gemfire之间的区别是什么？让Spring Data Geode / Gemfire领导John Blum阐明本文中的差异 。TL; DR：不多！
  • Project Reactor负责人Stephane Maldini刚刚发布了Reactor Bismuth SR3 ，该版本包含许多错误修复和改进。Spring Framework 5.0.1中提供了Reactor Bismuth SR3。
  • Spring REST Docs负责人Andy Wilkinson 刚刚宣布了Spring REST Docs 2.0.0。RC1 。新版本需要Spring Framework 5.0和Java 8。2.0版本的主要亮点是支持使用Spring WebFlux的WebTestClient记录API。
  • Spring Web Services负责人Greg Turnquist 刚刚宣布了Spring Web Services 3.0.0。发布 。新版本具有许多新功能，包括更好地处理大型附件，SAML回调，升级到AspectJ的最新版本，支持即将到来的Axiom 1.3等。值得一提的是，感谢3.0版的75％是由社区贡献的！
  • Mark Pollack博士刚刚宣布了Spring Cloud Skipper 1.0 M1 。摘自文章：“ Skipper是一种轻量级工具，可让您发现Spring Boot应用程序并管理它们在多个Cloud Platform上的生命周期。您可以独立使用Skipper或将其与Continuous Integration管道集成在一起，以帮助实施Continuous Deployment的实践。”
  • 我喜欢从Spring Framework 5.0.1开始的新Spring Framework文档 。新的登录页面非常漂亮，只需单击任何部分，您就会看到轮廓如何真正被很好地清理。
  • Spring Integration忍者Artem Bilan 在这篇推荐的新文章中探讨了集成测试Spring Cloud Stream应用程序的艰巨主题。
  • 在谈到消费者驱动的合同测试时，Spring Cloud合同负责人Marcin Grzejszczak 在今年早些时候的演讲中提出了更有趣的观点
  • Oleh Dokuka 查看了Reactive Streams规范的Publisher根据Pivotal的Reactor实施类型输入 。正如Lightbend的Konrad Malawski所说 ：“可能唯一有关#reactivestreams实现和#flow＃jdk9的帖子可以很好地说明如何使用tck / rules ：）”
  • 您是否可以相信，自从上一届SpringOne Platform活动以来已经有一年多了？我们距离2018年旧金山的大型活动只有一个月！（你有票吗？）在SpringOne平台上 ， 将会有很多精彩的Apache Geode演讲，我们希望您能加入我们。
  • Spring创始人兼Atomist首席执行官Rod Johnson和我写了一篇文章，探讨了如何使用Atomist自动化服务自动化基于Kotlin的反应式Spring Boot 2.0应用程序的妊娠 - 快来看看！
  • Reactor忍者Simon Basle简要介绍了如何使用Project Reactor和java.nio.*支持。
  • 我喜欢Simon Wirtz的这篇文章，该文章探讨了Spring Framework 5和Kotlin的一些有趣的集成 。
  • Raja Sundaram Ganesan在关于使用Spring Boot 2.0和Kotlin开发反应微服务的非常不错的文章中。
  • 您正在使用Microsoft Azure吗？您会很高兴地知道 ，Spring Initializr上现在有专用于Spring Boot的专用启动器和自动配置。边看代码边看吧！ 。
