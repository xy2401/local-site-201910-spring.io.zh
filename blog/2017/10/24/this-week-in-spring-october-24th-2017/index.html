春天的这个星期-2017年10月24日

工程
乔什·朗（Josh Long）
十月24，2017

嗨，春季粉丝！欢迎来到春季本周的另一期！您可以相信我们已经盯着2017年10月底吗？今年飞逝了！太着急了！本周，我在阳光明媚的旧金山的家中与客户交谈并进行演讲。

我将在vJUG 24在线会议上发言 。会议持续24小时！每小时都会有一个话题，话题来自尽可能多的时区！我希望您能和我（以及成千上万的其他人）一起参加，我们将在周三格林尼治标准时间04:00看到Reactive Spring。

而且，当然，我希望您能注册并加入我的朋友和Grails的共同创始人Jeff Scott Brown和我，一起在 2017年11月9日看一看Grails for Spring Developers 。

  • 在上周的Spring Tips系列中 ， 我将介绍如何在Spring Boot 2.0应用程序上下文中将JAX-RS与Jersey参考实现一起使用 。
  • Spring Integration忍者Artem Bilan 刚刚宣布了适用于AWS 1.1 RC1的Spring Integration 。新版本包括KinesisMessageDrivenChannelAdapter ， 一种DynamoDbMetaDataStore ，Spring Cloud Stream Kinesis活页夹等等。看看这个！
  • 春天引导1.x中不支持Java 9。Spring Boot 2.0将于12月发布。就是说，如果您想将Spring Boot 1.x与Java 9一起使用，您可能会喜欢此Wiki页面，其中包含有关使用Java 9的一些技巧 。
  • 我们自己的Richard Seroter汇集了五种方法的清单，这些方法可以像软件公司一样购买软件并提高速度。
  • 我喜欢Spring框架中的这个问题，其中Thymeleaf的领导者Daniel Fernandez就如何在基于Spring WebFlux的应用程序中使用Thymeleaf进行了很多讨论。
  • 您现在可以按需配置多实例RabbitMQ集群 cf create-serivice在Pivotal Cloud Foundry上：欢呼！
  • Andreas Falk非常了解如何使用Spring Cloud Vault等在Cloud Foundry上管理机密 。
  • Randhir Randhir Singh对如何构建Spring Cloud Data Flow进行了很好的介绍，以通过将Spring Boot应用程序结合在一起来构建实时数据集成和数据处理管道 。
  • 我喜欢Biju Kunjummen撰写的All and Sundry博客， 比较了Spring Boot 1.0和Spring Boot 2.0的性能 ，后者当然基于Spring WebFlux和Reactor Project。剧透：异步非阻塞IO在SLA级别春季启动2.0支持更多的需求。
  • 好吧，首先：这是超级早期，无非就是一个原型！但是，我们自己的Vinicius Carvalho已经为Spring和RSocket集成了原型集成 。RSocket是一种在Reactive Streams类型之上的高性能协议（尤其是它使用Reactor）。它得到了Facebook，Netflix和AEron项目等高级工程师的贡献。所有这一切， 真是令人兴奋！但是还早呢！（我有提到吗？）现在还很早， 所以我相信Vinicius会感谢您的任何反馈。
  • 我喜欢Spring框架，响应式Spring和Kotlin ninjaSébastienDeleuze的惊人示例，展示了如何在应用程序的前端和后端使用Kotlin 。前端利用Kotlin2JS，后端利用JVM上的Kotlin（带有Spring Boot，natch）
  • 我还没有读过这本书，但是它确实很有趣：Ravi Kant Soni写了一本书，关于用Spring Boot构建Angular应用程序 。看看这个。
