切尔西春季云流助阵GA！

发布
马里乌斯·波哥维奇（Marius Bogoevici）
2017年4月13日

我很高兴代表社区宣布Spring Cloud Stream切尔西发布火车的全面上市。对于此发行系列，第一个一般可用性发行版是Chelsea。SR1，解决了切尔西的许多问题。释放（均在度量标准导出功能的范围内）。切尔西。SR1也包含在Spring Cloud Dalston中。发布 。

新版本可以在Maven Central中获得，有关其功能的详细说明可以在参考文档中找到。有关工件和最新更改的信息，请参考以下发行说明：

  • 切尔西笔记。发布

  • 切尔西笔记。SR1

有什么新东西？

新版本包含许多对先前版本候选版本的修复和改进。提醒一下，以下是以前版本中新增的功能（带有指向原始公告的链接）的一些要点。

  • 调度功能已添加到StreamListener

  • 指标导出

  • 搜索和缓存的架构改进

  • RabbitMQ自定义基础架构支持

  • 设置SPI

接下来是什么：

发布后，我们将继续扩展Spring Cloud Stream生态系统：

  • 改进活页夹中的错误处理支持，并提供用户友好的API进行恢复和错误处理

  • 里程碑（1.0.0。M1）发布用于JMS活页夹

  • 基于Spring Kafka中新增的支持，预览Kafka Streams支持

  • 专注于基于云基础架构的活页夹，例如Google Pub Sub和Kinesis

  • 预览Reactive Streams绑定程序支持
