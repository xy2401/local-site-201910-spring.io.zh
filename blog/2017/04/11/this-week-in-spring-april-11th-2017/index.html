    春天的这个星期-2017年4月11日

    工程
    乔什·朗（Josh Long）
    2017年4月11日

    欢迎来到春季本周的另一个装置！

    • Spring Integration忍者Artem Bilan 刚刚宣布了Spring Integration 5.0。M3 。新版本包括集成测试框架，速度改进，对Spring 5 React的支持WebClient ，使用（S）FTP和AWS S3入站适配器，更智能的SOAP Web服务网关等时的文件树的追溯还原。
    • Spring Integration负责人Gary Russell刚刚宣布了适用于Apache Kafka 1.2.0和1.1.4的Spring。
    • 安迪·威尔金森（Andy Wilkinson）刚刚亮相了新的Gradle Spring Boot 2.0插件
    • 在上周的Spring Tips中，我研究了如何支持Spring Cloud Security支持的OAuth身份验证服务
    • 春天AMQP导致加利罗素刚刚宣布春季AMQP 2.0.0M3 ， 挤满了新的功能配套的测试，更聪明的处理null消息等等。
    • 您是否要参加德克萨斯州奥斯丁的不容错过的OSCON活动？我们很高兴见到你 ！我也要去
    • 我很喜欢从Cloud Foundry启动活动中回顾这段视频 。我的记忆朦胧，但我想该事件是4月11日，并于4月12日上传。无论哪种方式，Cloud Foundry生日快乐！您已经用风格改变了世界。
    • Jared Ruckle具有PCF 1.10中的所有新功能 -快来查看！
    • 如果您想了解有关我们如何将Cloud Foundry推向市场的更多信息，您会喜欢Pivotal的一位有远见者James Watters的 a16z播客
    • 我喜欢Gaurav Rai Mazra的快速浏览，将Spring Data JPA与嵌入式数据库一起使用
    • Andreas Falk有一篇不错的文章，介绍了如何保护Spring和Angular.js应用程序
    • Baeldung网站对Java的趋势做了又一个非常有趣的观察，再次指出Spring Boot和Java 8的持续增长 -好消息！
    • 这只是一个友好的提醒，阿里巴巴在整个过程中都使用Spring ，而服务100亿次以上使用的AliExpress在使用Spring Boot和Spring Cloud
    • 我希望您能和我一起参加Voxxed Days Singapore
    • Alex Falappa暗示了对Apache NetBeans的一些非常酷的新支持
    • Stefan Rempfer提出了将Spring Cloud Zuul与Vaadin结合使用的一个很好的例子
    • 我喜欢 Robert Kapala撰写的有关他们如何将投资银行迁移到Spring Boot的博客
    • Jonas Hecht 撰写了一篇有关使用Ansible在Windows上运行Spring Boot应用程序的不错的文章
