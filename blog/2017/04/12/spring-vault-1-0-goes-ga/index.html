Spring Vault 1.0正式发布

发布
马克·帕鲁奇
2017年4月12日

我很荣幸地代表社区宣布Spring Vault 1.0的全面可用性-这是经过近一年的开发后Spring Vault的第一个GA版本。

这些工件可以从Maven Central和Bintray获得。


  org.springframework.vault
  spring-vault-core
  1.0.0.RELEASE

此版本共发行了50张以上的票。以下是该版本附带的最重要功能的简短说明：

  • 通过AppRole，AWS-EC2，客户端证书，Cubbyhole（包装令牌）和静态令牌进行可插拔应用程序身份验证。
  • 通过凭据轮换支持可更新的@VaultPropertySource 。
  • 终端令牌到期后，具有令牌更新和重新登录的生命周期感知会话管理。
  • 模板API，可直接与保险柜互动。
  • 与Vault 0.5到0.7的兼容性。

会费

没有社区，我们就不可能成为今天的成功项目。我要感谢所有创建问题并提供反馈的人。

反馈请

如果您对此版本有任何反馈，建议您通过StackOverflow ， GitHub Issues或评论部分进行接触。您也可以在Twitter上对我@ mp911或@SpringCloud团队执行ping操作。

下一步

即将发布的Spring Cloud Vault Config 1.0 GA版本将接管Spring Vault GA，并作为Spring Cloud Dalston版本培训的一部分提供。
