春天的这个星期-2017年4月4日

工程
乔什·朗（Josh Long）
2017年4月4日

欢迎来到春季本周的另一期！这周我在英国伦敦。法国巴黎（在史诗般的Devoxx France活动中，我将与我的好友Matt Raible进行交谈 ）和德国斯图加特，在此我将在会议上和对客户讲话！如果您在附近，请毫不犹豫地说“嗨”！

  • 新版本的Spring Tool Suite挤满了支持Spring Boot和Cloud Foundry的新功能。检查新的和值得注意的 ，并在热的时候得到它！（我刚买了一台新笔记本电脑，所以我现在也正在做！）
  • 上周在最新版的Spring Tips中，我研究了Spring Framework中的各种实用程序类
  • 我刚刚在首届Voxxed Days CERN活动上演讲， 组织者采访了令人惊叹的Spring Framework负责人Juergen Hoeller，还有我（他的最大粉丝）
  • 不要错过4月13日即将举行的网络研讨会， Spring Boot Under the Hood
  • 在OSCON Austin 2017上查看Spring Team和Pivotal！
  • 我真的很喜欢Okta博客上Matt Raible的新帖子，其中介绍了如何将Spring Boot和Okta 与SAML和OAuth 2.0一起使用
  • 我来自阿里巴巴的朋友Jacky Chan 编写了一个出色的示例应用程序，展示了Spring Boot和Angular4
  • 最近发布了项目Find Security Bugs （用于Java Web应用程序安全审核的FindBugs插件）的1.6.0版本。这个新版本增加了一些特定于Spring的检查，例如禁用CSRF保护 ，无限制使用RequestMapping导致CSRF漏洞以及使用用户输入进行危险的服务器端重定向 。感谢Pablo Tamarit引起我的注意-非常酷！
  • 我们的朋友Nicolas Frankel有一篇有趣的文章，介绍了在Spring MVC应用程序中的通用代码放置位置
  • 是的，愚人节已经过去了，但我只想分享这个由2016年春季测试负责人Sam Polynnlot的Sam Brannen创作的史诗作品，展示了针对不同（人类）语言的组合应用程序
  • 尽管您可以使用Spring Cloud Zuul或新的Spring Cloud Gateway技术来实现，但这并不是与Spring专门相关的，但是Stripe的这篇关于负载均衡器和卸料机细微差别的文章非常有用。
  • 我非常喜欢这篇介绍了Kotlin语言最佳实践的文章
  • Spring Cloud忍者Dave Turanski撰写了一篇很棒的文章，介绍了针对事件驱动架构的Spring Cloud Stream
