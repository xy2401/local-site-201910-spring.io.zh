春天的这个星期-2017年4月25日

工程
乔什·朗（Josh Long）
2017年4月25日

欢迎来到春季本周的另一期！本周，我在伟大的伟大的印度开发者峰会第十版上 ，在印度美丽的班加罗尔！然后，我要去澳大利亚的珀斯， 因为YOW！西 。如果您在附近，打个招呼！ 。我们有很多内容要讲，所以让我们开始吧！

  • Spring REST Docs领导Andy Wilkinson 刚刚宣布了Spring REST Docs 1.2.0 ！这个新版本包括更智能的Asciidoctor集成，对请求和响应片段的支持以及对大型和复杂负载的更好支持。同时，Andy 还 发布了Spring REST Docs 1.1.3（维护版本）
  • 在ZTLive上收听Spring WebFlux忍者塞巴斯蒂安·德勒兹（Sebastien Deleuze）史诗般的Reactive Spring演讲的重播
  • Spring Boot 忍者StéphaneNicoll刚刚发布了Spring Boot 1.5.3 ，这是一个维护版本，其中包含许多修复和改进功能。他还宣布了Spring Boot 1.4.6 。
  • Spring Cloud Data Flow负责人和联合创始人Mark Pollack博士刚刚宣布了Spring Cloud Data Flow 1.2。RC1 。该版本挤满了新功能，包括对组合任务的支持，实时指标和监视，OAuth改进，Spring Cloud Stream应用程序启动程序Bacon的新版本（俗称）。发布，以及Spring Cloud Task App Starters Belmont的新版本。RC1。尽您所能获取位！踢轮胎和反馈！
  • Spring Cloud Pipelines的领导者和持续交付专家Marcin Grzejszczak刚刚宣布了Spring Cloud Pipelines 1.0.0。M4 。Spring Cloud Pipelines是一种现成的连续交付管道，可以在Jenkins或Pivotal的Concourse中使用 。
  • 史泰芬·尼科尔 （ StéphaneNicoll）还没有人为之骄傲，他还宣布了Spring Framework 4.3.8
  • Baeldung博客上有一篇不错的文章，介绍了如何使用Spring AMAQP项目的服务调用程序支持
  • Spring Cloud Zipkin负责人和OpenZipkin项目贡献者Adrian Cole结合日志、、指标和跟踪对可观察性进行了非常好的演讲
  • 该连接器使将RabbitMQ连接到Apache Kafka变得很容易 -看起来很有趣！
  • Spring Cloud Contract负责人Marcin Grzejszczak（还记得他吗？他也是Spring Cloud Pipelines的负责人！）在InfoQ上做了很好的采访 。
  • Bozho做了一个不错的博客，着眼于Spring Boot应用程序中的Spring MVC用例
  • 出于多种原因，用于Cloud Foundry的Microsoft Azure服务代理非常有趣：它使您可以使用Cloud Foundry的Microsoft Azure支持服务（马上！）它是用Spring Boot编写的， 因此它是一个非常好的模板
  • 是否想了解Cloud Foundry生态系统中正在发生的事情？我喜欢 2017年1月至2017年3月的这次综述 。
  • Auth0博客上有一篇不错的文章， 介绍如何在Spring Boot中使用JWT
  • Pierre Besson组合了一个非常方便的JHipster 生成器，该生成器生成自定义的Spring Bootbanner.txt
