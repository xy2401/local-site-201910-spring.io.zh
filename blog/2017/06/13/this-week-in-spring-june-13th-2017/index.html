春天的这个星期-2017年6月13日

工程
乔什·朗（Josh Long）
2017年6月13日

嗨，春季粉丝！本周，我在雷德蒙德（Redmond），与我们的朋友和Pivotal合作伙伴Microsoft一起讨论Microsoft Azure上的Cloud Foundry。如您所知，Microsoft的Azure IaaS平台是使用最广泛的云＃2，并且还在不断增长！在更多地区（包括德国和中国！）中，它的数据中心比其他两个云供应商要多。以及大量支持Windows和Linux操作系统的服务。出现了很多令人兴奋的东西，包括在内部运行Azure的Azure Stack，以及Azure和PCF之间的惊人集成。我期待本周在圣克拉拉举行的CF峰会上听到更多有关Microsoft Azure的信息，我将在星期三在那儿开会，然后前往犹他州参加犹他州JUG比赛。走出来，和我一起，说您好 ！

  • Baeldung在使用Spring Boot启动应用程序方面看起来很不错
  • Microsoft Azure上发生了一些非常酷的事情，并且与Spring和Pivotal Cloud Foundry配合使用特别好。通过免费的白皮书了解更多信息。
  • Spring 5具有许多主要功能，还有许多较小但非常方便的功能。其中之一是新引入的对@Nullable批注输入到您的编译器和IDE反馈中以进行方法调用
  • Auth0博客上有一篇不错的文章，内容涉及使用Spring Session扩展JSF应用程序
  • 社区传奇人物Michael Simons在此推文中指出了一个重点 ：如果您想了解Spring Data存储库的全部内容，请查阅Eric Evans的“域驱动设计”书！
  • Spring Boot传奇人物StéphaneNicoll刚刚宣布了Spring Boot 1.5.4
  • Spring Integration忍者Artem Bilan 刚刚宣布了Spring Integration 4.3.10 ，其中包括JSON序列化方面的改进，改进的错误处理等
  • 阿尔泰姆（ Artem）还宣布了 Spring AMQP 1.6.10和1.7.3
  • Artem 还宣布了适用于Apache Kafka 1.2.2和1.1.6的Spring
  • Spring Security主管Rob Winch刚刚宣布了CVE-2017-4995的修复程序-立即更新！
  • Spring Data忍者Mark Paluch刚刚宣布了Spring Data Ingalls SR4和Hopper SR11 ，其中包含新功能和更新版本！
  • StéphaneNicoll 还 宣布了Spring Framework 4.3.9
  • Marcin Grzejszczak刚刚宣布了Spring Cloud Pipelines 1.0.0。M5
