春天的这个星期-2017年6月20日

工程
乔什·朗（Josh Long）
2017年6月20日

嗨，春季粉丝们，欢迎来到春季本周的另一期！本周，我将在波兰克拉科夫参加史诗般的Devoxx PL活动，然后前往卢森堡参加VoxxedDays Luxembourg活动 ！如果你周围，不要犹豫，说喜

  • Spring Framework的负责人Juergen Hoeller 刚刚宣布了Spring Framework 5.0 RC2 。新版本是RC1的主要修订版，重构了反应式支持和路径模式解析。它介绍了基于JSR 305的@Nullable / @NonNullApi将Spring注释添加到代码库中。
  • Spring Data Train凯M4发布 -Spring Data忍者Mark Paluch刚刚宣布了Spring Data Train凯M4，其中包含新功能和更新。立即获取位！
  • Spring IO PlatformAthens-SR6 -Spring IO Platform负责人Andy Wilkinson刚刚宣布了Spring IO PlatformAthens-SR6，可从repo.spring.io和Maven Central获得。新的维护版本包括Spring Boot 1.4.7，Spring Framework 4.3.9，Sspring Integration 4.3.10，Spring REST Docs 1.1.3，Spring Seurity OAuth 2.0.14，Spring Web Flow 2.4.5和Spring Cloud Connectors 1.2。 4，除其他外。
  • Spring IO PlatformBrussels-SR3 -Andy 还不止于此，Andy 还发布了Spring IO Platform布鲁塞尔-SR3，其中包括大量支持库的更新。立即获取位！
  • Spring Boot 2.0.0。M2现在可用 -Spring Boot忍者StéphaneNicoll继Spring Framework RC2之后宣布了Spring Boot 2.0M2。
  • Spring Session 2.0.0 M2发布 -Spring Session的负责人Rob Winch刚刚宣布了Spring Session 2.0.0。M2。新版本确保了与Spring Framework 5.0.0的兼容性。RC2和Spring Data Kay-M4，都是新版本的基准。
  • Spring Security主管Rob Winch刚刚宣布了Spring Security 5.0 。新版本包括改进的反应式支持Java配置，并改进了WebClient测试支持。
  • 现在，Spring Session MongoDB的负责人Greg Turnquist 刚刚宣布了2.0.0。M1
  • 如果应用程序可以通过每天轮换其访问凭据自动抵御攻击者，从而使每个被盗密码在短时间内变得无用，该怎么办？这就是CredHub背后的想法，CredHub是今天为Pivotal Cloud Foundry引入的一项新功能 。阅读此VentureBeat文章的其余部分以获取更多信息。
  • Dormain Drewitz引用了最近的Cloud Foundry峰会中成功的Cloud Foundry用户的一些秘密
  • Jagdish Mirani在本文中介绍了HCSC如何使用Gemfire来帮助集成旧版应用程序 。
  • 马克·赫克勒（Mark Heckler） 在这篇简明的文章中着眼于按标题类型过滤HTTP请求
  • 从DynaTrace获取Duncan Winn关于Cloud Foundry的新书的预览章节
  • TechCrunch考察了Cloud Foundry在企业中的主导地位 -我最喜欢的事情之一是，有超过一半的《财富》 500强企业都在使用Cloud Foundry。
  • Tim Schimandle 在将Angular 4前端与他的Spring Cloud后端集成在一起的过程中继续了他的系列。
  • 蒂姆Schimandle 还写了一个很好的介绍了分布式跟踪与Spring云猎犬和基普金在这个岗位
  • Tim Schimandle希望确保Spring Cloud Services的安全
