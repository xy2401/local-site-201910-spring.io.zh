Spring Session MongoDB 2.0.0。M1发布

工程
格雷格·特恩奎斯特
2017年6月16日

亲爱的春季社区，

上个月，Spring Session的负责人Rob Winch宣布发布Spring Session 2.0.0。M1 （注意到那里缺少MongoDB吗？）在那篇文章中，他将Spring Session缩减为正式支持Redis，JDBC和Hazelcast。没有更多的MongoDB。

我在这里宣布，我已经接手了Spring Session MongoDB的工作 。同时管理Spring Session 和 Spring Security（还有其他职责），Rob无法通过过多的数据存储来维持高水平的支持。作为Spring Data团队的一员，我感觉更适合提供MongoDB对Spring Session的支持，因此我将其恢复为一个单独的项目 。

为了赶上一切，Spring Session MongoDB的这个发行版是针对以下关键项目构建的：

  • Spring Session 2.0.0。M2

  • Spring Data凯M4

  • Spring Framework 5.0.0。RC1

  • Spring Security 5.0.0。M1

  • Java 8

重要
 工件坐标已更改为org.springframework.session:spring-session-data-mongodb （注意数据库名称后附加了吗？）

我正在与Spring Boot团队合作，将Spring Session MongoDB添加到针对Boot 2.0的大量版本管理项目中。请继续关注这一方面。

同时，如果您访问项目站点，获取坐标并在您的Spring Boot应用程序中包含版本号 ，今天就可以获取。
