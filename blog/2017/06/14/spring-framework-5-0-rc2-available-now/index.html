Spring Framework 5.0 RC2现在可用

发布
于尔根·霍勒（Juergen Hoeller）
2017年6月14日

尊敬的Spring社区，

我很高兴地宣布，我们的第二个Spring Framework 5.0发行候选版现已上市 。这是RC1的主要修订版，在我们的新的响应式工具和新的路径模式解析器中进行了重构，并对整个框架中的代码样式进行了改进。

此版本为我们的所有软件包引入了一个正式的非null API声明，带有显式可为空的参数和返回值（现在已注释为此类）（而不仅仅是javadoc'ed）。我们的可空性注释符合JSR 305，并得到IntelliJ IDEA和即将推出的Kotlin版本的特别支持，并有可能为Kotlin用户带来特别强大的收益（这就是为什么我们仍将它们推广到5.0的原因）。请注意，我们现在在某些Spring API中应用了更强大的null强制，从而取代了我们以前宽大的null处理。

请尝试RC2，让我们知道如何进行！接下来是7月的RC3，即将在我们发布Google Analytics（分析）之前，因此仍然有机会根据您的反馈进行进一步完善。除了代码库中的一些剩余问题之外，我们还将选择Reactor 3.1和Jackson 2.9的最终版本，并总结与Servlet 4.0和JPA 2.2的兼容性。

干杯，
于尔根
