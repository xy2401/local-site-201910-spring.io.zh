    春天的这个星期-2017年7月18日

    工程
    乔什·朗（Josh Long）
    2017年7月18日

    嗨，春季粉丝！本周我将在希腊克里特岛举行史诗般的JCrete非会议，然后下周前往土耳其伊斯坦布尔，与客户进行下一次访问，并参加春季聚会。希望您能加入我的伊斯坦布尔 ！

    和往常一样，我们有很多内容需要介绍，让我们开始吧！

    • Spring Integration忍者Artem Bilan刚刚宣布了Hazlecast 1.0.0的Spring Integration Extension。GA 。此新版本的主要功能是错误修复和改进。它还通过HazelcastMessageStore ， HazelcastLockRegistry和HazelcastMetadataStore实现添加了对分布式集群应用程序的支持。
    • Spring Security忍者Joe Grandja 刚刚宣布了Spring Security OAuth 2.2.0。RC1 。这个新版本包括对JWT令牌和声明集处理方式的改进。不要错过更新！
    • 是否想在Cloud Foundry上分析Java应用程序 ？来自Cloud Foundry的Java经验主管Ben Hale的这个有用的技巧应该使您入门！
    • 来自Pivotal自己的Luke Shannon的技巧提示如何在升级时在Spring Tool Suite中修复损坏的“组织导入”命令可能对某些人有用！
    • 我喜欢这种法语语言， 看看Spring Framework 5中的一些新功能 。
    • 我喜欢这个名为Trampoline的项目，该项目有助于启动和停止基于Spring Boot的服务。在开发过程中，这看起来超级方便！
    • 开放服务代理API的工作正在全速进行。最近，创建按需服务经纪人的工作变得容易得多，这篇文章很好地解释了如何做。
    • Java 9即将来临，除了模块化支持之外，还有很多值得推荐的地方。我喜欢Oracle博客上有关Java 9 Collections更新的外观。
    • 想要感受一下JDK 9中的一些新工具吗？在Oracle博客上查看这篇不错的文章。
    • Atomist的Jessica Karr研究了使用HTTPS将应用程序从AWS迁移到Pivotal Web Services （或PWS，由Pivotal运行的Pivotal Cloud Foundry的多租户托管安装）所需的条件。该博客介绍了一些需要注意的事情。Pivotal自己的Glen Oppegard在评论中还提供了一些额外的信息，同样值得一试。
    • Pivotal自己的Richard Seroter 研究了在Pivotal Cloud Foundry上支持事件驱动的基于消息的应用程序的一些方法 。他研究了Spring Cloud Data Flow，典型应用程序包括的事件类型以及PCF中的自动伸缩节点。
    • 这篇文章着眼于用Java，Kotlin和Scala 构建基本的Spring Web应用程序
    • 这是应用程序间事件选项的不错比较 。三种选择来自Spring和Pivotal！这是一篇很酷的文章，但我要提醒您记住，他们正在查看Reactor已有两年多的旧版本，鉴于今天的Reactive Streams实现，它看起来非常不同。
    • 通过Spring Data ElasticSearch使用ElasticSearch吗？您可能喜欢这篇有关如何与ElasticSearch进行集成测试的文章 -
    • Ben Wilcock重新审视了2015年编写的旧应用程序，该应用程序使用Axon CQRS框架构建了逻辑微服务应用程序 。在这个新版本中，Ben着眼于Axon v3，Cloud Foundry和Concourse等。所有这些极大地简化了他的代码。
    • 这是一个老家伙，但又是一个好东西，着眼于我们告诉自己的回滚和其他部署神话的想法 。绝对值得一读，即使它与Spring本身并没有多大关系！
