Spring Boot 1.5.6现在可用

发布
安迪·威尔金森
2017年7月27日

我很高兴代表团队宣布Spring Boot 1.5.6已发布，现在可以从repo.spring.io和Maven Central中获得 。

Spring Boot 1.5.6修复了1.5.5中引入的Spring Data依赖管理问题 。抱歉给你带来不便。

下一步是什么？

在最近发布了Spring Framework 5.0 RC3以及紧随其后的其他版本之后，Spring Boot 2.0 M3 才刚刚发布 。如果您想早期了解Spring Boot 2，并且希望收到您的反馈，请访问start.spring.io并选择Spring Boot 2.0.0.M3或2.0.0.BUILD-SNAPSHOT 。

您能提供什么帮助？

如果您想提供帮助，请查看问题存储库中的“ ideal fortribution”标签 。如果您有一般性问题，请使用spring-boot标签在stackoverflow.com上提问，或在Gitter上与社区聊天。
