    春天的这个星期-2017年7月25日

    工程
    乔什·朗（Josh Long）
    2017年7月24日

    嗨，春季粉丝！本周，我在土耳其的伊斯坦布尔与客户交谈，并在今晚的Spring和Java聚会上讲话。希望您能加入我的行列，并且我们将谈论云原生Java！像往常一样，本周我们有很多工作要做，让我们开始吧！

    • Spring Integration的贡献者Artem Bilan刚刚宣布了Spring Integration 4.3.11 。此版本包括安全更新和错误修复。
    • Spring Batch主管Michael Minella刚刚宣布了Spring Batch 3.0.8 ，主要是维护版本和错误修正版本。
    • Spring NinjaStéphaneNicoll发布了Spring Framework 4.3.10 ，它是即将发布的Spring Boot 1.5.5维护版本的维护版本。它包含25个修复和改进。
    • Spring Cloud联合创始人Spencer Gibb刚刚宣布了Spring Cloud Dalston SR2 。该版本主要是一个错误修正。同样值得注意的是，此版本标志着Spring Cloud Angel和Spring Cloud Brixton生命的尽头。
    • Spring Framework的负责人Juergen Hoeller刚刚宣布了Spring Framework 5.0 RC3 。新版本是扩展版本候选阶段的一部分，该阶段允许Reactor 3.1，JUnit 5.0，Jackson 2.9等等，包括Spring WebFlux中的API改进，Kotlin支持，改进的可空性声明以及更新的JDK9支持。
    • Spring Security的贡献者Joe Grandja刚刚宣布了Spring Security 5.0.0。M3 。新版本包括对JSON Web令牌（JWT），JSON Web签名（JWS）的支持以及对OpenID Connect身份验证流的集成ID令牌支持。
    • Spring集成和消息传递忍者Artem Bilan刚刚宣布了Spring AMQP 2.0.0里程碑5 。此版本包括安全修复程序和许多不错的新功能。
    • Micrometer为最流行的监视系统提供了一个基于仪表客户端的简单外观，使您无需供应商锁定即可对基于JVM的应用程序代码进行仪表。考虑使用SLF4J，但要考虑指标。该项目将用作Spring Boot 2.0的度量标准收集子系统，并将反向移植到Spring Boot 1.0。值得注意的是，虽然是千不需要春天引导工作。看看这个。这里有很多很棒的东西，几乎不知道从哪里开始。
    • Gyula Lakatos整理了一篇富有启发性的详细博客，详细介绍了他的第一本新鲜的关于AWS Lambda上的Spring Cloud Function的内容 。干得好！
    • 这是一个老套，但又是个好东西：Antonio Simoes谈到了将其公司的架构从整体架构转变为基于Spring Cloud的无阻塞架构 。
    • 微服务是由供应商驱动的 -这并不是严格地依赖于Spring，但是我喜欢Stephen O'Grady关于微服务的讨论-人们为什么采用微服务以及它们与SOA有何区别。
    • James Governer 跟踪了 Stephen O'Grady的微服务职位。
    • 在绝对绝妙的新闻中 -没有什么可以做的-Spring-se-se专栏文章：10年前，有2,600名女学生参加了AP计算机科学考试。2017年，有29,000名女学生参加了考试。自2016年以来，女学生的增长令人难以置信，使AP CS考试的参与率增加了135％。人数不多，但参与人数不多，与去年相比增加了近170％！事情正在朝着正确的方向发展。我衷心希望我们的生态系统看到多样性的增加，更多的包容性和参与度，并继续增长。
    • Spring Boot用户NickyMølholm表示， 新的Lego Life应用程序在后台由Spring Boot支持 。作为Lego和Spring Boot的粉丝，我认为这太酷了！恭喜乐高人寿团队！
    • 快速行动，永无止境-Spring创始人兼Atomist首席执行官Rod Johnson着，Rod Johnson详细介绍了他公司的主要产品如何为使用Spring Boot的团队带来福音。这是一个非常酷的产品，它简化了创建新服务（可以并且应该在Spring Boot中）的端到端故事，支持协作，检测引起中断和中断的更改以及管理生产路径（例如Cloud Foundry）。我已经报名了！
    • 本月的《 Oracle Java杂志》 很好地介绍了Java 9中的一些新功能，这些功能即将来临。
    • 这是Spring忍者Greg Turnquist的老歌，但又是个好东西，着眼于人们认为他们在REST中遇到的一些问题以及如何解决它们。这是一本好书。
