春天的这个星期-2017年7月11日

工程
乔什·朗（Josh Long）
2017年7月11日

嗨，春季粉丝！欢迎来到春季本周的另一期！这周我在明尼苏达州的明尼阿波利斯。堪萨斯州堪萨斯城；亚利桑那州的凤凰城（Phoenix）与客户和Pivotal合作伙伴就Spring和Cloud Foundry的所有事宜进行了交谈。如果您在附近，请打个招呼 ！

这周我们有很多工作要做，让我们开始吧！

  • Mark Fisher首次展示了Spring Cloud Function ，它是一种抽象功能 ，它将功能即服务的范式带到您的Spring应用程序中。这是必读 ！
  • Spring（大数据）忍者Thomas Risberg刚刚宣布了适用于Apache Hadoop 2.5.0 GA的Spring，该版本主要包含错误修复和依赖项更新。
  • Spring Cloud联合创始人刚刚宣布了Spring Cloud Finchley的第一个里程碑 。该版本是第一个基于即将发布的Spring Boot 2.0基准并具有一个主要新功能的新功能：名为Spring Cloud Gateway的新微代理，我们认为它是Zuul 1的替代产品。
  • Spring Tool Suite 3.9.0已发布-Spring Tool Suite的负责人Martin Lippert刚刚宣布了Spring Tool Suite 3.9.0，它又基于Eclipse Oxygen。它还包括改进的Spring Cloud集成以及对编辑Cloud Foundry清单的支持。
  • 这篇文章着眼于将Spring Framework JmsTemplate与嵌入式ActiveMQ实例一起使用。
  • 大多数人没有意识到Spring集成了一个非常强大的面向Aspect的编程库，称为AspectJ。这篇文章从Spring Boot应用程序的角度介绍了AspectJ 。
  • 简·伊丽莎白（Jane Elizabeth）在JAXEnter上撰写了有关Spring Cloud Function的出色文章
  • 甲骨文宣布将通过Spring Boot开发REST应用程序，作为大规模在线课程。听起来不错！
  • 查看Spring Data领导者Oliver Gierke 关于域驱动设计与REST的JAX演讲
  • 这看起来很有趣-这是一个微型工作流程引擎-Piper-建立在Spring Boot的基础上。做得好，阿里克！
