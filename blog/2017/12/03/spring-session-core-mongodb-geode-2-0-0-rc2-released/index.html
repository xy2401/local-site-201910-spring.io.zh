Spring Session（Core / MongoDB / Geode）2.0.0。RC2发布了！

工程
格雷格·特恩奎斯特
十二月03，2017

Spring 问候社区，

为了准备即将发布的GA版本，Spring Session，Spring Session MongoDB以及适用于Apache Geode和Pivotal GemFire的Spring Session均已与2.0.0版保持一致。此发行版的RC2。

如果您错过了它，那么每个模块都包含以下内容：

  • 核心 -包括对Redis，JDBC和Hazelcast的支持

  • MongoDB-传统阻止以及对MongoDB的反应式支持

  • 阿帕奇的Geode -为Apache的Geode，以及枢纽的GemFire支持（包括PCC与服务SSC的扩展PCF ）

在此版本中，Spring Session在其API中进行了一些重构和较小的调整。在某些合同中有微小的重大更改，但没有影响最终用户的更改。

我们正在努力开发类似于其他几个Spring项目（例如Spring Cloud和Spring Data）的BOM（物料清单）。一旦发布，这将使您选择的商店特定解决方案的版本与Spring Session core的版本保持一致。敬请关注！

版本号

Spring Session中受支持的主要库包括：

  • Spring Framework 5.0.2。发布

  • 项目反应堆铋-SR4

  • Spring Data Kay-SR2

  • Spring Security 5.0.0。发布

  • Hazelcast 3.9

  • 生菜驱动程序5.0.1。发布

Spring Session MongoDB包括以下附加依赖项：

  • MongoDB 3.5

  • MongoDB（反应式串流）1.6.0

Apace Geode（和Pivotal GemFire）的 Spring 会议包括以下附加依赖项：

  • Apache Geode 1.2.1

  • 关键GemFire 9.1.1
