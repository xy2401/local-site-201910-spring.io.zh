Spring 的这个星期-2017年12月19日

工程
乔什·朗（Josh Long）
2017年12月20日

嗨，春季粉丝！欢迎来到春季本周的另一期！我刚从伦敦回来，我周一在这里与Pivotal客户进行了交谈，并帮助团队在Spring方面开发软件。

现在我回家度假。您能相信我们离2018年只有几周了！？时间过去了！光明节，圣诞节和节礼日将结束，宽扎节将在下周的本周春季本周开始之前开始，因此，我代表春季团队和整个Pivotal，祝您假期最温暖，最快乐！

如果您碰巧有一点时间，我们有很多事情可以帮您解决，所以事不宜迟，让我们开始吧。

  • 在最新版的Spring Tips中 ，我将介绍Spring Boot 2.0甚至是Spring的最重要功能之一：ASCII横幅！不要错过这个！
  • 我喜欢这个由Spring Cloud Contract负责人Marcin Grzejszczak和Adib Saikali介绍消费者驱动合同（CDC）的演讲。
  • 我真的很喜欢Project Riff的第一篇博客文章，这是我们在SpringOne Platform 2017上宣布的无服务器平台。
  • 这是个老歌，但可能对某些人有用。在这篇文章中， 我和Redmonk的James Governor 一起通过Spring Boot看云原生 。
  • 在此SpringOne Platform 2017演讲中，Spring Framework Juergen Hoeller 讨论了Java 9上的Spring Framework 5.0 。
  • 这是一个有趣的示例Spring Boot项目，它使用Giraphe CMS框架和Spring Boot 。Giraphe是基于Spring Boot的无头CMS框架。
  • Baeldung已使用Spring教程更新了他们的REST ，并且它总是值得一看的。
  • 优秀的Syer博士整理了一个非常不错的教程， 探讨如何使用Angular接受Spring Boot项目 。
  • IntelliJ工程师YannCébron知道如何开始生产过程 ： 开始..点..Spring ..点..io ！
  • 又到了每年的这个时候！所有SpringOne Platform 2017视频均已发布并可用 ！狂欢看一切，给自己知识的礼物！
  • 在响应式Spring Data实现的背后做了很多工作的Spring Data忍者Mark Paluch在响应式数据访问的幕后作了出色的SpringOne Platform 2017演讲
  • Spring社区传奇人物Jenn Strater在SpringOne Platform 2017上发表了有关使用Spring REST Docs _Documenting RESTful API的演讲]（ https://www.youtube.com/watch?v=CaARz49u1Mc&list=PLSzU7snapMVZ4n5s5glQz4Dd_C3572O_2 ），非常值得一看。
  • Linux基金会提供了有关使用Cloud Foundry构建云原生软件的很好的课程。
  • InfoQ的Victor Grazi着眼于新的Spring Security 5.0版本
  • 该epages博客上有一篇不错的文章， 介绍了Spring REST Docs和RAML的RESTful API文档。
  • InfoQ的Victor Grazi对反应堆工程师Simon Basle和市场经理Pieter Humphrey进行了很好的采访
  • 春季忍者Thomas Risberg集合了一个很酷的Project Riff-我们最近宣布的无服务器平台-演示，您可以在这里查看
  • Microsoft的公开博客查看了支持Microsoft Azure的 Pivotal Cloud Foundry的最新版本
