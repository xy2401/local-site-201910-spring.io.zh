Spring 的这个星期-2017年12月5日

工程
乔什·朗（Josh Long）
2017年12月6日

各位Spring粉丝们，大家好，欢迎您来参加JVM语言活动SpringOne Platform 2017， 这是春季的本周 ！有东西一个巨大的量覆盖，尤其是在SpringOne上平台的光，让我们开始吧！

  • Spring Cloud联合创始人Spencer Gibb刚刚宣布了Spring Cloud Finchley M5
  • Spring Tools的负责人Martin Lippert 刚刚宣布了Spring Tool Suite 4和与IDE无关的新语言服务器的首次预览 。另外，我做了特别的（很早！）Spring Tip着眼于这些主题 。
  • Spring Batch主管Michael Minella 刚刚宣布了Spring Batch 4.0.0
  • Dave Syer博士更新了Spring Security和Angular_以及Spring Boot和OAuth教程
  • 春季忍者Greg Turnquist刚刚宣布了Core，MongoDB和Apache Geode的Spring Session更新 。
  • Spring Cloud Data Flow负责人和联合创始人Mark Pollack博士刚刚宣布了Spring Cloud Data Flow 1.3.0。M3 。
  • Spring Integration负责人Gary Russell刚刚宣布了适用于Apache Kafka 2.1.0和1.3.2和2.0.2的Spring。
  • Spring Boot的联合创始人兼领导Phil Webb刚刚宣布了Spring Boot 2.0。M7
  • Spring IO Platform 负责人Andy Wilkinson刚刚宣布了Spring IO Platform Brussels SR6
  • Spring Integration忍者Artem Bilan 刚刚宣布了Spring Integration 5.0。GA
  • 上周，我在Spring Tips的一部分中研究了新的Spring Cloud Gateway项目。
  • Spring Integration忍者刚刚发布了Spring AMQP 1.7.5和2.0.1
  • Spring REST Docs安迪·威尔金森（Andy Wilkinson）刚刚宣布了Spring REST Docs 2.0.0
  • Spring Security主管Rob Winch刚刚宣布了Spring Security 5.0
  • Spring REST Docs领导Andy Wilkinson 还宣布了Spring REST Docs 1.2.3
  • Spring Boot忍者StéphaneNicoll刚刚宣布了Spring Boot 1.5.9
  • Spring Data忍者Mark Paluch 用Spring in Cloud Foundry展示了将绑定应用程序绑定到Hashicorp Vault的惊人发现
