春天的这个星期-2017年8月22日

工程
乔什·朗（Josh Long）
2017年8月22日

嗨，春季粉丝！欢迎来到春季本周的另一个安装。您能相信我们已经快到八月底了吗？时间过得真快！

上周和周末，我在中国杭州与阿里巴巴进行了访问，并与中国使用Spring Boot和Spring Cloud的一些团队进行了交谈。

本周，我在中国上海，一直在与客户交流，并与Microsoft中国的团队一起度过了几天，他们正在为Microsoft Azure进行Spring Boot和Pivotal Cloud Foundry集成。他们正在做一些了不起的工作，很荣幸与他们在一起。看看他们为各种Azure服务构建的所有出色的Spring Boot启动器。

（ 我和微软上海团队一起 ）

现在要去中国北京参加春季峰会了！我肯定希望在那里见到你！如果您在附近， 请随时在微信上对我ping 。

  • Spring消息传递大师Artemm Bilan刚刚宣布了适用于Apache Kafka 1.3M2的Spring 。该发行版是Spring Framework 5兼容2.0系列的子集。它集成了Apache Kafka 0.11.0.0。它介绍了KafkaTransationManager和KafkaListenerContainer ， 除其他事项外。
  • Spring Batch和Spring Cloud Task负责人Michael Minella刚刚宣布了Spring Cloud Task 2.0.0。M1是第一个与Spring Boot 2.0兼容的。这会升级所有其他依赖项，包括Spring Batch，Spring Cloud Stream等。
  • Spring Cloud贡献者Ryan Baxter刚刚宣布Spring Cloud Dalston SR3可用。这个新版本包括更新Zookeeper，Sleuth，Netflix和Contract模块。
  • Spring Boot 2.0中新的Spring Boot Actuator端点很辣 ！它们与基础架构无关。它们可以与Spring MVC，Spring WebFlux（一种反应性API）和Jersey一起使用（并且很容易集成其他JAX-RS实现）。
  • 这是社区驱动的Spring Boot入门者的一个很好的清单，这些入门者尚未进入Spring Initializr。
  • 我喜欢Dan Vega的这篇关于Spring Boot和Angular 4入门的文章 。简短而甜蜜！看看这个。
  • Simon Wirtz 研究如何构建反应式 @RestController和使用Kotlin语言WebFlux DSL的功能性反应性端点。
  • 查看跟踪幻灯片（他为Spring Cloud Sleuth做出了贡献！）和可观察性大师Adrian Cole在2017年dotScale会议上的演讲。
  • 我们看到许多Pivotal Cloud Foundry客户正在争论他们应该使用哪种基础设施来运行PCF。尽管PCF不在乎，但从技术上讲，您可能会对这篇文章比较两者感兴趣 。
  • 查阅David Syer博士的演讲《无服务器之路：作为 I / O 2017的应用程序功能》
  • Didispace博客上有一篇不错的（中文）帖子，介绍了将Spring Boot与Swagger 2.0一起使用
  • 共享了此工具SonarLint的 Oliver Gierke的技巧提示 ，可让您在IDE中以交互方式检测代码质量问题
  • 从严格意义上讲，这与Spring无关，但这是一个有趣的文章： Stripe如何处理API版本控制 。
  • 我喜欢我的好友和关键CTO Josh McKenty的访谈，内容涉及平台为何抵消了Conway定律的原因
