Helm is a package manager for Kubernetes, similar to apt, yum or homebrew. It is very easy to install and it greatly simplifies installation of an application and its dependencies into your Kubernetes cluster. The application package contents and configuration is defined in a chart. When you install it you can override any default configuration values. Helm will install any required services in addition to the ones defined in the chart. For Spring Cloud Data Flow, you have three required services: MySQL and Redis are used as the stores for Spring Cloud Data Flow state and RabbitMQ is used for the pipelines' messaging layer.

使用Helm轻松安装Kubernetes的Data Flow

工程
托马斯·里斯伯格
2017年8月31日

通过适用于Kubernetes的Spring Cloud Data Flow的新Helm图表，现在有了安装软件的简单得多的方法。

Helm是Kubernetes的软件包管理器，类似于apt，yum或homebrew。它非常容易安装 ，并且极大地简化了应用程序及其依赖关系在Kubernetes集群中的安装。在图表中定义了应用程序包的内容和配置。安装时，您可以覆盖任何默认配置值。除图表中定义的服务外，Helm还将安装所有必需的服务。对于Spring Cloud Data Flow ，您具有三个必需的服务：MySQL和Redis被用作Spring Cloud Data Flow状态的存储，RabbitMQ被用作管道的消息传递层。

一旦在本地系统上安装了Helm客户端，就可以继续进行Helm服务器部分Tiller的集群安装。

要安装分er ，请运行helm init 。完成此操作后，您应该可以通过kubectl get po --namespace kube-system命令。

现在，您可以为Spring Cloud Data Flow安装图表包了。由于该图表当前位于孵化器存储库中，因此您需要通过运行以下命令将此存储库添加到我们的Helm安装程序中：

helm repo add incubator https://kubernetes-charts-incubator.storage.googleapis.com
helm repo update

接下来，使用以下命令运行图表安装：

helm install --name my-release incubator/spring-cloud-data-flow

注意：

如果您在没有负载均衡器的集群（例如Minikube）上运行 ，则应覆盖服务类型以使用NodePort 。请使用以下命令：

helm install --name my-release --set server.service.type=NodePort \
    incubator/spring-cloud-data-flow

您应该看到以下输出：

您刚刚在Kubernetes集群的默认名称空间中创建了一个新版本 。注释部分提供了有关连接到新安装的服务器的说明。该应用程序及其所需的服务需要花费几分钟才能启动。您可以通过发出get pod -w命令。等待READY列为所有窗格显示“ 1/1”。完成后，您可以使用通过以下方式列出的外部IP连接到数据流服务器： kubectl get svc my-release-data-flow-server命令。默认用户名是user ， 密码为password 。

要查看您正在运行的Helm版本，可以使用helm list命令。是时候删除发布运行了helm delete my-release 。这会删除为发行版创建的所有资源，但会保留发行版信息，以便您可以使用helm rollback my-release 1命令。要完全删除发行版并清除所有发行版元数据，请使用helm delete my-release --purge

注意：

有一个问题，与用于所需要的服务在图表上更新，不断旋转产生的秘密。为避免这种情况，请在安装图表时为这些服务设置密码。您可以使用：

helm install --name my-release \
    --set rabbitmq.rabbitmqPassword=rabbitpwd \
    --set mysql.mysqlRootPassword=mysqlpwd \
    --set redis.redisPassword=redispwd incubator/spring-cloud-data-flow
