春天的这个星期-2017年8月29日

工程
乔什·朗（Josh Long）
2017年8月29日

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我在中国北京参加了神话般的春季峰会活动，来自中国各地的技术人员来到国会大厦，讨论云原生和春季的所有问题。然后我要去中国深圳和香港，拜访客户和史诗般的YOW！香港秀 ！

  • Pivotal已发布用于Cloud Foundry 的商业版Concourse CI管道工具 。检查一下，然后Spring Cloud Pipelines出来！
  • 我喜欢Bruno Leite的关于基于Spring Boot的REST API中的错误处理的文章。
  • 您可能会喜欢Baeldung的有关在Spring Boot上使用GraphQL的文章
  • 这个项目看起来很有趣，尽管我还没有使用过。它使您可以将ModelMapper与Spring Boot应用程序一起使用 。
  • 这是社区传奇人物汤米·路德维希 （ Tommy Ludwig ）的绝佳发现，他注意到Bitbucket现在正在Spring Boot上运行 ！
  • 由于我的JavaScript-fu并不是最强的，所以这篇文章是个老歌，但也是个好东西。它探讨了如何在由Spring Boot应用程序支持的websocket中使用Angular（2或更高版本）。
  • IntelliJ倡导者和超级巨星Trisha Gee着眼于IntelliJ IDEA 2017.2中旨在支持Spring Boot应用程序的新功能 。
  • 分布式跟踪大师Adrian Cole研究了如何在这次史诗般的microXchg演讲中适当地归咎于延迟 。
  • 本文是David Dawson演讲的回顾。它与Spring本身无关，与事件驱动的体系结构无关 。这是一个很好的讨论，您会发现Spring Cloud Stream，Spring Cloud Data Flow和Axon框架等技术中的支持工具。
  • 我喜欢Java 9对正则表达式的新支持 。
  • 美国政府的分支机构（成立于多年以前） 一直在寻找数字转换，他们正在寻找一些熟悉Cloud Foundry的云原生者来帮助他们 。
