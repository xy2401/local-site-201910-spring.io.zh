春天的这个星期-2017年8月8日

工程
乔什·朗（Josh Long）
2017年8月8日

嗨，春季粉丝们，欢迎来到春季本周的另一期！本周，本周三，我将与我的伙伴Matt Raible一起在带有Spring Boot和Angular 4的渐进式Web应用程序的VJUG（虚拟JUG）上进行演讲。加入我们 ！然后去纽约市，在那里我将与我的好友Marcin Grzejszczak合作进行一个令人兴奋的新项目。

  • Spring IO Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform Brussels-SR4 。此版本修订了Spring Batch 3.0.8，Spring Boot 1.5.6，Spring Data Ingalls SR6，Spring Framework 4.3.10，Spring Integration 4.3.11和Spring Retry 1.2.1的版本。
  • Spring Cloud Data Flow 1.3.0的第一个里程碑已经发布！许多不错的新功能，包括基于Angular 4的新颖的用户界面，更好的文档，更好的测试覆盖率等等。
  • Spring Cloud Pipelines负责人Marcin Grzejszczak刚刚宣布了Spring Cloud Pipelines 1.0.0。M6 。此版本支持称为流水线描述符的强大机制，以简化流水线定义。
  • Kubernetes v1.2.2的Spring Cloud Data FlowRELEASE现在可用。此版本改进了文档，更新了与Helm Chart对齐的服务器部署脚本，等等。
  • 作为RabbitMQ项目的管理员， 我们很乐意提供介绍RabbitMQ的课程 。
  • ErnestOrtuño引入了Trampoline这个开源社区项目，它使在本地计算机上启动和停止基于Spring Boot的服务变得更加容易。
  • 我喜欢RedMonk分析师James Governor的这篇文章，该文章着眼于Dell （Pivotal的父母）在市场上的地位。
  • 我被问及对版本API的见解，并且做对了事情 -我问Spring Data负责人Oliver Gierke，后者随后引用了这篇史诗性的文章， Roy Fielding有关版本，Hypermedia和REST 。
  • InfoQ在Spring Cloud Function框架上有一篇不错的文章，该框架支持“功能即服务”开发范例 。如果您还不了解这个令人兴奋的新项目，那么这是一本好书。
  • 对拟议的Java演变感兴趣？观看JVM Language Summit中的视频 。
  • 来自Devoxx UK的我的演讲Reactive Spring现在可以在线获得。看看这个！
  • 这篇中文帖子 （称为“ Spring Boot，Dubbo项目模拟测试踩坑与总结”）着眼于基于Spring Boot的阿里巴巴Dubbo服务框架中的模拟对象。（我认为）
  • 我喜欢这篇有关Atomist的InfoWorld文章，Atomist是集成了Chatops的开发人员生命周期管理解决方案。Atomist由Spring的创建者Rod Johnson创建，因此与Spring完美地集成在一起。
  • 在与Dropwizard，Restlet，Vert.x和Spark进行比较之后，NewRelic谈论了他们为什么选择Spring Boot作为其New Relic Alerts产品 。
  • 您是否看到过Spring Boot和Spring Cloud联合创始人Dave Syer博士的Spring Boot瘦启动器插件 ？这为您的Spring Boot应用程序支持了一个精简版式布局-在启动时解析库依赖关系。
  • Andrey Cheptsov谈论了IntelliJ IDEA 2017.2.1中的一个非常酷的新功能 ，这是一个Spring Run Dashboard，可让您一次管理和启动多个运行配置。太酷了！当然，如果您使用的是Spring Tool Suite，那么您在Spring Boot Dashboard中也会有类似的经验。
  • Spring Cloud Pipelines和Spring Cloud Contract的负责人Marcin Grzejszczak和我进行了一次讨论，讨论了由Devoxx Poland开发的以消费者为导向的合同 。该视频在线播放，值得一看！
