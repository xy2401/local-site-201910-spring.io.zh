春天的这个星期-2017年8月15日

工程
乔什·朗（Josh Long）
2017年8月15日

嗨，春季粉丝们，欢迎来到春季本周的另一期！这是忙碌的一周！上周，我开始与北卡罗来纳州的一组开发人员交谈，然后在旧金山与Matt Raible进行了VJUG演示，然后在圣路易斯与开发人员进行了交谈，然后飞往纽约拍摄了连续三天的视频在热工作室中（整个周末不少！）现在，当我飞往旧金山时，我希望在前往亚洲（杭州，上海，北京，深圳，香港和新加坡）大约四个星期之前，在地面上度过一个幸福的14个小时！不过，我不是唯一一个一直很忙的人！像往常一样，本周我们有很多工作要做，让我们开始吧！

  • 我的朋友和Spring Framework的负责人Juergen Hoeller在Devoxx PL 2017上做了演讲， 介绍了Spring的反应式编程 。希望您能看完这本书。
  • 我的好友Matt Raible和我上周做了一个演讲， 着眼于使用Angular和Spring Boot构建渐进式Web应用程序和强大的后端服务 。我和马特（Matt）进行了几次演讲，而且总是很有趣。我们现场直播。我写了后端服务，而他写了前端客户端。我使用Apache Groovy，Kotlin和Java编写后端服务。他在前端使用TypeScript。我认为马特是真正的英雄在这里，当然，因为即使我写了3个服务，马特最后写里姆斯更多打字稿获得客户的工作！ ：D该视频很疯狂。希望您能检查一下。
  • Cloud Native Java 终于完成并发布了 ！我和我的朋友肯尼·巴斯塔尼（Kenny Bastani）为O'Reilly写了一本小书-Cloud Native Java-关于如何从Spring Boot，Spring Cloud和Cloud Foundry的角度构建Cloud Native Java应用程序。这本书花了我们两年时间，已经由Spring团队的人们以不同的方式进行了回顾，并由Spring创作者Rod Johnson和Cloud Foundry传奇人物James Watters转发。这不是您可以买到的最糟糕的书，希望您能考虑一下。非常感谢您的耐心等待，春季粉丝！ :-)
  • SpringOne Platform 2017在短短四个月内在美丽的加利福尼亚州旧金山的Moscone中心举行！我要去，我希望你也能。如果您需要更令人信服的内容，请阅读此内容 。如果您已经确信，请单击该链接以获取可笑的动画.gif营销团队放在一起！
