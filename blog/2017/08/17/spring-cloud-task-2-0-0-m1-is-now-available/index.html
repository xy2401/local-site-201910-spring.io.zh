Spring Cloud Task 2.0.0。M1现在可用

发布
迈克尔·米内拉
2017年8月17日

我们很高兴地宣布Spring Cloud Task 2.0.0。现在可以通过Github和Pivotal下载存储库使用M1。非常感谢所有对此版本做出贡献的人。

该版本是与Spring Boot 2.0里程碑（特别是Spring Boot 2.0.0）兼容的第一个版本。此版本的M3）。通过该升级，还可以升级所有依赖项（Spring Cloud Stream，Spring Batch等），以及将基准Java版本升级到Java 8。

你怎么看？

我们期待您在Github ， StackOverflow中对这些新功能的反馈，或者直接通过Twitter @michaelminella给我！
