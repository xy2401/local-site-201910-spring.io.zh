春天的这个星期-2017年3月14日

工程
乔什·朗（Josh Long）
2017年3月14日

欢迎来到春季本周的另一期！本周，我在芝加哥做生意，为下周做准备， 这将是我在美国Devoxx举办的第一场活动！ 。我希望能在那里看到你！

  • 上周，我继续了Spring Tips系列，探讨了如何在今年夏天发布的Spring Framework 5中使用经典的 Spring MVC和Spring WebFlux实现服务器发送的事件。
  • 上周，Pivotal和Google宣布在BOSH之上与Cloud Foundry一起推出Kubo （ 运行和管理Kubernetes的功能）。 这意味着现在可以同时支持Pivotal Cloud Foundry和Kubernetes部署，同时支持云原生应用程序和低级容器编排。
  • Spring I / O平台负责人Andy Wilkinson宣布了Spring I / O平台Brussels SR1和Spring I / O平台Athens sr4
  • Disid刚刚宣布了Spring Roo 2.0。RC1
  • Spring I / O Platform负责人Andy Wilkinson刚刚宣布了依赖项管理插件Platform的更新版本
  • Spring Integration忍者Artem Bilan刚刚宣布了适用于AWS的Spring Integration扩展的更新版本
  • Spring Cloud忍者Ryan Baxter刚刚宣布了Spring Cloud Camden SR6
  • 不要错过3月29日（星期三）太平洋时间上午10:00的Michael Plod，他将通过Spring Boot谈论战略（域驱动）设计
  • Greg Turnquist宣布所有Spring指南现已更新，以反映IntelliJ IDEA的用法
  • Spring Security团队正在为OAuth制作一些令人惊奇的东西。看看超级初步 ，而不是最终的一切
  • MuleSoft和Pivotal合作伙伴支持Pivotal Cloud Foundry和MuleSoft AnyPoint之间的应用程序网络
  • Baeldung博客对Spring LDAP有很好的了解
  • 对于那些不了解的人， BOSH是一个开源工具，用于发布工程，部署，生命周期管理和监视分布式系统。它是由Cloud Foundry团队（其中许多人以前是Google员工）开发的，用于支持Cloud Foundry本身的部署。它是服务而不是服务器的程序包管理器。可以使用不可变的基础结构（与融合的基础结构选项（如Puppet或Chef）相对）从操作系统重新创建整个分布式系统。如果要管理大型分布式系统，则在系统下方需要这样的内容。您还可以在分布式模式下部署任意其他软件包， 例如Apache JMeter 。
  • Google的CRE支持和可靠性工程师以及Pivotal也在上周宣布，他们已经通过标签合作来支持和认证在Google Cloud上运行的Pivotal Cloud Foundry 。要了解为什么它如此重要，就意味着Pivotal Cloud Foundry （如今正变得越来越白热化）如今已融入Google的网站可靠性工程最佳实践中， 如此处所述 。
  • Michael Gruczel 在这篇德语文章中对使用Spring Boot构建REST API进行了很好的介绍
  • 我喜欢StefanPröll 一起使用Hibernate Search和Spring Boot进行介绍的帖子
  • 与Pivotal没有特别关系，我喜欢CAP定理之父Eric Brewer的这篇文章，阐述了CAP 如何应用于Google的Spanner数据库
  • 我真的很喜欢这份JAX调查报告，其中详细介绍了人们最感兴趣的框架 。 Spring MVC和Spring Boot在各自类别中排名第一。非常感谢，Java社区！
  • 观看本次讲座， 由Spring Data为Apache Geode贡献者John Blum和Luke Shannon介绍Apache Geode 。
  • Couchbase开发倡导者Laurent Doguin对使用Test Containers和Spring测试Spring Data Couchbase应用程序进行了很好的介绍
  • 亚历山大·科萨列夫（Alexander Kosarev）的这篇俄语文章很好地介绍了Spring Data Repository方法技巧
  • 我喜欢Oracle高级应用程序工程师Orlando L. Otero的关于使用Spring Cloud进行多版本服务发现的文章

  • Oliver Antoine完美集成了Spring Data Cassandra应用程序的集成测试
