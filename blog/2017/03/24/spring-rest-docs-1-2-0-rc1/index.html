Spring REST文件1.2.0。RC1

发布
安迪·威尔金森
2017年3月24日

谨代表每个贡献者，宣布Spring REST Docs 1.2.0。RC1已发布，可从https://repo.spring.io/milestone/获取 。

什么是新的？

在发行说明中可以找到1.2新增功能的完整概述。以下是一些重点。

改进的Asciidoctor集成

REST Docs现在有了一个新模块， spring-restdocs-asciidoctor ，这样可以更轻松地在文档中使用生成的代码段。一个新的宏意味着您可以在一行中为同一操作导入多个片段。对示例的此更新显示了采用宏的好处。

更好地支持记录大型和复杂的有效载荷

为了使您的文档更易于使用，现在可以拆分大型和复杂有效负载的文档。可以分别记录有效载荷的各个部分，从而使您的文档可以指导用户完成重要的工作。

请求和响应正文的片段

现在，默认片段中包含了记录请求或响应正文的片段。您还可以显式使用这些代码片段来记录大型或复杂主体的子部分。
