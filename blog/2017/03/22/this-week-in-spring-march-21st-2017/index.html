春天的这个星期-2017年3月21日

工程
乔什·朗（Josh Long）
2017年3月22日

欢迎来到春季本周的另一期！本周，我要拜访客户，并出席在加利福尼亚州圣何塞举行的第一场Devoxx US活动 。如果您在附近，不要犹豫，在博览会大厅的Pivotal摊位打个招呼。

和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Spring Cloud Dalston RC1现在可用！趁热烧碎！
  • Spring Cloud Task的贡献者Glenn Renfro刚刚宣布了Spring Cloud Task 1.2.0。M2
  • Marius Bogoevici刚刚宣布了Spring Cloud Stream切尔西RC1
  • Spring Cloud Vault 负责人Mark Paluch刚刚发布了1.0。项目的RC1
  • 上周，我继续了我的Spring Tips系列，这次是看反应式HTTP客户端， WebClient ，在Spring Framework 5的初步构建中（用于Spring Boot 2.0的初步构建中）。
  • 通过我一直以来最喜欢的演讲者之一，InnoQ的Michael MichaelPlöd观看有关领域驱动设计的网络研讨会
  • 该博客着眼于/auditevents Spring Boot 1.5中的新促动器功能
  • 观看 Pivotal的Daniel Carwin录制的此网络研讨会的录像， 了解RabbitMQ世界中正在发生的事情
  • 我们的朋友Christophe F.继续回顾了Pivotal Cloud Foundry中的新容器网络及其性能
  • 我的朋友丹尼尔·布莱恩特（Daniel Bryant）的微服务周新闻综述一直做得很好
  • Spring社区的朋友NicolasFränkel返回了一个有趣的帖子，他尝试在非Spring Boot应用程序中使用Spring Boot的Actuator 。我不建议这样做！但是，这是使执行器框架采用何种方式的非常有趣的剖析。让尼古拉斯带您踏上旅途。
  • 我喜欢这个博客， The Cloud Native Opportunity
  • 我知道这与Pivotal生态系统关系不大，但这是Adobe的一篇很棒的文章，谈到设计概念的演变
