Spring Cloud Stream Elmhurst。M3 / 2.0.0。M3发布公告

工程
奥列格·朱拉库斯基（Oleg Zhurakousky）
2017年11月9日

我们很高兴宣布2.0.0。Spring Cloud Stream Elmhurst发布火车的M3版本。

Spring Cloud Stream Elmhurst 2.0.0。M3可在Spring Milestone存储库中使用。发行说明包括有关与Spring Boot，Spring Cloud，Spring AMQP和Spring for Apache Kafka的版本兼容性的相关信息。

以下各节列出了此版本中包含的新功能和改进：

改进的内容类型分辨率

内容类型的分辨率已得到改进，并在通道拦截器中得到了整合，从而使其可用于需要应用程序级自定义的情况。另外，根据社区反馈，我们现在将默认内容类型设置为application/json 。但是，仍然可以通过以下方式覆盖它BindingProperties或者可以将其与Message来自上游应用程序。

支持Apache Kafka 0.11+

该发行版完全支持Apache Kafka 0.11+，包括对本机头的支持。

尽管我们强烈建议升级您的Kafka经纪人，但此版本仍支持0.10 Kafka经纪人，但是您需要设置headerMode到none要么embedded 。

关于样本的注释

与往常一样，学习任何新技术或框架的好方法是对其进行尝试，并将其应用于用例。我们看到来自社区和客户的各种用例。为了进一步改善学习体验，我们将继续添加更多示例 ，并且最近添加了一些新示例 。使用它或叉子；另外，通过提交示例与更广泛的社区互动来分享您的经验。欢迎您的贡献！

下一步

我们希望在SpringOne Platform 2017之前达到一个稳定的发布候选状态。请尝试一下并给我们反馈，并经常这样做！

与往常一样，我们欢迎您提供反馈和意见，因此请通过Stackoverflow或GitHub或通过Gitter与我们联系。
