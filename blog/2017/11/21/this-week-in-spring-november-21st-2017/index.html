春天的这个星期-2017年11月21日

工程
乔什·朗（Josh Long）
2017年11月21日

嗨，春季粉丝！欢迎来到春季本周的另一期！您能相信我们距2018年不到六周吗？当您玩得开心时光飞逝！

本周，我在洛杉矶与社区成员交谈，然后在美国，由朋友和家人包围的感恩节假期。感恩是一个反思我们所感恩的机会。我敢肯定，当我说我们对您表示感谢时，我会在这里代表我们所有人在Spring和Pivotal团队中发言。感谢您使它成为技术领域最令人惊奇，最活跃，令人振奋和最有趣的社区。如果您在美国，感恩节快乐！无论哪种方式，本周我们都有很多工作要做，让我们开始吧。

  • Spring Cloud联合创始人兼联合创始人Spencer Gibb刚刚宣布了Spring Cloud Finchley M4 。这个新版本更新了所有内容以支持Spring Boot 2.0.0。M6。与往常一样，这是一个里程碑式的发行，它为您提供了一个尝试的机会。
  • 在上周的Spring Tips中，我研究了Spring Boot构建插件的一些更奇特的功能，包括胖 .jar和薄 .jar s。
  • 查看Spring Boot社区传奇人物NicolasFränkel的演讲，介绍Spring Boot的幕后机制
  • Abhinav Mehla 在Pivotal Cloud Foundry上介绍了MongoDB Enterprise的可用性。这太棒了！
  • 这真是个好消息！罗德·约翰逊（Rod Johnson）的初创公司Atomist旨在启动围绕应用程序开发的端到端生命周期自动化， 并已推出并宣布了其自动化平台 。恭喜罗德！这是关于原子论从隐身中崛起的另一篇好文章。如果您想了解更多信息，请加入他的SpringOne Platform
  • 您在圣路易斯地区吗？与我的Spring Developer倡导者Mario Gray和我一起参加Bootiful Testing ，2017年12月13日。
  • Auth0有一篇有趣的文章，介绍了如何使用Gradle构建手动嵌入Apache Tomcat 5的Spring 5应用程序。（我认为您最好使用Spring Boot 2，如果它在CLASSPATH上，它会自动配置Apache Tomcat）。
  • 我们的@SpringCentral Twitter帐户甚至在我意识到之前就热闹地宣布了这个消息 ，但是我最近在Java refcard中更新了我的DZone微服务，现在可以使用 。
  • Spring Data负责人Oliver Gierke 与该示例应用程序 Salespoint 共享此示例应用程序 ， Salespoint是基于Spring Boot的POS框架，可用于德累斯顿工业大学（我认为是德国德累斯顿工业大学）的教学目的，Oliver在那里教书。这是项目的主页和存储库 。
  • 这篇文章介绍了T-Mobile如何重构其主要的整体应用程序， 并介绍了Cloud Foundry来改变其文化 。
  • OAuth专家Will Tran在这个出色的InfoQ视频中讨论了如何使用OAuth 2.0实施微服务的安全策略 。
  • Neo4J传奇人物Michael Hunger向我们介绍了使用Neo4J的Cypher语言和基于Devoxx会议演讲的数据集进行的网络分析 。如果您正在使用Neo4j（也许与Spring Data Neo4J一起使用），那么这是您阅读的大师班。
  • 当云平台演示Spring Boot应用程序时，我会喜欢它。
