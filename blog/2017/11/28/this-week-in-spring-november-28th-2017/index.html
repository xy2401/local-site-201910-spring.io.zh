Spring 的这个星期-2017年11月28日

工程
乔什·朗（Josh Long）
2017年11月28日

嗨， Spring 粉丝们，欢迎来到 Spring 本周的另一个装置！本周，我在旧金山，然后去德克萨斯州奥斯汀参加史诗般的G3峰会 。如果您在附近，请打个招呼。然后，直接回到旧金山，参加下周在全球范围内举行的首要敏捷，操作，应用和以云为中心的活动，即SpringOne Platform 2017 。千万不要错过这个！

  • 在上周的Spring Tips中 ，我将介绍基于Spring的应用程序中的测试。看看这个！
  • Spring Framework 负责人Juergen Hoeller在这篇博文中深入探讨了我们决定终止对Spring 的一流JDK 6支持的决定 。
  • Spring Framework负责人和Spring联合创始人Juergen Hoeller刚刚发布了Spring Framework 5.0.2和4.3.13 。建议对新版本进行改进。特别要注意的是，我们已不再支持Castor XML支持，而实际上已弃用了对JDK 6的支持。
  • Spring Data负责人Mark Paluch刚刚宣布了Spring Data Ingalls SR9和Spring Data Kay SR2 。新版本已修复90多个问题，建议所有用户升级。
  • Spring Cloud联合创始人Spencer Gibb 刚刚宣布了Spring Cloud Edgware。可发布 。新版本更新了Spring Cloud Bus，Spring Cloud Sleuth和Zipkin，Spring Cloud Contract，Spring Cloud Security和Spring Cloud Stre
  • Spring Integration忍者Artem Bilan刚刚宣布适用于AWS 1.1 GA的Spring Integration现在可用。该版本包括对S3，Kinesis和DynamoDB的强大支持。
  • Mark Pollack博士刚刚宣布了Spring Cloud Skipper 1.0 M2 。上一篇文章：“ Skipper是一种轻量级工具，可让您发现Spring Boot应用程序并管理它们在多个Cloud Platform上的生命周期。您可以独立使用Skipper或将其与Continuous Integration管道集成在一起，以帮助实施Continuous Deployment的实践。”
  • Spring Security OAuth忍者Joe Grandja刚刚发布了Spring Security OAuth 2.2.1 。此版本主要是一个错误修复版本
  • 好吧，这篇文章本身与Spring没有太多关系，但是仍然值得一提。关于如何激发和激励敏捷哲学的知识非常有趣， 那就是学习和改进 。
  • 我真的很喜欢Arne Vandamme的这个帖子，关于 Spring 的提示和技巧CacheManager抽象 ，尤其是支持它的注释。
  • 我喜欢这篇关于埃森哲和Pivotal合作伙伴关系的德语文章。
  • 这是对几天之内在美丽的旧金山举行的SpringOne Platform 2017会议活动中即将发布的一些精彩内容的很好看。
  • 社区朋友Eberhard Wolff在这个有趣的视频中介绍了使用Spring Boot和Java构建微服务 。看看这个！
  • 迈克尔·西蒙斯（Michael Simons）在本文中介绍了与Spring Boot，Gradle和Docker Compose的集成测试。一如往常，值得一看。
  • 我真的很喜欢这篇有关JUnit 5和AssertJ集成的文章。向我的朋友Billy Korando推荐此链接 。
