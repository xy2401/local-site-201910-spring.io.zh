春天的这个星期-2017年11月7日

工程
乔什·朗（Josh Long）
2017年11月7日

嗨，春季粉丝！欢迎来到春季本周的另一期！本周，我在比利时安特卫普参加令人惊叹的Devoxx（比利时）活动。我将与我的朋友Matt Raible和Mark Heckler共同介绍渐进式Web应用程序和Spring。我将和很多Pivotal和Spring的队友一起加入，因此请务必检查时间表。

本周晚些时候，我将与Grails联合创始人Jeff Scott Brown举行联合网络研讨会-Grails for Spring Boot开发人员 。

然后，前往摩洛哥卡萨布兰卡参加Devoxx MA活动。

如果您在任何一个活动中， 都可以打个招呼 ！

  • Spring Cloud Pipelines负责人Marcin Grzejszczak 刚刚宣布了Spring Cloud Pipelines 1.0.0。M7 。这个新版本首次发布了Kubernetes对Jenkins和Concourse的支持。除其他功能外，它还通过150多次Bash测试大大提高了代码覆盖率。
  • Spring Cloud联合负责人Spencer Gibb刚刚宣布Spring Cloud Finchley M3现已上市 。Spring Cloud Finchley在Spring Framework 5中集成了反应式API，并引入了一个名为Spring Cloud Gateway的新项目。
  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 5.0.0。RC1 。这个版本很大，因此您一定要检查新版本，看看代码现在是否适合您。它包括Spring Web Flux和Spring Framework 5的集成，OAuth 2.0集成（直接在Spring Security本身中！），密码存储更新，UnboundID LDAP内存中支持等等。
  • Spring Security和Spring Session的负责人Rob Winch 刚刚宣布了Spring Session 2.0.0。RC1 。新版本对即将发布的2.0.0作了最后的修改。通过简化与Servlet API的集成，并增加对Redis会话清除cron流程的支持，来进行发布。
  • Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration 5.0 RC1 。新发行版除其他事项外，还修复了20个问题，这些问题是基于HTTP WebFlux的适配器，并且Java DSL的行为发生了变化，以支持为DSL中创建的每个组件注册新的bean定义。
  • Spring Batch主管Michael Minella刚刚宣布了Spring Batch 4.0.0。RC1包括更新的基准API，新的构建器API，更新的Java配置和文档。总而言之，值得升级。那些使用Spring Boot的人应该得到一些好处，踢一下轮胎，然后立即尝试。
  • Spring Boot忍者StéphaneNicoll 刚刚宣布了Spring Boot 2.0.0。M6最初支持HTTP / 2，改进了对基于WebFlux的应用程序的支持（添加了TLS配置和错误页面），并且首次实现了Kotlin集成。
  • 在最新版的Spring Tips中，我研究了Spring Shell项目 。
  • Thymeleaf 团队刚刚发布了Thymeleaf 3.0.9 ，其命中率有所提高。 StandardCache ，改进的限制表达式评估模式以支持更好的安全性，以及针对Spring Framework 5集成的一些错误修复。
  • Pivotal自己的Toshiaki Maki撰写了一篇很棒的日语文章，介绍了如何使用Pivotal Cloud Foundry指标转发器服务将指标从Spring Boot应用程序转发到Pivotal Cloud Foundry指标服务。
  • 我对这个演讲 （来自Spring Atlanta，亚特兰大 ）的建议“ Spring Notn Invent It！”不够推荐。劳拉摩尔（Laura Moore）着 。我只希望她能再继续几个小时！很高兴这个谈话存在。
  • 我和其他建筑师克里斯·理查森，丹尼尔·布莱恩特
    Glenn Engstrand和Alex Silva参加了有关服务编排与编排的InfoQ虚拟小组 。这很有趣，希望您喜欢这个讨论。
  • InfoQ的Amit K Gupta对Spring Tool Suite 3.9.1中的新功能做了很好的撰写。
  • 严格地说，这与Spring无关，但是我总是喜欢Java语言中新功能的讨论和介绍。Brian Goetz的这篇文章既是一本精彩的书，也是对Java未来版本中“数据类”如何工作的深刻见解。向我的朋友和Java粉丝Billy Korando分享这些技巧。
  • 我喜欢Spring Framework和Kotlin ninja Sebastien Deleuze的这个示例项目，该示例项目演示了Spring Boot中的一些API及其Kotlin的用法。
  • Baeldung博客对Micrometer度量标准发布库进行了很好的介绍。Micrometer是一个Pivotal项目，用作Spring Boot 2.0对Actuator的支持的指标骨干。
  • 有一个很棒的NetBeans Spring Boot插件版本1.6.1 ，它支持新的提示和快速修复。它将警告您使用@ConfigurationProperties没有存在spring-boot-configuration-processor ，删除不推荐使用或未知的属性，以及更新Spring Boot 1.5.8等。
  • 您现在可以在Spring Initializr上使用Spring Shell和Spring Cloud Contract 。
