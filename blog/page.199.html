Spring XD 1.0.1发布

发布
马克·波拉克
2014年10月2日

我非常荣幸地代表Spring XD团队宣布Spring XD 1.0.1的全面上市！

此版本包括错误修复和增强功能以及一些新功能：

  • 对Admin Server的HTTPS访问和身份验证
  • UI中的群集和流视图
  • 配置自定义模块的位置
  • 空水槽

您可以下载zip发行版或使用homebrew在OSX上安装。在RHEL / CentO上，您可以使用yum进行安装 。

反馈非常重要，因此请通过以下方式与问题和评论联系
* StackOverflow spring-xd标签
* Spring JIRA或GitHub问题

Spring Framework 4.1.1发布

发布
于尔根·霍勒（Juergen Hoeller）
2014年10月1日

尊敬的Spring社区，

我很高兴地宣布，Spring Framework 4.1.1现在可用。作为4.1版本中的第一个维护版本 ，此版本包含针对4.1 GA报告的大量回归和其他问题的修复程序。非常感谢所有的问题记者！

4.1.1还提供了一些次要的增强功能 ，例如：一致地处理Java 8的Optional类型，具有改进的默认值的Jackson ObjectMapper构建器类，一些要求的JDBC改进以及特定的性能改进 。

网络研讨会：Spring和OAuth2的微服务安全性

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年10月1日

演讲者：Dave Syer

使用Spring和OAuth2的微服务的安全性OAuth2是一种轻量级的安全协议，非常适合与HTTP结合使用，HTTP是许多现代体系结构的核心。Spring Security OAuth2具有许多新功能，其中最重要的一点就是2.0版中的@Configuration支持。将它们与Spring Boot结合使用，您将拥有一个平台，该平台可以通过大约20行代码为您提供安全的HTTP服务应用程序。此演示文稿展示了现代Spring堆栈中快速开发和生产就绪功能的结合如何完美结合在由微服务组成的系统中开发安全组件。我们探索了Spring OAuth2中的新功能，指导您选择使用哪种功能以及何时使用，并展示了快速启用它们的难易程度。

2014年9月26日
