春天的这个星期-2012年12月11日

工程
乔什·朗（Josh Long）
2012年12月11日

欢迎来到春季本周的另一期！我们在周末完成了SpringOne China ，今天是在班加罗尔举办的SpringOne India和在海得拉巴举办的SpringOne两项活动的第一场。表演真是太神奇了！特别是，我们非常喜欢这里涌现的美食，景点和令人惊叹的社区！











我在Cloud Foundry上的Spring上发表演讲时，一位与会者对我开了一枪 Mark Pollack博士，本人，Gary Russell和Chris Richardson
在北京的一家餐馆
我在班加罗尔演讲，人们在舞台上提问和问候。我请最后一批人和我合影。 LtoR：Gary Russel，Josh Long，Jennifer Hickey，Jeremy Grelle，Oliver Gierke，Chris Richardson –整个SpringOne印度阵容

 


  1.  Roy Clarkson has announced Spring for Android 1.0, which     adds support for Jackson 2.x in RestTemplate through the new MappingJackson2HttpMessageConverter. It also addresses several bugs and compatibility issues with RestTemplate and Android Jelly Bean.
    2. 

  2. The TomcatExpert.com portal has a nice post on Spring Insight Developer to analyze code, install it with Tomcat, and extend it with plugins. 
    3. 


  3. Spring Scala lead Arjen Poutsma has a nice post introducing Spring Scala.  
    4.

  4. Trifork上的人们有一个关于正确测试Spring MVC控制器的好博客。

  5. 在这里我要说老实话-我确切地不知道这篇文章的意思，因为它是用韩语编写的，并且Google翻译不是特别有用。看来作者发现了与Spring MVC的@SessionAttribute有关的警告，并提供了解决方法。希望这对您读韩语很有用！

  6. 您是否在.NET环境中使用RabbitMQ？在EasyNewQ上查看此帖子， EasyNewQ是RabbitMQ的.NET客户端。

  7. GoSmarter博客上有一篇有趣的文章，介绍了如何设置一个假FTP服务器来对Spring Integration FTP流进行单元测试 。该博客还提供了一篇关于在Spring Integration中模拟SFTP支持的不错的文章。

  8. GoSmarter博客也有一篇关于使用Spring Security和容器管理的安全性的好文章

  9. 同样来自GoSmarter博客，这是一篇关于使用Spring Webflow进行测试驱动的开发的不错的文章。

  10. MonoCaffe博客上有一篇关于使用Spring MVC JSP标签的文章，该文章允许在Bootstrap的输入字段中预先添加和添加图标 。

  11. Zen Java博客上有一篇不错的文章，介绍如何创建从Java FX使用的RESTful Spring MVC服务 。这真的很滑。

  12. Cafe Techno博客上有一篇不错的文章，比较了Spring MVC和Struts 。我想说，春季MVC的票价非常好！

  13. Java博客快速指南上有一篇关于Spring MVC国际化的好文章。
