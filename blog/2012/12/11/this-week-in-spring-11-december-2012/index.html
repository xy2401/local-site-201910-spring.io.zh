春天的这个星期-2012年12月11日

工程
乔什·朗（Josh Long）
2012年12月11日

欢迎来到春季本周的另一期！我们在周末完成了SpringOne China ，今天是在班加罗尔举办的SpringOne India和在海得拉巴举办的SpringOne两项活动的第一场。表演真是太神奇了！特别是，我们非常喜欢这里聚集的美食，景点和令人惊叹的社区！











当我在Cloud Foundry上的Spring上发表演讲时，一位与会者对我开了一枪 Mark Pollack博士，本人，Gary Russell和Chris Richardson
在北京的一家餐馆
我在班加罗尔演讲，人们在舞台上提问和问候。我请最后一批人和我合影。 LtoR：Gary Russel，Josh Long，Jennifer Hickey，Jeremy Grelle，Oliver Gierke，Chris Richardson –整个SpringOne印度阵容

 


    <LI> Roy Clarkson has announced <a href = "http://www.springsource.org/spring-android/news/1.0.1-released">Spring for Android 1.0</a>, which     adds support for Jackson 2.x in <code>RestTemplate</code> through the new <code>MappingJackson2HttpMessageConverter</code>. It also addresses several bugs and compatibility issues with <code>RestTemplate</code> and Android Jelly Bean.</LI>
<LI>The <a href="http://www.tomcatexpert.com">TomcatExpert.com portal</a> has a nice post on Spring Insight Developer to analyze code, <a href="http://www.springsource.org/node/3761">install it with Tomcat, and extend it with plugins</a>. </LI>

<LI>Spring Scala lead Arjen Poutsma has a nice <a href="http://blog.springsource.org/2012/12/10/introducing-spring-scala/">post introducing Spring Scala</a>.  </LI>

  1. Trifork上的人们有一个关于正确测试Spring MVC控制器的好博客。

  2. 在这里我要说老实话-我确切地不知道这篇文章的意思，因为它是用韩语编写的，并且Google翻译不是特别有用。看来作者发现了与Spring MVC有关的警告@SessionAttribute并提供了解决方法。希望这对您读韩语很有用！

  3. 您是否正在从.NET环境使用RabbitMQ？在EasyNewQ上查看此帖子， EasyNewQ是RabbitMQ的.NET客户端。

  4. GoSmarter博客上有一篇有趣的文章，介绍了如何设置一个假FTP服务器来对Spring Integration FTP流进行单元测试 。该博客还提供了有关在Spring Integration中模拟SFTP支持的不错的文章。

  5. GoSmarter博客也有一篇关于使用Spring Security和容器管理的安全性的好文章

  6. 同样来自GoSmarter博客，这是一篇关于使用Spring Webflow进行测试驱动的开发的不错的文章。

  7. 该MonoCaffe博客对使用Spring MVC的JSP标签，让一个漂亮的后预未决和图标追加到引导的输入字段 。

  8. Zen Java博客上有一篇不错的文章，介绍如何创建从Java FX使用的RESTful Spring MVC服务 。这是真的光滑。

  9. Cafe Techno博客上有一篇不错的文章，比较了Spring MVC和Struts 。我想说，春季MVC的票价非常好！

  10. 在一个快速指南的Java博客具有良好的岗位上Spring MVC的国际化 。
