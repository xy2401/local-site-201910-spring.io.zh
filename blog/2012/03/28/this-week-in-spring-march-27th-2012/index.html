春天的这个星期-2012年3月27日

工程
乔什·朗（Josh Long）
2012年3月28日

欢迎来到春季本周的另一个装置。
和往常一样，我们有很多内容要讲。随着这篇文章的上升，
Cloud Foundry Open Tour正在北京进行，很快就会来到您附近的城市。该节目是一个非常难得的机会，可以从专家那里了解有关Cloud Foundry和Spring的更多信息-不要错过，请立即注册。

  1. Spring Web花花公子Rossen Stoyanchev宣布Spring Web Flow 2.3.1已发布 。这是一个维护版本，具有对Spring 3.1.1和JavaServer Faces 2.1.7的升级以及许多错误修复。

  2. SpringSource工具套件负责人Martin Lippert宣布了针对Eclipse 1.0的Cloud Foundry集成的发布。
    该版本为所有Eclipse发行版（包括SpringSource Tool Suite）带来了完整的，紧密的Cloud Foundry集成。

  3. 马丁 还宣布发布SpringSource Tool Suite版本2.9.1 ，其中包含新功能和错误修复。

  4.     Mark Fisher and Thomas Risberg's epic talk from SpringOne 2GX 2011  - Architecture Choices for Scalable Cloud Apps -  that introduces how to build scalable architectures in the cloud using technologies like Spring Integration and Cloud Foundry is now up on InfoQ. 
      
    5.

  5. Maciej Walkowiak的博客介绍了如何将Spring 3.1配置文件与一些自定义Tomcat配置结合使用来激活Spring配置文件，而无需更改已部署的二进制文件。


  6. Michal Jastak撰写了一篇精彩的文章，介绍了如何使用Spring MVC 3.1对flash属性的支持 。


  7. Tobias Flohre又来了，这一次在Spring Batch上有两个帖子。第一篇文章介绍了Spring Batch中事务的基础知识，第二篇文章介绍了重新启动基于游标的读者和作家的一些要点 。

  8. Artur Mkrtchyan的精彩文章介绍了如何安装Redis （快速，高度优化的数据结构服务器）和如何使用Spring Data Redis （Spring Data伞项目的一部分，该项目有助于访问各种所谓的NoSQL。和大数据存储） 来构建与Redis对话的Spring应用程序 。

  9. Vijay Rawat详细介绍了如何在Spring应用程序中使用Memcached来实现会话复制 。这种方法依赖于开发人员专门委派Memcached来实现其代码中的Session持久性。在此可行的同时，我希望用户可以看看更透明的选项。Tomcat支持可插入会话存储引擎。有许多可用的实现，包括基于Redis的实现和基于GemFire的实现 。

  10. 说到Apache Tomcat，博主Ramki有一篇有趣的文章，介绍了如何使用Apache Tomcat，Apache和ModJK设置虚拟主机（“ vhosts”） 。
