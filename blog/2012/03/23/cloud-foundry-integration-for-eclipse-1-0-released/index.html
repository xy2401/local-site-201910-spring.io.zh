面向Eclipse 1.0的Cloud Foundry集成发布

新闻与活动
马丁·利珀特
2012年3月23日

我很高兴宣布Cloud Foundry Integration for Eclipse 1.0的第一版。

此版本具有：


  • 直接从您的工作区部署，更新，启动，停止应用程序

  • 支持Java / Web，Java / Spring，Groovy / Grails和Scala / Lift应用程序

  • 服务创建和绑定（直接在部署时或稍后）

  • 实例扩展和统计

  • 远程文件浏览直接集成

  • 对Micro Cloud Foundry的全面调试支持

  • 改进的增量更新性能

  • 集成到Eclipse和STS中

最后但并非最不重要的一点是，该项目现已在EPL下开源，可从GitHub获得：
https://github.com/SpringSource/eclipse-integration-cloudfoundry

您可以使用STS仪表板将其安装到STS（需要2.9.0或更高版本）中，也可以使用Eclipse Marketplace将其直接安装到纯Eclipse JEE软件包中（建议使用Indigo）。

注意：如果您安装了以前版本的Cloud Foundry工具（旧的里程碑或夜间版本，上面安装了旧的STS 2.7.0），请在安装新版本之前将其卸载。自从我们移至GitHub以来，这是一个新的独立项目，除了您拥有的旧内容外，还将安装它。因此，请先卸载旧功能。

更详细的用户文档将很快发布 http://start.cloudfoundry.com/ 。敬请关注！
