Spring 本周：2012年3月6日

工程
亚当·菲茨杰拉德
2012年3月7日

欢迎来到 Spring 本周的另一期。本周刚刚发布的Spring for Apache Hadoop项目有很多令人兴奋的地方，该项目为Apache Hadoop提供了非常非常强大的Spring API集成，Apache Hadoop是一个最初旨在支持map-reduce样式的开源框架。 批处理解决方案 。


  1. Costin Leau 宣布了适用于Apache Hadoop 1.0.0的Spring，从而拉开了帷幕。上周M1 。我无法比Costin更好地解释它，所以我不会打扰。从他的帖子：

    无论是编写独立的原始MapReduce应用程序，与整个企业中多个数据存储中的数据进行交互，还是协调HDFS，Pig或Hive作业的复杂工作流，或在两者之间进行任何协调，Spring for Apache Hadoop始终如一。 Spring哲学提供了简化的编程模型，并解决了由基础架构引起的“意外复杂性”。

    你在等什么？走！看看这个！完成后，您也可以查看该事件的其他报道。
    如你所料，很多的 人 在博客中 关于 它 ， 太 。
  2. What are you guys doing in two days, on March 8th, 2012? I'll be at the Native Android Development with Spring for Android webinar that introduces native Android development practices, resource management and the Spring for Android integration library, which makes the whole process more natural for developers that wish to interface with services on the server side. There are, as usual, two editions, one for North America and one for Europe. Don't miss it!
  3. The inimitable Oleg Zhurakousky today announced the Spring Integration Scala DSL. The DSL extends the Spring Integration API to Scala developers using a very concise, idiomatic Scala-based DSL.

    就在您认为Spring Integration不能再简明的时候！

  4. SpringSource Tool Suite 2.9.0版的最新更新现已发布，建议所有Spring和Grails开发人员进行升级。该版本提供了对Spring Integration 2.1，Grails 2.0.1和Spring Roo 1.2.1的最新更新，并且在后台对基准Eclipse平台进行了一些更新。

  5. 说到Android的Spring，Roy Clarkson刚刚宣布了最新版本1.0.0。Spring for Android库的RC1 。除其他外，新版本还支持Spring Social 1.0.2。发布和Spring Security 3.1.0。发布并更新RestTemplate实现，与Spring 3.1的实现兼容。


  6. 项目负责人Michael Hunger刚刚宣布了Spring Data Neo4J项目的最新里程碑版本2.1.0 。有很多新功能和更新，因此请查看发行说明。

  7. 在InfoQ上， Sam Brennan和Rossen Stoyanchev在 SpringOne2GX的 基于Spring（和Spring MVC）的测试方面的精彩演讲已经结束。Rossen是Spring MVC的主要提交者之一，而Sam是Spring核心测试支持的主要提交者，所以……。还有什么要说的？这些家伙是巨人。走。学习。我会跟随！

  8. 本文提供了有关如何在基于RMI的远程处理中使用Spring的介绍。

  9. Cloud Hadoop博客上有一篇很棒的文章介绍如何注入Java 5 enum使用Spring 。

  10. InfoQ 在Thymeleaf 2.0上发布了很好的发布公告，Thymeleaf 2.0是另一个开放源代码项目，填补了SpringSource之外的空白，或填补了Spring的空白。这些项目中最好的，例如Thymeleaf，本身就是模范，值得我们关注。正如本综述的长期读者所知道的，Thymeleaf是一种模板引擎，与Spring MVC的集成故事非常出色。 View和ViewResolver机械，绝对值得一看的是那些受JSP支持的Spring MVC开发人员！
