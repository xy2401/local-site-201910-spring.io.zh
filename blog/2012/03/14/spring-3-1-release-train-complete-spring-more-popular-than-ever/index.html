Spring 3.1 Release Train完成，Spring比以往任何时候都受欢迎

发布
亚当·菲茨杰拉德
2012年3月14日

在今天的新闻发布中，SpringSource和VMware 宣布 Spring 3.1 Release Train已完成。Spring Integration ， Spring Security ， Spring Batch ， Spring Data ， Spring Mobile和Android的Spring现在都支持Spring 3.1，从而使企业开发人员可以更轻松，更快速地构建下一代企业应用程序。

该新闻稿还强调了Evans Data Research关于使用Spring的日益普及和好处的重要发现。如今，超过三分之二的Java开发人员正在使用Spring或计划在未来两年内使用Spring，而使用Spring的Java开发人员的生产率提高了50％。

SpringSource要感谢所有使该版本成为可能的令人惊叹的社区成员。社区贡献是许多项目的重要组成部分，所有尝试过里程碑和候选版本的用户都在很大程度上确保了Spring技术的高质量和生产准备。许多其他令人兴奋的项目已经在进行中，例如最近宣布的Spring for Apache Hadoop项目。邀请社区成员参加即将举行的SpringOne on the Road活动，与Spring专家见面，并了解Spring的最新功能。
