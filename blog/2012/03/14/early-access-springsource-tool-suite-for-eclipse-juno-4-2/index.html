How to install



  • Download and install the latest: Eclipse Juno JEE milestone build

  • Download the bookmarks.xml and import that to your list of update sites (Preferences -> Install/Update -> Available Update Sites -> Import).

  • Go to “Install New Software” and select SpringSource Tool Suite for Eclipse 4.2: http://dist.springsource.com/snapshot/TOOLS/nightly/e4.2.

  • Select the components you would like to use and install.

  • Restart and use STS on Eclipse 4.2.

抢先体验：用于Eclipse Juno的SpringSource工具套件（4.2）

工程
马丁·利珀特
2012年3月14日

是否要在最新的Eclipse Juno（4.2）里程碑构建中使用SpringSource Tool Suite？干得好：

如何安装



  • 下载并安装最新版本： Eclipse Juno JEE里程碑构建

  • 下载bookmarks.xml并将其导入您的更新站点列表（首选项->安装/更新->可用更新站点->导入）。

  • 转到“安装新软件”，然后选择适用于Eclipse 4.2的SpringSource Tool Suite： http://dist.springsource.com/snapshot/TOOLS/nightly/e4.2 。

  • 选择您要使用和安装的组件。

  • 在Eclipse 4.2上重新启动并使用STS。


可以从组合更新站点获得STS的必要依赖项： http://dist.springsource.com/snapshot/TOOLS/composite/e4.2 ，如果您想手动添加。

自己管理更新站点


以下是单个更新站点，您可以从中安装STS，AJDT和用于Juno（4.2）的Groovy-Eclipse：

  • http://dist.springsource.com/snapshot/TOOLS/nightly/e4.2

  • http://dist.springsource.org/snapshot/GRECLIPSE/e4.2/

  • http://download.eclipse.org/tools/ajdt/37/dev/update/


已知限制


由于Eclipse Juno发行版基于Eclipse 4.x流，因此在启动最新的Eclipse Juno构建时，您会在UI上注意到很多更改。而且我们尚未适应所有更改，因此STS的Maven支持（例如）尚未起作用。如果您还有其他发现，请告诉我们。

反馈


这是STS的早期访问版本，并由我们的CI服务器不断更新。如果您发现在Eclipse Juno（4.2）M5（或更高版本）安装之上运行STS时遇到问题，请告知我们并将问题报告给： https://issuetracker.springsource.com/browse/STS 。由于我们正计划在Eclipse Juno上进行STS 3.0发行，因此我们很高兴听到您对此的意见并尽早收集反馈。

外表


我们正计划在2012年7月Eclipse Juno发布之后立即发布STS 3.0，以提供基于Eclipse Juno的STS的现成发行版。
