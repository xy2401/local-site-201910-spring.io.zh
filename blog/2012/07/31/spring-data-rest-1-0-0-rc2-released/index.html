Spring Data REST 1.0.0。RC2发布

发布
乔恩·布里斯宾
2012年7月31日

我很高兴地宣布发布Spring Data REST 1.0.0。RC2！除了许多错误修复之外，此版本还增加了对JSONPE（具有错误处理功能的JSONP）的支持，具有使用@RestResource批注关闭CRUD方法的功能，并且现已针对Servlet 3.0 API进行了构建和测试（尽管尚未完成） 3.0专用，因此在Servlet 2.5容器中仍然可以正常工作）。

新功能包括：

  • JSONPE-只需添加URL参数即可将结果包装在对您指定的Javascript函数的调用中。还通过将错误转换为HTTP 200并将原始状态代码作为错误处理程序的第一个参数传递来处理服务器错误。
  • 关闭CRUD方法-导出器现在在CRUD方法上使用@RestResource批注。只需重写方法CrudRepository您想要关闭并用注释@RestResource(exported = false) 。
  • 与现有Spring MVC应用程序的更好集成-简化了内部Spring MVC配置，使其与现有Spring MVC应用程序的集成更加容易。只要将RepositoryRestMvcConfiguration bean包含到您自己的配置中就可以了。

新的或更新的文档包括：

  • JSONP支持
  • 配置REST URL路径
  • 将Spring Data REST添加到现有的Spring MVC应用程序
