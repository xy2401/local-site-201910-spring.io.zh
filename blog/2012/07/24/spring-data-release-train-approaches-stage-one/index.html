Spring数据发布训练接近第一阶段

发布
奥利弗·德罗博姆
2012年7月24日

我想宣布Spring Data Commons（1.3.2）的新里程碑和错误修正版本的可用性。发布1.4.0。M1），JPA（1.1.1。发布1.2.0。M1）和MongoDB（1.0.3。发布1.1.0。M2）。这些发行标志着通用发行流程的第一步，它将在8月中旬发布下一个主要发行版本，其中包括Spring Data Commons，JPA，MongoDB，Neo4J和Gemfire。发布过程是一项工作，旨在同时发布所有支持存储库抽象的商店模块，从而提供一组通用的功能。除此之外，还可以确保已发布的商店彼此无缝地互操作。

该第一个里程碑版本通过引入以下内容，支持基于JavaConfig的存储库配置： @EnableJpaRepositories ， @EnableMongoRepository等等。除此之外，我们当然有大量的错误修复和改进。有关详细信息，请单击下面的链接。

我们将在8月初发布候选版本的主要候选版本，之后再发布GA版本。有关发行火车的更多详细信息，请查看Spring Data Commons中的Wiki页面 。
