Spring 的这个星期-2012年7月3日

工程
乔什·朗（Josh Long）
2012年7月4日

欢迎回到 Spring 本周的另一期。
今天， 我们告别Spring的父亲Rod Johnson 。我们祝罗德接下来的工作顺利。



  1. Spring的贡献者和全方位的构建忍者Chris Beams组成了一个不错的博客，讨论了最近Spring项目中的一些基础架构更改 ，包括它们向GitHub和构建工具Gradle的迁移。
  2. Martin Lippert宣布了SpringSource Tool Suites 3.0.0的可用性。M2，一个用于Spring，另一个用于Groovy和Grails 。

  3. Spring Data Redis 1.0.1已经发布 。
    新版本支持的其他功能
    支持新的（第4个）Redis驱动程序SRP，
    Redis本机执行（ RedisConnection#execute ），并改进了管道执行过程，可在所有驱动程序中始终跟踪潜在的错误和批量结果
  4. Gary Russell has announced the availability of Spring Integration 2.1.3.RELEASE and 2.2.0.M3.

  5. Hubert Klein Ikkink撰写了一篇有趣的文章， 介绍如何在Cloud Foundry上设置环境变量 。

  6. Jan Machacek 在他的博客中宣布了Specs2 Spring 0.6.1的最新版本。请查看更多详细信息。

  7. Matt Vickery的另一部分是关于如何将Spring Integration与C24的iO产品一起使用的 。

  8. 我们的朋友
    威利·惠勒（Willie Wheeler） 在Spring Data JPA上关于可分页自定义查询的精彩文章中。

  9. 我们的朋友Roger Hughes回来了，这次是一篇帖子， 介绍了Spring Social的入门方法 。

  10. Nicolas Frankel的博客不错， 介绍了Spring Data JPA 。

  11. Java TV教程站点和聚合器都具有Spring Integration的贡献者Oleg Zhurakousky关于Spring Integration Tips'n Tricks的网络研讨会。聚合器收集了不错的内容非常好，但是读者也应该确保查看您所有SpringSource视频和教程的源代码，即SpringSource Dev YouTube频道 。

  12. A blog, seemingly on PHP, has a post on the correct way to utilize Spring's <p> and  <c> namespaces.

  13. doanduyhai博客上有一篇关于Spring 3.1缓存抽象的有趣文章。

  14. 通过代码学习博客中有一篇很棒的文章， 介绍了Spring对消息加载的支持

  15. JavaCode Geeks博客上有一篇有趣的帖子， 介绍了Spring的JSF支持中的异常处理

  16. 仅有一点与Spring相关的问题：Activiti BPMN2工作流引擎的撰稿人Joram Barrez，Spring Batch的负责人David Syer博士和我对Spring的支持对此做出了贡献，他在一篇不错的文章中比较了工作流引擎的速度和一些测试使用Spring支持。
