2012年2月28日，春季，本周

工程
乔什·朗（Josh Long）
2012年2月29日

欢迎来到春季本周的另一期。和往常一样，我们有很多好东西要看。

  Since you're here, though, let's talk about the Cloud Foundry Open Tour, which is an event bringing the industry's best talent and speakers on Spring, Cloud Foundry, and much more  to a town near you in America, Asia and Europe.  The full itinerary's provided on the linked page, but if you're in (or near) Shanhai, Beijing, Tokyo, London, Moscow, Kiev, San Francisco, Portland, Austin, and Washington D.C., then you should not  miss this event - the tour will be in your backyard! 
 I hope to see you there, as always. 



  1.  Spring Social  lead Craig Walls' video from SpringOne2GX 2011, Spring Social for the New Web of APIs,  is up on InfoQ. The video introduces the challenges of secure RESTful web services, OAuth, and the myriad of implementations across the various, popular social APIs, and then he introduces the Spring Social project, which provides the   solution to these problems.  
    2.  


  2.  My SpringOne2GX 2011 talk,  on Tailoring Spring for Custom Usage, is up on InfoQ, along with a lot of other great SpringOne 2GX content. The idea is that  while most of us are familiar with the surface level component model, Spring's real power lies just underneath that component model, in the numerous SPIs and extension hooks. The other Spring projects, including Spring Integration, Spring Batch and Spring MVC, all build on top of these slightly lower level framework hooks, and  you can too. The deck is available for download from my presentations page, as well.  
    3.  



  3. In the recent Messaging for Modern Applications webinar,  Tom McCuch explained how the landscape of enterprise messaging has changed and discusses what new styles of solutions are emerging. It's a great video by an industry legend, and definitely worth a watch.   
    4.  


  4.  Over on the Xebia blog,  Shekhar Gulati talks about how to use Spring 3.1's profile features to build PaaS-portable Spring applications. In the example, he uses Spring Roo and MongoDB.   
    5.

  5. Tomcat 7.0.26已发布 。新版本包括新功能和错误修复。
  6. Mark Baars put together a more comprehensive outline of some great tutorials that are available on YouTube.com introducing Hyperic, tcServer, and GemFire, all part of the vFabric stack.
    7.
  7. Mark Baars汇总了YouTube.com上一些出色的教程的更全面的概述， 介绍了HyperFab，tcServer和GemFire，它们都是vFabric堆栈的一部分 。

  8. Java Code Geeks的这个博客介绍了如何使用Spring管理线程池。
    线程池对于执行同步和异步过程非常重要，并且Spring很好地支持线程池（由TaskExecutor SPI建模和公开）
    但是，该博客似乎不必要冗长；结论是，借助Spring的TaskExecutor SPI，您可以选择适合自己需求的实现，并在许多不同的地方使用它，包括Java SE，Java EE和云中 。

  9. 威利·惠勒（Willie Wheeler）回来了，这次是关于Spring MVC（3.1）的另一篇精彩文章。在这篇文章中，Willie介绍了Spring的@ResponseBodyannotation，并描述了如何在Spring MVC中获取基于Jackson和JAXBannotation的封送处理，而仅需在实体上进行一次映射即可。


  10. Tobias Trelle又来了，这一次介绍了使用Spring Data Neo4j。 这篇文章很好地介绍了Neo4J的使用方法 ，并提供了逐步指南，以获取库，配置它们以及使用Spring Data Neo4J的映射功能构建API。

  11. 清莱研究博客有一个有趣的帖子建设有Spring MVC和jQuery的AJAX形式 。这是很标准的东西，但是很高兴能有参考。如果今天要这样做，请考虑使用Spring 3.1。

  12. 在ZK框架的网站上，Hawk Chen拥有一个有趣的博客， 介绍如何使用ZK6中的Spring。
