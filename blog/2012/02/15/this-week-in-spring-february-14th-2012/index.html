春季本周：2012年2月14日

工程
亚当·菲茨杰拉德
2012年2月15日

欢迎来到春季本周的另一期。这周有很多值得探讨的内容，所以让我们开始吧。

另外：如果您在瑞典的斯德哥尔摩参加JFokus Java会议，请确保在VMWare摊位前打个招呼或回答有关Spring的问题，
Cloud Foundry和vFabric 。

  1.  Alan Stewart has announced the release of Spring Roo 1.2.1, which includes fixes for multi-module Maven projects,
    huge performance enhancements, and updated support for  PrimeFaces 3.1 and Google App Engine 1.6.2.


    
    2. 



  2.  Oliver Gierke has announced that JPA 1.1.0 RC1 and 1.0.3 GA have been released. Among the many new features, the release includes support for locking, 
     @IdClass in entities,
    Support for LessThanEqual and  GreaterThanEquals, True/False keywords in query methods,
    added CDI integration for Spring Data repositories,

    3. 并改进了对用于空值的派生查询的参数绑定。

    4. 在SpringOne 2GX 2011的这段视频中，
    乔希·朗 （ Josh Long）和
    罗伊·克拉克森（Roy Clarkson）介绍
    本机Android开发实践
    使用Spring Android项目。

  4.        In another video from SpringOne2GX, 
       Dr. David Syer discusses identity management with Spring Security. He covers Single Sign On, security standards (including SAML, OpenID, OAuth, SCIM, JWT), how Spring Security can fit in, as well as demoing IdM as a service.
  5. Last week, Jason Weathersby and Josh Long put together an introduction to using BIRT and Spring together.

    BIRT是一种用于执行商务智能和报告的出色技术，Spring已了解您的数据，因此将其连接自然是下一步。请享用！

  6.  Tobias Flohre is back with not one, but two great posts on Spring! The first, a follow up to his series on business component architectures introduces  properties and their use in Spring 3.1. 

       The second post, on geospatial queries in Spring Data MongoDB, is a fantastic look at  
       both geolocation, and the specific application  from Spring Data MongoDB.
