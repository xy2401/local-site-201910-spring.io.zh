      1.  First, the big news: Spring Framework 3.1.1's Been Released!  

       The first maintenance release in the Spring 3.1.x line is now available via Maven Central, the SpringSource repository, or for direct download from our community download page.  This release includes many important bugfixes and minor improvements and is a recommended upgrade. 
       
        2. 
    
      2.  Dr. David Syer's SpringOne2GX talk, Identity Management With Spring Security,

      涵盖了单一登录，安全标准（包括SAML，OpenID，OAuth，SCIM，JWT），Spring Security的适用范围以及将身份管理作为服务进行演示。


    3. Roy Clarkson和我在SpringOne 2GX 2011上发表了演讲
      介绍本机Android开发实践 。
      该会话涵盖了Spring MVC对设备检测的支持，REST对处理设备请求的实现以及Spring Android对组织从移动客户端到服务器的调用的支持。

    4. 春季社交主管Craig Walls宣布了春季社交LinkedIn 1.0.0 RC1 。由于社区的贡献，此版本中已完成对LinkedIn API的支持。


    5. 新的Micro Cloud Foundry版本1.2已发布，并具有许多新功能，包括更轻松的离线体验以及SpringSource Tool Suite中的调试支持。

    6. InfoQ已发布Spring Found2GX 2011上 Cloud Foundry架构师Derek Collison的演讲“机器内部的Cloud Foundry” ，其中详细介绍了Cloud Foundry的体系结构。这种观点可以为我们作为Cloud Foundry的开发人员提供信息。

    7.   In the SpringOne 2GX video, Tuning Java for Virtual Environments with EM4J, Ben Corrie provides a

      8. 对Java内存管理的全面概述，在虚拟环境中事物如何工作以及如何避免出现问题的方法。

      9. 如果您错过了有关Spring开发的最近Cloud Foundry网络研讨会 ，则可以在Cloud Foundry YouTube频道上观看视频重播 。在此视频中， Ramnivas Laddad向您展示了如何使用STS和vmc命令行获取真正的Spring应用程序并将其定位到Cloud Foundry。

    9. Spring Data Document Lead Oliver Gierke announced that Spring Data Mongodb 1.0.1 GA has been released.
    10. Javier Manzano's posted a blog about efficient RESTful services with Spring Android. Interestingly, I also found this blog, in Spanish, which seems to be the same thing, just translated.
    11. The Java CodeGeeks has an article up, on setting up a Spring 3 Development environment. The article introduces the basics, but you can shortcut most of those steps by setting up a JDK and STS, as commenters note at the bottom of the page. 

    12.  In this blog, on Sending Email Attachments with Spring Integration, blogger Glenn introduces Spring Integration, the flow of a use case that requires email attachments, and  example code, to boot!

    13. Artur Mkrtchyan在DZone的这篇文章中介绍了Spring Data和MongoDB 。他首先在代码中建立一个示例，然后介绍了更简洁的基于Spring Data Document的代码。

    14.  Also on DZone, blogger Roger Hughes has written up a great article on the handy class, the SimpleMappingExceptionHandler, which maps exceptions to error pages in Spring MVC.
