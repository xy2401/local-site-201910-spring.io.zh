发布了Spring Data JPA 1.1.0 RC1和1.0.3 GA

发布
奥利弗·德罗博姆
2012年2月9日

亲爱的春季社区，

我很高兴地宣布发布Spring Data JPA 1.1.0 RC1和1.0.3 GA。候选版本包括大量新功能和各种错误修复。以下是最重要的清单：

  • 支持锁定
  • 支持实体中的@IdClass
  • 在查询方法中支持LessThanEqual和GreaterThanEquals，True / False关键字
  • 为存储库添加了CDI集成
  • 改进的参数绑定，用于派生查询空值

CDI集成在很大程度上由Buschmais的Dirk Mahler贡献。有关更改的完整列表，请参见changelog 。修复了许多错误，因此我们绝对建议升级到1.0.3并与候选版本一起使用，以帮助改进最终版本。可以从Maven Central获得1.0.3 GA版本，而候选版本可以通过我们的里程碑存储库获得： http://repo.springsource.org/libs-milestone 。
