Spring 的这个星期-2012年10月9日

工程
乔什·朗（Josh Long）
2012年10月10日

欢迎回到 Spring 本周的另一个安装！ 。
当我们开始向SpringOne进军的最后一站时，本周在SpringSource上有很多活动！
我们只有一个星期的路程，而且该节目每天都在编排，成为有史以来最好的节目！我们希望看到你在那里！不要错过Adrian Colyer，Jurgen Hoeller，Mark Pollack，Graeme Rocher的第1天和第2天主题演讲，以及过去4周我们在SpringSource.org上重点介绍的激动人心的会议： 异步-推送通知 ， 客户端-侧面UI Smackdown ， 分解应用程序的可部署性和可伸缩性 ， 如何使用OSS为Hadoop构建大数据管道 。



  1. Alvin J Rayes撰写了一篇关于将Spring MVC 3与模板引擎Apache Tiles一起使用的不错的文章。

  2. 工具套件负责人Martin Lippert宣布， Spring工具套件和Groovy / Grails工具套件3.1.0已发布 ！干得好，马丁！


  3. Spring Security负责人Rob Winch宣布已发布Spring Security 2.0.8、3.0.8和3.1.3 ！这使未解决的错误总数降至0。做得好，罗布！

  4. 您是否一直在关注Spring Integration团队令人兴奋的新博客，以了解即将推出的Spring Integration 2.2版本的新功能？

         <A href = "http://blog.springsource.org/2012/10/09/spring-integration-2-2-retry-and-more/">Gary Russell has put together a nice blog on the new support for retry in Spring Integration</A>.  The support for retry capabilities originally comes from Spring Batch's support for retrying  
     operations, and was then factored out to the <a href="http://www.github.com/springsource/spring-retry">Spring Retry</A> project. Now, you can take advantage of it in Spring Integration flows!


  5. Gunnar Hillert在如何使用Spring Integration 2.2中的新适配器与JPA一起使用方面发表了一篇不错的文章。

  6. 学习OAuth？是否想了解OAuth可以帮助更好地保护RESTful API的场景？加入Spring忍者David Syer博士的文章介绍OAuth，内容涉及Cloud Foundry如何将其用于UAA服务 。

    7.     <LI> Krishna Prasad has put together some very cool posts recently.  I liked his post on connecting systems using <a href="http://krishnasblog.com/2012/10/03/publisher-subscriber-using-vfabric-spring-integration-gemfire/">publish-subscribe through Spring Integration and vFabric GemFire</a>. His next post, on  
        <A href = "http://krishnasblog.com/2012/10/08/responsive-web-design-using-twitter-bootstrap-spring-mvc/">responsive web design using Twitter and Spring MVC </a>, is brilliant. Really well done, Krishna!


  7. vFabric ninja Al Sargent 迅速获得了JavaOne的现场报道 ，并向着简化迈进了坚实的一步。

  8. Broadleaf Commerce的人们写了一篇很好的文章， 介绍了将Spring Social与Broadleaf Commerce集成在一起


  9. Spring Social社区继续为Spring Social的扩展做出贡献。最近， 杰弗里·威廉姆斯 （ Jeffrey Williams）开始扩展与Intuit的Quickbooks Online集成的扩展 ！
    Spring Social将您连接到您的社交提供程序以及所有其他OAuth保护的服务。

  10. 你们在使用Twitter吗？一定要查看一下Twitter上代表的SpringSource人员列表 。
