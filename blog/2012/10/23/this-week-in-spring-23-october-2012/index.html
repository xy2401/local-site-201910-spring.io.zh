春天的这个星期-2012年10月23日

工程
乔什·朗（Josh Long）
2012年10月23日

欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！许多消息来自SpringSource本身，来自上周计划与SpringOne2GX同步的各种项目。这就像收到生日礼物！说到礼物，如果您错过了SpringOne2GX，请不要担心！我们记录了SpringOne会议，他们将很快发布。另外，您可以在SpringSource.org上查看第1天 ， 第2 天和第3 天的演出总结，以了解会议的情况。


  1. Costin Leau宣布为Apache Hadoop 1.0发布Spring。RC1 （春天的第一代曾经 RC为Apache Hadoop的！）包含一个模板，用于Pig和Hive脚本的异常转换和声明性运行器，增强的map / reduce .jar和工具执行以及完善的JVM脚本，M / R和级联运行器。
    不要错过即将在11月8日举行的网络研讨会 ！

  2.   Peter Bell did a    great presentation on Spring Data and the NoSQL   space for QCon. Be sure to check it out!  
    3.

  3. Yusuf 在开源Bitronix独立JTA-XA事务管理器方面，很好地介绍了Spring对XA的支持 。作为替代指南，我可以谦虚地建议这篇博客文章吗？

  4. Oleg Zhurakousky宣布发布Spring Integration 2.1.4和2.2.0。RC2 。新版本具有伪事务，JPA支持，内置重试支持以及更多功能，因此请务必检查一下！

  5. Costin Leau宣布发布Spring Data Redis 1.0.2 ，该版本具有缓存过期和ZSet方法等功能。

  6. Thomas Risberg宣布了具有Oracle数据库支持1.0.0版的Spring Data JDBC扩展的可用性。RC4 。

  7. Satish Ab在Spring MVC的示例发票应用程序上构建Web层有很好的文章。

  8. 该Mkyong博客提供了有关如何自动装配dependiecnes好贴到定制JSF验证 。很好的帖子！

  9. 我们的朋友Krishna Prasad在hwo上有一篇不错的文章，介绍如何通过Spring Integration和JMS实现控制总线模式
