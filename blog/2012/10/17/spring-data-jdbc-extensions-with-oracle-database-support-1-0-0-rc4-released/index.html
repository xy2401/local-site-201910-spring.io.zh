具有Oracle数据库支持1.0.0的Spring Data JDBC扩展。RC4发布

发布
托马斯·里斯伯格
2012年10月17日

亲爱的春季社区，

我们很高兴地宣布，第四个也是最后一个候选版本（1.0.0。带有Oracle数据库支持的Spring Data JDBC Extensions 1.0项目的RC4）现已发布！

创建新的Spring Data JDBC Extensions项目是为了提供对特定于供应商的JDBC扩展的额外支持，以及使用QueryDSL之类的JDBC的新方法。大部分支持由从SpringSource项目“ Oracle数据库高级包”移植的代码组成，该代码可用于支持订阅客户。现在，我们将该代码提供给所有Spring用户，并且任何新开发都将在Spring Data JDBC Extensions项目中进行。

要了解有关该项目的更多信息，请访问Spring Data JDBC Extensions 主页 。

这四个候选版本的最重要更改是：

一般
*更改了版本以使用Gradle
*添加了QueryDSL SQL模块的文档章节

核心
*添加一对多ResultSetExtractor实现
*将异常转换添加到QueryDslJdbcTemplate

甲骨文
*用于高级数据类型映射的参数化类
*除了标量值外，还支持映射STRUCT的ARRAY
