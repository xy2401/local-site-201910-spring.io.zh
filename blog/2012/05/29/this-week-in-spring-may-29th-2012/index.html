春天的一周：2012年5月29日

工程
亚当·菲茨杰拉德
2012年5月29日

欢迎回到春季本周的另一期！像往常一样，我们有很多内容要讲，所以让我们直接讲吧！



  1. 克里斯·比姆斯（Chris Beams）宣布了Spring 3.2的第一个里程碑版本
    现在可用     ！这个版本很棒！

        It includes 

    initial support for asynchronous  @Controller methods,
    early support for JCache-based cache providers,
    significant performance improvements in autowiring of non-singleton beans,
    initial delay support for  @Scheduled and  <task:scheduled>,
    ability to choose between multiple executuors with @Async,
    enhanced bean profile selection using the not (!) operator,  
    48 bugs fixed, 8 new features and 36 improvements implemented.

    立即查看最新和最重要的内容，并随时提供反馈！

    当我在Twitter帐户上要求任何要纳入本综述的项目时，一个用户立即回击：“ 在Spring 3.2新闻中，关于社区是否希望RC1跟随3.2 M1的民意调查怎么样？等待太好了 。”我同意，这个版本是非常史诗！

  2. Michael Isvy 在Spring的大部分工作中对低级代理机制进行了精彩的介绍 ，包括事务管理和缓存。一旦了解了它的工作原理，就很容易通过外科手术来使用它。

  3. 马丁·利珀特（Martin Lippert）宣布
    SpringSource Tool Suite 2.9.2已经发布 。
    该版本包括与tcServer 2.7的更新兼容性以及一些错误修复。

  4. Alan Stewart宣布了Spring Roo的1.2.2版本 。

     This is the second maintenance release for 1.2 and includes fixes for a number of issues and includes support for Spring Framework 3.1.1 and JDK 7.  Roo 1.2.2 also includes the excellent new "tailor" feature provided by our partner, Accenture. The H-Online website had coverage of the release.
  5. The Cake Solutions blog has put together (another!) great blog introducing how to run Neo4j embedded with the web administration tool, and Spring Data. /

  6. Zenika博客又来了一次，这次不是一个，而是本周综述的两个重要条目。第一个条目概述了Spring Data MongoDB ，第二个条目概述了如何一起使用Spring Batch和MongoDB 。很棒的东西，我一定会回来检查新内容的！

  7. JavaRevisted博客上有一篇不错的文章，介绍了如何在Spring中指定bean的作用域 。需要澄清的几点：Spring 3.0 确实提供了新的作用域（包括我最喜欢的功能之一，即SimpleThreadScope ） ，并且-尽管此博客介绍了可用的默认范围-您可以使用轻松注册新的范围CustomScopeConfigurer ，并填写自定义范围实例的地图，如下所示：

    @Bean 
	public static CustomScopeConfigurer csc(){ 
	  
	  Map scopes = new HashMap();
	  scopes.put("thread", new SimpleThreadScope());
	  	
	  CustomScopeConfigurer csc = new CustomScopeConfigurer();
	  csc.setScopes(scopes);
	  return csc;	
	}


  8. Blogger Angelo Zerr汇总了如何使用Spring Virgo（以前称为Spring DM），Spring Data JPA和Spring Remoting来构建Eclipse RCP应用程序的介绍 。有很多博客-而且它们似乎在顶部附近有指向上一个链接的链接，因此只需向后工作即可获得所有链接。

    文章有法文和英文两种版本。
