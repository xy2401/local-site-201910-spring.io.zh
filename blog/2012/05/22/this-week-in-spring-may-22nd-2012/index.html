Spring 的这个星期-2012年5月22日

工程
乔什·朗（Josh Long）
2012年5月22日

欢迎回到 Spring 本周的另一期。和往常一样，本周我们有很多精彩的内容！

  1. Rossen Stoyanchev在其有关Spring MVC 3.2 Preview的系列文章中建立了另一个博客。最新一期介绍Spring MVC聊天示例 。
  2. Oliver Gierke宣布了Spring Data JPA的1.1.0 GA版本 。 Spring Data JPA使构建基于JPA的存储库变得非常简单，从而避免了繁琐的样板代码。此新版本包括用于查询生成的新关键字（ LessThanEqual ， GreaterThanEqual ， Before ， After ， StartsWith ， EndsWith和Contains ），方便PersistenceUnitPostProcessor扫描JPA实体（将在3.1之前的Spring版本中使用），支持@Query ，并支持声明式锁定。
  3. Jonathan Brisbin宣布了1.0.0。M2版本的Spring Data REST 。 Spring Data REST让您轻松地将Spring存储库对象导出为RESTful端点。新版本包括对以下查询方法的调用的支持： Repository接口，对JSR 303和Spring Validator验证的支持以及对Spring的改进支持ApplicationEvent在每次保存或删除前后以及基于注解的配置之前发出的。

  4. Oleg Zhurakousky宣布了Spring Integration 2.2的第一个里程碑版本 。该发行版包括依赖项升级，JPA支持以及对“发布者确认和返回”的支持，Spring AMQP新增了这些支持。
  5. 加里·罗素（Gary Russell ）宣布了Spring AMQP的1.1.0版本，其中包括对RabbitMQ 2.8.x客户端的支持，而该客户端又支持镜像队列，代理故障转移，发布者确认，退货，联合交易所等。
  6. Matt Vickery在一篇很棒的文章中介绍了Spring Integration splitter-aggregator模式 。
  7. 威利·惠勒（Willie Wheeler）在他的自定义配置管理数据库（CMDB）上发表了精彩的文章。帖子详细介绍了该项目，然后讨论了他在重建CMDB后端时使用Spring Data的存储库。好的帖子，威利！
  8. 道格·哈伯组建了一个美妙的岗位上
    使用Spring Data和Spring 3.1中的REST支持处理分页 。
  9. Blogger panbhatt详细介绍了如何使用Spring MVC的REST支持来解决他遇到的特定问题。
  10. Blogger OBSERWATORZY 在尝试使用RESTful服务时描述了他的特殊思考过程，并想知道Spring是否提供了答案（当然可以了！）。请继续阅读以获取他的解决方案。
  11. Vishal Biyani很好地介绍了Spring Roo和Cloud Foundry入门 。

  12. RabbitMQ博客上有一篇很棒的文章， 介绍了排队理论（并介绍了吞吐量，延迟和带宽。
  13. JAXEnter 很好地概述了本文中描述的一些新闻发布，包括Spring AMQP和Spring Data JPA的GA。
