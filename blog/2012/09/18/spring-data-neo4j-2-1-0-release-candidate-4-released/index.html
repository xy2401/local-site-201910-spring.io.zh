Spring Data Neo4j 2.1.0 Release Candidate 4发布

发布
迈克尔·汉格
2012年9月18日

在Spring Data GA发布培训即将到来之前，我们希望收集有关Spring Data - Neo4j中的修复和更新的一些反馈。因此，我们发布了Release Candidate 4。

这里是对该候选版本所做的更改的快速概述，虽然比我们预期的要长得多，但仍然非常有用。

2.1.0版中的更改。RC4（2012-08-19）

  • DATAGRAPH-296更新为Neo4j 1.8的稳定版本。更改API时采用RC1，空间，cypher-dsl和java-rest-binding
  • DATAGRAPH-285添加用于保存和删除的应用程序事件
  • DATAGRAPH-263，DATAGRAPH-212，DATAGRAPH-272，DATAGRAPH-147重构了派生查询的创建，增加了对多个索引字段和所有查询关键字的支持
  • DATAGRAPH-294导出的用于数字索引值的查找器方法
  • DATAGRAPH-293通过graph-id查找对象
  • DATAGRAPH-275 TypeRepresentationStrategyFactory在空图上的非正常回退
  • DATAGRAPH-246允许实体作为派生查找器的参数
  • DATAGRAPH-281添加了对枚举和日期的支持，作为Cypher的参数

只需将您的依赖版本指向2.1.0.RC4 。目前，由于人工和时间限制，在线资源尚未更新。这些将在2.1.0中可用。发布。
