    春天的这个星期-2012年11月6日

    工程
    乔什·朗（Josh Long）
    2012年11月6日

    欢迎回到春季本周的另一期！我们有很多内容要讲，所以让我们开始吧！


    1. 你们错过了SpringOne2GX 2012吗？
      我不会骗你-你真的错过了！但是，请不要绝望！我绝对喜欢的两个
      展览的演讲现在可以在网上进行-主题演讲 ！两者都很棒，但是如果您正在寻找有关下一代Web应用程序潜在趋势的精彩介绍，以及很棒的（ 很棒！ ）浏览RabbitMQ，查看第2天的主题演讲！

          More sessions will be posted on that page, so check back often.
    2. Spring 3.2 is drawing ever nearer! The H has some nice coverage of the new features in the framework.
    3. Chris Beams has announced the latest maintenance release, Spring 3.1.3 - the third in the 3.1 line, has been released!
    4. Spring 3.2越来越近了！H 很好地涵盖了框架中的新功能 。
    5. 克里斯·比姆斯（Chris Beams）宣布了最新的维护版本Spring 3.1.3-3.1系列中的第三个版本已发布 ！

    6. 伙计们，我很高兴告诉您SpringOne即将来到印度 ！现在注册！


    7. 您的PaaS是否支持您的工作负载？如果您使用的是Cloud Foundry，那么很有可能会这么做！


    8. Michael Isvy 在Spring MVC和View层上发表了一篇不错的文章

    9. Spring框架的TestContext框架现在支持用于集成测试的会话/请求范围的Bean

    10. Spring Data SOLR的第一个里程碑已经发布！

    11. Jonathan Brisbin宣布了Spring REST Shell的1.2版本 。
      新版本具有SSL，基本身份验证，dotrc支持，更新的Homebrew安装公式，HATEOAS和超媒体。

    12. 本周四，请查看Costin Leau的网络研讨会，介绍Spring Data Hadoop ！

    13. RabbitMQ开发人员的拥护者Alvaro Videla链接到一篇有关可视化RabbitMQ拓扑的出色文章 。

    14. Spring Data Batch和Spring Security OAuth忍者以及Cloud Foundry UAA提交者David Syer博士撰写了一篇关于使用Cloud Foundry的UAA代理OAuth2终结点的精彩文章 。这与Spring几乎没有什么关系，只不过Syer博士的帖子具有很好的教育意义，并极大地帮助了我了解Spring Security OAuth解决的问题。
