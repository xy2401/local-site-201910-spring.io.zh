春天的这个星期-2012年11月27日

工程
乔什·朗（Josh Long）
2012年11月28日

欢迎回到春季本周的另一期！如果您在美国，那么希望您在一个希望非常愉快的假期周末休息一下，因为本周我们有很多工作要做！希望您对此综述有用。如果您想仔细阅读前几周的综述，我们最近为博客档案创建了“ This Week in Spring”哈希标签。无论如何，本周我们有很多工作要做，让我们继续吧！

  1. 克里斯·比姆斯（Chris Beams）宣布Spring Framework 3.2 RC2
    已发布，主要由错误修正和重构组成，随着项目临近GA。

  2. 安迪·威尔金森（Andy Wilkinson）宣布了Spring Migration Analyzer的发布和可用性，
    一个命令行实用程序，用于分析企业Java应用程序并生成描述该应用程序以及如何将其迁移到Spring的报告。
    输入EAR文件，获取迁移工作报告。它在具有里程碑意义的版本中支持WebLogic，WebSphere，Java EE和JBoss，但我们正在寻求社区支持以使其更好地适用于GA！

  3. 赶紧参加12月13日举行的Spring Data网络研讨会系列的最新活动– 使用Spring Data，Hadoop，批处理，集成进行数据访问和处理 。我们还将在12月6日举办有关Spring Data Gemfire的网络研讨会： 数据复兴：使用VMWare vFabric GemFire 7.0和Spring进入内存 。

  4. 如果您错过了Spring Data网络研讨会系列中的其他网络研讨会，请查看SpringSource Dev Youtube频道，获取有关以下内容的最近发布的重放： Spring的数据访问–充分利用JPA，JDBC和REST，以及Spring for Apache Hadoop简介 。

  5. Gary Russell宣布了Spring Integration 2.2.0。RC3已发布 。

  6. Spring Social 1.1.0已发布 。新版本包括更轻松的XML和Java配置，更严格地遵守最新的OAuth 2规范草案，包括HTTP Basic客户端身份验证以及对资源所有者凭证授予和客户端凭证授予的支持，以及对Facebook和Twitter API绑定的更新。

  7. Jens Schauder撰写了一篇不错的文章，内容涉及使用singleton范围 。

  8. Java How to Program博客对以下内容进行了总结

      How to use @Component 
  annotation to automatically configure Spring beans.


  9. 这是一篇关于如何使用Spring和Java实现责任链模式的好文章。 @Autowired （或者@Inject ）。

  10. Viral Patel再次支持，这次发表了有关Spring 3 MVC拦截器的文章 。

  11. Mkyong博客还有一篇关于Spring和使用Spring的Java线程示例的精彩文章。 thread范围 。

  12. DZone在Spring和JUnit 4支持下的轻松集成测试上有一篇不错的文章。

  13. 我最近为初学者写了一个有关SpringSource Tool Suite和Spring入门的教程的版本。
