春天的这个星期-2012年11月20日

工程
乔什·朗（Josh Long）
2012年11月20日

哇！伙计们，您能相信我们又来了吗
盯着年底？时间过得真快！
在美国，这是感恩节的一周，这是一个假期，我们应该花一点时间反思一下我们要感恩的事情。我（我肯定会代表团队说我们是..）永远感谢你们，围绕Spring，RabbitMQ和Cloud Foundry的奇妙，生机勃勃且富有吸引力的社区，使这次综述变得如此愉快每周。


这样，让我们继续了解新闻：


  1. Thomas Risberg宣布了Cloud Foundry Maven插件1.0.0。M4现在可用。

  2. Did you miss out at SpringOneOne2GX 2012 in Washington D.C.? Don't fret. We'll release 2 sessions every week on springsource.org. We've already released Day 1 and 2 Keynotes. Available now: a talk from Rossen Stoyanchev on "What's New in Spring Mvc 3.2" and a talk from Roy Clarkson and Craig Walls on "Extending Spring Mvc With Spring Mobile and Javascript." Awesome.
  3. Our friend Tobias Trelle is at it again! This week, he's put together an interview with Oliver Gierke, the Spring Data ninja
  4. Teammate, friend, mentor and Grails Developer Advocate Peter Ledbrook had a great answer when I asked him if there was a Spring Integration / Grails solution. He linked me to this code by Stephane Maldini which lets you tie the Spring Integration project's wealth of adapters supporting system-to-system communication to the eventing system in Grails! That's really cool! "The nice thing," Peter adds, "is it can be used with RabbitMQ to extend the event bus across multiple application instances on Cloud Foundry." Brilliant. Thanks, Peter!
  5. 您是否错过了华盛顿特区的SpringOneOne2GX 2012 ？别担心我们每周都会在springsource.org上发布2节。我们已经发布了第一天和第二天的主题演讲。现在可用：Rossen Stoyanchev关于“ Spring Mvc 3.2的新功能”的演讲，以及Roy Clarkson和Craig Walls关于“使用Spring Mobile和Javascript扩展Spring Mvc”的演讲。太棒了
  6. 我们的朋友托比亚斯·特雷勒又来了！本周，他整理了Spring Data忍者Oliver Gierke的专访
  7. 当我问他是否有Spring Integration / Grails解决方案时，队友，朋友，导师和Grails开发倡导者Peter Ledbrook给出了很好的答案。他通过Stephane Maldini将我链接到该代码，使您可以将Spring Integration项目的大量适配器（支持系统对系统通信） 与Grails中的事件系统联系起来 ！这太酷了！彼得补充说：“很高兴，它可以与RabbitMQ一起使用，以将事件总线扩展到Cloud Foundry上的多个应用程序实例。”辉煌。谢谢，彼得！


  8. Mark Fisher，Jonas合伙人，Marius Bogoevici和Iwein Fuld就InfoQ刚发行的《 Spring Integration in Action》 接受了采访 。
  9. The all and sundry blog has a nice post on how to build standalone (not run in an application server) Spring Integration applications.
  10. all and杂物博客关于如何构建独立的（不在应用程序服务器中运行）Spring Integration应用程序一文很好。

  11. Jakub Nabrdalik的帖子很好地介绍了按合同设计样式的前提条件，后置条件和状态到Spring代码中，以使用Spring方面摆脱null参数 。


  12. Java Revisited博客上有一篇很好的文章，介绍了Spring中的setter-injection和构造函数-injection之间的区别

  13. “ 通过示例学习Java”博客中有一篇很好的介绍性文章，介绍如何使用@javax.inject.Inject您的Spring应用程序。

  14. Kevin Hookee有一个不错的博客，介绍了Spring框架中的一些基本概念 。

  15.     Yohan Liyanage has put together a nice post that 
    can be used to integration test Spring Data MongoDB solutions  by embedding MongoDB.   Read on for more!  
    16.  
    
  16.   The Technical Notes   blog has put together a nice post          
    introducing  the Spring internationalization support.  

        
    17.

火鸡节快乐！
