SpringOne 2GX 2012 Web Track：Spring MVC 3.2的新增功能+通过Spring Mobile和JavaScript扩展Spring MVC

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2012年11月16日

 

Spring MVC 3.2的新增功能



在Juergen关于即将发布的Spring 3.2版本的演讲之后，本演讲将重点关注Spring MVC领域中的新功能。该演示文稿将解释所有值得注意的功能，并且像每个新版本一样，将进行很多讨论，包括基于Servlet的异步请求支持，内容协商增强，REST错误处理，@ MVC测试支持等等。该演讲没有提供Spring MVC的概述，而是假设了一定的经验，并着重介绍了新功能。







关于罗森·斯托扬切夫



Rossen是一名Spring Framework开发人员，专注于Spring MVC和Spring Web Flow。他有17年以上的工作背景，包括交易和风险管理软件，投资会计，电子商务Web应用程序，目录服务等方面的工作。在成为全职Spring Framework开发人员之前，Rossen花费了数年的时间，就广泛的主题，为客户使用Spring构建企业Java应用程序提供咨询和咨询服务。

使用Spring Mobile和JavaScript扩展Spring MVC



现代网络不再局限于台式机浏览器。智能手机和平板电脑已成为我们日常生活不可或缺的一部分。在22英寸显示器上看起来不错的网站通常无法在较小的屏幕上格式化和显示。此外，网络速度可能会限制移动设备上网站的性能。由于这些原因，许多开发人员和组织正在考虑如何使人们使用的所有各种设备和屏幕尺寸都可以访问其网站。在本节中，我们将探讨Spring Mobile项目中提供的功能，以及如何使用它将Spring MVC应用程序扩展到移动设备和平板电脑设备上。然后，我们将通过演示如何与Spring Mobile结合使用一些流行的移动JavaScript框架来为移动设备上的用户提供一流的体验，从而继续进行讨论。
