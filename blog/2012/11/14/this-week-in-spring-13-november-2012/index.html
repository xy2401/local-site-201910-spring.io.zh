春天的这个星期-2012年11月13日

工程
乔什·朗（Josh Long）
2012年11月14日

欢迎回到春季本周的另一期！我本周开始在可爱的保加利亚索非亚参加Cloud Foundry Open Tour活动，与一群听众谈论在Cloud Foundry上构建Spring应用程序。现在，我在比利时的安特卫普举行的Devoxx会议上，我将介绍Spring 3.2，Cloud Foundry等方面的新增功能。如果您在这里，我邀请您参观SpringSource的展位，看看我和其他人将在Spring上进行的一些演讲。

  1. 罗伊·克拉克森（Roy Clarkson）本周宣布的不是Spring Mobile的一个，而是两个！春天的Android 1.0.1已经发布 。此版本包括对支持BlackBerry 10移动设备的更改。使用时黑莓10台的移动设备现在已经得到解决作为移动设备LiteDeviceResolver 。随后，他又发布了1.1.0。M1版本包括1.0.1支持以及站点首选项处理和站点切换中的平板电脑支持，对基于Java的容器配置的支持以及对基于servlet的配置的支持。
  2. 托马斯·里斯伯格（Thomas Risberg）宣布了具有QueryDSL和Advanced Oracle支持的Spring Data JDBC扩展的GA版本。
  3. 本周测试领域有很多有趣的消息！首先，Rossen Stoyanchev在3.2 RC1的Spring MVC测试框架上发布了一个不错的博客。然后，山姆Brannen张贴在一个伟大的博客在即将到来的春季3.2核心的新的测试功能（现在可作为3.2 RC1的一部分）。
  4. 这并不是特别新颖 ，但值得重申： Spring Framework 3.2 RC1已发布 ！借此机会对照您的应用程序进行检查，并开始学习一些新功能。
  5. David Turanski撰写了一篇有关使用新的Spring Integration Groovy DSL的不错的文章。谈论一二拳组合！
  6. 如今， Cloud Foundry上公布的Cloud Foundry核心的定义，并推出core.cloudfoundry.org ，它提供了大量的整齐的工具。纵观各种Cloud Foundry的实现，并希望货比三家 ？或者，您是否想请求将某些框架或服务添加到Cloud Foundry ？
  7. Viral Patel 在Spring MVC上整理了一篇不错的文章来处理多个文件的上载 。
  8. Erwin Vervaet撰写了一篇很棒的博客文章，介绍了新生的Spring Shell项目 。做得很好！
  9. Santosh Rangarajan在介绍Spring Security 3.1的概念方面做得很好。
