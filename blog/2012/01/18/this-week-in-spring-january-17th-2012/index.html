    欢迎来到春季本周的另一期。本周我们要看的内容很多，所以让我们继续吧。


    1. s你们这19号星期四在做什么？
      如果您还没有，我是否建议您注册
      与Juergen Hoeller和Chris Beams举行的Spring 3.1网络研讨会 ？

             If you're in Europe,  there's going to be a  webinar  at 3PM GMT, or 4PM CEST, and if you're in North America, there's going to be a webinar at 1PM EST or 10AM PST. 
        Don't be left out, this event's already seen the highest registration of any webinar thus far, so register now, and as usual, I hope to see you there.
    2. Another session from SpringOne2GX, on Writing Applications for Cloud Foundry using Spring and MongoDB, has been posted. The Spring material is presented by data expert Thomas Risberg and the Mongo material is presented by Jared Rosoff from 10gen.
    3. Willie Wheeler, author of Manning's Spring in Practice, has put together a great blog post called Calling the GitHub API using Spring's RestTemplate. Craig Walls, author of Manning's own Spring in Action and lead of the Spring Social Github project read this blog post and reached out to Willie. They started collaborating. A few pull requests later and that integrated, cleaned up idea and code is now in Spring Social Github master, ready to be incorporated in the next release. That's three days from blog to master! This is a great example of how powerful open source is. Don't hesitate to get involved, too!
    4. Cloud Foundry has just announced support for Manifests, which contain deployment information for an application. These manifests can be used to remove many of the tedious interactive prompts encountered when deploying an application using the vmc tool making deployments much more reproducible.
    5. Additionally the Cloud Foundry group has also announced support for installing and running the Java-based Jasper Reports server as a service on Cloud Foundry.
    6. Willie Wheeler's been busy, of late! Some content from his upcoming book is available on the internet. In particular, Creating Your First Spring MVC applications is really good, and worth a read. He also has another post, Making <form:select/> work nicely using Spring 3 Formatters, that is definitely worth a read! It answers a very common question and shows the latest and greatest in best practices. Nice job, Willie!
    7. The Rai Studies blog has a few great posts this week, including a post on Using Spring MVC 3 with JQuery's Ajax support and a post on File Uploads Using Spring MVC 3.
    8. Possibly inspired by the Cloud Foundry add-on for Spring Roo, Shekhar Gulati at Xebia has cooked up another PaaS add-on, this time for Openshift. This is a great example of Spring Roo's powerful addon mechanism and the ecosystem thats grown up around it. If Spring Roo doesn't have something for you, it's easy enough to extend it! Readers of this blog will recall that Shekhar wrote the great blogs introducing Spring Roo on his IBM Developer Works series, which, if you didn't read it then, you should read now.
    9. Ashish Sarin, author of the Spring Roo Cookbook, has put some great content up on Adding Dynamic Finder Methods in Spring Roo.
    10. The TomcatExpert.com site has a great roundup of some of the great content from 2011.
    11. Over on the Apache Tomcat mailing list comes news of an update to Apache Tomcat 6 and 7 designed to fix a severe DoS attack. Please consult the post for information on the attack as well as the appropriate fixes.
  • 来自SpringOne2GX的另一场会议涉及使用Spring和MongoDB编写Cloud Foundry应用程序 。Spring资料由数据专家Thomas Risberg呈现，Mongo资料由10gen的Jared Rosoff呈现。
  • Manning的Spring in Practice的作者Willie Wheeler撰写了一篇很棒的博客文章，名为使用Spring的RestTemplate调用GitHub API。 曼宁自己的《 Spring in Action》一书的作者和Spring Social Github项目的负责人Craig Walls阅读了此博客文章，并与Willie取得了联系。他们开始合作。几个引入请求后和集成，清理思路和代码现在是在春季社会Github的master ，准备在下一版本中被合并。从博客到master这三天！这是开源功能强大的一个很好的例子。也毫不犹豫地参与其中 ！
  • Cloud Foundry刚刚宣布支持Manifests ，其中包含应用程序的部署信息。这些清单可用于删除使用vmc工具部署应用程序时遇到的许多繁琐的交互式提示，从而使部署更具可重复性。
  • 此外，Cloud Foundry小组还宣布支持在Cloud Foundry上安装和运行基于Java的Jasper Reports服务器即服务 。
  • 威利·惠勒（Willie Wheeler）最近很忙！他即将出版的书中的一些内容可以从互联网上获得。特别是， 创建您的第一个Spring MVC应用程序确实很好，值得一读。他还有另一篇文章， 使用Spring 3 Formatters使很好地工作 ， 绝对值得一读！它回答了一个非常常见的问题，并显示了最新的最佳实践。干得好，威利！
  • Rai Studies博客本周有一些不错的帖子，包括关于使用带有JQuery的Ajax支持的Spring MVC 3的帖子，以及关于使用Spring MVC 3的 文件上传的帖子。
  • 可能是受Spring Roo的Cloud Foundry附加组件的启发，Xebia的Shekhar Gulati 编写了另一个PaaS附加组件， 这次是Openshift 。这是Spring Roo强大的插件机制以及围绕它成长的生态系统的一个很好的例子。如果Spring Roo没有适合您的东西， 扩展它就很容易了 ！该博客的读者会回想起Shekhar在他的IBM Developer Works系列文章中写了许多介绍Spring Roo的很棒的博客，如果您还没有阅读的话，现在就应该阅读。
  • Spring Roo Cookbook的作者Ashish Sarin在“在Spring Roo中 添加动态查找器方法”中添加了一些很棒的内容。
  • TomcatExpert.com网站对2011年以来的一些重要内容进行了 汇总 。
  • 关于Apache Tomcat邮件列表的信息是有关旨在修复严重的DoS攻击的Apache Tomcat 6和7的更新消息。请咨询该帖子以获取有关攻击以及适当的修复程序的信息。
