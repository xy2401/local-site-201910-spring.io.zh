春天的这个星期-2012年6月12日

工程
乔什·朗（Josh Long）
2012年6月13日

欢迎回到春季本周的另一期。这周我们有很多工作要做，让我们开始吧！


  1. Gordon Dickens又来了一次，这一次很好地介绍了Spring 3.1的构造函数名称空间 ，该名称空间提供了与之相对应的逻辑。 p:命名空间元素。

  2. Matt Vickery再来一次！他有一篇关于如何在Spring上使用C24 iO产品的有趣文章。

  3. Vaadin博客上有一篇有关Vaadin Web框架和Spring的序列化的有趣文章。
  4. Java Code Geeks的博客文章介绍了如何将RESTEasy REST框架与基于Spring的服务结合使用 。尽管我建议通过这种方法在Spring MVC中提供Spring REST支持，但是如果需要使用配方，至少是很有趣的。

  5. Java Code Geeks博客上还有一篇关于构建基于Spring的JPA服务的文章，该服务位于RESTful CXF后端的后面 。这是另外一种情况，虽然在必要时知道如何做是有用的，但最好还是使用Spring MVC的REST支持。它更容易，并且可以更自然地与组件模型集成。

  6. “ 撞墙撞我的头”博客有一篇关于从Spring 2.5升级到3.1的精彩文章。
    该博客显示，迁移非常简单，如果您还没有进行迁移，那么他已深入了解您可能会遇到的一个小难题 ，可以使迁移更加顺畅。

  7. TeamExtension博客上有一篇快速文章，介绍如何开始使用Spring Mobile 1.0 。他们建议使用带有m2e支持的Eclipse，但是，当然，如果使用SpringSource Tool Suite ，则无需进行任何设置。

  8. 您是.NET开发人员，正在寻找像Spring这样的可靠依赖注入框架吗？您是否从SpringSource背后的同一人那里听说过Spring.NET （依赖注入框架）？
    博主Åukasz Budnik撰写了一篇有趣的文章，内容是有关Spring.NET在该领域其他替代产品（微软的Unity和Ninject）上的优越性 。
