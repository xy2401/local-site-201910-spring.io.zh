SpringSource工具套件3.0.0。M2发布

发布
马丁·利珀特
2012年6月28日

亲爱的春季社区，

我很高兴地宣布即将推出的SpringSource Tool Suites 3.0的第二个里程碑版本已可用。

此M2版本包含许多重大更改，并反映了未来工具的新结构。主要目标是从一种大型的整体工具转向较小的独立组件。因此，SpringSource Tool Suite的不同组件已经彼此分离，并且现在可以单独使用。

所有这些使我们能够开始交付不同口味的工装项目。从此M2版本开始，我们开始交付两种不同的即用型发行版：

  • Spring Tool Suite：这与旧的SpringSource Tool Suite基本相同。它为您带来了所有相关功能，使您可以在IDE中进行Spring开发（包括完整的Spring IDE功能集，tc Server集成以及tc Server Developer Edition 2.7.0和Spring Roo 1.2.2的现成运行时）。
  • Groovy / Grails工具套件：这个家族的新成员为您提供了即用的Groovy / Grails开发经验，而无需安装任何其他组件。它具有预安装和预配置的所有必需部件：带有Groovy 1.8的Groovy-Eclipse，Grails IDE，tc Server集成以及用于tc Server Developer Edition 2.7.0和Grails 2.0.4的现成的运行时。

每个分发版都随附了仪表板，您可以根据需要将其余项目添加到安装中。

由于组件化工作，一些工件已重命名，并且这种重命名影响了项目中存储的某些Eclipse元数据。M2版本的发行版包含迁移工具，可以很好地帮助您自动更新项目元数据以使用新的命名方案。

有关新功能和更改功能的更多详细信息，请参阅3.0.0的新功能和值得注意的功能。M2文件。

如果您遇到不兼容，迁移问题，错误或任何其他建议，请在论坛中提供反馈，并通过JIRA解决文件问题。

可从STS下载页面获得下载 ，请查看“其他下载”部分。

3.0.0。M3计划于2012年7月上旬发布，随后3.0.0版本将于2012年8月上旬发布。
