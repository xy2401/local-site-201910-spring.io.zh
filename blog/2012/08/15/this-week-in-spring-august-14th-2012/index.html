2012年8月14日春季，本周

工程
乔什·朗（Josh Long）
2012年8月15日

欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，所以让我们开始吧。



  1. SpringSource工具套件已经开源 ！而且，它现在有两个不同的版本，支持两个不同的开发人员：由Spring Tool Suite提供服务的Spring开发人员，以及由Groovy / Grails工具套件提供服务的Groovy and Grails开发人员。有关此奇妙新闻的更多信息，请查看工具套件负责人Martin Lippert的博客。

  2. Spring Security主管Rob Winch宣布发布了Spring Security的最新版本3.1.2 。

         

  3.  The ZeroTurnaround blog has a  really cool little blog on rapid development with Spring and Hibernate. 
     Of course, Zero Turnaround has a handy little software agent that lets you reload Java classes on the fly. So that's a huge gain in productivity right there. That, coupled with XML-free Spring 3.1 and Hibernate 4.1, and you have yourself a really awesome combination. To learn more, check out the blog!  The example he illustrates are also well represented in this sample project on GitHub. The example even  includes web.xml, even though it isn't required, just as was done in that blog post, specifically because it's more reliable on the buggy versions of JBoss 7 and Servlet 3 environments aren't ubiquitous, yet. 
        This is a seriously cool blog post, be sure to read the followups!

    
    4.

  4. Joseph Kulandai 在春季有一篇不错的介绍性博客文章ApplicationContext ，其中包括我早已忘记的一些小知识！很好，约瑟夫！

  5. 之前我们已经讨论过这一点，但是InfoQ的报道也很棒。
    IG Group开源RESTdoclet ，这使得为基于Spring的RESTful服务生成文档变得非常简单。REST不管好坏，都没有SOAP的WSDL之类的东西，WSDL枚举了连接到基于SOAP的Web服务的客户端的接口协定。这项技术填补了这一空白。

  6. Paxcel Labs has an interesting blog post on that attempts to demystify IoC (inversion of control) and how Spring supports the concept.

  7. Hybris博客最近一直在流行，他们从客户端的角度介绍了使用OAuth的帖子，但是该帖子特别介绍了如何设置Spring Security OAuth。

    is the one I've been waiting for!  It's a very nice read, and I'm glad they've shared it with us, complete with working code and a blow-by-blow breakdown of the pieces. OAuth is complex, Spring Security OAuth (not yet GA!) makes it much simpler, but it's still nice to have guidance.

  8. 说到Spring Security，Java Code Geeks的博客文章中有一个经典的食谱（老歌但好吃）： 如何实现UserDetails由您自己设计的Hibernate实体支持的对象 。

  9. Biju Kunjummen有一篇很棒的博客文章介绍

    
     some of the nuances of using Java-based configuration 
. Specifically, he reminds us to let Spring provide the lifecycle callbacks by routing things through @Bean methods, instead of instatiating the objects directly.


  10. 如果您使用Spring.NET，并且想利用AOP方法的拦截功能 ，请查看这篇文章！

  11. Spring Data Neo4j领导迈克尔· 汉格 （Michael Hunger） 刚举办了非常酷的Spring Data Neo4J网络研讨会 （在我们的YouTube频道上找到更多很棒的内容！），并整理了一个后续博客，其中提供了有用的资源，如幻灯片的位置 。谢谢，迈克尔！

  12. Partha Bhattacharjee继续他的博客系列，介绍Spring Integration。这次，他谈论了Spring Integration的网关，这些网关使请求/答复语义交换变得非常简单 ！

  13. 调试队长在使用Spring 3.1的配置文件以XML定义特定于环境的bean方面有一篇很棒的文章。

  14. Halycon代码博客上有一篇关于使用Spring的子上下文的文章，可让您利用Spring在import陈述。通常，这些语句无法看到当前应用程序上下文的属性占位符，但是嵌套上下文可以看到父上下文的属性占位符。解？只需在子上下文中实例化所有内容，以便import语句可以使用父上下文的属性 ！这个把戏有点偷偷摸摸！我喜欢。

    记住，子上下文提供了作用域-一个孩子中定义的bean无法看到另一个孩子中定义的bean。这对于诸如Spring Batch Admin之类的事情非常有用，它使您可以即时上传和启动新的Spring Batch作业（它们本身基于Spring配置）。


  15. Java Assist博客上有一篇不错的文章，介绍了在3.0或更高版本中使用Spring MVC bean验证 。凉！

  16. Intelligrape博客上有一篇关于使用Grails对Spring Beans的支持以及利用Spring的生命周期回调（特别是利用它）的快速帖子。 InitializingBean#afterPropertiesSet 。 

  17.    The Middleware magic blog has a nice, complete post on configuring  Spring to talk to a JBoss AS 7-managed JMS instance. The  example is particularly involved because of all the work required to setup JBoss and to build up a client connection to it.
