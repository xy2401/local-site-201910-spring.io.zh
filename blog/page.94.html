  • 尽管您可以使用Spring Cloud Zuul或新的Spring Cloud Gateway技术来实现，但这并不是与Spring专门相关的，但是Stripe的这篇关于负载均衡器和卸料机细微差别的文章非常有用。

Spring Tool Suite 3.8.4发布

发布
马丁·利珀特
2017年4月3日

亲爱的春季社区，

我很高兴宣布Spring Tool Suite的3.8.4版本，这是我们基于Eclipse的工具。

此版本的重点包括：

  • 更新到Eclipse Neon.3
  • 彻底改造的向导，可直接访问start.spring.io
  • 引导仪表板：支持本地Spring Cloud CLI服务
  • 引导仪表板：将应用程序部署到CF时对路由的支持
  • 修复了主要STS仪表板上的字体渲染
  • 增加了对Buildship 2.0的支持
  • 显示正在运行的启动应用程序的状态时，提高了启动仪表板的可靠性
  • 问题跟踪已移至GitHub Issues

使用Spring Boot堆栈，您可以在几分钟内启动Spring项目。对于使用Spring和微服务采用者的开发人员而言，Spring Boot取得了巨大的成功。

2017年3月22日

我很高兴代表团队宣布发布Spring Cloud Data Flow 1.2的第三个里程碑。

注意：开始使用此新版本的一个好方法是遵循项目页面上的版本列表 ，其中包括下载坐标和参考指南的链接。

1.2 M3版本的亮点：

伴随元数据工件
