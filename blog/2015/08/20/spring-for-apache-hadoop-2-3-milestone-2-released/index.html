面向Apache Hadoop 2.3 Milestone 2的Spring发布

发布
托马斯·里斯伯格
2015年8月20日

我们很高兴宣布Spring for Apache Hadoop 2.3 M2里程碑版本。

此版本中最重要的增强功能：

  • 新的批处理小任务：支持运行简单的Spark应用[ SHDP-397 ]和运行简单的Sqoop2 Job [ SHDP-506 ]
  • 更好的启动支持：将jobHistoryAddress添加到SpringHadoopProperties中以进行启动配置[ SHDP-517 ]
  • 纱：纱线容器组名称中的支持点[ SHDP-515 ]

有关详细信息，请参见发行变更日志 。

通过Spring Batch Tasklet在YARN上运行Spark作业的新支持为将Spark任务集成到较大的Spring Batch流中提供了可能性。我们将在即将举行的“ Hadoop工作流和分布式YARN应用程序”的 SpringOne 2GX会议上展示一个简单的示例。 使用Spring技术”的演示文稿。在本演示中，我们还将探讨基于Hadoop的应用如何利用云原生开发方法。

我们将继续在Spring IO里程碑存储库中提供特定于版本的工件及其各自的传递依赖项：

  • 2.3.0。M2（默认-Apache Hadoop稳定版2.7.1）
  • 2.3.0。M2-hadoop26（Apache Hadoop 2.6.0）
  • 2.3.0。M2-phd30（Pivotal HD 3.0）
  • 2.3.0。M2-cdh5（Cloudera CDH 5.4）
  • 2.3.0。M2-hdp23（Hortonworks HDP 2.3）

请提供反馈，以便我们可以优先安排计划在2.3上使用的新功能（请参阅以前的博客文章以获取列表）和任何其他功能请求。您可以使用JIRA问题或GitHub问题（请参阅项目页面以获取链接）。
