春天的这个星期-2015年8月4日

工程
乔什·朗（Josh Long）
2015年8月4日

欢迎来到春季本周的另一期！本周，我在华盛顿特区，与我的朋友Andrew Clay Shafer在Agile 2015大会上进行了一些云原生的Realtalk交流，然后去了科罗拉多州的丹佛市，再次与客户交谈！如果您在任何一个地区，请打个招呼！

我们正在快速接近SpringOne2GX 2015！随着团队提供越来越多的精彩作品，这里的事情变得令人兴奋！进入SpringOne2GX的接下来的几周是我一年中最喜欢的时间。观看此空间以获取大量新版本！

  • 当然，本周最大的公告是Spring Framework 4.2的全面可用性 ！趁热烧碎！这是一个改变游戏规则的版本，包含许多新功能！在您阅读新功能列表时，我将等待。
  • ..欢迎回来！您是否已将应用程序升级到Spring Framework 4.2？好！列表中的下一个：Spring Framework 负责人Juergen Hoeller的后续文章期待Spring Framework 4.3的下一个功能版本
  • Spring Session（和Spring Security！）领队Rob Winch 刚刚宣布了Spring Session 1.0.2 。该版本是一个错误修复版本，为在Spring Session 1.1上开始工作铺平了道路。因此，这是推荐的简单升级。
  • Spring Statemachine的负责人Janne Valkealahti 刚刚发布了Spring Statemachine 1.0.0 M3 ，它具有新功能，包括分布式状态机，持久性，状态机上下文，轻松将枚举用作状态和事件，状态机的程序化实例等等。
  • 上周，我写了一篇博客，内容涉及将应用程序从遗留环境迁移到Cloud Foundry等现代云平台 。这篇文章是关于如何将应用程序推入新的闪亮平台的。
  • Apache Hadoop的Spring主管Thomas Risberg刚刚宣布了Apache Hadoop 2.3 M1的Spring 。此版本将Hadoop 2.7.1添加为默认版本，添加了对HDP 2.3和CDH 5.4.4版本的支持，包括错误修复，并更好地支持Spring Boot以及Hive 1.x和HiveServer 2。
  • Spring Data负责人Oliver Gierke 刚刚宣布了Spring Data Gosling的第一个候选版本，其中包括用于Spring Data REST和Spring MVC的QueryDSL集成，Spring Framework 4.2和Spring Boot 1.3的集成， RepositoryRestConfigurer便于配置，以及大量更新的模块版本。
  • 我喜欢James Watters 在Cloud Native Journey上的帖子。
  • Scott Frederick刚刚宣布Spring Cloud Connectors 1.2.0现已可用！Spring Cloud Connectors利用了依赖注入所提供的间接性来实例化和定义bean，这些bean通常是通过诸如环境变量之类的上下文在云环境（如Cloud Foundry）中公开的通用支持服务的客户端。此发行版包含一些关键的错误修正以及对许多事物的新支持，包括IBM DB2和Microsoft SQL Server关系数据库。
  • 在使用Spring和Swagger时，我不能完全读懂日语幻灯片，但是看起来很有趣 。但是，我建议人们在进入Swagger之前先咨询Spring REST文档 。
