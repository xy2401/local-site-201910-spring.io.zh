    Spring for Apache Hadoop 2.3 Milestone 1发布

    发布
    托马斯·里斯伯格
    2015年8月4日

    我们很高兴宣布Spring for Apache Hadoop 2.3 M1里程碑版本。

    Spring在Apache Hadoop 2.3 M1版本中最重要的增强功能：

    • 将Hadoop 2.7.1添加为默认版本
    • 添加对HDP 2.3和CDH 5.4.4的支持
    • 错误修复：修复附录重新打开可能会失败SHDP-510
    • 更好的启动支持：支持启动配置道具元数据SHDP-452
    • 添加对Hive 1.x和HiveServer2的支持

    有关详细信息，请参见发行变更日志 。

    为了能够使用Hive 1.x，它放弃了对HiveServer1和相应的Thrift客户端的支持，我们不得不在Spring中为Apache Hadoop重写Hive支持。现在，我们使用JDBC客户端支持HiveServer2，对于最常见的用途，您只需更改客户端配置即可使用hiveDataSource。这是一个例子：

    <hive-client-factory id="hiveClientFactory" hive-data-source-ref="hiveDataSource"/>
<beans:bean id="hiveDriver" class="org.apache.hive.jdbc.HiveDriver"/>
<beans:bean id="hiveDataSource" class="org.springframework.jdbc.datasource.SimpleDriverDataSource">
	<beans:constructor-arg name="driver" ref="hiveDriver"/>
	<beans:constructor-arg name="url" value="jdbc:hive2://localhost:10000"/>
</beans:bean>

    如果您使用了org.apache.hadoop.hive.service.HiveClient直接在您的代码中，您需要切换以使用新的org.springframework.data.hadoop.hive.HiveClient它模仿了旧的基于Thrift的客户端，但是使用JDBC驱动程序与HiveServer2进行通信。

    在此提交中可以看到我们对Spring for Apache Hadoop Hive示例代码所做的更改。

    我们将继续在Spring IO里程碑存储库中提供特定于版本的工件及其各自的传递依赖项：

    • 2.3.0。M1（默认-Apache Hadoop稳定版2.7.1）
    • 2.3.0。M1-hadoop26（Apache Hadoop 2.6.0）
    • 2.3.0。M1-phd30（Pivotal HD 3.0）
    • 2.3.0。M1-cdh5（Cloudera CDH 5.4）
    • 2.3.0。M1-hdp23（Hortonworks HDP 2.3）

    请提供反馈，以便我们可以优先安排计划在2.3上使用的新功能（请参阅以前的博客文章以获取列表）和任何其他功能请求。您可以使用JIRA问题或GitHub问题（请参阅项目页面以获取链接）。
