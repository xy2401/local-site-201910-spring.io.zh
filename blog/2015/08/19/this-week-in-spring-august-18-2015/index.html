Spring 的这个星期-2015年8月18日

工程
乔什·朗（Josh Long）
2015年8月19日

欢迎来到 Spring 本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！本周，我回到巴西圣保罗拜访客户，然后前往里约热内卢QCon Rio ！如果您在任何一个城市，请在Twitter上给我发消息，让我们来喝杯咖啡 ！

  • 今天，Pivotal首席执行官Paul Maritz辞职[并欢迎Pivotal Labs创始人Rob Mee担任新首席执行官。 保罗（Paul）和罗布（Rob）都有一个帖子。谢谢Paul，欢迎Rob！
  • 令人惊异的Scott Frederick刚刚宣布了Spring Cloud Connectors 1.2.0的全面上市！ - 看一下这个！
  • Spring 忍者StéphaneNicoll刚刚发布了Spring Boot 1.3M4， 并进行了大量修复和改进 ！
  • Spring Integration负责人Gary Russell刚刚宣布了一个CRAZY打包的新Spring Integration 4.2版本，包括但不限于对安全性上下文传播，STOMP客户端通道适配器，指标，新的Spring Framework 4.2事件通道适配器，流程障碍组件的支持，最后修改文件列表过滤器，编解码器，JMS共享订阅，（S）FTP改进，SOAP Action Propagation 等
  • Spring Integration和Spring AMQP负责人Gary Russel 还不为所动， 还宣布Spring AMQP 1.5.0已经发布 ！
  • SPR博客上有一篇不错的文章，介绍了Spring Boot应用程序中的国际化
  • Stephane Deraco对Spring Boot进行了很好的介绍
  • 检出此实现以将bean依赖项解析为Spring MVC处理程序参数
  • 看看这两个部分的播客，我与Michael Cote一起使用Spring Boot，Spring Cloud和Cloud Foundry进行云本地化
  • Pivotal Cloud Foundry 终于支持Apache Geode（该项目以前称为Gemfire）
  • 我们的朋友Petri又回来了，这一次是关于将Gradle用于Spring Boot Web应用程序的帖子
