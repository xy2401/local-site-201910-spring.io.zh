Spring 的这个星期-2015年6月9日

工程
乔什·朗（Josh Long）
2015年6月9日

欢迎来到 Spring 本周的另一期！好一个星期！本周，我在堪萨斯城参加了由单个公司的开发人员组成的6,000多人的开发人员会议，然后又回到纽约，参加QCon NYC，在这里我将讨论使用Spring Boot和Spring Cloud构建云原生应用程序。如果您在任何一个地方，请务必与我联系，然后聊一下商店（ Spring ，以及（可选）在堪萨斯城的BBQ。）

  • Spring联合创始人兼项目负责人Juergen Hoeller在本周整理了两篇必读文章： 第一篇是Java 8和Spring 4的采用 。而且， 非常重要的是： 他提到了本周的 Spring ：D #WIN 。
  • Juergen 还发布了我多年以来最喜欢的文章之一， 他对Java EE 7的现状持残酷诚实的看法，并且对生产支持的Java EE容器缺少（大厂商）支持的实现选项感到沮丧。你知道，这个人有一点。
  • 神奇的波拉克博士（见上文！）发布了Spring XD 1.2 RC1 。新版本中包含了许多新内容，包括用于将Spring XD打包和部署到生产中的Apache Ambari插件，针对Apache Kafka支持的新分析，新功能和性能改进，针对RabbitMQ的改进的HA配置，Sqoop元存储支持以及很多 ！看看这个！
  • 查看Spring XD ninja Janne Valkealahti的这篇文章，以了解有关Spring XD的Apache Ambari部署选项的更多信息。
  • Spring and Spring Boot忍者Stephane Nicoll刚刚宣布了Spring Boot 1.2.4
  • SébastienDeleuze刚刚发布了有关Spring framework 4.2即将提供的CORS支持的必读文章 。看起来真酷！
  • 我们的好朋友Jacob Severson在最近为Spring Data REST添加电子标签以及它们为我们的开发人员提供的信息中发表了一篇不错的文章
  • 不要错过Jamie O'Meara即将举行的与Jenkins，Spring和Cloud Foundry持续集成的网络研讨会
  • 看看Phil Dutson 即将举行的网络研讨会.jsp和JSTL页面到Thymeleaf
  • 我的朋友詹姆斯·沃德（James Ward）再次发表了精彩的文章，这次是关于2005年左右的部署与今天的应用程序部署 。它仅与最特定的Spring技术有关，但值得一读。
  • 查看史诗巨著《释放》的作者迈克尔·尼加德的演讲。 ， 为什么我们需要弹性软件
  • 我喜欢Camille Fournier 在各种共识软件（例如ZooKeeper和Etcd）上的采访
  • 我上个月在本地的旧金山Java用户组中进行了一次演讲， 介绍了Spring Boot ，人们似乎很喜欢它。希望你会的。
