Spring 的这个星期-2015年6月23日

工程
乔什·朗（Josh Long）
2015年6月23日

欢迎来到春季本周的另一期！上周，我在伦敦的Devoxx UK 。该节目在各个方面都取得了成功。我做了一个演讲，一个讲习班，也很高兴参加闭幕主题演讲！这个星期，我在波兰克拉科夫，为我Devoxx波兰 （NE 33度），这似乎是一个了不起的成就，以及！我要感谢组织者（分别来自 Devoxx UK和PL的Mark Hazell和Grzegorz Duda ）邀请我-一段时间以来我没有这么开心，我热爱这两项活动所服务的社区。

除了在主题演讲中，我有幸短暂地参加了本次主题演讲的自拍照，离开的阿伦·古普塔（Arun Gupta）花了一点时间来表彰最近提名的Java冠军（排在第二位）的Trisha Gee和David Blevins和我自己）。非常感谢，阿伦！我将其发布在这里，因为它对我来说非常特别。;-)

因此，最特别的是：令人敬畏的潮流，这是本周的“ 本周春季”中的重要内容 ！

  • Spring XD忍者Marius Bogoevici宣布了Spring Integration Kafka 1.2 。新版本包括对Kafka 0.8.2的支持，性能改进和改进的文档。所有这些都是值得推荐的升级！
  • 春季指南的忍者Greg Turnquist研究了Spring团队的Java 8优先方法，然后宣布史诗指南默认情况下已移至Java 8。
  • Biju Kunjummen在DZone上对Spring Cloud Eureka和Spring Cloud Config Server进行了很好的介绍- 简洁而有用！
  • 我们的常驻Spring Boot巫师Phil Webb 看着spring-boot-devtools在这个惊人的博客文章中。认真地，去读这个。我会等待..（您是否观看了视频？那不是很酷吗？）
  • 大数据医生Pollack博士刚刚宣布了Spring XD 1.2 GA，Spring XD 1.1.3和Spring XD Flo beta 。这些版本中有很多很棒的东西！我特别喜欢Spring XD Flo beta（甚至包括一个超酷的视频）！
  • Spring XD的忍者Glenn Renfro一直在进行 Spring XD的基准测试 （而非基准测试 ），并在第一部分中记录了他的旅程。
  • 我们的好朋友Eugen Paraschiv又回来了，这次提供了Spring Boot应用程序的超方便配置选项列表 。我一定会给这个评论！
  • 来到波兰的真正好处之一是，我终于见到了Marcin Grzejszczak和令人惊叹的4Finance IT团队的其他成员！他们负责对Spring Cloud做出一些真正的贡献，包括对Spring Cloud Zookeeper和Spring Cloud Sleuth的改进，它们提供了通过微服务进行跟踪的功能-令人兴奋！这些改进来自他们的其他工作-一个称为“微型红外”的项目-已被合并到Spring Cloud中。Dziękuję，我的朋友们！
  • 几乎没有这么真棒，但我很骄傲，最近合并的支持的云春DiscoveryClient Cloud Foundry的实施 。这意味着，很快，我们将拥有DiscoveryClient Netflix的Eureka，Zookeeper，Consul，Lattice以及现在的Cloud Foundry的实施！
  • 说到Spring XD，我想特别提请注意这些介绍性视频， Building Streams和Advanced Stream Creation 。
