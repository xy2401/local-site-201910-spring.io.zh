春季数据发布火车高斯林的第一个里程碑

发布
克里斯多夫·斯特罗布尔
2015年6月2日

我很高兴代表Spring Data团队宣布Spring Data发布火车Gosling的第一个里程碑。该发行版附带固定的164张票证 ，以及带有初始里程碑的Spring Data KeyValue加入Spring Data发行系列。该版本的重点包括：

  • Spring 4.1作为新的最低基准。
  • 简化了对在各个模块中的所有存储库中添加自定义方法的支持（请参见本示例 ）。
  • 临时JPA 2.1声明@EntityGraph存储库方法通过@EntityGraph获取图声明。
  • Gemfire 8.1支持包括多索引定义操作。
  • 在规划环境地政司表达式@Query以及支持$geoIntersects MongoDB的。
  • 对Spring Data Redis的增强的ZSET支持，包括ZRANGEBYLEX 。
  • 提供HAL浏览器的Spring Data REST模块以及对条件GET的支持。
  • 支持地图的存储库（请参见以下示例 ）
  • 支持Neo4J 2.2。
  • 在Spring Data Elasticsearch中支持派生的deleteBy...查询和批量更新。
