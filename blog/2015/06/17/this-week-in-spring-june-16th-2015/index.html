春天的这个星期-2015年6月16日

工程
乔什·朗（Josh Long）
2015年6月17日

欢迎来到春季本周的另一期！本星期
我在英国Devoxx的阳光灿烂的伦敦，在那里我将与开发人员讨论建筑
使用Spring Boot，Spring Cloud和Cloud Foundry的云原生应用程序。和往常一样，如果您在该地区，请打我 。本周的大新闻是Spring XD 1.2 GA打破了性能数字！此处没有基准测试，所有内容均已发布且可重现：使用内存传输时性能提高到约1200万消息/秒，使用Kafka（100字节消息）时达到260万消息/秒。表演博客中有更多详细信息。而且我们甚至都没有谈论过Amabari安装程序或新功能的容器。

  • 查看Spring Ninja Stephane Nicoll的博客，该博客在Spring Boot 1.3M1中引入了针对Spring Cache的智能新自动配置 （以及扩展的任何JSR107实现）
  • 春季社区成员Tadaya Tsuyukubo 在这篇精彩的嘉宾帖子中介绍了他的Spring Social Evernote绑定
  • Spring Lead Juergen Hoeller 解释了Spring framework 3.2最终计划
  • Spring Data Gemfire主管John Blum介绍了对Apache Geode的必要支持 。Apache Geode在Apache孵化项目中，并且基于开源Gemfire技术。
  • Spring Boot联合负责人Phillip Webb刚刚宣布了Spring Boot 1.3M1 ，其中包含许多强大功能
    包括神话般的 devtools支持在开发时特别有用的工具的模块：查看模板重新加载，远程调试，各种实时重新加载器等等。它还提供了我最喜欢的Spring Boot功能之一：可执行文件.jar s！使用此新功能，您可以运行mvn clean install然后运行./foo.jar在类似UNIX的环境中， 它可以正常工作 ！
  • Spring Data忍者Thomas Risberg 刚刚宣布了适用于Apache Hadoop 2.2 GA的Spring
  • Spring Framework 负责人Juergen Hoeller 邀请有关Spring Framework 5要求的反馈
  • 我们在NTT Data的朋友在Terasoluna上发表了一篇不错的文章 ，他们在Spring和Spring Boot的基础上构建了一个框架，并在构建（需要大量）持续使用的应用程序时进行认证。
  • 乔莎•斯特拉（Josha Stella） 很好地探讨了为什么我们应该选择不可变的基础设施 。Battery Ventures（以前是Netflix）的Adrian Cochroft谈到了这一点，他说：“像宠物而不是牛一样对待服务器。”这是一本不错的书，即使它与Spring毫无关系。
  • Boxfuse的人们研究如何构建Spring Boot应用程序，然后通过几个简单的步骤将其融合到Amazon EC2可运行的映像中。Boxfuse看起来很有趣，但是请记住Boxfuse是付费的。作为替代， 您现在可以在几乎任何地方简单地运行Docker映像 。
