Spring Boot 1.3.0。M1现在可用

发布
菲尔·韦伯
2015年6月12日

我很高兴地宣布Spring Boot v1.3.0。M1现在可以从Spring里程碑存储库中获得 。这个版本建立在Spring Framework 4.2.0之上。RC1并在Spring Boot 1.2上提供了许多改进和新功能 。重点包括：

  • 一个新的spring-boot-devtools支持自动重启，LiveReload和远程更新的模块。
  • 对EhCache，Hazelcast，Infinispan，Redis，Guava或任何兼容的JSR-107（JCache）实现的广泛缓存支持。缓存度量标准信息现在也通过执行器公开（当基础技术支持时）。
  • 改进的指标支持，包括导出和聚合功能，以及使用Java 8时的性能大幅提升。
  • 用于基于Linux的操作系统的完全可执行的JAR，包括“服务”支持。现在，将启动作为Linux服务启动应用程序就像键入命令一样简单： sudo ln -s /var/myapp/myapp.jar /etc/init.d/myapp 。
  • 其他OAuth2支持，包括@EnableAuthorizationServer ， @EnableResourceServer和@EnableOAuth2Client注释。
  • 新HealthEndpoints适用于Elasticsearch，Email和JMS。
  • 马文start和stop帮助集成测试的目标。

有关更改的完整列表和升级说明，请参阅WIKI上的《 Spring Boot 1.3发行说明 》和更新的参考文档 。

从现在开始，我们将以相当固定的节奏发布里程碑，以期发布1.3.0.RELEASE在九月（大约在春季一 ）。与往常一样，早期的反馈和错误报告对我们来说非常宝贵，非常欢迎。

再次感谢为发布做出贡献并支持该项目的所有人。
