春天的这个星期-2015年6月30日

工程
乔什·朗（Josh Long）
2015年6月30日

欢迎来到春季本周的另一期！本周，我将在德克萨斯州奥斯汀市一个温暖而美好的地方与客户见面，然后前往墨西哥墨西哥城在SG Conference and Expo上发表主题演讲。如果您在墨西哥城的春天或夏天在墨西哥城，请在墨西哥城设计一家啤酒店。不要犹豫，请联系Twitter（@starbuxman） ！

6月30日！今年飞来飞去 ！这周我们有很多工作要做，让我们开始吧！

  • Spring Framework 负责人Juergen Hoeller刚刚宣布了Spring Framework 4.1.7和3.2.14，它们都是维护版本。
  • Spring Framework 忍者Stephane Nicoll刚刚发布了Spring Framework 4.2 RC2，其中包含许多新功能！
  • Spring Tool Suite负责人Martin Lippert刚刚宣布了3.7.0更新 。更新包括Eclipse Mars 4.5 GA，Spring Boot YML支持，改进的Cloud Foundry集成，Pivotal tcServer 3.1.1支持以及对Spring Roo 2.0.0的早期支持。M1（感谢Disid）。
  • Spring Boot和Spring Cloud项目的共同创始人好人Syer博士刚刚宣布了首个Spring Cloud发布培训 Angel。SR3 。
  • Spring RESTDocs领导者Andy Wilkinson 刚刚宣布了Spring REST Docs 1.0.0。M1 。
  • Spring Framework Guru博客很好地论述了将Spring Boot与Thymeleaf结合使用 。
  • 希望您能抓住Spring Data主管Oliver Gierke的史诗般的vJUG演示，内容涉及使用Spring Data进行优雅的数据访问 ！
  • Spring Data艺术家Thomas Darimont整理了一个不错的GitHub GIST，介绍了如何在@Value注解 。这里有很多很棒的例子，请查看！
  • Alberto Souza撰写了一篇不错的（西班牙语）文章，介绍如何从Spring Boot的魔术开始
  • Mohamed Sanaulla上有一篇不错的文章，介绍了如何使用Spring Boot对基于Spring Boot的REST服务运行完全集成测试
  • Keyhole软件博客上有一篇不错的文章，内容涉及将Spring Batch作业从XML配置迁移到Java配置DSL。
  • Nixmash博客上有一篇不错的文章，介绍了如何更改默认值.favico在Spring MVC中显示的文件。这篇文章谈论的是Spring MVC，但实际上，给出的方法仅在Spring Boot中使用Spring MVC时有效 。否则，很棒的帖子！
  • 贾斯汀·穆斯格罗夫（Justin Musgrove）整理了一段很酷的视频，并演练了如何使用Spring MVC防止双击预防
  • 观看本CFSummit演讲， 了解Cloud Foundry Service Broker API中的一些新功能
  • 我们的朋友@Making为JSUG进行了精彩的（日语）演讲， 可在此处找到幻灯片
