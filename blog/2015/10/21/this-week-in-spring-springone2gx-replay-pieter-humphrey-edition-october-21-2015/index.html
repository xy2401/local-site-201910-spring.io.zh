This Week in Spring ( SpringOne2GX replay / Pieter Humphrey edition!) - October 21, 2015

Engineering
Josh Long
October 21, 2015

Welcome to another installation of This Week in Spring! This week is the first week (of many) where the crazy awesome [https://twitter.com/pieterhumphrey](Pieter Humphrey) has released scores of SpringOne2GX 2015 videos for our binge watching enjoyment! Most of this week’s bountiful roundup is thanks to Pieter’s meticulous transcoding, uploading and posting videos from SpringOne2GX 2015 - thanks Pieter! So without further ado, let’s take a look at some of this week’s selection including a LOT of content from SpringOne2GX, including..

  • ..Juergen Holler and Stephane Nicoll’s talk presenting the updates to Spring framework 4.2
  • ..the SpringOne2GX day 1 keynote featuring James Watters, Dr. David Syer, Scott Frederick, Juergen Hoeller and yours truly
  • ..the SpringOne2GX day 2 keynote featuring Guillaume LaForge, Graeme Rocher, Dr. David Syer, Mark Fisher, Scott Frederick and yours truly
  • ..the talk that Spring Boot and Spring Cloud co-founder Dr. Dave Syer and I, aided by a VERY gracious guest appearance by Spring creator and hero Dr. Rod Johnson, gave introducing Spring Cloud (I’ve never been so privileged as I was to be a part of this talk and I truly hope you enjoy this one)
  • ..Netflix’s Jon Schneider and Taylor Wicksell showing - live - how they build cloud native applications using Spring Boot and Spring Cloud at Netflix
  • ..Matt Stine’s talk introducing some of the powerful new features in Lattice
  • .. Grails lead Graeme Rocher’s Grails 3 update
  • ..Spencer Gibb’s talk introducing some of the various extensibility planes of Spring Cloud
  • ..John Field and Shawn McKinney’s talk discussing some of the security concerns when moving applications to a PaaS
  • ..Will Tran’s talk on securing microservices with Spring Cloud Security
  • ..Google’s Ray Tsang introducing how to run Spring Boot-based microservices on Kubernetes
  • ..Nicholas Frankel’s talk introducing devops with Spring Boot
  • ..Cornelia Davis’ talk on 12 factor Spring applications
  • ..AppDynamics’s Dustin Whittle’s talk about modern frontend engineering
  • ..VMWare’s Emad Benjamin’s talk about building and tunng high performance Java platforms in a world of virtual machines and the cloud
  • ..Colin Harrington’s talk on using Grails to support Internet-of-Things use cases
  • ..the amazing incomparable Jeff Brown’s talk introducing javax.cache
  • ..and Jeff’s talk on RESTful Grails 3
  • ..and Jeff’s talk on building a microservice architecture with Grails

Whew! That’s at least a week’s worth of watching! That’s all thanks to Pieter Humphrey working overtime to get this all available as early and abundantly as possible. Pieter is also responsible for working very hard over the last few years to ensure that all talks are recorded and available so to be able to binge watch everything so quickly after the show is a true privilege. Thank you, Pieter! We still have some news to catch up on, as well! So..

  • Spring and Spring Boot ninja Stéphane Nicoll just announced that Spring Boot 1.2.7 is now available
  • Spring Data ninja Thomas Risberg just announced that Spring for Apache Hadoop RC1 has been released
  • SN has also just announced that Spring framework 4.2.2, 4.1.8, and 3.2.15 - complete with lots of bug fixes, are all now available as well
  • Martin Lippert’s just posted the second part of his series on using the Spring Boot Dashboard in Spring Tool Suite
  • Spring lead and co-founder Juergen Hoeller put together a nice post on the compatible Java EE 7 implementations and Spring
  • Janne Valkealahti just announced Spring Statemachine 1.0.0 has been released
