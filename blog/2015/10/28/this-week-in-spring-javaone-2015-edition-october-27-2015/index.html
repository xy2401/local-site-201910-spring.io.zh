This Week in Spring (JavaOne 2015 Edition) - October 27, 2015

Engineering
Josh Long
October 28, 2015

Welcome to another installation of This Week in Spring! This week I’m at JavaOne 2015 in San Francisco along with the rest of the Pivotal team. This week the Pivotal Spring team is out in full force, come stop by and say hi!

There are some great new SpringOne2GX 2015 recordings on line as well as some great community content this week so let’s get to it!

  • Spring Cloud cofounder’s Spencer Gibb and Dr. David Syer’s talk on the Spring Cloud Developer experience from SpringOne2GX 2015!
  • Spring tools ninja Martin Lippert continues his series with this third installment on using Spring Boot Dashboard in STS
  • Gary Russell has just announced a few new maintenance releases of Spring Integration
  • Check out this upcoming IntelliJ IDEA webinar on the new Spring framework and Spring Boot support in IntelliJ IDEA with Spring ninja Stéphane Nicoll
  • this post by Rajith Delantha walks through using Spring Security OAuth with Spring Boot
  • My article on the The Power, Patterns, and Pains of Microservices of microservices is now up on DZone
  • Giau Ngo has a good example mapping on using JPA many-to-many relationship mapping with Spring Boot, MySQL and Maven
  • Papo Reto has a nice in Portguese-language video - I’m pretty sure - on using Spring Boot on Heroku
  • Dan Vega’s got a really cool look at using the @SpringBootApplication on annotation
  • Do you want to get Stormpath working with Spring Boot? Check out this handy getting-started example!
  • I thought this was a cool blog on using Cloud Watch with Cloud Foundry
  • Roberto Venanzi of The Spring Side blog has an interesting Italian-language post on using Spring Boot with JSF
