Spring 本周（JavaOne 2015版）-2015年10月27日

工程
乔什·朗（Josh Long）
2015年10月28日

欢迎来到 Spring 本周的另一个装置！本周，我将与Pivotal团队的其他成员一起在旧金山的JavaOne 2015上。本周，Pivotal Spring团队将全力以赴，赶快打个招呼吧！

本周有一些很棒的SpringOne2GX 2015录制新唱片以及一些很棒的社区内容，所以让我们开始吧！

  • Spring Cloud联合创始人的Spencer Gibb和David Syer博士就SpringOne2GX 2015的Spring Cloud Developer经验发表演讲 ！
  • Spring Tools 忍者Martin Lippert的第三部分继续了他在STS中使用Spring Boot Dashboard的系列
  • Gary Russell刚刚宣布了Spring Integration的一些新维护版本
  • 在新的Spring Framework 上查看即将举行的IntelliJ IDEA网络研讨会， 并通过Spring NinjaStéphaneNicoll在IntelliJ IDEA中支持Spring Boot。
  • Rajith Delantha的这篇文章逐步介绍了如何在Spring Boot中使用Spring Security OAuth
  • 我关于微服务的力量，模式和微服务的痛苦的文章现在在DZone上发表了
  • Giau Ngo有一个很好的示例映射，说明了如何在Spring Boot，MySQL和Maven中使用JPA多对多关系映射
  • Papo Reto的Portguese语言视频很不错-我很确定- 在Heroku上使用Spring Boot
  • 丹·维加的使用时有一个很酷的样子了@SpringBootApplication在注释上
  • 您想让Stormpath与Spring Boot一起工作吗？看看这个方便的入门示例！
  • 我认为这是有关将Cloud Watch与Cloud Foundry结合使用的不错的博客
  • The Spring Side博客的 Roberto Venanzi撰写了一篇有趣的意大利语文章，介绍如何将Spring Boot与JSF一起使用
