Spring XD 1.3 M1发布

发布
马克·波拉克
十月08，2015

我非常荣幸地代表Spring XD团队宣布Spring XD 1.3的第一个里程碑版本

此版本包括错误修复和一些增强功能 ：

  • Hadoop发行版更新到Apache Hadoop 2.7.1和Hortonworks Data Platform 2.3。Pivotal Hadoop 2.1、3.0和Cloudera Hadoop 5.4
  • Spark 1.3.1支持
  • 卡桑德拉水槽
  • 标头增强器处理器
  • 对Flo Designer用户界面和管理界面的错误修复
  • Gpfdist接收器现在支持更新操作和所有控制文件选项
  • 更新了Spring Integration 4.2，该功能在启用监视时提供了性能改进。
  • 升级到Spring Data Gosling发布培训

您可以下载zip发行版或使用homebrew在OSX上安装。在RHEL / CentOS上，您可以使用rpm进行安装 。更新的Flo UI也可以单独下载。

展望未来，候选1.3版本将包括用于创建批处理作业的新DSL以及用于编写和可视化作业的Flo的新版本。1.3版候选版的目标是在10月22日发布，而1.3 GA版则在11月的第一周发布。

在1.3 GA发行版之后，1.x发行版将处于维护模式，仅解决错误修复。下一个发行版将对基础架构进行重大重新设计。基于Spring XD Zookeeper的运行时将被服务提供商接口 （SPI）取代，该服务提供商接口委托用于运行和扩展应用程序的本机平台功能。考虑到架构的重大变化，下一个发行版将更名为Spring Cloud Data Flow，以更好地表示通过云原生编程和运营模型重新构想的XD。有关更多详细信息，请参考最近发布的博客 ，并随时关注即将发布的有关Spring Cloud Data Flow的工程博客。
