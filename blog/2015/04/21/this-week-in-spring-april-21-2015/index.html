Spring 的这个星期-2015年4月21日

工程
乔什·朗（Josh Long）
2015年4月21日

欢迎来到 Spring 本周的另一期！本周，我在印度班加罗尔举行的“ 伟大的印度开发者峰会”上 。本周 ，Spring团队的许多成员也在德国JAXConf上 。

谨在此提醒您，如果您位于欧洲的西南地区，尤其是在西班牙的巴塞罗那，并且想在阳光下找乐子，那么我想不出下周的 Spring I / O Conference再好不过了。不要错过许多惊人的Spring团队与Spring社区成员的许多伟大的会谈。我会在那里，期待与大家见面！

事不宜迟，让我们开始吧！

  • Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration Hazelcast适配器 。Hazelcast是一个开源的分布式数据网格，它支持java.util.Map ， java.util.Queue和java.util.Set语义，尤其是跨节点群集的语义。
  • Spring Security和Spring Session负责人Rob Winch 刚刚宣布了Spring Session 1.0.1
  • Spring Social主管Craig Walls刚刚宣布了Spring Social Facebook 2.0.0。RELEASE，其中包括Facebook Graph API中的所有众多重大更改。
  • «卡夫卡！»Marius Bogoevici和Mark Pollack将于4月28日在星期二举行网络研讨会，内容涉及Spring XD和Kafka ，这是任何大数据/快速数据爱好者必不可少的。
  • Spring集成忍者只是放在一起的一个非常好的书面记录Spring集成的Java DSL 1.1。M1版本，具有许多新功能，包括基于POJO的EIP组件， IntegrationFlowAdapter ，动态语言支持和Apache Kafka支持。
  • ..这是我本周写的。Apache的卡夫卡是一个分布式的消息队列是使铲大量的数据，周围很容易做到 。在Spring Integration和Spring XD的支持下，它非常适合基于集成的代码和大数据处理。
  • 本周的SpringOne2GX回放已经结束！Dealer.com的Ryan Gardner讨论了如何从应用程序中获取配置，并使用Spring和Apache ZooKeeper进入集中式应用程序配置中的分布式存储中
  • 我们的朋友XavierPadró撰写了一篇有关将Spring Boot与JMS一起使用的不错的文章 ，特别是利用了Spring Framework 4.1中的编程模型。
  • 我喜欢这篇关于从Spring Boot公开Git提交信息的帖子/info执行器端点 。
  • 您是否知道Spring Boot在嵌入式容器中支持对HTTP的声明式， 易于使用的支持 ？
  • 您正在使用Vaadin和Spring吗？您知道吗，Spring Initializr中现在有一个功能强大的基于Spring Boot和Vaadin的集成。
  • 您正在使用IntelliJ吗？您是否已经尝试了IntelliJ支持Spring Boot的所有新功能？查看Spring忍者Stephan Nicoll与IntelliJ团队关于将IntelliJ与Spring Boot结合使用的联合演讲。
  • 我一直在研究安全地协调分布式系统中群集状态的方法。在这个领域，Redis，Hazelcast和Apache Zookeeper之类的工具是非常强大的选项，Spring Cloud已经有了一些非常初步的支持 。去看一下！如果您有帮助，我很想听听您的用例协调。有一些明显的情况。您将如何选择一个处理消息队列的节点作为主节点？您将如何处理主员工拓扑？
  • 我在Spring Boot的Qiita上发现了很多（日语） 帖子 ，等等。有关Spring Boot和Thymeleaf的这篇文章很有趣。这是一个[使用Spring Boot（Spring MVC）和Swagger的使用]（用于Swagger的Spring MVC集成）。这篇关于单元测试Spring MVC服务的文章看起来非常详细 。我会不时地回来检查！看来，即使从Google翻译结果来看，资源也很丰富！
