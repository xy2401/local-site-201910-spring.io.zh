春天的这个星期-2015年4月29日

工程
乔什·朗（Josh Long）
2015年4月28日

欢迎来到春季本周的另一期！本周，我在西班牙的巴塞罗那参加春季I / O会议 。


（您能找到最喜欢的Spring团队或社区成员吗？）

  • Spring Social主管Craig Walls刚刚宣布发布Spring Social Facebook 2.0.1 ，这是Spring Social Facebook 2.0系列中的第一个维护版本。
  • 好的Syer博士刚刚发布了一个博客，介绍了Cloud Foundry中绑定服务的不同方法
  • Spring Security主管Rob Winch 刚刚发布了Spring Security 4.0.1 ，这是Spring Security 4.0产品线中的第一个维护版本
  • 本周的SpringOne2GX重播已经结束！Marius Bogoevici向我们展示了如何使用Spring Integration构建高度模块化和可测试的业务系统 ，Brian Cavalier谈到了通过差分同步和JSON补丁与Topher Bullock进行有效的客户端-服务器通信， Will Tran回顾了Nashorn和Spring的服务器端JavaScript 。
  • 您上周去过伦敦JAX吗？没有？然后您错过了域驱动设计作者Eric Evan关于微服务的有限上下文的演讲 。看看这个！
  • Zurelsoft的Sanjay Shah 为Spring Boot安排了一个不错的高级演讲
  • 埃伯哈德·沃尔夫 （ Eberhard Wolff ） 完美地表达了整体应用背后的惯性：“整体重力：大型系统将吸引新的功能和变化。在此过程中，它将变得更大，并最终崩溃。”
  • 我在Oracle Netbeans平台上工作的朋友Geertjan刚刚介绍了Aggelos Karalias的一个非常酷的贡献，该贡献集成了对Spring Boot有关配置属性的元数据的自动完成支持 。此支持类似于最新的STS和IntelliJ IDEA 14.1版本中提供的支持。很酷！
  • InfoQ刚刚在Spring Security 4上发表了一篇不错的文章 。
  • Apache Geode（Pivotal的Gemfire分布式数据网格技术的核心）刚刚通过投票，成为Apache Software Foundation中的孵化器项目 ！恭喜你！
  • 我们的朋友David Welch刚刚分享了JDBC支持的MetricRepository看起来很有用！看看这个！
