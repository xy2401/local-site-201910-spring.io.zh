春天的这个星期-2015年4月7日

工程
乔什·朗（Josh Long）
2015年4月7日

欢迎来到春季本周的另一期！本周，我要去法国巴黎的Devoxx France。Spring和Cloud Foundry团队中的许多人都将在那里，因此，如果您即将与我们联系，请随时与我们联系！此外，即使您不在现场，也可以在免费的Devoxx France活动中与Spring团队中的许多人见面（包括Juergen Hoeller本人，StéphaneNicoll，Brian Clozel，SébastienDeleuze，当然还有您） 。

另外，这个星期四，我将再次在vJUG上发言， 这次是关于使用Spring Cloud构建微服务 。不要错过！

勘误表：@ springjuergen，@ snicoll，@ sdeleuze，@ brianclozel mais aussitrèsprobablement @starbuxman。致辞：eventbrite.com/e/venez-decouv…

  • Spring Boot联合负责人Phil Webb刚刚宣布Spring Boot 1.1.12和Spring Boot 1.2.3现已上市
  • 大数据忍者Thomas Risberg刚刚为Apache Hadoop 2.1.2发布了Spring
  • Spring Social主管Craig Walls刚刚宣布了Spring Social Facebook 2.0.0。RC1 。新版本将Spring Social Facebook升级到Facebook API版本2.3，并包含向后不兼容的更改。
  • Spring Ninja Andy Wilkinson刚刚宣布了Spring IO Platform BOM，1.1.2和Spring IO Platform 1.0.5的最新版本
  • Matt Stine刚刚宣布提供了两个新指南以及相关的Spring Cloud支持，这些指南用于使用Docker构建Spring应用程序并在Lattice分布式运行时上运行它们 。什么是莱迪思？它是Cloud Foundry的一个很小的子集，可以在本地桌面上运行，与Kubernetes和Mesos + Marathon之类的东西有些可比。
  • 做过微服务炒作了吗？然后，让我们在4月21日开始实践。我将讨论Spring Cloud如何将服务注册（例如Eureka，Consul或Zookeeper），声明式REST客户端（具有Netflix的Feign），Hystrix的反应式编程和断路器模式集成在一起，以支持简单，可靠的服务到终端。服务调用。
  • Marius Bogoevici和Mark Pollack将于4月28日在星期二举行网络研讨会，内容涉及Spring XD和Kafka ，这是所有大数据/快速数据爱好者必不可少的。
  • Spring Framework 负责人Juergen Hoeller在一篇不错的文章中重点介绍了Spring Framework 中对Java 6、7和8的持续支持
  • 然后是Spring Framework ninjaStéphaneNicoll，接着研究了Spring如何在Java 6、7和8上保持不稳定的向后兼容性。
  • Spring Integration忍者Artem Bilan刚刚宣布了Spring AMQP 1.4.4和Spring Integration 3.0.7、4.0.7和4.1.3现已可用 。这些主要是维护版本。
  • Spring忍者Greg Turnquist已使用Spring Security OAuth，Spring Cloud和Angular.js将出色的Dave Syer博士的史诗般的博客系列有关单页应用程序安全性转换为教程 ！欢呼！
  • Spring Security负责人Rob Winch刚刚宣布了Spring Security Kerbreros 1.0.0。RC2
  • 与Spring无关，但我很喜欢这篇文章，《 大规模升级Java 8》
  • 查看Sergei Egorov的这篇文章（该文章是2014年的第一篇文章 ），内容涉及使用Spring Boot，Apache Thrift和Facebook的Swift（为服务器代码删除Swift IDL文件）构建微服务 。您可能还会在几周前重新访问我关于使用Spring Framework 的Google协议缓冲区支持的文章
  • 我们的朋友回来了，这次有一篇关于将配置属性干净地注入Spring bean的帖子。最终，这看起来与Spring Boot的非常相似@ConfigurationProperties机制，因此请检查您是否正在使用Spring Boot！
  • Packt刚刚由Ravi Kant Soni 出版了有关Spring Framework 的新书 。
  • 你知道我今天过得怎样吗？在Spring Initialzr的复选框中可以看到第三方Vaadin对Spring Boot的支持。棒极了！
  • 发行了SpringOne2GX的MOAR Groovy和Grails重播（庆祝Grails 3.0发布），Lari Hotari的Performance Tuning Grails应用程序 ，Jeff Scott Brown的Groovy运行时元编程 ，Bobby Warner的Groovy移动自动化以及Gareme Rocher的Advanced GORM：Beyond Relational 。请享用！
