Spring 的这个星期-2015年4月15日！

工程
乔什·朗（Josh Long）
2015年4月14日

欢迎来到春季本周的另一期！和往常一样，我们有很多讨论，所以让我们开始吧！

上周，我去了法国的Devoxx，在法国巴黎，这周，我在印度参加了“ 伟大的印度开发者峰会”，以及在班加罗尔和海得拉巴举行会议之前的一些会议。如果您位于孟买，海得拉巴或班加罗尔， 并且想谈谈Spring，Cloud Foundry和大数据，请告诉我 ！

一周后，我将在西班牙巴塞罗那进行Spring I / O。如果您在西班牙，请务必不要错过这个惊人的活动！我将与Spring团队的许多其他成员一起在那里，并期待与您见面！

  • 是否想了解有关云，微服务以及构建云原生应用程序的更多信息？请务必查看充满动感的CF Summit时间表 ，并附上Cory Doctorow的主题演讲！
  • Spring Data忍者Janne Valkealahti刚刚宣布发布Spring Statemachine ，这是一个用于构建状态机的通用API。这是从Spring for Hadoop中提取的，并且可以支持许多其他项目，在这些项目中，从一种状态正确迁移到另一种状态尤为重要，例如在分布式异步场景中。
  • 做过微服务炒作了吗？然后，让我们在4月21日开始实践。我将讨论Spring Cloud如何将服务注册（例如Eureka，Consul或Zookeeper），声明式REST客户端（具有Netflix的Feign），Hystrix的反应式编程和断路器模式集成在一起，以支持简单，可靠的服务到终端。服务调用。
  • Marius Bogoevici和Mark Pollack将于4月28日在星期二举行网络研讨会，内容涉及Spring XD和Kafka ，这是所有大数据/快速数据爱好者必不可少的。
  • 上周，我很高兴再次在Virtual JUG上发表演讲。这是我第二次在那发言。去年，我做了我的演讲，“ 使用Spring Boot构建Bootiful应用程序”，并且玩得很开心。今年，差不多一年后，我很高兴被邀请再次发表演讲，这次是关于使用Spring Cloud构建Bootiful微服务的 。在ZeroTurnaround赞助vJUG的优秀人才甚至对我进行了后续采访并发表了文章 。能够出席是我的荣幸和荣幸，非常感谢您让我vJUG！
  • 有一些很棒的与Groovy和Grails相关的SpringOne2GX重放！查看Lori Hotari的演讲， 性能调整Grails应用程序 ，深入了解Grails的性能瓶颈。查看Grails忍者Jeff Scott Brown的演讲，“ 使用Groovy运行时元编程” ，以了解Groovy如何通过元编程等高级功能实现比Java更多的动态API。Bobby Warner的演讲Groovy Mobile Automation将使用Groovy，Spock和Gradle进入iOS和Android功能测试领域。想更多地了解Grails的GORM？查看Grails项目负责人Graeme Rocher的GORM深入研究，并超越关系数据 。
  • 今年早些时候，我们在Pivotal宣布将开源整个大数据堆栈。我们正在向前迈进。该承诺的一部分是将Gemfire捐赠给Apache基金会，该基金会已被更名为Project Geode 。我们已经有了项目页面和源代码下载，因此请查看！提案已提交给Apache Software Foundation，以孵化Project Geode。在ASF同意孵化Project Geode之前，可通过仅评估版的EULA获得源代码。
  • Sergei Egorov的帖子， Spring Boot的胖子.jars与Docker ，充满了胜利！本文介绍了如何缓存不同微服务共有的共享工件，以便Docker映像可以从共享的基础Docker映像中受益。不错的帖子，绝对值得一读！
  • 我们的朋友Matt Raible在JHipster上写了一篇关于他最近在Denver JUG上的演讲的不错的帖子 ，这是一个Yeoman代码生成器，它生成基于Spring Boot的后端和基于JavaScript的前端。幻灯片，他的着装以及其他所有内容都使该文章成为必读内容！ ：D
  • 说到JHipster，请查看Driss Amri的这篇文章，该文章探讨了如何使用JHipster启动现代Java项目 。这篇文章着眼于生成Angular.js和基于Spring Boot的应用程序，然后利用一些Boot支持的集成来度量，安全和部署。真好！
  • 我不确定是否会推荐这样做，但这是一个有趣的读物： 如何在同一个JVM中运行多个Spring Boot应用程序 。
  • 这篇帖子-Java EE的JPA生产力提升器使我咯咯笑，但真是太棒了。这是一起来看看如何使股票的Java EE数据访问（但没有使用Spring，这似乎错过了这一点！）。第一部分重点介绍了Spring Data CDI集成。
