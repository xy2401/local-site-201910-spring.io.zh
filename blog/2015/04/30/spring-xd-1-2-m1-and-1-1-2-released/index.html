    Spring XD 1.2 M1和1.1.2发布

    发布
    马克·波拉克
    2015年4月30日

    我非常荣幸地代表Spring XD团队宣布Spring XD 1.2的第一个里程碑版本和1.1.2维护版本。

    下载链接：

    • 1.1.2。发布： zip
    • 1.2.0。M1： 拉链

    您也可以使用brew和rpm安装

    JIRA中提供了针对1.1.2修复的问题的完整列表。值得注意的是1.1.2版本提供了PHD 3.0支持。

    1.2 M1版本还包括错误修复以及几个新功能和增强功能：

    • PHD 3.0支持
    • MongoDB Source ，来自Abhinav Gandhi的社区贡献
    • HDFS支持的模块注册表
    • Greenplum gpload作为提供的批处理作业。这样可以将CSV文件有效地加载到Greenplum DB / HAWQ中。
    • 遵循gpfdist协议的gpfdist 接收器 。这允许将数据并行流传输到Greenplum DB / HAWQ中。
    • Zookeeper基于分布式队列的流和作业部署。
    • 使用Dead Letter Queue 改进了RabbitMQ的错误处理，并为发布/订阅命名通道提供了持久的队列支持（tap：和topic :)
    • Sqoop集成改进 ，对合并和代码生成命令的支持以及在受保护的Hadoop群集上运行。
    • Kafka消息总线的改进，针对消息总线创建的主题的自定义分区计数。 （模块。[模块名称] .producer.minParitionCount）
    • TupleBuilder和JDBC to HDFS作业的改进的性能特征
    • Spark Streaming集成改进，可靠的接收器支持和错误修复。

    JIRA中提供了针对1.2 M1修复的问题的完整列表。

    对于Spring XD 1.2 RC1的下一发行版，我们将包括对以下功能的支持：

    • Kafka消息总线改进
    • 卡桑德拉水槽
    • 弹性搜寻器
    • 改进了对RabbitMQ的HA支持
    • 从JDBC到HDFS作业的增量数据导入

    Spring Flo for XD的Beta版（一种用于创建数据管道，分析仪表板和监视的基于HTML5的画布）将与XD 1.2 GA一起发布。欲了解正在进行中的工作，请观看以下截屏视频：
