春季测试HtmlUnit 1.0.0 RC1发布

发布
罗伯·温奇
2015年5月14日

我很高兴地宣布发布Spring Test HtmlUnit 1.0.0 RC1。该版本的要点可以在下面找到：

迁移指南

从1.0.0开始的详细迁移指南。M2到1.0.0。RC1已添加。

新的Maven座标

我们已经更改了Maven组ID。有关详细信息，请参阅更新依赖项

默认上下文路径为“”

根据社区反馈，默认上下文路径现在为“”。这意味着，如果您使用的是先前版本，则需要更新配置的URL。

有关详细信息，请参阅默认上下文路径现在为“”

MockMvc WebClient和WebDriver构建器

此版本增加了MockMvc和HtmlUnit集成的构建器。建设者提供了许多好处：

  • 简化设置
  • 允许更多配置选项
  • 默认情况下，任何具有localhost主机的URL都由MockMvc（即http：// localhost / something ）处理，而其他任何主机都可以正常检索（即http://cdn.example.com/jquery.js） 。

有关详细信息，请参阅MockMvc WebClient和WebDriver构建器

新项目现场和文档

我们添加了一个新的Project Site 。我们还发布了测试所有代码的文档 。

转发的网址现已处理

转发的URL现在已正确处理。
