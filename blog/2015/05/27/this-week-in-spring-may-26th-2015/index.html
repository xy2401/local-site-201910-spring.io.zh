春天的这个星期-2015年5月26日

工程
乔什·朗（Josh Long）
2015年5月27日

欢迎来到春季本周的另一期！有很多要讨论的，所以让我们开始吧！

  • 首先，大新闻！Spring框架负责人和OG Spring Guy Juergen Hoeller刚刚宣布Spring框架4.2 RC1现已可用！这个新版本包含许多惊人的新功能，因此请务必检查一下！它包括对Java 8默认方法的注释检测，基于注释的应用程序事件，一流的注释属性别名，用于直接字段绑定的嵌套路径处理，针对JSR 354（货币和货币）的数据绑定和转换，Hibernate ORM 5.0支持（通过JPA和本机），一个STOMP客户端，可用于TCP和WebSocket通道， Listenable / CompletableFuture作为消息处理程序的返回值，CORS和声明性HTTP缓存支持等等。试一试，让我们确保这是迄今为止最好的版本！
  • Spring Cloud Connectors负责人Scott Frederick刚刚宣布Spring Cloud Connectors 1.2.0 RC1已发布。新版本包括许多向后兼容的更改和升级，其中包括对HikariCP作为连接池的支持，以及消除了令人恐惧的修补程序。 ClassNotFoundException在某些罕见的情况下！此升级绝对值得升级！
  • Spring Data忍者和Apache Hadoop的Spring领导者Thomas Risberg刚刚宣布了2.2之前为Apache Hadoop里程碑发布的最新计划Spring。GA大约2周。此版本包括Cloudera CDH支持，Hortonworks HDP 2.2和2.2.4.0支持，更新的Kite SDK，对HDFS writer的改进以支持可同步写入和新的超时选项，并且-有用的-此版本需要Java 7或更高版本。
  • 上周起，请查看Marius Bogoevici和Mark Pollack博士关于Spring XD和Kafka的反应式管道的惊人网络研讨会！即使您看到了它，也要再次签出。这是个好东西。
  • 您的谦虚作者被授予Java Champion的头衔。该头衔授予那些帮助推进Java平台的人，其他Java拥护者也提名了该奖项。这是一个稀有的团体，我不知道我是否值得。我之所以仅提及它，是因为我与您共享的友情世界-世界上最令人惊叹的社区-使其继续前进变得有趣而激动。谢谢我的朋友们。春季的本周是该社区为教我和激励他人所做的所有奇妙事情的近5年纪事。谢谢！
  • ChargeAhead提供了一个有趣的两部分视频，其中包含构建基于Spring Boot的Spring MVC应用程序的视频。这是第1部分 ，这是第2部分 。

  • Jose Diaz整理了有关如何使用Angular.js，Twitter Bootstrap，Spring MVC，Spring Data和Spring Security的视频
