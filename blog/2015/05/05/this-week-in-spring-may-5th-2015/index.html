Spring 的这个星期-2015年5月5日

工程
乔什·朗（Josh Long）
2015年5月5日

欢迎来到 Spring 本周的另一期！本周，我在拉斯维加斯参加大型EMC World活动。

  • Spring Security负责人Rob Winch刚刚宣布Spring Security SAML 1.0.1现已可用
  • Phillip Verheyden的演讲非常实用地介绍了如何移动Spring 3应用程序 （包括诸如Jackson和Hibernate之类的依赖项）以利用Spring 4.x中的新功能。
  • 我的朋友Adib Saikali在SpringOne2GX 2014上的关于使用Spring构建富客户后端的演讲现在已经结束
  • 不要错过Spring Ninja Greg Turnquist 关于调试和维护Spring Boot应用程序的演讲的重播
  • 达斯汀·惠特尔（Dustin Whittle） 关于如何进行性能测试（Spring）应用程序的演讲中包含了有用的工具，使用《攻城》进行性能测试的应用程序的提示和技巧，带机枪的Bees，Google PageSpeed，WBench等。
  • 不要错过我们的朋友Marten Deinum在5月12日举行的关于使旧版 Spring应用程序现代化的网络研讨会。我等不及要听这个了！ Spring 定期阅读本周的读者知道他几乎总是在我们的综述中露面！
  • Spring Data忍者Thomas Risberg刚刚宣布了适用于Apache Hadoop 2.2 M1的Spring 。新版本包括更新的受支持的依赖关系，对Hiveserver2的更好支持（包括批处理任务集），对用于运行Spark应用程序的批处理任务集的支持，Spring Boot集成，更智能的安全性集成以及与Spring Cloud组件的集成等等！
  • 优秀的Pollack博士刚刚发布了Spring XD 1.2 M1 （包括令人惊讶的Spring Flo可视编辑器的预览版，改进的Cassandra和Kafka集成以及更多内容）和1.1.2，其中包含了错误修正。我等不及要看Spring XD 1.2中的某些内容。M1。
  • Spring Security主管Rob Winch刚刚宣布了Spring LDAP 2.0.3。RELEASE， 主要是一个错误修正版本 。
  • 查看Unicon最近为Spring Boot开发的CAS客户端自动配置 。CAS是一种用于Web的单点登录。 https://github.com/Unicon/cas-client-autoconfig-support
  • 查阅Spring I / O的SébastienDeleuze的演讲，以了解更多有关Spring Framework 4.2即将使用React.js和Handlebars 支持同构 （在服务器和客户端上编写的）视图的信息。
  • 查阅这篇文章，了解如何使用Elastic Beanstalk将Spring Boot应用程序部署到AWS]（ http://mtdevuk.com/2015/02/10/how-to-deploy-a-spring-boot-application-to-亚马逊使用弹性 bean 茎/
  • Java Code Geek发表了一篇不错的文章，汇总了GitHub上约60,000多个库中由导入表示的顶级Java 库 。这些数字非常清楚：Spring库是最常使用的功能，它们合在一起使用，特别是Spring Test，紧接在JUnit之后（比Mockito快！）。就受欢迎程度而言。我也喜欢代表的许多其他非Spring程序库中经常包含Spring Boot启动程序。方便！
  • 如果您是Vaadin开发人员，请查看即将在5月7日举行的有关同时使用Spring和Vaadin的网络研讨会 ，您将想了解Spring Boot！
