This Week in Spring - May 19th, 2015

Engineering
Josh Long
May 19, 2015

Welcome to another installment of This Week in Spring! As usual, we’ve got a
lot to cover this week so let’s get to it!

  • the good Dr. Dave Syer is back it again with this seventh installment on how to better modularize the Angular.js client that talks to a backend Spring Security, Spring Boot, and Spring Security OAuth-powered service and with this eighth installment which looks at how to test that Angular.js application!
  • Spring Security lead and all around amazing, test-driven dude, Rob Winch just announced Spring Test HtmlUnit 1.0.0.RC1 with a lot of new polish.
  • Spring Batch lead Michael Minella just announced that Spring Batch 3.0.4, a maintenance release, is now available
  • Spring Social lead Craig Walls just announced that two maintenance releases, Spring Social 1.1.2 and Spring Social 1.1.1, have been released
  • On May 28th, join Pivotal’s own Greg Turnquist: How to Debug and Maintain your Spring Boot App
  • Turkcell’s Erdem Gunay, and our pal, is giving a webinar on June 9th explaining how Spring is pretty comprehensively used in Turkcell’s personal cloud service.
  • longtime Spring community member David Welch had a question about setting up a highly available Netflix Eureka installation and Spring Cloud co-founder Spencer Gibb helped him along on this super useful StackOverflow.com post.
  • Fancy using JUnit @Rule instances instead of being forced to specify Spring’s JUnit TestRunner (which effectively removes the ability to use other sometimes required TestRunner implementations)? Check out the upcoming Spring framework 4.2 support already in master!
  • on May 27th the Seattle Continuous Delivery Meetup is having a Spring Boot talk! Naturally. Check it out. Spring Boot fits well in a continuous delivery pipeline: it’s easy to build isolated, testable, and production-ready applications.
  • Yet another (well done, by the way!) post on how to setup Spring Boot applications and containerize them with Docker. Seems this is a popular topic! Good thing we have a guide!
  • Spring and Groovy Tool Suite ninja Kris De Volder announced the STS support for YAML last week - have you tried it yet?
  • the IS Applications Development Services blog has a nice post on documenting Spring Boot applications with Swagger. I like Swagger, but I’d recommend also trying out Spring REST Docs which are less invasive - they live in the unit-tests - and they’re Asciidoctor-friendly.
  • Wow, were are finally out of SpringOne2GX replays! This is the last batch, mostly Groovy and Grails. First, Baruch Sadogursky from JFrog talks about Grails Plugin Testing Strategies. Then David Dawson, from Simplicity Itself talks about Philosophical Architecture in Grails. Also, Ken Kousen Groovy Vampires: Combining Groovy, REST, NoSQL, and more
