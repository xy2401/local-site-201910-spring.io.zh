春天的这个星期-2015年5月19日

工程
乔什·朗（Josh Long）
2015年5月19日

欢迎来到春季本周的另一期！像往常一样，我们有一个
这周有很多事情要做，让我们开始吧！

  • 好的Dave Syer博士在第七部分中再次对它进行了支持，该部分是关于如何更好地模块化与后端由Spring Security，Spring Boot和Spring Security OAuth提供支持的服务进行对话的Angular.js客户端，以及在第八部分中，如何测试Angular.js应用程序 ！
  • Spring Security主管以及令人惊叹的测试驱动家伙，Rob Winch刚刚宣布了Spring Test HtmlUnit 1.0.0。RC1带有很多新的抛光剂。
  • Spring Batch负责人Michael Minella 刚刚宣布维护版本Spring Batch 3.0.4现已发布
  • Spring Social主管Craig Walls刚刚宣布已经发布了两个维护版本Spring Social 1.1.2和Spring Social 1.1.1 。
  • 5月28日，加入Pivotal自己的Greg Turnquist： 如何调试和维护Spring Boot应用程序
  • Turkcell的Erdem Gunay和我们的朋友将于6月9日举行网络研讨会，解释如何在Turkcell的个人云服务中非常全面地使用Spring 。
  • 长期的Spring社区成员David Welch对于建立一个高可用的Netflix Eureka安装存在疑问 ，Spring Cloud联合创始人Spencer Gibb帮助他完成了这个超级有用的StackOverflow.com帖子。
  • 看中使用JUnit @Rule实例，而不是被迫指定Spring的JUnit TestRunner （这有效地消除了使用其他有时需要的功能TestRunner实施）？掌握即将发布的Spring Framework 4.2支持 ！
  • 5月27日，西雅图持续交付聚会将举行Spring Boot演讲 ！自然。看看这个。Spring Boot非常适合连续交付管道：轻松构建隔离的，可测试的且可用于生产的应用程序。
  • 还有一个（顺便说一句，做得好！）关于如何设置Spring Boot应用程序以及如何使用Docker对其进行容器化的文章 。似乎这是一个受欢迎的话题！好东西，我们有指导 ！
  • Spring and Groovy Tool Suite忍者Kris De Volder上周宣布了对YAML的STS支持 -您是否尝试过？
  • IS Applications Development Services博客上有一篇不错的文章， 介绍了如何使用Swagger记录Spring Boot应用程序 。我喜欢Swagger，但我还建议您尝试使用Spring REST Doc ，它们的入侵性较小-它们生活在单元测试中- 并且它们对Asciidoctor友好。
  • 哇，SpringOne2GX的回放终于结束了！这是最后一批，主要是Groovy和Grails。首先，来自JFrog的Baruch Sadogursky谈论了Grails插件测试策略 。然后，来自《简单性》本身的David Dawson谈论了Grails中的哲学体系结构 。另外，Ken Kousen Groovy吸血鬼：结合Groovy，REST，NoSQL等
由Disqus提供动力的评论
