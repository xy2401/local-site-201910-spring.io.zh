春天的这个星期-2015年12月15日

工程
乔什·朗（Josh Long）
2015年12月16日

哇！朋友，这是12月15日；我们这个星球上的许多人将很快庆祝新的一年！希望您做得比我更好，要在2016年及时完成所有新年决议！如果您还没有，那么赶上Pivotal社区中的最新和最伟大的事物将是不可逾越的！让我们看看这周发生了什么。

  • 格雷格·特恩奎斯特（Greg Turnquist）关于将Spring与React.js结合使用的史诗博客系列现在可以作为教程在线获得
  • Spring Batch主管Michael Minella刚刚宣布了Spring Batch 3.0.6，其中包含了许多出色的贡献和社区中的错误修复
  • 良好的大卫Syer博士组建了一个非常 好的看看在各种配置Spring Boot的记忆特性
  • Spring RestDocs 负责人Andy Wilkinson刚刚宣布了Spring Rest Docs 1.0.1 ，它是维护版本，其中包括许多更改，包括对XML负载中的文档属性的改进支持。立即
  • Spring Social的负责人Craig Walls 一直在努力工作，以使Spring Social适应 Facebook的Graph API 的最新突破，该更改阻止了连接和登录与Spring Social Facebook一起使用。此外，API绑定已针对目标Graph API v2.5进行了调整。
  • Duncan Brown对如何将Spring Data Neo4J 3.0项目迁移到新的4.0系列进行了很好的介绍
  • 我认为这篇中文帖子介绍了如何使用Spring Boot ，在这种情况下-尽情享受吧！
  • Couchbase的SimonBaslé撰写了一篇很好的文章，介绍了如何使用Spring Data Couchbase及其支持的各种功能， 包括Cache抽象和Spring数据存储库
  • AppMite博客简单介绍了如何启动一个简单的Spring Boot应用程序，以用作基于Backbone + Marionette的JavaScript客户端的后端
  • MarkoŠvaljek撰写了一篇非常酷的文章，内容涉及使用Spring Boot，Kafka等处理流处理
  • DataStax的Apache Cassandra传播者Duy Hai Doan在基于Spring Boot，Cassandra等的示例应用程序KillrChat上撰写了一篇非常酷的文章
  • 想快速入门使用Spring Boot和AOP吗？
  • Christoph Strobl在SpringOne2GX 2015上的演讲“ 使用Spring引导您的搜索”现已在线上提供。
  • 了解有关Spring Boot 1.3对彩色横幅的新支持
  • 最近，社区的一位朋友Craig Burke在Spring Boot 1.3首次推出了上述对彩色横幅ASCII艺术的支持之后，提出了史诗般的拉动请求 ： Banner 也知道要寻找的实现banner.jpg ， 不只是banner.txt ！抓住这一点，我创建了一个简单的基于Boot的CLI和一个Cloud Foundry Web服务，其代码可在线使用 -感谢Craig！
