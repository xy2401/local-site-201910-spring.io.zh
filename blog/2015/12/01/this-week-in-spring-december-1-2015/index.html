春天的这个星期-2015年12月1日

工程
乔什·朗（Josh Long）
2015年12月1日

哇！您可以相信我们已经盯着2016年了吗？时间过得真快！请记住， 春季的本周始于 2011年的第一周，因此我们将很快纪念TWiS成立五周年！

本周，我将在中国上海参加亚洲Cloud Foundry峰会 ，然后前往Ft。美国佛罗里达州劳德代尔， 丰富的网络体验 ！如果您在任何一个地方， 请随时与我联系 ！

和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Spring Boot的共同创始人和Spring框架贡献者Phil Webb发表了一篇非常体贴的文章，帮助人们着眼于Spring（已有10多年的历史了），以避免2016年新开发人员面临的一些常见陷阱 。非常推荐阅读！
  • Florian Benz在讨论Spring Rest Docs生成的全面REST 文档的动机和使用方面做得非常出色。这是一个非常酷的快速阅读！
  • 在谈到Spring Rest Docs时，对项目负责人进行了一次精彩的采访，他是一位了不起的Andy Wilkinson，他在充足的空闲时间里还管理Spring IO Platform，并为Spring Boot做出了贡献。继续阅读以了解有关我的一位英雄的更多信息 ！
  • Spring Boot和Spring Cloud联合创始人Dave Syer博士发表了[关于将Spring Cloud应用程序从Spring Boot 1.2迁移到刚刚发布的1.3的不错的文章。
  • Spring Social主管Craig Walls刚刚宣布了Spring Social 1.1.4，其中包括一些建议的错误修复
  • Spring Data忍者Thomas Risberg 刚刚宣布了Spring Data JDBC Extensions 1.2和许多其他重大更新
  • Spring Tool Suite的负责人Martin Lippert 刚刚宣布了STS的STS 3.7.2版本，其中包括更新的Spring Boot仪表板，Spring Boot 1.3.0兼容性，快速文本搜索等。
  • Spring Boot，Spring Cloud和Spring Security OAuth贡献者，共同创始人兼忍者Dr. Dave Syer在将基于OAuth的代码从Spring Cloud Angel迁移到Brixton的帖子中介绍了Spring Boot 1.3中的某些功能，这是非常有用的很酷的阅读！
  • 说到Spring Cloud Security，请查看我的朋友Will Tran在SpringOne2GX 2015中的史诗般的演讲 ！
  • 查看Spring Cloud联合创始人Spencer Gibb 关于Spring Cloud通用扩展挂钩的替代实现的史诗般的演讲，来自SpringOne2GX
