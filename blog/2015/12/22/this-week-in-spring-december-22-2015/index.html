春天的这个星期-2015年12月22日

工程
乔什·朗（Josh Long）
2015年12月22日

欢迎来到春季本周的另一期！本周，我们许多人将休假圣诞节。如果您庆祝的话，那么我希望您能从我们的团队中获得“圣诞节最快乐”奖。如果您不庆祝的话，我们将在下周的春季“本年”与您见面！ 并欢迎2016年！

是否度假，这周有很多很棒的东西可供阅读，让我们开始吧！

  1. 如果您使用的是Spring Cloud Eureka和Jersey，则可能会遇到问题，因为Eureka本身使用Jersey 1.x，而Spring Boot的Jersey支持基于Jersey 1。阅读Aleksandar Stoisavljevic如何解决它
  • Spring Framework 负责人Juergen Hoeller宣布Spring Framework 4.2.4和4.1.9已发布
  • Spring Framework 贡献者StéphaneNicoll 刚刚宣布Spring Framework 3.2.16现已可用
  • Spring Boot的贡献者StéphaneNicoll 还宣布了Spring Boot 1.3.1和1.2.8也已经可用。
  • Spring Data负责人Oliver Gierke刚刚宣布，Spring Data relese火车Gosling SR2也已上市
  • Spring IO Platform负责人Andy Wilkinson刚刚宣布1.1.5和2.0.1现已发布
  • 您要在系统中构建多租户吗？考虑使用具有非常强大的Spring Boot自动配置功能的Stormpath Identity提供程序
  • 我们的朋友Biju Kunjummen又回来了，这一次是关于使用Spring Cloud的自动配置RestTemplate带
  • 本周，Pivotal宣布收购长期的Cloud Foundry合作伙伴Cloud Credo ：欢迎加入！
  • NedimŠabić的这篇文章很好地介绍了使用Spring Cloud和Go实现微服务的各种方法
  • Voxxed在Holger Steinhauer的一篇有趣的文章中介绍了一个简单的Spring Boot应用程序 ，该应用程序利用Spring MVC，Spring Data，JPA，使用Spring Boot执行器对其进行操作，添加模板等等。
