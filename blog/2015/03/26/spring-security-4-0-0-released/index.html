Spring Security 4.0.0发布

发布
罗伯·温奇
2015年3月26日

我很高兴宣布Spring Security 4.0.0的发布。发行将关闭超过175张门票 。您可以在下面找到要点：

WebSocket支持

Spring Security 4添加了WebSocket支持 。现在可以将Spring Security与Spring的WebSocket支持一起使用。

Spring数据集成

Spring Security 4添加了Spring数据集成 。现在可以使用SpEL在Spring Data查询中引用Spring Security的用户。

测试支持

Spring Security 4添加了测试支持 。现在，使用Spring Security应用程序编写测试要容易得多。

默认情况下更安全

随着针对应用程序的攻击不断发展，Spring Security也必须发展。作为主要发行版本，Spring Security团队借此机会进行了一些非被动更改，这些更改主要集中在：

  • 默认情况下，确保Spring Security更安全
  • 最小化信息泄漏
  • 删除不推荐使用的API

有关从Spring Security 3迁移到Spring Security 4的完整详细信息，请参阅参考中从3.x迁移到4.x的部分。
