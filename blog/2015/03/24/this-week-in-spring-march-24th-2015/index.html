春天的这个星期-2015年3月24日

工程
乔什·朗（Josh Long）
2015年3月24日

欢迎来到春季本周的另一期！本周，我在巴西圣保罗，在那里我举办了一个研讨会，该研讨会通过Spring Boot ， Spring Cloud和Cloud Foundry （通过Pivotal Web Services ）构建基于云的应用程序，特别是微服务 。

  • Spring Data的负责人Oliver Gierke刚刚宣布Spring Data Fowler已经退出了GA，并包含许多高级Java 8功能 ！
  • 西班牙巴塞罗那春季IO展示的议程现已发布 ！看看这个！
  • 做过微服务炒作了吗？然后，让我们在4月21日开始实践。我将讨论Spring Cloud如何将服务注册（例如Eureka，Consul或Zookeeper），声明式REST客户端（具有Netflix的Feign），Hystrix的反应式编程和断路器模式集成在一起，以支持简单，可靠的服务到终端。服务调用。
  • Marius Bogoevici和Mark Pollack将于4月28日在星期二举行网络研讨会，内容涉及Spring XD和Kafka ，这是所有大数据/快速数据爱好者必不可少的。
  • Colin Harrington的SpringOne2GX 2014 Replay现在可用： Groovy浏览器自动化
  • 彼得·莱德布鲁克（Peter Ledbrook）的SpringOne2GX 2014 Replay现已发布： 面向Java开发人员的Groovy
  • Pivotal Ninjas的梦想团队Derek Beauregard，Phil Berman，Michael Minella，Darrel Sharpe的梦想团队为您带来了SpringOne2GX 2014最酷的演示
    现在可用： 物联网已实现-互联汽车向您展示了Spring XD对于大数据和物联网的真正强大功能。
  • 对Java序列化性能感到沮丧吗？您是Spring Integration性能狂热者吗？还是您只是想将Java网络调用的时间缩短几毫秒和几毫秒？您不会想错过C24的John Davies，因为他向您展示了如何通过SpringOne2GX 2014重播： 内存中的大数据来利用Java序列化来恢复性能。
  • 优秀的Syer博士再次发表了他的史诗博客系列的后续文章，介绍了使用Spring Security OAuth，Spring Cloud Zuul等的单页应用程序。
  • 我汇总了使用Google协议缓冲区来构建有效的基于Spring的REST服务的方法
  • Spring Tool Suite的忍者Kris De Volder撰写了一篇很棒的教程，介绍了支持构建Spring Boot应用程序的STS 3.6.4中的一些新功能。看看这个！
  • 我将再进行一次vJUG会话，这次是使用Spring Cloud构建“ Bootiful”微服务 -希望您在那里加入我！让我们将其变成一场超级热闹的活动！
  • 这篇关于使用Spring Social和Spring Security来保护GWT应用程序安全的文章很有意思
  • 嘿，您是否想念Adrian Cockroft在O'Reilly的软件架构峰会上的演讲？查看此博客，总结关键要点
  • 我喜欢这篇博客文章中关于微服务可能的变体的想法
  • 还有一个将Spring Security与Spring Boot结合使用的超简单示例
