    春天的这个星期-2015年3月3日

    工程
    乔什·朗（Josh Long）
    2015年3月3日

    欢迎来到春季本周的另一期！本周，我在堪萨斯州堪萨斯城，与我的朋友和Spring Security负责人Rob Winch闲逛。本周，我将在本地用户组和Spring用户组中发言，并且我将与Cerner和其他一些大型企业讨论Spring，Spring Boot和Spring Cloud。

    和往常一样，我们有很多很棒的东西要谈论，让我们开始吧！

    1. Spring Boot的共同负责人Phil Webb 刚刚宣布了Spring Boot 1.2.2和Spring Boot 1.1.11 。这两个版本都是推荐的升级，并且Spring Boot 1.2.2甚至包括对Mustache模板库的新支持。真好！
    2. 查看John Hann的SpringOne2GX 2014演讲，介绍Rave.js，它将Spring Boot概念引入JavaScript。
    3. 我汇总了一个博客，内容涉及构建应用程序的过程，这些应用程序可以轻松地从应用程序服务器，Web服务器和云中以可扩展的方式和可移植的方式使用Servlet HTTP会话 。
    4. 克里斯·比姆斯（Chris Beams）在SpringOne2GX 2014演讲中， “革命不会被集中化”现在可以在线获得。
    5. 想更多地了解Spring的缓存？查看Michael Plod在SpringOne2GX 2014演讲中的重播内容，该演讲介绍了Spring框架中的高级缓存改进
    6. Brian Clozel和Rossen Stoyanchev的SpringOne2GX 2014演讲在Spring MVC 4.1中引入了资源处理， 真棒 ！
    7. Spring Security负责人Rob Winch 刚刚宣布了Spring Security 4.0 RC2 ，其中包括对Websocket的支持！检查一下，然后尝试一下！
    8. Rob Winch还刚刚宣布了Spring Security 3.2.6，它改进了Java configuratino和Spring Boot的集成 ，因此请查看！
    9. 这本身并不是特定于Spring或任何技术的，但它启发了人们对机器学习如何用于预测对抗癌药物的临床反应的启发
    10. 谈到Rob Winch和教育界的忍者Ken Kreuger，请查看他们在SpringOne2GX 2014上有关集成测试的演讲，该演讲现已在InfoQ上提供 。
    11. Alex Zhitnitsky将Spring Boot和Dropwizard进行了很好的比较
    12. 查看这些基于Cloud Foundry的IBM Bluemix的Spring Cloud PaaS连接器
    13. Rahman Usta组合了一个基于Web的SSH客户端的惊人酷演示（后端带有Spring Boot）
    14. 该视频演示了如何使用Codeenvy和Spring Boot
