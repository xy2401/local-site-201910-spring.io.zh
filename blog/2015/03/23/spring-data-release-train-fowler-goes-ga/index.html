Spring Data 发布培训Fowler参加GA

发布
奥利弗·德罗博姆
2015年3月23日

我谨代表Spring Data团队宣布Spring Data发布火车Fowler的GA版本的可用性。该版本发行了将近400张票。

培训最核心的主题是对Java 8的高级支持：MongoDB和JPA模块现在支持Stream用作存储库方法的返回类型，现在我们支持包装盒中的非时区JDK 8日期/时间类型，并且还与该API的Java 7反向端口集成。

Spring Data MongoDB附带了服务器3.0版本和相应驱动程序版本的兼容性。Spring Data Gemfire现在支持Gemfire 8.0。对象到存储的映射已经实现了一些广泛的性能改进，因此，例如MongoDB的每秒操作数应该会大幅增加。

可以在此处找到预定的更改日志，完整的固定票证列表可在JIRA中找到 。我们目前正在准备一个博客文章，以更详细地介绍这些新功能。下一个发行火车将以Java的父亲James Gosling的名字命名。
