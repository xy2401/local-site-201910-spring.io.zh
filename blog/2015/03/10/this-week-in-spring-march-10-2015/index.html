春天的这个星期-2015年3月10日

工程
乔什·朗（Josh Long）
2015年3月10日

欢迎来到春季本周的另一期！ 本周， 我和乔治亚州少数几个Spring团队成员以及Spring社区知名人士一起在佐治亚州亚特兰大的DevNexus工作 。我将发表有关Spring Boot入门的演讲，希望您也能加入我的行列，或者是Spring团队的众多其他演讲之一：

  • Rossen Stoyanchev撰写的Spring 4 Web Applications
  • Spencer Gibb的Spring Cloud提供的Cloud Native Apps
  • 使用Scala，功能域模型和Spring Boot构建微服务 Chris Richardson
  • Spring Integration 4有什么新功能？ 由Marius Bogoevici
  • Spring Data REST- Greg Turnquist和Roy Clarkson的数据与超媒体交汇
  • 物联网实现 -Michael Minella 的互联汽车
  • 保持清洁 -Phillip Webb 从开源中学到的有关如何完善代码的经验教训
  • Spring Integration Java DSL-深入探究 Gary Russell

我们还将参加春季用户组会议 。出来看看Rossen Stoyanchev谈论Spring MVC的新功能，看看Michael Minella谈论Spring Batch的新功能。

不管怎样，让我们开始吧！

  1. 令人惊叹的Syer博士刚刚宣布了Spring Cloud 1.0.0。发布 ！这是一个漫长而疯狂的开发周期的高潮，其中包括很多很棒的东西。Spring Cloud是作为一组Spring Boot样式的自动配置和对基础架构支持微服务的启动程序依赖项提供的。
  2. Spring Data忍者Christoph Strobl刚刚宣布了Spring Data Fowler发布火车的RC1 。这个新版本增加了许多新功能，包括对@Score Solr中的-annotation，更新的MongoDB 3支持（包括对两种类型的MongoDB GIS索引的支持）等。
  3. 剩下的三月网络研讨会将继续Javascript月。Scott Deeg将于3月24日检查Polymer，Spring Security和Spring Cloud的轻量级API网关。最后，Julien Dubois将于2015年3月17日推出AngularJS + Boot应用程序生成器JHipster。
  4. 本周的SpringOne2GX 2014回放已经结束！查看Gunnar关于使用Spring和AngularJS创建模块化测试驱动的SPA的精彩演讲。 Rob Winch和Ken Krueger在“寻求圣洁集成测试”中介绍了Spring MVC测试-这是Spring MVC的鲜为人知且非常有用的部分。 Scott Frederick和Ramnivas Laddad在他们的演讲“ 使用Spring Cloud Connectors简化云应用程序”中介绍了原始的Spring Cloud（现在称为Spring Cloud Connector），以及RedHat / OpenShift的一篇有关使用PaaS将您的应用程序导入云的内容 。
  5. EMC Academics刚刚发布了他们的第一本Flipboard Spring杂志 -请查看！
  6. JAXenter在最近发布的Spring Cloud 1.0.0上有不错的文章 。
  7. Parleys.com 上有来自Devoxx UK 2014在Spring Boot上的我的演讲（获得极高评价，因为我认为是这个话题。）！看看这个！
  8. 我很高兴为O'Reilly举办了两次网络研讨会，我刚刚学到的录音现在都在线上，其中一个是使用Spring Boot构建应用程序以及使用Spring Cloud构建微服务 。O'Reilly正在出版有关基于Spring和Cloud Foundry的微服务的书。
  9. Activiti BPMN工作流引擎的共同创始人，我们的好朋友和社区成员Joram Barrez共同撰写了一篇不错的文章，介绍了Spring Boot和Activiti的集成 。
  10. 在最近的Voxxed Days Vienna活动中， InfoQ采访了Spring框架负责人Juergen Hoeller的各种各样的事情 。
