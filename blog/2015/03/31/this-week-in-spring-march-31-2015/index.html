    春天的这个星期-2015年3月31日

    工程
    乔什·朗（Josh Long）
    2015年3月31日

    欢迎来到春季本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！

    • Spring Security主管Rob Winch 刚刚宣布了惊人的Spring Security 4.0版本 ！这其中包含了令人惊奇的功能，包括websocket支持，Spring Data集成，增强的测试支持以及大大改善的默认设置
    • Cloud Foundry Java buildpack现在通过使用命名环境变量来支持覆盖内置配置！这意味着您可以调整所有默认行为，而无需派生构建包！
    • 好消息！Spring XD 1.1.1发布了！它包括使用Kafka消息总线时的改进的性能和偏移管理，对Spark的流传输可靠接收器的支持，对Kryo序列化的改进，RabbitMQ自动清除选项以及Sqoop示例。
    • Graeme Rocher惊人的Groovy Web App框架Grails 3.0已发布 -现在基于Spring Boot！
    • Greg Turnquist刚刚发布了带有错误修复的Spring WS 2.2.1 。趁热拿到！
    • 哦，是的，还发布了具有大量错误修正的Spring Security 3.2.7 ，但是…下载4（见上文）！
    • 说到所有数据， Spring Integration Kafka 1.1刚刚发布！Spring XD 1.1.1中也包含了此功能，请检查一下！
    • 刚刚发布了带有补丁和错误修正的Spring框架4.1.6。对于Spring 4.x用户，这是一个推荐的更新 。
    • 我将通过Spring Cloud举办有关更智能的服务到服务调用的网络研讨会
    • 这次我将再次出现在虚拟JUG（vJUG）上， 以谈论基于Spring Cloud的微服务 。加入我们，全球各地！
    • Marius Bogoevici和Mark Pollack将于4月28日在星期二举行网络研讨会，内容涉及Spring XD和Kafka ，这是所有大数据/快速数据爱好者必不可少的。
    • Julien Dubois最近在基于Spring Boot的应用程序生成器JHipster上的网络研讨会上的重放现已可用！
    • Scott Deeg出色的网络研讨会重播：现在可以使用Spring后端构建安全的Polymer应用程序 ！这与Dave Syer博士的超级流行的Spring和Angular JS：安全单页应用程序博客和网络研讨会的前端JavaScript不同，它使用Spring Cloud Netflix / Zuul在前端使用安全的JS应用程序，这是一个轻量级的API网关， Spring Cloud和安全的Spring后端。
    • 为了庆祝现在基于Spring Boot的Grails 3.0版本，我们将在一段时间内发布SpringOne2GX 2014的所有G＆G演讲。本周的重播现在已经结束！Jeff Scott Brown 从内到外谈论GORM。PaulKing博士向我们展示了Groovy AST转换 ，而Ken Kousen与Groovy分享了高级Groovy技巧和窍门以及分布式平台开发
    • 查看Thomas Darimont的精彩文章《 Spring Data Fowler的新增功能》
    • Disid公司刚刚发布了Spring Roo 2.0的更新 。说起..
    • 我将与其他许多Spring团队成员一起，在西班牙巴塞罗那举行Spring I / O会议 。加入我们！（还有桑格利亚汽酒！）
