This Week in Spring - March 31, 2015

Engineering
Josh Long
March 31, 2015

Welcome to another installment of This Week in Spring! As usual, we’ve got a lot to cover so let’s get to it!

  • Spring Security lead Rob Winch just announced the amazing Spring Security 4.0 release! This one is packed with amazing features including websocket support, Spring Data integration, enhanced testing support, and much improved defaults
  • The Cloud Foundry Java buildpack now supports overriding inbuilt configuration through the use of well-named environment variables! This means you can tweak all manner of the default behavior without needing to fork the buildpack!
  • Good news! Spring XD 1.1.1 is released! It includes improved performance and offset management when using Kafka message bus, support for Spark’s streaming reliable receiver, Kryo serialization improvements, RabbitMQ auto-cleanup options and a Sqoop example.
  • Graeme Rocher’s amazing Groovy Web App framework, Grails 3.0 has been released - now based on the Spring Boot!
  • Greg Turnquist just announced Spring WS 2.2.1, with bugfixes, was just released. Get it while it’s hot!
  • Oh yah, Spring Security 3.2.7 with bugfixes aplenty is also released but… download 4 (see above)!
  • speaking of allll the data, Spring Integration Kafka 1.1 was just released! This is also included in Spring XD 1.1.1, so check it out!
  • Spring framework 4.1.6, with patches and bugfixes, was just released. It is a recommended update for Spring 4.x users.
  • I’ll be doing a webinar on smarter service-to-service invocations with Spring Cloud
  • I’ll be appearing at the Virtual JUG (vJUG) again, this time to talk about Spring Cloud-based microservices. Join us, worldwide!
  • Marius Bogoevici and Mark Pollack are teaming up for a webinar on Tues April 28th about Reactive data-pipelines with Spring XD and Kafka, a must for any big / fast data enthusiasts.
  • The replay from Julien Dubois’s recent webinar on the Spring Boot-based app generator, JHipster - used to generate AngularJS web apps - is now available!
  • Scott Deeg’s excellent Webinar Replay: Building a secure Polymer app with Spring backend is now available! This is a different frontend JavaScript take on Dr. Dave Syer’s uber popular Spring and Angular JS: A Secure Single Page Application blog and webinar, that uses Spring Cloud Netflix/Zuul to have a secure JS app on the frontend, a lightweight API gateway with Spring Cloud, and a secure Spring backend.
  • In celebration of the Grails 3.0 release - which is now based on Spring Boot - we’ll be publishing all G&G talks from SpringOne2GX 2014 for a while. This week’s replays are now up! Jeff Scott Brown talks about GORM inside and out Dr. Paul King shows us Groovy AST Transformations, and Ken Kousen shares Advanced Groovy Tips and Tricks and Distributed Platform Development with Groovy
  • Check out Thomas Darimont’s amazing post, What’s New in Spring Data Fowler
  • The Disid corporation just posted an update looking ahead to Spring Roo 2.0. Speaking of..
  • I, along with a great many other Spring team members, will be in Barcelona, Spain, for the Spring I/O conference. Join us! (also, Sangria!)
