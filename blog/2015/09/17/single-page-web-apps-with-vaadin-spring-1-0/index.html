Vaadin Spring 1.0的单页Web应用程序

工程
斯特凡·尼科尔（StéphaneNicoll）
2015年9月17日

这篇文章是社区成员Matti Tahvonen（@MattiTahvonen）的来宾帖子，他是Vaadin Ltd的开发倡导者，该公司最初是开发Vaadin Framework并为其提供商业服务和扩展。

Vaadin的Spring集成库自5月以来一直处于测试阶段，并且已经被多个生产应用程序使用。今天我们很自豪地宣布Beta标记被删除，稳定的1.0.0版本已经发布。

Vaadin是一个基于组件的Web UI框架，您的应用程序状态和逻辑存储在Java应用程序服务器的内存中。这种体系结构为您的UI代码提供了JVM和诸如Spring之类的框架的全部功能，尤其是将UI逻辑保持在数据旁边的巨大优势。由于桌面UI库熟悉了强大的抽象和基于组件的API，您几乎会忘记自己实际上是在使用Web应用程序，而只专注于域问题。

Vaadin并不是每个网站的工具，但是如果您要构建单页面Web应用程序，而不仅仅是一些简单的表单，那么您肯定应该在工具箱中包含Vaadin。如果您是Vaadin的新手，并且想知道它在后台如何工作，请查看介绍页面以获取更详细的说明。

作为纯Java库，Vaadin一直很容易装入基于Spring的软件堆栈中。使用最近发布的官方Vaadin Spring集成库，它变得更加容易。仅通过用注释标记UI对象，它们就成为Spring托管的bean，使用IoC技术并将例如基于Spring Data的服务直接连接到UI类变得很简单。Vaadin Spring自然支持Spring Boot，但也可以在旧式Spring部署中使用。

如何开始使用Spring和Vaadin？

为了轻松开始新的Spring + Vaadin项目， start.spring.io服务支持Vaadin Spring集成库。只需点击那里的Vaadin复选框，然后将生成的项目导入到您喜欢的IDE。对于基于Spring Boot的简单hello world应用程序，请尝试以下代码片段：

@SpringBootApplication
public class DemoApplication {

  public static void main(String[] args) {
    SpringApplication.run(DemoApplication.class, args);
  }

  @Service
  public static class MyService {
    public String sayHi() {
      return "Hello Spring!";
    }
  }

  @Theme("valo")
  @SpringUI(path = "")
  public static class VaadinUI extends UI {

    @Autowired
    MyService myService;

    @Override
    protected void init(VaadinRequest request) {
      Button button = new Button("Greet service");
      button.addClickListener(e -> Notification.show(myService.sayHi()));
      setContent(button);
    }
  }
}

有关如何继续使用基于Spring + Vaadin的软件策略的更多提示，请访问vaadin.com/spring和我们最近的介绍网络研讨会 。
