查阅新的“使用Vaadin创建CRUD UI”指南

工程
格雷格·特恩奎斯特
2015年9月28日

春季问候社区，

今天，我们发布了新指南： 使用Vaadin创建CRUD UI 。

该指南由Vaadin团队撰写，展示了如何很好地构建可利用Spring Data的UI，但不需要编写一行JavaScript或HTML代码。

Vaadin真的很酷：

  • 带有自己的Spring Boot启动器
  • 与构造函数注入效果很好
  • 使插件持久化解决方案（如Spring Data）变得非常简单
  • 拥有大量的插件以发展为实际应用程序

看看这个新创建的指南，玩得开心！

更多指南即将推出

作为SpringOne 2GX的后续，我们确定了一些还需要编写的指南。云原生的故事席卷了整个开发社区，人们正在寻求更多解决方案。我们的工作不应是机架和堆叠服务器或构建基础架构。相反，我们应该集中精力于轻松简便的解决方案。将内容编码为云原生可以实现这一点。使用Spring Boot和Spring Cloud，这比以往任何时候都更加容易。
