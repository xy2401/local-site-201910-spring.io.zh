春天的这个星期-2015年9月29日

工程
乔什·朗（Josh Long）
2015年9月30日

欢迎来到春季本周的另一个装置！对于那些在美国读书的人，让我成为第一个祝您全国生日快乐的咖啡 ！和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • 您是否一直在查看Annotated Spring网站 ？太好了 ！包含了有关Spring入门的容易理解的截屏视频（有点像过去的Rails Casts）
  • 我们的朋友Greg Turnquist再次发表了他的史诗《 React.js》和Spring Data REST教程的第3部分
  • Greg还不放松，他还发布了一篇很棒的新指南，介绍了如何使用Vaadin和Spring Boot构建CRUD应用程序
  • Spring Data忍者Thomas Risberg刚刚宣布了适用于Apache Hadoop 2.3 M3的Spring
  • Spring Integration Java Config DSL负责人Artem Bilan刚刚宣布了1.1 GA版本！这是一个发布了很多好东西的巨大版本，请查看
  • 我刚刚注意到Geoffroy Warin的这本书， 精通Spring MVC 4 ，现已发行
  • 我们的朋友Nicolas Frankel通过Spring Boot 首次了解了了解微服务
  • 观看9分钟的视频片段，该视频片段来自Phil Webb的监控微服务，以及我的培训视频“ 使用Spring Boot构建微服务”。
  • Rob Hinds 通过Spring Cloud在Netflix OSS堆栈中撰写了一篇介绍性文章
  • 想让JSF在Spring Boot上运行吗？
  • Boyko Dimitrov在与Google作为提供者一起使用Spring Boot的OAuth2支持方面撰写了一篇不错的文章
  • Patrick Grimard在使用Spring MVC的新服务器端JavaScript视图渲染器渲染EJS脚本模板方面发表了精彩的文章
  • Christophe Labouisse返回了有关使用ELK和Docker管理Spring Boot日志的帖子
