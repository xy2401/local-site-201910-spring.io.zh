Spring 的这个星期-2015年7月7日

工程
乔什·朗（Josh Long）
2015年7月7日

欢迎来到 Spring 本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！这周我将在中国上海，与一些客户交谈（在Twitter上打个招呼！）然后去中国深圳， 我将在ArchSummit会议上作主题演讲 。

  • Spring Integraton负责人Gary Russell 刚刚宣布了Spring Integration 4.2和4.1.6现已可用 。新版本包括对Apache Zookeeper Java 8的支持。 CompletableFuture s，（S）FTP -stream ，并在性能上进行了一些改进。
  • Spring Boot联合负责人Phil Webb 刚刚宣布了Spring Boot 1.2.5 。
  • Spring Framework 4.2是快速接近（在大约一个星期！）并且支持许多不同（有时是新的）项目的许多新版本。要了解更多信息，请查看Spring Framework 负责人Juergen Hoeller撰写的详细博客。
  • Spring Data忍者Christoph Strobl 刚刚宣布了Spring Data Fowler SR1版本 。这是一个维护版本，其中包含许多新功能，因此请务必了解。
  • Spring Data负责人Oliver Gierke刚刚宣布了最终的Spring Data Service版本（Evans SR3和Dijkstra SR6） 。
  • Daniel Jagielski对使用Spring Boot和DropWizard的优缺点进行了很好的介绍。应该指出的是，我们Spring Boot团队的确受到了DropWizard的启发
  • 我找到了一个老兄但又好吃的人：这是敏捷专家布莱恩·马里克（Brian Marick）在2008年博客中对敏捷开发实践的评论
  • 想赶上Redis的所有事情吗？我非常喜欢Redislabs通讯，我将其链接至最新一期
  • 我们的朋友Marten Deinum精心整理了一些出色的Spring Framework 瑰宝
  • 我喜欢Cloud Foundry After Dark ，直播的Google Hangouts每个星期日都有各种Cloud Foundry杰出人物。在这一集中， 我们的英雄之一-安德鲁·克莱·谢弗 （ Andrew Clay Shafer） – 剃了光头
  • 我真的用Pivotal Cloud Foundry挖了Michael Cote的关于连续交付的采访 -请查看！
