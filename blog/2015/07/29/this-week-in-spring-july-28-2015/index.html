春天的这个星期-2015年7月28日

工程
乔什·朗（Josh Long）
2015年7月29日

欢迎来到春季本周的另一期！我们有很多内容需要介绍，让我们开始吧！

  • Spring Data负责人Oliver Gierke 刚刚宣布了Spring Data SR2 ，该版本修复了40个问题，并且对对JTA 1.2的支持进行了重要更新。@Transactional
  • Spring Security主管Rob Winch 刚刚宣布了Spring Security Kerberos 1.0.1 ，其中包括Spring Security 4和LDAP 2.0支持，以及一个问题的修复程序。
  • Spring Security负责人Rob Winch没睡，所以他还 宣布了Spring Security 3.2.8
  • Spring Security负责人Rob Winch 并没有因此而感到惊讶， 还宣布了Spring Security 4.0.2，其中包括许多错误修复，对Spring Framework 4.2的支持以及对Spring Security本身的较小改进，包括返回的状态代码。HttpStatusReturningLogooutSuccessHandler
  • 我们的朋友Dan Woods 在将Spring Boot与Ratpack一起用于InfoQ 上写了一篇非常不错的文章 。看看这个！
  • 想更多地了解Spring Boot和Spring Cloud？我喜欢我们的朋友埃伯哈德·沃尔夫（Eberhard Wolff）的这些幻灯片
  • Spring XD联合负责人Mark Pollack博士刚刚宣布了Spring XD 1.2.1 ，其中包括对注册自定义Kryo序列化器的支持，以及用于调整Kryo性能的配置选项，包括Spring Integration，Spring Boot和Reactor在内的多个组件的更新；更新的管理界面，并改进了容器和作业的分页。
  • 我不会讲这种语言，但是我有几个人告诉我，他们喜欢 Igor Khotin在乌克兰基辅举行的JEEConf 上在Spring Cloud上进行的乌克兰语演讲 。看看这个！
  • 约翰·沃特斯（John K.Waters）撰写了一篇不错的文章，介绍了Pivotal如何通过Spring Boot和Spring Cloud引领云原生应用程序革命
  • Rambabu Posa有一篇关于使用Spring Initializr入门Spring Boot的不错的文章
  • 我们所有的朋友和其他博客再次支持它！（我错过了Biju的最新帖子！）最新的帖子是关于使用Spring Boot的@ConfigurationProperties 。
  • 我错误地忘记了包含上述项目的链接，因此当我再次谷歌搜索以查找链接时，我偶然发现了这个老兄，但是一个很好的人使用了RxJava的Observable在Spring MVC流程中
