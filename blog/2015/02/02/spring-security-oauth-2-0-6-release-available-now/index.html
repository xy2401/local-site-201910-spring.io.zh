Spring Security OAuth 2.0.6。现在发布

发布
戴夫·瑟
2015年2月2日

Spring Security OAuth 2.0.6。现在可以在通常的存储库中使用RELEASE。这是一个错误修复版本，用户为2.0.5。发布应该升级。唯一的严重错误是使用刷新令牌的JWT用户的，并且有些人正在经历重定向uris的双重编码，现已修复。新功能的亮点：

  • （非常需要）具有无需定制的未过期刷新令牌的能力（只需将有效期设置为零或更少）

  • / token端点默认仅接受POST请求

  • 资源服务器默认情况下不接受基于cookie的身份验证（您必须将其打开）

  • 资源服务器配置具有一些新选项，包括注入自定义错误处理程序的能力

与往常一样，感谢社区贡献代码和想法。感觉社区正在成长，这总是很高兴的。最后，注意即将在此处发布的有关带有Spring Security和OAuth2的Angular JS的新博客。
