春天的这个星期-2015年2月24日

工程
乔什·朗（Josh Long）
2015年2月24日

欢迎来到春季本周的另一期！我们有很多内容需要介绍，让我们开始吧！

  1. 我们的好朋友Adam Koblentz（来自ZeroTurnaround） 发表了一篇很棒的文章，介绍了如何使用Spring Boot和JRebel构建Websocket应用程序 。看看这个！
  2. 观看Mark Fisher，Mark Pollack博士和Sabby Anandan的网络研讨会的重播，该网络研讨会介绍了Spring XD- 大规模数据和开发人员生产力的平台
  3. 当然，Pivotal Data Suite的很大一部分是Spring XD。上周，我在Spring XD 1.1的新流处理支持中展示了一些惊人的Spring XD Wiki内容。查阅Wiki页面 ，了解Spring XD ninja Ilayaperumal Gopinathan的更详细外观。
  4. Spring Cloud联合负责人Spencer Gibb一直在改善Spring Cloud Netflix的集成。查看此示例， 演示使用RxJava的Observable Spring MVC的返回值 。 这是变化 。
  5. 3月的网络研讨会就在这里，其中包含Javascript，Spring Boot和Spring Cloud。
    这个杀手级阵容始于Dave Syer博士的史诗般的VII部分博客系列，该系列有关Angular的轻量级API网关-Spring Security- 于3月10日成为网络研讨会 。然后，斯科特·戴格（Scott Deeg）将在3月24日的网络研讨会上对同一件事进行研究，只是将Google的Polymer放在前端。最后，Julien Dubois将于2015年3月17日推出AngularJS + Boot应用程序生成器JHipster。
  6. Alvin Reyes的帖子介绍了如何配置Apache Tomcat的server.xml和users.xml是很棒的资源。
  7. Reactor项目负责人Jon Brisbin 刚刚宣布了Reactor RC1 ！
  8. Spring NinjaStéphaneNicoll刚刚宣布了Spring Framework 维护版本4.1.5
  9. 上周，我们在Pivotal宣布了一些重大变化；我们已经将Pivotal数据套件开源了！查看有关此主题的InfoQ帖子
  10. Spring IO Platform负责人Andy Wilkinson刚刚发布了对使用Gradle的Maven物料清单依赖管理的改进支持
  11. Eclipse基金会成员John Arthorne和Spring Tool Suite主持了Martin Lippert的演讲， Eclipse IDE到 JavaOne 2014 开发人员工具的基于云的时代已经可以在线获取，非常值得一看！
  12. 关于Pivotal Data Suite开源的《 SD Times》文章也值得一读。
  13. 无关，但仍然很棒： HTTP2现在已经成为问题 ！
  14. 查看Rick Hightower的有关将QBit框架与Spring Boot结合使用的文章。有趣..
  15. Hubert Klein Ikkink简要介绍了使用Spring Boot的OutputCapture JUnit的@Rule在Spock测试中 。
  16. 我们的好朋友Eugen Paraschiv对使用古老JdbcTemplate
  17. Disid的Enrique Ruiz撰写了一篇很好的文章，介绍了使用STS的Spring Roo 。
  18. 观看本周的SpringOne2GX 2014回放：
    SpringOne2GX 2014重放：小组会议：实际启动顺序
    SpringOne2GX 2014重放：Spring数据REST-数据遇到超媒体
    SpringOne2GX重播：Spring Batch性能调整
    SpringOne2GX 2014重放：具有Spring Boot的艺术性Spring Data Neo4j 3.x
