春天的这个星期-2015年2月3日

工程
乔什·朗（Josh Long）
2015年2月3日

欢迎来到春季本周的另一期！我们有很多内容需要介绍，因此，不费吹灰之力，让我们开始吧！

  1. Spring XD的共同负责人Mark Pollack博士刚刚宣布Spring XD 1.1 RC1现已上市 ！现在是时候得到点点滴滴，尝试一下，看看是否有任何缝隙！
  2. 如果您最近阅读过Syer博士令人惊叹的博客，您会知道他一直在向人们介绍如何为UI客户端使用公开和安全的REST服务。第四篇文章探讨如何在客户端和后端服务之间插入API网关。然后，第五篇文章介绍OAuth，以替代正在使用的定制身份验证会话令牌。如果您不遵循此系列，请返回并重新阅读它们。本系列讨论了我一直被问到的一个主题。
  3. Sabby Anandan将于2月17日星期二与我们一起参加Spring XD 1.1发布网络研讨会-Spring XD：大规模数据和开发人员生产力的平台
  4. 希望您能与我一起在2015年2月24日星期二的Spring Session项目中重新体验HTTP会话
  5. Simplicity Itself博客很好地处理了服务发现和注册的主题，并很好地介绍了各种服务注册表， Consul，Eureka和Zookeeper 。Spring Cloud包括对Consul，Eureka和ZooKeeper的支持。
  6. 本周，我们很幸运地邀请了Spring Social Slideshare项目的创始人Tadaya Tsuyukubo（@ttddyy） 在本来宾帖子中向我们介绍了该项目 。
  7. 好的Syer博士（@david_syer） 刚刚宣布Spring Cloud 1.0.0 RC2现已可用！新版本改善了安全性集成，并清理了一些入门工具。这艘船很快就升到了1.0！尽您所能检查出位！
  8. 谈到Syer博士（@david_syer） ，他还 宣布了Spring Security 2.0.6 。
  9. 查看Paul King博士在SpringOne2GX 2014上有关使用Groovy操纵数据库的演讲
  10. Disid 已发布Spring Roo 1.3.1 ，其中包括许多错误修复和一些调整。
  11. 查看Grails项目负责人Graeme Rocher在SpringOne2GX 2014上的演讲， 使Spring Boot变得Groovier
  12. Spring Data忍者Thomas Risberg在SpringOne2GX 2014上的帖子。Apache Hadoop的Spring简介现已在线提供。这是开始使用Spring和大数据的好地方。
  13. Groovy项目负责人Guillaume LaForge的演讲“ Java 8之光下的Groovy”现已在线上提供。检验一下以了解两种语言的相似之处以及Groovy在何处遗漏Java 8
  14. Spring Data负责人Oliver Gierke刚刚宣布Spring Data Evans SR2和Spring Data Dijkstra SR5现已上市！此版本包含许多改进，因此建议升级！
