春天的这个星期-2015年2月17日

工程
乔什·朗（Josh Long）
2015年2月17日

欢迎来到春季本周的另一期！本周，我去录音室录制了Spring Livelessons系列的下一个版本，这个系列是关于使用Spring Boot，Spring Cloud和Cloud Foundry构建云原生应用程序（微服务）的。这很有趣，而且-今天早上-我花了一点时间为O'Reilly做两个1.5小时的网络研讨会，以构建完全实时编码的Spring Boot应用程序和Spring Cloud微服务！对我来说，这是一个有趣的星期，我希望对您也是如此。

这周四19标记今年的农历新年之大成，或中国新年或春节 。这与Spring（技术）有什么关系？除了庆祝更好的事情和一个好名字以外，还不算什么！祝世界各地的读者新年快乐！

无论如何，事不宜迟，让我们开始吧！

  • 好的Mark Pollack博士刚刚宣布了Spring XD 1.1 ！新版本令人难以置信 ：基于Reactor，RxJava和Spark的流处理； Sqoop和Spark批处理作业（除了Spring Batch和Hadoop）； Python处理器模块；用于定制处理器模块的Java配置模型；有无数的后端，包括卡夫卡的整合，以及更多 ！把握机会，更快，更可靠地完成数据处理管道，并及时回家庆祝！
  • 好的Dave Syer博士刚刚宣布了Spring Cloud 1.0.0。RC3现在可用 ！希望这是1.0.0之前的最后一个里程碑，因此请务必尝试一下！
  • Spring XD忍者Sabby Anandan撰写了一篇不错的文章，介绍了Spring XD 1.1中的许多新功能
  • Spring Ninja Stephane Nicoll撰写了一篇不错的文章，介绍了Spring Framework 4.2中对新事件的支持
  • 本杰明·温特贝格（Benjamin Winterberg） 用Spring Boot和Nashorn构建了一个同构（使用React.js在客户端和服务器上呈现的JavaScript视图） 应用程序的漂亮文章！太酷了！
  • Spring（和Spring Boot）使组装无头处理应用程序变得非常简单-您知道，没有HTTP端点的应用程序吗？所以，自然地， 当我看到这个 Spring Boot 子时 ， 我笑了。 init.d脚本 ！
  • 这个在OpenShift上运行Spring Boot的“自己动手”示例很有趣 。OpenShift具有墨盒的概念。我认为-但不是100％肯定-他们在逻辑上提供了与Cloud Foundry和Heroku上buildpack概念的映射。显然，除了DIY墨盒无法放大外，因此不建议将其真正用于任何用途。
  • 我们的好朋友Dan Woods 对InfoQ的微服务Spring Boot进行了另一番精彩报道
  • 令人惊异的Brian Dussault在我们的朋友[James Wards的帖子， Java Not's Suck，You're Doing It Wrong》中整理了一篇很好的后续文章 。
  • 观看此重播，观看Andy Wilkinson的演讲，“ 记录REST API”
  • 想要了解更多有关Spring Integration Java配置DSL的信息（这很夸张！当然可以！），然后查看Gary Russell关于该主题的精彩网络研讨会的重播 。
  • 谈到丹·伍兹（Dan Woods），请观看他在2014年SpringOne2GX演讲中介绍RatPack的重播
  • Groovy语言忍者CédricChampeau的SpringOne2GX 2014演讲，“ 重新思考具有特质的API设计”已经发布！
  • 肯尼思·库森（Kenneth Kousen）在SpringOne2GX 2014演讲中介绍了Grails中的各种单元测试方法，现在可以在线获取
  • Ryan Vanderwerf在2014年的SpringOne2GX演讲中介绍了与Terracotta和Grails一起使用Web集群的内容，现已在线发布
  • Peter Niederwieser在SpringOne2GX 2014上的演讲，即使用Gradle和Jenkins构建连续交付管道现已发布 ！
  • Colin Harrington的有关保护Grails应用程序的SpringOne2GX 2014演讲现已发布
  • 另外， HTTP 2是最终版本 ！
