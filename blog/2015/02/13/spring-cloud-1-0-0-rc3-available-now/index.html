Spring Cloud 1.0.0。RC3现在可用

发布
戴夫·瑟
2015年2月13日

Spring Cloud 1.0.0。RC3现在可从https://repo.spring.io/libs-milestone-local获得 。 （希望如此）这是1.0.0之前的最后一个里程碑版本。自1.0.0开始，已修复了一些错误。RC2以及一些小的新功能：

  • 重构了Feign支持，使其看起来更像Spring Data（因此@EnableFeignClients代替@FeignClientScan ）。
  • 在Zuul代理中支持multipart / form-data
  • 支持在Zuul的自动路由注册中包括和排除远程服务
  • 在Zuul中支持声明式功能区重试
  • 清理了很多依赖项。如果您使用spring-cloud-starters，则应该获得一致的添加和减去功能的经验。Gradle用户需要使用依赖项管理插件才能获得相同的体验。
  • 添加了一些很小的示例项目，以对类路径隔离问题进行集成测试。
