Spring Framework 4.2.3现在可用

发布
斯特凡·尼科尔（StéphaneNicoll）
2015年11月16日

我代表团队高兴地宣布，现在可以从repo.spring.io和Maven Central获得Spring Framework 4.2.3。4.2行中的第三次维护版本包含60多个修复和改进 。

我们强烈建议所有4.x用户立即升级到4.2.3。此版本是为即将到来的春季启动1.3 GA版本很快上来了基础。
