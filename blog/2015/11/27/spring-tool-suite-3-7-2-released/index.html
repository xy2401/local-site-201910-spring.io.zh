Spring Tool Suite 3.7.2发布

发布
马丁·利珀特
2015年11月27日

亲爱的春季社区，

我很高兴宣布Spring Tool Suite的3.7.2版本，这是我们基于Eclipse的工具。

STS 3.7.2专注于解决现有问题并更新STS以与Cloud Foundry，Spring和各个Spring项目上的最新更新一起使用。

已解决的问题列表包括：

  • Spring Boot仪表板现在可与Cloud Foundry上的新Diego运行时一起使用
  • Spring Boot 1.3.0兼容性（由于内部更新了spring组件）
  • 快速文本搜索现在可以搜索转义的*和？人物
  • Eclipse Buildship的初步支持（Gradleware的Eclipse的新Gradle集成）

要下载发行版，请访问：

  • Spring Tool Suite： https : //spring.io/tools/sts/all

详细的新的和值得注意的笔记可以在这里找到： STS 3.7.2新的和值得注意的 。

STS 3.7.3计划于2016年3月上旬交付。
