    春天的这个星期-2015年1月20日

    工程
    乔什·朗（Josh Long）
    2015年1月20日

    欢迎来到春季本周的另一期！可以很容易地将本周的综述标题为“ 本周”（在春季视频和网络研讨会中） ，因为我们有很多内容！

    1. 优秀的Dave Syer博士继续他的系列文章， 介绍了Spring Security和Angular.js集成，这一次着眼于将资源服务器与授权服务器分离 。这篇文章看起来
    2. 我们的好朋友Nicoalas Frankel又回来了！尼古拉斯（Nicolas）居住在Vaadin和Spring社区。因此，自然而然，当我和PetterHolmström在2014年初启动Vaadin4Spring项目时，当尼古拉斯开始做出巨大贡献时，我们并不感到惊讶。他的最新贡献介绍了Vaadin的MVP模式的优雅实现，并且他对它进行了很好的介绍 -来看看！
    3. 我将在28日的Oakland JUG上发表有关使用Spring Cloud和Cloud Foundry构建微服务的演讲。
    4. Tadaya Tsuyukubo刚刚宣布了Spring Social Slideshare项目的第一个1.0.0版本 。我还没有机会玩这个游戏，但是看起来它可能会划伤我的一些瘙痒！干得好！
    5. 希望您下个月加入我的网络研讨会，介绍Spring Session项目 。
    6. 谈到Vaadin和Spring，我将在本周的GWT.create会议上共同演讲。我们的演讲将着眼于使用基于Vaadin的HTML5前端和Spring Boot驱动的后端和无人机客户端控制Parrot AR无人机。快来加入我们，一定会很有趣！
    7. InfoQ就Spring Boot 1.2.1的所有新功能对 Spring Boot联合创始人Phil Webb进行了采访。
    8. Alexandre Lourenco 在DZone上的这篇快速文章中发表了一篇文章，展示了使用Spring Boot来启动和运行REST API是多么容易
    9. 请务必在2月17日举行的Spring XD.next网络研讨会上查看Spring XD的Sabby Anandan
    10. 无与伦比，令人惊叹的Andy Wilkinson将在2月3日的网络研讨会上讨论如何记录REST API 。
    11. 我整理了一篇文章，介绍了Consul和Eureka等服务注册表， 以便在基于Spring Cloud的系统中提供更具弹性的服务到服务的调用 。
    12. 我们的好朋友Jakub Kubrynski撰写了一篇精彩的 文章，内容涉及在Spring Boot uber-jar中包含NewRelic Java代理
    13. 在整理有关使用Spring Cloud服务发现和注册的文章时，我偶然发现了Jason Wilder对服务注册中心的出色理解，并非常喜欢它，即使它本身与Spring无关。
    14. 观看Rossen Stoyanchev在SpringOne2GX 2014演讲中的重播，其中介绍了Spring 4对Web应用程序的支持
    15. 查阅Matthew Adams（SCI Spike）和Michael Nitschinger（Couchbase）在SpringOne2gx 2014中的演讲，内容涉及在企业中使用Spring Data Cassandra和Spring Data Couchbase。
    16. Garrett Wampole在SpringOne2GX 2014上有关应用于实时数据处理的企业集成模式的演讲值得关注！
    17. 查看我们的朋友和西班牙春季I / O会议的组织者Sergi Almar的演讲深入探讨websockets
    18. 谈到超赞的西班牙人， Disid团队刚刚宣布了Spring Roo 1.3.1 RC2
    19. Spring Tool Suite和Groovy Tool Suite的负责人Martin Lippert刚刚宣布了最新版本3.6.3 SR1 。
    20. 我的朋友卢克·香农（Luke Shannon）举办了精彩的网络研讨会，介绍了Spring Data Gemfire ，这是一个针对世界级数据网格的世界级集成。
    21. Spring Batch和Spring Batch Admin主管Michael Minella刚刚宣布了这两个项目的最新版本 。新的Spring Batch修订版包括对JSR 352自定义基础结构的支持，对分区结果的轮询等等。Spring Batch Admin修订版集成了Spring Batch中的最新功能和最新功能，包括JSR 352和Java配置支持。看看这个！
    22. Spring IO Platform负责人Andy Wilkinson发布了Spring IO Platform 1.1.1，其中包括Spring AMQP，Boot和Integration的最新版本
    23. Spring Integration主管Gary Russell在Spring Integration 2的2014年SpringOne2GX演讲中不容错过 ！
    24. 查看Spring框架负责人Juergen Hoeller和Stephane Nicoll关于Spring 4.1中的缓存和消息传递改进的演讲。
    25. Spring Security负责人Rob Winch 关于Spring Security的SpringOne2GX 2014入门演讲也已发布
    26. Spring框架负责人Juergen Hoeller在Spring框架和Java 8上的SpringOne2GX 2014演讲对于那些关注下一代Spring 4和4.1的人来说是不容错过的入门
    27. 另一个关于使用的好帖子SimpleJdbcInsert处理JDBC数据插入
    28. Alvin Henrick在一起使用Spring Boot和Docker时撰写了一篇不错的文章
    29. 我发现了这篇关于在Spring Boot上使用Groovy的有趣的问答文章
    30. 亚历克斯·柯林斯（Alex Collins）关于使用Docker，Elastic Search和Selenium创建盒中搜索的文章是史诗般的 ！看看这个！
    31. 这篇有关使用新的Spring Boot 1.2.1的Jersey支持的日语文章看起来不错（通过Google翻译！）
    32. 是否想通过Docker站起来服务？查看我们的朋友Chris Richardson的关于通过Docker站起来服务的帖子 ！例如，您可以从另一个Docker打包的Spring Boot应用程序或Cloud Foundry.next（至今称为“ Diego”）上使用这些服务。 http://plainoldobjects.com/2015/01/14/need-to-install-mongodb-rabbitmq-or-mysql-use-docker-to-simplify-dev-and-test/
    33. 您是否正在Cloud Foundry中关注本周 ？做！和往常一样，那里有很多好东西。

    网络研讨会构成了本周综述的大部分内容。很好， 彼得·汉弗莱 （ Pieter Humphrey） ！
