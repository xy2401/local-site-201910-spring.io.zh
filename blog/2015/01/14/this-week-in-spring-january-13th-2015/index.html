春天的这个星期-2015年1月13日

工程
乔什·朗（Josh Long）
2015年1月14日

欢迎来到春季本周的另一期。我们有很多话要说。我们自己的彼得·汉弗莱一直不懈努力，得到的SpringOne2GX 2014节目重播可在网上有他们摆这个星期！真是个胜利！

  • 多年以来，Spring Batch的前负责人，Spring Boot，Spring Cloud和Spring Security OAuth的共同负责人，以及包括Spring本身在内的多个Spring项目的摇滚明星，出色的Syer博士已经在非常与使用Spring保护Angular.js应用程序有关的实际问题。第一个是Spring和Angular JS：安全的单页应用程序 ，介绍了将基于Angular.js的客户端连接到后端API的基础。第二部分， 登录页面：Angular JS和Spring Security第二部分 ，介绍了登录表单。收藏本系列！
  • 令人惊叹，耐睡眠的Rob Winch刚刚宣布了Spring Session 1.0 。对于那些尚未关注的人来说，Spring Session是标准Servlet HTTP Session API的代理API的下降。安装它，然后将HTTP会话持久性委派给其他后端实现（例如Redis）。出于多种原因，这是理想的选择：如果您正在使用Java EE应用程序服务器，并且希望扩展会话复制，则可以使用针对它进行更好设计的引擎。可以肯定的是，Cassandra和Redis背后的团队比Glassfish背后的团队对数据复制的思考更多，不是吗？如果将应用程序部署到Cloud Foundry或Heroku等PaaS（默认情况下未启用会话复制），那么这为您提供了一种简单的添加方法，并且添加方式从一个环境到另一个环境都是一致的。另一个。
  • 我整理了一篇关于Spring如何支持12因子应用程序样式配置的博客
  • 我们的朋友尼古拉斯·弗兰克尔（Nicolas Frankel）又来了一次，这次有一篇不错的文章，介绍了如何使用Spring的配置文件和Maven的配置文件
  • Spring Boot 1.2.1（主要是维护版本）刚刚发布
  • 观看Janne Valkealahti的演讲“ Springless 2GX 2014重放”，该讲稿使用Spring为YARN应用程序进行无痛构建和部署。
  • 观看Groovy忍者Cedric Champeau在SpringOne2GX 2014上的演讲， Android和Groovy：双赢 。在苹果宣布推出Swift（一种类似Groovy的iOS语言）之后，他刚刚宣布在Android上支持Groovy。自然，它引起了人们的极大兴趣，而这次谈话填补了空白。看看这个！
  • Jeff Beck的演讲“ 在微服务体系结构中使用Grails的体验”现已发布。这是一个值得一提的手表，可以深入了解向micorservices过渡的痛点。
  • 长期阅读本综述的读者无疑会记得我的前队友Peter Ledbrook 。我是Spring Developer Advocate，他是Grails Developer Advocate，教了我很多东西。我要说的全部是，如果您想从最好的之一中学习，则应该查看他的演讲，包括来自SpringOne2GX 2014的如何使用Java 8获得Groovy 。
  • 谈到彼得，他还在 Groovy＆Java 8上写了这篇很棒的文章。
  • 在开源Spring IO参考应用程序和网站的内容中查看Brian Clozel的SpringOne2GX 2014演讲
  • ETrade的Durai Arasan和Monish Unni 在使用Spring Integration和Node.js构建REST服务方面做了很好的演讲 。
  • 谈到我们的朋友尼古拉斯·弗兰克尔（Nicolas Frankel），请阅读他的书《 来自Trenches的集成测试》 ，其中有很大一部分是关于Spring的。
