Spring 的这个星期-2015年1月6日

工程
乔什·朗（Josh Long）
2015年1月7日

欢迎来到春季本周的另一期！希望您的假期和新年真棒！这是新的一年的开始（我几乎输入了“ 2014”！），今年我们将迎来很多很棒的东西！让我们看看上周在Spring社区中互联网在做什么。

  • 我们的朋友Petri Kainulainen又回来了，他已经彻底修改了Spring Data JPA CRUD教程 ！看看这个！在本综述中，我们已经链接了它，但始终值得重新阅读！
  • 我真的很喜欢这篇文章，该文章根据基于JHipster的示例着眼于将多租户引入应用程序 。
  • Inforworld有一个有趣的帖子， 为什么2015年将成为微服务之年
  • 我们在西班牙的好朋友今年将再次举办他们的Spring IO（会议）会议！它正在塑造成一个非常棒的演出，我可以告诉您，没有任何人比这些人更努力地为社区做事。如果您在该地区，希望您会考虑！
  • 同时，在Codeleak.pl博客上，有关于使用Jersey和Spring Boot 1.2的精彩介绍。
  • …以及有关如何使用Spring Boot 1.2和Jersey建立HATEOAS API的后续文章
  • Marcin Grzejszczak很好地介绍了他公司的一些开源工具，这些工具使用Spring（和Spring Boot）来支持微服务。这是他的第一篇文章 。我没有探索所有内容，但看起来很有趣！
  • 这是一个新视频，显示了如何使用Spring Security XML名称空间来保护（在这种情况下，基于JAX-RS的）REST服务的安全
  • 我认为这是一个韩语视频，内容涉及为MyBatis编写单元测试 。我还不知道为什么，但是我喜欢看着人们重复。有时（一幅电影）值一千个字。
  • 我认为这是一个孟加拉语视频， 介绍如何开始使用Spring Boot 。无论哪种方式，视频都能说明一切！

此外，这篇文章标志着4年的写作“ Spring This Week”！感谢大家多年来给我带来的丰富内容，也感谢所有在Twitter或电子邮件 上提出建议的人。谢谢，并继续努力，Spring社区，您真了不起！
