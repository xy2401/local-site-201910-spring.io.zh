Spring Data Moore RC1和Lovelace SR9发布

发布
克里斯多夫·斯特罗布尔
六月17，2019

我很高兴代表团队宣布Spring Data发布Moore RC1和Lovelace SR9。这些新功能基于最新的Spring Framework版本，并将分别由Spring Boot 2.2 M4和2.1.6所采用。

值得注意的新功能包括：

  • EntityCallback API，用于在转换或保存之前修改实体。
  • Spring Data JPA支持存储过程中的多个OUT参数。
  • Spring Data MongoDB中的声明式聚合 。
  • 针对Gemfire和Apache Geode的增强的SSL支持和动态客户端端口配置。

请找到我们发行版Wiki中已添加内容的高级概述。
一如既往，我们期待您的反馈！
