在start.spring.io上探索项目

发布
达米安·维特拉克（Damien Vitrac）
七月04，2019

在start.spring.io上引入新功能： 探索项目 。
有了它，您现在可以浏览生成的项目文件，而无需实际下载，解压缩等。

许多开发人员要求使用此功能 ，因为他们想与另一个项目进行比较，或升级现有的应用程序。我们希望将来进一步探索这些可能性，因此请以我们的方式发送您的反馈 ！

探索项目：

  • 根据项目的不同，默认情况下将显示pom.xml / build.gradle / build.gradle.kts
  • 突出显示代码（1），预览降价
  • 复制并下载文件

（1）支持：Javascript，Markdown，Kotlin，Gradle，Gitignore，Java，XML，Properties，Groovy

请注意，该功能的第一个版本并不总是适合移动设备使用 ，因此我们正在收集反馈。

在继续学习和迭代网站和Initializr库的过程中，我们感谢大家的帮助和反馈 。
