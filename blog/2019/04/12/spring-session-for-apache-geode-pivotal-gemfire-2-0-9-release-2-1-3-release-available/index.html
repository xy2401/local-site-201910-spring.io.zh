Apache Geode和Pivotal GemFire 2.0.9的春季会议。发布＆2.1.3。可发布

工程
约翰·布鲁姆
2019年4月12日

我很高兴地宣布针对Apache Geode和Pivotal GemFire（SSDG）2.0.9发行了Spring Session。发布和2.1.3。发布。

这两个发行版都专注于依赖项更新，以使其各自发行版中的Spring产品组合的其余部分保持一致。

SSDG 2.0.9。发布基于：

  • Spring Framework 5.0.13。发布

  • 春季数据Kay-SR14

  • 春季会议2.0.10。发行（Apple-SR9）

  • 旨在在Spring Boot 2.0.9中使用。发布。

SSDG 2.1.3。发布基于：

  • Spring Framework 5.1.6。释放

  • 春季数据Lovelace-SR6

  • 春季会议2.1.5。发布（Bean-SR4）

  • 旨在在Spring Boot 2.1.4中使用。发布

所有的SSDG位在Maven Central（ 此处 ）中都可用。

下一步是什么

现在，所有注意力都集中在基于Spring Framework 5.2，Spring Data Moore，Spring Session 2.2并在Spring Boot 2.2中使用的SSDG 2.2版本。

建议和即将推出的功能包括：

  • 附加的会话（选项）。

  • 使用更强的一致性（可选） Map.replace(key, oldValue, newValue)轻量级交易取代了对较重的缓存/本地交易的需求。

  • PDX序列化支持方面的改进。

  • 还有更多...
