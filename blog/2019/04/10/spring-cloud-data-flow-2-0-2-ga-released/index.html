Spring Cloud Data Flow 2.0.2 GA发布

发布
Ilayaperumal Gopinathan
2019年4月10日

Spring Cloud Data Flow团队很高兴宣布2.0.2 GA Data Flow的发布。遵循在本地 ， Cloud Foundry和Kubernetes上运行的入门指南。

这是次要版本，主要包含错误修复和文档增强功能。

以下是此版本的重点：

  • 改进的文档

    • 文档更新，可在Helm Chart上启用Kafka

    • 改进的架构图

    • 入门指南改进

  • 关于Spring Cloud Data Flow，Spring Cloud Data Flow UI和Spring Cloud Deployer实施的错误修复
