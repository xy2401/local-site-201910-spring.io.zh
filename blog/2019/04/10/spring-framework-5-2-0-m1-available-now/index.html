Spring Framework 5.2.0。M1现已上市

发布
布赖恩·克洛泽尔
2019年4月10日

我很高兴代表团队和所有贡献者宣布，Spring Framework 5.2的第一个里程碑已经发布，可以从我们的里程碑存储库中获得 。此版本关闭了140多个问题并提出了请求 。

这个第一个里程碑包含许多功能和修补程序，其中包括：

  • 许多核心容器改进，从使用新的MergedAnnotations API解析注释数据到@Configuration类优化
  • 支持Kotlin协程
  • spring-webmvc模块中的新WebMvc.fn编程模型提供了对基于Servlet API的带注释的控制器的功能替代。现在，像spring-webmvc这样的spring-webflux提供了功能性和基于注释的编程模型。
  • Spring MVC和Spring WebFlux中的性能改进可减少请求映射，媒体类型解析，CORS检查等方面的开销
  • RSocket支持包括通过带注释的@MessageMapping方法进行 响应处理以及通过RSocketRequester 执行请求 。
  • 许多有趣的集成测试改进，尤其是当您在测试中处理应用程序和上下文事件时
  • 我们的参考文档焕然一新

请留意的是什么在Spring框架5.x的新的和升级到Spring框架5.x的 wiki页面，因为我们会继续在里程碑阶段增加信息。如果你想知道我们的支持政策，该专用页面应该帮助你 。

如果您想在Spring Boot应用程序中试用新的Spring Framework功能，则可以使用Spring Boot 2.2.0。M2刚出来的时候 。当然，这将在https://start.spring.io上可用。
