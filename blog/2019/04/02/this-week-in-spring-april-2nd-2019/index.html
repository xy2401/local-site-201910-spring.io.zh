春天的这个星期-2019年4月2日

工程
乔什·朗（Josh Long）
2019年4月2日

嗨，春季粉丝们，欢迎来到春季本周的另一期！本周，我将在纽约市参加卡夫卡峰会，在那里我将与Confluent开发人员经验负责人蒂姆·伯格隆德 （ Tim Berglund）和Pivotal高级副总裁James Watters一起探讨为何Pivotal和Confluent在一起会更好。加入我们！然后，下周去南非（参加在开普敦和约翰内斯堡举行的SpringOne巡回演出，然后去毛里求斯参加DevConf会议 。如果您在这些地方中的任何一个， 请随时打个招呼！） 。

LTR：合流联合创始人Jay Kreps，Tim Berglund，我和Pivotal高级副总裁James Watters

事不宜迟，让我们开始吧！

  • Spring Framework 5.1.6、5.0.13和4.3.23现在可用
  • Spring Tools 4.2.0发布
  • 上周， 在Bootiful Podcast （@BootifulPodcast）上，我与Josh Mckenty谈到了数据主权，微服务，Cloud Foundry，Python，云，Pivotal等
  • 如果您要阅读一个博客，请在Reactor Debugging Experience上阅读该博客。太神奇了 ！
  • Spring Tool Suite 3.9.8发布
  • 是否想知道Pivotal Cloud Foundry是运行Spring Boot应用程序的最佳场所？
  • 专注于集成测试，而不是基于模拟的测试
  • 温馨提示：您应该尽快升级到Spring Framework 5.1x
  • Sara Torrey用图解说明了她介绍 Spring Boot的笔记 ，它们真的很棒！
  • 在最新的OffHeap播客集中查看Pivotal的Ben Hale关于企业中响应式编程的状态，RSocket和R2DBC
  • 是否错过了这个炙手可热的Spring技巧，该技巧介绍了使Spring Boot变得懒惰和快速的方法？现在是时候观看了 ！还不到十分钟！
  • 我们正在进行一项调查，着眼于如何改善Spring Initializr（起点dot Spring dot io） ，这是我心中最亲近的话题。请在这里反馈
