可用Spring Integration 5.2 GA

发布
阿尔特姆·比兰（Artem Bilan）
十月02，2019

亲爱的春季社区，

我很高兴代表Spring Integration小组宣布Spring Integration 5.2.0.RELEASE可用。

可以从我们的发布存储库和Maven Central中下载：

compile "org.springframework.integration:spring-integration-core:5.2.0.RELEASE"

这代人有什么新东西？

  • RSocket支持-一个RSocketInboundGateway和RSocketOutboundGateway在RSocket协议中为响应流支持而实现。看到spring-integration-rsocket模块。

  • FluxAggregatorMessageHandler根据Project Reactor生成窗口或缓冲区结果Flux分组和窗口功能。

  • RateLimiterRequestHandlerAdvice和CacheRequestHandlerAdvice可用于AOP方面MessageHandler.handleMessage() 。

  • Kotlin脚本的JSR223实现已添加到spring-integration-scripting模块。

  • 现在，FTP和SFTP模块为某些Apache Mina FTP / SFTP服务器事件提供了事件侦听器。

  • 现在提供了简单的Apache Avro变压器。

  • 在整个项目中还有许多其他改进。

看到What’s New?有关详细信息和可能的重大更改，请参见参考手册和迁移指南 。

此外

维修5.1.8.RELEASE该版本也已发布，包括一些改进和错误修复。

的spring-integration-kafka:3.2.0.RELEASE也可用，并且它基于Spring Integration 5.2和Spring for Apache Kafka 2.3 。

另外，我们也想与您分享我们刚刚开始了Spring Integration Kotlin DSL的扩展项目。 Maven Central也提供预览版本：

compile "org.springframework.integration:spring-integration-kotlin-dsl:0.0.1.RELEASE"

有关更多信息，请参见GitHub Project页面 。

趁热还可以抓住它，欢迎您提供任何反馈。
