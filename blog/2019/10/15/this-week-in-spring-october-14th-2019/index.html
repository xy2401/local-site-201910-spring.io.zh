Spring 的这个星期-2019年10月14日

工程
乔什·朗（Josh Long）
十月15，2019

嗨， Spring 粉丝！哇！好一个星期！上周是疯狂的 SpringOne Platform 2019活动，我仍在恢复中！然后，我飞回家，在旧金山接待了Spring团队成员，Micrometer负责人和朋友Tommy Ludwig，并准备在星期一（在德国斯图加特）和星期二（在阿姆斯特丹）飞赴会议和参加用户组露面。

我准备好了，然后接到哥哥打来的电话，说我81岁的父亲身体不好（感谢大家的祝福！），我进入了Uber，去了旧金山机场。并被送往医院遗憾的是，我不得不取消在这些国家的现场演出。对于那些让我失望的人，我感到非常抱歉。能够远程进行一些会议，我感到非常高兴。感谢当我飞往洛杉矶与我父亲同住时，我的所有支持和支持。

无论如何，朋友们，本周还有很多东西要去！

  • Spring Cloud Stream-神秘化和简化
  • Spring Initializr 0.8.0现已发布
  • 令人垂涎的播客：疯狂的科学家安迪·克莱门特（Andy Clement），对AspectJ，SpEL，Eclipse，Graal等进行了研究
  • Azure Spring Cloud：在Kubernetes上运行Spring Boot应用程序的新方法
  • 希望看到一个真正令人信服的演示了举足轻重的PAS（A Cloud Foundry的发行版）？观看迪克的体育用品向您展示如何完成
  • Pivotal和Microsoft正在推出Azure Spring Cloud的预览版（荷兰文章）
  • 像往常一样，我喜欢在Spring Boot对Netbeans的支持中所做的更新
  • 我喜欢Jetbrains团队的Roman Elizarov的史诗般的解释器，涵盖了Kotlin和协程的所有方面
  • 使用Azure SignalR服务将实时添加到Java应用-DEV Community
  • 了解有关Azure Spring Cloud的更多信息并注册私有Beta
  • Spring Data JPA Projection对本机查询的支持
  • 上周，Syer博士，Juergen Hoeller和我在SpringOne Platform 2019的Azure Spring Cloud摊位上闲逛 。好玩。并以生产为中心。
  • 我喜欢乔·贝达和詹姆斯·沃特斯等人的访谈。请享用！
  • 我在今年的SpringOne平台上主持主题演讲非常开心 。谢谢大家，尽管我笑得很糟，但还是很有趣！我必须和一些我最喜欢的人一起出去玩 。
