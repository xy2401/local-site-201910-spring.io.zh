Spring Framework 5.2.0。M2、5.1.7、5.0.14和4.3.24现在可用

发布
斯特凡·尼科尔（StéphaneNicoll）
2019年5月10日

我很高兴代表团队和所有贡献者宣布Spring Framework 5.2.0.M2 ， 5.1.7 ， 5.0.14和4.3.24可从repo.spring.io现在以及Maven的中央。

首先，Spring Framework 5.2的第二个里程碑包括50多个修复和改进，具有改进的RSocket集成和对响应事务的初始支持（有关更多信息，请参见后续博客文章）。Spring Framework 5.1.7包含43个修复和改进 ，是所有5.x用户的建议升级。

最后，我们还发布了Spring Framework 5.0.14和4.3.24其中包含一些选定的修复程序。

一旦产品组合的其他版本可用，我们将在下周中旬发布Spring Boot对应版本，敬请期待！
