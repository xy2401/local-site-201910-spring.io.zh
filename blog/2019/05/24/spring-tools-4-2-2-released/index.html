    Spring Tools 4.2.2发布

    发布
    马丁·利珀特
    五月24，2019

    亲爱的春季社区，

    我很高兴地宣布适用于Eclipse，Visual Studio Code和Atom的Spring Tools 4的4.2.2版本。

    此版本的重点包括：

    • （所有语言服务器）性能：语言服务器启动时间的其他改进
    • （Spring Boot）新功能：导航Spring XML配置文件的Bean标识符，Bean类和属性名称
    • （Spring Boot）新增功能：内容助手推出了更多的Spring XML配置元素和属性
    • （春季启动）已修复：使用路径定义为常量而不是文字字符串（ ＃281 ）检测@RequestMapping
    • （CF清单）新功能：添加了对锚点和引用的支持
    • （Eclipse）新增功能：添加了与项目相关的XSD模式解析
    • （Eclipse）已修复：代码挖掘和实时悬停处于活动状态时，CPU和内存峰值很高（ ＃292 ）

    要下载Eclipse发行版并找到指向Visual Studio Code和Atom的市场条目的链接，请访问：

    • Spring Tools 4： https : //spring.io/tools/

    可以在这里找到详细的更改： https : //github.com/spring-projects/sts4/wiki/Changelog#2019-05-24-422-release

    Spring Tools 4.3.0计划于2019年6月下旬发布，并将基于Eclipse 2019-06。
