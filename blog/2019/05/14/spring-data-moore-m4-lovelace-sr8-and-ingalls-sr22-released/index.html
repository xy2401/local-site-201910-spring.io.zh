发布了Spring Data Moore M4，Lovelace SR8和Ingalls SR22

发布
马克·帕鲁奇
五月14，2019

我很高兴地宣布发布Spring Data版本Moore M4 ， Lovelace SR8和Ingalls SR22 。我们的发行版基于最新的Spring Framework发行版，并将分别由Spring Boot 2.2 M3、2.1.5和1.5.21所采用。

Moore M4也是Spring Data R2DBC 1.0 M2的先决条件。它附带70张固定的票 。其中有一些值得注意的新功能：

  • 支持MongoDB的反应式事务管理
  • 对MongoDB的基于注释的归类支持
  • Elasticsearch中的反应式索引操作，并通过查询删除

请找到我们发行版Wiki中已添加内容的高级概述。

由于Lovelace SR7和Ingalls SR22的故障，我们立即不得不分别发布Lovelace SR8和Ingalls SR22。Lovelace SR7和SR8共修复48张门票 。Ingalls SR21和SR22随附修复了31张票证 。Lovelace SR8和Ingalls SR22包含CVE-2019-3802的修复程序，强烈建议升级。
