适用于Apache Geode和Pivotal GemFire 1.0.0的Spring Boot。RC2发布！

发布
约翰·布鲁姆
五月01，2019

我很高兴地宣布发布适用于Apache Geode和Pivotal GemFire（SBDG）1.0.0的Spring Boot。RC2。

我想向Spring Boot团队致以特别感谢，以提供有关该项目的宝贵反馈。实际上，如果没有Spring Boot和Boot团队的出色努力，SBDG是不可能的。所以，谢谢Boot团队！

纳入反馈后，我决定推迟最终的GA，并再引入1个候选发布版本。最终1.0版GA（暂定）计划于5月6日星期一。

位（ org.springframework.geode:spring-geode-starter:1.0.0.RC2 ）可在Spring libs-milestone存储库中找到。

什么是新的

此版本包括以下更改：

  • 添加了一个新 示例，该示例显示了Spring Boot的Actuator对Apache Geode和Pivotal GemFire的支持

  • 将JCache API作为运行时依赖项包含在spring-geode-starter和spring-gemfire-starter为方便起见，在将Spring的Cache Abstraction与JSR-107（JCache API注释）一起使用时。

  • 删除排除在org.springframework.boot:spring-boot-starter-logging并且仅排除org.apache.logging.log4j:log4j-to-slf4j避免在Spring Boot和Apache Geode之间的日志依赖关系中发生冲突

  • 改善测试范围

  • 升级到Gradle 5.4.1

请参阅更改日志以获取详细信息的完整列表。

除了上述更改之外，该项目现在还包括一个Wiki页面 ，其中包含SBDG的当前版本以及所有必需的依赖项及其版本。

下一步是什么

最终GA！仅需要进行少量文档编辑和润色。

在1.0 GA之后，您可以期待SBDG 1.1.0.M1释放跟随。1.1将基于Spring Framework 5.1，Spring Boot 2.1，Spring Data Lovelace和Spring Session 2.1。

谢谢！
