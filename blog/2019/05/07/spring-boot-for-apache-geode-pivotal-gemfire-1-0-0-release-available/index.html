适用于Apache Geode和Pivotal GemFire 1.0.0的Spring Boot。发布可用！

发布
约翰·布鲁姆
五月07，2019

最后，我非常荣幸地宣布发布适用于Apache Geode和Pivotal GemFire （SBDG）的Spring Boot。 1.0.0.RELEASE ！

要感谢的人很长。

首先，请马克·波拉克（Mark Pollack）博士在项目开始之初就提供您真正的反馈。

感谢乔什·隆 （ Josh Long）先生在S1P-2018午夜之后与我一起坐了2个小时，以向我提供有关在Spring Boot中使用Apache Geode的反馈。

感谢Phil Webb先生和整个Spring Boot培训，以提供指导。

致少数贡献PR的社区成员 。

最后，在这个项目开始之际，感谢其他所有人的支持和耐心。

\ o / .oO（胜利！）

SBDG 1.0 GA 位在Maven Central中可用。

什么是新的

SBDG 1.0 GA版本包括：

  • 抛光参考文档 。

  • 后备缓存模式的新样本以及指南和示例应用程序代码。

  • 升级到Apache Geode和Pivotal GemFire （STDG）的Spring Test 0.0.1.RELEASE 。有关此新项目和即将进行的项目的更多详细信息，请参见此处 。

有关更改的完整列表，请参见changelog 。

下一步是什么

现在有了1.0 GA，所有注意力都转移到了1.1。

SBDG 1.1将基于：

  • Spring Framework 5.1

  • Spring Boot 2.1

  • 春天数据Lovelace

  • 春季会议2.1

的1.1.0.M1即将发布并宣布。

我们始终赞赏并欢迎对新发布的1.0 GA的反馈。
