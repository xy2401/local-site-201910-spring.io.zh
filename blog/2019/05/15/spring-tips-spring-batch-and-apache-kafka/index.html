春季提示：Spring Batch和Apache Kafka

工程
乔什·朗（Josh Long）
五月15，2019

嗨，春季粉丝！欢迎使用Spring Tips的另一部分（@SpringTipsLive） ！在本期中，我们将介绍Spring Batch中刚刚登陆的社区，其中增加了对Apache Kafka的支持。这项新的支持为桥接批处理和流式工作负载提供了诱人的机会。使用Kafka将大量写入工作转移到后端仓库，并使用Spring Batch耗尽数据流。从现有数据存储中加载大量数据，并将其集中到流处理管道中。当您考虑使用Spring Integration ， Spring Cloud Stream ， Spring Cloud Stream Kafka Streams和Spring Cloud Data Flow时，可能性变得更加有趣。如果您想快速了解Spring Batch基础知识，您可能会喜欢Spring Batch from- gasp 上的Spring Tip系列 ！ -两年前！
