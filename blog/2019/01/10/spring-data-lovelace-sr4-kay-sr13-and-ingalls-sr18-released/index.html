发布了Spring Data Lovelace SR4，Kay SR13和Ingalls SR18

发布
马克·帕鲁奇
一月10，2019

我谨代表Spring Data团队宣布Lovelace SR4，Kay SR13和Ingalls SR18维护版本的可用性。今天的发行版包含了最新的Spring Framework维护发行版：分别是5.1.4、5.0.12和4.3.22。为了方便起见，即将发布的Spring Boot版本（2.1.2、2.0.8和1.5.19）将选择Lovelace SR4，Kay SR13和Ingalls SR18。

该服务版本附带错误修复和依存关系升级，总共解决了170张票证。您可以通过以下链接找到票证的完整列表：

  • Lovelace SR4
  • 凯SR13
  • 英格尔斯SR18

Spring Data 1.x和2.0.x生命周期终止（EOL）

所有美好的事物都必须结束。对于 Spring Data 1.x （英语）和2.0.x （Kay）行，我们将停止与Spring Framework和Spring Boot保持一致的维护。寿命终止日期是：

  • Spring Data 1.x（英文）：2019年8月。
  • Spring Data 2.0.x（Kay）：2019年3月。

Spring Data用户应做出相应计划，以确保在该日期之前已升级到最新的2.1版本。请注意，Spring Data 2.0及更高版本需要Java 8+。
