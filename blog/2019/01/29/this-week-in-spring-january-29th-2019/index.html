春天的这个星期-2019年1月29日

工程
乔什·朗（Josh Long）
一月29，2019

嗨，春季粉丝！你能相信我们已经凝视着二月了吗？我仍然在休假中摆脱假期。我在太平洋标准时间（PST）的午夜过后几分钟，就在SFO机场，等待登机飞往休斯敦的航班，这是前往北卡罗来纳州夏洛特（Charlotte）的航班，这是史诗般的新春之行，这是新的一年的首次！我期待看到你们所有人！

无论如何，我们还有很多事情要做，让我们开始吧！

  • Spring Tools 4.1.1发布 -此新版本包含大量功能，请不要错过！
  • 听：在我的最新播客“ A Bootiful Podcast”中 ，我采访了Spring Boot传奇人物Madhura Bhave 。不要错过！
  • 观看：在上周的Spring Tips中，我看了用反应式对冲客户请求WebClient以及 Spring Cloud支持的服务注册表 DiscoveryClient抽象。
  • Spring IO PlatformCairo-SR7
  • Spring IO PlatformBrussels-SR16
  • 春天云格林威治。现在可以发布
  • IBM Pivotal的新工具为Java EE WebSphere Apps迁移到Cloud Foundry和Kubernetes铺平了道路
  • 使用Keycloak保护Spring Boot Admin和执行器端点
  • 这看起来很有趣： 通过Spring Data Projections（DTO）仅从数据库中获取所需的数据， 并通过虚拟属性丰富结果
  • NewStack在Spring Cloud GCP 1.1上有一篇不错的文章-请查看！ ）
  • Cloud Foundry基金会目录Abby Kearns 预测2019年趋势 ）
  • JUnit 5.4 RC1现在可用。请踢轮胎，并在5.4 GA前给我们反馈！5.4 RC1还包括@RegisterExtension订购null和空参数来源@ParameterizedTest方法， @TempDir现场注入
  • 我喜欢传奇的Kubernetes 社区成员Paul Czarkowski撰写的系列文章， 为Spring开发人员介绍Kubernetes。
  • 《软件工程日报》播客与Ryland Degnan一起在反应式流服务网络协议RSocket上播出了精彩的一集。Spring Framework 5.2也旨在集成RSocket。
  • 这是《 The New Stack》中一个有趣的播客片段，内容涉及大规模使用Cloud Foundry的人们 -请查看。
