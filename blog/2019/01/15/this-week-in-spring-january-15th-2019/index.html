春天的这个星期-2019年1月15日

工程
乔什·朗（Josh Long）
一月15，2019

嗨，春季粉丝！欢迎来到春季本周的另一期。我刚从加拿大蒙特利尔回来，明天我要去德克萨斯州达拉斯市进行客户访问。一年中的这个时候以明显的气候变化和对新的一年的乐观为特征。我喜欢它。在今天的综述中，这种乐观情绪是显而易见的。我们有很多内容需要介绍，让我们开始吧！

  • 您知道，我永远都不知道在这个美好的社区中会有什么人会变得有趣！出于这个原因，我尝试在Spring Tips系列中涵盖尽可能多的主题。最新一期已证明很受欢迎！截至撰写本文时，几乎有100人顶过YouTube。在本文中，我研究了Apache 2许可的Flowable BPMN和CMMN引擎的业务流程管理。
  • Spring Framework 团队刚刚从JIRA迁移到Github问题。这并不容易。在这里阅读以了解发生了什么以及如何发生
  • 发布了Spring Session Bean-SR2，Apple-SR8和1.3.5
  • Spring Cloud Task 2.1.0。RC1现在可用
  • 我将继续我的第六部分系列，探讨如何将Spring与Azure结合使用。这是六部分中的第三部分，介绍了如何使用CosmosDB扩展数据访问 。在昨天发布的第四部分中，我们研究了与Azure Service Bus的消息传递和集成 。
