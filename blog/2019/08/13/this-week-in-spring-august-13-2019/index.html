春天的这个星期-2019年8月13日

工程
乔什·朗（Josh Long）
2019年8月13日

嗨，春季粉丝！欢迎来到春季本周的另一期！我刚刚完成了我一生中最神奇的经历之一-奥赖利（O'Reilly）的“ Foo Camp”，这是一次非正式会议，邀请来宾可以在奥赖利媒体校园总部扎营并互相交谈。在不计其数的其他人中，我遇到了前加利福尼亚州州长杰里·布朗，爱沙尼亚前总统托马斯·亨德里克·伊尔维斯等等。现在，我在德克萨斯州，与我们在奥斯丁，圣安东尼奥，达拉斯和普莱诺的出色客户交谈。我也明天晚上在达拉斯MUG上演讲，如果您在周围并且想出来闲逛。

整整一周将是一个有趣的内容，所以让我们开始吧！

  • 我希望您喜欢上周的“ A Bootiful Podcast”，其中我采访了DevOps，云计算，流程等方面的John Willis。
  • Spring Statemachine从3.0.0开始变得活跃。M1
  • 查看消息传递项目的新版本：Spring AMQP（用于RabbitMQ的Spring），Spring Integration和用于Apache Kafka的Spring。
  • 春季会议Corn-M3和Bean-SR7发布
  • 我爱我们的社区 。感谢你们每个人，他们帮助使Spring生态系统变得如此惊人。
  • 我喜欢这部影片，您也可能喜欢。干得好Maciej！
  • 抢先体验Neo4j 4.0的Alpha版本，并尝试可与RxJava2和Project Reactor一起使用的反应式Java驱动程序，抢购Spring Boot入门，甚至尝试在此文章上尝试新的反应式Spring Data Repository
  • 想要为Spring Cloud Kubernetes做出贡献吗？ 这是一个很容易解决的问题，我们希望您能提供一些帮助
  • 提醒一下，Spring Cloud Edgware（与Spring Boot 1.5.x兼容）已在本月初停产。
  • 康卡斯特正在招聘！
