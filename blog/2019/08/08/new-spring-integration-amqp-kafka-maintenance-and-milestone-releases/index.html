此外，下一代的里程碑4（以及之前未宣布的里程碑3）也可用：

Spring Integration

  • 5.1.0。M4

  • 5.1.0。M3

  • 什么是新的？

这些里程碑中的亮点

  • RabbitMQ-支持用户端批处理（ List来自多个源消息或生产者批次的有效负载）

  • 多页参考手册可加快浏览器加载时间

  • 控制总线支持Pausable组件

  • Flux基于基础的聚合器

  • 对入站HTTP组件的验证支持

  • 新的标头聚合策略可自定义聚合器行为

  • 改进的异常信息，使识别故障发生的位置更加容易

  • Apache MINA服务器的FTP / SFTP服务器应用程序事件发布者
