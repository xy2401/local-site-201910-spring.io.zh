是时候了-再见Spring Boot 1.x！

工程
Madhura Bhave
2019年8月6日

如去年的公告中所述，Spring Boot 1.x产品线已经到了使用寿命。今天早些时候发布的Spring Boot 1.5.22将是1.x中的最新版本。线。

我们希望大多数用户已将其应用程序升级到Spring Boot2.x。对于尚未迁移的用户，我们强烈建议尽快进行迁移。迁移指南中提供了有关如何从1.x迁移到2.x的详细说明。为了获得平稳的升级体验，我们建议分阶段升级，而不是直接跳转到最新的稳定版本。从最新的1.5.x版本升级到最新的2.0.x版本完成后，您应该升级到2.1.x行中的最新版本。

如果您对迁移过程有任何疑问，请加入Gitter 。如果您认为迁移指南中缺少某些内容，请随时在我们的问题跟踪器中提出问题 。
