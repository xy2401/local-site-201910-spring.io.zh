Spring Security 5.2.0。M4发布

发布
菲利普·汉尼克（Filip Hanik）
2019年8月5日

我很高兴代表社区宣布Spring Security 5.2.0的发布。M4！您可以在变更日志和以下重点中找到完整的详细信息：

OAuth 2.0

gh-6811 for Servlet-介绍OAuth2AuthorizedClient管理器/提供程序
gh- 6886-未获取自定义声明的OpenID Connect用户信息
gh- 7033-添加资源服务器JWE示例
gh-7034 -Nimbus Jwt解码器不应强制执行SignedJWT

核心

gh-6506-添加通用AuthenticationFilter
gh-5300-允许配置CSRF的SessionAuthenticationStrategy
gh-5557-用于HTTP安全的DSL嵌套生成器
gh-7082-添加繁体中文本地化消息
gh- 7042-允许在不同的BCrypt编码之间升级
gh- 7057-允许在不同的SCrypt编码之间升级
从之前的5.2.0。M3版本-添加nohttp来构建

有关nohttp项目的更多信息，请参见此博客文章 。

网页

gh- 4310-向StrictHttpFirewall添加了主机名功能
GH-7168 - BASIC认证情况下灵敏度

Web助焊剂

gh-7107-在DSL中为响应式应用程序支持嵌套生成器
gh-7113-支持残疾人

依赖关系和CI

gh-7147-升级外部依赖项
travis- ci-在Travis CI上使用OpenJDK8进行构建
