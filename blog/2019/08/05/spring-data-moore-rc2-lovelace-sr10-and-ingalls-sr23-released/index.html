发布了Spring Data Moore RC2，Lovelace SR10和Ingalls SR23

发布
马克·帕鲁奇
2019年8月5日

我很高兴代表团队宣布Spring Data发布了Moore RC2，Lovelace SR10和Ingalls SR23。所有发行版都采用了最新的Spring Framework发行版，并将分别由Spring Boot 2.2 RC1、2.1.7和1.5.22所采用。

在许多依赖项升级中，我们迁移了几乎所有的Spring Data核心项目以使用AssertJ断言。根据我们先前的声明 ，Ingalls SR23是Spring Data 1.x产品线的最后一个发布版本，该产品即将到期 。我们强烈建议所有仍使用Spring Data 1.x的用户迁移到Spring Data Lovelace SR10。

该发行版总共发行了320张固定票。Moore RC2最显着的新功能（包括其他功能）是：

  • Geode / GemFire的网关发送者的基于注释的配置。
  • 适用于MongoDB的流畅的反应式ChangeStream API。
  • MongoDB聚合的基于类型的字段投影派生。
  • Cassandra删除查询的查询派生。
  • 升级到Apache Solr 8.2。
  • 升级到Elasticsearch 6.8.1。

我们正在努力争取在9月中旬发布最终的GA版本之前升级Spring Data Moore RC3（将于9月到期）。
