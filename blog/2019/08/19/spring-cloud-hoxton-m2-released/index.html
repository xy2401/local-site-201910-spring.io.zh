Spring Cloud Hoxton。M2发布

发布
瑞安·巴克斯特（Ryan Baxter）
2019年8月19日

我很荣幸地代表社区宣布， Spring Cloud Hoxton发布系列的里程碑2（M2）现已上市。该版本可以在Spring Milestone存储库中找到。您可以查看Hoxton 发行说明以获取更多信息 。此外，您可以查看GitHub Project for Hoxton。M2 。

Hoxton发布系列的重大变化

Spring Cloud Hoxton。M2建立在Spring Boot 2.2.0之上。M5。

文件变更

霍克斯顿。M2文档有一个新的登录页面以及一个新主题。登陆页面会将您链接到您感兴趣的特定项目的文档。

我们的文档有一个新主题。我们希望您发现新主题使文档更易于使用。

新的负载均衡器实施

Spring Cloud Hoxton。M2是第一个同时包含阻塞式和非阻塞式负载均衡器客户端实现的版本，以替代已进入维护模式的Netflix Ribbon。

要将新的BlockingLoadBalancerClient与RestTemplate一起使用，您需要在应用程序的类路径中包含org.springframework.cloud:spring-cloud-loadbalancer 。使用@LoadBalanced WebClient.Builder
