Spring 的这个星期-2019年9月17日

工程
乔什·朗（Josh Long）
九月17，2019

嗨，春季粉丝！欢迎来到春季本周的另一期-现在是一周！我正在参加史诗般的年度eventapalooza ，它是Oracle CodeOne（以前的JavaOne ）。我今天刚刚结束了关于反应式编程的演讲，下周三我将在Kotlin上发表另一个演讲。不要错过！

同时，本周还有很多事情要做，让我们开始吧！

  • Apache Geode和Pivotal GemFire 1.1.1的Spring Boot。发布＆1.2.0。M2可用！
  • 如果您在本周收听一个播客，请与Spring Framework联合创始人一起听这期最新，最伟大的A Bootiful Podcast ，并由Juergen Hoeller领导Spring Framework 5.2（开源），决定新版本的主题，他的作为Spring Framework的负责人的传奇之旅，等等 。
  • Spring Cloud 格林威治。SR3发布
  • Spring Vault 2.2 M1现在可用
  • 在上周史诗般的JavaZone 2019活动中查看Reactor队友Simon Basle在RSocket上的演讲
  • Spring Initializr-开始点spring dot io-现在支持黑暗模式 ！
  • 我真的很喜欢API网关技术（包括Spring Cloud Gateway）的技术比较
  • 我希望看到Flyway支持R2DBC，这是我们的反应式SQL数据存储访问SPI。你不是吗
  • Pivotal与阿里巴巴和Netifi等组织一起发起了Reactive Foundation ， 这是非常好的，Reactive Foundation的第一个项目将是RSocket。
  • FreeCodeCamp博客在Spring Boot上有一个有趣的外观教程
