春天的这个星期-2019年9月10日

工程
乔什·朗（Josh Long）
2019年9月10日

嗨，春季粉丝！欢迎来到春季本周的另一期晴天！您可以相信我们现在盯着2019年第四季度吗？时光飞逝！

在撰写本文时，我坐在挪威奥斯陆的Radisson Blu旅馆里，参加史诗般的JavaZone表演。我在这家酒店得到吨美好的回忆，经常与我的Spring团队的朋友，过了很多年享受。我喜欢这个节目，很高兴自2015年以来第一次回来！

我今天登陆，录制了几集Spring Tips （新赛季即将开始！），打了个不合时宜的午睡，现在我们来了！我们有很多工作要做，让我们开始吧！

  • Spring Boot 2.2.0。M6
  • Apache Geode和Pivotal GemFire 2.2.0的春季会议。M4可用！
  • Spring Data Moore RC3发布
  • Spring Security 5.2.0。RC1发布
  • Spring Cloud Data Flow 2.3.0。M1和2.2.1 GA发布
  • 令人垂涎的播客：加里·罗素
  • Spring Boot 2.1.8现在可用
  • Spring HATEOAS 1.0.0。RC2发布
  • Spring Framework 5.2 RC2现在可用
  • 随着Kubernetes赌注的回报，Cloud Foundry的开发人员体验翻了一番
  • 提醒您，Cloud Foundry是目前最大的开源项目之一
  • RabbitMQ中的本月
  • 将Spring Security与Azure Active Directory结合使用
  • Spring Data JDBC现在可以从Spring Initializr中获得-欢呼！
  • 很高兴看到我们的Micrometer项目发挥作用，因此在本文有关微服务指标的文章中发挥了关键作用
  • 这是一篇关于如何使用Spring Cloud Gateway编写自定义过滤器的很棒的文章
  • 我们正在取得很大进步！您是否知道我们的Spring Framework文档现在在“核心”部分中提供了适用于所有内容的Java 和 Kotlin示例？
