春天的这个星期-2019年9月24日

工程
乔什·朗（Josh Long）
九月24，2019

嗨，春季粉丝！欢迎阅读《 春季本周》的另一期， 这是每周的综述，在其中我可以浏览生态系统中的重要内容并学习事物，并希望与大家分享有趣的花絮。我将日历标记为Springs本周的数目，因此想象一下我高兴地意识到日历确实接近史诗般的SpringOne Platform 2019 ！我简直不敢相信！

该活动即将于2019年10月7日至10日在德克萨斯州奥斯汀举行。那指日可待，但是我仍然有一些地方可以到达。我在西班牙马德里参加史诗般的CodeMotion Madrid演出。然后去墨西哥瓜达拉哈拉参加墨西哥Java表演日。然后前往台湾台北，参加由台湾JUG领导的JCConf。然后，终于到了烧烤和Spring之地SpringOne Platform！我等不及要见你们了。

（希望到那时，我将在《 Reactive Spring》一书中再写一章！）

我们暂时不愿拖延：让我们开始吧！

  • Java 13发布了！多行字符串，提高了性能，主要是重写Socket API等。立即获取位 ！如果您使用的是macOS，并且拥有尊贵的sdkman安装后，也可以使用sdk install java 13.0.0-open 。
  • 当我们欢迎Java 13时， 一个令人愉快的播客：关于Java模块化的Nicolai Parlog
  • Spring Tools 4.4.0发布
  • 受Spanner启发的Yugabyte DB刚刚发布了他们的2.0版本 ，甚至还有一个Spring Data模块，他们正在致力于R2DBC的支持！
  • 我喜欢这个对Spring Data MongoDB文档的贡献之路的故事
  • 软件工程电台在Spring Boot的第381集中采访了我的播客
  • Spring Data负责人Mark Paluch的警告： 我们正在准备@ R2DBC 0.8.0。RC1下周下半年发布。关注GitHub里程碑以了解我们的发布时间表
  • 我喜欢Netifi上有关gRPC和RSocket之间的区别的帖子-Netifi-中
  • 上周在Oracle CodeOne社区的主题演讲中，这真是太有趣了。
  • 关于使用Reactor Blockhound查找阻塞线程的代码的不错的文章
  • 巧妙的Toshiaki Maki在Spring Boot中增加了对8位ASCII艺术的支持。所以，您正在升级，对吗？
  • 您是否知道，除了最近在Spring Initializr上添加的“暗模式”之外，还有一种“共享”配置以便其他人可以单击链接并获得相同配置的方法吗？
