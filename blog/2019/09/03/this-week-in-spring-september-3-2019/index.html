Spring 本周：2019年9月3日

工程
乔什·朗（Josh Long）
九月03，2019

嗨， Spring 粉丝！欢迎来到 Spring 本周的另一期！我们刚刚在美国旧金山长家中的劳动节庆祝活动结束。这是与家人和朋友度过的愉快的一天，我不确定是否还能再有其他的乐趣，但那是星期二，您知道这意味着什么： Spring 本周的另一部史诗般的部分。我不敢相信我们已经在9月，已经盯着一年的最后几个月了！像往常一样，我明年的旅程中最喜欢的部分之一就是史诗般的SpringOne平台秀的到来。今年也不例外：我迫不及待地想与德克萨斯州史诗般的奥斯丁的朋友们一起庆祝演出。你会在那儿吗？让我知道！

  • 我真的很喜欢上周的A Bootiful Podcast ，这是Spring Cloud Data Flow传奇人物Ilayaperumal Gopinathan的专访
  • 查看Spring Data Neo4j RX的精彩介绍
  • 关于DZone微服务的一篇不错的文章，关于带有OpenAPI和Spring Cloud Contract的微服务的合同
  • DZone Java有一个Spring Framework 注释指南
  • 我也很喜欢 Spring 自动接线概念的介绍。
  • 这很有趣：Spring用户很快将能够通过阿里云上的Netifi托管服务使用RSocket。
  • 我喜欢 Matt Cloud的Spring Cloud Gateway的安全反应微服务
  • 这是Emmanouil Gkatziouras在Dzone上发表的有趣文章，内容涉及如何向Spring Data存储库添加自定义功能
  • 这是一个关于使用Spring Boot创建状态机的有趣的文章，但不是有关 Spring Statemachine的内容。这篇文章很有趣，但是我不能完全说服他们无法使用Spring Statemachine完成他们所需的工作。无论哪种方式，有趣的阅读。
  • 朱利安·杜波依斯（Julien Dubois）发布了关于使用R2DBC 与Microsoft SQL Server进行动态对话的非常漂亮的外观
