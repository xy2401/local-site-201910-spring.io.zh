Spring Roo 1.3.2 RC1维护版本现已发布

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2015年8月11日

我很荣幸地代表DISID Corporation的Spring Roo团队宣布
Spring Roo 1.3.2 RC1发布的候选版本已经退出。

Spring Roo是用于Java的快速应用程序开发工具，
允许您在短短几分钟内创建完整的企业应用程序。

Spring Roo jar文件已发布到Maven Central。
与往常一样，该发行版在Spring IO存储库中可用。

候选版本修复了以下问题：

  • 在ITD生成ROO-3583上添加InnerType构造函数时出错
  • JavaBean实现定义吸气剂和吸气剂的接口ROO-3584
  • JavaBean实现接口并覆盖导致空指针的getter ROO-3585
  • 更改语言时要求宽松的附加参数ROO-3595
  • 添加支持以生成通用方法ROO-3648
  • 更新Selenium以支持更高版本的Firefox ROO-3634
  • 在表单字段标签ROO-3639中添加renderLabel属性
  • 实现与getId（）的接口的JpaActiveRecord实体：双代ROO-3587
  • 更新Tomcat7 Maven插件版本ROO-3652
  • 生成命令“全部硒” ROO-3636
  • 添加支持以生成通用方法ROO-3648

春季数据发布培训高斯林的第一个发布候选版本

发布
奥利弗·德罗博姆
2015年8月4日

我很高兴代表Spring Data团队宣布Spring Data发布火车Gosling的第一个也是最后一个发布候选版本的可用性。该版本发布了121个修复程序和增强功能，最值得注意的是以下功能：

  • Spring MVC和Spring Data REST的Querydsl集成（有关详细信息，请参见示例 ）。
  • 改进了与Spring Framework 4.2和Spring Boot开发工具的兼容性。
  • Spring Data JPA现在公开了一个JpaContext轻松获得EntityManager每个域类型。
  • RepositoryRestConfigurer在Spring Data REST中减少侵入性的配置。
  • Spring Data Redis使用Mark Paluch维护的生菜叉。
  • 支持ReadPreference在Spring Data MongoDB聚合中。
  • 在Spring Data Solr中支持范围方面。
  • 依赖性升级到Spring Framework 4.1.7，Guava 18和Jackson 2.6。
