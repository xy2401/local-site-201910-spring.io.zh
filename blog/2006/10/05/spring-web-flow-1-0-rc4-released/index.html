Spring Web Flow 1.0 RC4发布

发布
基思·唐纳德
2006年10月5日

亲爱的春季社区，
 
我们很高兴地宣布， Spring Web Flow 1.0 RC4已发布。
 

 
下载 | 说明文件 | 变更日志 | 升级指南

Spring Web Flow是Spring社区的产品，专注于Web应用程序中用户界面流程的编排。

此版本包含许多改进和一些令人兴奋的新功能。我们认为它是迄今为止最稳定的发行版，最后是使Spring Web Flow 1.0最终路线图功能完整的发行版。Spring Web Flow 1.0的最终版本将在下周发布，并且变化不大。从现在开始，我们建议您测试1.0 RC4，以帮助在1.0大版本发布之前发现所有剩余的问题。

请注意，此版本中有一些影响用户的更改。1.0 RC3或更早的用户应查看升级指南 ，其中详细概述了这些更改。

入门

开始使用Spring Web Flow的最佳方法之一是查看并演练示例应用程序。我们建议您检查所有样品，并从一开始就根据需要补充参考手册材料。该发行版附带十个示例应用程序，每个示例应用程序都展示了一组独特的产品功能。这些样本是：

  1. 电话簿-演示大多数功能（包括子流）的原始样本
  2. Sellitem-演示具有条件转换，流执行重定向，自定义文本字段格式和延续的向导
  3. Flowlauncher-演示启动和恢复流程的所有可能方法
  4. 项目清单-演示REST风格的URL和内联流
  5. Shippingrate-与Ajax技术一起展示Spring Web Flow
  6. NumberGuess-演示有状态Bean，评估操作以及“单键”流执行重定向。
  7. 生日-演示Struts集成
  8. Fileupload-演示分段文件上传，设置操作和Flash作用域
  9. Phonebook-Portlet-Portlet环境中的电话簿示例（请注意流定义如何保持不变）
  10. Sellitem-JSF-JSF环境中的Sellitem示例

要构建示例应用程序以进行快速评估，只需：

  1. 解压缩spring-webflow-1.0-rc4.zip版本档案
  2. 进入projects / spring-webflow / build-spring-webflow目录
  3. 执行“ ant dist”目标。
  4. 有关每个样本的可部署.war文件以及扩展的war目录，请参见“ target / artifacts”目录。
请参阅发行版readme.txt和projects / spring-webflow / spring-webflow-samples / readme.txt分别获得有关发行档案内容和示例的更多信息。

所有示例项目都是可直接导入Eclipse的Spring IDE项目。
