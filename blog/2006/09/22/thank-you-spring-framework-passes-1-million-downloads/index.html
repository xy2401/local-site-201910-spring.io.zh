谢谢！Spring Framework下载量突破一百万

工程
罗德·约翰逊
2006年9月22日

几周前，Spring Framework项目从其SourceForge的家中下载了100万次下载。实际的总数可能要高得多，因为该数字不包括夜间构建或可以从中下载Spring的其他站点。而且，当然，Spring包含在越来越多的其他产品的发行版中。然后是Spring.NET …

最重要的是，Spring继续保持增长势头：数字正在迅速增长。Spring的下载最多的版本是最新的生产版本1.2.8，该版本已下载175,000次，占总下载量的17％以上。以这种速度，我们将实现下百万次下载的速度比前100万次快得多！

这只是向前迈出的重要一步。随着Spring 2.0最终版本将于9月底发布，以及Spring Web Flow的最终版本大约在同一时间发布，未来看起来会更加令人兴奋。我对在Spring 2.0上进行工作或在其中进行培训感到特别满意（当然也要在其中进行工作），并且很高兴看到AOP增强功能和XML扩展机制等新功能如何激发用户的生活并使他们过上美好的生活更轻松。自从2003年底发布Spring 1.0最终版以来，我对前方的技术可能性并不感到兴奋。

Spring 2.0版本还包含一些功能，这些功能将已经丰富的Spring生态系统提升到新的水平。除其他事项外：

  • 它是Pitchfork项目的基础，该项目在WebLogic的下一版本中用于提供Java EE 5.0注入和侦听功能，并为Java EE组件添加Spring功能，例如更强大的注入和真正的AOP。Spring 2.0表面之下的许多功能（例如能够向bean定义中添加任意元数据以及用于非常高级使用的新容器回调的功能）使Spring 2.0更加强大，可作为构建其他框架的基础。
  • 我们已经完成了即将到来的Spring OSGi集成的后台工作，这将增强Spring作为服务器基础结构的基础的价值，并在组件化，版本控制和动态部署领域为用户带来好处。

我们还将很快宣布Spring Acegi Security将以Spring Security的名称进入Spring家族的心脏。同样，Spring核心的新功能将使用户受益。2.0 XML模式扩展机制将极大地简化安全性配置，从而使这款出色的产品更易于使用。

Spring团队为此付出了很多努力，但是如果没有您，这一切都不会发生！

我想借此机会向社区表示非常感谢。开源项目与我们的用户有着特别密切的关系，Spring社区的实力是Spring Framework作为软件的成功的重要组成部分。不断提供反馈，补丁和问题，我们将一如既往地倾听并竭尽所能。
