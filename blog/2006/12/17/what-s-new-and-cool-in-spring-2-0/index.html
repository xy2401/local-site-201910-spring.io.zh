Spring 2.0中有哪些新功能和新功能？

工程
本·亚历克斯
2006年12月17日

上个月，罗德·约翰逊（Rod Johnson）在三届澳大利亚春季用户小组会议上作了题为“春季2.0的新功能和新功能”的会议。罗德说我会做他的介绍中可用的那些会议期间提及，所以在这里它是 。

人们最近也通过电子邮件向我发送其他一些演示文稿。以下是最新的顺序（无特定顺序）：


  • 如果您正在寻找Rod的另一个会话“使用AOP简化企业应用程序”，则可以直接从Sydney Java User Group网站下载它 。

  • 我在澳大利亚的两次Java企业会议上发表的面向对象的（ROO）演示都不会发布。这些幻灯片已被我在The Spring Experience上进行的最近一次的详细会议所取代。

  • 我的“ 快速跳进真正面向对象（ROO）应用 ”从春天体验会议将在这个博客在短期内公布。我目前正在制作与之配套的视频。

  • 我的“ 超越低端的成果：域访问控制列表 ”部分的内容还将介绍Acegi Security 1.0.3的新ACL功能。同时，Acegi Security SVN现在包含示例应用程序（ SEC-415 ）。


对于那些参加演示的人，我希望您喜欢它们。
