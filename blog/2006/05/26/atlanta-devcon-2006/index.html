亚特兰大DevCon 2006

工程
本·黑尔
2006年5月26日

我刚刚在2006年亚特兰大DevCon上完成了Spring 2.0：新的和值得注意的演讲。首先让我说这次会议很棒。该网站和组织者都是一流的。我要特别鸣谢伯特·萨特（Burr Sutter）参加一次会议。您知道当会议中心没有无线网络时，一切都会好起来的，但是您可以从隔壁的咖啡厅那里获取无线网络。那是很好的业力！JUG成员都很有知识（甚至是不了解Spring的人），并提出了很好的问题。我提出了有关EJB 3，JPA，ESB，Spring学习曲线，Java 5和Spring Web Flow的问题。每个人都是一个好问题。

对于那些无法做到的人，我将以几种格式发布该演讲的幻灯片。我已决定为所有其他Mac用户发布Keynote文件，以及PDF和Keynote创建的非常酷的导出HTML 。

随意使用注释向我询问Spring 2.0的一些新功能和值得注意的功能。
