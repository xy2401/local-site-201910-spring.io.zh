Spring LDAP 1.1发布

发布
马蒂亚斯·阿瑟森（Mattias Arthursson）
2006年8月28日

亲爱的 Spring 社区，

我们很高兴地宣布，Spring LDAP 1.1版已经发布。这是该库作为Spring Framework子项目的第一个发行版。下载 | 说明文件 | API

关于Spring LDAP
Spring LDAP是一个Java库，用于基于Spring的JdbcTemplate模式简化LDAP操作。该框架减轻了用户查找和关闭上下文，遍历NamingEnumerations，编码/解码值和过滤器等的负担。

LdapTemplate类封装了传统LDAP编程中涉及的所有管道工作，例如创建，遍历NamingEnumerations，处理Exception和清理资源。这使程序员可以处理重要的事情-在哪里查找数据（DN和筛选器）以及如何处理数据（与域对象进行映射，绑定，修改，取消绑定等），方法与JdbcTemplate减轻负担的方式相同除了实际的SQL之外的所有程序员，以及数据如何映射到域模型。

除此之外，Spring LDAP还提供了从NamingExceptions到DataAccessExceptions的Exception转换，以及一些用于使用过滤器，LDAP路径和Attributes的实用程序。

Spring-LDAP需要J2SE 1.4。构建需要J2SE 1.4。运行该示例需要J2EE 1.4（Servlet 2.3，JSP 1.2）。

从哪儿开始
从上面的链接下载发行版。该发行版包含大量的JavaDoc文档以及完整的参考文档和一个示例应用程序，该示例应用程序说明了使用Spring LDAP的不同方法。

历史
