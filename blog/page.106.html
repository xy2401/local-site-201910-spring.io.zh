SpringOne Platform 2016重播：编写自己的Spring Boot Starter

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
一月3，2017

记录在SpringOne Platform 2016中。
演讲者：Dieter Hubau
幻灯片： http ： //www.slideshare.net/SpringCentral/writing-your-own-spring-boot-starter

任何积极使用Spring的人现在都必须已经听说过Spring Boot。它是从新的Spring项目开始的当前事实上的标准工具，并且在未来几年中将会越来越流行。在start.spring.io上易于使用的界面和非常好的文档旁边，有大量的所谓的Spring Boot Starters，开发人员可以使用它们来快速启动他们的开发。尽管自动配置和启动器通常似乎是来自恩多森林月亮最深处的魔法赋予的生物……却没有。实际上，任何人都可以编写自己的Spring Boot Starter。在演示过程中，我将解释Spring Boot Starter项目的内部工作原理，并介绍创建自己的代码。我将使用最近开发的入门程序将Spring Social与Redis支持的数据存储区集成。在会议结束时，听众将知道Spring Boot的自动配置和入门项目不是神奇的独角兽，而是易于理解的Spring Boot生态系统中的组件。他们还将学习如何创建自己的Spring Boot Starter项目，并希望他们会积极参与Spring Boot项目。

Spring AMQP 1.7 RC1现已上市

发布
阿尔特姆·比兰（Artem Bilan）
2016年12月28日

我谨代表Spring Integration团队宣布Spring AMQP 1.7的第一个（也是最后一个）发行候选版本，该版本可在Milestone资料库中找到 。

此发行版主要是介于1.6.x和2.0之间，用于实现Spring Boot 1.5和IO Platform Brussels兼容性。

无论如何，这里有一些改进和新功能可以解释：

  • 进行适当的RabbitConnectionFactoryBean更改升级到Amqp Client 4.0

  • 升级到Log4j 2.7，并修复log4j2.AmqpAppender
