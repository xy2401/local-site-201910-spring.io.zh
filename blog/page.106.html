新闻与活动
彼得·汉弗莱（Pieter Humphrey）
一月3，2017

记录在SpringOne Platform 2016中。
演讲者：Dieter Hubau
幻灯片： http ： //www.slideshare.net/SpringCentral/writing-your-own-spring-boot-starter

任何积极使用Spring的人现在都必须已经听说过Spring Boot。它是从新的Spring项目开始的当前事实上的标准工具，并且在未来几年中将会越来越流行。在start.spring.io上易于使用的界面和非常好的文档旁边，有大量的所谓的Spring Boot Starters，开发人员可以使用它们来快速启动他们的开发。尽管自动配置和启动器通常似乎是来自恩多森林月亮最深处的魔法赋予的生物……却没有。实际上，任何人都可以编写自己的Spring Boot Starter。在演示过程中，我将解释Spring Boot Starter项目的内部工作原理，并介绍创建自己的代码。我将使用最近开发的入门程序将Spring Social与Redis支持的数据存储集成。在会议结束时，听众将知道Spring Boot的自动配置和入门项目不是神奇的独角兽，而是易于理解的Spring Boot生态系统中的组件。他们还将学习如何创建自己的Spring Boot Starter项目，并希望他们会积极参与Spring Boot项目。

Spring AMQP 1.7 RC1现已上市

发布
阿尔特姆·比兰（Artem Bilan）
2016年12月28日

我谨代表Spring Integration团队宣布Spring AMQP 1.7的第一个（也是最后一个）发行候选版本，该版本可在Milestone资料库中找到 。

此版本主要是版本之间的中间版本1.6.x和2.0用于春季靴1.5和IO平台Brussels兼容性。

无论如何，这里有一些改进和新功能可以解释：

  • 适当升级到Amqp Client 4.0 RabbitConnectionFactoryBean变化

  • 升级到Log4j 2.7，并需要修复制动更改log4j2.AmqpAppender

  • 重要的升级到Spring Retry 1.2 StatefulRetryOperationsInterceptor.setUseRawKey(true)向后兼容

  • 一个新的spring-rabbit-junit工件提供了几个实用程序（例如BrokerRunning@Rule ），这对于测试Spring AMQP应用程序很有用

  • 的SimpleMessageListenerContainer现在可以开始播放，而无需排队聆听。稍后可以通过运行时提供它们addQueues()

  • 一种ConnectionNameStrategy提供给ConnectionFactory允许在Broker或其他监视和跟踪工具中标识应用程序连接

春季数据发布系列Ingalls RC1发布

发布
奥利弗·德罗博姆
2016年12月23日

我谨代表Spring Data小组宣布Spring Data火车Ingalls的第一个（也是最后一个）发布候选人。该版本提供了310多个已修复的问题（其中一些以前与Hopper服务版本一起发布）。最重要的新功能是：

  • 升级到Spring 4.3（.5）作为基线- ＃294 。
  • 支持轻松地从聚合根-DATACMNS-928将域事件公开为Spring应用程序事件。
  • 支持衍生查询方法DATACMNS-875中的存在投影。
  • 支持Javaslang Option ，关于仓库查询方法的收集和地图类型-DATACMNS-937 ， DATACMNS-940 。
  • 添加了Spring Data LDAP模块，以接管Spring LDAP的存储库实现。
  • 最新版本-DATAMONGO-1536中增加了对MongoDB聚合运算符的支持。
  • 现在，流执行将拒绝调用，而不会在JPA- DATAJPA-1023中进行周围事务。
  • 在Gemfire- SGF-106 ， SGF-516中支持基于注释的索引和到期。
  • 在Spring Data REST- DATAREST-573中支持CORS配置。
