Spring Data Redis 1.2.0 RC1发布

发布
克里斯多夫·斯特罗布尔
2014年2月19日

我们很高兴地宣布Spring Data Redis 1.2.0的第一个候选版本！
我们将最低Spring版本提高到3.2.8。此版本针对Redis 2.6和当前的2.8以及最新版本的Spring Framework 4.0.2进行了测试。它引入了一系列错误修复以及一些改进，例如支持Jackson 2序列化，支持事务的缓存管理器，对TIME命令的支持以及Redis驱动程序的升级。

有关更改的完整列表，请参见此[JIRA查询]（ https://jira.springsource.org/issues/?filter=14427&jql=project%20%3D%20DATAREDIS%20AND%20resolution%20in%20 ( Complete%2C ％20Fixed）％20AND％20fixVersion％20in％20（％221.2.0.RC1％22）％20ORDER％20BY％20updated％20DESC） 。下一步将是RC2，计划于3月发布。
我们期待您在Twitter或问题跟踪器上的反馈。

春天的这个星期-2014年2月18日

工程

2014年2月19日

  3. 3月25日的网络研讨会-请务必查看Spring领导者Juergen Hoeller即将举行的网络研讨会：基于Java 8的Spring Framework 4 。
  4. SpringOne2GX 2013重播： Spring和Sencha-天生一对（或至少是云计算）现已可用
  5. 查看Spring的创始之父-Rob Harrop谈“在Amazon Web Services中运行Spring” ，这是SpringOne2GX 2013的精彩演讲。
  6. HCL Technologies的高级副总裁Srinivas Ajjarapu 在Pivotal堆栈帮助IT降低支出并获得更多收益的某些方式上发表了一篇不错的文章 。

  8. Daniel Mikusa和Stuart Williams的介绍Apache Tomcat 8的网络研讨会重播现在可以在线获得 。
  9. 史蒂夫·珀金斯（Steve Perkins）在使用Spring Boot的第一步中撰写了一篇不错的文章。看看这个！
  10. John Thoms提出了一个很好的示例，该示例使用SSL支持配置Spring Boot的嵌入式Tomcat应用程序服务器 。看看这个！
  11. 位于Spring Boot之上的恒星JHipster项目现在可以使用新的监视端点完成了 ！
  12. 说到Spring Boot， Spring Boot RC 3现在可用 。不要犹豫，检查一下。
  13. 我们的好朋友XavierPadró回来了，对如何在Spring中使用视图模板库Thymeleaf进行了很好的介绍
  14. 我的好朋友Andy Piper和Eclipse Paho（一个MQTT客户）负责人Andy Piper撰写了一篇不错的文章，介绍了如何使用Spring Integration的MQTT支持 。

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2014年2月18日

在加利福尼亚州圣克拉拉的SpringOne2GX 2013上录制

主讲人：娄·克罗克，约翰·弗格森

就像Spring出现并成为Java的事实上的标准一样，Sencha也在争夺HTML5世界中的类似角色。从用于桌面开发的Ext JS，用于移动设备的Sencha Touch和作为设计工具的Sencha Architect，它已成为多客户端开发新范例中的一种普及技术。在本节中，我们将使用我们自己的SpringTrader参考应用程序来说明Spring与Sencha的集成技术。我们将不仅仅介绍将绑定结果绑定到UI组件的REST调用的基本方法，还将介绍如何映射Spring MVC和Sencha MVC体系结构，以及其他简化移动应用程序开发工作的技巧。
