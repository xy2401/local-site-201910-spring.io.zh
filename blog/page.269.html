适用于Android 1.0.0的Spring。RC1发布

发布
罗伊·克拉克森
2012年3月5日

亲爱的春季社区，

我们很高兴地宣布， Spring for Android项目的第一个候选版本现已发布！

Spring for Android支持在Android环境中使用Spring Framework。1.0.0。RC1版本致力于更新对本机Android应用程序中最新的Spring Social版本的支持，以及对Rest Template的增强。这包括：

  • 支持Spring Social 1.0.2。发布，以及Spring Security 3.1.0。通过Spring for Android Auth模块进行释放，该模块包括用于持久OAuth API连接的SQLite数据存储。
  • 更新了RestTemplate支持，现在处于Spring Framework 3.1.0级别。发布。
  • 添加了对基本身份验证的支持。
  • 添加了AssetResource，用于访问存储在Android项目的Assets目录中的静态资源。
  • 由于某些设备上与预装的Gson版本不兼容，因此删除了自动加载GsonHttpMessageConverter的功能。
  • 改进了RestTemplate中的gzip支持。
  • 按照Google的建议，现在默认使用Gingerbread和更高版本中的标准J2SE设施（HttpURLConnection）。
  • 使用HttpURLConnection时，在Eclair和较早版本上禁用了连接重用，因为这些版本的Android上的连接池存在缺陷。
  • 添加了一些其他错误修复和改进。

介绍Spring Integration Scala DSL

工程
奥列格·朱拉库斯基（Oleg Zhurakousky）
2012年3月5日

介绍

Spring Integration团队很高兴宣布第一个里程碑版本（1.0.0。Spring Integration Scala DSL的 M1） -Spring Integration产品组合的最新成员之一。

什么是Spring Integration Scala DSL？

Spring Integration Scala DSL是一种用Scala编写的领域特定语言，其目标是：

    • 为Spring集成提供XML配置的强类型替代
    • 提高对Scala社区中Spring集成的认识
    • 提供与各种Scala框架和产品（例如Akka）的一流集成
    • 提供与Java的无缝集成，Scala开发人员仍可以利用其现有的Java投资
