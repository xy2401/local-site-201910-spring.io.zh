马克·波拉克
二月15，2019

介绍

自2011年Cloud Foundry本身启动活动以来，Spring Cloud Connectors库就一直在我们身边。连接器库和Cloud Foundry的Java buildpack的主要目标之一是“减少您想开始使用Cloud Foundry时的初期投资”。连接器库使用VCAP_SERVICES环境变量中包含的信息创建连接到支持服务（如数据库）所需的Spring bean定义。然后，buildpack通过一种称为“自动重新配置”的功能，将应用程序中具有的这些bean定义替换为由连接器库创建的bean定义。当您将应用程序推送到Cloud Foundry时，您可能已经在日志中看到了该消息...

Spring Boot 2.1.3现已发布

发布
斯特凡·尼科尔（StéphaneNicoll）
二月15，2019

我很高兴代表团队和所有贡献者，宣布Spring Boot 2.1.3已发布，现在可以从repo.spring.io和Maven Central获得。

此版本包括70多个修复程序，改进和依赖项升级 。感谢所有为问题报告和请求请求做出贡献的人。

如果您仍在使用Spring Boot 2.0.x ，升级到Spring Boot 2.1.x强烈鼓励。在宣布 Spring Framework之后5.0.x将于三月Spring Boot达到其停产2.0.x将跟进并最终发布2.0.x生产线计划于三月下旬。如先前宣布的 ，Spring Boot 1.5.x将继续保持下去，直到8月达到停产。
