Spring Roo 2.0.0M2发布

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年5月18日

我很荣幸地代表DISID Corporation的Spring Roo团队宣布Spring Roo 2.0.0。M2已发布！

这个Spring Roo版本的主要目标是更新代码生成以使用最新的Spring技术：

  • 使用Spring IO Platform来管理依赖版本，并能够使用最新的Spring技术。
  • 更新代码生成以使用最新的Spring Framework 版本（4.x）
  • 更新代码生成，以在生成的项目中包括Spring Boot。
  • 删除所有基于XML文件生成的配置，并使用Spring Boot自动配置。
  • 删除Active Record数据模型，改用基于存储库（Spring数据）的模型。
  • 默认情况下基于服务层模式生成应用程序体系结构

网络研讨会重播：带有Spring Cloud Data Flow的数据微服务

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年5月18日

主讲人：Pivotal的Mark Fisher和Mark Pollack
可伸缩数据处理的未来是微服务！基于Spring Boot提供的易于开发和部署以及Spring Cloud的云原生功能，Spring Cloud Stream和Spring Cloud Task项目为微服务流和批处理提供了一个简单而强大的框架。
在更高的抽象层次上，Spring Cloud Data Flow是一个集成的编排层，可为部署和管理由独立微服务组成的复杂数据管道提供高生产力的体验。流和任务是使用DSL抽象定义的，可以通过外壳和Web UI进行管理。此外，可插入的运行时SPI允许Spring Cloud Data Flow在各种分布式运行时平台（如Cloud Foundry，Apache Mesos和Apache YARN）之间协调这些应用程序。
在此网络研讨会中，您将看到Spring Cloud Data Flow的概述，以及从本地集群到远程Cloud的不同平台上的流和批处理应用程序的实时演示，以显示开发人员体验的简便性。

Spring Boot 1.4.0。M3现在可用

发布
菲尔·韦伯
2016年5月17日

Spring Boot v1.4.0。M3现在可以从Spring里程碑资料库中获得 。此版本关闭了150多个问题，并引入了许多新功能。

重点包括：

  • M2上的许多错误修复和改进（感谢大家尝试里程碑）。
  • 基于约定的错误页面（想要自定义404，只需添加src/main/resources/public/error/404.html ）。
  • 改进的ErrorPage注册支持。
  • 支持可插拔的OAuth2主体提取。

有关更改的完整列表和升级说明，请参阅WIKI上的《 Spring Boot 1.4发行说明 》和更新的参考文档 。

Spring 的云彩布里克斯顿。可以发布

发布
斯宾塞·吉布
2016年5月11日

我很高兴地代表团队宣布Spring Cloud Brixton Release Train已达到通用。该版本今天就可用，可以在我们的Spring Release存储库和Maven Central中找到 。

我们要感谢许多社区贡献者和测试人员的帮助。有许多请求请求，问题和对话使Spring Cloud Brixton变得更好。

强调

布里克斯顿发行火车的一些亮点是：

  • Spring Boot 1.3.x和Spring Framework 4.2.x支持
  • Hashicorp Consul通过Spring Cloud Consul支持服务注册/发现和配置
  • Apache Zookeeper支持服务注册/发现，通过Spring Cloud Zookeeper进行配置以及Spring Cloud Cluster中的领导者选举
  • 通过Spring Cloud Sleuth抽象进行分布式跟踪，其中有两种现成的实现方式：一种支持日志记录（对于日志收集器和Logstash和Loggregator等多路复用器来说是理想的），另一种支持Twitter的Zipkin
  • Netflix Atlas遥测系统 ，下一代Spectator Metrics库以及Eureka，Ribbon，Hystrix和Feign的最新版本可在Spring Cloud Netflix中获得
  • Spring Cloud Bus现在由最近发布的Spring Cloud Stream支持
  • 通过Spring Cloud Cluster进行集群领导者选举和锁定
  • 将Spring Boot指标导出到Amazon Cloudwatch，以及对Amazon RDS的本地支持
