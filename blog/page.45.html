Spring Data Ingalls SR15和Kay SR10发布

发布
马克·帕鲁奇
九月10，2018

我谨代表Spring Data团队宣布Ingalls SR15和Kay SR10维护版本的可用性。Kay SR10随附在刚刚发布的Spring Framework 5.0.9之上。为了方便起见，Spring Boot 2.0.5将选择Kay SR10。Spring Boot 1.5.16将使用Ingalls SR15。两种版本都提供了大多数错误修正和一些依赖性升级。

Ingalls SR15附带40张票 ，Kay SR10包含75张票。

另外，请确保您注册了今年的SpringOne平台 （9月24日至27日，华盛顿特区），以了解有关Spring Data的最新功能，甚至可以预览计划用于下一版本的功能。整个团队将在现场，并乐意回答问题并获得反馈。

Spring Vault 2.1进入RC

发布
马克·帕鲁奇
九月10，2018

我谨代表社区宣布Spring Vault 2.1发行候选版本的可用性。Spring Vault 2.1需要JDK 8或更高版本，并且特别支持JDK 11作为下一个长期支持版本。此发行版附带15张票证，并针对Java 9至11进行了一些改进。最值得注意的变化是：

  • 支持使用VM上的托管服务实例进行Azure身份验证。
  • 代表服务帐户的GCP IAM身份验证支持。

此版本与从0.5.2到0.11.1 Vault版本兼容。

发布

我很高兴代表团队宣布Spring Framework 5.1 RC3、5.0.9和4.3.19 现已可用 。
