Spring CredHub 2.0.0发布

发布
斯科特·弗雷德里克
一月10，2019

我很高兴地宣布，Spring CredHub 2.0.0的一般可用性版本现已在Maven Central中提供。

CredHub

CredHub为Cloud Foundry平台提供集中的凭据管理。CredHub实现了HTTP API以安全地存储，生成，检索和删除各种类型的凭据。

Spring CredHub为CredHub API提供了Java绑定，可轻松将Spring应用程序与CredHub集成。

什么是新的

Spring CredHub 2.0.0发行版包含以下增强功能：

  • 支持CredHub服务器版本1.x和2.x，以及CredHub v1和v2 API

  • 提供与Spring Framework 5.1和Spring Boot 2.1的兼容性

  • 使用Reactor添加反应接口

有什么不同？
