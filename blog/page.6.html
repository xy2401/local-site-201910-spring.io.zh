Spring Cloud Data Flow 2.3.0。M1和2.2.1 GA发布

工程
Ilayaperumal Gopinathan
九月6，2019

Spring Cloud Data Flow团队很高兴宣布里程碑版本2.3.0。M1和2.2.1 GA发布。

除了错误修复和增强功能，2.3.0的第一个里程碑还具有以下新功能：

  • 基于Prometheus RSocket的改进Prometheus监控。它在流/任务应用程序和Prometheus RSocket代理实例之间使用持久的双向（RSocket）连接。当Prometheus抓取代理实例时，代理依次使用连接从应用程序中获取指标。
  • 能够将OAuth访问令牌作为组成任务的命令行参数传递
  • 用户现在可以在创建流时指定流的描述。谢谢Daniel Serleg为贡献！
  • Spring Cloud Scheduler SPI现在已合并到Spring Cloud Deployer中。我们将弃用显式spring-cloud-scheduler项目及其在Cloud Foundry，K8s上的实现，并将它们作为一部分提供spring-cloud-deployer 。

Spring HATEOAS 1.0.0。RC2发布

工程
格雷格·特恩奎斯特
九月5，2019

尊敬的Spring社区，我们很荣幸地宣布Spring HATEOAS 1.0.0的第二个候选版本。自从上一发行版以来，Spring HATEOAS有了两个更大的里程碑和两个RC，取得了长足的进步。

总结如下：

  • 对HAL和HAL-FORMS的国际化支持。

  • 性能提升

  • Spring HATEOS现在基于Spring Framework 5.2和Reactor的Dysprosium版本。这对于支持诸如Spring Data Moore和Spring Boot 2.2之类的下游项目至关重要。

  • 现在支持同时具有超媒体和非超媒体端点的Spring WebFlux应用程序。

  • 改进了大部分参考文档。

  • 创建了Spring HATEOAS Gitter频道 。

  • 修复了各种媒体类型中的问题，以确保它们符合规范定义，并且不会引发意外异常。
