Spring Cloud Task 2.1.0。M2现在可用

发布
格伦·伦弗罗（Glenn Renfro）
十二月20，2018

我们很高兴地宣布Spring Cloud Task 2.1.0。现在可以在Github和Pivotal下载存储库上使用M2。非常感谢所有对此版本做出贡献的人。

什么是新的？

Spring Cloud Task 2.1.0。M2旨在成为与Spring Boot 2.1.0保持一致的框架版本。从2.1.0更新。M1包括：

  • 更新所有依赖项。
  • 根据大众的需求， @EnableTask回来了。
  • Bug修复。

让我们更详细地介绍这些更新。

@EnableTask又回来了

根据大众的需求， @EnableTask又回来了！在这种情况下， AutoConfiguration仍用于创建TaskExplorer ， TaskConfigurer和其他bean，但不会创建TaskLifecycleListener （将“任务生命周期”添加到您的应用程序的bean）。的TaskLifecycleListener当用户添加@EnableTask他们的应用程序或配置。我们将此添加回去，以使正在构建自己的应用程序（不是任务，但监视或浏览任务存储库）的用户通过使用提供的自动配置来构建应用程序。

CredHub

CredHub为Cloud Foundry平台提供集中的凭据管理。CredHub实现了HTTP API以安全地存储，生成，检索和删除各种类型的凭据。

Spring CredHub为CredHub API提供了Java绑定，可轻松将Spring应用程序与CredHub集成。
