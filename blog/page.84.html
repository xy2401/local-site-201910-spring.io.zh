适用于Apache Kafka 2.0 Milestone 3的Spring

发布
阿尔特姆·比兰（Artem Bilan）
2017年7月24日

我们很高兴地宣布Apache Kafka的里程碑3的可用性2.0版。

可以从Milestone存储库下载：

repositories {
    maven { url 'http://repo.spring.io/libs-milestone' }
}
compile "org.springframework.kafka:spring-kafka:2.0.0.M3"

自上次发布以来，这些功能已成为当前的里程碑：

  • Apache Kafka 0.11.0.0客户端基准线；

  • 通过新的Apache Kafka基础，我们引入了KafkaTransactionManager和交易支持KafkaTemplate通过新executeInTransaction()操作

  • 的KafkaListenerContainer现在可以随TransactionManager执行具有事务边界的轮询和偏移提交；

  • 还提供了Kafka标头支持。现在，您可以发送和接收带有任意头的消息，例如，使用某些头来构建请求-答复方案correlationKey标题

  • 的ConsumerAwareErrorHandler提供给管理Consumer指出错误情况；

  • 的KafkaEmbedded现在可以提供任何任意brokerProperties 。

春天的这个星期-2017年7月25日

工程
乔什·朗（Josh Long）
2017年7月24日

嗨，春季粉丝！本周，我在土耳其的伊斯坦布尔与客户交谈，并在今晚的Spring和Java聚会上讲话。希望您能加入我的行列，并且我们将谈论云原生Java！像往常一样，本周我们有很多工作要做，让我们开始吧！

  • Spring Integration的贡献者Artem Bilan刚刚宣布了Spring Integration 4.3.11 。此版本包括安全更新和错误修复。

  • Spring NinjaStéphaneNicoll发布了Spring Framework 4.3.10 ，它是即将发布的Spring Boot 1.5.5维护版本的维护版本。它包含25个修复和改进。

  • Spring集成和消息传递忍者Artem Bilan刚刚宣布了Spring AMQP 2.0.0里程碑5 。此版本包括安全修复程序和许多不错的新功能。

  • Log4j2 AmqpAppender现在是线程安全的；

  • DefaultMessagePropertiesConverter现在可以将标头值的数组正确传递给AMQP客户端库；

  • Jackson2JsonMessageConverter现在可以为trustedPackages解决杰克逊序列化小工具的违规问题;

  • RabbitAdmin现在可以通过一个工作单元在同一AMQP通道上执行操作RabbitTemplate.invoke() ;

  • PossibleAuthenticationFailureException现在可以通过以下选项将其视为非致命的AbstractMessageListenerContainer ;

  • RepublishMessageRecoverer现在可以为deliveryMode以便重新发布消息。
