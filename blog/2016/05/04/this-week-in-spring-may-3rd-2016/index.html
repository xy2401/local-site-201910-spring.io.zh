Spring 的这个星期-2016年5月3日

工程
乔什·朗（Josh Long）
2016年5月4日

我什至不敢相信已经五月了！停止我想要的世界！好吧，至少时代是令人兴奋的！这周有很多很酷的东西要看，所以让我们开始吧！

  • 全新的会议在崭新的SpringOne Platform网站上进行，请查看！除了Adrian Cockcroft的主旨演讲外，我们还邀请了来自Netflix，Google，Comcast，Express Scripts，Kroger的演讲者，以及Juergen Hoeller，Dave Syer博士，Rossen Stoyanchev，Oliver Gierke等Spring领导人，当然还有您真的！
  • 并加入Spring Cloud Data Flow领导Mark Fisher和Mark Pollack一起研究带有Spring Cloud Data Flow的数据驱动微服务
  • Spring Session和Spring Security负责人Rob Winch 刚刚宣布了Spring Session 1.2.0。RC3 。
  • Spring REST Docs领导Andy Wilkinson刚刚宣布了Spring REST Docs
  • Spring Cloud Data Flow联合负责人Mark Fisher刚刚宣布了Spring Cloud Data Flow 1.0.0。M3
  • Janne Valkealahti 刚刚宣布了针对Apache YARN的Spring Cloud Data Flow
  • Spring和数据向导Thomas Risberg 刚刚宣布了适用于Apache Mesos和Kubernetes 1.0.0的Spring Cloud Data Flow。M2
  • Spring Cloud Data Flow忍者Eric Bottard刚刚宣布了Cloud Foundry 1.0.0的Spring Cloud Data Flow。M2
  • Spring Data主管Christoph Strobl撰写了大量有关Spring Data Hopper新增功能的文章
  • Spring Data忍者John Blum刚刚写了一篇很好的文章，介绍了Spring Data Geode的新功能
  • 我们的朋友迈克尔·西蒙斯（Michael Simmons）整理了一篇有关Spring Boot自动配置工作原理的精彩文章
  • 您是否一直在关注Cloud Foundry的最新动态 ？
  • Dan Vega整理了一段不错的视频， 介绍了如何将Lombok与Spring Boot一起使用
  • Anirudh Bhatnagar 撰写了一篇有关同时使用Spring Boot，Thymeleaf和Angular.js的不错的文章
  • 我喜欢这篇关于使用CXF在Spring Boot之上开发JAX-RS服务的文章 。（如果他们只是使用Jersey，那就更容易了！）
  • 是否想获得本地Cloud Foundry环境？认识PCF开发人员：您在本地运行Cloud Foundry的门票
