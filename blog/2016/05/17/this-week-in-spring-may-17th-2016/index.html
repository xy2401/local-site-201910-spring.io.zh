Spring 的这个星期-2016年5月17日

工程
乔什·朗（Josh Long）
2016年5月17日

欢迎来到春季本周的另一期！在撰写本文时，我正在印度钦奈，前往印度班加罗尔参加激动人心的一天客户会议，然后前往西班牙阳光明媚的巴塞罗那参加史诗般的Spring IO活动 ！然后，这个星期五，我要去乌克兰基辅参加JEEConf 。如果您像往常一样在周围， 请不要害羞，打个招呼！ 。就是说，让我们开始吧！

  • BIG新闻： Spring Cloud Brixton现在可用 ！从Spring Initializr获取位 。如果您允许进行一些后期修补，那么我认为通过阅读我写的这篇文章 ，您可能会对与Spring Cloud地址等技术有关的问题有很好的了解，并且您将了解如何在此中使用 Spring Boot和Spring Cloud 。视频 。就是说，抓住零碎去！
  • Spring Cloud Stream 1.0.0终于来了！ spring-cloud-stream使构建基于消息传递的协作微服务变得简单。
  • Spring Boot的共同负责人Phil Webb刚刚宣布了Spring Boot 1.4.0。M3现在可用 ， 正好赶上巴塞罗那的Spring I / O活动！1.4中有很多很棒的新功能，因此请确保尽早并经常尝试。
  • Spring Security和Spring Session的负责人Rob Winch刚刚宣布了Spring Session 1.2.0。发布 。此版本包含大量有趣的东西 。
  • Spring Integration的负责人Gary Russell刚刚宣布了Spring Integration 4.3.0。RC1 （最后一个RC！因此，请小心一点并踢轮胎！）
  • Spring IO Platform 负责人Andy Wilkinson刚刚宣布了Spring IO Platform 2.0.5 。该版本包括一些有趣的库更新，包括Spring Boot 1.3.5。发布和Spring AMQP 1.5.6。发布。
  • Spring Security团队刚刚发布了Spring LDAP 2.1.0 ， 这让他们不为所动 。
  • 我非常喜欢这个博客，了解如何将Pivotal的Concourse CI管道工具与Cloud Foundry结合使用
  • Pivotal自己的Richard Seroter 对消息传递技术RabbitMQ，NATS和Apache Kafka进行了很好的比较（尽管.NET颜色）
  • 这篇有关将Groovy语言Spock与Spring Boot 1.4的新测试功能结合使用的文章很老套！
  • 塔基皮（Takipi）的人们发表了一篇有趣的文章，他们比较并发现了最广泛使用的前100个Java项目 。当然，Spring和各种Spring模块的功能非常突出，在前100名中占44个！
