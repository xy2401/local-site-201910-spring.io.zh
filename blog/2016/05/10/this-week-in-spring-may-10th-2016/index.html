春天的这个星期-2016年5月10日

工程
乔什·朗（Josh Long）
2016年5月10日

Em妈！时间一定在飞！欢迎来到春季本周的另一期！ 本周，我将在华沙参加Cloud Native聚会，并在波兰的克拉科夫参加壮观的Geecon活动！如果您在附近，请随时打个招呼 。另外，一些非常重要的消息即将到来！敬请期待明天的Spring.io博客 ！无论如何，像往常一样，我们有很多事情要看，所以让我们开始吧！

  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 4.1.0 RELEASE ！这是一个很大的发布， 包含很多很棒的东西 ！
  • Spring IO Platform负责人Andy Wilkinson刚刚宣布了Spring IO Platform 2.0.4版本
  • Spring AMQP和消息传递专家Gary Russell刚刚宣布了Spring AMQP 1.6.0 RC（和1.5.6）
  • Spring Framework NinjaStéphaneNicoll刚刚宣布了Spring Framework 4.3 RC2、4.2.6和3.2.17现已可用
  • 史泰芬（ Stéphane）不再坐拥桂冠， 还宣布推出Spring Boot 1.3.5
  • 这个博客很好地介绍了微服务生态系统中的一些参与者 ，尽管它们是高级的。
  • 是否想了解支撑Cloud Foundry BOSH的服务包管理器？此博客是一个非常良好的开端 ！
  • 杰西卡·科尔（Jessica Kerr）和持续交付的杰出人物杰兹·汉布尔（Jez Humble）在Craft Conf上进行了精彩的讨论， 杰西卡（Jessica）撰写的有关团队之间协调权衡的博文非常值得一读
  • 使用数据科学理解非结构化文本
  • 我喜欢Spring Boot和HTTP 2上的这篇不错的文章
  • 我的好友Markus Eisele在O'Reilly的软件架构展览会上的联合演讲中写的一篇不错的帖子，我们在其中讨论了Spring和Java EE集成模式 。
