On behalf of the community, I’m pleased to announce the release of Spring Session 1.3.0.M2. This release release closes lots of community submitted Pull Requests. For a complete list of changes see the changelog.

Note
Spring Session 1.3.0.M1 release had some problems, so we followed up with an immediate release of Spring Session 1.3.0.M2

What’s New in Spring Session 1.3.0 M2

Highlights include:

  • First Class Support for Hazelcast

  • First class Spring Security’s concurrent session management

  • Redis Sample with GenericJackson2JsonRedisSerializer

  • spring.session.cleanup.cron.expression can be used to override the cleanup task’s cron expression

  • Lots of performance improvements and bug fixes

Spring Session 1.3.0 M2发布

发布
罗伯·温奇
九月14，2016

我很高兴代表社区宣布Spring Session 1.3.0的发布。M2。此版本释放了许多社区提交的拉取请求。有关更改的完整列表，请参见changelog 。

注意
 春季会议1.3.0。M1版本存在一些问题，因此我们紧跟着发布了Spring Session 1.3.0。M2

Spring Session 1.3.0 M2中的新增功能

重点包括：

  • 对Hazelcast的一流支持

  • 一流的Spring Security并发会话管理

  • 使用GenericJackson2JsonRedisSerializer的Redis样本

  • spring.session.cleanup.cron.expression可用于覆盖清理任务的cron表达式

  • 许多性能改进和错误修复

会费

没有社区，我们就不可能成为今天的成功项目。我要感谢所有创建问题并提供反馈的人。
