    春天的这个星期-2016年9月20日

    工程
    乔什·朗（Josh Long）
    2016年9月20日

    你好，世界！自上次交谈以来，我去过中国的多个城市，包括杭州，上海，北京，广州，深圳和香港，并与客户和社区成员进行了交谈。看到我们在中国的惊人客户如何在竞争异常激烈的中国市场中使用我们的技术，这真是令人着迷。而且，好吃！

    • Spring Cloud Camden快来了！Spring Cloud联合创始人Spencer Gibb刚刚宣布了Spring Cloud Camden RC1，它包括许多改进和一个非常酷的新模块Spring Cloud Contract
    • Apache Kafka的Spring负责人Artem Bilan刚刚宣布Apache Kafka 1.1 GA的Spring和Spring Integration Kafka 2.1 GA现已可用
    • Spring Integration忍者Artem Bilan 也 刚刚宣布Spring Integration 4.3.2刚刚发布 ，其中围绕RMI适配器，TCP以及一些新的全局调整（如spring.integration.readOnly.headers全局属性）进行了改进。这是Spring Integration 5.0的一个小但值得的更新，一个月之内就要完成一个里程碑！
    • Spring框架忍者StéphaneNicoll 刚刚宣布了Spring框架4.3.3和4.2.8现已可用
    • Spring Cloud Data Flow忍者Thomas Risberg刚刚宣布了Spring Cloud Data Flow 1.1 M1和1.0.1 GA ！1.1 M1，尤其是非常令人兴奋！它增加了LDAP后端支持，HTTP BASIC后端身份验证，文件后端身份验证，Spring Boot 1.3和1.4支持，UI改进以及许多其他优点。
    • Spring Integration Java DSL负责人Artem Bilan从来没有放松过， 也 刚刚宣布了1.2 M2 ，其中包括JPA支持，中间流事务支持，分散收集支持，改进的路由器等等！
    • Spring Security和Spring Session的负责人Rob Winch刚刚宣布了Spring Session 1.3.0 M2，其中包含对Hazelcast的一流支持，对Spring Security并发会话管理的支持，Redis JSON序列化示例，对Redis清理任务的更轻松配置以及许多其他功能 。性能改进和错误修复。趁热拿到！
    • 我是从社区英雄Michael Simmons上运行[通过ElasticSearch在Pivotal Cloud Foundry上进行休眠搜索]（ http://info.michael-simons.eu/2016/09/20/running-hibernate-search-with-elasticsearch）上获得的。 -on-pivotal-cf /
    • 我真的很喜欢Alex Blewitt 在Hotspot内部的演讲。另外：查看他的史诗般的领带！
    • 我非常感兴趣，可以从上海的eBay了解名为YiDB的配置服务器。可能值得一看！
    • Riot Games的这个博客，介绍了他们在使用Zuul时所做的一些优化， 非常有用
    • 这是来自Sebastian Dehne的JavaZone的非常酷的演讲，内容涉及零停机时间部署以及数据库更改。
    • 说到， Spring Cloud Netflix Github中有一个非常酷的话题，关于使用Eureka进行0停机的蓝色/绿色部署
    • 不幸的是，Java 9将延迟4个月
    • 我将重点研究边缘服务选择，尤其是Netflix Zuul过滤器 ，您当然可以将其与Spring Cloud Netflix结合使用
