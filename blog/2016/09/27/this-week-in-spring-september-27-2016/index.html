Spring 的这个星期-2016年9月27日

工程
乔什·朗（Josh Long）
九月27，2016

哇！已经快十月了！今年肯定过去了！

欢迎来到春季本周的另一期！本周，我将与Phil Webb一起在录音室中录制下我们的Livelessons系列的下一部分， 该系列有关使用Spring Boot构建微服务 。到目前为止，这很有趣，我们才刚刚开始！

今晚，我希望您能与我一起在vJUG的全球范围内参加Cloud Native Java ，并在世界各地的时区进行现场活动！

我们有很多内容要讲，所以让我们开始吧！

  • Spring领导者Juergen Hoeller刚刚宣布了Spring Framework 5.0。M2，这是前往Spring 5的又一个重要步骤！这个版本还包括Spring MVC的功能端点机制 ！
  • 欢呼！Reactor 3.0已经发布！了解最新信息，并了解Reactor为什么支撑Spring Framework 5的Reactive Runtime
  • 查看Spring响应式忍者Arjen Poutsma对Spring Framework 5中的功能端点机制的新见解
  • Spring Cloud忍者Spencer Gibb 刚刚宣布了Spring Cloud Camden。发布和Spring Cloud Brixton。SR6 ！Spring Cloud Camden 挤满了好东西，包括新的Spring Cloud Contract工作！趁热拿到！
  • Spring Integration Java DSL负责人Artem Bilan刚刚宣布Spring Integration Java DSL 1.2 RC1现在可用
  • 查看Rossen Stoyanchev 在4.3版本中对Spring MVC的出色观看的 SpringOne平台重播
  • 查看此SpringOne平台重播，了解Craig Walls和Roy Clarkson的Spring Cloud Services
  • Spring和数据忍者Thomas Risberg 刚刚宣布了适用于Apache Mesos 1.0的Spring Cloud Data Flow。GA
  • Spring Cloud Contract负责人Marcin Grzejszczak宣布了1.0.0。在Spring Cloud Camden本身的启动中发布Spring Cloud Contract 。Spring Cloud Contract是Spring Cloud产品组合中必看的一项补充，它使服务提供商和服务使用者可以轻松地停留在同一页面上并进行构建（和测试！）。确信节点之间的隐含契约是完整的。
  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 4.2。M1 ，解决了50多个问题，并进行了许多其他改进
  • 在Spring Cloud Camden诞生之前，Marius Bogoevici刚刚宣布了Spring Cloud Stream Brooklyn，这使得构建基于消息传递的微服务变得容易
  • Spring Data忍者Janne Valkealahti刚刚宣布了适用于Apache YARN 1.0.2的Spring Cloud Data Flow
  • Spring Platform负责人Andy Wilkinson 刚刚宣布了Spring IO Platform Athens和Spring IO Platform 2.0.8
  • Spring Boot忍者StéphaneNicoll 刚刚宣布了Spring Boot 1.4.1。和1.3.8
  • Atomist创始人和Spring创始人Rod Johnson有趣地发表了一篇关于如何使用他公司的技术自动执行代码编辑以促进代码库升级的有趣文章。当然，他的例子是关于Spring Cloud，尤其是Hystrix。
