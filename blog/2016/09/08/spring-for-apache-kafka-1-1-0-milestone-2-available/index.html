适用于Apache Kafka 1.1.0 Milestone 2的Spring

发布
加里·罗素
2016年9月8日

我很高兴地宣布，Apache Kafka 1.1.0版是Spring的第二个里程碑。M2现在可以在 Spring 里程碑仓库中使用 。

其中包括一些错误修复和以下新功能：

  • 使用以下命令时，现在可以处理一批消息（ 在上一个里程碑中引入 ）。 @KafkaListener例如注解...

@KafkaListener(id = "list", topics = "myTopic",
                          containerFactory = "batchFactory")
public void listen(List list) {
    ...
}

  • 现在，您可以从侦听器执行查找操作-使用组管理时，当Kafka分配分区时，这可以设置初始偏移量。您还可以在初始化后执行任意查找操作。

的KafkaTemplate现在可以访问metrics和partitionsFor上的方法Producer 。

此外，第一个里程碑spring-integration-kafka现在可以使用2.1版， spring-kafka 1.1.0。M2版本。它还支持批处理负载。

请参阅项目页面以获取文档链接和更多信息。
