Spring 的这个星期-2016年9月13日

工程
乔什·朗（Josh Long）
九月13，2016

欢迎来到 Spring 本周的另一期！在中国，这是一个了不起的一周，与阿里巴巴和华为一起度过了 Spring 。本周要去上海，北京，广州，深圳和香港！

我们的朋友彼得·汉弗莱（Pieter Humphrey）本周忙于将SpringOne Platform 2016视频上传并在线上供我们欣赏，因此本周有很多好东西！谢谢彼得！

  • 想要在9月29日看到Mark Fisher和Mark Pollack出席并通过Spring Cloud Data Flow进行现场演示吗？立即预订座位！
  • 是否想了解Spring Framework 4.3的新功能？加入Spring Framework 负责人Juergen Hoeller，看看这个很棒的框架
  • 我喜欢来自Home Depot的关于12因素应用程序对Java开发人员意味着什么的演讲，事实证明这是参加会议的人数最多的活动之一！
  • 我在SpringOne Platform 2016上进行了将近2个小时的演讲，名为Cloud Native Java，涵盖了Spring Boot（REST，HATEOAS，Actuator，JPA），Spring Cloud（集中配置，服务注册和发现，断路器，微代理，API网关，最终一致性）。 ，流处理，可观察性和监视，分布式跟踪），Spring Cloud Security（OAuth和SSO）和Spring Cloud Data Flow。趁热拿到！
  • 与Apache Tomcat忍者Mark Thomas一起在SpringOne Platform 2016的本次演讲中对Apache Tomcat路线图进行出色的了解
  • Spring Cloud Task和Spring Batch负责人Michael Minella 刚刚宣布了Spring Cloud Task 1.0.3和1.1.0。M1现在可用
  • Spring Integration的负责人Gary Russell刚刚宣布了适用于Apache Kafka 1.1.0 M2的Spring，该版本现在支持批量处理来自Kafka目标的消息并查找操作！趁热拿到！
  • Spring Cloud Stream负责人Marius Bogoevici刚刚宣布了Spring Cloud Stream Brooklyn RC1
  • Spring Session负责人Rob Winch刚刚宣布了Spring Session 1.2.2，其中包含许多错误修复和改进
  • 凉！特定于实例的路由刚刚落入Cloud Foundry的gorouter ！这是特定于实例的Spring Boot执行器端点的理想选择 ！
  • 我喜欢Spring Retry的中文介绍
  • 是否想与Spring Session一起使用穷人授权服务器？这是一个很好的介绍！
  • 我喜欢这个博客，其中介绍了Aliyun博客上的Spring Cloud Sleuth和Zipkin支持 。对于那些不知道的人来说，阿里云就像是阿里巴巴的亚马逊的AWS。这太酷了！
  • 我们的前CTO Adrian Coyler在他的博客“ The Morning Paper”上有一篇很棒的文章，内容涉及微服务体系结构中的数据处理。
