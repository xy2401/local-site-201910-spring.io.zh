春季本周-2016年1月5日（5周年纪念版！）

工程
乔什·朗（Josh Long）
2016年1月6日

欢迎来到春季本周的另一期，欢迎我度过一个充满活力，有趣的新年！

这个星期，我们纪念春天本周五周年，
我和亚当·菲茨杰拉德 （ Adam Fitzgerald ）刚从2010年假期开始（基于与2011年1月与基思·唐纳德 （ Keith Donald ）的讨论）。

从那时起，我竭尽全力每周发布一次 （无例外！没有错过的几周！）每周星期二在夏威夷的午夜之前，无论我在那个星期处于什么时区！这个世界很大，因此从某个人坐在纽约市的角度来看，有时候这个博客有时会出现在星期一中午（亚洲各个国家/地区的00:00 AM）或星期三清晨（美国夏威夷）-变化很大 ！但是，我向您保证，我总是在星期二发帖！

我会尽一切可能发布。如果我知道自己要坐10-20个小时的长途飞机，请确保将其预发布，或者如果仍然在星期一，则安排它，无论我在哪里。

起初， 亚当·菲茨杰拉德 （ Adam Fitzgerald）以及过去几年里的彼得·汉弗莱 （ Pieter Humphrey）在许多方面都具有无价之宝，其中包括确保帖子不丢失任何链接，或者我-急于-没发指。谢谢您，没有您，我真的无法做到！

当然，最大的贡献者这个星期在春天 -我喜欢做它的原因-是你们 ，社会各界，他们的精力和热情驱使你和超越;撰写博客，记录演讲，介绍幻灯片，并通常有助于在世界范围内传播精通知识。非常感谢！

这是更多的分期付款，并且一如既往地毫不犹豫地在Twitter（@starbuxman）上通过贡献，反馈或其他方式对我进行ping操作 。

现在，让我们继续进行下去，让我们进入另一个内容丰富的综述！

  • Spring Cloud自己的Marcin Grzejszczak刚刚在
    测试基于Spring Cloud的微服务
  • Spring XD和Spring Cloud Data Flow负责人Mark Pollack博士刚刚宣布了Spring Cloud Stream 1.0 M3和Spring Cloud Data Flow 1.0 M2
  • 我们的好朋友Pieter Humphrey又来了，与去年的SpringOne2GX 2015相比，它提供了很多很棒的内容。让我们从Drs的介绍开始。 Syer和Pollack 使用Spring Cloud Stream来开发消息驱动的微服务，这是Spring Cloud Data Flow的基础 。
  • 是否想向真正的大师学习Java 8？查看Venkat Subramaniam博士关于将代码转换为Java 8的演讲
  • 本讲座介绍如何在基于Spring Boot的应用程序中考虑RxJava
  • 让Spring Data忍者Christoph Strobl教您如何在Spring Boot应用程序中引入搜索功能
  • 查看Sarah Aerni，Srivatsan Ramanujam和Jarrod Vawdrey的关于如何对数据科学解决方案中的事件做出反应的演讲
  • 我和Adam Koblentz进行了演讲，介绍如何使用Spring Boot和JRebel
  • 查看我们自己的Kenny Bastani关于使用Neo4J和Spring创建页面排名引擎以对Twitter关注者进行排名的介绍。
  • 我真的从Anwar Ziani的这本法语幻灯片中挖了出来， 介绍了如何使用Spring Boot和Maven 。我走到尽头才意识到他和我认识了！（小世界！）我要从平台上更改的一件事是，Spring（在Spring Boot之前）需要XML的微妙含义，这根本不是事实。它支持Java配置。我还没有使用XML太多可言多年 。
  • Mert Caliskan 撰写了一篇不错的文章，内容涉及将Spring Boot应用程序部署到Payara Micro应用程序服务器上 。Payara是GlassFish的变体。这是一篇不错的文章，但也错误地暗示了Spring需要XML，但这是不正确的。确实，您可以使用Servlet 3.0，JPA，依赖项注入和许多其他Java EE技术， 而无需使用必需的Java EE XML文件（为什么本文有空web.xml ？）在基本的Spring中，Spring本身当然不需要XML。我也有些困惑，为什么这个应用程序不只是使用典型的Maven pom.xml由Spring Initializr生成的，Spring Initializr预先配置了Java 1.8支持。除此之外，如果您采用这种方式，那将是一篇不错的文章，而且很有趣！

（我差点将约会日期定为2015年 ！）
