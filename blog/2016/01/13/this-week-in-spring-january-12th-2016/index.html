春天的这个星期-2016年1月12日

工程
乔什·朗（Josh Long）
2016年1月13日

欢迎来到春季本周的另一个装置！ 今天，我在旧金山水罐公园的一个拥挤的房间里讲话，现在我要去圣路易斯参加一天的客户会议。在我们开始新的一年时，有很多很棒的东西要看，所以让我们开始吧！

  • 查看Spring Integration贡献者Artem Bilan在Spring Integration上最新，最出色的文章，其中包括新的Spring Integration扩展的首次亮相，该扩展支持飞行中消息的压缩和解压缩。
  • EuregJUG Maas-Rhine 在其网站上发布了有关该网站的信息！全部是Spring Boot 。看看这个。
  • 观看安迪·威尔金森（Andy Wilkinson）的SpringOne2GX 2015 演讲的重播，该演讲有关使用Spring REST Docs记录REST API。
  • 查看Fatima Casau 关于使用Spring Boot和Groovy的SpringOne2GX 2015演讲的重播
  • Marius Bogoevici的SpringOne2GX 2015演讲也将通过Spring XD和Apache Kafka进行大规模流处理
  • Spring Security负责人Rob Winch的SpringOne2GX 2015 演讲也介绍了如何使用Spring保护REST服务。
  • Spring Framework 的传奇人物和负责人Juergen Hoeller参与了接下来的几个月中有关Java的重要会议
  • 查看David Syer博士关于使用Spring Security OAuth和Spring Boot的权威指南
  • 然后在Malike的文章中跟进，有关使用Spring Security OAuth和MongoDB保护应用程序的安全
  • 我很确定之前已经发布了这些内容，但值得重申： 亚马逊如何使用微服务以及微服务的痛苦，力量和模式
  • 红帽公司的克里斯蒂安·波斯塔（Christian Posta）刚刚发布了一个博客，该博客讨论了使用ActiveMQ的某些用例可能会提高 25倍的性能 -绝对值得一读！
