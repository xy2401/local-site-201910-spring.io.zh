春天的这个星期-2016年1月26日

工程
乔什·朗（Josh Long）
2016年1月26日

欢迎来到春季本周的另一个装置！在这里，我们已经凝视着一年的第二个月了， 男孩 ！，事情变得令人兴奋了！我即将再次上路，如果您在2月的以下任何地点和活动附近， 很想 与您和您的组织进行对话 ：

  • 微创会议
  • JFokus
  • 天狼星
  • 康涅狄格州哈特福德（大约17/2/2016）
  • 康富
  • 上海（大约26/2/2016）

无论如何，事不宜迟，让我们开始吧！

  • 我们自己的保罗·查普曼（Paul Chapman）加入了更新的课件中，该课件可供想要了解Spring的开发人员使用
  • 有（最后！）有关使用Spring Cloud的新指南： 使用Netflix Zuul微代理库和Spring Cloud进行路由和过滤，使用 Spring Cloud Config Server进行集中配置， 断路器设计模式以及使用Spring Cloud进行服务注册和发现DiscoveryClient和Netflix Eureka的Spring Cloud集成 。请继续关注，我们还有更多产品正在准备中！
  • Spring Boot主管Phil Webb刚刚宣布发布1.3.2 ，其中包括对依赖项的更新和一些错误修复。
  • 查看Jetbrain的YannCébron以及我们自己的StéphaneNicoll的史诗般的SpringOne2GX 2015演讲，内容涉及Spring Boot如何使其适用于一流的工具
  • 我们的朋友Baeldung再次回来了， 这次带了一个有关监视Spring Boot应用程序的视频
  • http://tomaszdziurko.pl/2016/01/dockerizing-spring-boot-application/
  • Neo4j 对我们的GraphConnect 2015演讲有很好的转录
  • 关于使用Spring Boot和Spring Cloud构建微服务的西班牙语系列文章的第三篇文章看起来不错！
  • 我在QCon Rio de Janeiro上使用Spring Boot和Spring Cloud构建云本机Java应用程序的演讲
  • 我真的挖了我们的朋友Nicolas Frankel 关于将Spring Boot和Vaadin与Kotlin编程语言一起使用的帖子。Kotlin是一种非常好的语言，这篇文章涵盖了在使用Spring和Kotlin的第一步中会遇到的一些问题。非常便利！
  • Groovy 还是一种非常不错的编程语言，如果您需要进一步的证明，那么您只需要看一下这篇文章即可证明使用几乎没有任何代码就能站住Spring Cloud Config Server！
