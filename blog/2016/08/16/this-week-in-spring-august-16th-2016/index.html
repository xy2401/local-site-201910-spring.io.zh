Spring 的这个星期-2016年8月16日

工程
乔什·朗（Josh Long）
2016年8月16日

欢迎来到春季本周的另一期！自从上次发言以来，我已经在会议上向伦敦，北京，上海和新加坡的客户介绍过-我现在在这里。明天，星期三，我将在新加坡春季聚会上发言-加入我的行列 ！已经好几天了！

  • Spring Data忍者Mark Paluch对Spring Cloud Vault进行了另一篇精彩的文章
  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 4.1.2
  • Spring Integration忍者Artem Bilan刚刚宣布了Apache Kafka 1.0.3的春季版
  • 看看这个平台， 了解如何在PayPal使用Spring Boot
  • 几周前，我喜欢我的好友Richard Seroter 对我们的SpringOne Platform活动的评论
  • 我喜欢卡米尔·富尼耶（Camille Fournier）在微服务的某些现实体系结构模式上的帖子，尤其是她对单体仓库或单体应用所面临的挑战及其与人类动力学的关系的观察。
  • Hatena博客上的Shitsua Miukiyuki Shiiba对Pivotal的Reactor项目进行了简要而必要的考察
  • 我在过去几年的工作中经常看到这一点，并认为这篇文章很有趣： 为什么企业重新发现PaaS
  • Cloud Foundry基金会最近发布了一些很酷的视频，包括“ Cloud Foundry支持容器” ，“ Grape Up重新审视Cloud Foundry：UDP，VoIP和其他！”。
