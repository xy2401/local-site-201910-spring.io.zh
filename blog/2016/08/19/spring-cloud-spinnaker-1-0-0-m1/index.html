工程
格雷格·特恩奎斯特
2016年8月19日

春季问候社区，

我很高兴发布Spring Cloud Spinnaker的第一个里程碑。Spring Cloud Spinnaker捆绑了连续交付的Spinnaker平台 ，并提供一键式安装程序，可让您将其安装到任何经过认证的Cloud Foundry提供商 。

在今年的SpringOne Platform 2016大会上，有两个关于Spinnaker的话题。如果您拥有早期发行版的访问权限，但错过了访问权限，则可以立即观看。否则，发布它们后，您可以在SpringDeveloper YouTube频道上捕获它们。

如果您的团队/会议/ JUG有兴趣了解更多有关Spinnaker的信息， 请与我联系 ，我们可以安排Google Hangouts。

干杯，
