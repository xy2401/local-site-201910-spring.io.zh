Spring 的这个星期-2016年8月30日

工程
乔什·朗（Josh Long）
2016年8月30日

欢迎来到 Spring 本周的另一期！这个星期我去了旧金山（我住的地方），向硅谷 Spring 用户小组致词。现在是到美丽的中国来带些Spring和Pivotal了（也许要花点时间！）

和往常一样，我们还有很多事情要做！

  • Spring Ninja Greg Turnquist刚刚宣布Spring Web Services 2.3.1和2.4.0现已可用
  • Spring Cloud忍者Marcin Grzejszczak刚刚宣布Spring Cloud Camden M1现已上市
  • Spring Cloud Data Flow忍者Thomas Risberg刚刚宣布了适用于Mesos 1.0的Spring Cloud Data Flow。RC2现在可用
  • Marius Bogoevici刚刚宣布了Spring Cloud Stream Brooklyn M1现已可用。此版本包含了新功能，包括对反应式编程的支持
  • Spring Cloud Data Flow忍者Eric Bottard刚刚宣布适用于Cloud Foundry 1.0的Spring Cloud Data Flow。已经去GA
  • 如果您使用的是Spring Security SAML，则应查看此安全更新
  • Spring Security负责人Rob Winch 刚刚宣布了Spring Security 4.1.3，其中包含所有修复程序
  • Spring Cloud团队的新成员Ryan Baxter 刚刚宣布了Spring Cloud Brixton SR5
  • Spring Integration的负责人Gary Russell刚刚宣布Apache Kafka 1.1.0 M1的Spring现在可用。
  • 我在新加坡参加了一个为时4小时的研讨会，介绍了Spring Boot，Spring Cloud，Spring Coud Data Flow，Spring Cloud Security和Spring Security OAuth。整个过程已记录下来，可在此处在线获取
  • 8月14日，Reno Chou参加了我在中国的Cloud Native Java演讲之一，并开始撰写一系列博客中讨论的一些信息。这是他的第一 ， 第二部分中文。里诺非常酷！
  • 我在Antoine的法语文章中详细介绍了他如何将Spring Petclinic应用程序从Spring 4迁移到Spring Boot
  • 凉！Vojtech Ruzicka对他通过Pivotal认证的Spring Web Application Developer考试的经历进行了很好的回顾
  • 我喜欢Tim Spann的关于使用Spring Boot从Apache Hive访问数据的帖子
