Spring Cloud Camden M1可用

发布
马尔辛·格热兹扎克（Marcin Grzejszczak）
2016年8月29日

我很高兴地代表团队宣布Spring Cloud Camden Release Train的Milestone 1今天上市。该版本可以在我们的Spring Milestone存储库中找到。我们进行了许多增强和错误修复！您可以查看Camden.M1 发行说明以获取更多信息 。

以下模块已作为Camden的一部分进行了更新。M1：

Spring Cloud Build        1.2.0.RELEASE
Spring Cloud Stream       Brooklyn.M1
Spring Cloud Bus          1.2.0.M1
Spring Cloud Config       1.2.0.M1
Spring Cloud Netflix      1.2.0.M1
Spring Cloud Consul       1.1.0.M1
Spring Cloud Contract     1.0.0.M2
Spring Cloud CLI          1.2.0.M1

合并的发行培训文档可在此处获得

而且，一如既往，我们欢迎反馈：在GitHub上 ，在Gitter上 ，在Stack Overflow上或在Twitter上。

通过BOM使用Maven入门（仅依赖管理）


    
        spring-milestones
        Spring Milestones
        http://repo.spring.io/milestone
        
            false
        
    


  
    org.springframework.cloud
    spring-cloud-dependencies
    Camden.M1
    pom
    import
  


  
    org.springframework.cloud
    spring-cloud-starter-config
  
  
    org.springframework.cloud
    spring-cloud-starter-eureka
  
  ...

或使用Gradle：

buildscript {
  dependencies {
    classpath "io.spring.gradle:dependency-management-plugin:0.6.0.RELEASE"
  }
}
repositories {
    maven {
        url 'http://repo.spring.io/milestone'
    }
}
apply plugin: "io.spring.dependency-management"

dependencyManagement {
  imports {
    mavenBom 'org.springframework.cloud:spring-cloud-dependencies:Camden.M1'
  }
}

dependencies {
    compile 'org.springframework.cloud:spring-cloud-starter-config'
    compile 'org.springframework.cloud:spring-cloud-starter-eureka'
    ...
}
