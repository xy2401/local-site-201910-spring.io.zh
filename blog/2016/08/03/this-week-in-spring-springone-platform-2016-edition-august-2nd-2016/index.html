春季本周-SpringOne Platform 2016版！ -2016年8月2日

工程
乔什·朗（Josh Long）
2016年8月3日

欢迎使用春季本周的另一非常特别的部分-本周Pivotal在SpringOne Platform 2016上生效。今年有更多的人，更多的主题和更多的赞助商！有关更多信息，请查看我们的SpringOne Platform概述博客 ！

如果您不在这里，那么恐怕并不是所有的演讲都可以在线进行，但是直到那时.. 希望您在这里 ！与往年一样，与SpringOne的比赛充满了令人敬畏的发行版本，在您等待在线演讲作为重播时，应该会让您忙得不可开交！让我们开始吧！

  • Spring Boot的共同负责人Phillip Webb 发布了Spring Boot 1.4 ， 正好赶上 SpringOne Platform！
  • ..和Spring Boot忍者Stephane Nicoll 宣布了Spring Boot 1.3.7
  • Spring Data忍者Mark Paluch发布了Spring Data Ingalls M1 ，其中包含许多新功能（已修复250张票！）包括用于转换子系统（Commons，MongoDB）中的属性访问，针对Spring Data Cassandra的Cassandra 3.0更新，
    支持Cassandra存储库的声明性查询方法，支持Redis geo命令，按示例查询的任意匹配模式，支持基于REST负载的XML和JSON投影等等 ！
  • Spring Framework 负责人Juergen Hoeller刚刚宣布了Spring Framework 5.0.0 M1
  • 春季网络大师Rossen Stoyanchev 在Spring Framework 5.0.0 M1中引入了反应式编程支持 -这已经有很长的时间了，绝对值得一看！
  • Janne Valkealahti刚刚宣布了针对Apache YARN 1.0.1的Spring Cloud Data Flow
  • Spring平台负责人Andy Wilkinson宣布了Spring IO Platform 2.0.7 。Spring IO Platform是一个Maven物料清单发行版依赖项，它管理不同库之间的依赖项，并使您可以专注于不使用依赖项版本玩乱子。
  • Spring Cloud Task主管Michael Minella刚刚宣布了Spring Cloud Task 1.0.2
  • Spring工具领导者Martin Lippert 刚刚发布了Spring Tool Suite 3.8.1 ，该工具修复了一些小问题。
  • 我非常喜欢Richard Seroter的博客 ， 该博客关于快速部署到生产中
  • 这很有趣，有一篇关于使用Excelsior JET的文章-这是针对Microsoft Windows，Apple OS X和Linux的Java SE技术的实现，它将代码转换为本地应用程序- 将Spring Boot应用程序转换为本地应用程序 。
  • 我喜欢这篇关于同时使用Spring和Akka的文章
