春天的这个星期-2016年3月8日

工程
乔什·朗（Josh Long）
2016年3月8日

欢迎来到春季本周的另一期！本周，我在德国布鲁尔（Bruhl）参加令人惊叹的JavaLand展览，然后去苏格兰拜访一个大客户，然后去罗马尼亚参加Voxxed Days布加勒斯特 。如果您在这些地方中的任何一个， 我都想联系 ！

  • Spring Ninja Stephane Nicoll介绍了Spring Framework 4.3中的一些细节
  • 不要错过Micah Silverman即将举行的网络研讨会， 该研讨会介绍了令人惊叹的Stormpath和基于Spring / Spring Boot的集成和自动配置 。Stormpath是一种史诗般的身份提供者即服务。它是基于云的应用程序的理想选择。
  • Spring IO Platform负责人Andy Wilkinson刚刚宣布了Spring IO Platform 2.0.3
  • 集成负责人Gary Russell在SpringOne2GX 2015上向我们展示了RabbitMQ如何为Spring XD提供动力
  • 同样来自SpringOne2GX 2015的Thomas Risberg向我们展示了Hadoop工作流程的简化和发行版的可移植性如何使原始Hadoop对于Spring开发人员而言更加简单。查看重播！
  • Petar Tahchiev在此SpringOne2GX 2015重放中将Springdom应用于电子商务的问题领域，即使用Spring构建下一代云电子商务平台
  • JavaCodeGeeks发布了Srujana Cherukuri 关于使用Spring Boot的精彩介绍
  • 是否想在Openshift上使用Spring Boot ？
  • 我挖了这篇文章，解释了如何从Neo4J 4.0升级到4.1
  • Spring Data Redis 现在支持Redis集群
  • 即将到来的JUnit 5中有很多不错的东西：抓紧时间，踢轮胎然后走！
  • 如果您使用的是Apache Wicket，那么您可以在InfoQ 和 上进行讨论 ，正如我们在该文章中了解到的那样，显然还有Spring Boot自动配置！自动配置似乎有点奇怪，因为它取代了您在Spring Boot应用程序中看到的一些常规内容，但是看起来很容易上手。
  • 我们正在计划大型SpringOne平台会议，这将是惊人的！我们希望您能加入我们！如果您有故事要讲，那么请务必提交您的演讲！ 在3/24之前！
