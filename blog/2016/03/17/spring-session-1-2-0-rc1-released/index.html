Spring Session 1.2.0 RC1 Released

Releases
Rob Winch
March 17, 2016

On behalf of the community, I’m pleased to announce the release of Spring Session 1.2.0.RC1. The release can be found in the Spring Milestone repository (https://repo.spring.io/milestone).

Change Log

This release includes nearly 30 issues with some exciting new features. You can find a complete list of changes in the change log. The highlights can be found in What’s New in 1.2

Contributions

I continue to be amazed with the growing Spring Session community. It is because of our community that we were able to do this release within a month of 1.1.0 going GA. A special thank you to all those who contributed Pull Requests for this release:

  • #371 - Add MongoOperationsSessionRepository. Thanks Jakub Kubryński
  • #379 - Add JdbcOperationsSessionRepository. Thanks Vedran Pavić
  • #397 - Fix minor documentation errors. Thanks Vedran Pavić
  • #402 - @DirtiesContext for GemFire integration tests. Thanks John Blum
  • #404 - Polish GemFire. Thanks John Blum
  • #413 - Add Checkstyle to build. Thanks Vedran Pavić
  • #414 - Fix bad reference in IndexDocTests. Thanks Vedran Pavić
  • #418 - Add JDBC documentation. Thanks Vedran Pavić
  • #421 - Fix JdbcOperationsSessionRepository session cleanup. Thanks Erin Drummond
  • #425 - JacksonMongoSessionConverter. Thanks Jakub Kubryński
  • #426 - SessionRepositoryFilter caches null session lookup. Thanks Øyvind Horneland
  • #428 - Fix incorrect link in What’s new section. Thanks Vedran Pavić
  • #429 - Update to Sonarqube Plugin. Thanks Eddú Meléndez Gonzales
  • #430 - Extend Mongo documentation. Thanks Jakub Kubryński
  • #433 - JdkMongoSessionConverter supports custom ClassLoader. Thanks Jakub Kubryński
