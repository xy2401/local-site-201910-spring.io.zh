Spring Security 4.1.0。RC1发布

发布
罗伯·温奇
2016年3月24日

我很高兴代表社区宣布Spring Security 4.1.0的发布。RC1。此版本解决了100张票 。您可以在下面找到一些重点：

  • Web安全表达式中的路径变量
  • 内容安全政策（CSP）
  • HTTP公钥固定（HPKP）
  • 添加ForwardAuthenticationFailureHandler和ForwardAuthenticationSuccessHandler
  • SCrypt支持SCryptPasswordEncoder
  • 简化的UserDetailsService Java配置
  • 简化的AuthenticationProvider Java配置
  • 移至GitHub问题
  • 测试元注释
  • 方法安全性元注释
