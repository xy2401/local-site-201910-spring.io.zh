This Week in Spring - March 1, 2016

Engineering
Josh Long
March 01, 2016

Welcome to another installment of This Week in Spring! This week, I’m in Shenzhen, China and Zurich, Switzerland for Voxxed Days Zurich. It’s going to be a busy week indeed though I sure hope to see you there! Anyway, we’ve got a lot to cover this week so let’s get to it!

  • Spring Boot ninja Stéphane Nicoll has just announced Spring Boot 1.4.0M1 with excellent couchbase support
  • Spring Session lead Rob Winch, never resting on his laurels, just announced Spring Session 1.1.0
  • Also not resting on his laurels, Stéphane just announced Spring Framework 4.2.5, chock full ’o bug fixes and updates
  • Spring XD lead Mark Pollack has just announced that Spring XD 1.3.1 is now GA, along with Spring Flo for XD 1.0.1 GA
  • Martin Lippert, the lead developer on Spring Tool Suite, has added better support for both Spring Boot and Cloud Foundry, as well as picking up the latest Eclipse Mars distro, in STS 3.7.3
  • Webinar Replay: Creating a PageRank Analytics Platform Using Spring Boot Microservices
  • Webinar Replay: Spring Boot Authentication with Stormpath… and More!
  • Artem Bilan takes us through his SpringOne2GX 2015 talk, Spring Integration Java DSL
  • Mark Fisher and Patrick Peralta give us the first ever glimpse at Spring Cloud Data Flow in this SpringOne replay: Cloud Native Dataflow Orchestration
  • XML? Java Configuration? Annotations? Watch them duke it out in this SpringOne replay -Spring Framework - the ultimate configuration battle
  • A great overview of the state of the streaming open source ecosystem from Michael Minella and Gunnar Hillert, followed by a in - depth look at Who needs batch these days?
  • Rossen takes us through the latest in Spring MVC 4.2, New and Noteworthy
  • Our pal Geertjan, as part of last week’s epic Spring Boot and Netbeans webinar, has released the Spring Boot tools for Netbeans extensions. This is super cool!
  • I loved Spring ninja and team member Greg Turnquist’s post, Spring Boot is still a gem.. waiting to be discovered
