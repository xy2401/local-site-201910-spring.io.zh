Spring 的这个星期-2016年3月1日

工程
乔什·朗（Josh Long）
2016年3月1日

欢迎来到 Spring 本周的另一期！本周，我将在中国深圳和瑞士苏黎世参加“苏黎世Voxxed Days”活动 。尽管我肯定希望能在这里见到你，但实际上这将是忙碌的一周！无论如何，本周我们有很多工作要做，让我们开始吧！

  • Spring Boot忍者StéphaneNicoll刚刚发布了Spring Boot 1.4.0M1，具有出色的沙发床支持
  • Spring 会议的负责人罗布·温奇（Rob Winch）从来没有为之动容，只是宣布 Spring 会议1.1.0
  • Stéphane也不感到惊讶，他刚刚宣布了Spring Framework 4.2.5，全面修复和更新了“ o”错误
  • Spring XD负责人Mark Pollack刚刚宣布将Spring XD 1.3.1以及现在用于XD 1.0.1 GA的Spring Flo正式发布。
  • Spring Tool Suite的首席开发人员Martin Lippert增加了对Spring Boot和Cloud Foundry的更好支持，并在STS 3.7.3中获得了最新的Eclipse Mars发行版。
  • 网络研讨会重播： 使用Spring Boot微服务创建PageRank分析平台
  • 网络研讨会重播： 使用Stormpath进行Spring Boot认证…等等！
  • Artem Bilan带领我们完成了SpringOne2GX 2015演讲， Spring Integration Java DSL
  • 马克·费舍尔（Mark Fisher）和帕特里克·佩拉尔塔（Patrick Peralta）在SpringOne重播中首次展示了Spring Cloud Data Flow： Cloud Native Dataflow Orchestration
  • XML？Java配置？注释？观看他们在SpringOne重放中将其发布-Spring Framework-最终配置大战
  • 概述了Michael Minella和Gunnar Hillert的流式开源生态系统的状态，然后深入研究了这些天谁需要批量处理？
  • Rossen带我们了解Spring MVC 4.2的最新特性（新特性和值得注意的特性）
  • 作为上周史诗般的Spring Boot和Netbeans网络研讨会的一部分，我们的好朋友Geertjan已发布了用于Netbeans扩展的Spring Boot工具 。这太酷了！
  • 我喜欢Spring忍者和团队成员Greg Turnquist的帖子， Spring Boot仍然是一颗宝石..等待被发现
