Spring 的这个星期-2016年3月22日

工程
乔什·朗（Josh Long）
2016年3月22日

欢迎来到 Spring 本周的另一个装置！和往常一样，我们有很多内容需要介绍，让我们开始吧！我本周应该在巴黎和比利时。不幸的是，由于最近发生在布鲁塞尔机场和地铁上的悲剧性，令人恶心和怯ward的袭击 ，我不确定我是否会身处比利时。但是，今晚的见面会演讲将 通过流媒体远程进行 。请继续关注我的Twitter句柄-@starbuxman，以获取更多信息。

也就是说，有很多很棒的内容，所以让我们开始吧！

  • Spring Data负责人Oliver Gierke刚刚宣布了Spring Data Hopper的第一个候选版本 。
  • 我们当地的Spring和Kotlin忍者Sebastien Deleuze在他的示例应用程序GeoSpatial Messenger中创建了一个博客， 展示了Spring和Kotlin的一些有趣应用 。
  • Spring Security负责人Rob Winch 宣布了Spring Security SAML 1.0.2 。
  • Spring Tool Suite的负责人Martin Lippert继续他的系列文章，介绍如何使用Spring Tool Suite中的Spring Boot和Cloud Foundry 。本期文章主要介绍Cloud Foundry清单。
  • Spring Security和Spring Session的负责人Rob Winch刚刚宣布了两项新裁员： Spring Session 1.2.0。RC1和Spring Session 1.1.1 。
  • New Relic的Ashely Puls在她的SpringOne2GX 2015演讲中深入探讨了JVM- 及时发布意味着什么
  • Matt Raible的Epic框架smackdown演讲又回来了，听他比较ReactJS，AngularJS，EmberJS
  • InnQ的Michael Ploed讨论了使用事件源和CQRS构建 SpringOne2GX 2015形式的微服务
  • Spring Boot 太神奇了吗？不是。这只是一个抽象，很容易推断您是否了解抽象。我们的好伙伴Siva Prasad Reddy撰写了一篇不错的文章， 拉开了Spring Boot自动配置工作原理的帷幕
  • 我们来自Heroku的朋友又回来了，再次发表了一篇关于使用Spring Cloud的精彩文章， 这次是关于使用Spring Cloud的Zuul集成的 。
  • 我们的朋友Biju Kunjummen 在关于使用Cloud Foundry部署Spring Boot应用程序的精彩文章中
  • 我认为很多人都对如何使用Spring Cloud Hystrix断路器集成编写有效的服务客户存有疑问，而且-令人惊讶的是-本文档做得很好 ！
  • 我将在4月21日的达姆施塔特水罐（Darmstadt JUG）上演讲，快来加入我吧！
  • 看看JHipster 3.0的新功能，它与深度Spring Cloud集成完整！
