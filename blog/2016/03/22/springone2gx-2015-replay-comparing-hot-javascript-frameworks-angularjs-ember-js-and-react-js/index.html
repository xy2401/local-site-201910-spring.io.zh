SpringOne2GX 2015重播：比较热门JavaScript框架：AngularJS，Ember.js和React.js

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年3月22日

记录在SpringOne2GX 2015中。
演讲者：Matt Raible
Web / JavaScript跟踪
幻灯片： http ： //www.slideshare.net/SpringCentral/comparing-hot-javascript-frameworks
如今，JavaScript MVC框架风靡一时。他们席卷了Web开发世界。本课程探讨了三个最热门的JavaScript MVC框架的各种功能：AngularJS，Ember.js和React.js。它还比较了客户端模板与服务器模板，以及每个框架对同构JavaScript的支持程度（可以同时在客户端和服务器端运行的代码）。最后，我们将使用Yevgeniy Brikman的计分卡按照10个不同的标准对每个框架进行排名。
