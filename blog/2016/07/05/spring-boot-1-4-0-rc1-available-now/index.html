Spring Boot 1.4.0。RC1现已上市

发布
斯特凡·尼科尔（StéphaneNicoll）
2016年7月5日

我很高兴地宣布， Spring里程碑存储库中现已提供Spring Boot 1.4的候选发布版本 。这个里程碑结束了1.4线的6个月的工作，结束了140多个问题并提出了请求 ！感谢大家的贡献。

新版本的重点包括：

  • 统一@EntityScan for JPA，MongoDB，Neo4j，Couchbase和Cassandra
  • 自动配置的RestTemplateBuilder
  • 通过@RestClientTest支持纯休息客户端测试
  • 对Jest的支持（Elasticsearch Rest Client）
  • 升级到Spring Integration 4.3，Spring AMQP 1.6，Spring REST Docs 1.1，MongoDB Java Driver 3等

有关更改的完整列表和升级说明，请参阅WIKI上的《 Spring Boot 1.4发行说明 》和更新的参考文档 。

在这一点上，给我们1.4.0.RC1
