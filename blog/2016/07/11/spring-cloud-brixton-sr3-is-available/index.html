春天的云彩布里克斯顿。SR3可用

发布
瑞安·巴克斯特（Ryan Baxter）
2016年7月11日

我很荣幸地代表团队宣布Spring Cloud Brixton Release Train的Service Release 3现已发布。该版本可以在我们的Spring Release存储库和Maven Central中找到 。

此版本中包括Spring Cloud Task的GA版本 。

除了添加Spring Cloud Task之外，此版本主要包括错误修复。

以下模块已作为Brixton的一部分进行了更新。SR3：

  • Spring Cloud Security 1.1.2。发布
  • Spring Cloud Netflix 1.1.4。发布
  • Spring Cloud Sleuth 1.0.3。发布
  • Spring Cloud Task 1.0.1。发布

而且，一如既往，我们欢迎您提供反馈：在GitHub ， gitter ， Stack Overflow或Twitter上 。

Spring Cloud团队将在Spring One Platform上讨论Spring Cloud，该平台将于今年8月1-4日在拉斯维加斯举行。还有许多其他精彩的演讲，因此请检查议程并获得入场券（如果您尚未这样做的话）。
