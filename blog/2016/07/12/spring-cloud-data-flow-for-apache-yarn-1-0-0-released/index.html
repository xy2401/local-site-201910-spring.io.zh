适用于Apache YARN 1.0.0的Spring Cloud Data Flow发布

发布
珍妮·瓦尔基亚拉赫蒂
2016年7月12日

我们很高兴地宣布针对Apache YARN 1.0.0发布Spring Cloud Data Flow 。发布。

适用于Apache YARN的Spring Cloud Data Flow提供了对在Apache YARN上协调长时间运行（流）和短期（任务/批处理）数据微服务的支持。

该项目最初是概念化的，目的是补充现有的Spring XD用户，这些用户在传统的裸机设置中投资运行流和批处理数据管道。我们希望提供迁移路径，以帮助移植他们现有的投资以及利用微服务风格架构的好处的工具和技术。

除了API清理和稳定之外，我们还在此主要版本中研究了以下主题。

  • 建立在1.0.0之上。发行Spring Cloud Data Flow Core（shell，UI，REST-API等）
  • 添加扩展基础架构以支持流分区
  • Spring Cloud Data Flow的Amabri插件的多项改进
  • 将迁移指南和其他文档片段添加到参考文档中
  • 添加项目站点
  • 添加了入门指南以在HDP 2.4沙箱上引导Spring Cloud Data Flow

有关更改和改进的完整列表，请参阅1.0.0。发布服务器版本和1.0.0。发布部署程序版本 。

Spring Cloud Data Flow团队将在Spring One Platform上进行演讲，该平台将于今年8月1-4日在拉斯维加斯举行。还有许多其他精彩的演讲，因此请检查议程并获得入场券（如果您尚未这样做的话）。
