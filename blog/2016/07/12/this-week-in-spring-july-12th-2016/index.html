春天的这个星期-2016年7月12日

工程
乔什·朗（Josh Long）
2016年7月12日

欢迎来到春季本周的另一期！本周，我在爱尔兰的都柏林，英国的伦敦以及马萨诸塞州的波士顿，在都柏林的枢纽和伦敦举行公开研讨会，并与当地客户交谈。这是一个有趣的一周，有很多内容要讲！所以，让我们开始吧！

这是我一年中最喜欢的时间！当我们进入SpringOne平台时 ，发布了太多好东西，几乎跟不上它！我真的很期待今年8月初的SpringOne平台展示。这是构建应用程序的绝佳时机，SpringOne Platform处于独特的位置，可以吸引更大的讨论：为什么我们要＃devops，＃cloud，＃agile或#microservice？参加2016年8月1-4日在美丽的拉斯维加斯的讨论，让我们找出答案！

所以，让我们开始吧！

  • Spring Cloud Data Flow 1.0已发布 ！欢呼！！也有YARN ， Cloud Foundry和Kubernetes适配器。
  • Dzone要求我整理一下有关使用Java构建微服务的参考，上周它上线了。在本裁判中，我们讨论了微服务背后的动机，以及在分布式系统（微服务）中构建有价值的服务时需要解决的问题。自然，我们谈论的是Spring Cloud，但我也提到了可用的竞争产品。
  • 第一个春天启动1.4的候选发布版现在可用， 挤满了很多伟大的事情！此新版本的功能包括统一@EntityScan适用于JPA，MongoDB，Neo4j，Couchbase和Cassandra（自动配置） RestTemplateBuilder ，通过支持纯REST客户端测试@RestClientTest ，支持Jest（Elasticsearch REST客户端），并升级到Spring Integration 4.3，Spring AMQP 1.6，Spring REST Docs 1.1，MongoDB Java Driver 3 等 ！立即下载并踢轮胎！
  • Spring Batch负责人和Spring Cloud Task负责人Michael Minella刚刚宣布了Spring Cloud Task 1.0.1。RELEASE ，是Spring Cloud Task的第一个推荐的GA版本。该博客非常漂亮地介绍了导致Spring Cloud Task的演变及其背后的动机，以及Spring Cloud Task本身所有令人兴奋的功能！伟大的阅读，检查出来。
  • Spring Tool Suite引领着全能的人Martin Lippert刚刚宣布，基于Eclipse Neon的Spring Tool Suite 3.8.0现已发布 。Eclipse Neon是一个很棒的更新，其中包含新功能，并且Spring Tool Suite在Eclipse Neon之上集成了同类最佳，稳定和集成的功能，并向其中添加了新的Spring Boot代码模板和改进的Spring Boot。 .yml文件配置和编辑，以及Spring Boot本身的大量工作效率增强。
  • Spring Security主管Rob Winch发布了Spring Security 4.1.1。RELEASE已发布。它包含针对50多个问题的解决方案，缓解了CVE-2016-5007和一些显着的增强功能： MvcRequestMatcher提供与Spring MVC的深度集成，以确保授权规则的路径匹配策略一致，可以利用Spring MVC的CORS配置的CORS支持，以及CookieCsrfTokenRepository.withHttpOnlyFalse()便于AngularJS应用程序集成。
  • Spring IO Platform的领导者和所有的伟人Andy Wilkinson 刚刚宣布了Spring IO Platform 2.0.6，该版本更新了许多库
  • Spring Security负责人Rob Winch刚刚发布了CVE 2016-5007的修复程序，该修复程序解决了Spring MVC和Spring Security路径匹配不一致的问题
  • Spring Integration 忍者Artem Bilan刚刚宣布了Spring Integration Java DSL 1.2。M1和1.1.3，除其他外，还支持Apache Kafka 0.9和流的运行时注册
  • 我们最新的Spring Cloud团队成员Ryan Baxter 刚刚宣布Spring Cloud SR3现已可用，并且是建议的更新 。
  • 我真的很喜欢Laurent Doguin 关于将 Ratpack 与Spring Boot应用程序结合使用的精彩文章
  • 是否想知道像Twitter，Google和Mozilla这样的不同公司如何偏爱他们的Java代码样式 ？
  • 选择PaaS时，我真的很喜欢这种不同的体系结构考虑 。 大礼帽给理查德·塞罗特 ！
  • Spring Cloud Services的新功能-在Cloud Foundry上支持Spring Cloud OSS服务
    充满了很酷的功能和更新的兼容性，绝对值得一看。它包括与Spring Cloud Brixton的发行版兼容性，Spring Boot 1.3，异步服务配置和零宕机时间升级/更新，Config Server和Service Registry的高可用性拓扑，Config Server Git Backend的增强功能等等！趁热拿到！
