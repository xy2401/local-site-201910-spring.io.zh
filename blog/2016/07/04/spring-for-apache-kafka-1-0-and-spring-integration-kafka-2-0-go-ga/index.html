适用于Apache Kafka 1.0的Spring和适用于Spring Kafka 2.0的Spring go

发布
阿尔特姆·比兰（Artem Bilan）
2016年7月4日

我很高兴地宣布， spring-kafka （Apache Kafka的Spring） 1.0 GA和Spring Integration Kafka 2.0 GA终于上市了！

工件org.springframework.kafka:spring-kafka:1.0.0.RELEASE和org.springframework.kafka:spring-kafka-test:1.0.0.RELEASE 版本库和Maven Central中分别提供了RELEASE 。

另外org.springframework.integration:spring-integration-kafka:2.0.0.RELEASE ，完全基于spring-kafka-1.0.0.RELEASE ，可在相同的存储库，以及。

重要
 刚发布后，我们发现了一个令人讨厌的错误 ，它在RECORD模式下具有未确认的偏移提交。特别感谢Adam Dec ！所以，请见spring-kafka:1.0.1.RELEASE 。

当然，非常感谢参与其中的每个人，也感谢积极的社区成员，他们提供了功能要求和贡献。

自从发布候选版本1开始，已经进行了一些返工和错误修复，其中包括：

  • 对Manual AckMode进行返工，尽管由于MANUAL_IMMEDIATE_SYNC模式已被删除，它仍指示出重大变化；

  • ackTime现在有5秒作为默认值。与Kafka默认的auto.commit.interval.ms相同；

  • 如果是COUNT和COUNT_TIME模式，则ackCount必须> 0 ；

  • 参考手册中新的Spring Integration Kafka支持章节。

有关更多信息，另请参见参考手册 。

下一步

我们希望尽快发布1.1版Milestone 1，它将支持Kafka 0.10，包括一些Spring Boot自动配置和执行器端点等等！
