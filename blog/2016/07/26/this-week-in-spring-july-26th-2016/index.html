春天的这个星期-2016年7月26日

工程
乔什·朗（Josh Long）
2016年7月26日

欢迎来到春季本周的另一个装置！这周我主要在旧金山和圣路易斯，忙着准备下周的大型活动！

这是我一年中最喜欢的时间！当我们进入SpringOne平台时 ，发布了太多好东西，几乎跟不上它！我真的很期待今年8月初的SpringOne平台展示。这是构建应用程序的绝佳时机，SpringOne Platform处于独特的位置，可以吸引更大的讨论：为什么我们要＃devops，＃cloud，＃agile或#microservice？参加2016年8月1-4日在美丽的拉斯维加斯的讨论，让我们找出答案！

我们有很多工作要做，而且时间很少，所以让我们开始吧！

  • Spring Cloud Contract 负责人Marcin Grzejszczak刚刚宣布M1现在可用 。该博客还很好地了解了消费者驱动的合同背后的概念，因此非常值得一读！
  • Spring Cloud Data Flow产品经理Sabby Anandan 在Spring Cloud Data Flow 1.0上为Pivotal博客撰写了一篇非常不错的文章 。
  • 查看Spring Cloud Data Flow领导的Pollack博士和Mark Fisher的史诗性演讲，以了解有关数据驱动微服务的更多信息
  • Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration AWS 1.0.0。现在可以发布 。对于所有使用Spring Integration AWS M1的用户，建议使用最终版本
  • Artem 还从未宣布过赞叹不已， 还 宣布推出Spring Integration 4.3.1、4.2.9、4.1.9、4.0.9。
  • 您要在Cloud Foundry上使用Hazelcast吗？在这里查看服务代理及其功能信息
  • Spring REST Docs领导者Andy Wilkinson 刚刚宣布了Spring REST Docs 1.1.1 ，该版本具有许多小的改进和错误修正 。
  • Rossen Stoyanchev刚刚宣布Spring Web Flow 2.4.4现已可用。这是一个维护版本，其中包括确保与Hibernate 5.2的兼容性以及修复JSF中的一些小问题。
  • 优秀的Syer博士在他的系列文章中发布了另一篇有关构建响应式应用程序的文章， 这次着眼于构建一个简单的响应式HTTP服务器
  • 克里希纳·斯里尼瓦桑（Krishna Srinivasan）很好地研究了如何使用某些Spring Boot Actuator模块的暴露端点
  • 我喜欢他们在Stormpath所做的事情， 这个最新的截屏视频很好地说明了为什么
  • 您是否知道Pivotal提供了一些课程来学习有关Pivotal Cloud Foundry和Spring Cloud Services的信息 ？
  • CAS项目第5系列，继续前进到GA，并在其最新版本M3中进行代码的全面Spring Bootification 。
