Spring Data 发布系列Ingalls M1发布

发布
马克·帕鲁奇
2016年7月27日

我很高兴代表Spring Data团队宣布Ingalls发布培训的第一个里程碑。该版本已发行230张门票 ！最值得注意的新功能是：

  • 使用方法句柄在转换子系统（Commons，MongoDB）中进行属性访问。
  • 升级到Cassandra 3.0 for Spring Data Cassandra（有关详细信息，请参阅更新的示例 ）。
  • 支持Cassandra存储库的声明性查询方法。
  • 支持Redis地理命令。
  • 示例查询的任意匹配模式。
  • 支持REST有效负载的基于XML和JSON的投影（有关详细信息，请参见示例 ）

在我们的发行培训Wiki中找到经过策划的变更日志，或浏览JIRA中变更的完整列表 。

达到里程碑的最简单方法是声明spring-data-releasetrain.version财产Ingalls-M1在您的Spring Boot应用程序中。如果您使用的是Spring Data REST，请确保设置spring-hateoas.version财产0.21.0.M1 。Spring Data Redis用户需要升级到Jedis 2.9.0（引导用户设置了jedis.version属性）。如果您是Spring Data Cassandra用户，则需要将Cassandra驱动程序升级到3.0.x（引导用户将cassandra-driver.version属性）。另外，请查看我们更新的Spring Data示例 ，以获取有关集成特定组件的更多示例。
