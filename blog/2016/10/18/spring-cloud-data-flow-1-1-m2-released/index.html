Spring Cloud Data Flow 1.1 M2发布

工程
马克·波拉克
2016年10月18日

我很高兴代表团队宣布发布Spring Cloud Data Flow 1.1的第二个里程碑。您可以在此处下载此版本一部分的本地服务器。

1.1 M2版本包含以下新功能和改进：

  • 在Boot 1.4.1和Spring Cloud Camden改进的基础上构建

  • 现在可以使用非前缀属性名称引用任务应用程序属性

  • 为相关流添加视觉表示。此表示形式还包括嵌套的TAP和整个拓扑视图中的下游处理节点。

  • UI添加了对任务定义的批量导入支持 。这允许从文件导入任务定义，或者使用Flo-editor添加多个任务定义的功能。

  • 浮动编辑器嵌入CodeMirror，用于增量DSL语法验证和错误报告。还有一个开/关按钮来启用和禁用增量验证。

  • 为非OAUTH后端身份验证方法（例如LDAP，基本和基于文件的选项）添加基于表单的登录页面。

  • 添加了通过YAML文件传递应用程序特定属性的功能 。当部署具有许多部署属性的流时，这特别有用。

  • 合并以下Shell命令以实现顺序和一致性。将“任务显示”更改为“任务执行状态”，删除“任务状态”， 并对“ app info ： ”和“ app unregister ： ”使用： 语法。

查看1.0.1。M2版本标记，以了解有关增量改进的更多信息。
