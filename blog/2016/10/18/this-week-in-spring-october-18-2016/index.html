Spring 的这个星期-2016年10月18日

工程
乔什·朗（Josh Long）
2016年10月18日

欢迎来到 Spring 本周的另一期！这又是疯狂的美好一周。本周，我将在伦敦和英国剑桥与客户进行访问，并进行演讲（尤其是在O'Reilly 软件体系结构大会上 ）。如果您要打个招呼！现在，让我们开始吧！

  • 由克里斯·斯特林 （ Chris Sterling）领导的Spring Cloud Services团队刚刚使Eureka Service Discovery在Pivotal Cloud Foundry上的工作更加出色。新的1.2版本允许跨安装和组织进行对等复制，从而实现多站点复制。
  • Spring Cloud Data Flow 1.1。M2今天刚刚发布，它选择了Boot 1.4.1和Spring Cloud Camden版本。在Mark Pollack的博客文章中查看所有其他好处。
  • 通过这次令人难以置信的网络研讨会，学习如何使用Spring Cloud Data Flow构建以数据为中心的微服务
  • Spring Integration忍者Artem Bilan刚刚宣布了用于Spring Integration 1.2的Java DSL
  • 上周，我整理了一下快速浏览-Spring Tips系列的另一部分！ -关于如何使用Spring Integration （和上述Java DSL！） 构建应用程序集成流程 。请享用。
  • 我真的很喜欢InfoQ的这篇文章Edith Harbaugh 所讲的系统中功能标记的危险和潜力
  • 您是否在macOS Sierra上并且在IntelliJ IDEA中滚动时遇到问题？请查看此超级有用的技巧以解决问题！
  • 使用Java 8构建API时，我喜欢这篇关于设计原则的文章
  • 是否想了解响应式Spring Security的未来？在Spring Security团队开始工作时 ， 请 遵循此存储库，但是请注意一切 ！
  • 大众，又是一家获得此 殊荣的汽车公司， 选择了Pivotal Cloud Foundry将周期时间从几年的数量级降低到一周的时间（如果他们能够将其进一步降低，我也不会感到惊讶！）
  • 我们的朋友迈克尔·西蒙斯（Michael Simmons）撰写了一篇很棒的参考书@MockBean在 Spring 测试中按名称
  • 你在日本吗不要错过这个 Spring 活动！
  • Spring Data负责人Oliver Gierke在构建可演化系统的权衡取舍上发表了一篇不错的文章。
  • 乔纳斯赫克特已发布但在他的系列另一分期付款寻求如何与CXF和Spring，这个时候开张工作好看 Spring 开机自动配置和启动器，简化了相当的东西
  • ZeroTurnaround 发布了他们对Java社区的最新调查的数据混合 ，这次细分了调查参与者中对不同技术（Spring和Java EE）的使用。
  • TheNewStack又回来了，这一次是一篇名为Cloud Foundry的文章，在不断变化的平台环境中仍然是国王 。有趣的分析！
