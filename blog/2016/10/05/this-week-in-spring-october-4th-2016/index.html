    春天的这个星期-2016年10月4日

    工程
    乔什·朗（Josh Long）
    2016年10月5日

    欢迎来到春季本周的另一期！我们有很多东西可以看，所以让我们开始吧！

    • Spring Security主管Rob Winch 刚刚发布了Spring Security LDAP 2.2RC1
    • Spring Integration的负责人Gary Russell 刚刚宣布了Spring Integration和Spring AMQP的发布候选人 ；现在得到的位！
    • Spring Data忍者Thomas Risberg刚刚发布了Kubernetes 1.1的Spring Cloud Data Flow。M1
    • Spring Data 负责人Oliver Gierke宣布了Spring Data Hopper SR4和Spring Data Gosling SR6
    • Spring Statemachine负责人Janne Valkealahti 刚刚宣布了Spring Statemachine 1.2.0M1
    • 我喜欢Jenn Strater的GR8Conf演讲，该演讲涉及一种测试驱动的方法来使用Spring REST Docs记录REST API 。
    • 这次德国新闻综述如此简短地着眼于Spring 5和Rod Johnson的新举措Atomist
    • JAXenter也采访了我
    • 在这篇不错的InfoQ帖子中查看Netflix Zuul未来的发展趋势
    • EMC的Paul Warren可能会挖掘一些有趣的项目： 检查Ginkgo4j使用Java进行BDD和Ginkgo样式的测试，并检查他的Spring Data扩展以支持任意Blob存储

    另外，我想指出今天Pieter Hintjens所遭受的损失，Pieter Hintjens在为世界做出的许多贡献中，是将成为RabbitMQ的AMQP协议的原始设计者。彼得，安息吧。
