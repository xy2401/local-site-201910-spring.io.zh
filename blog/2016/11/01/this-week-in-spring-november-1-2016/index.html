这个星期在 Spring - 2016年11月1日

工程
乔什·朗（Josh Long）
2016年11月1日

您好，欢迎回到 Spring 本周的另一分期付款！你能相信我们现在2周是到2017年 - 时间肯定是在飞！这周我在美国加利福尼亚的洛杉矶，然后去伦敦参加2016年的ConCon ， 比利时的Devoxx和Oredev 。如果您参加任何这些活动，请务必打个招呼！现在，事不宜迟，让我们开始吧！

  • 首先，请务必查看Marcin Grzejszczak撰写的有关Spring Cloud Contract的网络研讨会
  • Spring Data忍者Mark Paluch刚刚宣布了Spring Vault和Spring Cloud Vault 1.0.0。M1现在可用
  • Spring Security主管Rob Winch宣布发布Spring Security 4.2.0。RC1具有许多新功能和更新。
  • 在上周的“ Spring Tips”一期中，我研究了Spring中的断路器， 包括Hystrix断路器和Spring Retry中的新断路器功能。
  • 相关，Spring Cloud团队成员Ryan Baxter分享了重试功能区负载平衡请求的新支持
  • Spring Cloud Pipelines负责人Marcin Grzejszczak刚刚宣布了Spring Cloud Pipelines M2
  • Spring Security和Spring Security LDAP 2.2.0负责人Rob Winch刚刚宣布了Spring LDAP 2.2.0
  • 在Pivotal工程博客上查看有关构建Spring Cloud Data Flow接收器的文章
  • Cloud Foundry基金会正在帮助标准化 Cloud Foundry 的服务代理模型，使其适用于Kubernetes
  • 我们的朋友@make 为Concourse CI整合了一项有用的一次性任务，以运行Flyway数据库迁移
  • 基于Spring Boot的Grails Framework 3.2.2现在可用
  • 我们认为，这个可打印的ZeroTurnaround备忘单详细介绍了一些常见的Spring生态系统注释及其帮助方式，但我们认为其中一些注释的解释不充分。
