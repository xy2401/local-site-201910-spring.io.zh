春天的这个星期-2016年11月8日

工程
乔什·朗（Josh Long）
2016年11月8日

欢迎来到春季本周的另一期！本周我将在英国伦敦的µCon ，在比利时安特卫普的Devoxx和在瑞典的马尔默的Oredev 。如果您在附近，请打个电话给我打个招呼 。

  • Spring Framework和Spring Boot忍者Stephane Nicoll刚刚宣布，现已提供Spring Framework 4.3.4（主要是错误修复） 。
  • Spring Security忍者Joe Grandja刚刚宣布了Spring Security OAuth 2.0.12 ，它主要是错误修复，并为一些不错的新功能铺平了道路。
  • Spring Cloud Data Flow和Spring Cloud Task忍者Glenn Renfro刚刚宣布了Spring Cloud Task 1.1.0。RC1 。Spring Cloud Task继续了1.1的旅程，它增加了许多新功能，包括错误处理，用于自定义分区作业工人启动的改进选项。
  • Spring Data负责人Oliver Gierke刚刚宣布了Spring Data发布火车Hopper SR5 。此版本包括所有模块的大量更新，包括JPA，KeyValue，Cassandra，SOLR，Gemfire，Neo4J，MongoDB，Envers，Redis，ElasticSearch，Couchbase的模块，以及Spring Data REST和Spring Data Commons的更新。
  • Spring Cloud联合负责人Spencer Gibb刚刚宣布了Spring Cloud CLI启动器 。这是我最喜欢的新实用程序之一-使用Spring Cloud可以更轻松地完成工作！安装并运行spring cloud configserver dataflow eureka h2 hystrixdashboard kafka zipkin您将在适当的端口上运行这些服务！简单。
  • Spring Cloud Data Flow忍者Thomas Risberg 刚刚宣布了Kubernetes 1.1 M2的Spring Cloud Data Flow
  • Spring Cloud忍者Ryan Baxter刚刚宣布了Spring Cloud Camden SR2 ，其中包括对各种Spring Cloud模块的错误修复。
  • 上周，我研究了如何显式地了解Spring Framework组件模型中鲜为人知的方面
  • William Witt参加了我的演讲之一，并修改了这些示例以在PCFDev Cloud Foundry环境中工作，并逐步将其与代码一起记录了下来-这是一个了不起的资源！ 干得好威廉！
  • 想同时使用JHipster和Stormpath？春季社区英雄Matt Raible的解决方案
  • Spring社区的朋友Michael Simons展示了一个非常有趣的外观，展示了将基于JOOQ的分析API作为Spring Boot和基于Spring MVC的REST API 。
  • Pivotal自己的Ben Wilcock将Spring Boot和Axon 2.0框架结合在一起使用非常好看。
  • 如果您想了解有关Axon 3.0的新版本的更多信息，请务必与Axon负责人Allard Buijze和我一起参加11月16日的网络研讨会！
