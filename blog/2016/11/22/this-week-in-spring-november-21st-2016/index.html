Spring 的这个星期-2016年11月21日

工程
乔什·朗（Josh Long）
2016年11月22日

欢迎来到 Spring 本周的另一期！这周我回家了！毕竟，这周在美国的感恩节。当我说我们感谢您这个地球上最美好的社区时，我确定我会代表整个Pivotal团队发言。非常感谢，如果您正在庆祝感恩节，那么也祝您感恩节快乐！当您吃完饭时-几乎不能睁开眼睛-我希望您能找到一个舒适的扶手椅，并在本周的综述中接受一些内容。

  • 优秀的David Syer博士并不总是会做视图模板， 而是在做模板时。
  • 观看Spring Security主管Rob Winch和忍者Joe Grandja的重播，他们期待Spring Security 4.1
  • 延续我们的Spring Tips系列，上周我们将Spring Cloud Data Flow视为快速构建流处理解决方案的一种方式
  • 欢呼！祝贺Apache Geode项目，他们刚刚看到他们的项目被提升为Apache顶级项目
  • 我真的很喜欢Spring Cloud忍者Ryan Baxter的这篇文章，该文章研究了Spring Cloud Sleuth的分布式跟踪
  • 想麻烦吗？然后查看这个有趣的新的Trouble MakerSpring Boot器
  • 公司成长后，他们就变成了云原生的网络公司， 行业分析师RedMonk的这篇有趣的帖子断言
  • 了解如何从Spring Data存储库中流式传输数据
  • 我真的很喜欢Stormpath的这篇文章，其中展示了如何使用基于React.js的前端构建Spring Boot应用程序，所有这些都使用Stormpath完美地保护了
  • 上周，我的朋友Allard Buijze（以CQRS为中心和Spring Boot认证的Axon Framework 3的负责人）和我一起研究了构建CQRS应用程序。一位Pivotal同事表示，这是一次不错的网络研讨会，并认为我们的听众可能想了解更多，并推荐了Microsoft的这一深层次资源，《 The CQRS Journey》
  • Ceylon语言的最新版本1.3.1已发布， 并具有与Spring Boot等技术的互操作性
  • 最新版本的RabbitMQ-3.5.8已发布
