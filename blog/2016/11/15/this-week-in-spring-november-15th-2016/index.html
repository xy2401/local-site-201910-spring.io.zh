Spring 的这个星期-2016年11月15日

工程
乔什·朗（Josh Long）
2016年11月15日

欢迎来到 Spring 本周的另一期！时间一定在飞！这个星期我们有很多事情要做。您可以相信我们盯着2017年吗？我们有很多内容需要介绍，让我们开始吧！

  • 我希望您能加入Axon框架负责人Allard Buijze，明天我将介绍如何使用Axon Framework和Spring Boot构建以CQRS为中心的应用程序
  • 还有更多的SpringOne Platform 2016讲座，如果您对Spring和反应式编程如何结合感兴趣，您将希望直接从Rossen听到有关团队在哪里开始的旅程！
  • 任何在微服务领域工作的人都应该观看这个令人惊叹的 SpringOne Platform 2016重播，向我们展示Oliver Gierke如何将域驱动设计概念应用于创建REST API。
  • 请务必查看我们自己的Stephane Nicoll和Vaadin的Matti Tahvonen的博客以及相应的网络研讨会，他们将研究如何构建与后端Spring服务无缝集成的美观的Web应用程序用户界面。
  • Spring IO Platform负责人Andy Wilkinson刚刚发布了Spring IO Platform Athens SR1，其中包含许多更新
  • Spring Data Geode负责人John Blum刚刚发布了Spring Data Geode 1.0.0。发行发行
  • Spring Security主管Rob Winch 刚刚宣布了Spring Security 4.2.0 。
  • Spring Cloud Data Flow联合创始人Mark Pollack博士刚刚宣布发布Spring Cloud Data Flow 1.1 RC1
  • 上周，在Spring Tips系列的另一部分中，我们研究了如何使用Spring Cloud Stream构建基于消息的微服务。
  • Spring Data Geode负责人John Blum刚刚发布了Spring Data Geode 1.0.0。发行发行
  • 想了解事件来源？看看与Jakub Pilimon一起录制的这个时髦记录的Google Hangout
  • Spring Integration和Spring AMQP负责人Gary Russell刚刚宣布了Spring AMQP和Spring Integration Core和Java DSL维护版本可用
  • Spring Framework 负责人兼艺术大师Juergen Hoeller刚刚宣布
    Spring Framework 5.0 M3发布
  • 使用此功能强大的新Atomist生成器创建新的Spring Cloud Config Server
  • 我听说过好东西约阿尔瓦罗·桑切斯的Grails的Spring Security的REST插件
  • 很高兴看到Spring Boot 在Thougthworks Radar的最新一期中的“ 采用”栏中保持领先地位
  • 这篇日语帖子看起来在介绍Spring Security方面做得很好
  • 这根本不是特定于Spring的，但是鉴于本周我们还有很多其他有关事件源，有限上下文和微服务的内容， 您可能会喜欢Russ Miles关于事件源的最新思路
  • 一定要参加Tino Tereshko（Google大数据主管）和Joshua McKenty（Pivotal平台工程高级总监）的网络研讨会，讨论如何在运行于Google Compute Engine的Pivotal Cloud Foundry中使用Google的BigQuery API
