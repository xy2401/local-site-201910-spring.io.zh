春天的这个星期-2016年11月29日

工程
乔什·朗（Josh Long）
2016年11月29日

欢迎来到春季本周的另一期！ 我简直不敢相信今年已经过去了！这周我在澳大利亚墨尔本参加YOW！发布会 ，然后是一周，然后是布里斯班，然后是悉尼，以举办同一场演出的下一版。澳大利亚是我离开“本地”时区以来最遥远的地方-因此，即使我总是在每个星期二的春季发布“本周”，我还是很高兴现在仍然是欧洲西部的任何一个星期一！今晚，我将加入我的朋友Intellij的Trisha Gee，我们将在Melbourne JVM User Group上发言。我很高兴第一次来到这里，帮助将Spring推倒。如果您在附近，请打个招呼（@starbuxman）！

Spring Cloud Dataflow团队的发布周非常艰巨！
Spring Cloud Data Flow负责人Mark Pollack博士刚刚宣布了Spring Cloud Data Flow 1.1。GA
* Spring Cloud Task主管Michael Minella刚刚宣布了Spring Cloud Task 1..1.0，其中更新了错误处理，对分区Spring Batch进行了改进Job ，外部执行ID持久性，其他数据库支持等等。
* Spring Cloud Data Flow忍者Thomas Risberg刚刚宣布了Kubernetes 1.1的Spring Cloud Data Flow。RC1改进了对运行批处理和流处理的支持，同时将其部署为Kubernetes作为服务结构。
* Spring Cloud Data Flow忍者Janne Valkealahti刚刚宣布了适用于Apache YARN 1.1.0 RC1的Spring Cloud Data Flow

活跃的一周
* Spring Data的忍者Mark Paluch刚刚发表了一篇超酷的文章，介绍了Spring Data中对响应式编程的一些即将支持，远远超出了Spring Data中对异步类型的有限支持。我个人等不及要看到MongoDB与@EnableReactiveMongoRepositories ！
* Spring Data负责人Oliver Gierke刚刚宣布了Spring Data Kay的第一个里程碑，这不仅仅是一个新版本：它将Java 8的基准修订版更新，包括对Spring Data MongoDB，Cassandra和Redis中的反应式编程的支持，以及这么多

  • Spring Sesssion的负责人Rob Winch刚刚宣布了Spring Session 1.3.0。RC1包括一流的Spring Security RememberMe集成，使用Lettuce（Redis驱动程序）的更新，OrientDB支持，性能改进等等
  • 上周，在进行中的Spring Tips专栏中，我们将Cloud Foundry（开源Cloud Native Platform） 视为一种快速发布软件的方法

  • 我真的很喜欢这个视觉演示，介绍了OpenCredo的CQRS和事件源。

  • 我的朋友西蒙·布朗（Simon Brown） 撰写的有关软件体系结构的优秀著作是（仅在有限的时间内！）免费提供 ！尽你所能！
  • 我们的朋友Matta Tahvonen是Vaadin的开发倡导者，他使用Hibernate Spatial，Vaadin和Spring Boot组成了一个很好的例子
