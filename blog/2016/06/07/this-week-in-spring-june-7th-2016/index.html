Spring 的这个星期-2016年6月7日

工程
乔什·朗（Josh Long）
2016年6月7日

欢迎来到春季本周的另一期！已经是六月了！哪里。有。的。时间。走？？本周，我将在伊利诺伊州芝加哥参加芝加哥编码员会议，在波士顿和新罕布什尔州进行客户访问，在英国伦敦的Devoxx UK和在爱沙尼亚的Talin的Geekout EE。如果您在附近，请务必打个招呼 ！现在，本周有很多内容需要介绍，让我们开始吧！

  • Spring Cloud Services忍者Scott Frederick刚刚宣布了Spring Cloud Cloud Foundry Service Broker，这是一个为Cloud Foundry平台构建服务代理的好框架
  • Spring Boot和Spring Cloud联合创始人（好人）Dave Syer博士刚刚发表了有关反应式编程前景的系列文章中的第一篇文章
  • Spring Integration忍者Artem Bilan 刚刚宣布了Spring Integration Kafka 2.0.0。RC1现在可用
  • Artem 还 宣布了Apache Kafka 1.0的春季版。RC1
  • Spring Cloud Streams主管Marius Bogoevici刚刚宣布了Spring Cloud Stream 1.0.1。发布 。它提供了附加的配置选项，并且能够贡献绑定而不会覆盖默认值和一系列小的更新。
  • Reactor项目已经走了很长一段路，因此我们已经弃用了一些较旧的Reactor指南
  • Spring Integration和Spring AMQP忍者Artem Bilan 也刚刚宣布Spring AMQP 1.6 GA（1.6.0。RELEASE）现在可用
  • 这个关于将Hibernate集成为类级和跨参数约束使用Spring Expression Language的博客非常有趣 。
  • 我真的很喜欢Michael Redlich在Spring Statemachine 1.1上发布的 InfoQ帖子
  • New Stack上有一篇很不错的文章，名为Cloud Foundry：旨在生存
  • 想快速介绍MQTT吗？
  • 这个关于使用Spring Boot和CXF开发以文档为中心的基于JAX-WS SOAP的服务的博客很有趣
