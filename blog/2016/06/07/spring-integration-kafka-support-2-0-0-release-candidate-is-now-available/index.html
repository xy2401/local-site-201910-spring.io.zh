Spring Integration Kafka支持2.0.0版本候选版现已发布

发布
阿尔特姆·比兰（Artem Bilan）
2016年6月7日

我很高兴地宣布，版本2.0的spring-integration-kafka （Spring Integration Kafka支持）发行候选版现已可用。

工件org.springframework.integration:spring-integration-kafka:2.0.0.RC1 。

该版本基于刚刚发布的Spring for Apache Kafka候选发布者1.0.0。RC1 。

自上一个里程碑1以来，变化不大。只是一些常规的内部修复和升级，以适应Apache Kafka的最新Spring更改。

Spring Integration Java DSL 1.2及其Kafka09 Factory也已更改，以赶上Apache Kafka API的最新Spring。 DSL的里程碑1即将面世。

我们将发布2.0.0.RELEASE在未来一周左右，与春节一起为Apache 1.0.0卡夫卡。发布。因此，这是一个尝试一下并报告您发现的任何问题的机会。
