Spring 的这个星期-2016年6月14日

工程
乔什·朗（Josh Long）
六月14，2016

欢迎来到春季本周的另一期！这周我将在伦敦和阿姆斯特丹（参加史诗般的GOTO阿姆斯特丹会议以及JDriven聚会和Dutch Spring用户组 ）。像往常一样，我们有很多内容需要介绍，所以让我们开始吧！

  • Spring Framework 4.3包含了各种各样的细节，现在可用 。该版本为各种库（Hibernate ORM，Jackson，Netty，Undertow，Tomcat）提供了各种很酷的更新，并支持我非常兴奋的两个功能：在Spring托管组件的构造函数中自动注入，其参数被其他对象明确解析。应用程序上下文中的类型以及将类型注入到应用程序中的能力@Configuration类的构造函数。
  • Marius Bogoevici刚刚宣布了对Spring Cloud Stream 1.0.2的重要更新。RELEASE，现在可以立即使用
  • 优秀的David Syer博士发布了一系列有关反应式编程前景的系列文章中的第二篇。这篇文章专门针对Project Reactor。看看这个！
  • Spring Integration忍者Artem Bilan刚刚宣布了Spring Integration 4.3。GA现在可用 ！这个版本包含了很多很酷的东西，所以赶快抓住机会！
  • 我真的在挖这个博客， 了解Cloud Foundry如何将世界免于割
  • 想要快速构建用于在MySQL中编辑数据的UI？Vaadin博客有一个不错的博客，介绍了使用Vaadin和Spring Boot进行构建的方法
  • 这看起来很有趣： npm用于通过JavaScript使用Spring Data REST API的模块
  • 像往常一样，我们的好友Baeldung 在将JOOQ与Spring Boot结合使用的这篇文章中再次强调了这一点。
  • 这是Bartosz Jedrzejewski的有趣帖子
    关于通过Spring Cloud演示的微服务隔离和重用功能的说明。
  • Apereo CAS应用程序旨在成为企业单点登录解决方案，刚刚发布了CAS版本5.0.0。M2 。该版本建立在Spring Boot和Spring Cloud的基础上，因此，该版本中的许多工作都提供了智能的自动配置。看看这个！
