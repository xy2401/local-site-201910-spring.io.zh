用于Cloud Foundry 1.0.0的Spring Cloud Data Flow。M3发布

发布
埃里克·博塔德
2016年6月21日

我很高兴宣布1.0.0。面向Cloud Foundry的Spring Cloud Data Flow的M3版本，这是一个团队的工作，其中包括许多新功能：

这是第三个里程碑，它基于Spring Cloud Data Flow的最新RC1版本，还增加了Cloud Foundry的特定改进。一些重点包括：

  • 与Cloud Foundry的全局环境属性（例如ORG，API和URL）保持一致，而不是以前的命名变体

  • 添加安全集成以针对与Cloud Foundry兼容的OAUTH后端服务器进行身份验证

  • 发布安全配置，以便在连接到与Cloud Foundry兼容的OAUTH后端服务器时更轻松地覆盖

  • 在有和没有水龙头的实时流水线管道中对Cloud Foundry进行性能测试的初步迭代

  • 基于Spring Cloud Data Flow核心项目的功能切换支持，有条件地完善了每个版本中包含的功能

自行开始使用此新版本的一种好方法是遵循参考文档的本部分 。

有关更改和改进的完整列表，请参阅1.0.0。M3 版本以及Cloud Foundry部署程序版本详细信息 。
