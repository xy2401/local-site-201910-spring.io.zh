Gradle依赖管理插件0.6.0发布

发布
安迪·威尔金森
2016年6月30日

我很高兴地宣布Gradle依赖性管理插件 0.6.0。现在可以从Maven Central，JCenter和Gradle Plugin Portal中获得RELEASE。

Gradle 3.0兼容性

此新版本与Gradle 3.0兼容。在撰写本文时，它已经针对3.0 M2进行了测试，并且0.6.x系列将继续跟踪Gradle 3.0的开发，以尽可能保持兼容性。

对Bom属性覆盖的更精细控制

以前，可以使用项目属性来覆盖导入的Bom的属性。这个新版本增加了对覆盖属性的支持，作为导入的一部分。 bomProperty可用于一次覆盖一个属性（并可多次使用以覆盖多个属性）：

dependencyManagement {
    imports {
        mavenBom('com.example:example-bom:1.2.0') {
            bomProperty 'foo.version', '1.1.1'
            bomProperty 'bar.version', '2.0.3'
        }
    }
}

另外，可以将bomProperties与属性覆盖图一起使用：

dependencyManagement {
    imports {
        mavenBom('com.example:example-bom:1.2.0') {
            bomProperties([
                'foo.version': '1.1.1',
                'bar.version': '2.0.3'
            ])
        }
    }
}
