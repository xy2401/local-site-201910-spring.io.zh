Spring Integration AWS 1.0.0。M1现在可用

发布
阿尔特姆·比兰（Artem Bilan）
2016年4月14日

亲爱的 Spring 社区，

我很高兴代表Spring Integration Team宣布Spring Integration AWS 1.0.0.M1现在可以从Spring里程碑资料库下载。

从0.5.0的预发行版本开始，我们已经完全重新设计了核心部分，以简化编程模型并引入了新组件。
当前，Spring Integration AWS以其基础上的Spring Cloud AWS项目为基础。

感谢所有为这一里程碑做出贡献的人！

该里程碑的重点包括：

  • Amazon Simple Storage Service（S3）适配器

  • Amazon Simple Queue Service（SQS）适配器

  • Amazon Simple Notification Service（SNS）适配器

请参阅项目自述文件以获取更多信息。

下一步

  • 添加AWS集成测试环境；
  • 添加Kinesis适配器；
  • 添加DynamoDB适配器并MessageStore实施；
  • 添加SimpleDb适配器并MessageStore实施；
  • 添加Spring Cloud Stream绑定器；
  • 为AWS适配器添加Spring Cloud Stream应用程序;
