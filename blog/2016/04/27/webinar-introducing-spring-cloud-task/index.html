网络研讨会：Spring Cloud Task介绍

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年4月27日

演讲者：Pivotal的Michael Minella

云的主要承诺之一就是灵活性。如今，大多数部署到云的应用程序都是长期运行的流程，它们利用云扩展的灵活性。但是计算充满了短暂的任务，这些任务可以启动，完成工作然后终止。这些任务是出色的云用例，因为可以快速调配和回收资源。

在此网络研讨会中，我们将探索Spring Cloud产品组合中的一个新项目Spring Cloud Task，这是一个用于开发和编排短期微服务的新框架。我们将探索各种用例，构建您的第一个任务，并讨论如何使用各种技术来编排任务。最后，我们将窥视Spring Cloud Task项目的路线图。
