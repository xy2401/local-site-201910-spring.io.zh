适用于Apache Kafka 1.0 Milestone 2的Spring

发布
阿尔特姆·比兰（Artem Bilan）
2016年4月11日

我很高兴地宣布spring-kafka （Apache Kafka的春季版）第二里程碑1.0现在可用。

文物org.springframework.kafka:spring-kafka:1.0.0.M2和org.springframework.kafka:spring-kafka-test:1.0.0.M2在Milestone存储库中可用。

自第一个里程碑以来的更改包括：

  • KafkaTemplate.send()方法现在返回ListenableFuture>
    异步跟踪发布到Kafka主题。
  • 的MessagingMessageListenerAdapter （对于@KafkaListener方法）现在可以推断出payload从方法参数转换。
  • 一种StringJsonMessageConverter ，可以使用Jackson使用Jackson的类型推断从JSON内容构建目标对象ObjectMapper 。
  • RECEIVED_标头已从中添加Listener避免冲突（重新路由） topic一样的messageKey发送时，例如使用Spring Integration Adapters。
  • 更多测试工具。

有关更多信息，另请参见参考手册 。

Spring Integration Kafka支持

不要错过Spring集成Kafka 2.0的第一个里程碑 ，它完全基于此Spring for Apache Kafka基础。

我们也在研究版本1.1已经使用Reactor的Reactive Streams实现与Kafka的交互Flux和Mono抽象。同时，具有Kafka Streams功能的Apache Kafka 0.10.0 ，对于Apache Kafka 1.1，这也是Spring的不错补充。
