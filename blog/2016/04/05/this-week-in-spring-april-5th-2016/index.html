Spring 的这个星期-2016年4月5日

新闻与活动
乔什·朗（Josh Long）
2016年4月5日

欢迎来到 Spring 本周的另一期！和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Spring Security主管Rob Winch刚刚宣布了Spring LDAP 2.1.0。RC1已发布！ 获取的位，并尝试一下。
  • Oracle惊人的Geertjan Wielenga进行了一个很棒的 100％现场演示， 探讨了如何将Spring Boot与NetBeans结合使用
  • 我喜欢Spring Cloud Data Flow！无论您是想从简单的类似于actor的组件中描述和组成复杂的处理流，还是要编排流处理解决方案，管理诸如大规模数据提取等任务，还是其他方面，Spring Cloud Data Flow可能都是您想要的。快速浏览一下如何站起来的Spring Cloud Data Flow服务器，shell和我组合在一起的自定义模块
  • Spring Cloud团队成员Marcin Grzejszczak在最近发布的Spring Cloud的风口浪尖上，研究了Spring Cloud Sleuth中的更改和令人敬畏的事物，Spring Cloud Sleuth （ Spring Cloud中的分布式跟踪抽象）
  • 别错过令人惊叹的Danny Hyun 对Spring Boot和Ratpack的反应式编程的史诗性观看 。
  • Rossen Stoyanchev和Stephane Maldini对他们从SpringOne2GX 2015 开始的反应式编程简介的后续讲话现在在Spring Developer频道上进行。检出反应式Web应用程序
  • 来自汉莎航空的Michael Wilmes和来自Couchbase的Laurent Dougin在他们的SpringOne2GX 2015演讲， Spring Data Daily Double-Couchbase和Neo4J的演讲中加倍关注NoSQL和Spring
  • 迈克尔·沙哈格（Michael Scharhag）对Spring Retry进行了一次非常酷的展示，这是我最喜欢的项目之一 。根据我们的下载数量，Spring Retry 非常受欢迎！
  • Pivotal数据科学团队的这篇文章探讨了他们如何将完全不同的， 多种语言的服务与RabbitMQ和Spring XD集成在一起，以实现评分服务
  • 我们在Codecentric的朋友又来了一次，这一次让我们对Spring Cloud Stream有了一个很好的外观
  • 正在进行的系列中的一个不错的新条目，该条目使用Spring Cloud的Netflix Feign支持
  • MyBatis Spring Boot启动器不断进步。我很高兴看到这一事实（并为此做出了贡献），因为MyBatis在中国（中国范围内）被大量使用。否则会讨厌ORM技术（例如Hibernate）的服务。
  • 本周[Spring Initializr]的世界上有许多激动人心的新选择！选择Spring Boot 1.3.4 (SNAPSHOT)然后您会看到Zipkin UI （将其与Zipkin Stream Server获得即时跨度收集和可视化服务）， Apache Camel ， Spring Cloud Data Flow Shell （ Spring Cloud Data Flow Local Server ）。
