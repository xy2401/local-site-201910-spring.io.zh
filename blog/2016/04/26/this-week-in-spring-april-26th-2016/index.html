春天的这个星期-2016年4月26日

工程
乔什·朗（Josh Long）
2016年4月26日

欢迎来到春季本周的另一期！ 本周，我在匈牙利的布达佩斯参加令人惊叹的Craft Conf 。这个节目确实是一个非常特别的节目。对我而言，特别是，首先，我将与行业巨头如Adrian Cochroft，Jez Humble，Sam Newman和Kyle Kingsbury一同加入，并与来自Pivotal的我的两个朋友-Andrew Clay Shafer和Bridget Kromhout一起加入 （还有泰坦）！我很高兴能参与其中，甚至更多，以至于我几乎无法控制它，并且我建议您考虑自己的行程，如果方便的话。

现在，我们有很多工作要做，让我们开始吧！

  • Spring Integration忍者Artem Bilan 刚刚宣布了Spring Integration 4.3 M2可用 。新版本支持在运行中的Spring Integration系统中所有组件的图形化枚举，并在FileReadingMessageSource使用JDK 7中的DirectoryWatcher（最终！我甚至开发了用于本地实现和JNI来应对文件系统的变化，几年前和使用的 ，以支持文件的消息源在2009年的代码！），谷歌云邮件扩展（XEP）到XMPP，静态可编译Groovy脚本，等等多得多！获取位并检查出来！
  • Janne Valkealahti刚刚宣布了Spring Statemachine 1.1.0。RC1 。新版本包括许多小功能和一项重大功能：Eclipse Papyrus UML建模支持！
  • Spring Batch主管Michael Minelle刚刚宣布Spring Batch 3.0.7现已可用！ 。此版本包括，如果你正在使用从最新的Spring数据发布新的依赖，是一个非常推荐的更新几个有用的修复。
  • Spring Security负责人Rob Winch刚刚宣布了Spring Security 4.1.0。RC2 。新版本包含新功能！看一看，让我们知道！
  • Spring和Spring Reactive忍者Sebastien Deleuze的文章很好地介绍了为何Reactive类型有用，以及如何根据团队在使用即将到来的Spring Framework 5 Reactive支持时所学到的知识，将它们与其他异步类型进行比较。
  • Spring Boot和Spring的共同负责人David Syer博士刚刚宣布了Spring Cloud Brixton，RC2的最新版本 ，其中包括许多很酷的新功能和改进功能。绝对可以尝试一下！
  • 上周我参加了西门枫，丹尼尔·布莱恩和Markus艾泽勒的第一次正式ZeroTurnaround研讨会 -这一项上微服务-这是一个很大的乐趣！视频已经播出，但是不幸的是，当我们录制GotoWebinar基础设施时，它似乎已经崩溃了。某一时刻，马库斯无法共享他的屏幕（尽管恢复了足够长的时间以便他以后可以工作），并且录制的视频还是很差。 ：/无论如何，它仍然是可见的，尽管您必须斜视才能看到类型。请享用！
  • 查看我们的朋友Julien Dubois的博客，了解如何在Heroku上使用JHipster出色的新Spring Cloud集成。
  • 我们自己的Kenny Bastani 发布了有关如何将Spring Cloud与Reactor一起使用以解决一些CQRS和事件源用例的文章。
  • Cloud Foundry现在拥有一个入门资源，可全面了解Cloud Foundry及其生态系统The Cloud Foundry Planet 。阅读此内容，然后再继续，并修复您的Cloud Foundry生态系统。
