春天的这个星期-2016年4月12日

工程
乔什·朗（Josh Long）
2016年4月12日

欢迎来到春季本周的另一期！这个星期我在潮湿而美好的纽约市。在美国，这已经差不多是纳税日了！ew！ 时间肯定飞逝。其他（明显更令人愉悦的）年度地标也将很快在这里出现，包括我们的大型SpringOne Platform活动 ，请立即预订门票！

另外，我将与我的朋友Simon Maple，Daniel Bryant和Markus Eisele 一起参加4月19日举行的微服务网络研讨会 -加入我们！它应该很有趣，而且-希望-很有趣！

和往常一样，我们有很多内容需要介绍，让我们开始吧！

  • Spring Data Hopper上线了 ！此版本包括395张已关闭的门票，代表了大量工作！取得点点滴滴，立即尝试！
  • Spring Integration和一般消息传递忍者Artem Bilan刚刚宣布了Spring Integration Kafka 2.0系列的第一个里程碑 。此版本代表对Spring Kafka中提供的支持的完全重写，基于Producer和Consumer Kafka 0.9x中的API。检查一下，踢轮胎并提供反馈！
  • Spring Cloud Stream负责人Marius Bogoevici刚刚宣布了Spring Cloud Stream 1.0RC2 。此版本包括对ObjectMapper在转换对象，标题嵌入和更好的文档时。
  • Spring Security和Spring Session负责人Rob Winch 刚刚宣布了Spring Session 1.2.0 RC2
  • Spring框架负责人Juergen Hoeller刚刚宣布了Spring Framework 4.3 RC1 。这是第一个功能完整的Spring Framework 4.3候选发行版，现已上市，并已加载新功能！查看博客以获取详细信息，获取细节并尽情享受！千万不要错过这个！
  • 春季忍者Greg Turnquist刚刚宣布了春季WS 2.3.0 。许多较旧的依赖项已更新，这反映出重大更新。确保您已参考变更日志。
  • 在这种情况下，保存惊人的最好的SpringOne2GX回放之一，在这种情况下来自惊人的Matt Raible-不要错过Spring项目中的Elimination XML
  • 来自C24的John Davies是Spring团队的长期合作伙伴，并且是Spring集成，数据技术和金融服务的领先专家，他谈到了Spring中企业中FastData的微服务。
  • 切勿错过Stephane Maldini，Glenn Renfo和David Turanski，他们在讨论Spring XD的高性能流处理时展示了春季中有关响应处理的最早工作。
  • XavierPadró刚刚用Spring Data MongoDB组成了一个有关数据聚合的博客
  • 我真的很喜欢Domain Driven Design作者Eric Evans博士的演讲
  • Benedikt Ritter组成了一个不错的博客，介绍Spring Boot的@ConfigurationProperties
  • Spring社区的长期朋友Joris Kuipers 在Spring Session的并发环境中写了一篇不错的关于会话控制的博客
  • 我喜欢James Ward的博客文章，该博客文章在不到十分钟的时间内引导了从代码到云的完整Spring Boot开发经验。
