    工程
    乔什·朗（Josh Long）
    2016年2月23日

    欢迎来到春季本周的另一期！我们已经在盯着2月的最后几个星期！

    • Spring Data负责人Oliver Gierke 刚刚发布了包含45个错误修复程序的Spring Data Gosling SR4 。趁热拿到！
    • 想更多地了解Spring Data Gosling？看看Spring Data领导Oliver Gierke的介绍的重播， 还有新内容
    • Spring Integration 4.3的第一个里程碑，4.x系列的最后一个里程碑已经到来！在Spring Framework 5之前，有许多小的新功能可以帮助您解决问题， 那里的反应式支持成为Spring集成5的基准
    • 让我们为高科技女性倾听吧！Ludwine Probst和Ilayaperumal Gopinathan为您带来了SpringOne2GX 2015重播： 用于大数据处理的Apache Spark
    • 我们的好朋友Sergi Almar是2016年5月19日至20日在美丽的巴塞罗那举行的Spring IO会议的组织者，就如何使用Spring WebSocket从零变身为英雄进行了另一番精彩演讲
    • 我们的Securitay忍者Rob Winch从SpringOne2GX 2015重放的基础知识开始： 动手进行Spring Security 4.1
    • Craig Walls的史诗级Spring Boot in Action现已上市，并暂时打折！趁热拿到！
    • 我们的朋友Eberhard Wolff在JAX上就使用Spring Boot和Spring Cloud构建微服务进行了很好的演讲：请查看 ！
    • 我想花点时间向您警告互联网的危险， 此类帖子仅旨在鼓励禁用基于Spring Boot的应用程序。这是一个可怕的主意！您将如何获得像这样的Nyan Cat横幅和在这里建议的ImageBanner PR的惊人横幅？我原本不打算在春季本周中收录本文 （原则上来说！）但是受到史诗级的 IntellIJ IDEA开发人员YannCébron的 史诗般的响应的安慰， 这表明以前禁用Spring Boot标语的复选框不再启用！ ：D
    • 有一个有趣的示例， 基于Spring Boot配置OpenUI5
    • 我们的朋友Nicolas Frankel在Spring Boot上写了一个不错的博客来管理日志
    • 是否想将Apache CXF与Spring Boot结合使用（2016年）来构建SOAP Web服务？
    • 我对InfoQ进行了有关Spring Boot，Spring Cloud和微服务的总体采访
    • 说到IntellIJ，新版本16的第一个预览版已经发布，其中包含对Spring MVC，Spring框架和Spring Boot的更新支持以及对Kotlin的支持！如果发现任何内容，请检查位和文件问题
