SpringOne2GX 2015重播：适用于大数据处理的Apache Spark

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年2月23日

记录在SpringOne2GX 2015
演讲者：Ludwine Probst＆Ilayaperumal Gopinathan
大数据轨道
幻灯片： http ： //www.slideshare.net/SpringCentral/apache-spark-for-big-data-processing

今天，我们生活在大数据世界中。Hadoop和MapReduce在大规模数据处理领域中占主导地位。但是，MapReduce模型显示了其对各种类型的处理的局限性，尤其是对于机器学习领域中经常遇到的高度迭代的算法。

Spark是一种内存数据处理框架，与Hadoop不同，它提供了对大型数据集的交互式实时分析。此外，Spark具有比Hadoop更灵活的编程模型，并提供了更好的性能。

在本次演讲中，我们旨在提供Spark的画像并浏览其生态系统，尤其是带有具体示例的Spark Streaming和MLlib。我们还将展示如何将Spark与Spring XD结合使用，从而使您能够利用每个平台的优势。
