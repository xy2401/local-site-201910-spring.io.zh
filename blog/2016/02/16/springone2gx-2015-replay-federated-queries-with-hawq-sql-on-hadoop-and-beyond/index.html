This presentation will show different integration approaches that allow integration and data exchange between HAWQ and GemFire. SpringOne2GX 2015重播：使用HAWQ进行联合查询-Hadoop和更高版本上的SQL

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年2月16日

记录在SpringOne2GX 2015
主讲人：克里斯蒂安·特佐洛夫（Christian Tzolov）
大数据轨道
幻灯片： http : //www.slideshare.net/SpringCentral/federated-queries-with-hawq-sql-on-hadoop-and-beyond-53208753

在大数据领域，Pivotal提供了两个功能强大的数据处理工具，即HAWQ和GemFire。HAWQ是可伸缩的OLAP SQL-on-Hadoop系统，而GemFire是类似于OLTP的内存中数据网格和事件处理系统。本演示将展示允许HAWQ与GemFire之间进行集成和数据交换的不同集成方法。在指导您完成不同集成策略的实现时，将分享在某些用例中应用Spring Boot和Spring XD的实践经验。我们将展示一个利用SpringXD摄取GemFire数据并将其存储在HDFS中的集成路径，以及使用Spring Boot为HAWQ Web Table集成场景实现REStful代理的好处。
