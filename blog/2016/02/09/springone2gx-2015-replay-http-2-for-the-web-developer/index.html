SpringOne2GX 2015重放：Web开发人员的HTTP / 2

新闻与活动
彼得·汉弗莱（Pieter Humphrey）
2016年2月9日

记录在SpringOne2GX 2015
演讲人：Brian Clozel
Web / Javascript跟踪
幻灯片： http ： //www.slideshare.net/SpringCentral/http2-for-the-web-developer

HTTP / 2规范就在这里-客户端服务器正在努力实现它，而浏览器支持每天都在增加……JVM世界已经在Servlet 4.0和JDK9规范中对其进行了开发。但是我们的开发人员仍在尝试理解它的含义。

我们是否应该重新考虑构建JavaScript应用程序并对其进行优化以实现前端性能的方式？我们的服务器应用程序将具有新的角色和功能吗？

在本讲座中，我们将研究HTTP / 2的主要功能，以了解它如何显着改善Web上的延迟。我们还将通过具体示例讨论如何将HTTP / 2集成到我们的前端和Spring Web应用程序中。
