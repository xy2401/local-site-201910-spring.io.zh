Spring 的这个星期-2016年2月9日

工程
乔什·朗（Josh Long）
2016年2月9日

欢迎来到春季本周的另一个装置！本周，我在美丽的瑞典斯德哥尔摩举行的JFokus会议上 。在瑞典，这也是星期二 的节假日 ，习俗是过度放纵，所以当我键入此内容时，我正在吃Semla 。希望您也喜欢本周的史诗般的综述！

  • 终于揭开了世界上最糟糕的秘密！我们一直在为Spring MVC开发一个反应式运行时，您可以开始尝试了！看看这个关于Spring 5的反应式Web编程野心的惊人博客 ！
  • Spring Cloud Stream负责人Marius Bogoevici刚刚宣布了Spring Cloud Stream 1.0.0。M4除其他事项外，默认情况下是发布/订阅语义，它支持使用者组概念，提供了简化的绑定器SPI，可重置的Kafka偏移量等等 ！看看这个！
  • 2.0.0。Spring Data Couchbase的发布刚刚下降！ 。我怀疑会有博客或宣布它的东西，但与此同时，请抓住机会并尝试一下！
  • 优秀的David Syer博士为Spring＆OAuth教程添加了新的章节
  • 我整理了一些有关一起使用Activit和Spring Integration的文章
  • 我们的朋友Nicolas Frankel整理了一篇不错的文章，揭开了Spring Boot自动配置功能的神秘面纱
  • 如果您无法在SpringOne2GX 2015 上看到Brian Clozel和StéphaneNicoll的有关在Web层中使用Spring Boot的演示 ，那么您一定会喜欢它在线上提供
  • Oracle Netbean的领导者和全面的伟人Geertjan Wielenga和我在谈论与Spring Boot和Netbeans的集成， 他在一个出色的Oracle博客上写了他的故事
  • Heroku文档最近已更新，包括有关部署Spring Boot应用程序的扩展讨论
  • 我们的朋友Michael Simmons 开发了一个外观非常实用的WRO4J Spring Boot启动器
  • Dan Vega整理了一个很好的小视频，介绍如何使用Spring Boot发送电子邮件
  • 我们的朋友Petri Kainulainen再次发表了一篇不错的文章，内容涉及从Gradle获取Spring Batch应用程序所需的依赖项
  • 我喜欢这篇日语文章，内容涉及在Spring Boot应用程序中使用HikariCP以及公开其指标
  • 我喜欢我们自己的Mark Heckler 在Spring Boot上为IoT开发人员 撰写的这篇文章。这是系列中的第一篇，敬请期待！
  • Broadleaf Commerce上有关防止Spring会话劫持的精彩文章
