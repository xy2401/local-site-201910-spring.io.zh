Spring Data 发布训练器Hopper M1发布

发布
奥利弗·德罗博姆
2016年2月12日

我很高兴代表Spring Data团队宣布Hopper发布火车的第一个里程碑。该发行会固定发行250张门票！最重要的新功能是：

  • 升级到Querydsl 4。
  • Spring Data Neo4j 4.1，Spring Data Couchbase 2.1和Spring Data Solr 2（在Solr 5上）的集成。
  • 支持Redis集群。
  • 支持对JPA和MongoDB中的存储库查询方法的预测（有关详细信息，请参见示例 ）。
  • 将Spring Data Envers添加到发布系列中（以前是单独维护的）。

在我们的发行培训Wiki中找到经过策划的变更日志，或浏览JIRA中完整的变更列表 。我们正在为3月初发布的候选版本进行射击，并在月底发布GA发布的版本。

达到里程碑的最简单方法是声明spring-data-releasetrain.version财产Hopper-M1在您的Spring Boot应用程序中。如果您使用的是Solr或Querydsl，请确保也按照示例所示升级这些版本。

我们期待您的反馈！
