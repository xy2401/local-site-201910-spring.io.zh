Spring Data 发布系列Ingalls RC1发布

发布
奥利弗·德罗博姆
2016年12月23日

我谨代表Spring Data小组宣布Spring Data火车Ingalls的第一个（也是最后一个）发布候选人。该版本提供了310多个已修复的问题（其中一些以前与Hopper服务版本一起发布）。最重要的新功能是：

  • 升级到Spring 4.3（.5）作为基线- ＃294 。
  • 支持轻松地从聚合根-DATACMNS-928将域事件公开为Spring应用程序事件。
  • 支持衍生查询方法DATACMNS-875中的存在投影。
  • 支持Javaslang Option ，关于仓库查询方法的收集和地图类型-DATACMNS-937 ， DATACMNS-940 。
  • 添加了Spring Data LDAP模块，以接管Spring LDAP的存储库实现。
  • 最新版本-DATAMONGO-1536中增加了对MongoDB聚合运算符的支持。
  • 现在，流执行将拒绝调用，而不会在JPA- DATAJPA-1023中进行周围事务。
  • 在Gemfire- SGF-106 ， SGF-516中支持基于注释的索引和到期。
  • 在Spring Data REST- DATAREST-573中支持CORS配置。

该版本当然包含更多功能。在此处找到固定票证的完整列表，并在我们的发行列车Wiki中找到有关Ingall最重要功能的更详细的更改日志。我们正计划在一月中下旬发布GA版本，其中包含一些错误修复和完善的文档资料。因此，现在是释放当前版本并帮助我们消除故障的最佳时机。
